# Solana (Français)

> Solana est la blockchain haute performance conçue pour l'adoption massive. Cette documentation fournit des guides complets et des références pour les développeurs qui construisent des applications blockchain évolutives sur Solana.

## Concepts Fondamentaux

- [Comptes](https://solana.com/docs/fr/core/accounts): Comment Solana stocke les données
- [Transactions](https://solana.com/docs/fr/core/transactions): Interaction avec le réseau Solana
- [Programmes](https://solana.com/docs/fr/core/programs): Contrats intelligents sur Solana
- [Adresses Dérivées de Programme](https://solana.com/docs/fr/core/pda): Adresses déterministes pour les comptes
- [Invocation Inter-Programme](https://solana.com/docs/fr/core/cpi): Comment les programmes appellent d'autres programmes
- [Frais](https://solana.com/docs/fr/core/fees): Coûts de transaction et frais de priorité

## Démarrage

- [Démarrage Rapide](https://solana.com/docs/fr/intro/quick-start): Créez votre premier programme Solana
- [Installation](https://solana.com/docs/fr/intro/installation): Configurez votre environnement de développement local

## Tokens

- [Bases des Tokens](https://solana.com/docs/fr/tokens/basics): Fondamentaux des SPL Token
- [Extensions de Token](https://solana.com/docs/fr/tokens/extensions): Fonctionnalités du programme Token-2022

## Méthodes RPC

- [getAccountInfo](https://solana.com/docs/fr/rpc/http/getaccountinfo): Récupérer les données du compte
- [getBalance](https://solana.com/docs/fr/rpc/http/getbalance): Obtenir le solde SOL
- [sendTransaction](https://solana.com/docs/fr/rpc/http/sendtransaction): Envoyer des transactions signées
