# Solana

> Solana is the high-performance blockchain designed for mass adoption, capable of processing thousands of transactions per second with sub-second finality.

This documentation provides comprehensive guides, references, and tutorials for developers building on Solana.

- [Accounts](https://solana.com/docs/ko/core/accounts): How Solana stores data in accounts
- [Transactions](https://solana.com/docs/ko/core/transactions): The fundamental building blocks for interacting with Solana
- [Programs](https://solana.com/docs/ko/core/programs): Smart contracts on Solana
- [Program Derived Addresses](https://solana.com/docs/ko/core/pda): Deterministic addresses for program-controlled accounts
- [Cross Program Invocation](https://solana.com/docs/ko/core/cpi): How programs invoke other programs
- [Fees on Solana](https://solana.com/docs/ko/core/fees): Transaction costs and priority fees
