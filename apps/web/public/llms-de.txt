# Solana (Deutsch)

> Solana ist die Hochleistungs-Blockchain für Massenadoption. Diese Dokumentation bietet umfassende Anleitungen und Referenzen für Entwickler, die skalierbare Blockchain-Anwendungen auf Solana erstellen.

## Kernkonzepte

- [Konten](https://solana.com/de/docs/core/accounts): Wie Solana Daten speichert
- [Transaktionen](https://solana.com/de/docs/core/transactions): Interaktion mit dem Solana-Netzwerk
- [Programme](https://solana.com/de/docs/core/programs): Smart Contracts auf Solana
- [Programmabgeleitete Adressen](https://solana.com/de/docs/core/pda): Deterministische Kontoadressen
- [Cross Program Invocation](https://solana.com/de/docs/core/cpi): Programmaufruf anderer Programme
- [Gebühren](https://solana.com/de/docs/core/fees): Transaktionskosten und Prioritätsgebühren

## Erste Schritte

- [Schnellstart](https://solana.com/de/docs/intro/quick-start): Erstellen Sie Ihr erstes Solana-Programm
- [Installation](https://solana.com/de/docs/intro/installation): Lokale Entwicklungsumgebung einrichten

## Token

- [Token-Grundlagen](https://solana.com/de/docs/tokens/basics): SPL Token Grundlagen
- [Token-Erweiterungen](https://solana.com/de/docs/tokens/extensions): Token-2022 Programmfunktionen

## RPC-Methoden

- [getAccountInfo](https://solana.com/de/docs/rpc/http/getaccountinfo): Kontodaten abrufen
- [getBalance](https://solana.com/de/docs/rpc/http/getbalance): SOL-Guthaben abrufen
- [sendTransaction](https://solana.com/de/docs/rpc/http/sendtransaction): Signierte Transaktionen senden
