# Solana (Español)

> Solana es la blockchain de alto rendimiento diseñada para la adopción masiva. Esta documentación proporciona guías completas y referencias para desarrolladores que construyen aplicaciones blockchain escalables en Solana.

## Conceptos Fundamentales

- [Cuentas](https://solana.com/docs/es/core/accounts): Cómo Solana almacena datos
- [Transacciones](https://solana.com/docs/es/core/transactions): Interacción con la red de Solana
- [Programas](https://solana.com/docs/es/core/programs): Contratos inteligentes en Solana
- [Direcciones Derivadas de Programa](https://solana.com/docs/es/core/pda): Direcciones determinísticas para cuentas
- [Invocación Entre Programas](https://solana.com/docs/es/core/cpi): Cómo los programas invocan otros programas
- [Tarifas](https://solana.com/docs/es/core/fees): Costos de transacción y tarifas de prioridad

## Comenzar

- [Inicio Rápido](https://solana.com/docs/es/intro/quick-start): Crea tu primer programa en Solana
- [Instalación](https://solana.com/docs/es/intro/installation): Configura tu entorno de desarrollo local

## Tokens

- [Fundamentos de Tokens](https://solana.com/docs/es/tokens/basics): Fundamentos de SPL Token
- [Extensiones de Token](https://solana.com/docs/es/tokens/extensions): Características del programa Token-2022

## Métodos RPC

- [getAccountInfo](https://solana.com/docs/es/rpc/http/getaccountinfo): Obtener datos de cuenta
- [getBalance](https://solana.com/docs/es/rpc/http/getbalance): Obtener saldo de SOL
- [sendTransaction](https://solana.com/docs/es/rpc/http/sendtransaction): Enviar transacciones firmadas
