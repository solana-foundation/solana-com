Nopea asennus paikallisen Solana-kehitysympäristösi määrittämiseksi yhdellä
komennolla. Asenna Rust, Solana CLI ja Anchor Framework Windowsissa (WSL),
Linuxissa ja Macissa. (Halutessasi voit
[asentaa jokaisen riippuvuuden erikseen](/docs/intro/installation/dependencies).)

## Edellytykset

<Accordions>
<Accordion title="Windows">

Kehittääksesi Solana-ohjelmia Windowsissa sinun **täytyy** käyttää
[Windows Subsystem for Linux](https://learn.microsoft.com/en-us/windows/wsl/install)
-järjestelmää (WSL). WSL:n asentamisen jälkeen kaikki muut riippuvuudet
asennetaan Linux-terminaalin kautta.

Asentaaksesi WSL:n suorita seuraava komento Windows PowerShellissä:

```terminal
$ wsl --install
```

Asennusprosessi pyytää sinua luomaan oletuskäyttäjätilin.

![WSL-asennus](/assets/docs/intro/installation/wsl-install.png)

Oletuksena WSL asentaa Ubuntun. Voit avata Linux-terminaalin hakemalla "Ubuntu"
hakupalkista.

![WSL Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-search.png)

Jos Ubuntu-terminaalisi näyttää alla olevan kuvan kaltaiselta, saatat kohdata
ongelman, jossa `ctrl + v` (liitä-pikanäppäin) ei toimi terminaalissa.

![Ubuntu-terminaali](/assets/docs/intro/installation/wsl-ubuntu-terminal-1.png)

Jos kohtaat tämän ongelman, avaa Windows Terminal hakemalla "Terminal"
hakupalkista.

![Windows Terminal](/assets/docs/intro/installation/wsl-windows-terminal.png)

Sulje seuraavaksi Windows Terminal ja avaa Linux-terminaali uudelleen hakemalla
Ubuntu uudestaan. Terminaalin pitäisi nyt näyttää alla olevan kuvan kaltaiselta,
jossa `ctrl + v` (liitä-pikanäppäin) toimii.

![Ubuntu-terminaali](/assets/docs/intro/installation/wsl-ubuntu-terminal-2.png)

Jos käytät VS Codea,
[WSL-laajennus](https://code.visualstudio.com/docs/remote/wsl-tutorial)
mahdollistaa WSL:n ja VS Coden käytön yhdessä.

![WSL-määritys VS Codessa](/assets/docs/intro/installation/wsl-vscode.png)

Näet seuraavan VS Coden tilapalkissa:

![WSL: Ubuntu](/assets/docs/intro/installation/wsl-vscode-ubuntu.png)

Kun WSL on asennettu, olet valmis joko:

- Suorittamaan [pika-asennuksen](#installation) Ubuntu (Linux) -päätteellä
- Asentamaan [Rustin](/docs/intro/installation/dependencies#install-rust),
  [Solana CLI:n](/docs/intro/installation/dependencies#install-solana-cli) ja
  [Anchor CLI:n](/docs/intro/installation/dependencies#install-anchor-cli)
  erikseen, kuten alla on kuvattu

</Accordion>

<Tab value=".deb" >
Jos käytät Debiania tai Debian-pohjaisia vaihtoehtoja kuten Ubuntua tai Mintiä, seuraa alla olevia ohjeita:

1. Päivitä pakettiluettelot varmistaaksesi, että kaikki paketit ovat ajan
   tasalla:

```terminal
$ sudo apt-get update
```

2. Asenna seuraavat riippuvuudet:

```terminal
$ sudo apt-get install -y \
    build-essential \
    pkg-config \
    libudev-dev llvm libclang-dev \
    protobuf-compiler libssl-dev
```

<Callout>
Jos kohtaat seuraavan virheen asentaessasi `protobuf-compiler`, varmista
että suoritat ensin _shell`sudo apt-get update`

```
Package protobuf-compiler is not available, but is referred to by another package.
This may mean that the package is missing, has been obsoleted, or
is only available from another source
```

</Callout>

    RHEL-johdannaisille kuten Fedoralle tai Red Hat Linuxille, seuraa seuraavia ohjeita.

Päivitä pakettiluettelot saadaksesi uudemmat versiot paketeista suorittamalla
seuraava komento.

```terminal
$ sudo dnf upgrade --refresh
```

Seuraavaksi asenna nämä riippuvuudet:

```terminal
$ sudo dnf group install c-development development-tools \
&& sudo dnf install pkg-config openssl-devel
```

## Asennus

1. Syötä seuraava komento päätteeseen asentaaksesi kaikki tarvittavat
   riippuvuudet:

```terminal
$ curl --proto '=https' --tlsv1.2 -sSfL https://solana-install.solana.workers.dev | bash
```

Onnistunut asennus näyttää seuraavanlaiselta:

```
Installed Versions:
Rust: rustc 1.86.0 (05f9846f8 2025-03-31)
Solana CLI: solana-cli 2.2.12 (src:0315eb6a; feat:1522022101, client:Agave)
Anchor CLI: anchor-cli 0.31.1
Node.js: v23.11.0
Yarn: 1.22.1
```

2. Varmista onnistunut asennus tarkistamalla kunkin asennetun riippuvuuden
   versio.

```terminal
$ rustc --version && solana --version && anchor --version && node --version && yarn --version
```

<Callout>
  Jos pika-asennuskomento epäonnistuu, katso ohjeet kunkin riippuvuuden
  asentamiseen erikseen [Asenna
  riippuvuudet](/docs/intro/installation/dependencies) -osiosta.
</Callout>

### Seuraavat vaiheet

Kun olet asentanut kaikki tarvittavat riippuvuudet, olet valmis tutustumaan
[Solana CLI -perusteisiin](/docs/intro/installation/solana-cli-basics) ja
[Anchor CLI -perusteisiin](/docs/intro/installation/anchor-cli-basics).
