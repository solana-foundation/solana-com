h1: voteSubscribe RPC-metodi
---

Tilaa ilmoitukset aina kun uusi ääni havaitaan gossip-verkossa. Nämä äänet ovat
ennakkokonsensuksen mukaisia, joten ei ole takeita siitä, että nämä äänet
päätyvät kirjanpitoon.

<Callout type="warn" title={"Epävakaa metodi"}>
  Tämä tilaus on epävakaa ja käytettävissä vain, jos validator käynnistettiin
  `--rpc-pubsub-enable-vote-subscription` lipulla. Tämän tilauksen muoto saattaa
  muuttua tulevaisuudessa.
</Callout>

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "voteSubscribe"
}
```

### !params

**Ei mitään**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

tilauksen tunnus (tarvitaan tilauksen peruuttamiseen)

</APIMethod>

#### Ilmoitusmuoto:

Ilmoitus on objekti, jossa on seuraavat kentät:

- `hash: <string>` - Äänen hajautusarvo
- `slots: <array>` - Äänen kattamat slotit u64-kokonaislukutaulukkona
- `timestamp: <i64|null>` - Äänen aikaleima
- `signature: <string>` - Tämän äänen sisältäneen tapahtuman allekirjoitus
- `votePubkey: <string>` - Äänestystilin julkinen avain base-58 -koodattuna
  merkkijonona

```json
{
  "jsonrpc": "2.0",
  "method": "voteNotification",
  "params": {
    "result": {
      "hash": "8Rshv2oMkPu5E4opXTRyuyBeZBqQ4S477VG26wUTFxUM",
      "slots": [1, 2],
      "timestamp": null
    },
    "subscription": 0
  }
}
```
