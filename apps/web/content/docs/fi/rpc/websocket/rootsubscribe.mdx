h1: rootSubscribe RPC-metodi
---

Tilaa ilmoitukset aina, kun validator asettaa uuden juuren.

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**Ei mitään**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

tilauksen tunnus (tarvitaan tilauksen peruuttamiseen)

</APIMethod>

#### Ilmoitusmuoto:

Tulos on viimeisin root slot -numero.

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
