h1: Go SDK Solanalle
---

On olemassa [Go SDK](https://github.com/gagliardetto/solana-go), joka tukee
useimpia toimintoja Solanan kanssa vuorovaikuttamiseen. Jos käytät Anchoria
Solana- ohjelmissasi, voit käyttää
[anchor-go](https://github.com/gagliardetto/anchor-go) -kirjastoa idl:n kanssa
luodaksesi Golang-yhteensopivan asiakassovelluksen.

## Asennus

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```

## Lisää dokumentaatiota

Käy
[Go SDK:n dokumentaatiossa](https://github.com/gagliardetto/solana-go?tab=readme-ov-file#installation)
