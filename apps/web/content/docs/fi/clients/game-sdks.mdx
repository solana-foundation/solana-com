Solanan ollessa suorituskykyinen lohkoketju, jossa on alhaiset maksut ja nopeat
tapahtumat, sen ympärille on muodostunut loistava pelikehittäjäyhteisö.

Täältä löydät tietoja ekosysteemissä saatavilla olevista Solana-pelikehityksen
SDK:ista. Kaikki SDK:t ovat avoimen lähdekoodin ja yhteisön rakentamia. Jos
työskentelet itse SDK:n parissa, voit vapaasti avata PR:n tälle sivulle ja
lisätä sen.

## Unity SDK

Unity-pelimoottori tunnetaan aloittelijaystävällisestä lähestymistavastaan ja
alustariippumattomasta tuestaan, mukaan lukien WebGL, iOS ja Android. Rakenna
kerran, vie kaikkialle.

- [Dokumentaatio](https://docs.magicblock.gg/introduction)
- [Lähdekoodi](https://github.com/magicblock-labs/Solana.Unity-SDK)
- [Esimerkkipelit](https://github.com/solana-developers/solana-game-examples)
- [Ristinolla](https://blog.magicblock.gg/bolt-tic-tac-toe/)

Solana Unity SDK:ta ylläpitää [Magicblock](https://www.magicblock.gg/) ja se
sisältää:

- NFT-tuen
- Candy machine -integraation
- Transaktiot
- RPC-toiminnot
- Phantom-syvälinkit
- WebGL-yhdistimen
- Luotettavan WebSocket-yhteyden tuen
- Mobiililompakko-adapterin
- Istuntoavaimet transaktioiden automaattiseen hyväksymiseen
- Anchor-asiakaskoodin generoinnin ja paljon muuta.

## Godot SDK

[Godot Engine](https://godotengine.org/) on avoimen lähdekoodin pelimoottori,
joka on saanut paljon tukea viime vuosina. Solana SDK:ta Godotille ylläpitävät
[ZenRepublic](https://twitter.com/ZenRepublicNDM) ja
[Virus-Axel](https://twitter.com/AxelBenjam). Se sisältää lompakkoadapteri-
integraation, transaktiot, RPC-toiminnot ja Anchor-asiakaskoodin generoinnin.
Jopa täysin toimivan Metaplex
[candy machine -integraation](https://zenwiki.gitbook.io/solana-godot-sdk-docs/guides/setup-candy-machine).

- [Solana Godot Engine SDK](https://github.com/Virus-Axel/godot-solana-sdk)
- [Dokumentaatio](https://zenwiki.gitbook.io/solana-godot-sdk-docs)
- [Demo](https://github.com/ZenRepublic/GodotSolanaSDKDemoPackage)
- [Opastus](https://www.youtube.com/watch?v=tszFPInYmXQ)
- [Ylläpitäjä](https://twitter.com/ZenRepublicNDM)

## Turbo.Computer - Rust-pelimoottori

Turbo on alusta alkaen Solanaan keskittyvä Rust-pelimootori, jolla on kevyt
arkkitehtuuri ja nopeat iterointiajat. Se on aloittelijaystävällinen ja sisältää
täyden Solana RPC -tuen. Voit jopa käyttää sen tekoälytyökaluja kokonaisten
pelien luomiseen.

- [Verkkosivusto](https://turbo.computer/)
- [Dokumentaatio](https://docs.turbo.computer/)
- [Twitter-tili](https://twitter.com/makegamesfast)
- [Ylläpitäjä](https://twitter.com/jozanza)

# Honeycomb Protocol

Honeycomb tarjoaa kokoelman lohkoketjuohjelmia ja tilan pakkaustyökaluja, jotka
hoitavat kaikki keskeiset pelin elinkaaren toiminnot ja yhteensopivuuden Solana
Virtual Machinen (SVM) sisällä. Se tukee monia NFT-toimintoja kuten staking,
tehtävät, loot boxit, pelaajaprofiilit, tilan pakkaus, transaktioiden
automaattinen hyväksyminen ja paljon muuta.

- [Twitter-tili](https://twitter.com/honeycomb_prtcl)
- [Dokumentaatio](https://docs.honeycombprotocol.com/)

# Unreal SDK:t

Unreal Engine tunnetaan upeasta grafiikastaan ja solmupohjaisesta
skriptauskehyksestään. Solanalle on olemassa useita SDK:ita, joita ylläpitävät
eri yritykset.

### Varmeta Unreal SDK

VAR META tarjoaa avoimen lähdekoodin Unreal SDK:n Solana-lohkoketjutoimintojen
integroimiseksi Unreal Engine -projekteihin. Sen avulla kehittäjät voivat olla
vuorovaikutuksessa Solanan älysopimuksien kanssa, hallita lompakoita ja
suorittaa transaktioita suoraan Unreal Enginen sisällä.

- [Unreal SDK Varmetan toimesta](https://github.com/VAR-META-Tech/UnrealSolanaSDK)
- [Ylläpitäjä](https://www.var-meta.com/)

### Bitfrost Unreal SDK

Bitfrost kehittää Unreal SDK:ta, jota on jo käytetty useissa peleissä
viimeaikaisissa Solana-pelijameissa. Se sisältää C# `solnet` tuen C++:ssa ja
Blueprinteissä, Metaplex NFT:iden luomisen, maksuprosessoriesimerkkejä
pelinsisäisessä lompakossa ja paljon muuta.

- [Solana Unreal SDK Bitfrostin toimesta](https://github.com/Bifrost-Technologies/Solana-Unreal-SDK)
- [Opastus](https://www.youtube.com/watch?v=S8fm8mFeUkk)
- [Ylläpitäjä](https://twitter.com/BifrostTitan)

### Thugz Unreal SDK

Thugz on teknologiastudio ja NFT-projekti, joka myös ylläpitää avoimen
lähdekoodin Unreal SDK:ta Solanalle. Se sisältää paljon NFT-keskeisiä
toimintoja, ja lisäosa on yhteensopiva myös iOS:n ja Androidin kanssa.

- [Thugz Blockchain -lisäosa](https://www.fab.com/listings/a7e220ba-93e5-4a3b-99e7-17b13da76f05)
- [Repositorio](https://github.com/ThugzLabs/Thugz-BC-Plugin-Packaged-for-UE5.4)
- [Dokumentaatio](https://thugzblockchainplugin-docs.vercel.app/)
- [Video-opas](https://www.youtube.com/watch?v=dS7sTZd_E9U&ab_channel=ThugzNFT)

### Star Atlas Foundation Kit

[Star Atlas](https://staratlas.com/) -tiimi julkaisi avoimen lähdekoodin version
osasta SDK-pinoa nimeltä _Foundation Kit_. Sitä ei aktiivisesti ylläpidetä,
mutta se voi toimia lähtökohtana projektillesi.

- [Star Atlas Unreal SDK](https://github.com/staratlasmeta/FoundationKit)
- [Opas](https://www.youtube.com/watch?v=S8fm8mFeUkk)

## Next.js/React + Anchor

Yksi helpoimmista tavoista kehittää Solanalle on käyttää Web3js JavaScript
-kehystä yhdessä Solana Anchor -kehyksen kanssa. Monimutkaisempiin peleihin
suosittelemme kuitenkin pelimoottoria kuten [Unity](#unity-sdk) tai
[Unreal](#unreal-sdks).

Nopein tapa pystyttää Next.js-pohjainen peli:

```shell
npx create-solana-game your-game-name
```

Tämä luo erinomaisen lähtösovelluksen, jossa on `wallet-adapter` -tuki,
Anchor-ohjelma, React-sovellus ja Unity-asiakas, jotka on jo valmiiksi
määritetty toimimaan yhdessä. Next.js:n käytön etu on, että voit käyttää samaa
koodia sekä back-endissä että front-endissä, mikä nopeuttaa kehitystä.

- [Web3.js](https://solana.com/docs/clients/javascript)
- [Video-opas](https://www.youtube.com/watch?v=fnhivg_pemI&t=1s&ab_channel=Solana)

## Phaser

Nopea, ilmainen ja hauska avoimen lähdekoodin kehys Canvas- ja WebGL-pohjaisille
selainpeleille. [Phaser](https://phaser.io/) on erinomainen tapa aloittaa
pelikehitys Solanalla. Phaser Solana Platformer Template on loistava lähtökohta
pelillesi.

- [Lähdekoodi](https://github.com/Bread-Heads-NFT/phaser-solana-platformer-template)
- [Ylläpitäjä](https://twitter.com/blockiosaurus)

## Flutter

Flutter on Googlen ylläpitämä avoimen lähdekoodin kehys, jolla voi rakentaa
kauniita, natiivisti käännettyjä, monialustaisia sovelluksia yhdestä
koodipohjasta. Solana Flutter SDK:ta ylläpitää
[Espresso Cash](https://www.espressocash.com/) -tiimi.

- [Lähdekoodi](https://github.com/espresso-cash/espresso-cash-public)

## Python

Python on helposti opittava ja laajasti käytetty ohjelmointikieli (jota
käytetään usein koneoppimissovelluksissa). Seahorse-kehys mahdollistaa
kehittäjille Solana-ohjelmien kirjoittamisen Pythonilla. Seahorse on rakennettu
Anchor-kehyksen päälle ja muuntaa Python-koodin Anchor-pohjaiseksi
rust-koodiksi. Seahorse on tällä hetkellä beta-vaiheessa.

- [Dokumentaatio](https://www.seahorse.dev/)
- [Github-repo](https://github.com/solana-developers/seahorse)
- [Anchor Playground -esimerkki](https://beta.solpg.io/tutorials/hello-seahorse)

## Natiivi C#

Alkuperäinen Web3js-porttaus C#:lle. Se sisältää paljon toiminnallisuuksia kuten
transaktiot, RPC-funktiot ja Anchor-asiakaskoodin generoinnin.

- [Lähdekoodi ja dokumentaatio](https://github.com/bmresearch/Solnet/blob/master/docs/articles/getting_started.md)
