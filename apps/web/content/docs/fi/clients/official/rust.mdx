## Ohjelma-cratet

Käytä `solana-program`-crateta on-chain-ohjelmien rakentamiseen.

- [Esimerkki: Rakenna Solana-ohjelma](/docs/programs/rust)

| Crate          | Kuvaus                     | Dokumentaatio                           | GitHub                                                                   |
| -------------- | -------------------------- | --------------------------------------- | ------------------------------------------------------------------------ |
| solana-program | Rakenna lohkoketjuohjelmia | [Näytä](https://docs.rs/solana-program) | [Lähdekoodi](https://github.com/anza-xyz/solana-sdk/tree/master/program) |

Pinocchio on riippuvuuksista vapaa kirjasto Solana-ohjelmien luomiseen Rustilla,
ja sitä voidaan käyttää `solana-program`-craten korvaajana on-chain-ohjelmien
kirjoittamiseen.

| Crate     | Kuvaus                            | Dokumentaatio                      | GitHub                                              |
| --------- | --------------------------------- | ---------------------------------- | --------------------------------------------------- |
| pinocchio | Riippumattomat lohkoketjuohjelmat | [Näytä](https://docs.rs/pinocchio) | [Lähdekoodi](https://github.com/anza-xyz/pinocchio) |

<Callout type="warn">
  Pinocchio on vielä kehitysvaiheessa ja siihen tulee todennäköisesti
  merkittäviä muutoksia lähitulevaisuudessa.
</Callout>

Pinocchio sisältää ohjelmakohtaisia crateja Solana-ohjelmien kanssa
vuorovaikutukseen:

| pinocchio-token                    | Vuorovaikutus Token Program -ohjelman kanssa            | [Näytä](https://docs.rs/pinocchio-token/)                    | [Lähdekoodi](https://github.com/anza-xyz/pinocchio/tree/main/programs/token)                    |
