**Chave da funcionalidade**:
[`tSynMCspg4xFiCj1v3TDb4c7crMR5tSBhLz4sF7rrNA`](https://explorer.solana.com/address/tSynMCspg4xFiCj1v3TDb4c7crMR5tSBhLz4sF7rrNA)

Este gate de funcionalidade adiciona uma nova instrução de voto como padrão. As
instruções de voto existentes continuarão funcionando, mas novos votos usarão o
novo formato de instrução daqui para frente.

## Impacto

**Provedores de dados** precisarão ser atualizados para rastrear a nova
instrução de voto, já que o novo formato de instrução pode não ser capturado nos
pipelines de voto existentes.

## Ativação

Esta funcionalidade está ativa na mainnet beta desde o
[epoch 749](https://explorer.solana.com/epoch/748) (por volta das 22h ET de
2025-02-27)
