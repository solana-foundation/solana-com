`@solana/kit` é o SDK TypeScript recomendado para construir na Solana.

| @solana/kit                    | SDK principal                          | [Source](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | Interagir com o System Program         | [Source](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Interagir com o Token Program          | [Source](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Interagir com o Token-2022 Program     | [Source](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Interagir com o Memo Program           | [Source](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Interagir com o Compute Budget Program | [Source](https://github.com/solana-program/compute-budget/tree/main/clients/js) |
