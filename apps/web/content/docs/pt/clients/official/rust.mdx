## Crates de programa

Use `solana-program` para construir programas on-chain.

- [Exemplo: Construir um programa Solana](/docs/programs/rust)

| Crate          | Descrição                    | Docs                                  | GitHub                                                              |
| -------------- | ---------------------------- | ------------------------------------- | ------------------------------------------------------------------- |
| solana-program | Construir programas on-chain | [Ver](https://docs.rs/solana-program) | [Fonte](https://github.com/anza-xyz/solana-sdk/tree/master/program) |

Pinocchio é uma biblioteca sem dependências para criar programas Solana em Rust
e pode ser usada como substituta para `solana-program` para escrever programas
on-chain.

| Crate     | Descrição                           | Docs                             | GitHub                                         |
| --------- | ----------------------------------- | -------------------------------- | ---------------------------------------------- |
| pinocchio | Programas on-chain sem dependências | [Ver](https://docs.rs/pinocchio) | [Fonte](https://github.com/anza-xyz/pinocchio) |

<Callout type="warn">
  Pinocchio ainda está em desenvolvimento e provavelmente terá mudanças
  significativas no futuro próximo.
</Callout>

Pinocchio inclui crates específicos para interagir com programas Solana:

| Crate                              | Descrição                                | Docs                                                       | GitHub                                                                                     |
| ---------------------------------- | ---------------------------------------- | ---------------------------------------------------------- | ------------------------------------------------------------------------------------------ |
| pinocchio-system                   | Interagir com o System Program           | [Ver](https://docs.rs/pinocchio-system/)                   | [Fonte](https://github.com/anza-xyz/pinocchio/tree/main/programs/system)                   |
| pinocchio-token                    | Interagir com o Token Program            | [Ver](https://docs.rs/pinocchio-token/)                    | [Fonte](https://github.com/anza-xyz/pinocchio/tree/main/programs/token)                    |
| pinocchio-token-2022               | Interagir com o Token-2022 Program       | [Ver](https://docs.rs/pinocchio-token-2022/)               | [Fonte](https://github.com/anza-xyz/pinocchio/tree/main/programs/token-2022)               |
| pinocchio-associated-token-account | Interagir com o Associated Token Program | [Ver](https://docs.rs/pinocchio-associated-token-account/) | [Fonte](https://github.com/anza-xyz/pinocchio/tree/main/programs/associated-token-account) |
| pinocchio-memo                     | Interagir com o Memo Program             | [Ver](https://docs.rs/pinocchio-memo/)                     | [Fonte](https://github.com/anza-xyz/pinocchio/tree/main/programs/memo)                     |
