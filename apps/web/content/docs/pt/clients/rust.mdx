Os crates Rust da Solana são
[publicados no crates.io](https://crates.io/search?q=solana-) e podem ser
encontrados [no docs.rs](https://docs.rs/releases/search?query=solana-) com o
prefixo `solana-`.

## Crates Rust

A seguir estão os crates Rust mais importantes e comumente usados para o
desenvolvimento na Solana:

- [`solana-program`] &mdash; Importado por programas executados na Solana,
  compilados para SBF. Este crate contém muitos tipos de dados fundamentais e é
  reexportado de [`solana-sdk`], que não pode ser importado de um programa
  Solana.

- [`solana-sdk`] &mdash; O SDK básico offchain, ele reexporta [`solana-program`]
  e adiciona mais APIs sobre isso. A maioria dos programas Solana que não são
  executados on-chain importarão este.

- [`solana-client`] &mdash; Para interagir com um nó Solana através da
  [API JSON RPC](/docs/rpc).

- [`solana-cli-config`] &mdash; Carregamento e salvamento do arquivo de
  configuração da CLI Solana.

- [`solana-clap-utils`] &mdash; Rotinas para configurar uma CLI, usando
  [`clap`], como usado pela CLI principal da Solana. Inclui funções para
  carregar todos os tipos de signatários suportados pela CLI.

[`solana-program`]: https://docs.rs/solana-program
[`solana-sdk`]: https://docs.rs/solana-sdk
[`solana-client`]: https://docs.rs/solana-client

## Instalação

Para enviar transações e interagir com a Solana:

```shell title="Terminal"
cargo add solana-sdk solana-client
```

Para construir programas Solana, instale:

```shell title="Terminal"
cargo add solana-program
```
