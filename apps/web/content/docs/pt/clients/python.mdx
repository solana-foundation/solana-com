Existe um [SDK Python](https://github.com/michaelhly/solana-py) que suporta a
maioria das funções para interagir com Solana. Se você estiver usando Anchor
para seus programas Solana, poderá usar
[anchor-py](https://github.com/kevinheavey/anchorpy) com seu idl para produzir
um cliente compatível com Python.

## Instalação

```bash title="Terminal"
pip install solders
pip install solana
```

## Mais documentação

Por favor, visite a
[documentação do SDK Python](https://github.com/michaelhly/solana-py?tab=readme-ov-file#general-usage)
para mais informações sobre como usar este SDK Python com Solana.

- [Documentação mais recente](https://michaelhly.github.io/solana-py/).

[`@kevinheavey/solders`](https://github.com/kevinheavey/solders)

- `solders` é um kit de ferramentas Python de alto desempenho para Solana,
  escrito em Rust. Ele fornece soluções robustas nas seguintes áreas:

- Funcionalidade principal do SDK: keypairs, pubkeys, assinatura e serialização
  de transações.
- RPC: construção de solicitações e análise de respostas (para redes e APIs
  assíncronas,
  [solana-py](https://michaelhly.github.io/solana-py/rpc/async_api/) é a opção
  recomendada).
- Testes de integração: o módulo `solders.litesvm` é uma alternativa ao
  `solana-test-validator` que é muito mais conveniente e **muito** mais rápido.
  É baseado em
  [solana-program-test](https://crates.io/crates/solana-program-test).

## SDK Python para cliente Solana

Se você deseja interagir com contratos Solana usando Python, pode utilizar o SDK
de cliente para renderizar contratos Rust da Solana em clientes Python, desde
que você tenha o arquivo IDL do contrato.

[`Codama-py`](https://www.npmjs.com/package/codama-py)

- Padroniza qualquer programa de contrato on-chain da Solana para
  [`Codama IDL`](https://github.com/codama-idl/codama) e gera um cliente Python
  interativo.
- Suporta tanto programas Anchor quanto programas personalizados.

[`Anchor-py`](https://github.com/kevinheavey/anchorpy)

- Anchor-py é um cliente Python para programas Anchor. Ele pode gerar clientes a
  partir de arquivos IDL do Anchor e interagir com programas Anchor.
