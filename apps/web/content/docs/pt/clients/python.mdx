h1: SDK Python para Solana
---

Existe um [SDK Python](https://github.com/michaelhly/solana-py) que suporta a
maioria das funções para interagir com Solana. Se você estiver usando Anchor
para seus programas Solana, poderá usar
[anchor-py](https://github.com/kevinheavey/anchorpy) com seu idl para produzir
um cliente compatível com Python.

## Instalação

```bash title="Terminal"
pip install solders
pip install solana
```
