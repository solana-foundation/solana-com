h1: SDK Go para Solana
---

Existe um [SDK Go](https://github.com/gagliardetto/solana-go) que suporta a
maioria das funções para interagir com Solana. Se você estiver usando Anchor
para seus programas Solana, poderá usar
[anchor-go](https://github.com/gagliardetto/anchor-go) com seu idl para produzir
um cliente compatível com Golang.

## Instalação

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```

## Mais documentação

Por favor, visite a
