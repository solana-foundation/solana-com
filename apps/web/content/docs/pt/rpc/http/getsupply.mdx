h1: Método RPC getSupply
---

Retorna informações sobre o fornecimento atual.

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getSupply",
  "params": [
    // !hover(1:3) 0
    {
      // !hover commitment
      "commitment": "finalized"
    }
  ]
}
```

```ts !!request title="Kit"
import { createSolanaRpc } from "@solana/kit";

const rpc_url = "https://api.devnet.solana.com";
const rpc = createSolanaRpc(rpc_url);

let supply = await rpc.getSupply().send();

console.log(supply);
```

```ts !!request title="web3.js"
import { Connection, clusterApiUrl } from "@solana/web3.js";

const connection = new Connection(clusterApiUrl("devnet"), "confirmed");

let supply = await connection.getSupply();

console.log(supply);
```

### !params

#### !! 0

!type object

Objeto de configuração contendo os seguintes campos:

##### !! commitment

!type string

O commitment descreve quão finalizado está um bloco naquele momento. Veja
[Configurando State Commitment](/docs/rpc#configuring-state-commitment).

##### !! excludeNonCirculatingAccountsList

!type bool

Excluir lista de contas não circulantes da resposta

!type object

O resultado será um objeto JSON contendo:

##### !! circulating

!type u64

Fornecimento circulante em lamports

##### !! nonCirculating

!type u64

Fornecimento não circulante em lamports

##### !! nonCirculatingAccounts

!type array

Uma matriz de endereços de contas não circulantes, como strings. Se
`excludeNonCirculatingAccountsList` estiver ativado, a matriz retornada será
vazia.

##### !! total

!type u64

Fornecimento total em lamports

</APIMethod>
