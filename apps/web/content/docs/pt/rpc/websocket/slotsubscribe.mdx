h1: Método RPC slotSubscribe
---

Inscreva-se para receber notificações sempre que um slot for processado pelo
validator

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "slotSubscribe"
}
```

### !params

**Nenhum**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

Id de inscrição (necessário para cancelar a inscrição)

</APIMethod>

#### Formato de notificação:

A notificação será um objeto com os seguintes campos:

- `parent: <u64>` - O slot pai
- `root: <u64>` - O slot raiz atual
- `slot: <u64>` - O valor do slot recém-definido

Exemplo:

```json
{
  "jsonrpc": "2.0",
  "method": "slotNotification",
  "params": {
    "result": {
      "parent": 75,
      "root": 44,
      "slot": 76
    },
    "subscription": 0
  }
}
```
