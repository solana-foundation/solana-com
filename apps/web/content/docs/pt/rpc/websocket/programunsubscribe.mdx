h1: Método RPC programUnsubscribe
---

Cancelar inscrição das notificações de alteração de conta pertencente a programa

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "programUnsubscribe",
  "params": [
    // !hover 0
    0
  ]
}
```

### !params

#### !! 0

!type number  
!required

id da inscrição da conta a ser cancelada

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": true,
  "id": 1
}
```

!type boolean

mensagem de sucesso do cancelamento da inscrição

</APIMethod>
