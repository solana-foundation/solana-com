h1: Método RPC rootSubscribe
---

Inscreva-se para receber notificações sempre que uma nova raiz for definida pelo
validator.

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**Nenhum**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

id de inscrição (necessário para cancelar a inscrição)

</APIMethod>

#### Formato de notificação:

O resultado é o número do slot raiz mais recente.

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
