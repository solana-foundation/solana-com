h1: Método RPC getSnapshotSlot
---

Retorna o slot mais alto para o qual o nó possui um snapshot

<Callout type="warn" title={"Método Obsoleto"}>
  Este método deverá ser removido na `solana-core` v2.0. Por favor, use
  [getHighestSnapshotSlot](/docs/rpc/http/gethighestsnapshotslot) em vez disso.
</Callout>
