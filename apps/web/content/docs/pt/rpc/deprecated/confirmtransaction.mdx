h1: Método RPC confirmTransaction
---

Busque o status atual de uma assinatura de transação (processada, confirmada,
finalizada).

<Callout type="warn" title={"Método Obsoleto"}>
  Este método deve ser removido na `solana-core` v2.0. Por favor, use
  [getSignatureStatuses](/docs/rpc/http/getsignaturestatuses) em vez disso.
</Callout>
