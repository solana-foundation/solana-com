Surfpool é uma ferramenta de desenvolvimento local que simplifica o
desenvolvimento em Solana automaticamente carregando programas e contas de um
cluster (mainnet, devnet). Use o Surfpool CLI para iniciar um validator local
para testar seus programas e transações.

Visite a [documentação do Surfpool](https://docs.surfpool.run/) para mais
informações.

## Iniciar um validator local

Para iniciar um validator local, execute o seguinte comando:

```terminal
$ surfpool start
```

Por padrão, o Surfpool carrega automaticamente quaisquer programas e contas que
suas transações dependam da mainnet para o validator local.

## Surfpool Studio

O Surfpool Studio fornece uma interface web para interagir com seu validator
local. Use-o para:

- Ver detalhes das transações
- Inspecionar dados de contas
- Fazer airdrop de SOL e outros tokens usando a faucet da UI

Para acessar o Surfpool Studio:

1. Inicie o seu validator local com `surfpool start`
2. Abra o seu navegador e aceda a
   [http://127.0.0.1:18488](http://127.0.0.1:18488)
