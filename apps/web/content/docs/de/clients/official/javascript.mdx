## Solana Kit

`@solana/kit` ist das empfohlene TypeScript SDK für die Entwicklung auf Solana.

| Paket                          | Beschreibung                           | GitHub                                                                          |
| ------------------------------ | -------------------------------------- | ------------------------------------------------------------------------------- |
| @solana/kit                    | Core SDK                               | [Source](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | Interaktion mit System Program         | [Source](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Interaktion mit Token Program          | [Source](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Interaktion mit Token-2022 Program     | [Source](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Interaktion mit Memo Program           | [Source](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Interaktion mit Compute Budget Program | [Source](https://github.com/solana-program/compute-budget/tree/main/clients/js) |

- [Offizielle Kit-Dokumentation](https://www.solanakit.com/)
- [Beispiel: Wallet mit Next.js verbinden](/docs/frontend/kit)
