h1: Go SDK für Solana
---

Es gibt ein [Go SDK](https://github.com/gagliardetto/solana-go), das die meisten
Funktionen zur Interaktion mit Solana unterstützt. Wenn Sie Anchor für Ihre
Solana- Programme verwenden, können Sie
[anchor-go](https://github.com/gagliardetto/anchor-go) mit Ihrer IDL nutzen, um
einen Golang-kompatiblen Client zu erstellen.

## Installation

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```
