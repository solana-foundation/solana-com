h1: rootSubscribe RPC-Methode
---

Abonnieren Sie Benachrichtigungen, wann immer ein neuer Root durch den Validator
festgelegt wird.

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**Keine**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

Abonnement-ID (wird zum Abbestellen ben√∂tigt)

</APIMethod>

#### Benachrichtigungsformat:

Das Ergebnis ist die neueste Root-Slot-Nummer.

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
