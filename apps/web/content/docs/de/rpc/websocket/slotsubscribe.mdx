h1: slotSubscribe RPC-Methode
---

Abonnieren Sie Benachrichtigungen, wann immer ein Slot vom Validator verarbeitet
wird

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "slotSubscribe"
}
```

### !params

**Keine**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

Abonnement-ID (wird zum Abbestellen benötigt)

</APIMethod>

#### Benachrichtigungsformat:

Die Benachrichtigung ist ein Objekt mit den folgenden Feldern:

- `parent: <u64>` - Der übergeordnete Slot
- `root: <u64>` - Der aktuelle Root-Slot
- `slot: <u64>` - Der neu gesetzte Slot-Wert

Beispiel:

```json
{
  "jsonrpc": "2.0",
  "method": "slotNotification",
  "params": {
    "result": {
      "parent": 75,
      "root": 44,
      "slot": 76
    },
    "subscription": 0
  }
}
```
