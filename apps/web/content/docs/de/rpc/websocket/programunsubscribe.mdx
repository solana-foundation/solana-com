Abmeldung von Benachrichtigungen über Änderungen an programmgesteuerten Konten

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "programUnsubscribe",
  "params": [
    // !hover 0
    0
  ]
}
```

### !params

#### !! 0

!type number  
!required

ID des Konten-Abonnements, das gekündigt werden soll

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": true,
  "id": 1
}
```

!type boolean

Erfolgsmeldung zur Abmeldung

</APIMethod>
