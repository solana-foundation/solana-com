Surfpool ist ein lokales Entwicklungstool, das die Solana-Entwicklung
vereinfacht, indem es Programme und Konten automatisch von einem Cluster
(Mainnet, Devnet) lädt. Verwenden Sie die Surfpool CLI, um einen lokalen
Validator für das Testen Ihrer Programme und Transaktionen zu starten.

Besuchen Sie die [Surfpool-Dokumentation](https://docs.surfpool.run/) für
weitere Informationen.

## Starten eines lokalen Validators

Um einen lokalen Validator zu starten, führen Sie den folgenden Befehl aus:

```terminal
$ surfpool start
```

Standardmäßig lädt Surfpool automatisch alle Programme und Konten, von denen
Ihre Transaktionen abhängen, vom Mainnet auf den lokalen Validator.

## Surfpool Studio

Surfpool Studio bietet eine webbasierte Oberfläche für die Interaktion mit Ihrem
lokalen Validator. Verwenden Sie es, um:

- Transaktionsdetails anzuzeigen
- Kontodaten zu inspizieren
- SOL und andere Token über die UI-Faucet per Airdrop zu erhalten

Um auf Surfpool Studio zuzugreifen:

1. Starten Sie Ihren lokalen Validator mit `surfpool start`
2. Öffnen Sie Ihren Browser und navigieren Sie zu
   [http://127.0.0.1:18488](http://127.0.0.1:18488)
