Solanaクイックスタートガイドへようこそ！この実践的なガイドでは、あなたの経験レベルに関わらず、核となる概念を学ぶことができる**Solana
Playground**（Solpg）を紹介します。このチュートリアルでは、以下について学びます：

- **Solanaアカウント**：Solanaネットワークがデータを保存する方法
- **トランザクションの送信**：トランザクションを送信してSolanaネットワークとやり取りする方法
- **プログラムの構築とデプロイ**：最初のSolanaプログラムを作成し、ネットワークにデプロイする方法
- **Program Derived Addresses
  (PDAs)**：アカウントの決定論的なアドレスを作成するためにPDAを使用する方法
- **Cross-Program Invocations
  (CPIs)**：プログラム内から他のプログラムを呼び出す方法を学び、Solana上の異なるプログラム間の複雑な相互作用と組み合わせ可能性を実現する

## Solana Playground

Solana
Playgroundは、ブラウザベースの開発環境で、ローカルに何もインストールすることなくSolanaプログラムの開発、デプロイ、テストができます。手順に従って、コードをコピー＆ペーストし、結果をすぐに確認できます。基本的なプログラミングの知識があると役立ちますが、必須ではありません。さあ、Solanaでの構築を始めましょう！

新規ユーザーとして、Solana
Playgroundを操作するための最初のステップは、Playground
[ウォレット](/docs/references/terminology#wallet)を作成し、
[SOL](/docs/references/terminology#sol)を追加することです。このウォレットを使用すると、ブラウザから直接Solanaネットワークとやり取りできます。

## Playgroundウォレットの作成

<Steps>
<Step>

### Solana Playgroundに移動する

ウェブブラウザを使用して、[Solana Playground](https://beta.solpg.io/)にアクセスしてください。

</Step>
<Step>

### Playgroundに接続する

画面左下隅にある🔴 **Not
connected**（接続されていません）テキストをクリックします。

![接続されていません](/assets/docs/intro/quickstart/pg-not-connected.png)

</Step>
<Step>

### keypairを保存する

**Save
keypair**をクリックして、キーペアをローカルに保存します。準備ができたら、**Continue**をクリックしてください。

![プレイグラウンドウォレットの作成](/assets/docs/intro/quickstart/pg-create-wallet.png)

画面下部に、プレイグラウンドウォレットに接続されたことが表示されます。また、[接続されたクラスター](/docs/references/terminology#connected-cluster)（`devnet`）、[ウォレットアドレス](/docs/references/terminology#wallet-address)、および[SOL](/docs/references/terminology#sol)の残高も表示されます。

![接続済み](/assets/docs/intro/quickstart/pg-connected.png)

<Callout>
  プレイグラウンドウォレットはブラウザのローカルストレージに保存されます。ブラウザのキャッシュをクリアすると、保存されたウォレットは削除されます。
</Callout>

<Callout type="warn">
  プレイグラウンドウォレットはテストと開発のみに使用してください。実際の資産（[`mainnet-beta`](/docs/references/terminology#connected-cluster)から）をこのアドレスに送らないでください。
</Callout>

おめでとうございます！最初のSolanaウォレットを作成しました。これでウォレットにSOLを追加する準備ができました。

</Step>
<Step>

### ウォレットにSOLを追加する

開発を始める前に、devnet
SOLをウォレットに追加する必要があります。開発者として、SOLは主に2つのユースケースで使用されます：

- データを保存したりプログラムをデプロイしたりするための新しいアカウントの作成
- Solanaネットワークと対話する際の取引手数料の支払い

プレイグラウンドターミナルまたはDevnet Faucetを使用して、ウォレットにdevnet
SOLを追加できます。各オプションについて以下に説明します：

#### オプション1：プレイグラウンドターミナルを使用する

プレイグラウンドターミナルで次のコマンドを実行します：

```terminal
$ solana airdrop 5
```

#### オプション2：Devnet Faucetを使用する

レート制限やその他のエラーによりオプション1が失敗した場合は、Solanaの[Web Faucet](https://faucet.solana.com/)を使用してください。

- ウォレットアドレスを入力します
- 金額を選択します
- **Confirm Airdrop**をクリックします

![Faucetエアドロップ](/assets/docs/intro/quickstart/faucet-airdrop.gif)

</Step>
</Steps>
