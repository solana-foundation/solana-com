## プログラムクレート

`solana-program` を使用してオンチェーンプログラムを構築します。

- [例：Solanaプログラムを構築する](/docs/programs/rust)

| クレート       | 説明                         | ドキュメント                           | GitHub                                                               |
| -------------- | ---------------------------- | -------------------------------------- | -------------------------------------------------------------------- |
| solana-program | オンチェーンプログラムの構築 | [表示](https://docs.rs/solana-program) | [ソース](https://github.com/anza-xyz/solana-sdk/tree/master/program) |

Pinocchioは、Rustで Solana プログラムを作成するための依存関係ゼロのライブラリであり、`solana-program`
の代替としてオンチェーンプログラムの記述に使用できます。

| クレート  | 説明                                 | ドキュメント                      | GitHub                                          |
| --------- | ------------------------------------ | --------------------------------- | ----------------------------------------------- |
| pinocchio | 依存関係のないオンチェーンプログラム | [表示](https://docs.rs/pinocchio) | [ソース](https://github.com/anza-xyz/pinocchio) |

<Callout type="warn">
  Pinocchioはまだ開発中であり、近い将来に互換性を破壊する変更が行われる可能性があります。
</Callout>

Pinocchioには、Solanaプログラムと対話するためのプログラム固有のクレートが含まれています：

| クレート                           | 説明                             | ドキュメント                                                | GitHub                                                                                      |
| ---------------------------------- | -------------------------------- | ----------------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| pinocchio-system                   | System programとの対話           | [表示](https://docs.rs/pinocchio-system/)                   | [ソース](https://github.com/anza-xyz/pinocchio/tree/main/programs/system)                   |
| pinocchio-token                    | Token programとの対話            | [表示](https://docs.rs/pinocchio-token/)                    | [ソース](https://github.com/anza-xyz/pinocchio/tree/main/programs/token)                    |
| pinocchio-token-2022               | Token-2022 programとの対話       | [表示](https://docs.rs/pinocchio-token-2022/)               | [ソース](https://github.com/anza-xyz/pinocchio/tree/main/programs/token-2022)               |
| pinocchio-associated-token-account | Associated Token programとの対話 | [表示](https://docs.rs/pinocchio-associated-token-account/) | [ソース](https://github.com/anza-xyz/pinocchio/tree/main/programs/associated-token-account) |
| pinocchio-memo                     | Memo programとの対話             | [表示](https://docs.rs/pinocchio-memo/)                     | [ソース](https://github.com/anza-xyz/pinocchio/tree/main/programs/memo)                     |
