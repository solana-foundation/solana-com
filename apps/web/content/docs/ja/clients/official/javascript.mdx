## Solana Kit

`@solana/kit` はSolana上での開発に推奨されるTypeScript SDKです。

| パッケージ                     | 説明                           | GitHub                                                                          |
| ------------------------------ | ------------------------------ | ------------------------------------------------------------------------------- |
| @solana/kit                    | コアSDK                        | [ソース](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | System Programとの連携         | [ソース](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Token Programとの連携          | [ソース](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Token-2022 Programとの連携     | [ソース](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Memo Programとの連携           | [ソース](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Compute Budget Programとの連携 | [ソース](https://github.com/solana-program/compute-budget/tree/main/clients/js) |

- [公式Kit ドキュメント](https://www.solanakit.com/)
- [例: Next.jsでウォレットを接続する](/docs/frontend/kit)
