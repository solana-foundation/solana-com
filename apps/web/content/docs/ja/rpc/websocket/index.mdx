RPC PubSub websocketに`ws://<ADDRESS>/`で接続した後：

- 以下のメソッドを使用してwebsocketにサブスクリプションリクエストを送信します
- 複数のサブスクリプションを同時にアクティブにすることができます
- 多くのサブスクリプションはオプションの
  [`commitment`パラメータ](/docs/rpc/#configuring-state-commitment)を受け付け、通知をトリガーするための変更がどの程度確定されるべきかを定義します。サブスクリプションでは、commitmentが指定されていない場合、デフォルト値は`finalized`です。

## RPC PubSub WebSocketエンドポイント

デフォルトポート：`8900`

- ws://localhost:8900
- http://192.168.1.88:8900
