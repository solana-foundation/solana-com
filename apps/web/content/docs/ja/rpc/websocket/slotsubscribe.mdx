h1: slotSubscribe RPCメソッド
---

validatorによって処理されるたびにslotの通知を受け取るためにサブスクライブします

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "slotSubscribe"
}
```

### !params

**なし**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

サブスクリプションID（サブスクリプション解除に必要）

</APIMethod>

#### 通知フォーマット：

通知は以下のフィールドを持つオブジェクトになります：

- `parent: <u64>` - 親slot
- `root: <u64>` - 現在のルートslot
- `slot: <u64>` - 新しく設定されたslot値

例：

```json
{
  "jsonrpc": "2.0",
  "method": "slotNotification",
  "params": {
    "result": {
      "parent": 75,
      "root": 44,
      "slot": 76
    },
    "subscription": 0
  }
}
```
