h1: rootSubscribe RPCメソッド
---

validatorによって新しいルートが設定されるたびに通知を受け取るためにサブスクライブします。

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**なし**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

サブスクリプションID（サブスクリプション解除に必要）

</APIMethod>

#### 通知フォーマット：

結果は最新のルートslot番号です。

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
