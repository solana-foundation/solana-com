Μια γρήγορη εγκατάσταση για να ρυθμίσετε το τοπικό περιβάλλον ανάπτυξης Solana
με μία εντολή. Εγκαταστήστε Rust, Solana CLI και το πλαίσιο Anchor σε Windows
(WSL), Linux και Mac. (Αν προτιμάτε, μπορείτε να
[εγκαταστήσετε κάθε εξάρτηση ξεχωριστά](/docs/intro/installation/dependencies).)

## Προαπαιτούμενα

<Accordions>
<Accordion title="Windows">

Για να αναπτύξετε προγράμματα Solana στα Windows **πρέπει** να χρησιμοποιήσετε
το
[Windows Subsystem for Linux](https://learn.microsoft.com/en-us/windows/wsl/install)
(WSL). Μετά την εγκατάσταση του WSL, όλες οι άλλες εξαρτήσεις θα εγκατασταθούν
μέσω του τερματικού Linux.

Για να εγκαταστήσετε το WSL, εκτελέστε την ακόλουθη εντολή στο Windows
PowerShell:

```terminal
$ wsl --install
```

Η διαδικασία εγκατάστασης θα σας ζητήσει να δημιουργήσετε έναν προεπιλεγμένο
λογαριασμό χρήστη.

![Εγκατάσταση WSL](/assets/docs/intro/installation/wsl-install.png)

Από προεπιλογή, το WSL εγκαθιστά το Ubuntu. Μπορείτε να ανοίξετε ένα τερματικό
Linux αναζητώντας "Ubuntu" στη γραμμή αναζήτησης.

![WSL Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-search.png)

Αν το τερματικό Ubuntu σας μοιάζει με την παρακάτω εικόνα, μπορεί να
αντιμετωπίσετε ένα πρόβλημα όπου το `ctrl + v` (συντόμευση πληκτρολογίου για
επικόλληση) δεν λειτουργεί στο τερματικό.

![Τερματικό Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-terminal-1.png)

Αν αντιμετωπίσετε αυτό το πρόβλημα, ανοίξτε το Windows Terminal αναζητώντας
"Terminal" στη γραμμή αναζήτησης.

![Windows Terminal](/assets/docs/intro/installation/wsl-windows-terminal.png)

Στη συνέχεια, κλείστε το Windows Terminal και ανοίξτε ξανά ένα τερματικό Linux
αναζητώντας ξανά το Ubuntu. Το τερματικό θα πρέπει τώρα να μοιάζει με την
παρακάτω εικόνα, όπου το `ctrl + v` (συντόμευση πληκτρολογίου για επικόλληση)
λειτουργεί.

![Τερματικό Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-terminal-2.png)

Αν χρησιμοποιείτε VS Code, η
[επέκταση WSL](https://code.visualstudio.com/docs/remote/wsl-tutorial) σας
επιτρέπει να χρησιμοποιείτε το WSL και το VS Code μαζί.

![Ρύθμιση WSL στο VS Code](/assets/docs/intro/installation/wsl-vscode.png)

Θα δείτε τα ακόλουθα στη γραμμή κατάστασης του VS Code:

![WSL: Ubuntu](/assets/docs/intro/installation/wsl-vscode-ubuntu.png)

Μόλις ρυθμίσετε το WSL, είστε έτοιμοι είτε να:

- Ολοκληρώσετε την [γρήγορη εγκατάσταση](#installation) χρησιμοποιώντας το
  τερματικό Ubuntu (Linux)
- Εγκαταστήσετε το [Rust](/docs/intro/installation/dependencies#install-rust),
  το [Solana CLI](/docs/intro/installation/dependencies#install-solana-cli), και
  το [Anchor CLI](/docs/intro/installation/dependencies#install-anchor-cli)
  ξεχωριστά, όπως περιγράφεται παρακάτω

</Accordion>

Αν χρησιμοποιείτε Debian ή παράγωγα του Debian όπως Ubuntu ή Mint, ακολουθήστε τα παρακάτω βήματα:

1. Ενημερώστε τις λίστες πακέτων για να διασφαλίσετε ότι όλα τα πακέτα είναι
   ενημερωμένα:

```terminal
$ sudo apt-get update
```

2. Εγκαταστήστε τις ακόλουθες εξαρτήσεις:

```terminal
$ sudo apt-get install -y \
    build-essential \
    pkg-config \
    libudev-dev llvm libclang-dev \
    protobuf-compiler libssl-dev
```

## Εγκατάσταση

1. Εισάγετε την ακόλουθη εντολή στο τερματικό σας για να εγκαταστήσετε όλες τις
   απαραίτητες εξαρτήσεις:

```terminal
$ curl --proto '=https' --tlsv1.2 -sSfL https://solana-install.solana.workers.dev | bash
```

Μια επιτυχημένη εγκατάσταση θα επιστρέψει έξοδο όπως η ακόλουθη:

```
Installed Versions:
Rust: rustc 1.86.0 (05f9846f8 2025-03-31)
Solana CLI: solana-cli 2.2.12 (src:0315eb6a; feat:1522022101, client:Agave)
Anchor CLI: anchor-cli 0.31.1
Node.js: v23.11.0
Yarn: 1.22.1
```

2. Επαληθεύστε την επιτυχή εγκατάσταση ελέγχοντας την έκδοση κάθε εγκατεστημένης
   εξάρτησης.

```terminal
$ rustc --version && solana --version && anchor --version && node --version && yarn --version
```

<Callout>
  Εάν η εντολή γρήγορης εγκατάστασης αποτύχει, παρακαλούμε ανατρέξτε στην
  ενότητα [Εγκατάσταση Εξαρτήσεων](/docs/intro/installation/dependencies)
  παρακάτω για οδηγίες σχετικά με την εγκατάσταση κάθε εξάρτησης ξεχωριστά.
</Callout>
