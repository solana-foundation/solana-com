Το Surfpool είναι ένα εργαλείο τοπικής ανάπτυξης που απλοποιεί την ανάπτυξη στο
Solana φορτώνοντας αυτόματα προγράμματα και λογαριασμούς από ένα cluster
(mainnet, devnet). Χρησιμοποιήστε το Surfpool CLI για να ξεκινήσετε έναν τοπικό
validator για τη δοκιμή των προγραμμάτων και των συναλλαγών σας.

Επισκεφθείτε την [τεκμηρίωση του Surfpool](https://docs.surfpool.run/) για
περισσότερες πληροφορίες.

## Εκκίνηση ενός τοπικού validator

Για να ξεκινήσετε έναν τοπικό validator, εκτελέστε την ακόλουθη εντολή:

```terminal
$ surfpool start
```

Από προεπιλογή, το Surfpool φορτώνει αυτόματα όλα τα προγράμματα και τους
λογαριασμούς από τους οποίους εξαρτώνται οι συναλλαγές σας από το mainnet στον
τοπικό validator.

## Surfpool Studio

Το Surfpool Studio παρέχει μια διεπαφή βασισμένη στο web για αλληλεπίδραση με
τον τοπικό σας validator. Χρησιμοποιήστε το για να:

- Προβάλετε λεπτομέρειες συναλλαγών
- Επιθεωρήσετε δεδομένα λογαριασμών
- Κάνετε airdrop SOL και άλλα tokens χρησιμοποιώντας το UI faucet

Για να αποκτήσετε πρόσβαση στο Surfpool Studio:

1. Ξεκινήστε τον τοπικό σας validator με `surfpool start`
2. Ανοίξτε το πρόγραμμα περιήγησής σας και μεταβείτε στο
   [http://127.0.0.1:18488](http://127.0.0.1:18488)
