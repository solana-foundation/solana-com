Μετά τη σύνδεση στο RPC PubSub websocket στο `ws://<ADDRESS>/`:

- Υποβάλετε αιτήματα συνδρομής στο websocket χρησιμοποιώντας τις παρακάτω
  μεθόδους
- Πολλαπλές συνδρομές μπορούν να είναι ενεργές ταυτόχρονα
- Πολλές συνδρομές δέχονται την προαιρετική παράμετρο
  [`commitment`](/docs/rpc/#configuring-state-commitment), που καθορίζει πόσο
  οριστικοποιημένη πρέπει να είναι μια αλλαγή για να ενεργοποιήσει μια
  ειδοποίηση. Για τις συνδρομές, αν δεν καθοριστεί το commitment, η
  προεπιλεγμένη τιμή είναι `finalized`.

## Τελικό σημείο RPC PubSub WebSocket

Προεπιλεγμένη θύρα: `8900`

- ws://localhost:8900
- http://192.168.1.88:8900
