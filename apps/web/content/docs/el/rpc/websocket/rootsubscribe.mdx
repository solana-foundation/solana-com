h1: Μέθοδος RPC rootSubscribe
---

Εγγραφείτε για να λαμβάνετε ειδοποιήσεις κάθε φορά που ορίζεται ένα νέο root από
τον validator.

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**Κανένα**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

αναγνωριστικό συνδρομής (απαιτείται για την κατάργηση της εγγραφής)

</APIMethod>

#### Μορφή ειδοποίησης:

Το αποτέλεσμα είναι ο πιο πρόσφατος αριθμός slot του root.

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
