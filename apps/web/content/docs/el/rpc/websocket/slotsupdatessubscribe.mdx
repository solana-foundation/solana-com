Εγγραφείτε για να λαμβάνετε ειδοποιήσεις από τον validator σχετικά με διάφορες
ενημερώσεις σε κάθε slot

<Callout type="warn">
  Αυτή η συνδρομή είναι ασταθής. Η μορφή αυτής της συνδρομής μπορεί να αλλάξει
  στο μέλλον και ενδέχεται να μην υποστηρίζεται πάντα.
</Callout>

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "slotsUpdatesSubscribe"
}
```

### !params

**Κανένα**

### !!result

</APIMethod>

### Μορφή ειδοποίησης

Η ειδοποίηση θα είναι ένα αντικείμενο με τα ακόλουθα πεδία:

- `err: <string|undefined>` - Το μήνυμα σφάλματος. Παρουσιάζεται μόνο αν η
  ενημέρωση είναι τύπου "dead".
- `parent: <u64|undefined>` - Το γονικό slot. Παρουσιάζεται μόνο αν η ενημέρωση
  είναι τύπου "createdBank".
- `slot: <u64>` - Το πρόσφατα ενημερωμένο slot
- `stats: <object|undefined>` - Το μήνυμα σφάλματος. Παρουσιάζεται μόνο αν η
  ενημέρωση είναι τύπου "frozen". Ένα αντικείμενο με τα ακόλουθα πεδία:
  - `maxTransactionsPerEntry: <u64>`,
  - `numFailedTransactions: <u64>`,
  - `numSuccessfulTransactions: <u64>`,
  - `numTransactionEntries: <u64>`,
- `timestamp: <i64>` - Η χρονική σήμανση Unix της ενημέρωσης σε χιλιοστά του
  δευτερολέπτου
- `type: <string>` - Ο τύπος ενημέρωσης, ένα από τα εξής:
  - "firstShredReceived"
  - "completed"
  - "createdBank"
  - "frozen"
  - "dead"
  - "optimisticConfirmation"
  - "root"

```shell
{
  "jsonrpc": "2.0",
  "method": "slotsUpdatesNotification",
  "params": {
    "result": {
      "parent": 75,
      "slot": 76,
      "timestamp": 1625081266243,
      "type": "optimisticConfirmation"
    },
    "subscription": 0
  }
}
```
