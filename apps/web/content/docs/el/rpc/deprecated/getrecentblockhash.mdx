h1: Μέθοδος RPC getRecentBlockhash
---

Επιστρέφει ένα πρόσφατο block hash από το καθολικό, και ένα πρόγραμμα χρεώσεων
που μπορεί να χρησιμοποιηθεί για τον υπολογισμό του κόστους υποβολής μιας
συναλλαγής με τη χρήση του.

<Callout type="warn" title={"Μέθοδος σε Απόσυρση"}>
  Αυτή η μέθοδος αναμένεται να αφαιρεθεί στην `solana-core` v2.0. Παρακαλώ
  χρησιμοποιήστε [getLatestBlockhash](/docs/rpc/http/getlatestblockhash) αντ'
  αυτού.
</Callout>
