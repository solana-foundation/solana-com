Αυτή η πύλη λειτουργίας προσθέτει μια νέα οδηγία ψηφοφορίας ως προεπιλογή. Οι
υπάρχουσες οδηγίες ψηφοφορίας θα συνεχίσουν να λειτουργούν, αλλά οι νέες
ψηφοφορίες θα χρησιμοποιούν τη νέα μορφή οδηγιών στο μέλλον.

## Επίδραση

**Οι πάροχοι δεδομένων** θα πρέπει να ενημερωθούν για να παρακολουθούν τη νέα
οδηγία ψηφοφορίας, καθώς η νέα μορφή οδηγίας μπορεί να μην καταγράφεται στις
υπάρχουσες διαδικασίες ψηφοφορίας.
