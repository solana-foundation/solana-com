## Solana Kit

Το `@solana/kit` είναι το προτεινόμενο TypeScript SDK για ανάπτυξη στο Solana.

| Πακέτο                         | Περιγραφή                                  | GitHub                                                                                   |
| ------------------------------ | ------------------------------------------ | ---------------------------------------------------------------------------------------- |
| @solana/kit                    | Βασικό SDK                                 | [Πηγαίος κώδικας](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | Αλληλεπίδραση με το System Program         | [Πηγαίος κώδικας](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Αλληλεπίδραση με το Token Program          | [Πηγαίος κώδικας](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Αλληλεπίδραση με το Token-2022 Program     | [Πηγαίος κώδικας](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Αλληλεπίδραση με το Memo Program           | [Πηγαίος κώδικας](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Αλληλεπίδραση με το Compute Budget Program | [Πηγαίος κώδικας](https://github.com/solana-program/compute-budget/tree/main/clients/js) |

- [Επίσημη τεκμηρίωση Kit](https://www.solanakit.com/)
