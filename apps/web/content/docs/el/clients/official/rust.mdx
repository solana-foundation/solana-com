## Crates προγραμμάτων

Χρησιμοποιήστε το `solana-program` για να δημιουργήσετε on-chain προγράμματα.

- [Παράδειγμα: Δημιουργία προγράμματος Solana](/docs/programs/rust)

| Crate          | Περιγραφή                        | Τεκμηρίωση                                | GitHub                                                                        |
| -------------- | -------------------------------- | ----------------------------------------- | ----------------------------------------------------------------------------- |
| solana-program | Δημιουργία προγραμμάτων on-chain | [Προβολή](https://docs.rs/solana-program) | [Πηγαίος κώδικας](https://github.com/anza-xyz/solana-sdk/tree/master/program) |

Το Pinocchio είναι μια βιβλιοθήκη χωρίς εξαρτήσεις για τη δημιουργία
προγραμμάτων Solana σε Rust και μπορεί να χρησιμοποιηθεί ως εναλλακτική του
`solana-program` για τη συγγραφή on-chain προγραμμάτων.

| Crate     | Περιγραφή                             | Τεκμηρίωση                           | GitHub                                                   |
| --------- | ------------------------------------- | ------------------------------------ | -------------------------------------------------------- |
| pinocchio | Προγράμματα on-chain χωρίς εξαρτήσεις | [Προβολή](https://docs.rs/pinocchio) | [Πηγαίος κώδικας](https://github.com/anza-xyz/pinocchio) |

<Callout type="warn">
  Το Pinocchio βρίσκεται ακόμα υπό ανάπτυξη και πιθανότατα θα έχει σημαντικές
  αλλαγές στο εγγύς μέλλον.
</Callout>

Το Pinocchio περιλαμβάνει crates ειδικά για προγράμματα που αλληλεπιδρούν με
προγράμματα Solana:

| pinocchio-token                    | Αλληλεπίδραση με το Token Program            | [Προβολή](https://docs.rs/pinocchio-token/)                    | [Πηγαίος κώδικας](https://github.com/anza-xyz/pinocchio/tree/main/programs/token)                    |
| pinocchio-token-2022               | Αλληλεπίδραση με το Token-2022 Program       | [Προβολή](https://docs.rs/pinocchio-token-2022/)               | [Πηγαίος κώδικας](https://github.com/anza-xyz/pinocchio/tree/main/programs/token-2022)               |
