---

Υπάρχει ένα [Python SDK](https://github.com/michaelhly/solana-py) που
υποστηρίζει τις περισσότερες λειτουργίες για αλληλεπίδραση με το Solana. Αν
χρησιμοποιείτε το Anchor για τα προγράμματά σας στο Solana, θα μπορείτε να
χρησιμοποιήσετε το [anchor-py](https://github.com/kevinheavey/anchorpy) με το
idl σας για να δημιουργήσετε έναν συμβατό client για Python.

## Εγκατάσταση

```bash title="Terminal"
pip install solders
pip install solana
```

## Περισσότερη τεκμηρίωση

Παρακαλούμε επισκεφθείτε την
[τεκμηρίωση του Python SDK](https://github.com/michaelhly/solana-py?tab=readme-ov-file#general-usage)
για περισσότερες πληροφορίες σχετικά με τη χρήση αυτού του Python SDK με το
Solana.
