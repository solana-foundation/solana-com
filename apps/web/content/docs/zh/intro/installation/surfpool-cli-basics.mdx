Surfpool 是一个本地开发工具，通过自动从集群（主网、开发网）加载程序和账户，简化了 Solana 的开发。使用 Surfpool
CLI 启动本地 validator 来测试您的程序和交易。

访问 [Surfpool 文档](https://docs.surfpool.run/) 了解更多信息。

## 启动本地 validator

要启动本地 validator，请运行以下命令：

```terminal
$ surfpool start
```

默认情况下，Surfpool 会自动从主网加载您的交易所依赖的任何程序和账户到本地 validator。

## Surfpool Studio

Surfpool
Studio 提供了一个基于网页的界面，用于与您的本地 validator 交互。您可以使用它来：

- 查看交易详情
- 检查账户数据
- 使用 UI 水龙头空投 SOL 和其他代币

访问 Surfpool Studio 的步骤：

1. 使用 `surfpool start` 启动本地 validator
2. 打开浏览器，访问 [http://127.0.0.1:18488](http://127.0.0.1:18488)
