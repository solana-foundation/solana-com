## Solana Python SDK

有两个对使用 Python 开发非常有用的包：

[`@kevinheavy/solana.py`](https://github.com/michaelhly/solana-py)

- Solana.py 是与 Solana 交互的基础 Python 库。
- 您可以使用它构建交易并与 [Solana JSON RPC API](/docs/rpc) 交互，就像使用
  [solana-web3.js](https://github.com/solana-labs/solana-web3.js/) 一样。

- 它还涵盖了 [SPL Token Program](https://spl.solana.com/token)。

- [最新文档](https://michaelhly.github.io/solana-py/)。

[`@kevinheavey/solders`](https://github.com/kevinheavey/solders)

- `solders`
  是一个为 Solana 提供的高性能 Python 工具包，用 Rust 编写。它在以下领域提供了强大的解决方案：

- 核心 SDK 功能：密钥对、公钥、签名和序列化交易。
- RPC：构建请求和解析响应（对于网络和异步 API，[solana-py](https://michaelhly.github.io/solana-py/rpc/async_api/)
  是推荐选项）。
- 集成测试：`solders.litesvm` 模块是 `solana-test-validator`
  的替代方案，使用起来更加方便且**快得多**。它基于
  [solana-program-test](https://crates.io/crates/solana-program-test)。

## Solana 客户端 Python SDK

如果您想使用 Python 与 Solana 合约交互，可以使用客户端 SDK 将 Solana 的 Rust 合约转换为 Python 客户端，只要您能获取到合约的 IDL 文件。

[`Codama-py`](https://www.npmjs.com/package/codama-py)

- 将任何 Solana 链上合约程序标准化为
  [`Codama IDL`](https://github.com/codama-idl/codama)
  并生成一个交互式 Python 客户端。
- 支持 Anchor 和自定义程序。

[`Anchor-py`](https://github.com/kevinheavey/anchorpy)

- Anchor-py 是一个用于 Anchor 程序的 Python 客户端。它可以从 Anchor
  IDL 文件生成客户端，并与 Anchor 程序交互。
