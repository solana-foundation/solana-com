## Solana Kit

`@solana/kit` 是构建 Solana 应用的推荐 TypeScript SDK。

| 包名                           | 描述                           | GitHub                                                                        |
| ------------------------------ | ------------------------------ | ----------------------------------------------------------------------------- |
| @solana/kit                    | 核心 SDK                       | [源码](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | 与 System Program 交互         | [源码](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | 与 Token Program 交互          | [源码](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | 与 Token-2022 Program 交互     | [源码](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | 与 Memo Program 交互           | [源码](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | 与 Compute Budget Program 交互 | [源码](https://github.com/solana-program/compute-budget/tree/main/clients/js) |

- [官方 Kit 文档](https://www.solanakit.com/)
- [示例：使用 Next.js 连接钱包](/docs/frontend/kit)

`@solana/web3.js` 是 Solana 的旧版 TypeScript SDK。

| 包名              | 描述                                                  | GitHub                                                                           |
| ----------------- | ----------------------------------------------------- | -------------------------------------------------------------------------------- |
| @solana/web3.js   | 核心 SDK                                              | [源码](https://github.com/solana-foundation/solana-web3.js)                      |
| @solana/spl-token | 与 Token、Token-2022 和 Associated Token Program 交互 | [源码](https://github.com/solana-program/token-2022/tree/main/clients/js-legacy) |
| @solana/spl-memo  | 与 Memo Program 交互                                  | [源码](https://github.com/solana-program/memo/tree/main/clients/js-legacy)       |
