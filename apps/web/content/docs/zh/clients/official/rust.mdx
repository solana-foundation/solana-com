## 程序 Crates

使用 `solana-program` 构建链上程序。

- [示例：构建一个 Solana 程序](/docs/programs/rust)

| Crate          | 描述         | 文档                                   | GitHub                                                             |
| -------------- | ------------ | -------------------------------------- | ------------------------------------------------------------------ |
| solana-program | 构建链上程序 | [查看](https://docs.rs/solana-program) | [源码](https://github.com/anza-xyz/solana-sdk/tree/master/program) |

Pinocchio 是一个零依赖库，用于在 Rust 中创建 Solana 程序，并且可以作为
`solana-program` 的替代品来编写链上程序。

| Crate     | 描述           | 文档                              | GitHub                                        |
| --------- | -------------- | --------------------------------- | --------------------------------------------- |
| pinocchio | 零依赖链上程序 | [查看](https://docs.rs/pinocchio) | [源码](https://github.com/anza-xyz/pinocchio) |

<Callout type="warn">Pinocchio 仍在开发中，近期可能会有重大变更。</Callout>

Pinocchio 包括用于与 Solana 程序交互的特定程序 Crates：

| Crate                              | 描述                             | 文档                                                        | GitHub                                                                                    |
| ---------------------------------- | -------------------------------- | ----------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| pinocchio-system                   | 与 System Program 交互           | [查看](https://docs.rs/pinocchio-system/)                   | [源码](https://github.com/anza-xyz/pinocchio/tree/main/programs/system)                   |
| pinocchio-token                    | 与 Token Program 交互            | [查看](https://docs.rs/pinocchio-token/)                    | [源码](https://github.com/anza-xyz/pinocchio/tree/main/programs/token)                    |
| pinocchio-token-2022               | 与 Token-2022 Program 交互       | [查看](https://docs.rs/pinocchio-token-2022/)               | [源码](https://github.com/anza-xyz/pinocchio/tree/main/programs/token-2022)               |
| pinocchio-associated-token-account | 与 Associated Token Program 交互 | [查看](https://docs.rs/pinocchio-associated-token-account/) | [源码](https://github.com/anza-xyz/pinocchio/tree/main/programs/associated-token-account) |
