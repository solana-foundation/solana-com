h1: slotSubscribe RPC 方法
---

订阅以在验证器处理 slot 时接收通知

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "slotSubscribe"
}
```

### !params

**无**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

订阅 ID（用于取消订阅）

</APIMethod>

#### 通知格式：

通知将是一个包含以下字段的对象：

- `parent: <u64>` - 父 slot
- `root: <u64>` - 当前根 slot
- `slot: <u64>` - 新设置的 slot 值

示例：

```json
{
  "jsonrpc": "2.0",
  "method": "slotNotification",
  "params": {
    "result": {
      "parent": 75,
      "root": 44,
      "slot": 76
    },
    "subscription": 0
  }
}
```
