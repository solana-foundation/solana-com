Instalasi cepat untuk menyiapkan lingkungan pengembangan Solana lokal Anda
dengan satu perintah. Instal Rust, Solana CLI, dan Anchor Framework di Windows
(WSL), Linux, dan Mac. (Jika diinginkan, Anda dapat
[menginstal setiap dependensi secara terpisah](/docs/intro/installation/dependencies).)

## Prasyarat

<Accordions>
<Accordion title="Windows">

Untuk mengembangkan program Solana di Windows, Anda **harus** menggunakan
[Windows Subsystem for Linux](https://learn.microsoft.com/en-us/windows/wsl/install)
(WSL). Setelah menginstal WSL, semua dependensi lainnya akan diinstal melalui
terminal Linux.

Untuk menginstal WSL, jalankan perintah berikut di Windows PowerShell:

```terminal
$ wsl --install
```

Proses instalasi akan meminta Anda untuk membuat akun pengguna default.

![Instalasi WSL](/assets/docs/intro/installation/wsl-install.png)

Secara default, WSL menginstal Ubuntu. Anda dapat membuka terminal Linux dengan
mencari "Ubuntu" di bilah Pencarian.

![WSL Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-search.png)

Jika terminal Ubuntu Anda terlihat seperti gambar di bawah ini, Anda mungkin
mengalami masalah di mana `ctrl + v` (pintasan keyboard tempel) tidak berfungsi
di terminal.

![Terminal Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-terminal-1.png)

Jika Anda mengalami masalah ini, buka Windows Terminal dengan mencari "Terminal"
di bilah Pencarian.

![Windows Terminal](/assets/docs/intro/installation/wsl-windows-terminal.png)

Selanjutnya, tutup Windows Terminal dan buka kembali terminal Linux dengan
mencari Ubuntu lagi. Terminal sekarang seharusnya terlihat seperti gambar di
bawah ini, di mana `ctrl + v` (pintasan keyboard tempel) berfungsi.

![Terminal Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-terminal-2.png)

Jika Anda menggunakan VS Code, ekstensi
[WSL](https://code.visualstudio.com/docs/remote/wsl-tutorial) memungkinkan Anda
menggunakan WSL dan VS Code bersama-sama.

![Pengaturan WSL di VS Code](/assets/docs/intro/installation/wsl-vscode.png)

Anda akan melihat hal berikut di bilah status VS Code:

![WSL: Ubuntu](/assets/docs/intro/installation/wsl-vscode-ubuntu.png)

Setelah Anda mengatur WSL, Anda siap untuk:

- Menyelesaikan [instalasi cepat](#installation) menggunakan terminal Ubuntu
  (Linux)
- Menginstal [Rust](/docs/intro/installation/dependencies#install-rust),
  [Solana CLI](/docs/intro/installation/dependencies#install-solana-cli), dan
  [Anchor CLI](/docs/intro/installation/dependencies#install-anchor-cli) secara
  terpisah, seperti yang dijelaskan di bawah ini

</Accordion>

<Tab value=".deb" >
Jika menggunakan Debian atau alternatif turunan Debian seperti Ubuntu atau Mint, ikuti langkah-langkah di bawah ini:

1. Perbarui daftar paket untuk memastikan semua paket sudah terbaru:

```terminal
$ sudo apt-get update
```

2. Instal dependensi berikut:

```terminal
$ sudo apt-get install -y \
    build-essential \
    pkg-config \
    libudev-dev llvm libclang-dev \
    protobuf-compiler libssl-dev
```

<Callout>
Jika Anda mengalami kesalahan berikut saat menginstal `protobuf-compiler`, pastikan
Anda terlebih dahulu menjalankan _shell`sudo apt-get update`

```

This may mean that the package is missing, has been obsoleted, or
is only available from another source
```

</Callout>

## Instalasi

1. Masukkan perintah berikut ke terminal Anda untuk menginstal semua dependensi
   yang diperlukan:

```terminal
$ curl --proto '=https' --tlsv1.2 -sSfL https://solana-install.solana.workers.dev | bash
```

Instalasi yang berhasil akan menampilkan output seperti berikut:

```
Installed Versions:
Rust: rustc 1.86.0 (05f9846f8 2025-03-31)
Solana CLI: solana-cli 2.2.12 (src:0315eb6a; feat:1522022101, client:Agave)
Anchor CLI: anchor-cli 0.31.1
Node.js: v23.11.0
Yarn: 1.22.1
```

2. Verifikasi instalasi yang berhasil dengan memeriksa versi dari setiap
   dependensi yang diinstal.

```terminal
$ rustc --version && solana --version && anchor --version && node --version && yarn --version
```

<Callout>
  Jika perintah instalasi cepat gagal, silakan merujuk ke bagian [Instal
  Dependensi](/docs/intro/installation/dependencies) di bawah untuk petunjuk
  cara menginstal setiap dependensi secara individual.
</Callout>

### Langkah selanjutnya

Setelah Anda menginstal semua dependensi yang diperlukan, Anda siap untuk
mempelajari [Dasar-dasar CLI Solana](/docs/intro/installation/solana-cli-basics)
dan [Dasar-dasar CLI Anchor](/docs/intro/installation/anchor-cli-basics).
