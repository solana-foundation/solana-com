Surfpool adalah alat pengembangan lokal yang menyederhanakan pengembangan Solana
dengan secara otomatis memuat program dan akun dari sebuah cluster (mainnet,
devnet). Gunakan Surfpool CLI untuk memulai validator lokal guna menguji program
dan transaksi Anda.

Kunjungi [dokumentasi Surfpool](https://docs.surfpool.run/) untuk informasi
lebih lanjut.

## Memulai validator lokal

Untuk memulai validator lokal, jalankan perintah berikut:

```terminal
$ surfpool start
```

Secara default, Surfpool secara otomatis memuat program dan akun apa pun yang
transaksi Anda butuhkan dari mainnet ke validator lokal.

## Surfpool Studio

Surfpool Studio menyediakan antarmuka berbasis web untuk berinteraksi dengan
validator lokal Anda. Gunakan untuk:

- Melihat detail transaksi
- Memeriksa data akun
- Airdrop SOL dan token lainnya menggunakan faucet UI

Untuk mengakses Surfpool Studio:

1. Mulai validator lokal Anda dengan `surfpool start`
2. Buka browser Anda dan navigasikan ke
   [http://127.0.0.1:18488](http://127.0.0.1:18488)
