Selamat datang di panduan mulai cepat Solana! Panduan praktis ini akan
memperkenalkan Anda kepada **Solana Playground** (Solpg), di mana Anda dapat
mempelajari konsep-konsep inti, terlepas dari pengalaman Anda sebelumnya. Dalam
tutorial ini, Anda akan mempelajari tentang:

- **Akun Solana**: Bagaimana jaringan Solana menyimpan data
- **Mengirim transaksi**: Cara berinteraksi dengan jaringan Solana dengan
  mengirim transaksi
- **Membangun dan men-deploy program**: Buat program Solana pertama Anda dan
  deploy ke jaringan
- **Alamat Program-Derived**: Pelajari cara membuat alamat deterministik untuk
  akun
- **Cross-Program Invocations**: Pelajari cara memanggil program lain dari dalam
  program Anda, memungkinkan interaksi kompleks dan komposabilitas program

## Solana Playground

Solana Playground adalah lingkungan pengembangan berbasis browser yang
memungkinkan Anda mengembangkan, men-deploy, dan menguji program Solana tanpa
menginstal apa pun secara lokal. Ikuti langkah-langkahnya, salin, dan tempel
kode, dan lihat hasilnya segera. Pengetahuan pemrograman dasar membantu tetapi
tidak diperlukan. Saatnya menyelami dan mulai membangun dengan Solana!

Sebagai pengguna baru, langkah pertama untuk berinteraksi dengan Solana
Playground adalah membuat [dompet](/docs/references/terminology#wallet)
Playground dan menambahkan [SOL](/docs/references/terminology#sol). Dompet ini
akan memungkinkan Anda berinteraksi dengan jaringan Solana langsung dari browser
Anda.

### Simpan keypair Anda

Klik **Save keypair** untuk menyimpan keypair anda secara lokal. Ketika anda
siap, klik **Continue**.

![Membuat Dompet Playground](/assets/docs/intro/quickstart/pg-create-wallet.png)

Di bagian bawah layar, Anda akan melihat bahwa Anda sekarang terhubung ke dompet
Playground. Anda juga akan melihat
[kluster yang terhubung](/docs/references/terminology#connected-cluster)
(`devnet`), [alamat dompet](/docs/references/terminology#wallet-address), dan
saldo [SOL](/docs/references/terminology#sol) Anda.

![Terhubung](/assets/docs/intro/quickstart/pg-connected.png)

<Callout>
  Penyimpanan lokal browser anda menyimpan dompet Playground anda. Menghapus
  cache browser anda akan menghapus dompet yang tersimpan.
</Callout>

<Callout type="warn">
  Gunakan dompet Playground Anda hanya untuk pengujian dan pengembangan. Jangan
  pernah mengirim aset nyata (dari
  [`mainnet`](/docs/references/terminology#connected-cluster)) ke alamat ini.
</Callout>

Selamat! Anda telah membuat Dompet Solana pertama anda. Anda sekarang siap untuk
menambahkan SOL ke dompet anda.

</Step>
<Step>

### Tambahkan SOL ke dompet Anda

Sebelum memulai pengembangan, anda perlu menambahkan devnet SOL ke dompet anda.
Sebagai pengembang, SOL digunakan untuk dua kasus penggunaan utama:

- Untuk membuat akun baru untuk menyimpan data atau men-deploy program
- Membayar biaya transaksi saat berinteraksi dengan jaringan Solana

Anda dapat menambahkan devnet SOL ke dompet anda menggunakan terminal Playground
atau dengan Devnet Faucet. Setiap opsi dijelaskan di bawah ini:

#### Opsi 1: Gunakan terminal Playground

Jalankan perintah berikut di terminal Playground:

```terminal
$ solana airdrop 5
```

#### Opsi 2: Gunakan faucet Devnet

Jika opsi 1 gagal, karena batas rate atau kesalahan lainnya, gunakan
[Web Faucet](https://faucet.solana.com/) Solana.

- Masukkan alamat dompet anda
- Pilih jumlah
- Klik **Confirm Airdrop**

![Faucet Airdrop](/assets/docs/intro/quickstart/faucet-airdrop.gif)

</Step>
</Steps>
