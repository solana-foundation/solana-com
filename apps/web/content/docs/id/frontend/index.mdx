title: Frontend

---

Pengembangan frontend di Solana melibatkan bekerja dengan program, dompet, dan
framework JavaScript populer seperti React. Berinteraksi dengan
komponen-komponen ini memerlukan penanganan koneksi, pembuatan transaksi, dan
pembacaan dari akun Solana.

Untuk membantu pekerjaan ini, berbagai pustaka klien Solana tersedia dalam
framework yang berbeda.

## Pustaka utama

<Cards>
  <Card
    title="@solana/client"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/client"
  >
    - Klien Solana sederhana yang menggabungkan RPC, dompet, transaksi -
    Menyertakan state store bawaan, actions, watchers, connectors
  </Card>
  <Card
    title="@solana/react-hooks"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks"
  >
    - Hooks lengkap untuk dompet, saldo, transfer, tanda tangan, query - Hooks
    provider React yang membungkus `@solana/client` runtime state
  </Card>
  <Card
    title="@solana/web3-compat"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/web3-compat"
  >
    - Toolkit kompatibel Web3.js untuk menyederhanakan upgrade. - Internal yang
    lebih baru mengandalkan campuran web3.js dan kit.
  </Card>
  <Card
    title="@solana/kit"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/kit"
  >
    - SDK Solana tingkat rendah yang menggerakkan pustaka Solana lainnya seperti
    `@solana/react-hooks` - Sepenuhnya tree-shakable, menggunakan standar web
    modern, dan menggerakkan runtime
  </Card>
</Cards>

<Callout type="warn" title="@solana/web3.js sudah deprecated`">
  Banyak proyek ekosistem Solana masih mengandalkan `@solana/web3.js` yang sudah
  deprecated. Lebih baik gunakan `@solana/web3-compat` untuk menyederhanakan
  jalur migrasi Anda.
</Callout>

- [Panduan @solana/client](/docs/frontend/client): runtime lean dan headless
  untuk RPC, dompet, dan transaksi.
- [Panduan @solana/react-hooks](/docs/frontend/react-hooks): hooks React yang
  dibangun di atas runtime klien yang sama.
- [Panduan @solana/web3-compat](/docs/frontend/web3-compat): lapisan
  kompatibilitas untuk migrasi dari `@solana/web3.js` ke stack berbasis Kit.
