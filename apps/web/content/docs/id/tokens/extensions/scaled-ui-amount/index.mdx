## Apa itu ekstensi Jumlah UI Terskala?

Ekstensi jumlah UI terskala memungkinkan penerbit untuk menerapkan pengali yang
dapat diperbarui pada jumlah UI token. Ini berguna untuk skenario di mana token
digunakan untuk mewakili aset dunia nyata, seperti saham atau dividen.

Misalnya, pemecahan saham menggandakan jumlah saham untuk semua pemegang. Tanpa
ekstensi ini, saat ini tidak praktis untuk mencetak token baru kepada semua
pemegang selama peristiwa pemecahan dengan cara yang efisien dan terukur.

Dengan model ekstensi Token-2022, bagaimanapun, kita memiliki kemungkinan untuk
mengubah cara jumlah UI token direpresentasikan. Menggunakan ekstensi
ScaledUiAmount dan instruksi `amount_to_ui_amount`, Anda dapat mengatur pengali
UI pada token Anda dan mengambil jumlah UI-nya kapan saja.
[kode sumber](https://github.com/solana-program/token-2022/tree/main/program/src/extension/scaled_ui_amount).

Fitur ini juga dapat digunakan untuk dividen atau mendistribusikan hasil.

Catatan: Tidak ada token baru yang pernah dibuat, jumlah UI mengembalikan jumlah
mentah token dikalikan dengan pengali saat ini. Fitur ini sepenuhnya bersifat
kosmetik.

<Callout type="info">
  Beberapa ekstensi tidak kompatibel satu sama lain dan Anda tidak dapat
  mengaktifkannya secara bersamaan pada mint token atau token account yang sama.
  Anda tidak boleh mengaktifkan ekstensi `ScaledUiAmount` dan ekstensi
  `InterestBearingMint` pada mint token atau token account yang sama.
</Callout>
