Setelah terhubung ke websocket RPC PubSub di `ws://<ADDRESS>/`:

- Kirim permintaan langganan ke websocket menggunakan metode di bawah ini
- Beberapa langganan dapat aktif sekaligus
- Banyak langganan mengambil parameter opsional
  [`commitment` parameter](/docs/rpc/#configuring-state-commitment), yang
  mendefinisikan seberapa final suatu perubahan harus terjadi untuk memicu
  notifikasi. Untuk langganan, jika commitment tidak ditentukan, nilai
  defaultnya adalah `finalized`.

## Endpoint WebSocket RPC PubSub

Port default: `8900`

- ws://localhost:8900
- http://192.168.1.88:8900
