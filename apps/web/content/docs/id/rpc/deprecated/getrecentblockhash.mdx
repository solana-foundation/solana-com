h1: Metode RPC getRecentBlockhash
---

Mengembalikan hash blok terbaru dari buku besar, dan jadwal biaya yang dapat
digunakan untuk menghitung biaya pengiriman transaksi menggunakan hash tersebut.

<Callout type="warn" title={"Metode yang Tidak Digunakan Lagi"}>
  Metode ini diperkirakan akan dihapus di `solana-core` v2.0. Silakan gunakan
  [getLatestBlockhash](/docs/rpc/http/getlatestblockhash) sebagai gantinya.
</Callout>
