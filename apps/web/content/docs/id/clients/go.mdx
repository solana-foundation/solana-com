h1: Go SDK untuk Solana
---

Ada [Go SDK](https://github.com/gagliardetto/solana-go) yang mendukung sebagian
besar fungsi untuk berinteraksi dengan Solana. Jika Anda menggunakan Anchor
untuk program Solana Anda, Anda akan dapat menggunakan
[anchor-go](https://github.com/gagliardetto/anchor-go) dengan idl Anda untuk
menghasilkan klien yang kompatibel dengan Golang.

## Instalasi

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```

## Dokumentasi Lebih Lanjut

Silakan kunjungi
[dokumentasi Go SDK](https://github.com/gagliardetto/solana-go?tab=readme-ov-file#installation)
