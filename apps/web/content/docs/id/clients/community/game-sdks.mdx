Dengan Solana sebagai blockchain berkinerja tinggi dengan biaya rendah dan
transaksi cepat, komunitas pengembang game yang hebat telah terbentuk di
sekitarnya.

Di sini Anda dapat menemukan detail tentang SDK pengembangan game Solana yang
tersedia dalam ekosistem. Semua SDK bersifat open source dan dibangun oleh
komunitas. Jika Anda sedang mengerjakan SDK sendiri, silakan buka PR ke halaman
ini dan tambahkan.

## SDK Unity

Game engine Unity dikenal dengan pendekatannya yang ramah pemula dan dukungan
lintas platform termasuk WebGL, iOS, dan Android. Bangun sekali, ekspor ke mana
saja.

- [Dokumentasi](https://docs.magicblock.gg/introduction)
- [Sumber](https://github.com/magicblock-labs/Solana.Unity-SDK)
- [Contoh Game](https://github.com/solana-developers/solana-game-examples)
- [Tic Tac Toe](https://blog.magicblock.gg/bolt-tic-tac-toe/)

SDK Solana Unity dikelola oleh [Magicblock](https://www.magicblock.gg/) dan
dilengkapi dengan:

- Dukungan NFT
- Integrasi candy machine
- Transaksi
- Fungsi RPC
- Deep link Phantom
- Konektor WebGL
- Dukungan koneksi WebSocket yang andal
- Wallet-adapter untuk mobile
- Kunci sesi untuk menyetujui transaksi secara otomatis
- Pembuatan kode klien Anchor dan lainnya.

## SDK Godot

[Godot Engine](https://godotengine.org/) adalah game engine open source yang
mendapatkan banyak dukungan dalam beberapa tahun terakhir. SDK Solana untuk
Godot dikelola oleh [ZenRepublic](https://twitter.com/ZenRepublicNDM) dan
[Virus-Axel](https://twitter.com/AxelBenjam). SDK ini dilengkapi dengan
integrasi wallet adapter, transaksi, fungsi RPC, dan pembuatan kode klien
Anchor. Bahkan integrasi Metaplex candy machine yang berfungsi penuh
[integrasi candy machine](https://zenwiki.gitbook.io/solana-godot-sdk-docs/guides/setup-candy-machine).

- [SDK Solana Godot Engine](https://github.com/Virus-Axel/godot-solana-sdk)
- [Dokumentasi](https://zenwiki.gitbook.io/solana-godot-sdk-docs)
- [Demo](https://github.com/ZenRepublic/GodotSolanaSDKDemoPackage)
- [Tutorial](https://www.youtube.com/watch?v=tszFPInYmXQ)
- [Pengelola](https://twitter.com/ZenRepublicNDM)

## Turbo.Computer - Game Engine Rust

Turbo adalah mesin game rust yang ditulis dari awal dengan fokus pada Solana,
dengan arsitektur ringan dan waktu iterasi yang cepat. Mesin ini ramah untuk
pemula dan dilengkapi dengan dukungan Solana RPC penuh. Anda bahkan dapat
menggunakan alat AI-nya untuk menghasilkan game lengkap.

- [Website](https://turbo.computer/)
- [Docs](https://docs.turbo.computer/)
- [Akun Twitter](https://twitter.com/makegamesfast)
- [Pengelola](https://twitter.com/jozanza)

# Honeycomb Protocol

Honeycomb menyediakan rangkaian program onchain dan alat kompresi status yang
menangani semua fungsi siklus hidup game yang penting dan komposabilitas dalam
Solana Virtual Machine (SVM). Ini mendukung banyak fungsionalitas NFT seperti
staking, misi, loot box, profil pemain, kompresi status, persetujuan transaksi
otomatis dan banyak lagi.

- [Akun Twitter](https://twitter.com/honeycomb_prtcl)
- [Docs](https://docs.honeycombprotocol.com/)

# Unreal SDKs

Unreal Engine dikenal dengan visual yang hebat dan kerangka skrip berbasis node.
Ada beberapa SDK Solana yang dikelola oleh perusahaan yang berbeda.

### Varmeta Unreal SDK

VAR META menyediakan SDK Unreal open-source untuk mengintegrasikan
fungsionalitas blockchain Solana ke dalam proyek Unreal Engine. Ini memungkinkan
pengembang untuk berinteraksi dengan kontrak pintar Solana, mengelola dompet,
dan mengeksekusi transaksi langsung dalam Unreal Engine.

- [Unreal SDK oleh Varmeta](https://github.com/VAR-META-Tech/UnrealSolanaSDK)
- [Pengelola](https://www.var-meta.com/)

### Bitfrost Unreal SDK

Bitfrost sedang mengerjakan SDK Unreal yang sudah digunakan dalam beberapa game
yang dibangun selama game jam Solana baru-baru ini. SDK ini hadir dengan
dukungan C# `solnet` dalam C++ dan Blueprints, minting NFT metaplex, contoh
prosesor pembayaran dalam dompet game dan lainnya.

- [Solana Unreal SDK oleh Bitfrost](https://github.com/Bifrost-Technologies/Solana-Unreal-SDK)
- [Tutorial](https://www.youtube.com/watch?v=S8fm8mFeUkk)
- [Pengelola](https://twitter.com/BifrostTitan)

### Thugz Unreal SDK

Thugz adalah studio teknologi dan proyek NFT yang juga memelihara SDK Unreal
open source untuk Solana. SDK ini hadir dengan banyak fungsionalitas yang
berfokus pada NFT dan plugin ini juga kompatibel dengan iOS dan Android.

- [Plugin Thugz Blockchain](https://www.fab.com/listings/a7e220ba-93e5-4a3b-99e7-17b13da76f05)
- [Repositori](https://github.com/ThugzLabs/Thugz-BC-Plugin-Packaged-for-UE5.4)
- [Dokumentasi](https://thugzblockchainplugin-docs.vercel.app/)
- [Tutorial Video](https://www.youtube.com/watch?v=dS7sTZd_E9U&ab_channel=ThugzNFT)

### Star Atlas Foundation Kit

Tim [Star Atlas](https://staratlas.com/) telah membuka sumber (open source)
beberapa SDK mereka yang disebut _Foundation Kit_. Meskipun tidak aktif
dipelihara, ini bisa menjadi titik awal yang baik untuk proyek Anda.

- [Star Atlas Unreal SDK](https://github.com/staratlasmeta/FoundationKit)
- [Tutorial](https://www.youtube.com/watch?v=S8fm8mFeUkk)

## Next.js/React + Anchor

Salah satu cara termudah untuk membangun di Solana adalah menggunakan kerangka
kerja JavaScript Web3js dikombinasikan dengan kerangka kerja Solana Anchor.
Untuk game yang lebih kompleks, kami merekomendasikan untuk menggunakan Game
Engine seperti [Unity](#unity-sdk) atau [Unreal](#unreal-sdks).

Cara tercepat untuk menyiapkan game berbasis Next.js:

```shell
npx create-solana-game your-game-name
```

Ini akan menghasilkan aplikasi awal yang bagus dengan dukungan `wallet-adapter`,
program Anchor, aplikasi React, dan klien Unity yang sudah dikonfigurasi untuk
bekerja bersama. Keuntungan menggunakan Next.js adalah Anda dapat menggunakan
kode yang sama di backend dan frontend, mempercepat pengembangan.

- [Web3.js](https://solana.com/docs/clients/javascript)
- [Tutorial Video](https://www.youtube.com/watch?v=fnhivg_pemI&t=1s&ab_channel=Solana)

## Phaser

Kerangka kerja sumber terbuka yang cepat, gratis, dan menyenangkan untuk game
berbasis browser yang didukung Canvas dan WebGL. [Phaser](https://phaser.io/)
adalah cara yang bagus untuk memulai pengembangan game di Solana. Template
Phaser Solana Platformer adalah titik awal yang bagus untuk game Anda.

- [Kode Sumber](https://github.com/Bread-Heads-NFT/solana-phaser-template)
- [Pengelola](https://twitter.com/blockiosaurus)

## Flutter

Flutter adalah kerangka kerja sumber terbuka yang dikelola oleh Google untuk
membangun aplikasi multi-platform yang indah, dikompilasi secara native, dari
satu basis kode. SDK Solana Flutter dikelola oleh tim
[Espresso Cash](https://www.espressocash.com/).

- [Kode Sumber](https://github.com/espresso-cash/espresso-cash-public)

## Python

Python adalah bahasa pemrograman yang mudah dipelajari dan banyak digunakan
(sering digunakan dalam aplikasi pembelajaran mesin). Framework Seahorse
memungkinkan pengembang untuk menulis program Solana dalam Python. Seahorse
dibangun di atas framework Anchor dan mengkonversi kode Python menjadi kode rust
berbasis Anchor. Seahorse saat ini masih dalam tahap beta.

- [Dokumentasi](https://www.seahorse.dev/)
- [Repositori Github](https://github.com/solana-developers/seahorse)
- [Contoh Anchor Playground](https://beta.solpg.io/tutorials/hello-seahorse)
