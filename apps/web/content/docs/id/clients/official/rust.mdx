## Program Crates

Gunakan `solana-program` untuk membangun program on-chain.

- [Contoh: Membangun program Solana](/docs/programs/rust)

| Crate          | Deskripsi                  | Docs                                   | GitHub                                                               |
| -------------- | -------------------------- | -------------------------------------- | -------------------------------------------------------------------- |
| solana-program | Membangun program on-chain | [View](https://docs.rs/solana-program) | [Source](https://github.com/anza-xyz/solana-sdk/tree/master/program) |

Pinocchio adalah library tanpa dependensi untuk membuat program Solana dalam
Rust dan dapat digunakan sebagai pengganti `solana-program` untuk menulis
program on-chain.

| Crate     | Deskripsi                         | Docs                              | GitHub                                          |
| --------- | --------------------------------- | --------------------------------- | ----------------------------------------------- |
| pinocchio | Program on-chain tanpa dependensi | [View](https://docs.rs/pinocchio) | [Source](https://github.com/anza-xyz/pinocchio) |

<Callout type="warn">
  Pinocchio masih dalam pengembangan dan kemungkinan akan mengalami perubahan
  yang signifikan dalam waktu dekat.
</Callout>

Pinocchio mencakup crate khusus program untuk berinteraksi dengan program
Solana:

| Crate                              | Deskripsi                                    | Docs                                                        | GitHub                                                                                      |
| ---------------------------------- | -------------------------------------------- | ----------------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| pinocchio-system                   | Berinteraksi dengan System Program           | [View](https://docs.rs/pinocchio-system/)                   | [Source](https://github.com/anza-xyz/pinocchio/tree/main/programs/system)                   |
| pinocchio-token                    | Berinteraksi dengan Token Program            | [View](https://docs.rs/pinocchio-token/)                    | [Source](https://github.com/anza-xyz/pinocchio/tree/main/programs/token)                    |
| pinocchio-token-2022               | Berinteraksi dengan Token-2022 Program       | [View](https://docs.rs/pinocchio-token-2022/)               | [Source](https://github.com/anza-xyz/pinocchio/tree/main/programs/token-2022)               |
| pinocchio-associated-token-account | Berinteraksi dengan Associated Token Program | [View](https://docs.rs/pinocchio-associated-token-account/) | [Source](https://github.com/anza-xyz/pinocchio/tree/main/programs/associated-token-account) |
| pinocchio-memo                     | Berinteraksi dengan Memo Program             | [View](https://docs.rs/pinocchio-memo/)                     | [Source](https://github.com/anza-xyz/pinocchio/tree/main/programs/memo)                     |
