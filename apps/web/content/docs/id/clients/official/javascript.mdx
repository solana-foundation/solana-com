## Solana Kit

`@solana/kit` adalah TypeScript SDK yang direkomendasikan untuk membangun di
Solana.

| Paket                          | Deskripsi                                  | GitHub                                                                          |
| ------------------------------ | ------------------------------------------ | ------------------------------------------------------------------------------- |
| @solana/kit                    | SDK Utama                                  | [Source](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | Berinteraksi dengan System Program         | [Source](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Berinteraksi dengan Token Program          | [Source](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Berinteraksi dengan Token-2022 Program     | [Source](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Berinteraksi dengan Memo Program           | [Source](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Berinteraksi dengan Compute Budget Program | [Source](https://github.com/solana-program/compute-budget/tree/main/clients/js) |

- [Dokumentasi Kit Resmi](https://www.solanakit.com/)
- [Contoh: Menghubungkan dompet dengan Next.js](/docs/frontend/kit)

| Paket             | Deskripsi                                                           | GitHub                                                                             |

| @solana/web3.js   | SDK Utama                                                           | [Source](https://github.com/solana-foundation/solana-web3.js)                      |
