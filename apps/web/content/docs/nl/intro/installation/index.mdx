Een snelle installatie om je lokale Solana-ontwikkelomgeving op te zetten met
één commando. Installeer Rust, de Solana CLI en Anchor Framework op Windows
(WSL), Linux en Mac. (Indien gewenst, kun je ook
[elke dependency afzonderlijk installeren](/docs/intro/installation/dependencies).)

## Vereisten

<Accordions>
<Accordion title="Windows">

Om Solana-programma's op Windows te ontwikkelen **moet** je
[Windows Subsystem for Linux](https://learn.microsoft.com/en-us/windows/wsl/install)
(WSL) gebruiken. Na het installeren van WSL worden alle andere dependencies
geïnstalleerd via de Linux-terminal.

Om WSL te installeren, voer je het volgende commando uit in Windows PowerShell:

```terminal
$ wsl --install
```

Het installatieproces vraagt je om een standaard gebruikersaccount aan te maken.

![WSL Install](/assets/docs/intro/installation/wsl-install.png)

Standaard installeert WSL Ubuntu. Je kunt een Linux-terminal openen door
"Ubuntu" te zoeken in de zoekbalk.

![WSL Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-search.png)

Als je Ubuntu-terminal eruitziet zoals de afbeelding hieronder, kun je een
probleem tegenkomen waarbij `ctrl + v` (sneltoets voor plakken) niet werkt in de
terminal.

![Ubuntu Terminal](/assets/docs/intro/installation/wsl-ubuntu-terminal-1.png)

Als je dit probleem tegenkomt, open dan Windows Terminal door te zoeken naar
"Terminal" in de zoekbalk.

![Windows Terminal](/assets/docs/intro/installation/wsl-windows-terminal.png)

Sluit vervolgens de Windows Terminal en open opnieuw een Linux-terminal door te
zoeken naar Ubuntu. De terminal zou er nu uit moeten zien zoals de afbeelding
hieronder, waar `ctrl + v` (sneltoets voor plakken) wel werkt.

![Ubuntu Terminal](/assets/docs/intro/installation/wsl-ubuntu-terminal-2.png)

Als je VS Code gebruikt, maakt de
[WSL-extensie](https://code.visualstudio.com/docs/remote/wsl-tutorial) het
mogelijk om WSL en VS Code samen te gebruiken.

![WSL Setup in VS Code](/assets/docs/intro/installation/wsl-vscode.png)

Je ziet het volgende in de statusbalk van VS Code:

![WSL: Ubuntu](/assets/docs/intro/installation/wsl-vscode-ubuntu.png)

Zodra je WSL hebt ingesteld, ben je klaar om:

- De [snelle installatie](#installation) te voltooien via de Ubuntu (Linux)
  terminal
- [Rust](/docs/intro/installation/dependencies#install-rust), de
  [Solana CLI](/docs/intro/installation/dependencies#install-solana-cli), en de
  [Anchor CLI](/docs/intro/installation/dependencies#install-anchor-cli)
  afzonderlijk te installeren, zoals hieronder beschreven

<Tab value=".deb" >
Als je Debian of Debian-afgeleide alternatieven zoals Ubuntu of Mint gebruikt, volg dan de onderstaande stappen:

1. Update de pakketlijsten om ervoor te zorgen dat alle pakketten up-to-date
   zijn:

```terminal
$ sudo apt-get update
```

2. Installeer de volgende afhankelijkheden:

```terminal
$ sudo apt-get install -y \
    build-essential \
    pkg-config \
    libudev-dev llvm libclang-dev \
    protobuf-compiler libssl-dev
```

<Callout>
Als je de volgende fout tegenkomt bij het installeren van `protobuf-compiler`, zorg
er dan voor dat je eerst _shell`sudo apt-get update` uitvoert

```
Package protobuf-compiler is not available, but is referred to by another package.
This may mean that the package is missing, has been obsoleted, or
is only available from another source
```

</Callout>

## Installatie

1. Voer het volgende commando in je terminal in om alle benodigde
   afhankelijkheden te installeren:

```terminal
$ curl --proto '=https' --tlsv1.2 -sSfL https://solana-install.solana.workers.dev | bash
```

Een succesvolle installatie zal output geven zoals het volgende:

```
Installed Versions:
Rust: rustc 1.86.0 (05f9846f8 2025-03-31)
Solana CLI: solana-cli 2.2.12 (src:0315eb6a; feat:1522022101, client:Agave)
Anchor CLI: anchor-cli 0.31.1
Node.js: v23.11.0
Yarn: 1.22.1
```

2. Verifieer een succesvolle installatie door de versie van elke geïnstalleerde
   dependency te controleren.

```terminal
$ rustc --version && solana --version && anchor --version && node --version && yarn --version
```

<Callout>
  Als het snelle installatiecommando mislukt, raadpleeg dan het gedeelte
  [Dependencies installeren](/docs/intro/installation/dependencies) hieronder
  voor instructies om elke dependency afzonderlijk te installeren.
</Callout>

### Volgende stappen

Zodra je alle benodigde dependencies hebt geïnstalleerd, ben je klaar om de
[Solana CLI Basics](/docs/intro/installation/solana-cli-basics) en
[Anchor CLI Basics](/docs/intro/installation/anchor-cli-basics) te bekijken.
