Er is een [Go SDK](https://github.com/gagliardetto/solana-go) beschikbaar die de
meeste functies ondersteunt om met Solana te werken. Als je Anchor gebruikt voor
je Solana programma's, kun je
[anchor-go](https://github.com/gagliardetto/anchor-go) gebruiken met je idl om
een Golang-compatibele client te maken.

## Installatie

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```
