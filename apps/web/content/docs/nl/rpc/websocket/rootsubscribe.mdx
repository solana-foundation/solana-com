h1: rootSubscribe RPC-methode
---

Abonneer je om een melding te ontvangen wanneer een nieuwe root wordt ingesteld
door de validator.

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**Geen**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

abonnement-id (nodig om het abonnement op te zeggen)

</APIMethod>

#### Meldingsformaat:

Het resultaat is het laatste root slot-nummer.

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
