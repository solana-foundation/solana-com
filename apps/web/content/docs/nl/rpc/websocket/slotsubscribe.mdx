h1: slotSubscribe RPC-methode
---

Abonneer je om een melding te ontvangen wanneer een slot wordt verwerkt door de
validator

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "slotSubscribe"
}
```

### !params

**Geen**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

Abonnement-id (nodig om het abonnement op te zeggen)

</APIMethod>

#### Meldingsformaat:

De melding is een object met de volgende velden:

- `parent: <u64>` - Het bovenliggende slot
- `root: <u64>` - Het huidige root-slot
- `slot: <u64>` - De nieuw ingestelde slotwaarde

Voorbeeld:

```json
{
  "jsonrpc": "2.0",
  "method": "slotNotification",
  "params": {
    "result": {
      "parent": 75,
      "root": 44,
      "slot": 76
    },
    "subscription": 0
  }
}
```
