h1: voteSubscribe RPC-methode
---

Abonneer je om een melding te ontvangen wanneer een nieuwe stem wordt
waargenomen in gossip. Deze stemmen zijn pre-consensus, daarom is er geen
garantie dat deze stemmen in het grootboek terechtkomen.

<Callout type="warn" title={"Onstabiele methode"}>
  Dit abonnement is onstabiel en alleen beschikbaar als de validator is gestart
  met de `--rpc-pubsub-enable-vote-subscription` vlag. Het formaat van dit
  abonnement kan in de toekomst veranderen.
</Callout>

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "voteSubscribe"
}
```

### !params

**Geen**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

abonnement-id (nodig om je af te melden)

</APIMethod>

#### Meldingsformaat:

De melding is een object met de volgende velden:

- `hash: <string>` - De stem-hash
- `slots: <array>` - De slots die door de stem worden gedekt, als een array van
  u64 integers
- `timestamp: <i64|null>` - De tijdstempel van de stem
- `signature: <string>` - De handtekening van de transactie die deze stem
  bevatte
- `votePubkey: <string>` - De publieke sleutel van het stem-account, als base-58
  gecodeerde string

```json
{
  "jsonrpc": "2.0",
  "method": "voteNotification",
  "params": {
    "result": {
      "hash": "8Rshv2oMkPu5E4opXTRyuyBeZBqQ4S477VG26wUTFxUM",
      "slots": [1, 2],
      "timestamp": null
    },
    "subscription": 0
  }
}
```
