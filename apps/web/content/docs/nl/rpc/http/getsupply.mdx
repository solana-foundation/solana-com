h1: getSupply RPC-methode

Geeft informatie over de huidige voorraad.

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getSupply",
  "params": [
    // !hover(1:3) 0
    {
      // !hover commitment
      "commitment": "finalized"
    }
  ]
}
```

```ts !!request title="Kit"
import { createSolanaRpc } from "@solana/kit";

const rpc_url = "https://api.devnet.solana.com";
const rpc = createSolanaRpc(rpc_url);

let supply = await rpc.getSupply().send();

console.log(supply);
```

```ts !!request title="web3.js"
import { Connection, clusterApiUrl } from "@solana/web3.js";

const connection = new Connection(clusterApiUrl("devnet"), "confirmed");

let supply = await connection.getSupply();

console.log(supply);
```

### !params

#### !! 0

!type object

Configuratieobject met de volgende velden:

##### !! commitment

!type string

De commitment beschrijft hoe definitief een blok is op dat moment. Zie
[Configureren van state commitment](/docs/rpc#configuring-state-commitment).

##### !! excludeNonCirculatingAccountsList

!type bool

Sluit niet-circulerende accountslijst uit van de respons

!type object

Het resultaat is een JSON-object dat het volgende bevat:

##### !! circulating

!type u64

Circulerende voorraad in lamports

##### !! nonCirculating

!type u64

Niet-circulerende voorraad in lamports

##### !! nonCirculatingAccounts

!type array

Een array van accountadressen van niet-circulerende accounts, als strings. Als
`excludeNonCirculatingAccountsList` is ingeschakeld, zal de geretourneerde array
leeg zijn.

##### !! total

!type u64

Totale voorraad in lamports

</APIMethod>
