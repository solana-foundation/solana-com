## Solana Kit

`@solana/kit` Solana üzerinde geliştirme yapmak için önerilen TypeScript
SDK'dır.

| Paket                          | Açıklama                             | GitHub                                                                          |
| ------------------------------ | ------------------------------------ | ------------------------------------------------------------------------------- |
| @solana/kit                    | Çekirdek SDK                         | [Kaynak](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | System Program ile etkileşim         | [Kaynak](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Token Program ile etkileşim          | [Kaynak](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Token-2022 Program ile etkileşim     | [Kaynak](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Memo Program ile etkileşim           | [Kaynak](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Compute Budget Program ile etkileşim | [Kaynak](https://github.com/solana-program/compute-budget/tree/main/clients/js) |

- [Resmi Kit Dokümantasyonu](https://www.solanakit.com/)
- [Örnek: Next.js ile cüzdan bağlama](/docs/frontend/kit)
