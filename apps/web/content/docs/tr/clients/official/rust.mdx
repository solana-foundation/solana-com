## Program Crate'leri

Zincir üzerindeki programları oluşturmak için `solana-program` kullanın.

- [Örnek: Bir Solana programı oluşturun](/docs/programs/rust)

| Crate          | Açıklama                          | Dokümanlar                                  | GitHub                                                               |
| -------------- | --------------------------------- | ------------------------------------------- | -------------------------------------------------------------------- |
| solana-program | Zincir üzerinde program oluşturun | [Görüntüle](https://docs.rs/solana-program) | [Kaynak](https://github.com/anza-xyz/solana-sdk/tree/master/program) |

Pinocchio, Rust'ta Solana programları oluşturmak için bağımlılık içermeyen bir
kütüphanedir ve zincir üzerindeki programları yazmak için `solana-program`
yerine kullanılabilir.

| Crate     | Açıklama                                    | Dokümanlar                             | GitHub                                          |
| --------- | ------------------------------------------- | -------------------------------------- | ----------------------------------------------- |
| pinocchio | Bağımlılık içermeyen zincir üstü programlar | [Görüntüle](https://docs.rs/pinocchio) | [Kaynak](https://github.com/anza-xyz/pinocchio) |

<Callout type="warn">
  Pinocchio hala geliştirme aşamasındadır ve yakın gelecekte önemli
  değişiklikler içerebilir.
</Callout>

Pinocchio, Solana programlarıyla etkileşim için program özelinde crate'ler
içerir:

| Crate                              | Açıklama                                     | Dokümanlar                                                       | GitHub                                                                                      |
| ---------------------------------- | -------------------------------------------- | ---------------------------------------------------------------- | ------------------------------------------------------------------------------------------- |

| pinocchio-token                    | Token Program ile etkileşim kurun            | [Görüntüle](https://docs.rs/pinocchio-token/)                    | [Kaynak](https://github.com/anza-xyz/pinocchio/tree/main/programs/token)                    |
| pinocchio-token-2022               | Token-2022 Program ile etkileşim kurun       | [Görüntüle](https://docs.rs/pinocchio-token-2022/)               | [Kaynak](https://github.com/anza-xyz/pinocchio/tree/main/programs/token-2022)               |
| pinocchio-associated-token-account | Associated Token Program ile etkileşim kurun | [Görüntüle](https://docs.rs/pinocchio-associated-token-account/) | [Kaynak](https://github.com/anza-xyz/pinocchio/tree/main/programs/associated-token-account) |
