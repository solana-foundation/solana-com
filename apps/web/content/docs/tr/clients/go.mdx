h1: Solana için Go SDK
---

Solana ile etkileşim kurmak için çoğu işlevi destekleyen bir
[Go SDK](https://github.com/gagliardetto/solana-go) bulunmaktadır. Solana
programlarınız için Anchor kullanıyorsanız, Golang uyumlu bir istemci oluşturmak
için idl'niz ile birlikte [anchor-go](https://github.com/gagliardetto/anchor-go)
kullanabilirsiniz.

## Kurulum

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```

## Daha Fazla Dokümantasyon

Bu Go SDK'yı Solana ile kullanma hakkında daha fazla bilgi için lütfen
[Go SDK dokümantasyonunu](https://github.com/gagliardetto/solana-go?tab=readme-ov-file#installation)
ziyaret edin.
