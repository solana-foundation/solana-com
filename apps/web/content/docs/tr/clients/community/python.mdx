## Solana Python SDK

Python ile geliştirme yaparken faydalı olan iki paket vardır:

[`@kevinheavy/solana.py`](https://github.com/michaelhly/solana-py)

- Solana.py, Solana ile etkileşim kurmak için temel Python kütüphanesidir.
- Bunu, [solana-web3.js](https://github.com/solana-labs/solana-web3.js/) ile
  yapacağınız gibi işlemler oluşturmak ve [Solana JSON RPC API](/docs/rpc) ile
  etkileşim kurmak için kullanabilirsiniz.

- Ayrıca [Token Program](/docs/rpc) programını da kapsar.

- [En güncel dokümantasyon](https://michaelhly.github.io/solana-py/).

[`@kevinheavey/solders`](https://github.com/kevinheavey/solders)

- `solders` Rust ile yazılmış, Solana için yüksek performanslı bir Python araç
  kitidir. Aşağıdaki alanlarda sağlam çözümler sunar:

- Temel SDK işlevselliği: anahtar çiftleri, pubkey'ler, işlemleri imzalama ve
  serileştirme.
- RPC: istekleri oluşturma ve yanıtları ayrıştırma (ağ ve asenkron API'ler için
  [solana-py](https://michaelhly.github.io/solana-py/rpc/async_api/) önerilen
  seçenektir).
- Entegrasyon testi: `solders.litesvm` modülü, `solana-test-validator` için çok
  daha kullanışlı ve **çok** daha hızlı bir alternatiftir.
  [solana-program-test](https://crates.io/crates/solana-program-test)
  tabanlıdır.

## Solana Client Python SDK

Python ile Solana sözleşmeleriyle etkileşim kurmak istiyorsanız, sözleşmenin IDL
dosyasını alabildiğiniz sürece Solana rust sözleşmelerini Python istemcilerine
dönüştürmek için istemci SDK'sını kullanabilirsiniz.

[`Codama-py`](https://www.npmjs.com/package/codama-py)

- Herhangi bir Solana zincir üstü sözleşme programını
  [`Codama IDL`](https://github.com/codama-idl/codama) ile standartlaştırın ve
  etkileşimli bir Python istemcisi oluşturun.
- Hem Anchor hem de özel programları destekler.

[`Anchor-py`](https://github.com/kevinheavey/anchorpy)

- Anchor-py, Anchor programları için bir Python istemcisidir. Anchor IDL
  dosyalarından istemciler oluşturabilir ve Anchor programlarıyla etkileşim
  kurabilir.
