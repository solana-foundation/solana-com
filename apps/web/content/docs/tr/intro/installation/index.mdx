Tek bir komutla yerel Solana geliştirme ortamınızı kurmak için hızlı kurulum.
Windows (WSL), Linux ve Mac üzerinde Rust, Solana CLI ve Anchor Framework'ü
kurun. (Tercih ederseniz,
[her bağımlılığı ayrı ayrı kurabilirsiniz](/docs/intro/installation/dependencies).)

## Ön koşullar

<Accordions>
<Accordion title="Windows">

Windows'ta Solana programları geliştirmek için **mutlaka**
[Windows Subsystem for Linux](https://learn.microsoft.com/en-us/windows/wsl/install)
(WSL) kullanmalısınız. WSL'yi kurduktan sonra, diğer tüm bağımlılıklar Linux
terminali üzerinden kurulacaktır.

WSL'yi kurmak için Windows PowerShell'de aşağıdaki komutu çalıştırın:

```terminal
$ wsl --install
```

Kurulum işlemi sırasında bir varsayılan kullanıcı hesabı oluşturmanız
istenecektir.

![WSL Kurulumu](/assets/docs/intro/installation/wsl-install.png)

Varsayılan olarak, WSL Ubuntu kurar. Arama çubuğunda "Ubuntu" yazarak Linux
terminalini açabilirsiniz.

![WSL Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-search.png)

Ubuntu terminaliniz aşağıdaki resme benziyorsa, terminalde `ctrl + v`
(yapıştırma klavye kısayolu) çalışmaması sorunu ile karşılaşabilirsiniz.

![Ubuntu Terminali](/assets/docs/intro/installation/wsl-ubuntu-terminal-1.png)

Bu sorunla karşılaşırsanız, Arama çubuğunda "Terminal" yazarak Windows
Terminal'i açın.

![Windows Terminal](/assets/docs/intro/installation/wsl-windows-terminal.png)

Ardından, Windows Terminal'i kapatın ve tekrar Ubuntu'yu arayarak Linux
terminalini yeniden açın. Terminal artık aşağıdaki resme benzemeli ve `ctrl + v`
(yapıştırma klavye kısayolu) çalışmalıdır.

![Ubuntu Terminali](/assets/docs/intro/installation/wsl-ubuntu-terminal-2.png)

VS Code kullanıyorsanız,
[WSL uzantısı](https://code.visualstudio.com/docs/remote/wsl-tutorial) WSL ve VS
Code'u birlikte kullanmanızı sağlar.

![VS Code'da WSL Kurulumu](/assets/docs/intro/installation/wsl-vscode.png)

VS Code durum çubuğunda aşağıdakini göreceksiniz:

![WSL: Ubuntu](/assets/docs/intro/installation/wsl-vscode-ubuntu.png)

WSL'yi kurduktan sonra, aşağıdakilerden birini yapmaya hazırsınız:

- Ubuntu (Linux) terminali kullanarak [hızlı kurulumu](#installation) tamamlayın
- Aşağıda açıklandığı gibi
  [Rust](/docs/intro/installation/dependencies#install-rust),
  [Solana CLI](/docs/intro/installation/dependencies#install-solana-cli) ve
  [Anchor CLI](/docs/intro/installation/dependencies#install-anchor-cli)'yi ayrı
  ayrı kurun

</Accordion>
<Accordion title="Linux">

<Tabs items={[".deb", ".rpm"]}>
<Tab value=".deb" >
Debian veya Ubuntu ya da Mint gibi Debian türevi sistemlerde, aşağıdaki adımları izleyin:

1. Tüm paketlerin güncel olduğundan emin olmak için paket listelerini
   güncelleyin:

```terminal
$ sudo apt-get update
```

2. Aşağıdaki bağımlılıkları yükleyin:

```terminal
$ sudo apt-get install -y \
    build-essential \
    pkg-config \
    libudev-dev llvm libclang-dev \
    protobuf-compiler libssl-dev
```

<Callout>
`protobuf-compiler` yüklerken aşağıdaki hatayla karşılaşırsanız,
önce _shell`sudo apt-get update` komutunu çalıştırdığınızdan emin olun

```
Package protobuf-compiler is not available, but is referred to by another package.
This may mean that the package is missing, has been obsoleted, or
is only available from another source
```

</Callout>

  </Tab>

    <Tab value=".rpm" >
    Fedora veya Red Hat Linux gibi RHEL türevleri için aşağıdaki adımları izleyin.

Paketlerin daha yeni sürümlerini almak için paket listelerini aşağıdaki komutu
çalıştırarak güncelleyin.

```terminal
$ sudo dnf upgrade --refresh
```

Ardından, aşağıdaki bağımlılıkları yükleyin:

```terminal
$ sudo dnf group install c-development development-tools \
&& sudo dnf install pkg-config openssl-devel
```

## Kurulum

1. Gerekli tüm bağımlılıkları yüklemek için aşağıdaki komutu terminalinize
   girin:

```terminal
$ curl --proto '=https' --tlsv1.2 -sSfL https://solana-install.solana.workers.dev | bash
```

Başarılı bir kurulum aşağıdakine benzer bir çıktı verecektir:

```
Installed Versions:
Rust: rustc 1.86.0 (05f9846f8 2025-03-31)
Solana CLI: solana-cli 2.2.12 (src:0315eb6a; feat:1522022101, client:Agave)
Anchor CLI: anchor-cli 0.31.1
Node.js: v23.11.0
Yarn: 1.22.1
```

2. Her yüklenen bağımlılığın sürümünü kontrol ederek başarılı bir kurulum
   olduğunu doğrulayın.

```terminal
$ rustc --version && solana --version && anchor --version && node --version && yarn --version
```

<Callout>
  Eğer hızlı kurulum komutu başarısız olursa, lütfen her bağımlılığı ayrı ayrı
  kurmak için [Bağımlılıkları Yükle](/docs/intro/installation/dependencies)
  bölümüne bakın.
</Callout>
