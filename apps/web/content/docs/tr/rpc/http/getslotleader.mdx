h1: getSlotLeader RPC Metodu
---

Mevcut slot liderini döndürür

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getSlotLeader",
  "params": [
    // !hover(1:3) 0
    {
      // !hover commitment
      "commitment": "finalized"
    }
  ]
}
```

```ts !!request title="Kit"
import { createSolanaRpc } from "@solana/kit";

const rpc_url = "https://api.devnet.solana.com";
const rpc = createSolanaRpc(rpc_url);

let slotLeader = await rpc.getSlotLeader().send();

console.log(slotLeader);
```

```ts !!request title="web3.js"
import { Connection, clusterApiUrl } from "@solana/web3.js";

const connection = new Connection(clusterApiUrl("devnet"), "confirmed");

let slotLeader = await connection.getSlotLeader();

console.log(slotLeader);
```

### !params

#### !! 0

!type object

Aşağıdaki alanları içeren yapılandırma nesnesi:

##### !! commitment

!type string

Commitment, bir bloğun o zaman noktasında ne kadar sonlandırıldığını tanımlar.
Bkz. [Durum Taahhüdünü Yapılandırma](/docs/rpc#configuring-state-commitment).

##### !! minContextSlot

!type number

İsteğin değerlendirilebileceği minimum slot
