Trả về thông tin về epoch hiện tại

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getEpochInfo",
  "params": [
    // !hover(1:3) 0
    {
      // !hover commitment
      "commitment": "finalized"
    }
  ]
}
```

```ts !!request title="Kit"
import { createSolanaRpc } from "@solana/kit";

const rpc_url = "https://api.devnet.solana.com";
const rpc = createSolanaRpc(rpc_url);

let epochInfo = await rpc.getEpochInfo().send();

console.log(epochInfo);
```

```ts !!request title="web3.js"
import { Connection, clusterApiUrl } from "@solana/web3.js";

const connection = new Connection(clusterApiUrl("devnet"), "confirmed");

let epochInfo = await connection.getEpochInfo();

console.log(epochInfo);
```

```rs !!request title="Rust"
use anyhow::Result;
use solana_client::nonblocking::rpc_client::RpcClient;
use solana_sdk::commitment_config::CommitmentConfig;

#[tokio::main]
async fn main() -> Result<()> {
    let client = RpcClient::new_with_commitment(
        String::from("https://api.devnet.solana.com"),
        CommitmentConfig::confirmed(),
    );

    let epoch_info = client.get_epoch_info().await?;

    println!("{:#?}", epoch_info);

    Ok(())
}
```

### !params

#### !! 0

!type object

Đối tượng cấu hình.

##### !! commitment

!type string

Commitment mô tả mức độ hoàn tất của một block tại thời điểm đó. Xem
[Cấu hình State Commitment](/docs/rpc#configuring-state-commitment).

##### !! minContextSlot

!type number

Slot tối thiểu mà yêu cầu có thể được đánh giá.

!type object

Trường kết quả sẽ là một đối tượng với các trường sau:

##### !! absoluteSlot

!type u64

Slot hiện tại

##### !! blockHeight

!type u64

Chiều cao khối hiện tại

##### !! epoch

!type u64

Epoch hiện tại

##### !! slotIndex

!type u64

Slot hiện tại tính từ đầu epoch hiện tại

##### !! slotsInEpoch

!type u64

Số lượng slot trong epoch này

##### !! transactionCount

!type u64 | null

Tổng số giao dịch được xử lý không có lỗi kể từ khi khởi tạo
