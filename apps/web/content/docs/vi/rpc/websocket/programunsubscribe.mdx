Hủy đăng ký nhận thông báo thay đổi tài khoản thuộc sở hữu của chương trình

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "programUnsubscribe",
  "params": [
    // !hover 0
    0
  ]
}
```

### !params

#### !! 0

!type number  
!required

id của Đăng ký tài khoản cần hủy

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": true,
  "id": 1
}
```

!type boolean

thông báo hủy đăng ký thành công

</APIMethod>
