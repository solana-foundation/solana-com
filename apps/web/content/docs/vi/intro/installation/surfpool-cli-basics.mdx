Surfpool là công cụ phát triển cục bộ giúp đơn giản hóa việc phát triển Solana
bằng cách tự động tải các chương trình và tài khoản từ một cụm (mainnet,
devnet). Sử dụng Surfpool CLI để khởi động validator cục bộ để kiểm tra các
chương trình và giao dịch của bạn.

Truy cập [tài liệu Surfpool](https://docs.surfpool.run/) để biết thêm thông tin.

## Khởi động validator cục bộ

Để khởi động validator cục bộ, chạy lệnh sau:

```terminal
$ surfpool start
```

Theo mặc định, Surfpool tự động tải bất kỳ chương trình và tài khoản nào mà các
giao dịch của bạn phụ thuộc vào từ mainnet đến validator cục bộ.

## Surfpool Studio

Surfpool Studio cung cấp giao diện web để tương tác với validator cục bộ của
bạn. Sử dụng nó để:

- Xem chi tiết giao dịch
- Kiểm tra dữ liệu tài khoản
- Airdrop SOL và các token khác bằng cách sử dụng faucet UI

Để truy cập Surfpool Studio:

1. Khởi động validator cục bộ của bạn với `surfpool start`
2. Mở trình duyệt và truy cập vào
   [http://127.0.0.1:18488](http://127.0.0.1:18488)
