## Solana Python SDK

Có hai gói thư viện hữu ích cho việc phát triển với Python:

[`@kevinheavy/solana.py`](https://github.com/michaelhly/solana-py)

- Solana.py là thư viện Python cơ bản để tương tác với Solana.
- Bạn có thể sử dụng nó để xây dựng các giao dịch và tương tác với
  [Solana JSON RPC API](/docs/rpc), tương tự như cách bạn sử dụng
  [solana-web3.js](https://github.com/solana-labs/solana-web3.js/)

- Nó cũng hỗ trợ [Token Program](/docs/rpc).

- [Tài liệu mới nhất](https://michaelhly.github.io/solana-py/).

[`@kevinheavey/solders`](https://github.com/kevinheavey/solders)

- `solders` là bộ công cụ Python hiệu suất cao cho Solana, được viết bằng Rust.
  Nó cung cấp các giải pháp mạnh mẽ trong các lĩnh vực sau:

- Chức năng SDK cốt lõi: keypairs, pubkeys, ký và tuần tự hóa giao dịch.
- RPC: xây dựng yêu cầu và phân tích phản hồi (đối với mạng và API bất đồng bộ,
  [solana-py](https://michaelhly.github.io/solana-py/rpc/async_api/) là lựa chọn
  được khuyến nghị).
- Kiểm thử tích hợp: mô-đun `solders.litesvm` là một giải pháp thay thế cho
  `solana-test-validator` thuận tiện hơn và **nhanh hơn nhiều**. Nó dựa trên
  [solana-program-test](https://crates.io/crates/solana-program-test).

## Solana Client Python SDK

Nếu bạn muốn tương tác với các hợp đồng Solana bằng Python, bạn có thể sử dụng
SDK client để chuyển đổi các hợp đồng Rust của Solana thành client Python, miễn
là bạn có tệp IDL của hợp đồng.

[`Codama-py`](https://www.npmjs.com/package/codama-py)

- Chuẩn hóa bất kỳ chương trình hợp đồng on-chain Solana nào thành
  [`Codama IDL`](https://github.com/codama-idl/codama) và tạo ra client Python
  tương tác.
- Hỗ trợ cả chương trình Anchor và chương trình tùy chỉnh.

[`Anchor-py`](https://github.com/kevinheavey/anchorpy)

- Anchor-py là client Python cho các chương trình Anchor. Nó có thể tạo ra
  client từ tệp IDL của Anchor và tương tác với các chương trình Anchor.
