## Solana Kit

`@solana/kit` là TypeScript SDK được khuyến nghị để xây dựng trên Solana.

| Gói                            | Mô tả                                | GitHub                                                                          |
| ------------------------------ | ------------------------------------ | ------------------------------------------------------------------------------- |
| @solana/kit                    | SDK cốt lõi                          | [Source](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | Tương tác với System Program         | [Source](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Tương tác với Token Program          | [Source](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Tương tác với Token-2022 Program     | [Source](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Tương tác với Memo Program           | [Source](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Tương tác với Compute Budget Program | [Source](https://github.com/solana-program/compute-budget/tree/main/clients/js) |

- [Tài liệu chính thức về Kit](https://www.solanakit.com/)
