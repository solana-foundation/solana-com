Có một [Python SDK](https://github.com/michaelhly/solana-py) hỗ trợ hầu hết các
chức năng để tương tác với Solana. Nếu bạn đang sử dụng Anchor cho các chương
trình Solana của mình, bạn sẽ có thể sử dụng
[anchor-py](https://github.com/kevinheavey/anchorpy) với idl của bạn để tạo ra
một client tương thích với Python.

## Cài đặt

```bash title="Terminal"
pip install solders
pip install solana
```

## Tài liệu thêm

Vui lòng truy cập
[Tài liệu Python SDK](https://github.com/michaelhly/solana-py?tab=readme-ov-file#general-usage)
để biết thêm thông tin về việc sử dụng Python SDK này với Solana.

- [Tài liệu mới nhất](https://michaelhly.github.io/solana-py/).

[`@kevinheavey/solders`](https://github.com/kevinheavey/solders)

- `solders` là bộ công cụ Python hiệu suất cao cho Solana, được viết bằng Rust.
  Nó cung cấp các giải pháp mạnh mẽ trong các lĩnh vực sau:

- Chức năng SDK cốt lõi: keypair, pubkey, ký và tuần tự hóa giao dịch.
- RPC: xây dựng yêu cầu và phân tích phản hồi (đối với mạng và API bất đồng bộ,
  [solana-py](https://michaelhly.github.io/solana-py/rpc/async_api/) là lựa chọn
  được khuyến nghị).
- Kiểm thử tích hợp: mô-đun `solders.litesvm` là một giải pháp thay thế cho
  `solana-test-validator` thuận tiện hơn nhiều và nhanh hơn **rất nhiều**. Nó
  dựa trên [solana-program-test](https://crates.io/crates/solana-program-test).

## SDK Python cho Solana Client

Nếu bạn muốn tương tác với các hợp đồng Solana bằng Python, bạn có thể sử dụng
SDK client để chuyển đổi các hợp đồng rust Solana thành client Python, miễn là
bạn có file IDL của hợp đồng.

[`Codama-py`](https://www.npmjs.com/package/codama-py)

- Chuẩn hóa bất kỳ chương trình hợp đồng on-chain Solana nào thành
  [`Codama IDL`](https://github.com/codama-idl/codama) và tạo ra client Python
  tương tác.
- Hỗ trợ cả chương trình Anchor và tùy chỉnh.

[`Anchor-py`](https://github.com/kevinheavey/anchorpy)

- Anchor-py là client Python cho các chương trình Anchor. Nó có thể tạo client
  từ file IDL Anchor và tương tác với các chương trình Anchor.
