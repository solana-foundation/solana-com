## Solana Kit

`@solana/kit` est le SDK TypeScript recommandé pour développer sur Solana.

| Package                        | Description                           | GitHub                                                                          |
| ------------------------------ | ------------------------------------- | ------------------------------------------------------------------------------- |
| @solana/kit                    | SDK principal                         | [Source](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | Interagir avec System Program         | [Source](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Interagir avec Token Program          | [Source](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Interagir avec Token-2022 Program     | [Source](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Interagir avec Memo Program           | [Source](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Interagir avec Compute Budget Program | [Source](https://github.com/solana-program/compute-budget/tree/main/clients/js) |

- [Documentation officielle de Kit](https://www.solanakit.com/)
- [Exemple : Connecter un portefeuille avec Next.js](/docs/frontend/kit)
