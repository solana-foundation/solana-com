h1: Méthode RPC getRecentBlockhash
---

Renvoie un hachage de bloc récent du registre, ainsi qu'un barème de frais qui
peut être utilisé pour calculer le coût de soumission d'une transaction en
l'utilisant.

<Callout type="warn" title={"Méthode dépréciée"}>
  Cette méthode devrait être supprimée dans `solana-core` v2.0. Veuillez
  utiliser [getLatestBlockhash](/docs/rpc/http/getlatestblockhash) à la place.
</Callout>
