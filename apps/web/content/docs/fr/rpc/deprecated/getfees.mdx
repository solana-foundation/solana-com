h1: Méthode RPC getFees
---

Renvoie un hash de bloc récent du registre, un barème de frais qui peut être
utilisé pour calculer le coût de soumission d'une transaction l'utilisant, et le
dernier slot dans lequel le hash de bloc sera valide.

<Callout type="warn" title={"Méthode dépréciée"}>
  Cette méthode devrait être supprimée dans `solana-core` v2.0. Veuillez
  utiliser [getFeeForMessage](/docs/rpc/http/getfeeformessage) à la place.
</Callout>
