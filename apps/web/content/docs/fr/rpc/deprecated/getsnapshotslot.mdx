h1: Méthode RPC getSnapshotSlot
---

Renvoie le slot le plus élevé pour lequel le nœud possède un instantané

<Callout type="warn" title={"Méthode dépréciée"}>
  Cette méthode devrait être supprimée dans `solana-core` v2.0. Veuillez
  utiliser [getHighestSnapshotSlot](/docs/rpc/http/gethighestsnapshotslot) à la
  place.
</Callout>
