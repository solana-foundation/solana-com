h1: Méthode RPC getConfirmedBlock
---

Renvoie les informations d'identité et de transaction concernant un bloc
confirmé dans le registre

<Callout type="warn" title={"Méthode dépréciée"}>
  Cette méthode devrait être supprimée dans `solana-core` v2.0. Veuillez
  utiliser [getBlock](/docs/rpc/http/getblock) à la place.
</Callout>
