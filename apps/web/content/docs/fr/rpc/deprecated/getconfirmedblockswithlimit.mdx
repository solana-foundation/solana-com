h1: Méthode RPC getConfirmedBlocksWithLimit
---

Renvoie une liste de blocs confirmés à partir du slot spécifié

<Callout type="warn" title={"Méthode dépréciée"}>
  Cette méthode devrait être supprimée dans `solana-core` v2.0. Veuillez
  utiliser [getBlocksWithLimit](/docs/rpc/http/getblockswithlimit) à la place.
</Callout>
