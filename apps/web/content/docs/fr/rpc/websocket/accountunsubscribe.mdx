h1: Méthode RPC accountUnsubscribe
---

Se désabonner des notifications de changement de compte

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "accountUnsubscribe",
  "params": [
    // !hover 0
    0
  ]
}
```

### !params

#### !! 0

!type number  
!required

identifiant de l'abonnement au compte à annuler

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": true,
  "id": 1
}
```

!type boolean

message de confirmation du désabonnement

</APIMethod>
