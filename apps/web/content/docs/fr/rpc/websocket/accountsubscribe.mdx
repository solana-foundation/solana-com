h1: Méthode RPC accountSubscribe
---

Abonnez-vous à un compte pour recevoir des notifications lorsque les lamports ou
les données d'une clé publique de compte donnée changent

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "accountSubscribe",
  "params": [
    // !hover 0
    "CM78CPUeXjn8o3yroDHxUtKsZZgoy4GPkPPXfouKNH12",
    // !hover(1:4) 1
    {
      // !hover encoding
      "encoding": "jsonParsed",
      // !hover commitment
      "commitment": "finalized"
    }
  ]
}
```

### !params

#### !! 0

!type string  
!required

Pubkey du compte, sous forme de chaîne encodée en base-58

#### !! 1

!type object

Objet de configuration contenant les champs suivants :

##### !! commitment

!type string

Le commitment décrit le niveau de finalisation d'un bloc à un moment donné. Voir
[Configuration de l'engagement d'état](/docs/rpc#configuring-state-commitment).

##### !! encoding

!type string  
!values base58 base64 base64+zstd jsonParsed

Format d'encodage pour les données du compte

- `base58` est lent.
- L'encodage `jsonParsed` tente d'utiliser des analyseurs d'état spécifiques au
  programme pour renvoyer des données d'état de compte plus lisibles et
  explicites
- Si `jsonParsed` est demandé mais qu'aucun analyseur n'est trouvé, le champ
  revient à l'encodage binaire, détectable lorsque le champ `data` est de type
  `string`.

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 23784,
  "id": 1
}
```

!type number

Identifiant d'abonnement (nécessaire pour se désabonner)

</APIMethod>

#### Format de notification :

Le format de notification est le même que celui vu dans la méthode HTTP RPC
[getAccountInfo](/docs/rpc/http/getaccountinfo).

Encodage Base58 :

```json
{
  "jsonrpc": "2.0",
  "method": "accountNotification",
  "params": {
    "result": {
      "context": {
        "slot": 5199307
      },
      "value": {
        "data": [
          "11116bv5nS2h3y12kD1yUKeMZvGcKLSjQgX6BeV7u1FrjeJcKfsHPXHRDEHrBesJhZyqnnq9qJeUuF7WHxiuLuL5twc38w2TXNLxnDbjmuR",
          "base58"
        ],
        "executable": false,
        "lamports": 33594,
        "owner": "11111111111111111111111111111111",
        "rentEpoch": 635,
        "space": 80
      }
    },
    "subscription": 23784
  }
}
```
