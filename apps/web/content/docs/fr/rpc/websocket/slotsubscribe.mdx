h1: Méthode RPC slotSubscribe
---

S'abonner pour recevoir une notification chaque fois qu'un slot est traité par
le validator

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "slotSubscribe"
}
```

### !params

**Aucun**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

Identifiant d'abonnement (nécessaire pour se désabonner)

</APIMethod>

#### Format de notification :

La notification sera un objet avec les champs suivants :

- `parent: <u64>` - Le slot parent
- `root: <u64>` - Le slot racine actuel
- `slot: <u64>` - La nouvelle valeur de slot définie

Exemple :

```json
{
  "jsonrpc": "2.0",
  "method": "slotNotification",
  "params": {
    "result": {
      "parent": 75,
      "root": 44,
      "slot": 76
    },
    "subscription": 0
  }
}
```
