h1: Méthode RPC rootSubscribe
---

Abonnez-vous pour recevoir une notification chaque fois qu'une nouvelle racine
est définie par le validator.

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**Aucun**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

identifiant d'abonnement (nécessaire pour se désabonner)

</APIMethod>

#### Format de notification :

Le résultat est le numéro du dernier slot racine.

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
