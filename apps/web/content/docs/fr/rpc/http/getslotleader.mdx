h1: Méthode RPC getSlotLeader
---

Renvoie le leader du slot actuel

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getSlotLeader",
  "params": [
    // !hover(1:3) 0
    {
      // !hover commitment
      "commitment": "finalized"
    }
  ]
}
```

```ts !!request title="Kit"
import { createSolanaRpc } from "@solana/kit";

const rpc_url = "https://api.devnet.solana.com";
const rpc = createSolanaRpc(rpc_url);

let slotLeader = await rpc.getSlotLeader().send();

console.log(slotLeader);
```

```ts !!request title="web3.js"
import { Connection, clusterApiUrl } from "@solana/web3.js";

const connection = new Connection(clusterApiUrl("devnet"), "confirmed");

let slotLeader = await connection.getSlotLeader();

console.log(slotLeader);
```

### !params

#### !! 0

!type object

Objet de configuration contenant les champs suivants :

##### !! commitment

!type string

Le commitment décrit le niveau de finalisation d'un bloc à un moment donné. Voir
[Configuration de l'engagement d'état](/docs/rpc#configuring-state-commitment).

##### !! minContextSlot

!type number

Le slot minimum auquel la requête peut être évaluée
