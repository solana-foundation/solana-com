Biblioteki Rust od Solany są
[opublikowane na crates.io](https://crates.io/search?q=solana-) i można je
znaleźć [na docs.rs](https://docs.rs/releases/search?query=solana-) z prefiksem
`solana-`.

## Biblioteki Rust

Poniżej znajdują się najważniejsze i najczęściej używane biblioteki Rust do
tworzenia aplikacji na Solanie:

- [`solana-program`] &mdash; Importowane przez programy działające na Solanie,
  kompilowane do SBF. Ta biblioteka zawiera wiele podstawowych typów danych i
  jest re-eksportowana z [`solana-sdk`], której nie można importować z programu
  Solany.

- [`solana-sdk`] &mdash; Podstawowe SDK offchain, re-eksportuje
  [`solana-program`] i dodaje więcej API na tej podstawie. Większość programów
  Solany, które nie działają on-chain, importuje tę bibliotekę.

- [`solana-client`] &mdash; Do interakcji z węzłem Solany za pomocą
  [JSON RPC API](/docs/rpc).

- [`solana-cli-config`] &mdash; Ładowanie i zapisywanie pliku konfiguracyjnego
  CLI Solany.

- [`solana-clap-utils`] &mdash; Funkcje do konfiguracji CLI, używając [`clap`],
  jak w głównym CLI Solany. Zawiera funkcje do ładowania wszystkich typów
  podpisujących obsługiwanych przez CLI.

[`solana-program`]: https://docs.rs/solana-program
[`solana-sdk`]: https://docs.rs/solana-sdk
[`solana-client`]: https://docs.rs/solana-client

## Instalacja

Do wysyłania transakcji i interakcji z Solaną:

```shell title="Terminal"
cargo add solana-sdk solana-client
```

Do budowania programów Solany, zainstaluj:

```shell title="Terminal"
cargo add solana-program
```
