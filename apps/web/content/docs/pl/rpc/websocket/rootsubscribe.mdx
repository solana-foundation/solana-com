h1: Metoda RPC rootSubscribe
---

Subskrybuj, aby otrzymywać powiadomienia za każdym razem, gdy nowy root zostanie
ustawiony przez validator.

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**Brak**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

id subskrypcji (potrzebny do anulowania subskrypcji)

</APIMethod>

#### Format powiadomienia:

Wynik to numer najnowszego slotu root.

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
