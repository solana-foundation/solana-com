h1: Metoda RPC programUnsubscribe
---

Anuluj subskrypcję powiadomień o zmianach w kontach należących do programu

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "programUnsubscribe",
  "params": [
    // !hover 0
    0
  ]
}
```

### !params

#### !! 0

!type number  
!required

id subskrypcji konta do anulowania

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": true,
  "id": 1
}
```

!type boolean

wiadomość o pomyślnym anulowaniu subskrypcji

</APIMethod>
