Solana의 Rust 크레이트는
[crates.io에 게시되어 있으며](https://crates.io/search?q=solana-)
[docs.rs에서](https://docs.rs/releases/search?query=solana-) `solana-` 접두사로
찾을 수 있습니다.

## Rust 크레이트

다음은 Solana 개발에 가장 중요하고 일반적으로 사용되는 Rust 크레이트입니다:

- [`solana-program`] &mdash; Solana에서 실행되는 프로그램에서 가져오며, SBF로
  컴파일됩니다. 이 크레이트는 많은 기본 데이터 타입을 포함하고 있으며
  [`solana-sdk`]에서 재내보내기됩니다. Solana 프로그램에서는 직접 가져올 수
  없습니다.

- [`solana-sdk`] &mdash; 기본 오프체인 SDK로, [`solana-program`]를
  재내보내기하고 그 위에 더 많은 API를 추가합니다. 온체인에서 실행되지 않는
  대부분의 Solana 프로그램은 이것을 가져옵니다.

- [`solana-client`] &mdash; [JSON RPC API](/docs/rpc)를 통해 Solana 노드와
  상호작용하기 위한 크레이트입니다.

- [`solana-cli-config`] &mdash; Solana CLI 구성 파일을 로드하고 저장하는
  크레이트입니다.

- [`solana-clap-utils`] &mdash; 주요 Solana CLI에서 사용하는 것처럼 [`clap`]를
  사용하여 CLI를 설정하기 위한 루틴입니다. CLI에서 지원하는 모든 유형의 서명자를
  로드하는 기능을 포함합니다.

[`solana-program`]: https://docs.rs/solana-program
[`solana-sdk`]: https://docs.rs/solana-sdk
[`solana-client`]: https://docs.rs/solana-client
[`solana-cli-config`]: https://docs.rs/solana-cli-config
[`solana-clap-utils`]: https://docs.rs/solana-clap-utils

## 설치

트랜잭션 전송 및 Solana와 상호작용을 위해:

```shell title="Terminal"
cargo add solana-sdk solana-client
```

Solana 프로그램 빌드를 위한 설치:

```shell title="Terminal"
cargo add solana-program
```
