## Solana Kit

`@solana/kit`은 Solana에서 개발하기 위한 권장 TypeScript SDK입니다.

| 패키지                         | 설명                              | GitHub                                                                        |
| ------------------------------ | --------------------------------- | ----------------------------------------------------------------------------- |
| @solana/kit                    | 핵심 SDK                          | [소스](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | System Program과 상호작용         | [소스](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Token Program과 상호작용          | [소스](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Token-2022 Program과 상호작용     | [소스](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Memo Program과 상호작용           | [소스](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Compute Budget Program과 상호작용 | [소스](https://github.com/solana-program/compute-budget/tree/main/clients/js) |

- [공식 Kit 문서](https://www.solanakit.com/)
- [예제: Next.js로 지갑 연결하기](/docs/frontend/kit)
