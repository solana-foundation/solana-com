h1: Solana용 Go SDK
---

Solana와 상호작용하기 위한 대부분의 기능을 지원하는
[Go SDK](https://github.com/gagliardetto/solana-go)가 있습니다. Solana
프로그램에 Anchor를 사용하고 있다면,
[anchor-go](https://github.com/gagliardetto/anchor-go)를 idl과 함께 사용하여
Golang 호환 클라이언트를 생성할 수 있습니다.

## 설치

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```
