Surfpool은 클러스터(메인넷, 데브넷)에서 프로그램과 계정을 자동으로 로드하여
Solana 개발을 단순화하는 로컬 개발 도구입니다. Surfpool CLI를 사용하여 프로그램
및 트랜잭션 테스트를 위한 로컬 validator를 시작하세요.

더 많은 정보는 [Surfpool 문서](https://docs.surfpool.run/)를 참조하세요.

## 로컬 validator 시작하기

로컬 validator를 시작하려면 다음 명령어를 실행하세요:

```terminal
$ surfpool start
```

기본적으로 Surfpool은 트랜잭션이 의존하는 모든 프로그램과 계정을 메인넷에서 로컬
validator로 자동 로드합니다.

## Surfpool Studio

Surfpool Studio는 로컬 validator와 상호작용할 수 있는 웹 기반 인터페이스를
제공합니다. 다음과 같은 기능을 사용할 수 있습니다:

- 트랜잭션 세부 정보 보기
- 계정 데이터 검사
- UI 수도꼭지를 사용하여 SOL 및 기타 토큰 에어드롭

Surfpool Studio에 접근하는 방법:

1. `surfpool start`로 로컬 validator를 시작하세요
2. 브라우저를 열고 [http://127.0.0.1:18488](http://127.0.0.1:18488)로 이동하세요
