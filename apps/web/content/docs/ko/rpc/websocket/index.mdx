RPC PubSub 웹소켓에 `ws://<ADDRESS>/`로 연결한 후:

- 아래 메소드를 사용하여 웹소켓에 구독 요청을 제출하세요
- 여러 구독이 동시에 활성화될 수 있습니다
- 많은 구독은 선택적으로
  [`commitment` 매개변수](/docs/rpc/#configuring-state-commitment)를 사용합니다.
  이는 알림을 트리거하기 위해 변경 사항이 얼마나 확정되어야 하는지 정의합니다.
  구독의 경우, commitment가 지정되지 않으면 기본값은 `finalized`입니다.

## RPC PubSub 웹소켓 엔드포인트

기본 포트: `8900`

- ws://localhost:8900
- http://192.168.1.88:8900
