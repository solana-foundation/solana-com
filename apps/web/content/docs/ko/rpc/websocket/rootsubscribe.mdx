h1: rootSubscribe RPC 메소드
---

Validator가 새로운 루트를 설정할 때마다 알림을 받기 위해 구독합니다.

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**없음**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

구독 ID (구독 취소 시 필요)

</APIMethod>

#### 알림 형식:

결과는 최신 루트 slot 번호입니다.

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
