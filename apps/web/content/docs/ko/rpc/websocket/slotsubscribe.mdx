h1: slotSubscribe RPC 메소드
---

validator가 처리할 때마다 slot 알림을 받기 위한 구독

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "slotSubscribe"
}
```

### !params

**없음**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

구독 ID (구독 취소 시 필요)

</APIMethod>

#### 알림 형식:

알림은 다음 필드를 포함하는 객체입니다:

- `parent: <u64>` - 부모 slot
- `root: <u64>` - 현재 루트 slot
- `slot: <u64>` - 새로 설정된 slot 값

예시:

```json
{
  "jsonrpc": "2.0",
  "method": "slotNotification",
  "params": {
    "result": {
      "parent": 75,
      "root": 44,
      "slot": 76
    },
    "subscription": 0
  }
}
```
