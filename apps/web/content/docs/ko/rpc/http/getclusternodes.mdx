클러스터에 참여하는 모든 노드에 대한 정보를 반환합니다

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getClusterNodes"
}
```

```ts !!request title="Kit"
import { createSolanaRpc } from "@solana/kit";

const rpc_url = "https://api.devnet.solana.com";
const rpc = createSolanaRpc(rpc_url);

let nodes = await rpc.getClusterNodes().send();

console.log(nodes);
```

```ts !!request title="web3.js"
import { Connection, clusterApiUrl } from "@solana/web3.js";

const connection = new Connection(clusterApiUrl("devnet"), "confirmed");

let nodes = await connection.getClusterNodes();

console.log(nodes);
```

```rs !!request title="Rust"
use anyhow::Result;
use solana_client::nonblocking::rpc_client::RpcClient;
use solana_sdk::commitment_config::CommitmentConfig;

#[tokio::main]
async fn main() -> Result<()> {
    let client = RpcClient::new_with_commitment(
        String::from("https://api.devnet.solana.com"),
        CommitmentConfig::confirmed(),
    );

    let block_time = client.get_cluster_nodes().await?;

    println!("{:#?}", block_time);

    Ok(())
}
```

!type array

result 필드는 JSON 객체의 배열이며, 각 객체는 다음과 같은 하위 필드를
포함합니다:

##### !! featureSet

!type u32 | null

노드의 기능 세트에 대한 고유 식별자

##### !! gossip

!type string | null

노드의 가십 네트워크 주소

##### !! pubkey

!type string

base-58로 인코딩된 문자열 형태의 노드 공개 키

##### !! rpc

!type string | null

노드의 JSON RPC 네트워크 주소, 또는 JSON RPC 서비스가 활성화되지 않은 경우
`null`

##### !! shredVersion

!type u16 | null

노드가 사용하도록 구성된 shred 버전

##### !! tpu

!type string | null

노드의 TPU 네트워크 주소

##### !! version

!type string | null

노드의 소프트웨어 버전, 또는 버전 정보를 사용할 수 없는 경우 `null`

</APIMethod>
