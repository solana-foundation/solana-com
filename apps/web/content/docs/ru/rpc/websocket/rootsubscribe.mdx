h1: Метод RPC rootSubscribe
---

Подпишитесь, чтобы получать уведомления каждый раз, когда валидатор
устанавливает новый корневой слот.

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**Нет**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

идентификатор подписки (необходим для отмены подписки)

</APIMethod>

#### Формат уведомления:

Результатом является номер последнего корневого слота.

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
