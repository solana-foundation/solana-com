h1: Метод RPC slotSubscribe
---

Подпишитесь, чтобы получать уведомления каждый раз, когда слот обрабатывается
валидатором

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "slotSubscribe"
}
```

### !params

**Нет**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

ID подписки (необходим для отмены подписки)

</APIMethod>

#### Формат уведомления:

Уведомление будет объектом со следующими полями:

- `parent: <u64>` - Родительский слот
- `root: <u64>` - Текущий корневой слот
- `slot: <u64>` - Новое установленное значение слота

Пример:

```json
{
  "jsonrpc": "2.0",
  "method": "slotNotification",
  "params": {
    "result": {
      "parent": 75,
      "root": 44,
      "slot": 76
    },
    "subscription": 0
  }
}
```
