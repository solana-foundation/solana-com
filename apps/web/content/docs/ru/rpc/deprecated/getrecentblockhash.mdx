h1: Метод RPC getRecentBlockhash
---

Возвращает недавний хэш блока из реестра и таблицу комиссий, которую можно
использовать для расчёта стоимости отправки транзакции с его использованием.

<Callout type="warn" title={"Устаревший метод"}>
  Ожидается, что этот метод будет удалён в версии `solana-core` v2.0.
  Пожалуйста, используйте
  [getLatestBlockhash](/docs/rpc/http/getlatestblockhash) вместо него.
</Callout>
