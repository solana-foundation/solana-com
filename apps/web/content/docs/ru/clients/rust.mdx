Rust-библиотеки Solana
[опубликованы на crates.io](https://crates.io/search?q=solana-) и доступны
[на docs.rs](https://docs.rs/releases/search?query=solana-) с префиксом
`solana-`.

## Rust-библиотеки

Ниже приведены самые важные и часто используемые Rust-библиотеки для разработки
на Solana:

- [`solana-program`] &mdash; Импортируется программами, работающими на Solana, и
  компилируется в SBF. Эта библиотека содержит множество основных типов данных и
  реэкспортируется из [`solana-sdk`], которая не может быть импортирована из
  программы Solana.

- [`solana-sdk`] &mdash; Основной SDK для работы вне блокчейна, реэкспортирует
  [`solana-program`] и добавляет дополнительные API. Большинство программ
  Solana, которые не работают на блокчейне, импортируют эту библиотеку.

- [`solana-client`] &mdash; Для взаимодействия с узлом Solana через
  [JSON RPC API](/docs/rpc).

- [`solana-cli-config`] &mdash; Для загрузки и сохранения конфигурационного
  файла Solana CLI.

- [`solana-clap-utils`] &mdash; Процедуры для настройки CLI с использованием
  [`clap`], как в основном CLI Solana. Включает функции для загрузки всех типов
  подписантов, поддерживаемых CLI.

[`solana-program`]: https://docs.rs/solana-program
[`solana-sdk`]: https://docs.rs/solana-sdk

## Установка

Для отправки транзакций и взаимодействия с Solana:

```shell title="Terminal"
cargo add solana-sdk solana-client
```

Для создания программ Solana, установите:

```shell title="Terminal"
cargo add solana-program
```
