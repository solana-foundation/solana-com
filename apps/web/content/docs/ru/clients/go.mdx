h1: Go SDK для Solana
---

Существует [Go SDK](https://github.com/gagliardetto/solana-go), который
поддерживает большинство функций для взаимодействия с Solana. Если вы
используете Anchor для своих программ на Solana, вы сможете использовать
[anchor-go](https://github.com/gagliardetto/anchor-go) вместе с вашим idl для
создания клиента, совместимого с Golang.

## Установка

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```
