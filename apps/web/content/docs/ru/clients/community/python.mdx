## Solana Python SDK

Существует два пакета, которые полезны для разработки на Python:

[`@kevinheavy/solana.py`](https://github.com/michaelhly/solana-py)

- Solana.py — это основная библиотека Python для взаимодействия с Solana.
- Вы можете использовать её для создания транзакций и взаимодействия с
  [Solana JSON RPC API](/docs/rpc), так же, как вы бы делали это с
  [solana-web3.js](https://github.com/solana-labs/solana-web3.js/).

- Она также охватывает [SPL Token Program](https://spl.solana.com/token).

- [Последняя документация](https://michaelhly.github.io/solana-py/).

[`@kevinheavey/solders`](https://github.com/kevinheavey/solders)

- `solders` — это высокопроизводительный Python-инструментарий для Solana,
  написанный на Rust. Он предоставляет надёжные решения в следующих областях:

- Основной функционал SDK: ключевые пары, публичные ключи, подпись и
  сериализация транзакций.
- RPC: создание запросов и разбор ответов (для сетевых и асинхронных API
  [solana-py](https://michaelhly.github.io/solana-py/rpc/async_api/) является
  рекомендуемым вариантом).
- Интеграционное тестирование: модуль `solders.litesvm` является альтернативой
  `solana-test-validator`, которая гораздо удобнее и **значительно** быстрее. Он
  основан на
  [solana-program-test](https://crates.io/crates/solana-program-test).

## Solana Client Python SDK

Если вы хотите взаимодействовать с контрактами Solana на Python, вы можете
использовать клиентские SDK для преобразования контрактов Solana на Rust в
клиентские библиотеки Python, если у вас есть IDL-файл контракта.

[`Codama-py`](https://www.npmjs.com/package/codama-py)

- Стандартизируйте любую программу контракта Solana на блокчейне в
  [`Codama IDL`](https://github.com/codama-idl/codama) и создайте интерактивный
  клиент на Python.
- Поддерживает как Anchor, так и пользовательские программы.

[`Anchor-py`](https://github.com/kevinheavey/anchorpy)

- Anchor-py — это клиент Python для программ Anchor. Он может генерировать
  клиенты из IDL-файлов Anchor и взаимодействовать с программами Anchor.
