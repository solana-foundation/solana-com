## Solana Kit

`@solana/kit` — это рекомендуемый TypeScript SDK для разработки на Solana.

| Пакет                          | Описание                                | GitHub                                                                                |
| ------------------------------ | --------------------------------------- | ------------------------------------------------------------------------------------- |
| @solana/kit                    | Основной SDK                            | [Исходный код](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | Взаимодействие с System Program         | [Исходный код](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Взаимодействие с Token Program          | [Исходный код](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Взаимодействие с Token-2022 Program     | [Исходный код](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Взаимодействие с Memo Program           | [Исходный код](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Взаимодействие с Compute Budget Program | [Исходный код](https://github.com/solana-program/compute-budget/tree/main/clients/js) |

- [Официальная документация Kit](https://www.solanakit.com/)
- [Пример: Подключение кошелька с использованием Next.js](/docs/frontend/kit)
