## Модули программ

Используйте `solana-program` для создания программ на блокчейне.

- [Пример: Создание программы Solana](/docs/programs/rust)

| Модуль         | Описание                       | Документация                                  | GitHub                                                                     |
| -------------- | ------------------------------ | --------------------------------------------- | -------------------------------------------------------------------------- |
| solana-program | Создание программ на блокчейне | [Просмотреть](https://docs.rs/solana-program) | [Исходный код](https://github.com/anza-xyz/solana-sdk/tree/master/program) |

Pinocchio — это библиотека без зависимостей для создания программ Solana на
Rust, которая может использоваться как замена `solana-program` для написания
программ на блокчейне.

| Модуль    | Описание                                | Документация                             | GitHub                                                |
| --------- | --------------------------------------- | ---------------------------------------- | ----------------------------------------------------- |
| pinocchio | Программы на блокчейне без зависимостей | [Просмотреть](https://docs.rs/pinocchio) | [Исходный код](https://github.com/anza-xyz/pinocchio) |

<Callout type="warn">
  Pinocchio все еще находится в разработке, и в ближайшем будущем возможны
  изменения, нарушающие совместимость.
</Callout>

Pinocchio включает модули, специфичные для программ, для взаимодействия с
программами Solana:

| Модуль                             | Описание                                  | Документация                                                       | GitHub                                                                                            |
| ---------------------------------- | ----------------------------------------- | ------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------- |
| pinocchio-system                   | Взаимодействие с System Program           | [Просмотреть](https://docs.rs/pinocchio-system/)                   | [Исходный код](https://github.com/anza-xyz/pinocchio/tree/main/programs/system)                   |
| pinocchio-token                    | Взаимодействие с Token Program            | [Просмотреть](https://docs.rs/pinocchio-token/)                    | [Исходный код](https://github.com/anza-xyz/pinocchio/tree/main/programs/token)                    |
| pinocchio-token-2022               | Взаимодействие с Token-2022 Program       | [Просмотреть](https://docs.rs/pinocchio-token-2022/)               | [Исходный код](https://github.com/anza-xyz/pinocchio/tree/main/programs/token-2022)               |
| pinocchio-associated-token-account | Взаимодействие с Associated Token Program | [Просмотреть](https://docs.rs/pinocchio-associated-token-account/) | [Исходный код](https://github.com/anza-xyz/pinocchio/tree/main/programs/associated-token-account) |
| pinocchio-memo                     | Взаимодействие с Memo Program             | [Просмотреть](https://docs.rs/pinocchio-memo/)                     | [Исходный код](https://github.com/anza-xyz/pinocchio/tree/main/programs/memo)                     |
