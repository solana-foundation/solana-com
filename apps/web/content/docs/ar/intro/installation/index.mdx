تثبيت سريع لإعداد بيئة تطوير سولانا المحلية الخاصة بك بأمر واحد. قم بتثبيت Rust
وواجهة سطر أوامر سولانا وإطار عمل Anchor على نظام ويندوز (WSL) ولينكس وماك. (إذا
كنت تفضل ذلك، يمكنك
[تثبيت كل تبعية بشكل منفصل](/docs/intro/installation/dependencies).)

## المتطلبات الأساسية

<Accordions>
<Accordion title="ويندوز">

لتطوير برامج سولانا على نظام ويندوز **يجب** عليك استخدام
[نظام ويندوز الفرعي للينكس](https://learn.microsoft.com/en-us/windows/wsl/install)
(WSL). بعد تثبيت WSL، سيتم تثبيت جميع التبعيات الأخرى من خلال طرفية لينكس.

لتثبيت WSL، قم بتشغيل الأمر التالي في Windows PowerShell:

```terminal
$ wsl --install
```

ستطلب منك عملية التثبيت إنشاء حساب مستخدم افتراضي.

![تثبيت WSL](/assets/docs/intro/installation/wsl-install.png)

بشكل افتراضي، يقوم WSL بتثبيت أوبونتو. يمكنك فتح طرفية لينكس عن طريق البحث عن
"Ubuntu" في شريط البحث.

![WSL أوبونتو](/assets/docs/intro/installation/wsl-ubuntu-search.png)

إذا كانت طرفية أوبونتو لديك تشبه الصورة أدناه، فقد تواجه مشكلة حيث لا يعمل
اختصار لوحة المفاتيح `ctrl + v` (لصق) في الطرفية.

![طرفية أوبونتو](/assets/docs/intro/installation/wsl-ubuntu-terminal-1.png)

إذا واجهت هذه المشكلة، افتح Windows Terminal عن طريق البحث عن "Terminal" في شريط
البحث.

![طرفية ويندوز](/assets/docs/intro/installation/wsl-windows-terminal.png)

بعد ذلك، أغلق Windows Terminal وأعد فتح طرفية لينكس عن طريق البحث عن أوبونتو مرة
أخرى. يجب أن تبدو الطرفية الآن كما في الصورة أدناه، حيث يعمل اختصار لوحة
المفاتيح `ctrl + v` (لصق).

![طرفية أوبونتو](/assets/docs/intro/installation/wsl-ubuntu-terminal-2.png)

إذا كنت تستخدم VS Code، فإن
[امتداد WSL](https://code.visualstudio.com/docs/remote/wsl-tutorial) يتيح لك
استخدام WSL و VS Code معًا.

![إعداد WSL في VS Code](/assets/docs/intro/installation/wsl-vscode.png)

سترى ما يلي في شريط حالة VS Code:

![WSL: Ubuntu](/assets/docs/intro/installation/wsl-vscode-ubuntu.png)

بمجرد إعداد WSL، ستكون جاهزًا إما لـ:

- إكمال [التثبيت السريع](#installation) باستخدام طرفية أوبونتو (لينكس)
- تثبيت [Rust](/docs/intro/installation/dependencies#install-rust)، و
  [واجهة سطر أوامر سولانا](/docs/intro/installation/dependencies#install-solana-cli)،
  و
  [واجهة سطر أوامر Anchor](/docs/intro/installation/dependencies#install-anchor-cli)
  بشكل فردي، كما هو موضح أدناه

إذا كنت تستخدم ديبيان أو البدائل المشتقة من ديبيان مثل أوبونتو أو مينت، اتبع الخطوات التالية:

1. قم بتحديث قوائم الحزم للتأكد من أن جميع الحزم محدثة:

```terminal
$ sudo apt-get update
```

2. قم بتثبيت التبعيات التالية:

```terminal
$ sudo apt-get install -y \
    build-essential \
    pkg-config \
    libudev-dev llvm libclang-dev \
    protobuf-compiler libssl-dev
```

```
This may mean that the package is missing, has been obsoleted, or
is only available from another source
```

## التثبيت

1. أدخل الأمر التالي في الطرفية الخاصة بك لتثبيت جميع التبعيات الضرورية:

```terminal
$ curl --proto '=https' --tlsv1.2 -sSfL https://solana-install.solana.workers.dev | bash
```

سيُظهر التثبيت الناجح مخرجات مثل ما يلي:

```
Installed Versions:
Rust: rustc 1.86.0 (05f9846f8 2025-03-31)
Solana CLI: solana-cli 2.2.12 (src:0315eb6a; feat:1522022101, client:Agave)
Anchor CLI: anchor-cli 0.31.1
Node.js: v23.11.0
Yarn: 1.22.1
```

2. تحقق من نجاح التثبيت عن طريق فحص إصدار كل تبعية مثبتة.

```terminal
$ rustc --version && solana --version && anchor --version && node --version && yarn --version
```

<Callout>
  إذا فشل أمر التثبيت السريع، يرجى الرجوع إلى قسم [تثبيت
  التبعيات](/docs/intro/installation/dependencies) أدناه للحصول على تعليمات
  لتثبيت كل تبعية بشكل فردي.
</Callout>
