## حزم Rust

فيما يلي أهم وأكثر حزم Rust استخدامًا لتطوير سولانا:

- [`solana-program`] &mdash; يتم استيرادها بواسطة البرامج التي تعمل على سولانا،
  ويتم تجميعها إلى SBF. تحتوي هذه الحزمة على العديد من أنواع البيانات الأساسية
  ويتم إعادة تصديرها من [`solana-sdk`]، والتي لا يمكن استيرادها من برنامج
  سولانا.

- [`solana-sdk`] &mdash; مجموعة التطوير الأساسية خارج السلسلة، وهي تعيد تصدير
  [`solana-program`] وتضيف المزيد من واجهات برمجة التطبيقات فوق ذلك. ستقوم معظم
  برامج سولانا التي لا تعمل على السلسلة باستيراد هذه الحزمة.

- [`solana-client`] &mdash; للتفاعل مع عقدة سولانا عبر
  [واجهة برمجة تطبيقات JSON RPC](/docs/rpc).

- [`solana-cli-config`] &mdash; تحميل وحفظ ملف تكوين واجهة سطر أوامر سولانا.

- [`solana-clap-utils`] &mdash; روتينات لإعداد واجهة سطر الأوامر، باستخدام
  [`clap`]، كما هو مستخدم في واجهة سطر أوامر سولانا الرئيسية. تتضمن وظائف لتحميل
  جميع أنواع الموقعين المدعومين من قبل واجهة سطر الأوامر.

## التثبيت

لإرسال المعاملات والتفاعل مع سولانا:

```shell title="Terminal"
cargo add solana-sdk solana-client
```

لبناء برامج سولانا، قم بتثبيت:

```shell title="Terminal"
cargo add solana-program
```
