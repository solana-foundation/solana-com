هناك [مكتبة تطوير برمجيات Go](https://github.com/gagliardetto/solana-go) تدعم
معظم الوظائف للتفاعل مع سولانا. إذا كنت تستخدم Anchor لبرامج سولانا الخاصة بك،
ستتمكن من استخدام [anchor-go](https://github.com/gagliardetto/anchor-go) مع ملف
idl الخاص بك لإنتاج عميل متوافق مع Golang.

## التثبيت

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```
