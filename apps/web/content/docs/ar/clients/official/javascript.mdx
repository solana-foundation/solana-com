## مجموعة أدوات سولانا

`@solana/kit` هي مكتبة تايبسكريبت SDK الموصى بها للبناء على سولانا.

| الحزمة                         | الوصف                             | جيثب                                                                            |
| ------------------------------ | --------------------------------- | ------------------------------------------------------------------------------- |
| @solana/kit                    | المكتبة الأساسية                  | [المصدر](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | التفاعل مع System Program         | [المصدر](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | التفاعل مع Token Program          | [المصدر](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | التفاعل مع Token-2022 Program     | [المصدر](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | التفاعل مع Memo Program           | [المصدر](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | التفاعل مع Compute Budget Program | [المصدر](https://github.com/solana-program/compute-budget/tree/main/clients/js) |

- [الوثائق الرسمية لمجموعة الأدوات](https://www.solanakit.com/)
- [مثال: ربط محفظة مع Next.js](/docs/frontend/kit)

## سولانا Web3.js

`@solana/web3.js` هي مكتبة تايبسكريبت SDK القديمة لسولانا.

| الحزمة            | الوصف                                                   | جيثب                                                                               |
| ----------------- | ------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| @solana/web3.js   | المكتبة الأساسية                                        | [المصدر](https://github.com/solana-foundation/solana-web3.js)                      |
| @solana/spl-token | التفاعل مع Token وToken-2022 وAssociated Token Programs | [المصدر](https://github.com/solana-program/token-2022/tree/main/clients/js-legacy) |
| @solana/spl-memo  | التفاعل مع Memo Program                                 | [المصدر](https://github.com/solana-program/memo/tree/main/clients/js-legacy)       |
