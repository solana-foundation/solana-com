## مكتبة سولانا بايثون SDK

هناك حزمتان مفيدتان للتطوير باستخدام بايثون:

[`@kevinheavy/solana.py`](https://github.com/michaelhly/solana-py)

- Solana.py هي مكتبة بايثون الأساسية للتفاعل مع سولانا.
- يمكنك استخدامها لبناء المعاملات والتفاعل مع
  [واجهة برمجة تطبيقات سولانا JSON RPC](/docs/rpc)، تمامًا كما تفعل مع
  [solana-web3.js](https://github.com/solana-labs/solana-web3.js/)

- كما أنها تغطي [برنامج SPL Token](https://spl.solana.com/token).

- [أحدث توثيق](https://michaelhly.github.io/solana-py/).

[`@kevinheavey/solders`](https://github.com/kevinheavey/solders)

- `solders` هي مجموعة أدوات بايثون عالية الأداء لسولانا، مكتوبة بلغة Rust. وهي
  توفر حلولًا قوية في المجالات التالية:

- وظائف SDK الأساسية: keypairs، وpubkeys، وتوقيع وتسلسل المعاملات.
- RPC: بناء الطلبات وتحليل الاستجابات (للشبكات وواجهات برمجة التطبيقات غير
  المتزامنة، [solana-py](https://michaelhly.github.io/solana-py/rpc/async_api/)
  هو الخيار الموصى به).
- اختبار التكامل: وحدة `solders.litesvm` هي بديل لـ `solana-test-validator` وهي
  أكثر ملاءمة و**أسرع بكثير**. إنها تعتمد على
  [solana-program-test](https://crates.io/crates/solana-program-test).

## مكتبة سولانا كلاينت بايثون SDK

إذا كنت ترغب في التفاعل مع عقود سولانا باستخدام بايثون، يمكنك استخدام مكتبات SDK
للعملاء لتحويل عقود سولانا المكتوبة بلغة Rust إلى عملاء بايثون، إذا كان بإمكانك
الحصول على ملف IDL للعقد.

[`Codama-py`](https://www.npmjs.com/package/codama-py)

- توحيد أي برنامج عقد سولانا على السلسلة إلى
  [`Codama IDL`](https://github.com/codama-idl/codama) وإنشاء عميل بايثون
  تفاعلي.
- يدعم كلاً من برامج Anchor والبرامج المخصصة.

[`Anchor-py`](https://github.com/kevinheavey/anchorpy)

- Anchor-py هو عميل بايثون لبرامج Anchor. يمكنه إنشاء عملاء من ملفات Anchor IDL،
  والتفاعل مع برامج Anchor.
