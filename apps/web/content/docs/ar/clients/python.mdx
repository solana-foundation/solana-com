هناك [مكتبة بايثون SDK](https://github.com/michaelhly/solana-py) تدعم معظم
الوظائف للتفاعل مع سولانا. إذا كنت تستخدم Anchor لبرامج سولانا الخاصة بك، ستتمكن
من استخدام [anchor-py](https://github.com/kevinheavey/anchorpy) مع ملف idl الخاص
بك لإنشاء عميل متوافق مع بايثون.

## التثبيت

```bash title="Terminal"
pip install solders
pip install solana
```
