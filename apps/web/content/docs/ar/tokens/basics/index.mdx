استخدامًا.

جميع الأمثلة في هذا القسم تعمل بشكل متطابق مع كل من Token Program الأصلي وبرنامج
Token Extensions Program (Token 2022)، حيث يشتركان في نفس التنفيذ الأساسي.

<Callout type="info">
  راجع [وثائق Metaplex](https://developers.metaplex.com/token-metadata) للمزيد
  من المعلومات حول كيفية إضافة بيانات وصفية إلى رموز SPL.
</Callout>
