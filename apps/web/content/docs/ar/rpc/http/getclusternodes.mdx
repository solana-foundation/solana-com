<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getClusterNodes"
}
```

```ts !!request title="Kit"
import { createSolanaRpc } from "@solana/kit";

const rpc_url = "https://api.devnet.solana.com";
const rpc = createSolanaRpc(rpc_url);

let nodes = await rpc.getClusterNodes().send();

console.log(nodes);
```

```ts !!request title="web3.js"
import { Connection, clusterApiUrl } from "@solana/web3.js";

const connection = new Connection(clusterApiUrl("devnet"), "confirmed");

let nodes = await connection.getClusterNodes();

console.log(nodes);
```

```rs !!request title="Rust"
use anyhow::Result;
use solana_client::nonblocking::rpc_client::RpcClient;
use solana_sdk::commitment_config::CommitmentConfig;

#[tokio::main]
async fn main() -> Result<()> {
    let client = RpcClient::new_with_commitment(
        String::from("https://api.devnet.solana.com"),
        CommitmentConfig::confirmed(),
    );

    let block_time = client.get_cluster_nodes().await?;

    println!("{:#?}", block_time);

    Ok(())
}
```

!type array

سيكون حقل النتيجة عبارة عن مصفوفة من كائنات JSON، كل منها يحتوي على الحقول
الفرعية التالية:

##### !! featureSet

!type u32 | null

المعرف الفريد لمجموعة ميزات العقدة

##### !! gossip

!type string | null

عنوان شبكة Gossip للعقدة

##### !! pubkey

!type string

المفتاح العام للعقدة، كسلسلة مشفرة بنظام base-58

##### !! rpc

!type string | null

عنوان شبكة JSON RPC للعقدة، أو `null` إذا كانت خدمة JSON RPC غير مفعّلة

##### !! shredVersion

!type u16 | null

إصدار الشريحة (shred) الذي تم تكوين العقدة لاستخدامه

##### !! tpu

!type string | null

عنوان شبكة TPU للعقدة

##### !! version

!type string | null

إصدار البرنامج للعقدة، أو `null` إذا كانت معلومات الإصدار غير متوفرة
