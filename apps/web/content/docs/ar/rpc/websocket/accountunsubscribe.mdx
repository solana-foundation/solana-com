إلغاء الاشتراك من إشعارات تغيير الحساب

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "accountUnsubscribe",
  "params": [
    // !hover 0
    0
  ]
}
```

### !params

#### !! 0

!type number  
!required

معرّف اشتراك الحساب المراد إلغاؤه

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": true,
  "id": 1
}
```

!type boolean

رسالة نجاح إلغاء الاشتراك

</APIMethod>
