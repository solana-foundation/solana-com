## SDK de Python para Solana

Hay dos paquetes que son útiles para desarrollar con Python:

[`@kevinheavy/solana.py`](https://github.com/michaelhly/solana-py)

- Solana.py es la biblioteca base de Python para interactuar con Solana.
- Puedes usarla para construir transacciones e interactuar con la
  [API JSON RPC de Solana](/docs/rpc), de manera similar a como lo harías con
  [solana-web3.js](https://github.com/solana-labs/solana-web3.js/)

- También cubre el [Programa SPL Token](https://spl.solana.com/token).

- [Documentación más reciente](https://michaelhly.github.io/solana-py/).

[`@kevinheavey/solders`](https://github.com/kevinheavey/solders)

- `solders` es un kit de herramientas de Python de alto rendimiento para Solana,
  escrito en Rust. Proporciona soluciones robustas en las siguientes áreas:

- Funcionalidad básica del SDK: keypairs, pubkeys, firma y serialización de
  transacciones.
- RPC: construcción de solicitudes y análisis de respuestas (para redes y APIs
  asíncronas, [solana-py](https://michaelhly.github.io/solana-py/rpc/async_api/)
  es la opción recomendada).
- Pruebas de integración: el módulo `solders.litesvm` es una alternativa a
  `solana-test-validator` que es mucho más conveniente y **mucho** más rápida.
  Está basada en
  [solana-program-test](https://crates.io/crates/solana-program-test).

## SDK de Python para cliente de Solana

Si quieres interactuar con contratos de Solana con Python, puedes usar SDKs de
cliente para convertir contratos de Rust de Solana a clientes de Python, siempre
que puedas obtener el archivo IDL del contrato.

[`Codama-py`](https://www.npmjs.com/package/codama-py)

- Estandariza cualquier programa de contrato on-chain de Solana a
  [`Codama IDL`](https://github.com/codama-idl/codama) y genera un cliente
  interactivo de Python.
- Compatible tanto con programas Anchor como personalizados.

[`Anchor-py`](https://github.com/kevinheavey/anchorpy)

- Anchor-py es un cliente Python para programas Anchor. Puede generar clientes a
  partir de archivos IDL de Anchor e interactuar con programas Anchor.
