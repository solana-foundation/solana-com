  Aprende sobre la construcción de interfaces para programas de Solana.
---

El desarrollo frontend en Solana implica trabajar con programas, wallets y
frameworks populares de JavaScript como React. Interactuar con estos componentes
requiere manejar la conexión, la creación de transacciones y la lectura de
cuentas de Solana.

Para ayudar con este trabajo, hay disponibles diversas bibliotecas cliente de
Solana en diferentes frameworks.

## Bibliotecas principales

<Cards>
  <Card
    title="@solana/client"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/client"
  >
    - Cliente simple de Solana que agrupa RPC, wallets, transacciones - Incluye
    almacén de estado integrado, acciones, observadores, conectores
  </Card>
  <Card
    title="@solana/react-hooks"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks"
  >
    - Hooks completos para wallets, balances, transferencias, firmas, consultas
    - Hooks de proveedor de React que envuelven el estado de ejecución de
    `@solana/client`
  </Card>
  <Card
    title="@solana/web3-compat"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/web3-compat"
  >
    - Kit de herramientas compatible con Web3.js para simplificar la
    actualización. - Componentes internos más nuevos que dependen de una
    combinación de web3.js y kit.
  </Card>
  <Card
    title="@solana/kit"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/kit"
  >
    - SDK de bajo nivel de Solana que impulsa las otras bibliotecas de Solana
    como `@solana/react-hooks` - Completamente tree-shakable, usa estándares web
    modernos y alimenta el tiempo de ejecución
  </Card>
</Cards>

<Callout type="warn" title="@solana/web3.js está obsoleto`">
  Muchos proyectos del ecosistema Solana todavía dependen del obsoleto
  `@solana/web3.js`. Prefiere `@solana/web3-compat` para simplificar tu ruta de
  migración.
</Callout>

- [Guía de @solana/client](/docs/frontend/client): tiempo de ejecución ligero y
  sin interfaz para RPC, wallets y transacciones.
- [Guía de @solana/react-hooks](/docs/frontend/react-hooks): hooks de React en
  capas sobre el mismo tiempo de ejecución del cliente.
- [Guía de @solana/web3-compat](/docs/frontend/web3-compat): capa de
  compatibilidad para migrar de `@solana/web3.js` a stacks impulsados por Kit.
