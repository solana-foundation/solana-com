h1: Método RPC programUnsubscribe
---

Cancelar la suscripción a las notificaciones de cambios en cuentas propiedad del
programa

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "programUnsubscribe",
  "params": [
    // !hover 0
    0
  ]
}
```

### !params

#### !! 0

!type number  
!required

id de la suscripción de cuenta a cancelar

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": true,
  "id": 1
}
```

!type boolean

mensaje de éxito de cancelación de suscripción

</APIMethod>
