h1: Método RPC rootSubscribe
---

Suscríbete para recibir notificaciones cada vez que se establezca una nueva raíz
por el validator.

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**Ninguno**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

id de suscripción (necesario para cancelar la suscripción)

</APIMethod>

#### Formato de notificación:

El resultado es el número del último slot raíz.

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
