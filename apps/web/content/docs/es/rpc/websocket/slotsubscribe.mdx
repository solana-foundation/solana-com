h1: Método RPC slotSubscribe
---

Suscríbete para recibir notificaciones cada vez que un slot es procesado por el
validator

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "slotSubscribe"
}
```

### !params

**Ninguno**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

Id de suscripción (necesario para cancelar la suscripción)

</APIMethod>

#### Formato de notificación:

La notificación será un objeto con los siguientes campos:

- `parent: <u64>` - El slot padre
- `root: <u64>` - El slot raíz actual
- `slot: <u64>` - El nuevo valor de slot establecido

Ejemplo:

```json
{
  "jsonrpc": "2.0",
  "method": "slotNotification",
  "params": {
    "result": {
      "parent": 75,
      "root": 44,
      "slot": 76
    },
    "subscription": 0
  }
}
```
