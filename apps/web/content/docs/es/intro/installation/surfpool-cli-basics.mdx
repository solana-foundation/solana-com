Surfpool es una herramienta de desarrollo local que simplifica el desarrollo en
Solana automáticamente cargando programas y cuentas desde un clúster (mainnet,
devnet). Utiliza Surfpool CLI para iniciar un validator local para probar tus
programas y transacciones.

Visita la [documentación de Surfpool](https://docs.surfpool.run/) para más
información.

## Iniciar un validator local

Para iniciar un validator local, ejecuta el siguiente comando:

```terminal
$ surfpool start
```

Por defecto, Surfpool carga automáticamente cualquier programa y cuenta que tus
transacciones necesiten desde mainnet al validator local.

## Surfpool Studio

Surfpool Studio proporciona una interfaz web para interactuar con tu validator
local. Úsalo para:

- Ver detalles de transacciones
- Inspeccionar datos de cuentas
- Solicitar SOL y otros tokens usando el faucet de la interfaz

Para acceder a Surfpool Studio:

1. Inicia tu validator local con `surfpool start`
2. Abre tu navegador y dirígete a
   [http://127.0.0.1:18488](http://127.0.0.1:18488)
