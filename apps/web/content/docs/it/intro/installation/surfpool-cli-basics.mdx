Surfpool Ã¨ uno strumento di sviluppo locale che semplifica lo sviluppo su Solana
caricando automaticamente programmi e account da un cluster (mainnet, devnet).
Utilizza la CLI di Surfpool per avviare un validator locale per testare i tuoi
programmi e transazioni.

Visita la [documentazione di Surfpool](https://docs.surfpool.run/) per maggiori
informazioni.

## Avvia un validator locale

Per avviare un validator locale, esegui il seguente comando:

```terminal
$ surfpool start
```

Per impostazione predefinita, Surfpool carica automaticamente nel validator
locale qualsiasi programma e account da cui dipendono le tue transazioni,
prelevandoli dalla mainnet.

## Surfpool Studio

Surfpool Studio fornisce un'interfaccia web per interagire con il tuo validator
locale. Utilizzalo per:

- Visualizzare i dettagli delle transazioni
- Ispezionare i dati degli account
- Effettuare airdrop di SOL e altri token utilizzando il faucet dell'interfaccia
  utente

Per accedere a Surfpool Studio:

1. Avvia il tuo validator locale con `surfpool start`
2. Apri il browser e vai su [http://127.0.0.1:18488](http://127.0.0.1:18488)
