I crate Rust di Solana sono
[pubblicati su crates.io](https://crates.io/search?q=solana-) e possono essere
trovati [su docs.rs](https://docs.rs/releases/search?query=solana-) con il
prefisso `solana-`.

## Crate Rust

I seguenti sono i crate Rust più importanti e comunemente utilizzati per lo
sviluppo su Solana:

- [`solana-program`] &mdash; Importato dai programmi in esecuzione su Solana,
  compilato in SBF. Questo crate contiene molti tipi di dati fondamentali ed è
  ri-esportato da [`solana-sdk`], che non può essere importato da un programma
  Solana.

- [`solana-sdk`] &mdash; L'SDK offchain di base, ri-esporta [`solana-program`] e
  aggiunge più API su questa base. La maggior parte dei programmi Solana che non
  vengono eseguiti on-chain importeranno questo.

- [`solana-client`] &mdash; Per interagire con un nodo Solana tramite
  [API JSON RPC](/docs/rpc).

- [`solana-cli-config`] &mdash; Caricamento e salvataggio del file di
  configurazione della CLI Solana.

- [`solana-clap-utils`] &mdash; Routine per configurare una CLI, utilizzando
  [`clap`], come utilizzato dalla CLI principale di Solana. Include funzioni per
  caricare tutti i tipi di firmatari supportati dalla CLI.

[`solana-program`]: https://docs.rs/solana-program
[`solana-sdk`]: https://docs.rs/solana-sdk
[`solana-client`]: https://docs.rs/solana-client
[`solana-cli-config`]: https://docs.rs/solana-cli-config
[`solana-clap-utils`]: https://docs.rs/solana-clap-utils

## Installazione

Per inviare transazioni e interagire con Solana:

```shell title="Terminal"
cargo add solana-sdk solana-client
```

Per costruire programmi Solana, installa:

```shell title="Terminal"
cargo add solana-program
```
