## Crate dei programmi

Usa `solana-program` per costruire programmi on-chain.

- [Esempio: Costruire un programma Solana](/docs/programs/rust)

| Crate          | Descrizione                   | Documentazione                               | GitHub                                                                        |
| -------------- | ----------------------------- | -------------------------------------------- | ----------------------------------------------------------------------------- |
| solana-program | Costruisci programmi on-chain | [Visualizza](https://docs.rs/solana-program) | [Codice sorgente](https://github.com/anza-xyz/solana-sdk/tree/master/program) |

Pinocchio è una libreria senza dipendenze per creare programmi Solana in Rust e
può essere utilizzata come sostituto di `solana-program` per scrivere programmi
on-chain.

| Crate     | Descrizione                         | Documentazione                          | GitHub                                                   |
| --------- | ----------------------------------- | --------------------------------------- | -------------------------------------------------------- |
| pinocchio | Programmi on-chain senza dipendenze | [Visualizza](https://docs.rs/pinocchio) | [Codice sorgente](https://github.com/anza-xyz/pinocchio) |

<Callout type="warn">
  Pinocchio è ancora in fase di sviluppo e probabilmente avrà cambiamenti
  importanti nel prossimo futuro.
</Callout>

Pinocchio include crate specifici per programmi per interagire con i programmi
Solana:

| pinocchio-token                    | Interagisci con Token Program            | [Visualizza](https://docs.rs/pinocchio-token/)                    | [Codice sorgente](https://github.com/anza-xyz/pinocchio/tree/main/programs/token)                    |
