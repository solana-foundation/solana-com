## Solana Kit

`@solana/kit` è l'SDK TypeScript consigliato per sviluppare su Solana.

| Package                        | Descrizione                            | GitHub                                                                          |
| ------------------------------ | -------------------------------------- | ------------------------------------------------------------------------------- |
| @solana/kit                    | SDK principale                         | [Source](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | Interagisci con System Program         | [Source](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Interagisci con Token Program          | [Source](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Interagisci con Token-2022 Program     | [Source](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Interagisci con Memo Program           | [Source](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Interagisci con Compute Budget Program | [Source](https://github.com/solana-program/compute-budget/tree/main/clients/js) |
