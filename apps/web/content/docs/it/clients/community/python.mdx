## Solana Python SDK

Ci sono due pacchetti utili per lo sviluppo con Python:

[`@kevinheavy/solana.py`](https://github.com/michaelhly/solana-py)

- Solana.py è la libreria Python di base per interagire con Solana.
- Puoi utilizzarla per costruire transazioni e interagire con la
  [Solana JSON RPC API](/docs/rpc), proprio come faresti con
  [solana-web3.js](https://github.com/solana-labs/solana-web3.js/)

- Copre anche il [Token Program](/docs/rpc).

- [Documentazione più recente](https://michaelhly.github.io/solana-py/).

[`@kevinheavey/solders`](https://github.com/kevinheavey/solders)

- `solders` è un toolkit Python ad alte prestazioni per Solana, scritto in Rust.
  Fornisce soluzioni robuste nelle seguenti aree:

- Funzionalità core SDK: keypair, pubkey, firma e serializzazione delle
  transazioni.
- RPC: costruzione di richieste e analisi delle risposte (per il networking e le
  API asincrone,
  [solana-py](https://michaelhly.github.io/solana-py/rpc/async_api/) è l'opzione
  consigliata).
- Test di integrazione: il modulo `solders.litesvm` è un'alternativa a
  `solana-test-validator` che è molto più conveniente e **molto** più veloce. È
  basato su [solana-program-test](https://crates.io/crates/solana-program-test).

## Solana Client Python SDK

Se vuoi interagire con i contratti Solana con Python, puoi utilizzare gli SDK
client per convertire i contratti Solana in Rust in client Python, se riesci a
ottenere il file IDL del contratto.

[`Codama-py`](https://www.npmjs.com/package/codama-py)

- Standardizza qualsiasi programma contratto on-chain di Solana in
  [`Codama IDL`](https://github.com/codama-idl/codama) e genera un client Python
  interattivo.
- Supporta sia programmi Anchor che programmi personalizzati.

[`Anchor-py`](https://github.com/kevinheavey/anchorpy)

- Anchor-py è un client Python per programmi Anchor. Può generare client da file
  IDL Anchor e interagire con programmi Anchor.
