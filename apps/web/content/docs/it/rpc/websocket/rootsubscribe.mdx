h1: Metodo RPC rootSubscribe
---

Iscriviti per ricevere notifiche ogni volta che viene impostata una nuova root
dal validator.

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**Nessuno**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

id di sottoscrizione (necessario per annullare l'iscrizione)

</APIMethod>

#### Formato della notifica:

Il risultato Ã¨ il numero dell'ultimo slot root.

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
