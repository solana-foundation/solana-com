h1: Metodo RPC slotSubscribe

Sottoscrivi per ricevere notifiche ogni volta che uno slot viene elaborato dal
validator

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "slotSubscribe"
}
```

### !params

**Nessuno**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

Id di sottoscrizione (necessario per annullare la sottoscrizione)

</APIMethod>

#### Formato della notifica:

La notifica sar√† un oggetto con i seguenti campi:

- `parent: <u64>` - Lo slot padre
- `root: <u64>` - Lo slot root corrente
- `slot: <u64>` - Il nuovo valore dello slot impostato

Esempio:

```json
{
  "jsonrpc": "2.0",
  "method": "slotNotification",
  "params": {
    "result": {
      "parent": 75,
      "root": 44,
      "slot": 76
    },
    "subscription": 0
  }
}
```
