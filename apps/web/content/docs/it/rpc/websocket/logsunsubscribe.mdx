h1: Metodo RPC logsUnsubscribe
---

Annulla l'iscrizione alla registrazione delle transazioni

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "logsUnsubscribe",
  "params": [
    // !hover 0
    0
  ]
}
```

### !params

#### !! 0

!type integer  
!required

id di sottoscrizione da cancellare

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": true,
  "id": 1
}
```

!type boolean

messaggio di successo dell'annullamento dell'iscrizione

</APIMethod>
