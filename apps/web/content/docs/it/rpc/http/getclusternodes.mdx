h1: Metodo RPC getClusterNodes
---

Restituisce informazioni su tutti i nodi che partecipano al cluster

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getClusterNodes"
}
```

```ts !!request title="Kit"
import { createSolanaRpc } from "@solana/kit";

const rpc_url = "https://api.devnet.solana.com";
const rpc = createSolanaRpc(rpc_url);

let nodes = await rpc.getClusterNodes().send();

console.log(nodes);
```

```ts !!request title="web3.js"
import { Connection, clusterApiUrl } from "@solana/web3.js";

const connection = new Connection(clusterApiUrl("devnet"), "confirmed");

let nodes = await connection.getClusterNodes();

console.log(nodes);
```

```rs !!request title="Rust"
use anyhow::Result;
use solana_client::nonblocking::rpc_client::RpcClient;
use solana_sdk::commitment_config::CommitmentConfig;

#[tokio::main]
async fn main() -> Result<()> {
    let client = RpcClient::new_with_commitment(
        String::from("https://api.devnet.solana.com"),
        CommitmentConfig::confirmed(),
    );

    let block_time = client.get_cluster_nodes().await?;

    println!("{:#?}", block_time);

    Ok(())
}
```

!type array

Il campo result sarà un array di oggetti JSON, ciascuno con i seguenti
sotto-campi:

##### !! featureSet

!type u32 | null

L'identificatore univoco del set di funzionalità del nodo

##### !! gossip

!type string | null

Indirizzo di rete Gossip per il nodo

##### !! pubkey

!type string

Chiave pubblica del nodo, come stringa codificata in base-58

##### !! rpc

!type string | null

Indirizzo di rete JSON RPC per il nodo, o `null` se il servizio JSON RPC non è
abilitato

##### !! shredVersion

!type u16 | null

La versione shred con cui il nodo è stato configurato

##### !! tpu

!type string | null

Indirizzo di rete TPU per il nodo

##### !! version

!type string | null

La versione software del nodo, o `null` se le informazioni sulla versione non
sono disponibili

</APIMethod>
