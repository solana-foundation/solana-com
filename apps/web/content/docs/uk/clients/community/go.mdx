h1: Go SDK для Solana
---

Існує [Go SDK](https://github.com/gagliardetto/solana-go), який підтримує
більшість функцій для взаємодії з Solana. Якщо ви використовуєте Anchor для
своїх програм Solana, ви зможете використовувати
[anchor-go](https://github.com/gagliardetto/anchor-go) з вашим idl для створення
клієнта, сумісного з Golang.

## Встановлення

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```

## Додаткова документація

Будь ласка, відвідайте
[документацію Go SDK](https://github.com/gagliardetto/solana-go?tab=readme-ov-file#installation)
для отримання додаткової інформації про використання цього Go SDK з Solana.
