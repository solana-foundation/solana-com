## Крейти програм

Використовуйте `solana-program` для створення програм у блокчейні.

- [Приклад: Створення програми Solana](/docs/programs/rust)

| Крейт          | Опис                          | Документація                                  | GitHub                                                                     |
| -------------- | ----------------------------- | --------------------------------------------- | -------------------------------------------------------------------------- |
| solana-program | Створення програм у блокчейні | [Переглянути](https://docs.rs/solana-program) | [Вихідний код](https://github.com/anza-xyz/solana-sdk/tree/master/program) |

Pinocchio — це бібліотека без залежностей для створення програм Solana на Rust,
яку можна використовувати замість `solana-program` для написання програм у
блокчейні.

| Крейт     | Опис                                 | Документація                             | GitHub                                                |
| --------- | ------------------------------------ | ---------------------------------------- | ----------------------------------------------------- |
| pinocchio | Програми у блокчейні без залежностей | [Переглянути](https://docs.rs/pinocchio) | [Вихідний код](https://github.com/anza-xyz/pinocchio) |

<Callout type="warn">
  Pinocchio все ще перебуває в розробці і, ймовірно, матиме критичні зміни в
  найближчому майбутньому.
</Callout>

Pinocchio включає спеціальні крейти для взаємодії з програмами Solana:

| Крейт                              | Опис                                 | Документація                                                       | GitHub                                                                                            |
| ---------------------------------- | ------------------------------------ | ------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------- |
| pinocchio-system                   | Взаємодія з System Program           | [Переглянути](https://docs.rs/pinocchio-system/)                   | [Вихідний код](https://github.com/anza-xyz/pinocchio/tree/main/programs/system)                   |
| pinocchio-token                    | Взаємодія з Token Program            | [Переглянути](https://docs.rs/pinocchio-token/)                    | [Вихідний код](https://github.com/anza-xyz/pinocchio/tree/main/programs/token)                    |
| pinocchio-token-2022               | Взаємодія з Token-2022 Program       | [Переглянути](https://docs.rs/pinocchio-token-2022/)               | [Вихідний код](https://github.com/anza-xyz/pinocchio/tree/main/programs/token-2022)               |
| pinocchio-associated-token-account | Взаємодія з Associated Token Program | [Переглянути](https://docs.rs/pinocchio-associated-token-account/) | [Вихідний код](https://github.com/anza-xyz/pinocchio/tree/main/programs/associated-token-account) |
| pinocchio-memo                     | Взаємодія з Memo Program             | [Переглянути](https://docs.rs/pinocchio-memo/)                     | [Вихідний код](https://github.com/anza-xyz/pinocchio/tree/main/programs/memo)                     |
