Існує [Python SDK](https://github.com/michaelhly/solana-py), який підтримує
більшість функцій для взаємодії з Solana. Якщо ви використовуєте Anchor для
своїх програм Solana, ви зможете використовувати
[anchor-py](https://github.com/kevinheavey/anchorpy) з вашим idl для створення
клієнта, сумісного з Python.

## Встановлення

```bash title="Terminal"
pip install solders
pip install solana
```

## Додаткова документація

Будь ласка, відвідайте
[документацію Python SDK](https://github.com/michaelhly/solana-py?tab=readme-ov-file#general-usage)
для отримання додаткової інформації про використання цього Python SDK з Solana.

- [Найновіша документація](https://michaelhly.github.io/solana-py/).

[`@kevinheavey/solders`](https://github.com/kevinheavey/solders)

- `solders` - це високопродуктивний Python-інструментарій для Solana, написаний
  на Rust. Він надає надійні рішення у таких сферах:

- Основна функціональність SDK: keypair, pubkey, підписання та серіалізація
  транзакцій.
- RPC: створення запитів та аналіз відповідей (для мережевих та асинхронних API
  [solana-py](https://michaelhly.github.io/solana-py/rpc/async_api/) є
  рекомендованим варіантом).
- Інтеграційне тестування: модуль `solders.litesvm` є альтернативою
  `solana-test-validator`, яка набагато зручніша та **значно** швидша. Він
  базується на
  [solana-program-test](https://crates.io/crates/solana-program-test).

## Python SDK для клієнта Solana

Якщо ви хочете взаємодіяти з контрактами Solana за допомогою Python, ви можете
використовувати SDK для клієнтів, щоб перетворити контракти Solana на Rust у
Python-клієнти, якщо у вас є IDL-файл контракту.

[`Codama-py`](https://www.npmjs.com/package/codama-py)

- Стандартизує будь-яку програму контракту Solana на блокчейні до
  [`Codama IDL`](https://github.com/codama-idl/codama) та генерує інтерактивний
  клієнт Python.
- Підтримує як програми Anchor, так і власні програми.

[`Anchor-py`](https://github.com/kevinheavey/anchorpy)

- Anchor-py - це Python-клієнт для програм Anchor. Він може генерувати клієнти з
  файлів Anchor IDL та взаємодіяти з програмами Anchor.
