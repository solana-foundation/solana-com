Цей розділ охоплює основи взаємодії з SPL токенами, зосереджуючись на найбільш
часто використовуваних інструкціях.

Усі приклади в цьому розділі працюють однаково як з оригінальною Token Program,
так і з Token Extension Program (Token 2022), оскільки вони мають однакову
базову реалізацію.

<Callout type="info">
  Зверніться до [документації
  Metaplex](https://developers.metaplex.com/token-metadata) для отримання
  додаткової інформації про те, як додавати метадані до SPL токенів.
</Callout>
