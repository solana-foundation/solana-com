h1: RPC-метод getSupply
---

Повертає інформацію про поточну пропозицію.

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getSupply",
  "params": [
    // !hover(1:3) 0
    {
      // !hover commitment
      "commitment": "finalized"
    }
  ]
}
```

```ts !!request title="Kit"
import { createSolanaRpc } from "@solana/kit";

const rpc_url = "https://api.devnet.solana.com";
const rpc = createSolanaRpc(rpc_url);

let supply = await rpc.getSupply().send();

console.log(supply);
```

```ts !!request title="web3.js"
import { Connection, clusterApiUrl } from "@solana/web3.js";

const connection = new Connection(clusterApiUrl("devnet"), "confirmed");

let supply = await connection.getSupply();

console.log(supply);
```

### !params

#### !! 0

!type object

Об'єкт конфігурації, що містить такі поля:

##### !! commitment

!type string

Параметр commitment описує, наскільки фіналізованим є блок у певний момент часу.
Дивіться
[Налаштування підтвердження стану](/docs/rpc#configuring-state-commitment).

##### !! excludeNonCirculatingAccountsList

!type bool

Виключити список рахунків, що не перебувають в обігу, з відповіді

!type object

Результатом буде JSON-об'єкт, що містить:

##### !! circulating

!type u64

Обсяг в обігу в lamport

##### !! nonCirculating

!type u64

Некругова пропозиція в лампортах

##### !! nonCirculatingAccounts

!type array

Масив адрес рахунків некругових коштів у вигляді рядків. Якщо
`excludeNonCirculatingAccountsList` увімкнено, повернений масив буде порожнім.

##### !! total

!type u64

Загальна пропозиція в лампортах

</APIMethod>
