h1: RPC-метод getClusterNodes
---

Повертає інформацію про всі вузли, що беруть участь у кластері

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getClusterNodes"
}
```

```ts !!request title="Kit"
import { createSolanaRpc } from "@solana/kit";

const rpc_url = "https://api.devnet.solana.com";
const rpc = createSolanaRpc(rpc_url);

let nodes = await rpc.getClusterNodes().send();

console.log(nodes);
```

```ts !!request title="web3.js"
import { Connection, clusterApiUrl } from "@solana/web3.js";

const connection = new Connection(clusterApiUrl("devnet"), "confirmed");

let nodes = await connection.getClusterNodes();

console.log(nodes);
```

```rs !!request title="Rust"
use anyhow::Result;
use solana_client::nonblocking::rpc_client::RpcClient;
use solana_sdk::commitment_config::CommitmentConfig;

#[tokio::main]
async fn main() -> Result<()> {
    let client = RpcClient::new_with_commitment(
        String::from("https://api.devnet.solana.com"),
        CommitmentConfig::confirmed(),
    );

    let block_time = client.get_cluster_nodes().await?;

    println!("{:#?}", block_time);

    Ok(())
}
```

!type array

Поле результату буде масивом JSON-об'єктів, кожен з яких має такі підполя:

##### !! featureSet

!type u32 | null

Унікальний ідентифікатор набору функцій вузла

##### !! gossip

!type string | null

Мережева адреса gossip для вузла

##### !! pubkey

!type string

Публічний ключ вузла у вигляді рядка, закодованого в base-58

##### !! rpc

!type string | null

Мережева адреса JSON RPC для вузла, або `null`, якщо сервіс JSON RPC не
увімкнено

##### !! shredVersion

!type u16 | null

Версія шреду, яку налаштовано для використання вузлом

##### !! tpu

!type string | null

Мережева адреса TPU для вузла

##### !! version

!type string | null

Версія програмного забезпечення вузла, або `null`, якщо інформація про версію
недоступна
