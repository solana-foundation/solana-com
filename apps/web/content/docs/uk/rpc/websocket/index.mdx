Після підключення до RPC PubSub websocket за адресою `ws://<ADDRESS>/`:

- Надсилайте запити на підписку до websocket, використовуючи методи нижче
- Кілька підписок можуть бути активними одночасно
- Багато підписок приймають необов'язковий параметр
  [`commitment`](/docs/rpc/#configuring-state-commitment), який визначає,
  наскільки фіналізованою має бути зміна, щоб викликати сповіщення. Для
  підписок, якщо commitment не вказано, значення за замовчуванням — `finalized`.

## Кінцева точка RPC PubSub WebSocket

Порт за замовчуванням: `8900`

- ws://localhost:8900
- http://192.168.1.88:8900
