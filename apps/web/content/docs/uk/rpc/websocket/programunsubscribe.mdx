h1: RPC-метод programUnsubscribe
---

Скасування підписки на сповіщення про зміни в облікових записах програми

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "programUnsubscribe",
  "params": [
    // !hover 0
    0
  ]
}
```

### !params

#### !! 0

!type number  
!required

ідентифікатор підписки на обліковий запис, яку потрібно скасувати

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": true,
  "id": 1
}
```

!type boolean

повідомлення про успішне скасування підписки

</APIMethod>
