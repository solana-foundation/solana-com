Скасувати підписку на сповіщення про зміни облікового запису

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "accountUnsubscribe",
  "params": [
    // !hover 0
    0
  ]
}
```

### !params

#### !! 0

!type number  
!required

ідентифікатор підписки на обліковий запис, яку потрібно скасувати

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": true,
  "id": 1
}
```

!type boolean

повідомлення про успішне скасування підписки

</APIMethod>
