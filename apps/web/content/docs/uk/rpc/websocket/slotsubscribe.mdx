h1: RPC-метод slotSubscribe
---

Підписатися на отримання сповіщень щоразу, коли validator обробляє slot

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "slotSubscribe"
}
```

### !params

**Відсутні**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

Ідентифікатор підписки (потрібен для скасування підписки)

</APIMethod>

#### Формат сповіщення:

Сповіщення буде об'єктом з такими полями:

- `parent: <u64>` - Батьківський slot
- `root: <u64>` - Поточний кореневий slot
- `slot: <u64>` - Нове встановлене значення slot

Приклад:

```json
{
  "jsonrpc": "2.0",
  "method": "slotNotification",
  "params": {
    "result": {
      "parent": 75,
      "root": 44,
      "slot": 76
    },
    "subscription": 0
  }
}
```
