h1: RPC-метод rootSubscribe
---

Підпишіться, щоб отримувати сповіщення щоразу, коли validator встановлює новий
root.

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**Відсутні**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

ідентифікатор підписки (потрібен для скасування підписки)

</APIMethod>

#### Формат сповіщення:

Результатом є номер останнього root slot.

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
