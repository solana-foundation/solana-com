Surfpool — це інструмент локальної розробки, який спрощує розробку на Solana
шляхом автоматичного завантаження програм та облікових записів з кластера
(mainnet, devnet). Використовуйте Surfpool CLI для запуску локального validator
для тестування ваших програм та транзакцій.

Відвідайте [документацію Surfpool](https://docs.surfpool.run/) для отримання
додаткової інформації.

## Запуск локального validator

Щоб запустити локальний validator, виконайте наступну команду:

```terminal
$ surfpool start
```

За замовчуванням Surfpool автоматично завантажує будь-які програми та облікові
записи, від яких залежать ваші транзакції, з mainnet до локального validator.

## Surfpool Studio

Surfpool Studio надає веб-інтерфейс для взаємодії з вашим локальним validator.
Використовуйте його для:

- Перегляду деталей транзакцій
- Перевірки даних облікових записів
- Аірдропу SOL та інших токенів за допомогою UI-фаусету

Щоб отримати доступ до Surfpool Studio:

1. Запустіть свій локальний validator за допомогою `surfpool start`
2. Відкрийте браузер і перейдіть за адресою
   [http://127.0.0.1:18488](http://127.0.0.1:18488)
