# ¿Qué es una billetera?

Una billetera es tu herramienta principal para usar Solana. Almacena tus activos
digitales, te permite conectarte a aplicaciones y te facilita enviar y recibir
tokens. Este capítulo te ayudará a entender las billeteras y a configurar tu
primera billetera de forma segura.

## Lo que aprenderás

Al final de este capítulo, entenderás:

- Qué hacen las billeteras y cómo funcionan
- Diferentes tipos de billeteras y sus ventajas y desventajas
- Cómo crear tu primera billetera
- Mejores prácticas de seguridad
- Características y usos comunes de las billeteras

## Entendiendo las billeteras

{/* Illustration recommendation: Simple diagram showing a wallet as a container for keys, tokens, and NFTs, with arrows showing interactions with various applications */}

Una billetera cumple varias funciones:

- **Almacena tus claves**: Las claves criptográficas que prueban que eres dueño
  de tus activos
- **Guarda tus activos**: Muestra tus tokens, NFTs y otros elementos digitales
- **Firma transacciones**: Aprueba acciones en tu nombre
- **Se conecta a aplicaciones**: Tu método de inicio de sesión para aplicaciones
  de Solana

Piensa en ella como una combinación de:

- Un gestor de contraseñas (almacena tus claves)
- Un visor de cuenta bancaria (muestra tus saldos)
- Una herramienta de firma digital (aprueba transacciones)

## Cómo funcionan las billeteras

Cada billetera se basa en claves criptográficas:

### Frase semilla

Una serie de 12-24 palabras que pueden recrear tu billetera. Este es tu respaldo
definitivo - mantenla segura y privada.

### Clave privada

Derivada de tu frase semilla, esta clave firma las transacciones. Tu billetera
la gestiona por ti - nunca necesitarás verla o manejarla directamente.

### Clave pública/Dirección

Derivada de tu clave privada, es lo que compartes para recibir fondos. Es seguro
compartirla públicamente.

## Tipos de billeteras

### Billeteras de software

**Extensiones de navegador**

- Convenientes para aplicaciones web
- Fáciles de usar
- Buenas para transacciones diarias
- Gratuitas

**Aplicaciones móviles**

- Accede a tu billetera desde cualquier lugar
- Escaneo de códigos QR para pagos
- Seguridad biométrica (huella digital/facial)
- Gratuitas

**Aplicaciones de escritorio**

- Interfaces completas
- Buenas para traders activos
- Funcionalidad avanzada
- Gratuitas

### Billeteras hardware

**Dispositivos físicos**

- Máxima seguridad
- Almacenamiento de claves offline
- Ideales para grandes cantidades
- Costo: $50-200

### Eligiendo tu primera billetera

Para principiantes, recomendamos comenzar con una billetera de extensión de
navegador. Son:

- Amigables para el usuario
- Bien respaldadas
- Seguras para uso diario
- Fáciles de configurar

## Creando tu primera billetera

{/* Recommendation de ilustración: Guía visual paso a paso mostrando el proceso de creación de billetera */}

Veamos el proceso general para configurar una billetera:

### Paso 1: Elige e instala una billetera

1. Investiga billeteras populares de Solana
2. Visita el sitio web oficial (siempre verifica las URLs)
3. Descarga la billetera para tu plataforma
4. Instálala siguiendo las instrucciones oficiales

### Paso 2: Crea una nueva billetera

1. Abre tu aplicación de billetera
2. Selecciona la opción "Crear una nueva billetera"
3. Crea una contraseña fuerte para el acceso al dispositivo
4. **Importante**: Anota tu frase seed

### Paso 3: Asegura tu frase semilla

{/* Illustration recommendation: Do's and don'ts for seed phrase storage - physical paper in safe (✓), screenshot (✗), cloud storage (✗) */}

Tu frase semilla es crítica. Así es como almacenarla de forma segura:

**SÍ DEBES:**

- Escribirla en papel
- Guardarla en un lugar seguro
- Considerar tener varias copias en diferentes lugares
- Usar una caja fuerte ignífuga si es posible

**NO DEBES:**

- Hacer capturas de pantalla
- Guardarla en correo electrónico o servicios en la nube
- Compartirla con nadie
- Escribirla en sitios web aleatorios

### Paso 4: Verificar y finalizar

1. Confirma tu frase semilla cuando se te solicite
2. Tu billetera ahora está lista para usar
3. Verás tu dirección de billetera en la parte superior
4. Tu saldo inicial será de 0 SOL

## Características básicas de la billetera

Las billeteras modernas incluyen:

### Gestión de activos

- Ver saldos de tokens
- Mostrar colecciones de NFT
- Seguir el historial de transacciones
- Monitorear el valor de la cartera

### Herramientas de transacción

- Enviar y recibir tokens
- Intercambiar entre diferentes tokens
- Conectar con aplicaciones
- Aprobar o rechazar transacciones

### Características de seguridad

- Protección con contraseña
- Autenticación biométrica (móvil)
- Advertencias de transacción
- Detección de phishing

## Tus primeros pasos

Ahora que tienes una billetera, esto es lo que debes hacer a continuación:

1. **Guarda tu dirección**: Cópiala en un lugar seguro para referencia
2. **Consigue algo de SOL**: Necesitarás una pequeña cantidad para las
   comisiones de transacción
3. **Practica recibiendo**: Pide a un amigo que te envíe una cantidad mínima
4. **Explora la interfaz**: Familiarízate con el diseño

## Recordatorios importantes de seguridad

- **Nunca compartas tu frase semilla**: Ningún servicio legítimo te la pedirá
- **Verifica los sitios web**: Los estafadores crean sitios falsos de billeteras
- **Comienza con poco**: Usa pequeñas cantidades mientras aprendes
- **Mantén el software actualizado**: Instala las actualizaciones de la
  billetera con prontitud
- **Sé escéptico**: Si algo parece demasiado bueno para ser verdad,
  probablemente lo sea

## Preguntas frecuentes

### ¿Necesito pagar por una billetera?

La mayoría de las billeteras de software son gratuitas. Solo pagas por
billeteras físicas o funciones premium.

### ¿Puedo tener múltiples billeteras?

Sí, muchos usuarios tienen varias billeteras para diferentes propósitos (uso
diario, ahorros, aplicaciones específicas).

### ¿Qué pasa si olvido mi contraseña?

La contraseña de tu dispositivo puede restablecerse con tu seed phrase. Por eso
el seed phrase es tan importante.

### ¿Mis activos están almacenados en la billetera?

No, los activos se almacenan en Solana. Tu billetera guarda las claves que
prueban la propiedad.
