# Introduzione ai token Solana

I token sono asset digitali costruiti su Solana. Come il dollaro americano è la
valuta nativa degli Stati Uniti, SOL è la valuta nativa di Solana. I token
rappresentano tutti gli altri asset - dagli stablecoin come USDC ai token di
progetto e NFT. Questo capitolo spiega come funzionano i token e cosa puoi fare
con essi.

## Cosa imparerai

Alla fine di questo capitolo, comprenderai:

- Cosa sono i token e come funzionano su Solana
- Diversi tipi di token e i loro usi
- Come gestire i token nel tuo wallet
- Token account e commissioni
- Considerazioni sulla sicurezza quando si ha a che fare con i token

## Comprendere i token su Solana

Solana ha due tipi principali di asset digitali:

- **SOL**: La valuta nativa utilizzata per transazioni e commissioni
- **Token**: Tutti gli altri asset digitali su Solana

Proprio come le app sul tuo telefono seguono determinati standard per funzionare
correttamente, i token su Solana seguono uno standard comune che li rende
compatibili con wallet e applicazioni.

## Tipi di token

{/* Illustration recommendation: Grid showing different token categories with examples and use cases */}

### Stablecoin

Versioni digitali di valute tradizionali che mantengono un valore stabile.

**Esempi:**

- **USDC**: Dollari statunitensi digitali
- **USDT**: Tether, un altro stablecoin in USD
- **EURC**: Euro digitali

**Usi:** Risparmio, pagamenti, trading, evitare la volatilità

### Asset wrapped

Token che rappresentano asset provenienti da altre reti.

**Esempi:**

- **Wrapped Bitcoin (BTC)**: Bitcoin utilizzabile su Solana
- **Wrapped Ethereum (ETH)**: Ethereum su Solana
- **Wrapped SOL**: Utilizzato in alcuni protocolli DeFi

**Utilizzi:** Accedi ad altre criptovalute con la velocità e le basse
commissioni di Solana

### Token di progetto

Token creati da applicazioni e protocolli.

**Esempi:**

- Token di governance per votare sulle decisioni del protocollo
- Token di exchange che offrono sconti sulle commissioni
- Token di protocollo per la partecipazione all'ecosistema

**Utilizzi:** Votazioni di governance, sconti sulle commissioni, ricompense

### Token di utilità

Token con funzioni specifiche all'interno delle applicazioni.

**Esempi:**

- Token di gaming per economie in-game
- Token sociali per community di creator
- Token di ricompensa dai protocolli

**Utilizzi:** Accesso a funzionalità, guadagno di ricompense, partecipazione
agli ecosistemi

## Come funzionano i token account

{/* Illustration recommendation: Wallet diagram showing main SOL account and separate token accounts as sub-compartments */}

A differenza di alcune reti dove tutti gli asset condividono un unico indirizzo,
Solana utilizza un sistema di token account:

1. **Wallet principale**: Contiene i tuoi SOL
2. **Token account**: Account separati per ogni tipo di token

### Perché i token account?

- Migliori prestazioni e organizzazione
- Chiara separazione dei diversi asset
- Archiviazione di rete efficiente

### Rent dell'account

- Ogni token account richiede un deposito di ~0.002 SOL
- Questo "rent" è rimborsabile quando si chiude l'account
- Previene lo spam e finanzia l'archiviazione di rete
- Tipicamente costa $0.20-0.50 per tipo di token

## Gestione dei tuoi token

### Ricezione di token

1. Condividi l'indirizzo del tuo wallet
2. Il mittente avvia il trasferimento
3. Il wallet crea un token account se necessario
4. I token appaiono nel tuo wallet

### Visualizzazione dei tuoi token

I wallet moderni mostrano:

- Simboli e loghi dei token
- Saldi attuali
- Valori in USD
- Transazioni recenti

### Invio di token

1. Seleziona il token da inviare
2. Inserisci l'indirizzo del destinatario
3. Scegli l'importo
4. Conferma la transazione
5. Paga una piccola commissione in SOL (~$0.0005)

### Chiusura degli account token

Quando non hai più bisogno di un token:

1. Invia tutti i token
2. Chiudi l'account
3. Recupera il deposito di rent

## Nozioni di base sulla creazione di token

Chiunque può creare token su Solana:

### Requisiti

- Una piccola quantità di SOL per le commissioni (~0.5 SOL)
- Informazioni di base sul token (nome, simbolo, offerta)
- Opzionale: logo e metadati

### Usi comuni

- Token comunitari
- Sistemi di ricompensa
- Raccolta fondi
- Economie di gioco
- Progetti sperimentali

### Nota importante

Creare un token non gli conferisce valore. Il valore deriva dall'utilità,
dall'adozione e dalle dinamiche di mercato.

## Sicurezza dei token

{/* Illustration recommendation: Warning signs and safety checklist */}

### Verifica prima di fidarti

- Controlla i siti web ufficiali del progetto
- Verifica gli indirizzi dei token
- Cerca report di audit
- Fai ricerche sul team

### Truffe comuni da evitare

- **Token falsi**: Impersonificazione di progetti popolari
- **Rug pull**: Creatori che abbandonano i progetti
- **Honeypot**: Token che puoi comprare ma non vendere
- **Pump and dump**: Manipolazione artificiale dei prezzi

### Migliori pratiche

1. **Ricerca approfonditamente** prima di acquistare
2. **Inizia con piccole somme** con nuovi token
3. **Usa fonti affidabili** per le informazioni sui token
4. **Sii scettico** riguardo ai rendimenti garantiti
5. **Controlla la liquidità** prima di effettuare acquisti importanti

## Lavorare con i token

### Scambio di token

Usa aggregatori DEX per trovare i tassi migliori:

1. Connetti il wallet
2. Seleziona i token da scambiare
3. Rivedi il tasso e le commissioni
4. Conferma lo scambio

### Verifica delle informazioni sui token

Risorse per la ricerca:

- Block explorer: visualizza dettagli e possessori dei token
- Aggregatori di prezzi: controlla prezzi e liquidità
- Siti ufficiali dei progetti: verifica l'autenticità

### Gestione di più token

- Organizza nascondendo i saldi zero
- Aggiungi ai preferiti i token utilizzati frequentemente
- Utilizza tracker di portafoglio
- Chiudi i token account inutilizzati

## Domande frequenti

### Ho bisogno di SOL per inviare token?

Sì, tutte le transazioni richiedono piccole commissioni in SOL, anche quando si
inviano altri token.

### Perché alcuni token appaiono automaticamente?

I progetti possono effettuare airdrop di token ai wallet. Fai attenzione ai
token non richiesti.

### Posso recuperare i token inviati?

No, le transazioni su Solana sono permanenti. Controlla sempre due volte gli
indirizzi.

### Cosa determina il valore di un token?

Domanda e offerta, utilità, sviluppo del progetto e sentiment del mercato
influenzano tutti i prezzi dei token.
