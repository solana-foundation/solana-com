# Cos'è un wallet?

Un wallet è il tuo strumento principale per utilizzare Solana. Conserva i tuoi
asset digitali, ti permette di connetterti alle applicazioni e ti consente di
inviare e ricevere token. Questo capitolo ti aiuterà a comprendere i wallet e a
configurare il tuo primo wallet in modo sicuro.

## Cosa imparerai

Alla fine di questo capitolo, comprenderai:

- Cosa fanno i wallet e come funzionano
- I diversi tipi di wallet e i loro compromessi
- Come creare il tuo primo wallet
- Le migliori pratiche di sicurezza
- Funzionalità e usi comuni dei wallet

## Comprendere i wallet

{/* Illustration recommendation: Simple diagram showing a wallet as a container for keys, tokens, and NFTs, with arrows showing interactions with various applications */}

Un wallet svolge diverse funzioni:

- **Conserva le tue chiavi**: Le chiavi crittografiche che dimostrano che
  possiedi i tuoi asset
- **Contiene i tuoi asset**: Visualizza i tuoi token, NFT e altri elementi
  digitali
- **Firma le transazioni**: Approva le azioni per tuo conto
- **Si connette alle app**: Il tuo metodo di accesso per le applicazioni Solana

Pensalo come una combinazione di:

- Un gestore di password (conserva le tue chiavi)
- Un visualizzatore di conto bancario (mostra i tuoi saldi)
- Uno strumento di firma digitale (approva le transazioni)

## Come funzionano i wallet

Ogni wallet è costruito su chiavi crittografiche:

### Seed phrase

Una serie di 12-24 parole che possono ricreare il tuo wallet. Questo è il tuo
backup definitivo - conservalo in modo sicuro e privato.

### Chiave privata

Derivata dalla tua seed phrase, questa chiave firma le transazioni. Il tuo
wallet la gestisce per te - non avrai mai bisogno di vederla o gestirla
direttamente.

### Chiave pubblica/Indirizzo

Derivato dalla tua chiave privata, questo è ciò che condividi per ricevere
fondi. È sicuro da condividere pubblicamente.

## Tipi di wallet

### Wallet software

**Estensioni per browser**

- Comode per le applicazioni web
- Facili da usare
- Ideali per le transazioni quotidiane
- Gratuite

**App mobile**

- Accedi al tuo wallet ovunque
- Scansione di codici QR per i pagamenti
- Sicurezza biometrica (impronta digitale/volto)
- Gratuite

**Applicazioni desktop**

- Interfacce complete di funzionalità
- Ideali per trader attivi
- Funzionalità avanzate
- Gratuite

### Wallet hardware

**Dispositivi fisici**

- Massima sicurezza
- Archiviazione offline delle chiavi
- Ideali per grandi patrimoni
- Costo: 50-200€

### Scegliere il tuo primo wallet

Per i principianti, consigliamo di iniziare con un wallet a estensione per
browser. Sono:

- Facili da usare
- Ben supportati
- Sicuri per l'uso quotidiano
- Facili da configurare

## Creare il tuo primo wallet

{/* Illustration recommendation: Step-by-step visual guide showing the wallet creation process */}

Vediamo il processo generale per configurare un wallet:

### Fase 1: Scegliere e installare un wallet

1. Ricerca i wallet Solana più popolari
2. Visita il sito web ufficiale (verifica sempre gli URL)
3. Scarica il wallet per la tua piattaforma
4. Installa seguendo le istruzioni ufficiali

### Fase 2: Creare un nuovo wallet

1. Apri l'applicazione wallet
2. Seleziona l'opzione "Crea un nuovo wallet"
3. Crea una password forte per l'accesso al dispositivo
4. **Importante**: Annota la tua seed phrase

### Passo 3: proteggi la tua seed phrase

{/* Illustration recommendation: Do's and don'ts for seed phrase storage - physical paper in safe (✓), screenshot (✗), cloud storage (✗) */}

La tua seed phrase è fondamentale. Ecco come conservarla in modo sicuro:

**DA FARE:**

- Scriverla su carta
- Conservarla in un luogo sicuro
- Considerare più copie in luoghi diversi
- Usare una cassaforte ignifuga se possibile

**DA NON FARE:**

- Fare screenshot
- Conservarla in email o servizi cloud
- Condividerla con chiunque
- Digitarla su siti web casuali

### Passo 4: verifica e completa

1. Conferma la tua seed phrase quando richiesto
2. Il tuo wallet è ora pronto per l'uso
3. Vedrai il tuo indirizzo wallet in alto
4. Il tuo saldo iniziale sarà di 0 SOL

## Funzionalità base del wallet

I wallet moderni includono:

### Gestione degli asset

- Visualizzazione dei saldi dei token
- Visualizzazione delle collezioni NFT
- Monitoraggio della cronologia delle transazioni
- Controllo del valore del portafoglio

### Strumenti per le transazioni

- Invio e ricezione di token
- Scambio tra token diversi
- Connessione alle applicazioni
- Approvazione o rifiuto delle transazioni

### Funzionalità di sicurezza

- Protezione con password
- Autenticazione biometrica (mobile)
- Avvisi sulle transazioni
- Rilevamento del phishing

## I tuoi primi passi

Ora che hai un wallet, ecco cosa fare:

1. **Salva il tuo indirizzo**: copialo in un luogo sicuro per riferimento
2. **Ottieni un po' di SOL**: ti servirà una piccola quantità per le commissioni
   di transazione
3. **Esercitati a ricevere**: fatti inviare una piccola quantità da un amico
4. **Esplora l'interfaccia**: familiarizza con il layout

## Importanti promemoria sulla sicurezza

- **Non condividere mai la tua seed phrase**: nessun servizio legittimo te la
  chiederà
- **Verifica i siti web**: i truffatori creano siti wallet falsi
- **Inizia in piccolo**: usa piccole quantità mentre impari
- **Mantieni il software aggiornato**: installa prontamente gli aggiornamenti
  del wallet
- **Sii scettico**: se qualcosa sembra troppo bello per essere vero,
  probabilmente lo è

## Domande frequenti

### Devo pagare per un wallet?

La maggior parte dei wallet software sono gratuiti. Si paga solo per i wallet
hardware o per funzionalità premium.

### Posso avere più wallet?

Sì, molti utenti hanno diversi wallet per scopi differenti (uso quotidiano,
risparmi, applicazioni specifiche).

### Cosa succede se dimentico la password?

La password del tuo dispositivo può essere reimpostata con la tua seed phrase.
Ecco perché la seed phrase è così importante.

### I miei asset sono conservati nel wallet?

No, gli asset sono conservati su Solana. Il tuo wallet conserva le chiavi che ne
provano la proprietà.
