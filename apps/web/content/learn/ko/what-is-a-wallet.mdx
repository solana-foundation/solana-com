# 지갑이란 무엇인가요?

지갑은 솔라나를 사용하기 위한 주요 도구입니다. 디지털 자산을 저장하고,
애플리케이션에 연결할 수 있게 해주며, 토큰을 주고받을 수 있게 합니다. 이
장에서는 지갑을 이해하고 첫 번째 지갑을 안전하게 설정하는 데 도움을 드립니다.

## 배우게 될 내용

이 장을 마치면 다음 내용을 이해하게 됩니다:

- 지갑의 기능과 작동 방식
- 다양한 유형의 지갑과 각각의 장단점
- 첫 번째 지갑을 만드는 방법
- 보안 모범 사례
- 일반적인 지갑 기능과 용도

## 지갑 이해하기

![지갑의 네 가지 주요 기능을 보여주는 다이어그램: 트랜잭션 서명, 키 저장, 앱 연결, 자산 보유](/images/learn/understanding_wallets.webp)

지갑은 여러 기능을 수행합니다:

- **키 저장**: 자산 소유권을 증명하는 암호화 키
- **자산 보관**: 토큰, NFT 및 기타 디지털 아이템 표시
- **트랜잭션 서명**: 사용자를 대신하여 작업 승인
- **앱 연결**: 솔라나 애플리케이션의 로그인 방법

다음의 조합이라고 생각하세요:

- 비밀번호 관리자(키 저장)
- 은행 계좌 뷰어(잔액 표시)
- 디지털 서명 도구(트랜잭션 승인)

## 지갑 작동 방식

모든 지갑은 암호화 키를 기반으로 합니다:

### Seed 구문

지갑을 복구할 수 있는 12-24개의 단어 시리즈입니다. 이것은 최종 백업이므로
안전하고 비공개로 보관하세요.

### 개인 키

시드 구문에서 파생된 이 키는 트랜잭션에 서명합니다. 지갑이 이를 관리해주므로
직접 보거나 다룰 필요가 없습니다.

### 공개 키/주소

개인 키에서 파생된 이것은 자금을 받기 위해 공유하는 정보입니다. 공개적으로
공유해도 안전합니다.

## 지갑 유형

### 소프트웨어 지갑

**브라우저 확장 프로그램**

- 웹 애플리케이션에 편리함
- 사용하기 쉬움
- 일상적인 거래에 적합
- 무료로 사용 가능

**모바일 앱**

- 어디서나 지갑에 접근 가능
- QR 코드 스캔으로 결제
- 생체 인식 보안(지문/얼굴)
- 무료로 사용 가능

**데스크톱 애플리케이션**

- 모든 기능을 갖춘 인터페이스
- 활발한 트레이더에게 적합
- 고급 기능
- 무료로 사용 가능

### 하드웨어 지갑

**물리적 장치**

- 최대 보안
- 오프라인 키 저장
- 대규모 자산 보유에 최적
- 가격: $50-200

### 첫 번째 지갑 선택하기

초보자에게는 브라우저 확장 프로그램 지갑으로 시작하는 것을 권장합니다. 이유는:

- 사용자 친화적
- 잘 지원됨
- 일상적인 사용에 안전
- 설정이 쉬움

## 첫 번째 지갑 만들기

![지갑 생성을 위한 4단계 과정: 선택 및 설치, 새 지갑 생성, seed 구문 보안, 확인 및 완료](/images/learn/creating_your_first_wallet.webp)

지갑 설정의 일반적인 과정을 살펴보겠습니다:

### 1단계: 지갑 선택 및 설치

1. 인기 있는 솔라나 지갑 조사하기
2. 공식 웹사이트 방문(항상 URL 확인)
3. 사용 중인 플랫폼에 맞는 지갑 다운로드
4. 공식 지침에 따라 설치

### 2단계: 새 지갑 만들기

1. 지갑 애플리케이션 열기
2. "새 지갑 만들기" 옵션 선택
3. 기기 접근을 위한 강력한 비밀번호 생성
4. **중요**: seed 구문 적어두기

### 3단계: 시드 구문 보안

![seed 구문 보안 수칙 - 종이에 기록하기, 안전한 장소에 보관하기 vs 스크린샷 찍거나 온라인에 저장하지 않기](/images/learn/step_3_secure_your_seed_phrase.webp)

시드 구문은 매우 중요합니다. 다음은 안전하게 보관하는 방법입니다:

**해야 할 것:**

- 종이에 적기
- 안전한 장소에 보관하기
- 여러 장소에 여러 사본 보관 고려하기
- 가능하면 내화 금고 사용하기

**하지 말아야 할 것:**

- 스크린샷 찍기
- 이메일이나 클라우드 서비스에 저장하기
- 타인과 공유하기
- 무작위 웹사이트에 입력하기

### 4단계: 확인 및 완료

1. 메시지가 표시되면 시드 구문 확인
2. 이제 지갑 사용 준비 완료
3. 상단에 지갑 주소가 표시됨
4. 초기 잔액은 0 SOL

## 기본 지갑 기능

현대적인 지갑에는 다음이 포함됩니다:

### 자산 관리

- 토큰 잔액 확인
- NFT 컬렉션 표시
- 거래 내역 추적
- 포트폴리오 가치 모니터링

### 거래 도구

- 토큰 보내기 및 받기
- 다양한 토큰 간 교환
- 애플리케이션 연결
- 거래 승인 또는 거부

### 보안 기능

- 비밀번호 보호
- 생체 인증(모바일)
- 거래 경고
- 피싱 감지

## 첫 단계

이제 지갑이 생겼으니 다음 단계를 따르세요:

1. **주소 저장하기**: 참조를 위해 안전한 곳에 복사해두기
2. **SOL 구하기**: 거래 수수료를 위해 소량이 필요합니다
3. **수신 연습하기**: 친구에게 아주 작은 금액을 보내달라고 요청하기
4. **인터페이스 탐색하기**: 레이아웃에 익숙해지기

## 중요한 보안 알림

- **절대 시드 구문을 공유하지 마세요**: 정당한 서비스는 절대 이를 요구하지
  않습니다
- **웹사이트 확인하기**: 사기꾼들은 가짜 지갑 사이트를 만듭니다
- **소액으로 시작하기**: 학습하는 동안 소액을 사용하세요
- **소프트웨어 업데이트 유지하기**: 지갑 업데이트를 즉시 설치하세요
- **의심하기**: 너무 좋아 보이는 것은 대개 사실이 아닙니다

## 자주 묻는 질문

### 지갑 사용에 비용을 지불해야 하나요?

대부분의 소프트웨어 지갑은 무료입니다. 하드웨어 지갑이나 프리미엄 기능에만
비용을 지불합니다.

### 여러 개의 지갑을 가질 수 있나요?

네, 많은 사용자들이 다양한 목적(일상 사용, 저축, 특정 애플리케이션)에 맞게 여러
지갑을 사용합니다.

### 비밀번호를 잊어버리면 어떻게 하나요?

기기 비밀번호는 seed 구문으로 재설정할 수 있습니다. 그래서 seed 구문이 매우
중요합니다.

### 내 자산이 지갑에 저장되나요?

아니요, 자산은 Solana에 저장됩니다. 지갑은 소유권을 증명하는 키를 저장합니다.
