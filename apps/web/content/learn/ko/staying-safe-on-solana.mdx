# 솔라나에서 안전하게 지내기

솔라나를 사용할 때 보안은 매우 중요합니다. 자신의 자산을 직접 관리하기 때문에
그것을 안전하게 지키는 것은 여러분의 책임입니다. 이 가이드는 지갑 보호부터
일반적인 사기 방지까지 모든 사용자가 알아야 할 필수 보안 관행을 다룹니다.

## 배우게 될 내용

이 장을 마치면 다음 내용을 이해하게 됩니다:

- 지갑과 seed 구문을 보호하는 방법
- 일반적인 사기와 이를 피하는 방법
- 거래 안전을 위한 모범 사례
- 보안 도구와 그 용도
- 문제가 발생했을 때 대처 방법

## 황금 규칙: 시드 구문

{/* Illustration recommendation: Visual showing seed phrase security best practices - physical storage methods (safe, paper) with red X marks over digital storage (screenshots, cloud, email) */}

시드 구문은 보호해야 할 가장 중요한 것입니다. 이는 어떤 기기에서든 지갑을 복원할
수 있는 12-24개의 단어 세트입니다. 다음과 같은 마스터 키라고 생각하세요:

- 절대 변경할 수 없음
- 지갑에 완전한 접근 권한을 제공
- 절대 누구와도 공유해서는 안 됨

### 시드 구문 보호하기

**해야 할 일:**

- 종이에 적기(절대 타이핑하지 말 것)
- 안전한 장소에 보관하기
- 여러 장소에 여러 사본을 고려하기
- 가능하면 내화성 금고 사용하기

**하지 말아야 할 일:**

- 스크린샷이나 사진 찍기
- 이메일이나 클라우드 서비스에 저장하기
- 누구와도 공유하기(예외 없음)
- 웹사이트에 입력하기

## 피해야 할 일반적인 사기 수법

### 가짜 지원팀

사기꾼들은 종종 지원 담당자인 척합니다:

- 진짜 지원팀은 절대 먼저 DM을 보내지 않습니다
- 절대 시드 구문을 요구하지 않습니다
- 항상 공식 채널을 통해 확인하세요
- 요청하지 않은 도움은 의심하세요

### 피싱 웹사이트

정보를 훔치는 가짜 사이트:

- 항상 URL을 주의 깊게 확인하세요
- 미묘한 철자 오류를 확인하세요
- 실제 사이트는 북마크하세요
- 공식 링크만 사용하세요

### "공짜 돈" 사기

너무 좋아 보이는 것은 사실이 아닙니다:

- 정당한 에어드롭은 절대 시드 구문을 요구하지 않습니다
- 보장된 수익에 대해 의심하세요
- 참여하기 전에 조사하세요
- 새로운 것을 시도할 때는 소액으로 시작하세요

### 지갑 드레이너

지갑을 비우는 악성 앱:

- 검증된 애플리케이션에만 연결하세요
- 거래 요청을 주의 깊게 검토하세요
- 사용 후 지갑 연결을 해제하세요
- 다양한 활동에 별도의 지갑을 사용하세요

## 거래 안전

{/* Illustration recommendation: Transaction approval screen with callouts showing what to check - address, amount, what you're approving */}

거래를 승인하기 전에:

1. **수신자 주소 확인**
   - 첫 글자와 마지막 글자 확인
   - 큰 금액 이체 시 테스트 금액 보내기
   - 자주 사용하는 주소 저장하기

2. **서명하는 내용 이해하기**
   - 거래 세부 정보 읽기
   - 비정상적인 권한 확인하기
   - 혼란스러우면 승인하지 마세요

3. **출처 확인**
   - 본인이 시작한 작업인지 확인
   - 올바른 웹사이트에 있는지 확인
   - 새로운 앱에는 특히 주의하세요

## 보안 도구

### 핫 월렛 (브라우저/모바일)

**용도:** 일상적 사용, 소액

- 일반 거래에 편리함
- 무료로 쉽게 설정 가능
- 제한된 자금만 보관하세요

### 하드웨어 지갑

**최적 용도:** 장기 보관, 대량 자산

- 최대 보안(오프라인 저장)
- 가격 $50-200
- 상당한 자산 보유 시 가치 있음

### 멀티시그 지갑

**최적 용도:** 공유 자금, 추가 보안

- 다중 승인 필요
- 조직에 적합
- 단일 실패 지점 방지

## 모범 사례

### 여러 지갑 사용하기

- **메인 지갑**: 가치 있는 자산용(하드웨어 지갑 사용)
- **일상 지갑**: 일반 거래용
- **테스트 지갑**: 새로운 애플리케이션 시도용

### 소액으로 시작하기

- 새로운 앱은 아주 작은 금액으로 테스트
- 자신감을 얻으면서 점진적으로 금액 증가
- 잃어도 괜찮은 금액만 투자하기

### 정보 유지하기

- 공식 프로젝트 계정 팔로우
- 위험에 대해 배우기 위해 커뮤니티 가입
- 보안 업데이트 유지
- 다른 사람들의 경험에서 배우기

## 주의해야 할 위험 신호

{/* Illustration recommendation: Warning signs checklist */}

다음과 같은 상황에서는 매우 주의하세요:

- 빠른 행동을 촉구하는 압박
- seed 구문 요청
- 보장된 수익 약속
- 어색한 한국어나 문법 오류
- 확인할 수 없는 팀원
- 믿기 힘들 정도로 좋은 제안

## 문제가 발생했을 때

### 즉각적인 조치

1. **당황하지 마세요** - 신속하지만 신중하게 행동하세요
2. **남은 자산 이동** - 새 지갑으로 전송하세요
3. **모든 것을 기록** - 스크린샷, 주소, 금액
4. **사고 보고** - 커뮤니티에 알리세요
5. **교훈 얻기** - 무슨 일이 있었는지 이해하세요

### 도움을 받을 수 있는 곳

- 공식 프로젝트 디스코드/텔레그램
- Solana 재단 리소스
- 커뮤니티 보안 채널
- 복구를 제안하는 DM은 절대 신뢰하지 마세요

## 좋은 습관 기르기

### 정기 보안 점검

- 매월 연결된 앱 검토
- 지갑 소프트웨어 업데이트
- 보안 알림 확인
- 보안 습관 점검

### 모든 거래 전에

- 주소 재확인
- 금액 확인
- 작업 내용 이해
- 직감 신뢰하기

### 커뮤니티 안전

- 지식 공유(개인 정보는 제외)
- 발견한 사기 신고
- 초보자들의 안전한 학습 돕기
- 보안 이니셔티브 지원

## 자주 묻는 질문

### seed 구문을 잃어버리면 어떻게 되나요?

seed 구문이 없으면 지갑 복구가 불가능합니다. 이것이 안전한 백업이 매우 중요한
이유입니다.

### 거래를 취소할 수 있나요?

아니요. 모든 Solana 거래는 영구적입니다. 항상 확인 전에 다시 한번 확인하세요.

### 내 지갑 주소를 공유해도 안전한가요?

네, 공개 지갑 주소는 자금을 받기 위해 공유해도 안전합니다. 절대로 seed 구문은
공유하지 마세요.

### 앱이 정상적인지 어떻게 알 수 있나요?

공식 웹사이트, 인증된 소셜 계정, 감사 보고서 및 커뮤니티 평판을 확인하세요.

## 보안 체크리스트

- [ ] seed 구문을 오프라인에 안전하게 보관
- [ ] 큰 금액에는 하드웨어 지갑 사용
- [ ] 다양한 목적에 맞는 다른 지갑 사용
- [ ] 공식 웹사이트 북마크
- [ ] 거래 확인 방법 숙지
- [ ] 일반적인 사기 수법 이해
- [ ] 문제 발생 시 대처 계획 마련

## 다음 단계

처음에는 보안이 부담스럽게 느껴질 수 있지만, 연습을 통해 자연스러워집니다.
기본부터 시작하세요 - seed 구문을 안전하게 보관하고 새로운 애플리케이션에
주의하세요. 경험이 쌓이면서 좋은 보안 직감을 기르게 될 것입니다.

기억하세요: 솔라나에서 안전하게 활동하는 것은 과도한 불안감이 아닌 정보를 갖추고
주의하는 것입니다. 합리적인 예방 조치를 취하고, 지속적으로 학습하며, 생태계를
안전하게 탐험하세요.
