# Τι είναι ένα Πορτοφόλι;

Ένα πορτοφόλι είναι το κύριο εργαλείο σας για τη χρήση του Solana. Αποθηκεύει τα
ψηφιακά σας περιουσιακά στοιχεία, σας επιτρέπει να συνδέεστε με εφαρμογές και
σας δίνει τη δυνατότητα να στέλνετε και να λαμβάνετε tokens. Αυτό το κεφάλαιο θα
σας βοηθήσει να κατανοήσετε τα πορτοφόλια και να δημιουργήσετε το πρώτο σας με
ασφάλεια.

## Τι θα Μάθετε

Μέχρι το τέλος αυτού του κεφαλαίου, θα κατανοήσετε:

- Τι κάνουν τα πορτοφόλια και πώς λειτουργούν
- Διαφορετικούς τύπους πορτοφολιών και τα πλεονεκτήματα/μειονεκτήματά τους
- Πώς να δημιουργήσετε το πρώτο σας πορτοφόλι
- Βέλτιστες πρακτικές ασφαλείας
- Συνήθεις λειτουργίες και χρήσεις πορτοφολιών

## Κατανόηση των Πορτοφολιών

{/* Illustration recommendation: Simple diagram showing a wallet as a container for keys, tokens, and NFTs, with arrows showing interactions with various applications */}

Ένα πορτοφόλι εξυπηρετεί διάφορες λειτουργίες:

- **Αποθηκεύει τα κλειδιά σας**: Τα κρυπτογραφικά κλειδιά που αποδεικνύουν ότι
  κατέχετε τα περιουσιακά σας στοιχεία
- **Διατηρεί τα περιουσιακά σας στοιχεία**: Εμφανίζει τα tokens, NFTs και άλλα
  ψηφιακά αντικείμενά σας
- **Υπογράφει συναλλαγές**: Εγκρίνει ενέργειες για λογαριασμό σας
- **Συνδέεται με εφαρμογές**: Η μέθοδος σύνδεσής σας για εφαρμογές Solana

Σκεφτείτε το ως έναν συνδυασμό από:

- Έναν διαχειριστή κωδικών πρόσβασης (αποθηκεύει τα κλειδιά σας)
- Μια προβολή τραπεζικού λογαριασμού (δείχνει τα υπόλοιπά σας)
- Ένα εργαλείο ψηφιακής υπογραφής (εγκρίνει συναλλαγές)

## Πώς Λειτουργούν τα Πορτοφόλια

Κάθε πορτοφόλι βασίζεται σε κρυπτογραφικά κλειδιά:

### Seed Phrase

Μια σειρά από 12-24 λέξεις που μπορούν να αναδημιουργήσουν το πορτοφόλι σας.
Αυτό είναι το απόλυτο αντίγραφο ασφαλείας σας - κρατήστε το ασφαλές και
ιδιωτικό.

### Ιδιωτικό Κλειδί

Προέρχεται από τη φράση seed σας, αυτό το κλειδί υπογράφει συναλλαγές. Το
πορτοφόλι σας το διαχειρίζεται για εσάς - δεν θα χρειαστεί ποτέ να το δείτε ή να
το χειριστείτε άμεσα.

### Δημόσιο Κλειδί/Διεύθυνση

Προέρχεται από το ιδιωτικό σας κλειδί, αυτό είναι που μοιράζεστε για να λάβετε
χρήματα. Είναι ασφαλές να το μοιράζεστε δημόσια.

## Τύποι Πορτοφολιών

### Πορτοφόλια Λογισμικού

**Επεκτάσεις Προγράμματος Περιήγησης**

- Βολικά για διαδικτυακές εφαρμογές
- Εύκολα στη χρήση
- Καλά για καθημερινές συναλλαγές
- Δωρεάν χρήση

**Εφαρμογές Κινητών**

- Πρόσβαση στο πορτοφόλι σας από παντού
- Σάρωση κωδικών QR για πληρωμές
- Βιομετρική ασφάλεια (δακτυλικό αποτύπωμα/πρόσωπο)
- Δωρεάν χρήση

**Εφαρμογές Υπολογιστή**

- Διεπαφές με πλήρεις λειτουργίες
- Καλές για ενεργούς traders
- Προηγμένη λειτουργικότητα
- Δωρεάν χρήση

### Πορτοφόλια Υλικού

**Φυσικές Συσκευές**

- Μέγιστη ασφάλεια
- Αποθήκευση κλειδιών εκτός σύνδεσης
- Καλύτερα για μεγάλα ποσά
- Κόστος: 50-200€

### Επιλέγοντας το Πρώτο σας Πορτοφόλι

Για αρχάριους, συνιστούμε να ξεκινήσετε με ένα πορτοφόλι επέκτασης προγράμματος
περιήγησης. Είναι:

- Φιλικά προς το χρήστη
- Με καλή υποστήριξη
- Ασφαλή για καθημερινή χρήση
- Εύκολα στην εγκατάσταση

## Δημιουργώντας το Πρώτο σας Πορτοφόλι

{/* Illustration recommendation: Step-by-step visual guide showing the wallet creation process */}

Ας δούμε τη γενική διαδικασία δημιουργίας ενός πορτοφολιού:

### Βήμα 1: Επιλέξτε και Εγκαταστήστε ένα Πορτοφόλι

1. Ερευνήστε δημοφιλή πορτοφόλια Solana
2. Επισκεφθείτε την επίσημη ιστοσελίδα (πάντα να επαληθεύετε τα URLs)
3. Κατεβάστε το πορτοφόλι για την πλατφόρμα σας
4. Εγκαταστήστε ακολουθώντας τις επίσημες οδηγίες

### Βήμα 2: Δημιουργήστε ένα Νέο Πορτοφόλι

1. Ανοίξτε την εφαρμογή του πορτοφολιού σας
2. Επιλέξτε "Δημιουργία νέου πορτοφολιού"
3. Δημιουργήστε έναν ισχυρό κωδικό πρόσβασης για την πρόσβαση στη συσκευή
4. **Σημαντικό**: Καταγράψτε τη φράση seed σας

### Βήμα 3: Ασφαλίστε το Seed Phrase σας

{/* Illustration recommendation: Do's and don'ts for seed phrase storage - physical paper in safe (✓), screenshot (✗), cloud storage (✗) */}

Το seed phrase σας είναι κρίσιμο. Δείτε πώς να το αποθηκεύσετε με ασφάλεια:

**ΚΑΝΕΤΕ:**

- Γράψτε το σε χαρτί
- Αποθηκεύστε το σε ασφαλές μέρος
- Εξετάστε το ενδεχόμενο πολλαπλών αντιγράφων σε διαφορετικά μέρη
- Χρησιμοποιήστε πυρίμαχο χρηματοκιβώτιο αν είναι δυνατόν

**ΜΗΝ:**

- Βγάζετε στιγμιότυπο οθόνης
- Το αποθηκεύετε σε email ή υπηρεσίες cloud
- Το μοιράζεστε με κανέναν
- Το πληκτρολογείτε σε τυχαίους ιστότοπους

### Βήμα 4: Επαλήθευση και ολοκλήρωση

1. Επιβεβαιώστε το seed phrase σας όταν σας ζητηθεί
2. Το πορτοφόλι σας είναι τώρα έτοιμο για χρήση
3. Θα δείτε τη διεύθυνση του πορτοφολιού σας στο πάνω μέρος
4. Το αρχικό σας υπόλοιπο θα είναι 0 SOL

## Βασικές λειτουργίες πορτοφολιού

Τα σύγχρονα πορτοφόλια περιλαμβάνουν:

### Διαχείριση περιουσιακών στοιχείων

- Προβολή υπολοίπων token
- Εμφάνιση συλλογών NFT
- Παρακολούθηση ιστορικού συναλλαγών
- Παρακολούθηση αξίας χαρτοφυλακίου

### Εργαλεία συναλλαγών

- Αποστολή και λήψη token
- Ανταλλαγή μεταξύ διαφορετικών token
- Σύνδεση με εφαρμογές
- Έγκριση ή απόρριψη συναλλαγών

### Χαρακτηριστικά ασφαλείας

- Προστασία με κωδικό πρόσβασης
- Βιομετρική ταυτοποίηση (κινητό)
- Προειδοποιήσεις συναλλαγών
- Ανίχνευση phishing

## Τα πρώτα σας βήματα

Τώρα που έχετε ένα πορτοφόλι, δείτε τι πρέπει να κάνετε στη συνέχεια:

1. **Αποθηκεύστε τη διεύθυνσή σας**: Αντιγράψτε την κάπου με ασφάλεια για
   αναφορά
2. **Αποκτήστε λίγα SOL**: Θα χρειαστείτε ένα μικρό ποσό για τα τέλη συναλλαγών
3. **Εξασκηθείτε στη λήψη**: Ζητήστε από έναν φίλο να σας στείλει ένα μικρό ποσό
4. **Εξερευνήστε το περιβάλλον**: Εξοικειωθείτε με τη διάταξη

## Σημαντικές υπενθυμίσεις ασφαλείας

- **Ποτέ μην μοιράζεστε το seed phrase σας**: Καμία νόμιμη υπηρεσία δεν θα σας
  το ζητήσει
- **Επαληθεύστε τους ιστότοπους**: Οι απατεώνες δημιουργούν ψεύτικους ιστότοπους
  πορτοφολιών
- **Ξεκινήστε με μικρά ποσά**: Χρησιμοποιήστε μικρά ποσά κατά τη διάρκεια της
  εκμάθησης
- **Διατηρείτε το λογισμικό ενημερωμένο**: Εγκαταστήστε άμεσα τις ενημερώσεις
  του πορτοφολιού
- **Να είστε σκεπτικοί**: Αν κάτι φαίνεται πολύ καλό για να είναι αληθινό,
  πιθανότατα δεν είναι

## Συχνές ερωτήσεις

### Χρειάζεται να πληρώσω για ένα πορτοφόλι;

Τα περισσότερα ψηφιακά πορτοφόλια είναι δωρεάν. Πληρώνετε μόνο για πορτοφόλια
υλικού (hardware wallets) ή για premium λειτουργίες.

### Μπορώ να έχω πολλαπλά πορτοφόλια;

Ναι, πολλοί χρήστες έχουν διάφορα πορτοφόλια για διαφορετικούς σκοπούς
(καθημερινή χρήση, αποταμίευση, συγκεκριμένες εφαρμογές).

### Τι γίνεται αν ξεχάσω τον κωδικό μου;

Ο κωδικός της συσκευής σας μπορεί να επαναφερθεί με το seed phrase σας. Γι' αυτό
το seed phrase είναι τόσο σημαντικό.

### Αποθηκεύονται τα περιουσιακά μου στοιχεία στο πορτοφόλι;

Όχι, τα περιουσιακά στοιχεία αποθηκεύονται στο Solana. Το πορτοφόλι σας
αποθηκεύει τα κλειδιά που αποδεικνύουν την ιδιοκτησία.

## Τι ακολουθεί

Με το πορτοφόλι σας ρυθμισμένο, είστε έτοιμοι να αρχίσετε να χρησιμοποιείτε το
Solana. Στο επόμενο κεφάλαιο, θα εξερευνήσουμε τα τέλη συναλλαγών και γιατί
είναι σημαντικά για την προσβασιμότητα. Θα μάθετε γιατί τα χαμηλά τέλη του
Solana επιτρέπουν περιπτώσεις χρήσης που δεν είναι πρακτικές σε άλλα δίκτυα.

Να θυμάστε: όλοι ήταν κάποτε αρχάριοι. Αφιερώστε χρόνο για να εξοικειωθείτε με
το πορτοφόλι σας πριν μεταφέρετε μεγάλα ποσά. Η κοινότητα του Solana είναι εδώ
για να βοηθήσει αν έχετε απορίες.
