# Αποστολή και λήψη SOL

Η δυνατότητα αποστολής και λήψης SOL είναι η πρώτη σας πρακτική δεξιότητα στο
Solana. Το SOL είναι το εγγενές token του δικτύου Solana, που χρησιμοποιείται
για συναλλαγές, staking και αλληλεπίδραση με εφαρμογές. Σε αυτό το κεφάλαιο, θα
μάθετε πώς λειτουργούν οι μεταφορές και θα εξασκηθείτε κάνοντας την πρώτη σας
συναλλαγή.

## Τι θα μάθετε

Μέχρι το τέλος αυτού του κεφαλαίου, θα μπορείτε να:

- Κατανοήσετε τη διεύθυνση του πορτοφολιού σας και πώς να τη μοιράζεστε με
  ασφάλεια
- Στέλνετε SOL σε άλλο πορτοφόλι
- Λαμβάνετε SOL από άλλους
- Ελέγχετε την κατάσταση των συναλλαγών
- Κατανοήσετε τι συμβαίνει κατά τη διάρκεια μιας μεταφοράς

## Κατανόηση της διεύθυνσης του πορτοφολιού σας

Η διεύθυνση του πορτοφολιού σας είναι σαν τον αριθμό λογαριασμού σας στο Solana.
Είναι:

- **Δημόσια**: Ασφαλής για κοινοποίηση όταν λαμβάνετε χρήματα
- **Μοναδική**: Δεν υπάρχουν δύο πορτοφόλια με την ίδια διεύθυνση
- **Μόνιμη**: Δεν αλλάζει ούτε λήγει

Παράδειγμα διεύθυνσης: `7xKXtg2CW87d8V6zXKBq7cM8Y2FPHwzaAkxqEqD7x4rH`

### Κοινοποίηση της διεύθυνσής σας

Μπορείτε να μοιραστείτε τη διεύθυνσή σας μέσω:

- **Αντιγραφής/Επικόλλησης**: Κάντε κλικ στο κουμπί αντιγραφής στο πορτοφόλι σας
- **Κωδικού QR**: Επιτρέψτε σε άλλους να τον σαρώσουν με το τηλέφωνό τους
- **Ονομάτων ENS/Domain**: Ονόματα αναγνώσιμα από ανθρώπους (όπως
  `yourname.sol`)

## Λήψη SOL

{/* Illustration recommendation: Step-by-step visual showing someone sharing their address and seeing SOL arrive in their wallet */}

Η λήψη SOL είναι απλή:

1. **Ανοίξτε το πορτοφόλι σας**
2. **Βρείτε τη διεύθυνσή σας** (συνήθως εμφανίζεται στο πάνω μέρος)
3. **Μοιραστείτε την** με τον αποστολέα
4. **Περιμένετε λίγα δευτερόλεπτα** για την ολοκλήρωση της συναλλαγής
5. **Δείτε το νέο υπόλοιπό σας** να ενημερώνεται αυτόματα

### Τι χρειάζονται οι αποστολείς

Όταν κάποιος σας στέλνει SOL, χρειάζεται μόνο:

- Τη διεύθυνση του πορτοφολιού σας
- Το ποσό που θέλει να στείλει

Δεν χρειάζεται το όνομά σας, την τοποθεσία σας ή οποιαδήποτε προσωπική
πληροφορία.

## Αποστολή SOL

{/* Illustration recommendation: Wallet interface mockup showing the send flow with important elements highlighted */}

Έτσι μπορείτε να στείλετε SOL σε κάποιον άλλο:

### Βήμα 1: Λάβετε τη διεύθυνση του παραλήπτη

- Ζητήστε τους να μοιραστούν τη διεύθυνση του πορτοφολιού Solana
- Ελέγξτε διπλά ότι έχετε τη σωστή διεύθυνση
- Θυμηθείτε: οι συναλλαγές δεν μπορούν να αντιστραφούν

### Βήμα 2: Ξεκινήστε τη μεταφορά

1. Κάντε κλικ στο "Αποστολή" στο πορτοφόλι σας
2. Επικολλήστε τη διεύθυνση του παραλήπτη
3. Εισαγάγετε το ποσό του SOL
4. Ελέγξτε το τέλος συναλλαγής (συνήθως < $0.001)

### Βήμα 3: Επιβεβαιώστε και στείλτε

1. Ελέγξτε προσεκτικά όλες τις λεπτομέρειες
2. Επιβεβαιώστε τη συναλλαγή
3. Περιμένετε 1-2 δευτερόλεπτα για την ολοκλήρωση
4. Αποθηκεύστε το αναγνωριστικό συναλλαγής για τα αρχεία σας

## Κατανόηση των λεπτομερειών συναλλαγής

Κάθε συναλλαγή δημιουργεί μια εγγραφή που περιέχει:

- **Υπογραφή**: Μοναδικό αναγνωριστικό συναλλαγής
- **Από/Προς**: Διευθύνσεις αποστολέα και παραλήπτη
- **Ποσό**: Πόσο SOL μεταφέρθηκε
- **Τέλος**: Τέλος δικτύου που καταβλήθηκε (σε SOL)
- **Χρονική σήμανση**: Πότε συνέβη
- **Κατάσταση**: Επιτυχία ή αποτυχία

Μπορείτε να δείτε οποιαδήποτε συναλλαγή σε έναν εξερευνητή μπλοκ όπως το Solscan
ή το Solana Explorer.

## Συχνές ερωτήσεις

### Πόσο χρόνο διαρκούν οι μεταφορές;

Συνήθως 1-2 δευτερόλεπτα. Η Solana επεξεργάζεται τις συναλλαγές πολύ γρήγορα σε
σύγκριση με τις παραδοσιακές τραπεζικές συναλλαγές.

### Τι γίνεται αν στείλω σε λάθος διεύθυνση;

Οι συναλλαγές είναι μόνιμες και δεν μπορούν να αντιστραφούν. Πάντα να ελέγχετε
διπλά τις διευθύνσεις πριν την αποστολή.

### Γιατί χρειάζομαι SOL για τέλη;

Κάθε συναλλαγή απαιτεί ένα μικρό τέλος (συνήθως $0.00025) για να αποζημιώσει
τους validators του δικτύου που την επεξεργάζονται.

### Μπορώ να στείλω άλλα tokens εκτός από SOL;

Ναι, αλλά θα χρειάζεστε πάντα μια μικρή ποσότητα SOL για τέλη συναλλαγών, ακόμα
και όταν στέλνετε άλλα tokens.

## Πρακτική Άσκηση

Έτοιμοι να δοκιμάσετε την πρώτη σας συναλλαγή; Ορίστε ένας ασφαλής τρόπος
εξάσκησης:

1. **Στείλτε ένα μικρό ποσό** (όπως 0.001 SOL) σε έναν φίλο
2. **Ζητήστε τους να σας το στείλουν πίσω**
3. **Αναζητήστε τη συναλλαγή** σε έναν εξερευνητή μπλοκ
4. **Σημειώστε το τέλος** και τον χρόνο επιβεβαίωσης

Αυτό σας βοηθά να εξοικειωθείτε με τη διαδικασία χρησιμοποιώντας ελάχιστα ποσά.

## Βέλτιστες Πρακτικές Συναλλαγών

- **Ξεκινήστε με μικρά ποσά** όταν στέλνετε σε μια νέα διεύθυνση
- **Αποθηκεύστε σημαντικές διευθύνσεις** στο βιβλίο διευθύνσεων του πορτοφολιού
  σας
- **Κρατήστε αρχείο συναλλαγών** για φορολογικούς ή προσωπικούς λόγους
- **Ελέγξτε διπλά τις διευθύνσεις** - ειδικά τους πρώτους και τους τελευταίους 4
  χαρακτήρες
- **Να γνωρίζετε την τρέχουσα τιμή του SOL** όταν κάνετε αποστολή

## Τι Ακολουθεί

Τώρα που μπορείτε να στέλνετε και να λαμβάνετε SOL, είστε έτοιμοι να
εξερευνήσετε πιο προηγμένες λειτουργίες. Στο επόμενο κεφάλαιο, θα μάθουμε για το
staking - έναν τρόπο να κερδίζετε ανταμοιβές βοηθώντας στην ασφάλεια του δικτύου
Solana.

Θυμηθείτε: κάθε ειδικός ήταν κάποτε αρχάριος. Πάρτε τον χρόνο σας, ξεκινήστε με
μικρά ποσά και χτίστε την αυτοπεποίθησή σας μέσω της πρακτικής.
