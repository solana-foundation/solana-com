# Что такое кошелек?

Кошелек — это ваш основной инструмент для работы с Solana. Он хранит ваши
цифровые активы, позволяет подключаться к приложениям и дает возможность
отправлять и получать токены. В этой главе вы узнаете, что такое кошельки и как
безопасно настроить свой первый кошелек.

## Чему вы научитесь

К концу этой главы вы будете понимать:

- Что делают кошельки и как они работают
- Различные типы кошельков и их особенности
- Как создать свой первый кошелек
- Лучшие практики безопасности
- Основные функции и использование кошельков

## Понимание кошельков

{/* Рекомендация для иллюстрации: Простой диаграмма, показывающая кошелек как контейнер для ключей, токенов и NFT, со стрелками, указывающими на взаимодействие с различными приложениями */}

Кошелек выполняет несколько функций:

- **Хранит ваши ключи**: Криптографические ключи, которые доказывают, что вы
  владеете своими активами
- **Хранит ваши активы**: Отображает ваши токены, NFT и другие цифровые предметы
- **Подписывает транзакции**: Одобряет действия от вашего имени
- **Подключается к приложениям**: Ваш метод входа в приложения Solana

Думайте о нем как о сочетании:

- Менеджера паролей (хранит ваши ключи)
- Просмотрщика банковского счета (показывает ваши балансы)
- Инструмента цифровой подписи (одобряет транзакции)

## Как работают кошельки

Каждый кошелек основан на криптографических ключах:

### Сид-фраза

Серия из 12-24 слов, которые могут восстановить ваш кошелек. Это ваш главный
резервный инструмент — храните его в безопасности и конфиденциальности.

### Приватный ключ

Этот ключ, который используется для подписания транзакций, создаётся на основе
вашей seed-фразы. Ваш кошелёк управляет этим за вас — вам никогда не придётся
видеть или использовать его напрямую.

### Публичный ключ/адрес

Создаётся на основе вашего приватного ключа. Это то, что вы делитесь, чтобы
получать средства. Делиться им безопасно.

## Типы кошельков

### Программные кошельки

**Расширения для браузера**

- Удобны для веб-приложений
- Просты в использовании
- Подходят для ежедневных транзакций
- Бесплатны в использовании

**Мобильные приложения**

- Доступ к кошельку в любом месте
- Сканирование QR-кодов для платежей
- Биометрическая защита (отпечаток пальца/лицо)
- Бесплатны в использовании

**Приложения для компьютера**

- Полнофункциональные интерфейсы
- Подходят для активных трейдеров
- Расширенные функции
- Бесплатны в использовании

### Аппаратные кошельки

**Физические устройства**

- Максимальная безопасность
- Офлайн-хранение ключей
- Лучший выбор для крупных активов
- Стоимость: $50-200

### Выбор первого кошелька

Для начинающих мы рекомендуем начать с кошелька в виде расширения для браузера.
Они:

- Удобны для пользователя
- Хорошо поддерживаются
- Безопасны для повседневного использования
- Легко настраиваются

## Создание первого кошелька

{/* Рекомендация для иллюстрации: пошаговое визуальное руководство по процессу создания кошелька */}

Давайте рассмотрим общий процесс настройки кошелька:

### Шаг 1: Выберите и установите кошелёк

1. Изучите популярные кошельки Solana
2. Посетите официальный сайт (всегда проверяйте URL-адреса)
3. Скачайте кошелёк для вашей платформы
4. Установите, следуя официальным инструкциям

### Шаг 2: Создайте новый кошелёк

1. Откройте приложение кошелька
2. Выберите опцию "Создать новый кошелёк"
3. Создайте надёжный пароль для доступа к устройству
4. **Важно**: Запишите вашу seed-фразу

### Шаг 3: Защитите свою seed-фразу

{/* Рекомендация для иллюстрации: что можно и нельзя делать для хранения seed-фразы — физическая бумага в сейфе (✓), скриншот (✗), облачное хранилище (✗) */}

Ваша seed-фраза крайне важна. Вот как её безопасно хранить:

**МОЖНО:**

- Записать на бумаге
- Хранить в безопасном месте
- Сделать несколько копий и хранить их в разных местах
- Использовать огнеупорный сейф, если возможно

**НЕЛЬЗЯ:**

- Делать скриншоты
- Хранить в электронной почте или облачных сервисах
- Делиться с кем-либо
- Вводить на случайных сайтах

### Шаг 4: Подтвердите и завершите

1. Подтвердите свою seed-фразу, когда будет предложено
2. Ваш кошелёк готов к использованию
3. Ваш адрес кошелька будет отображаться вверху
4. Ваш начальный баланс составит 0 SOL

## Основные функции кошелька

Современные кошельки включают:

### Управление активами

- Просмотр баланса токенов
- Отображение коллекций NFT
- Отслеживание истории транзакций
- Мониторинг стоимости портфеля

### Инструменты для транзакций

- Отправка и получение токенов
- Обмен между разными токенами
- Подключение к приложениям
- Подтверждение или отклонение транзакций

### Функции безопасности

- Защита паролем
- Биометрическая аутентификация (мобильные устройства)
- Предупреждения о транзакциях
- Обнаружение фишинга

## Ваши первые шаги

Теперь, когда у вас есть кошелёк, вот что делать дальше:

1. **Сохраните свой адрес**: Скопируйте его в безопасное место для справки
2. **Получите немного SOL**: Вам понадобится небольшая сумма для оплаты комиссий
   за транзакции
3. **Попробуйте получить перевод**: Попросите друга отправить вам небольшую
   сумму
4. **Изучите интерфейс**: Ознакомьтесь с расположением элементов

## Важные напоминания о безопасности

- **Никогда не делитесь своей seed-фразой**: Ни один легитимный сервис не
  попросит её
- **Проверяйте сайты**: Мошенники создают поддельные сайты кошельков
- **Начинайте с малого**: Используйте небольшие суммы, пока учитесь
- **Обновляйте программное обеспечение**: Устанавливайте обновления кошелька
  своевременно
- **Будьте скептичны**: Если что-то кажется слишком хорошим, чтобы быть правдой,
  скорее всего, так и есть

## Часто задаваемые вопросы

### Нужно ли платить за кошелек?

Большинство программных кошельков бесплатны. Вы платите только за аппаратные
кошельки или премиум-функции.

### Могу ли я иметь несколько кошельков?

Да, многие пользователи имеют несколько кошельков для разных целей (ежедневное
использование, сбережения, конкретные приложения).

### Что делать, если я забуду свой пароль?

Пароль устройства можно сбросить с помощью вашей seed-фразы. Именно поэтому
seed-фраза так важна.

### Хранятся ли мои активы в кошельке?

Нет, активы хранятся на Solana. Ваш кошелек хранит ключи, которые подтверждают
право собственности.
