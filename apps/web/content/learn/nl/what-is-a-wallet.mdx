# Wat is een wallet?

Een wallet is je belangrijkste hulpmiddel voor het gebruik van Solana. Het
bewaart je digitale bezittingen, laat je verbinding maken met applicaties en
stelt je in staat om tokens te verzenden en te ontvangen. Dit hoofdstuk helpt je
wallets te begrijpen en je eerste wallet veilig in te stellen.

## Wat je gaat leren

Aan het einde van dit hoofdstuk begrijp je:

- Wat wallets doen en hoe ze werken
- Verschillende soorten wallets en hun voor- en nadelen
- Hoe je je eerste wallet aanmaakt

- Veelvoorkomende walletfuncties en -toepassingen

## Wallets begrijpen

{/* Illustration recommendation: Simple diagram showing a wallet as a container for keys, tokens, and NFTs, with arrows showing interactions with various applications */}

Een wallet heeft verschillende functies:

- **Bewaart je sleutels**: De cryptografische sleutels die bewijzen dat je
  eigenaar bent van je bezittingen
- **Bewaart je bezittingen**: Toont je tokens, NFT's en andere digitale items
- **Ondertekent transacties**: Keurt acties namens jou goed
- **Maakt verbinding met apps**: Je inlogmethode voor Solana-applicaties

Zie het als een combinatie van:

- Een wachtwoordbeheerder (bewaart je sleutels)
- Een bankrekeningviewer (toont je saldo's)
- Een digitale handtekeningtool (keurt transacties goed)

## Hoe wallets werken

Elke wallet is gebouwd op cryptografische sleutels:

### Seed phrase

Een reeks van 12-24 woorden waarmee je wallet kan worden hersteld. Dit is je
ultieme back-up - houd het veilig en privé.

### Private key

Afgeleid van je seed phrase, deze sleutel ondertekent transacties. Je wallet
beheert dit voor je - je hoeft deze nooit direct te zien of te gebruiken.

### Public key/adres

Afgeleid van je private key, dit is wat je deelt om geld te ontvangen. Het is
veilig om dit publiekelijk te delen.

## Soorten wallets

### Software wallets

**Browser-extensies**

- Handig voor webapplicaties
- Eenvoudig in gebruik
- Goed voor dagelijkse transacties
- Gratis te gebruiken

**Mobiele apps**

- Toegang tot je wallet overal
- QR-code scannen voor betalingen
- Biometrische beveiliging (vingerafdruk/gezicht)
- Gratis te gebruiken

**Desktop-applicaties**

- Volledig uitgeruste interfaces
- Goed voor actieve traders
- Geavanceerde functionaliteit
- Gratis te gebruiken

### Hardware wallets

**Fysieke apparaten**

- Maximale beveiliging
- Offline sleutelopslag
- Beste voor grote tegoeden
- Kosten: €50-200

### Je eerste wallet kiezen

Voor beginners raden we aan om te starten met een browser-extensie wallet. Deze
zijn:

- Gebruiksvriendelijk
- Goed ondersteund
- Veilig voor dagelijks gebruik
- Eenvoudig in te stellen

## Je eerste wallet aanmaken

{/* Illustration recommendation: Step-by-step visual guide showing the wallet creation process */}

Laten we het algemene proces van het opzetten van een wallet doorlopen:

### Stap 1: Kies en installeer een wallet

1. Onderzoek populaire Solana wallets
2. Bezoek de officiële website (verifieer altijd URL's)
3. Download de wallet voor jouw platform
4. Installeer volgens de officiële instructies

### Stap 2: Maak een nieuwe wallet aan

1. Open je wallet-applicatie
2. Selecteer de optie "Nieuwe wallet aanmaken"
3. Maak een sterk wachtwoord voor toegang tot het apparaat
4. **Belangrijk**: Schrijf je seed phrase op

### Stap 3: Beveilig je seed phrase

{/* Illustration recommendation: Do's and don'ts for seed phrase storage - physical paper in safe (✓), screenshot (✗), cloud storage (✗) */}

Je seed phrase is essentieel. Zo bewaar je het veilig:

**WEL DOEN:**

- Schrijf het op papier
- Bewaar het op een veilige plek
- Overweeg meerdere kopieën op verschillende plekken
- Gebruik indien mogelijk een brandwerende kluis

**NIET DOEN:**

- Er een screenshot van maken
- Opslaan in e-mail of clouddiensten
- Delen met anderen
- Intypen op willekeurige websites

### Stap 4: Verifiëren en afronden

1. Bevestig je seed phrase wanneer hierom wordt gevraagd
2. Je wallet is nu klaar voor gebruik
3. Je ziet je walletadres bovenaan
4. Je beginsaldo zal 0 SOL zijn

## Basisfuncties van de wallet

Moderne wallets bevatten:

### Activabeheer

- Bekijk tokensaldo's
- Toon NFT-collecties
- Volg transactiegeschiedenis
- Monitor portfoliowaarde

### Transactietools

- Tokens verzenden en ontvangen
- Wisselen tussen verschillende tokens
- Verbinding maken met applicaties
- Transacties goedkeuren of afwijzen

### Beveiligingsfuncties

- Wachtwoordbeveiliging
- Biometrische authenticatie (mobiel)
- Transactiewaarschuwingen
- Phishingdetectie

## Je eerste stappen

Nu je een wallet hebt, kun je het volgende doen:

1. **Bewaar je adres**: Kopieer het ergens veilig ter referentie
2. **Verkrijg wat SOL**: Je hebt een kleine hoeveelheid nodig voor
   transactiekosten
3. **Oefen met ontvangen**: Laat een vriend je een klein bedrag sturen
4. **Verken de interface**: Raak vertrouwd met de indeling

## Belangrijke beveiligingsherinneringen

- **Deel nooit je seed phrase**: Geen enkele legitieme dienst zal ernaar vragen
- **Verifieer websites**: Oplichters maken nepwallet-sites
- **Begin klein**: Gebruik kleine bedragen tijdens het leren
- **Houd software bijgewerkt**: Installeer wallet-updates direct
- **Wees sceptisch**: Als iets te mooi lijkt om waar te zijn, is dat
  waarschijnlijk ook zo

## Veelgestelde vragen

### Moet ik betalen voor een wallet?

De meeste software wallets zijn gratis. Je betaalt alleen voor hardware wallets
of premium functies.

### Kan ik meerdere wallets hebben?

Ja, veel gebruikers hebben verschillende wallets voor verschillende doeleinden
(dagelijks gebruik, sparen, specifieke toepassingen).

### Wat als ik mijn wachtwoord vergeet?

Je apparaatwachtwoord kan worden gereset met je seed phrase. Daarom is de seed
phrase zo belangrijk.

### Worden mijn assets in de wallet opgeslagen?

Nee, assets worden opgeslagen op Solana. Je wallet bewaart de sleutels die
eigendom bewijzen.
