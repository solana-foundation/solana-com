# Introduction aux tokens Solana

Les tokens sont des actifs numériques construits sur Solana. Tout comme le
dollar américain est la monnaie native des États-Unis, SOL est la monnaie native
de Solana. Les tokens représentent tous les autres actifs - des stablecoins
comme l'USDC aux tokens de projet et NFT. Ce chapitre explique comment
fonctionnent les tokens et ce que vous pouvez faire avec eux.

## Ce que vous allez apprendre

À la fin de ce chapitre, vous comprendrez :

- Ce que sont les tokens et comment ils fonctionnent sur Solana
- Les différents types de tokens et leurs utilisations
- Comment gérer les tokens dans votre portefeuille
- Les token accounts et les frais
- Les considérations de sécurité lors de la manipulation des tokens

## Comprendre les tokens sur Solana

Solana possède deux types principaux d'actifs numériques :

- **SOL** : La monnaie native utilisée pour les transactions et les frais
- **Tokens** : Tous les autres actifs numériques sur Solana

Tout comme les applications sur votre téléphone suivent certaines normes pour
fonctionner correctement, les tokens sur Solana suivent un standard commun qui
les rend compatibles avec les portefeuilles et les applications.

## Types de tokens

![Grille montrant quatre catégories de tokens : Stablecoins, Actifs enveloppés, Tokens de projet et Tokens utilitaires avec des exemples](/images/learn/types_of_tokens.webp)

### Stablecoins

Versions numériques des monnaies traditionnelles qui maintiennent une valeur
stable.

**Exemples :**

- **USDC** : Dollars américains numériques
- **USDT** : Tether, un autre stablecoin en USD
- **EURC** : Euros numériques

**Utilisations :** Épargne, paiements, trading, évitement de la volatilité

### Actifs enveloppés

Tokens représentant des actifs provenant d'autres réseaux.

**Exemples :**

- **Bitcoin enveloppé (BTC)** : Bitcoin utilisable sur Solana
- **Ethereum enveloppé (ETH)** : Ethereum sur Solana
- **SOL enveloppé** : Utilisé dans certains protocoles DeFi

**Utilisations :** Accédez à d'autres cryptomonnaies avec la rapidité et les
frais réduits de Solana

### Jetons de projet

Jetons créés par des applications et des protocoles.

**Exemples :**

- Jetons de gouvernance pour voter sur les décisions de protocole
- Jetons d'échange offrant des réductions de frais
- Jetons de protocole pour la participation à l'écosystème

**Utilisations :** Vote de gouvernance, réductions de frais, récompenses

### Jetons utilitaires

Jetons avec des fonctions spécifiques au sein des applications.

**Exemples :**

- Jetons de jeu pour les économies in-game
- Jetons sociaux pour les communautés de créateurs
- Jetons de récompense des protocoles

**Utilisations :** Accès aux fonctionnalités, gains de récompenses,
participation aux écosystèmes

## Comment fonctionnent les token accounts

![Diagramme de portefeuille montrant le compte SOL principal et des comptes de tokens séparés pour USDC et WBTC](/images/learn/how_token_accounts_work.webp)

Contrairement à certains réseaux où tous les actifs partagent une seule adresse,
Solana utilise un système de token account :

1. **Portefeuille principal** : Contient vos SOL
2. **Token accounts** : Comptes séparés pour chaque type de jeton

### Pourquoi des token accounts ?

- Meilleure performance et organisation
- Séparation claire des différents actifs
- Stockage réseau efficace

### Rent du compte

- Chaque token account nécessite un dépôt d'environ 0,002 SOL
- Ce "rent" est remboursable lors de la fermeture du compte
- Prévient le spam et finance le stockage réseau
- Coûte généralement entre 0,20 et 0,50 $ par type de jeton

## Gestion de vos jetons

### Réception de jetons

1. Partagez l'adresse de votre portefeuille
2. L'expéditeur initie le transfert
3. Le portefeuille crée un token account si nécessaire
4. Les jetons apparaissent dans votre portefeuille

### Visualisation de vos jetons

Les portefeuilles modernes affichent :

- Symboles et logos des jetons
- Soldes actuels
- Valeurs en USD
- Transactions récentes

### Envoi de tokens

1. Sélectionnez le token à envoyer
2. Saisissez l'adresse du destinataire
3. Choisissez le montant
4. Confirmez la transaction
5. Payez de petits frais en SOL (~0,0005 $)

### Fermeture des token accounts

Lorsque vous n'avez plus besoin d'un token :

1. Envoyez tous les tokens
2. Fermez le compte
3. Récupérez le dépôt de rent

## Principes de base de la création de tokens

Tout le monde peut créer des tokens sur Solana :

### Prérequis

- Une petite quantité de SOL pour les frais (~0,5 SOL)
- Informations de base sur le token (nom, symbole, offre)
- Optionnel : logo et métadonnées

### Utilisations courantes

- Tokens communautaires
- Systèmes de récompense
- Collecte de fonds
- Économies de jeux
- Projets expérimentaux

### Note importante

Créer un token ne lui donne pas de valeur. La valeur provient de l'utilité, de
l'adoption et des dynamiques du marché.

## Sécurité des tokens

![Liste de contrôle de sécurité des tokens montrant ce qu'il faut vérifier et ce dont il faut se méfier lors de la manipulation des tokens](/images/learn/token_safety.webp)

### Vérifiez avant de faire confiance

- Consultez les sites web officiels des projets
- Vérifiez les adresses des tokens
- Recherchez des rapports d'audit
- Renseignez-vous sur l'équipe

### Arnaques courantes à éviter

- **Faux tokens** : Usurpation de projets populaires
- **Rug pulls** : Créateurs abandonnant leurs projets
- **Pots de miel** : Tokens que vous pouvez acheter mais pas vendre
- **Pump and dumps** : Manipulation artificielle des prix

### Bonnes pratiques

1. **Faites des recherches approfondies** avant d'acheter
2. **Commencez petit** avec de nouveaux tokens
3. **Utilisez des sources fiables** pour les informations sur les tokens
4. **Soyez sceptique** face aux promesses de rendements garantis
5. **Vérifiez la liquidité** avant les achats importants

## Travailler avec des tokens

### Échange de tokens

Utilisez des agrégateurs DEX pour trouver les meilleurs taux :

1. Connectez votre portefeuille
2. Sélectionnez les tokens à échanger
3. Vérifiez le taux et les frais
4. Confirmez l'échange

### Vérification des informations sur les tokens

Ressources pour la recherche :

- Explorateurs de blocs : consultez les détails des tokens et leurs détenteurs
- Agrégateurs de prix : vérifiez les prix et la liquidité
- Sites officiels des projets : vérifiez l'authenticité

### Gestion de plusieurs tokens

- Organisez en masquant les soldes nuls
- Mettez en favori les tokens fréquemment utilisés
- Utilisez des trackers de portefeuille
- Fermez les token accounts inutilisés

## Questions fréquentes

### Ai-je besoin de SOL pour envoyer des tokens ?

Oui, toutes les transactions nécessitent de petits frais en SOL, même lors de
l'envoi d'autres tokens.

### Pourquoi certains tokens apparaissent-ils automatiquement ?

Les projets peuvent airdropper des tokens aux portefeuilles. Soyez prudent avec
les tokens non sollicités.

### Puis-je récupérer des tokens envoyés ?

Non, les transactions sur Solana sont permanentes. Vérifiez toujours les
adresses.

### Qu'est-ce qui détermine la valeur d'un token ?

L'offre et la demande, l'utilité, le développement du projet et le sentiment du
marché affectent tous le prix des tokens.
