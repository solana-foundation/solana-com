# Débuter avec Solana

Bienvenue sur Solana ! Ce guide vous aidera à comprendre les bases et comment
commencer à utiliser le réseau Solana.

## Qu'est-ce que Solana ?

![Diagramme isométrique montrant le réseau haute performance de Solana avec des nœuds interconnectés permettant des applications rapides et sécurisées](/images/learn/solana_network_isometric.webp)

Solana est un réseau haute performance qui permet des applications rapides,
sécurisées et évolutives.

## Pourquoi choisir Solana ?

### Vitesse et performance

![Visualisation de course montrant Solana traitant des milliers de transactions par seconde, dépassant les autres réseaux](/images/learn/speed_and_performance.webp)

- **Rapide** : Solana peut traiter des milliers de transactions par seconde
- **Efficace** : Le traitement parallèle assure un débit élevé

### Coûts réduits

- **Économique** : Les frais de transaction représentent une fraction de centime
- **Prévisible** : Les frais restent stables même pendant les périodes de forte
  activité

### Conçu pour l'évolutivité

- **Évolutif** : Construit pour croître avec l'adoption mondiale
- **Pérenne** : Conçu pour une utilisation grand public

## Premiers pas : vos premières étapes

![Parcours étape par étape montrant trois étapes : Configurez votre portefeuille, Obtenez votre premier SOL, Effectuez votre première transaction](/images/learn/getting_started.webp)

### 1. Configurez un portefeuille

Votre portefeuille est votre passerelle vers l'écosystème Solana. Choisissez
entre les extensions de navigateur, les applications mobiles ou les
portefeuilles matériels selon vos besoins.

### 2. Obtenez votre premier SOL

Vous aurez besoin de SOL pour payer les transactions. Commencez avec un petit
montant provenant d'une plateforme d'échange ou recevez-en de la part d'un ami.

### 3. Effectuez votre première transaction

Envoyez une petite quantité de SOL pour expérimenter par vous-même la rapidité
et le faible coût des transactions Solana.

## Et ensuite ?

Poursuivez votre parcours en explorant :

- La compréhension des différents types de portefeuilles
- L'apprentissage des frais de transaction
- La découverte des applications Solana

Restez à l'écoute pour des tutoriels plus détaillés !
