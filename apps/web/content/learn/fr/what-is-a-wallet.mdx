# Qu'est-ce qu'un portefeuille ?

Un portefeuille est votre outil principal pour utiliser Solana. Il stocke vos
actifs numériques, vous permet de vous connecter aux applications et vous permet
d'envoyer et de recevoir des jetons. Ce chapitre vous aidera à comprendre les
portefeuilles et à configurer votre premier portefeuille en toute sécurité.

## Ce que vous allez apprendre

À la fin de ce chapitre, vous comprendrez :

- Ce que font les portefeuilles et comment ils fonctionnent
- Les différents types de portefeuilles et leurs compromis
- Comment créer votre premier portefeuille
- Les meilleures pratiques de sécurité
- Les fonctionnalités et utilisations courantes des portefeuilles

## Comprendre les portefeuilles

{/* Illustration recommendation: Simple diagram showing a wallet as a container for keys, tokens, and NFTs, with arrows showing interactions with various applications */}

Un portefeuille remplit plusieurs fonctions :

- **Stocke vos clés** : Les clés cryptographiques qui prouvent que vous possédez
  vos actifs
- **Contient vos actifs** : Affiche vos jetons, NFT et autres éléments
  numériques
- **Signe des transactions** : Approuve des actions en votre nom
- **Se connecte aux applications** : Votre méthode de connexion pour les
  applications Solana

Considérez-le comme une combinaison de :

- Un gestionnaire de mots de passe (stocke vos clés)
- Un visualiseur de compte bancaire (affiche vos soldes)
- Un outil de signature numérique (approuve les transactions)

## Comment fonctionnent les portefeuilles

Chaque portefeuille est construit sur des clés cryptographiques :

### Phrase seed

Une série de 12 à 24 mots qui peut recréer votre portefeuille. C'est votre
sauvegarde ultime - gardez-la en sécurité et confidentielle.

### Clé privée

Dérivée de votre phrase de récupération, cette clé signe les transactions. Votre
portefeuille la gère pour vous - vous n'aurez jamais besoin de la voir ou de la
manipuler directement.

### Clé publique/Adresse

Dérivée de votre clé privée, c'est ce que vous partagez pour recevoir des fonds.
Il est sûr de la partager publiquement.

## Types de portefeuilles

### Portefeuilles logiciels

**Extensions de navigateur**

- Pratiques pour les applications web
- Faciles à utiliser
- Adaptés aux transactions quotidiennes
- Gratuits

**Applications mobiles**

- Accédez à votre portefeuille n'importe où
- Scan de codes QR pour les paiements
- Sécurité biométrique (empreinte digitale/visage)
- Gratuites

**Applications de bureau**

- Interfaces complètes
- Adaptées aux traders actifs
- Fonctionnalités avancées
- Gratuites

### Portefeuilles matériels

**Appareils physiques**

- Sécurité maximale
- Stockage de clés hors ligne
- Idéals pour les avoirs importants
- Coût : 50-200 €

### Choisir votre premier portefeuille

Pour les débutants, nous recommandons de commencer par un portefeuille en
extension de navigateur. Ils sont :

- Conviviaux
- Bien supportés
- Sécurisés pour une utilisation quotidienne
- Faciles à configurer

## Créer votre premier portefeuille

{/* Recommendation d'illustration : Guide visuel étape par étape montrant le processus de création de portefeuille */}

Parcourons le processus général de configuration d'un portefeuille :

### Étape 1 : Choisir et installer un portefeuille

1. Recherchez des portefeuilles Solana populaires
2. Visitez le site officiel (vérifiez toujours les URLs)
3. Téléchargez le portefeuille pour votre plateforme
4. Installez en suivant les instructions officielles

### Étape 2 : Créer un nouveau portefeuille

1. Ouvrez votre application de portefeuille
2. Sélectionnez l'option "Créer un nouveau portefeuille"
3. Créez un mot de passe fort pour l'accès à l'appareil
4. **Important** : Notez votre phrase de récupération

### Étape 3 : Sécurisez votre seed phrase

{/* Illustration recommendation: Do's and don'ts for seed phrase storage - physical paper in safe (✓), screenshot (✗), cloud storage (✗) */}

Votre seed phrase est essentielle. Voici comment la stocker en toute sécurité :

**À FAIRE :**

- L'écrire sur papier
- La conserver dans un endroit sécurisé
- Envisager plusieurs copies à différents endroits
- Utiliser un coffre-fort ignifuge si possible

**À NE PAS FAIRE :**

- Prendre une capture d'écran
- La stocker dans un email ou des services cloud
- La partager avec qui que ce soit
- La saisir sur des sites web aléatoires

### Étape 4 : Vérifier et terminer

1. Confirmez votre seed phrase lorsque demandé
2. Votre portefeuille est maintenant prêt à être utilisé
3. Vous verrez votre adresse de portefeuille en haut
4. Votre solde initial sera de 0 SOL

## Fonctionnalités de base du portefeuille

Les portefeuilles modernes incluent :

### Gestion des actifs

- Visualisation des soldes de tokens
- Affichage des collections de NFT
- Suivi de l'historique des transactions
- Surveillance de la valeur du portefeuille

### Outils de transaction

- Envoi et réception de tokens
- Échange entre différents tokens
- Connexion aux applications
- Approbation ou rejet des transactions

### Fonctionnalités de sécurité

- Protection par mot de passe
- Authentification biométrique (mobile)
- Avertissements de transaction
- Détection de phishing

## Vos premiers pas

Maintenant que vous avez un portefeuille, voici ce qu'il faut faire ensuite :

1. **Enregistrez votre adresse** : Copiez-la quelque part en lieu sûr pour
   référence
2. **Obtenez des SOL** : Vous aurez besoin d'une petite quantité pour les frais
   de transaction
3. **Entraînez-vous à recevoir** : Demandez à un ami de vous envoyer un petit
   montant
4. **Explorez l'interface** : Familiarisez-vous avec la disposition

## Rappels importants de sécurité

- **Ne partagez jamais votre seed phrase** : Aucun service légitime ne vous la
  demandera
- **Vérifiez les sites web** : Les escrocs créent de faux sites de portefeuilles
- **Commencez petit** : Utilisez de petits montants pendant l'apprentissage
- **Gardez vos logiciels à jour** : Installez rapidement les mises à jour du
  portefeuille
- **Soyez sceptique** : Si quelque chose semble trop beau pour être vrai, c'est
  probablement le cas

## Questions fréquentes

### Dois-je payer pour un portefeuille ?

La plupart des portefeuilles logiciels sont gratuits. Vous ne payez que pour les
portefeuilles matériels ou les fonctionnalités premium.

### Puis-je avoir plusieurs portefeuilles ?

Oui, de nombreux utilisateurs possèdent plusieurs portefeuilles pour différents
usages (utilisation quotidienne, épargne, applications spécifiques).

### Que faire si j'oublie mon mot de passe ?

Le mot de passe de votre appareil peut être réinitialisé avec votre seed phrase.
C'est pourquoi la seed phrase est si importante.

### Mes actifs sont-ils stockés dans le portefeuille ?

Non, les actifs sont stockés sur Solana. Votre portefeuille conserve les clés
qui prouvent la propriété.
