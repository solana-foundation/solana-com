# Apa itu Dompet?

Dompet adalah alat utama Anda untuk menggunakan Solana. Dompet menyimpan aset
digital Anda, memungkinkan Anda terhubung ke aplikasi, dan memungkinkan Anda
mengirim dan menerima token. Bab ini akan membantu Anda memahami dompet dan
mengatur dompet pertama Anda dengan aman.

## Apa yang Akan Anda Pelajari

Pada akhir bab ini, Anda akan memahami:

- Apa fungsi dompet dan bagaimana cara kerjanya
- Berbagai jenis dompet dan kelebihan serta kekurangannya
- Cara membuat dompet pertama Anda
- Praktik terbaik keamanan
- Fitur dan penggunaan dompet yang umum

## Memahami Dompet

{/* Illustration recommendation: Simple diagram showing a wallet as a container for keys, tokens, and NFTs, with arrows showing interactions with various applications */}

Dompet memiliki beberapa fungsi:

- **Menyimpan kunci Anda**: Kunci kriptografi yang membuktikan bahwa Anda
  memiliki aset Anda
- **Menyimpan aset Anda**: Menampilkan token, NFT, dan item digital lainnya
- **Menandatangani transaksi**: Menyetujui tindakan atas nama Anda
- **Terhubung ke aplikasi**: Metode login Anda untuk aplikasi Solana

Anggap saja sebagai kombinasi dari:

- Pengelola kata sandi (menyimpan kunci Anda)
- Penampil rekening bank (menunjukkan saldo Anda)
- Alat tanda tangan digital (menyetujui transaksi)

## Cara Kerja Dompet

Setiap dompet dibangun berdasarkan kunci kriptografi:

### Seed Phrase

Rangkaian 12-24 kata yang dapat membuat ulang dompet Anda. Ini adalah cadangan
utama Anda - simpan dengan aman dan rahasiakan.

### Kunci Privat

Diturunkan dari seed phrase Anda, kunci ini menandatangani transaksi. Dompet
Anda mengelola ini untuk Anda - Anda tidak perlu melihat atau menanganinya
secara langsung.

### Kunci Publik/Alamat

Diturunkan dari kunci privat Anda, ini adalah yang Anda bagikan untuk menerima
dana. Aman untuk dibagikan secara publik.

## Jenis-jenis Dompet

### Dompet Perangkat Lunak

**Ekstensi Browser**

- Nyaman untuk aplikasi web
- Mudah digunakan
- Baik untuk transaksi harian
- Gratis digunakan

**Aplikasi Mobile**

- Akses dompet Anda di mana saja
- Pemindaian kode QR untuk pembayaran
- Keamanan biometrik (sidik jari/wajah)
- Gratis digunakan

**Aplikasi Desktop**

- Antarmuka lengkap fitur
- Baik untuk trader aktif
- Fungsionalitas lanjutan
- Gratis digunakan

### Dompet Perangkat Keras

**Perangkat Fisik**

- Keamanan maksimum
- Penyimpanan kunci offline
- Terbaik untuk menyimpan aset dalam jumlah besar
- Biaya: $50-200

### Memilih Dompet Pertama Anda

Untuk pemula, kami merekomendasikan untuk memulai dengan dompet ekstensi
browser. Mereka:

- Ramah pengguna
- Didukung dengan baik
- Aman untuk penggunaan sehari-hari
- Mudah diatur

## Membuat Dompet Pertama Anda

{/* Illustration recommendation: Step-by-step visual guide showing the wallet creation process */}

Mari kita bahas proses umum pengaturan dompet:

### Langkah 1: Pilih dan Instal Dompet

1. Riset dompet Solana populer
2. Kunjungi situs web resmi (selalu verifikasi URL)
3. Unduh dompet untuk platform Anda
4. Instal mengikuti petunjuk resmi

### Langkah 2: Buat Dompet Baru

1. Buka aplikasi dompet Anda
2. Pilih opsi "Buat dompet baru"
3. Buat kata sandi yang kuat untuk akses perangkat
4. **Penting**: Catat seed phrase Anda

### Langkah 3: Amankan Seed Phrase Anda

{/* Illustration recommendation: Do's and don'ts for seed phrase storage - physical paper in safe (✓), screenshot (✗), cloud storage (✗) */}

Seed phrase Anda sangat penting. Berikut cara menyimpannya dengan aman:

**LAKUKAN:**

- Tuliskan di atas kertas
- Simpan di lokasi yang aman
- Pertimbangkan untuk membuat beberapa salinan di tempat berbeda
- Gunakan brankas tahan api jika memungkinkan

**JANGAN:**

- Mengambil tangkapan layar
- Menyimpan di email atau layanan cloud
- Membagikan kepada siapapun
- Mengetiknya di situs web acak

### Langkah 4: Verifikasi dan Selesaikan

1. Konfirmasi seed phrase Anda saat diminta
2. Dompet Anda sekarang siap digunakan
3. Anda akan melihat alamat dompet Anda di bagian atas
4. Saldo awal Anda akan 0 SOL

## Fitur Dasar Dompet

Dompet modern mencakup:

### Pengelolaan Aset

- Lihat saldo token
- Tampilkan koleksi NFT
- Lacak riwayat transaksi
- Pantau nilai portofolio

### Alat Transaksi

- Kirim dan terima token
- Tukar antar token yang berbeda
- Hubungkan ke aplikasi
- Setujui atau tolak transaksi

### Fitur Keamanan

- Perlindungan kata sandi
- Autentikasi biometrik (mobile)
- Peringatan transaksi
- Deteksi phishing

## Langkah Pertama Anda

Sekarang Anda memiliki dompet, inilah yang harus dilakukan selanjutnya:

1. **Simpan alamat Anda**: Salin ke tempat yang aman untuk referensi
2. **Dapatkan beberapa SOL**: Anda akan membutuhkan sejumlah kecil untuk biaya
   transaksi
3. **Berlatih menerima**: Minta teman mengirimkan jumlah yang sangat kecil
4. **Jelajahi antarmuka**: Biasakan diri dengan tata letaknya

## Pengingat Keamanan Penting

- **Jangan pernah membagikan seed phrase Anda**: Tidak ada layanan sah yang akan
  memintanya
- **Verifikasi situs web**: Penipu membuat situs dompet palsu
- **Mulai dengan jumlah kecil**: Gunakan jumlah kecil saat belajar
- **Perbarui perangkat lunak**: Segera instal pembaruan dompet
- **Bersikap skeptis**: Jika sesuatu terdengar terlalu bagus untuk menjadi
  kenyataan, mungkin memang begitu

## Pertanyaan Umum

### Apakah saya perlu membayar untuk dompet?

Sebagian besar dompet perangkat lunak gratis. Anda hanya membayar untuk dompet
perangkat keras atau fitur premium.

### Bisakah saya memiliki beberapa dompet?

Ya, banyak pengguna memiliki beberapa dompet untuk tujuan berbeda (penggunaan
harian, tabungan, aplikasi tertentu).

### Bagaimana jika saya lupa kata sandi saya?

Kata sandi perangkat Anda dapat diatur ulang dengan seed phrase Anda. Itulah
mengapa seed phrase sangat penting.

### Apakah aset saya disimpan di dompet?

Tidak, aset disimpan di Solana. Dompet Anda menyimpan kunci yang membuktikan
kepemilikan.

## Selanjutnya

Dengan dompet Anda yang sudah siap, Anda siap untuk mulai menggunakan Solana. Di
bab berikutnya, kita akan menjelajahi biaya transaksi dan mengapa hal tersebut
penting untuk aksesibilitas. Anda akan mempelajari mengapa biaya rendah Solana
memungkinkan kasus penggunaan yang tidak praktis di jaringan lain.

Ingat: semua orang pernah menjadi pemula. Luangkan waktu untuk merasa nyaman
dengan dompet Anda sebelum memindahkan jumlah besar. Komunitas Solana siap
membantu jika Anda memiliki pertanyaan.
