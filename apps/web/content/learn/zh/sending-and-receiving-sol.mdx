# 发送和接收 SOL

能够发送和接收 SOL 是您在 Solana 上的第一个实用技能。SOL 是 Solana 网络的原生代币，用于交易、质押以及与应用程序交互。在本章中，您将学习转账的工作原理并练习完成您的第一次交易。

## 您将学到什么

在本章结束时，您将能够：

- 了解您的钱包地址以及如何安全地分享它
- 向另一个钱包发送 SOL
- 接收他人发送的 SOL
- 检查交易状态
- 理解转账过程中发生的事情

## 了解您的钱包地址

您的钱包地址就像您在 Solana 上的账户号码。它具有以下特点：

- **公开**：在接收资金时可以安全分享
- **唯一**：没有两个钱包会有相同的地址
- **永久**：不会更改或过期

示例地址：`7xKXtg2CW87d8V6zXKBq7cM8Y2FPHwzaAkxqEqD7x4rH`

### 分享您的地址

您可以通过以下方式分享您的地址：

- **复制/粘贴**：点击钱包中的复制按钮
- **二维码**：让其他人用手机扫描
- **ENS/域名**：人类可读的名称（例如 `yourname.sol`）

## 接收 SOL

{/* 插图建议：逐步展示某人分享地址并看到 SOL 到达其钱包的视觉效果 */}

接收 SOL 非常简单：

1. **打开您的钱包**
2. **找到您的地址**（通常显示在顶部）
3. **将地址分享**给发送方
4. **等待几秒钟**完成交易
5. **查看您的新余额**会自动更新

### 发送方需要什么

当有人向您发送 SOL 时，他们只需要：

- 您的钱包地址
- 要发送的金额

他们不需要您的姓名、位置或任何个人信息。

## 发送 SOL

{/* 插图建议：展示钱包界面模拟图，突出显示发送流程中的重要元素 */}

以下是向他人发送 SOL 的方法：

### 第一步：获取接收方地址

- 请对方分享他们的 Solana 钱包地址
- 仔细核对您是否获得了正确的地址
- 请记住：交易无法撤销

### 第二步：发起转账

1. 在您的钱包中点击“发送”
2. 粘贴接收方的地址
3. 输入 SOL 的金额
4. 查看交易费用（通常 < $0.001）

### 第三步：确认并发送

1. 仔细检查所有详情
2. 确认交易
3. 等待 1-2 秒完成
4. 保存交易 ID 以备记录

## 了解交易详情

每笔交易都会生成一条包含以下内容的记录：

- **签名**：唯一的交易标识符
- **发件人/收件人**：发送方和接收方地址
- **金额**：转移的 SOL 数量
- **费用**：支付的网络费用（以 SOL 计）
- **时间戳**：交易发生的时间
- **状态**：成功或失败

您可以在区块浏览器（如 Solscan 或 Solana Explorer）上查看任何交易。

## 常见问题

### 转账需要多长时间？

通常 1-2 秒。与传统银行相比，Solana 的交易处理速度非常快。

### 如果我发送到错误的地址怎么办？

交易是永久性的，无法撤销。发送前请务必仔细核对地址。

### 为什么需要 SOL 来支付费用？

每笔交易都需要支付一小笔费用（通常为 $0.00025），以补偿处理交易的网络 validator。

### 我可以发送 SOL 以外的其他代币吗？

可以，但即使发送其他代币，您仍然需要少量的 SOL 来支付交易费用。

## 实践练习

准备好尝试您的第一笔交易了吗？以下是一个安全的练习方法：

1. **发送一小笔金额**（例如 0.001 SOL）给朋友
2. **让他们将其返还**给您
3. **在区块浏览器上查询交易**
4. **记录费用**和确认时间

这可以帮助您通过使用极少的金额熟悉交易流程。

## 交易最佳实践

- **从小额开始**，尤其是发送到新地址时
- **将重要地址**保存在钱包的地址簿中
- **保留交易记录**，以便税务或个人追踪
- **仔细核对地址**，特别是前后 4 个字符
- **了解当前 SOL 的价格**，以便发送时心中有数
