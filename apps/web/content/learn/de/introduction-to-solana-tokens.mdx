# Einführung in Solana Tokens

Tokens sind digitale Vermögenswerte, die auf Solana aufgebaut sind. So wie der
US-Dollar die native Währung der Vereinigten Staaten ist, ist SOL die native
Währung von Solana. Tokens repräsentieren alle anderen Vermögenswerte - von
Stablecoins wie USDC bis hin zu Projekt-Tokens und NFTs. Dieses Kapitel erklärt,
wie Tokens funktionieren und was Sie mit ihnen machen können.

## Was Sie lernen werden

Am Ende dieses Kapitels werden Sie verstehen:

- Was Tokens sind und wie sie auf Solana funktionieren
- Verschiedene Arten von Tokens und ihre Verwendungszwecke
- Wie man Tokens in seiner Wallet verwaltet
- Token Konten und Fee
- Sicherheitsüberlegungen beim Umgang mit Tokens

## Tokens auf Solana verstehen

Solana hat zwei Hauptarten von digitalen Vermögenswerten:

- **SOL**: Die native Währung für Transaktionen und Fee
- **Tokens**: Alle anderen digitalen Vermögenswerte auf Solana

Genau wie Apps auf Ihrem Smartphone bestimmten Standards folgen, um richtig zu
funktionieren, folgen Tokens auf Solana einem gemeinsamen Standard, der sie mit
Wallets und Anwendungen kompatibel macht.

## Arten von Tokens

![Raster mit vier Token-Kategorien: Stablecoins, Wrapped Assets, Projekt-Tokens und Utility-Tokens mit Beispielen](/images/learn/types_of_tokens.webp)

### Stablecoins

Digitale Versionen traditioneller Währungen, die einen stabilen Wert
beibehalten.

**Beispiele:**

- **USDC**: Digitale US-Dollar
- **USDT**: Tether, ein weiterer USD-Stablecoin
- **EURC**: Digitale Euro

**Verwendungszwecke:** Sparen, Zahlungen, Handel, Vermeidung von Volatilität

### Wrapped Assets

Tokens, die Vermögenswerte aus anderen Netzwerken repräsentieren.

**Beispiele:**

- **Wrapped Bitcoin (BTC)**: Bitcoin, verwendbar auf Solana
- **Wrapped Ethereum (ETH)**: Ethereum auf Solana
- **Wrapped SOL**: Wird in einigen DeFi-Protokollen verwendet

**Verwendungszwecke:** Zugriff auf andere Kryptowährungen mit Solanas
Geschwindigkeit und niedrigen Fee

### Projekt-Token

Token, die von Anwendungen und Protokollen erstellt wurden.

**Beispiele:**

- Governance-Token für Abstimmungen über Protokollentscheidungen
- Börsen-Token, die Rabatte auf Gebühren bieten
- Protokoll-Token für die Teilnahme am Ökosystem

**Verwendungszwecke:** Governance-Abstimmungen, Gebührenrabatte, Belohnungen

### Utility-Token

Token mit spezifischen Funktionen innerhalb von Anwendungen.

**Beispiele:**

- Gaming-Token für In-Game-Wirtschaftssysteme
- Social-Token für Creator-Communities
- Belohnungs-Token von Protokollen

**Verwendungszwecke:** Zugriff auf Funktionen, Belohnungen verdienen, Teilnahme
an Ökosystemen

## Wie token account funktionieren

![Wallet-Diagramm, das das Haupt-SOL-Konto und separate Token-Konten für USDC und WBTC zeigt](/images/learn/how_token_accounts_work.webp)

Im Gegensatz zu einigen Netzwerken, bei denen alle Assets eine Adresse teilen,
verwendet Solana ein token account-System:

1. **Haupt-Wallet**: Enthält dein SOL
2. **token account**: Separate Konten für jeden Token-Typ

### Warum token account?

- Bessere Leistung und Organisation
- Klare Trennung verschiedener Assets
- Effiziente Netzwerkspeicherung

### Account-rent

- Jeder token account benötigt eine Einlage von ~0,002 SOL
- Diese "rent" wird bei Schließung des Kontos zurückerstattet
- Verhindert Spam und finanziert Netzwerkspeicher
- Kostet typischerweise $0,20-0,50 pro Token-Typ

## Verwaltung deiner Token

### Token empfangen

1. Teile deine Wallet-Adresse
2. Absender initiiert die Überweisung
3. Wallet erstellt bei Bedarf einen token account
4. Token erscheinen in deiner Wallet

### Deine Token anzeigen

Moderne Wallets zeigen:

- Token-Symbole und Logos
- Aktuelle Guthaben
- USD-Werte
- Aktuelle Transaktionen

### Token senden

1. Token zum Senden auswählen
2. Empfängeradresse eingeben
3. Betrag wählen
4. Transaktion bestätigen
5. Kleine SOL Fee zahlen (~$0.0005)

### Token-Konten schließen

Wenn du einen Token nicht mehr benötigst:

1. Alle Token versenden
2. Das Konto schließen
3. Die rent-Einlage zurückerhalten

## Grundlagen der Token-Erstellung

Jeder kann Token auf Solana erstellen:

### Voraussetzungen

- Kleine Menge SOL für Fee (~0,5 SOL)
- Grundlegende Token-Informationen (Name, Symbol, Angebot)
- Optional: Logo und Metadaten

### Häufige Anwendungen

- Community-Token
- Belohnungssysteme
- Fundraising
- Spielökonomien
- Experimentelle Projekte

### Wichtiger Hinweis

Die Erstellung eines Tokens verleiht ihm keinen Wert. Wert entsteht durch
Nutzen, Akzeptanz und Marktdynamik.

## Token-Sicherheit

![Token-Sicherheitscheckliste, die zeigt, was zu prüfen ist und worauf man beim Umgang mit Tokens achten sollte](/images/learn/token_safety.webp)

### Vor dem Vertrauen verifizieren

- Offizielle Projekt-Websites prüfen
- Token-Adressen verifizieren
- Nach Audit-Berichten suchen
- Das Team recherchieren

### Häufige Betrugsmaschen

- **Gefälschte Token**: Nachahmung beliebter Projekte
- **Rug pulls**: Entwickler, die Projekte aufgeben
- **Honeypots**: Token, die du kaufen, aber nicht verkaufen kannst
- **Pump and dumps**: Künstliche Preismanipulation

### Bewährte Praktiken

1. **Gründlich recherchieren** vor dem Kauf
2. **Klein anfangen** bei neuen Token
3. **Seriöse Quellen nutzen** für Token-Informationen
4. **Skeptisch sein** bei garantierten Renditen
5. **Liquidität prüfen** vor größeren Käufen

## Arbeiten mit Token

### Token tauschen

DEX-Aggregatoren nutzen, um die besten Kurse zu finden:

1. Wallet verbinden
2. Token zum Tausch auswählen
3. Kurs und Fee überprüfen
4. Tausch bestätigen

### Token-Informationen überprüfen

Ressourcen für die Recherche:

- Block-Explorer: Token-Details und Inhaber anzeigen
- Preis-Aggregatoren: Preise und Liquidität prüfen
- Offizielle Projektwebseiten: Authentizität verifizieren

### Verwaltung mehrerer Token

- Organisieren durch Ausblenden von Nullsalden
- Häufig verwendete Token als Favoriten markieren
- Portfolio-Tracker verwenden
- Ungenutzte Token-Konten schließen

## Häufige Fragen

### Benötige ich SOL, um Token zu senden?

Ja, alle Transaktionen erfordern kleine SOL-Fee, auch beim Senden anderer Token.

### Warum erscheinen manche Token automatisch?

Projekte können Token an Wallets airdroppen. Sei vorsichtig bei unaufgefordert
erhaltenen Token.

### Kann ich gesendete Token zurückholen?

Nein, Transaktionen auf Solana sind permanent. Überprüfe immer die Adressen.

### Was bestimmt den Wert eines Tokens?

Angebot und Nachfrage, Nutzen, Projektentwicklung und Marktstimmung beeinflussen
alle den Tokenpreis.
