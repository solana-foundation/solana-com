Agave, Firedancer, Solana Go, and Surfpool shipped new releases this week, alongside new validator-client work and an offchain message signing specification.

## Releases

Several core and developer-facing projects published new versions across testnet, mainnet, and tooling.

- Anza shipped [Agave v4.1.0-beta.3](https://github.com/anza-xyz/agave/releases/tag/v4.1.0-beta.3) and [Agave v4.0.2](https://github.com/anza-xyz/agave/releases/tag/v4.0.2).
- Firedancer released [Testnet v0.1002.40103](https://github.com/firedancer-io/firedancer/releases/tag/v0.1002.40103) and [Mainnet v0.911.40002](https://github.com/firedancer-io/firedancer/releases/tag/v0.911.40002).
- The Solana Foundation released [Solana Go v1.21.0](https://github.com/solana-foundation/solana-go/releases/tag/v1.21.0).
- The Solana Foundation released [Surfpool v1.3.1](https://github.com/solana-foundation/surfpool/releases/tag/v1.3.1).

## Solana Improvement Documents

A new Solana Improvement Documents (SIMDs) discussion proposes a SysVar account for details about the upcoming leader.

- Contributors opened a discussion to [create a new SysVar account with details about the upcoming leader](https://github.com/solana-foundation/solana-improvement-documents/discussions/558).

## Validator client work

Agave and other validator-client projects continued work across networking, repair, bank changes, and replay.

- Anza worked on [Shred Tracer for Agave](https://github.com/anza-xyz/agave/pull/12928).
- Anza worked on [XdpSender for Agave](https://github.com/anza-xyz/agave/pull/12897).
- Anza worked on an [Agave Repair Signing Pool](https://github.com/anza-xyz/agave/pull/12987).
- Anza worked on [Bank: add_root for Agave](https://github.com/anza-xyz/agave/pull/12913).
- Anza worked on [Dynamic VAT for Agave](https://github.com/anza-xyz/agave/pull/12990).
- Anza added [802.1Q VLAN encapsulation to the XDP transmit loop](https://github.com/anza-xyz/agave/pull/12920).
- Anza is [reconfiguring Agave’s multicast implementation](https://github.com/anza-xyz/agave/pull/12923).
- Mithril is working to [implement Replay from Solana Turbine](https://github.com/Overclock-Validator/mithril/pull/243).

## Clients, wallets, and program frameworks

Web, Rust, and Go client work included wallet-adapter alternatives and offchain message signing updates, while program framework work continued in Pinocchio.

- Anza worked on a [kit-plugins alternative to wallet adapter](https://github.com/anza-xyz/kit-plugins/pull/173).
- Contributors advanced an offchain message signing spec across [wallet-standard pull request #92](https://github.com/anza-xyz/wallet-standard/pull/92), [Agave pull request #12965](https://github.com/anza-xyz/agave/pull/12965), the [Solana Off-Chain Message Signing document](https://app.notion.com/p/solanafoundation/Solana-Off-Chain-Message-Signing-OCMS-374d36dad52d80bf981ddbcebfeca5f0), and [wallet-standard pull request #93](https://github.com/anza-xyz/wallet-standard/pull/93).
- Pinocchio has a program-framework update in [pull request #421](https://github.com/anza-xyz/pinocchio/pull/421).

## Other ecosystem work

Kora Deploy added a developer workflow for deploying to Devnet without SOL.

- Kora Deploy allows developers to [deploy a Solana program on Devnet without SOL](https://x.com/dev_jodee/status/2064439356626964752).
