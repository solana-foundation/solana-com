Solana engineering teams shipped new Agave, Firedancer, Solana Go, and Surfpool releases for the week of June 10, 2026. The week also brought a new Solana Improvement Documents discussion, validator-client updates across Agave and Mithril, and client tooling work for wallets and program frameworks.

## Releases

New versions landed across validator clients, language clients, and local development tooling. Builders and operators can review the linked release notes for version-specific changes.

* Anza shipped [Agave v4.1.0-beta.3](https://github.com/anza-xyz/agave/releases/tag/v4.1.0-beta.3) and [Agave v4.0.2](https://github.com/anza-xyz/agave/releases/tag/v4.0.2).
* Firedancer released [Testnet v0.1002.40103](https://github.com/firedancer-io/firedancer/releases/tag/v0.1002.40103) and [Mainnet v0.911.40002](https://github.com/firedancer-io/firedancer/releases/tag/v0.911.40002).
* The Solana Foundation released [Solana Go v1.21.0](https://github.com/solana-foundation/solana-go/releases/tag/v1.21.0).
* The Solana Foundation released [Surfpool v1.3.1](https://github.com/solana-foundation/surfpool/releases/tag/v1.3.1).

## Proposed network changes

A new Solana Improvement Documents (SIMDs) discussion proposes adding a SysVar account with details about the upcoming leader. The proposal is available for review and discussion on GitHub.

* Review the [SIMD discussion for a new upcoming-leader SysVar account](https://github.com/solana-foundation/solana-improvement-documents/discussions/558).

## Validator client work

Agave and Mithril work continued across networking, repair, replay, and bank-related changes. The linked pull requests include implementation details and review history.

* Anza worked on [Shred Tracer in Agave](https://github.com/anza-xyz/agave/pull/12928).
* Anza worked on [XdpSender in Agave](https://github.com/anza-xyz/agave/pull/12897).
* Anza worked on [Repair Signing Pool in Agave](https://github.com/anza-xyz/agave/pull/12987).
* Anza worked on [Bank `add_root` in Agave](https://github.com/anza-xyz/agave/pull/12913).
* Anza worked on [Dynamic VAT in Agave](https://github.com/anza-xyz/agave/pull/12990).
* Anza worked on [adding 802.1Q VLAN encapsulation to the XDP transmit loop in Agave](https://github.com/anza-xyz/agave/pull/12920).
* Anza worked on [reconfiguring Agave’s multicast implementation](https://github.com/anza-xyz/agave/pull/12923).
* Mithril worked to implement [Replay from Solana Turbine](https://github.com/Overclock-Validator/mithril/pull/243).

## Client tooling and program frameworks

Web, Rust, and Go client work included wallet tooling and off-chain message signing updates. Program framework work also continued in Pinocchio.

* Anza worked on an [alternative to wallet adapter in kit-plugins](https://github.com/anza-xyz/kit-plugins/pull/173).
* An off-chain message signing specification moved across [wallet-standard pull request #92](https://github.com/anza-xyz/wallet-standard/pull/92), [Agave pull request #12965](https://github.com/anza-xyz/agave/pull/12965), the [Solana Off-Chain Message Signing specification](https://app.notion.com/p/solanafoundation/Solana-Off-Chain-Message-Signing-OCMS-374d36dad52d80bf981ddbcebfeca5f0), and [wallet-standard pull request #93](https://github.com/anza-xyz/wallet-standard/pull/93).
* Anza worked on [Pinocchio pull request #421](https://github.com/anza-xyz/pinocchio/pull/421).

## Other ecosystem notes

Developer tooling also saw work aimed at making devnet program deployment easier.

* Kora Deploy allows builders to deploy a Solana program on Devnet without SOL, according to [a post from Jodee](https://x.com/dev_jodee/status/2064439356626964752).
