Solana Crashcourse is a lighter-weight course inside Developers Learn. It
focuses on the fastest path from orientation to a working app, and more lessons
can be added here as they finish editing.

## What You Will Cover

- The Solana developer roadmap
- A working local development environment
- A fast first build-and-deploy flow

## How To Use This Course

- Watch the episodes in order.
- Use the companion pages for notes and follow-up links.
- Come back to the full bootcamp when you want the deeper, four-module path.
