## Lesson goal

Understand the major stages of the Solana developer journey so you know what to
learn first and what can wait until later.

## Checkpoints

- Identify the core topics you need before you start building.
- Note which tools and frameworks show up later in the roadmap.
- Decide which deeper path you want to take after this crashcourse.
