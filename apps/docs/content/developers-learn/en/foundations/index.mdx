product and protocol builds.

- Set up the local developer toolchain
- Start from `create-solana-dapp`
- Build with AI while verifying blockchain-critical details

## How To Use This Track

- Watch each episode first.
- Use the companion page for notes, checkpoints, and links.
- Mark a lesson complete locally before moving to the next track.
