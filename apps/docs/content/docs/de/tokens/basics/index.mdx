Der Abschnitt SPL Token Basics behandelt die grundlegenden Anweisungen für die
Interaktion mit SPL-Tokens und konzentriert sich auf Funktionen, die sowohl vom
ursprünglichen Token Program als auch vom Token Extensions Program (Token 2022)
unterstützt werden.

<Callout type="info">
  Weitere Informationen zum Hinzufügen von Metadaten zu SPL-Tokens finden Sie in
  der [Metaplex-Dokumentation](https://developers.metaplex.com/token-metadata).
</Callout>
