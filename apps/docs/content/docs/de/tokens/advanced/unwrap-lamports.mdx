## Was bewirkt „Lamports entpacken“?

_rs`UnwrapLamports`_ verschiebt SOL aus einem nativen (gepackten SOL) token
account in ein Ziel-Konten als lamport. Du kannst das gesamte Guthaben oder
einen bestimmten Betrag entpacken, und das token account bleibt dabei geöffnet.

Dies ist flexibler als das Schließen eines gepackten SOL-Konten: Ein teilweises
Entpacken ermöglicht es dir, einen Teil des SOL abzuheben, während das Konten
für die spätere Nutzung finanziert bleibt.

## Konten

| Konten        | Beschreibung                                                         |
| ------------- | -------------------------------------------------------------------- |
| `source`      | Beschreibbar. Das native (gepackte SOL) token account zum Entpacken. |
| `destination` | Beschreibbar. Empfängt die entpackten lamport.                       |
| `authority`   | Der Konten-Eigentümer oder ein Delegierter (oder Multisig-Signer).   |

<Callout type="warn">

Die Quelle muss ein natives token account sein; andere token accounts geben
_rs`NonNativeNotSupported`_ zurück. Ein Delegierter kann bis zu seinem
delegierten Betrag entpacken.

</Callout>

## So werden Lamports entpackt

Lasse _ts`amount`_ weg, um das gesamte Guthaben zu entpacken, oder setze es, um
eine bestimmte Anzahl von lamport zu entpacken. Das Beispiel verwendet einen
konfigurierten
[`@solana/kit`](/docs/clients/official/javascript#solana-kit)-Client – siehe
[Tokens übertragen](/docs/tokens/basics/transfer-tokens) für die
Client-Einrichtung.

```ts title="Unwrap wrapped SOL (Kit)"
import { getUnwrapLamportsInstruction } from "@solana-program/token";

// source: the native (wrapped SOL) token account to unwrap.
// destination: address that receives the unwrapped lamports.
// authority: a TransactionSigner (the account owner or a delegate).

// Unwrap a specific number of lamports.
const unwrapSome = getUnwrapLamportsInstruction({
  source,
  destination,
  authority,
  amount: 500_000_000n
});

// Or omit `amount` to unwrap the entire balance.
const unwrapAll = getUnwrapLamportsInstruction({
  source,
  destination,
  authority
});

// Send whichever instruction you need.
await client.sendTransaction([unwrapSome]);
```

Für eine Multisig-Autorität übergib die Mitunterzeichner mit _ts`multiSigners`_.
