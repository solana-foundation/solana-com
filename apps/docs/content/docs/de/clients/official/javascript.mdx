## Solana Kit

`@solana/kit` ist das empfohlene TypeScript-SDK für die Entwicklung auf Solana.
Sie stellen einen Client aus kombinierbaren Plugins zusammen, fügen bei Bedarf
React-Bindungen hinzu und binden generierte Programmclients für die von Ihnen
aufgerufenen On-Chain-Programme ein.

| Paket                     | Beschreibung                                    | GitHub                                                                                 |
| ------------------------- | ----------------------------------------------- | -------------------------------------------------------------------------------------- |
| @solana/kit               | Kern-SDK: RPC, Transaktionen, Codecs, Signer    | [Quelle](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | React-Bindungen: `ClientProvider` und Hooks     | [Quelle](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | RPC-Verbindung + Transaktionsversand            | [Quelle](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | Gebührenzahler-, Identitäts- und Signer-Plugins | [Quelle](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Wallet-Standard-Verbindung + React-Hooks        | [Quelle](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

Beispiel `tokenProgram()` → `client.token`).

| @solana-program/token          | Interaktion mit dem Token Program           | [Quelle](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Interaktion mit dem Token-2022 program      | [Quelle](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Interaktion mit dem Memo-Programm           | [Quelle](https://github.com/solana-program/memo/tree/main/clients/js)           |

## Solana Web3.js

`@solana/web3.js` ist das veraltete TypeScript-SDK für Solana. Neue Apps sollten
`@solana/kit` verwenden; siehe [Migration zu Kit](/docs/frontend/web3-compat).

| Paket             | Beschreibung                                                           | GitHub                                                                             |
| ----------------- | ---------------------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| @solana/web3.js   | Kern-SDK                                                               | [Quelle](https://github.com/solana-foundation/solana-web3.js)                      |
| @solana/spl-token | Interaktion mit Token Program, Token-2022 und Associated Token Program | [Quelle](https://github.com/solana-program/token-2022/tree/main/clients/js-legacy) |
| @solana/spl-memo  | Interaktion mit dem Memo-Programm                                      | [Quelle](https://github.com/solana-program/memo/tree/main/clients/js-legacy)       |
