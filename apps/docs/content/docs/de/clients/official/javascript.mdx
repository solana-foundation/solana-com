## Solana Client

`@solana/client` ist die headless Solana-Runtime, die RPC, Wallets,
Transaktionen und Subscriptions in einem Store bündelt.

| Paket          | Beschreibung            | GitHub                                                                                 |
| -------------- | ----------------------- | -------------------------------------------------------------------------------------- |
| @solana/client | Headless Client-Runtime | [Quelle](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [Anleitung: @solana/client Übersicht](/docs/frontend/client)

## Solana React Hooks

`@solana/react-hooks` legt einen React-Provider und Hooks über `@solana/client`
und verwendet dieselbe Runtime und denselben Cache.

| Paket               | Beschreibung                   | GitHub                                                                                      |
| ------------------- | ------------------------------ | ------------------------------------------------------------------------------------------- |
| @solana/react-hooks | React-Provider + Hooks für UIs | [Quelle](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [Anleitung: @solana/react-hooks](/docs/frontend/react-hooks)

## Solana Kit

`@solana/kit` ist das empfohlene TypeScript-SDK für die Entwicklung auf Solana.

| Paket                          | Beschreibung                           | GitHub                                                                          |
| ------------------------------ | -------------------------------------- | ------------------------------------------------------------------------------- |
| @solana/kit                    | Core-SDK                               | [Quelle](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | Interaktion mit System Program         | [Quelle](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Interaktion mit Token Program          | [Quelle](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Interaktion mit Token-2022 Program     | [Quelle](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Interaktion mit Memo Program           | [Quelle](https://github.com/solana-program/memo/tree/main/clients/js)           |
