## Programm-Crates

Verwenden Sie `solana-program`, um On-Chain-Programme zu erstellen.

- [Beispiel: Ein Solana-Programm erstellen](/docs/programs/rust)

| Crate          | Beschreibung                 | Docs                                      | GitHub                                                                  |
| -------------- | ---------------------------- | ----------------------------------------- | ----------------------------------------------------------------------- |
| solana-program | On-Chain-Programme erstellen | [Ansehen](https://docs.rs/solana-program) | [Quellcode](https://github.com/anza-xyz/solana-sdk/tree/master/program) |

Pinocchio ist eine Bibliothek ohne Abhängigkeiten zur Erstellung von
Solana-Programmen in Rust und kann als Ersatz für `solana-program` zum Schreiben
von On-Chain-Programmen verwendet werden.

| Crate     | Beschreibung                          | Docs                                 | GitHub                                             |
| --------- | ------------------------------------- | ------------------------------------ | -------------------------------------------------- |
| pinocchio | Abhängigkeitsfreie On-Chain-Programme | [Ansehen](https://docs.rs/pinocchio) | [Quellcode](https://github.com/anza-xyz/pinocchio) |

<Callout type="warn">
  Pinocchio befindet sich noch in der Entwicklung und wird in naher Zukunft
  wahrscheinlich grundlegende Änderungen erfahren.
</Callout>

Pinocchio enthält programmspezifische Crates für die Interaktion mit
Solana-Programmen:

| Crate                              | Beschreibung                             | Docs                                                           | GitHub                                                                                         |
| ---------------------------------- | ---------------------------------------- | -------------------------------------------------------------- | ---------------------------------------------------------------------------------------------- |
| pinocchio-system                   | Interaktion mit System Program           | [Ansehen](https://docs.rs/pinocchio-system/)                   | [Quellcode](https://github.com/anza-xyz/pinocchio/tree/main/programs/system)                   |
| pinocchio-token                    | Interaktion mit Token Program            | [Ansehen](https://docs.rs/pinocchio-token/)                    | [Quellcode](https://github.com/anza-xyz/pinocchio/tree/main/programs/token)                    |
| pinocchio-token-2022               | Interaktion mit Token-2022 Program       | [Ansehen](https://docs.rs/pinocchio-token-2022/)               | [Quellcode](https://github.com/anza-xyz/pinocchio/tree/main/programs/token-2022)               |
| pinocchio-associated-token-account | Interaktion mit Associated Token Program | [Ansehen](https://docs.rs/pinocchio-associated-token-account/) | [Quellcode](https://github.com/anza-xyz/pinocchio/tree/main/programs/associated-token-account) |
| pinocchio-memo                     | Interaktion mit Memo Program             | [Ansehen](https://docs.rs/pinocchio-memo/)                     | [Quellcode](https://github.com/anza-xyz/pinocchio/tree/main/programs/memo)                     |
