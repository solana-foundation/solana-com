## Syscalls

Programme interagieren mit der Runtime durch Syscalls, die in
[`create_program_runtime_environment_v1()`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/lib.rs#L282)
registriert sind. Jeder Syscall zieht Compute Units vom Compute-Budget der
Transaktion ab.

### Syscall-Kosten

Die Basis-Compute-Unit-Kostenkonstanten jedes Syscalls sind in
[`SVMTransactionExecutionCost`](https://github.com/anza-xyz/agave/blob/v3.1.8/program-runtime/src/execution_budget.rs#L198-L236)
definiert. Einige Syscalls kombinieren diese Konstanten in ihrer Implementierung
zu komplexeren Formeln (z. B. Skalierung mit EingabegrĂ¶Ăe oder HinzufĂŒgen von
`syscall_base_cost` Overhead). Siehe die einzelnen Syscall-Quelllinks unten fĂŒr
exakte Kostenberechnungen.
