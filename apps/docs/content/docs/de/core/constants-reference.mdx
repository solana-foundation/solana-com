Diese Seite ist eine zentrale Referenz für alle Laufzeitkonstanten, die auf den
Core-Konzeptseiten erwähnt werden. Jede Konstante verlinkt auf ihre
Quelldefinition im [agave](https://github.com/anza-xyz/agave/tree/v3.1.8)
Validator-Client oder [solana-sdk](https://github.com/anza-xyz/solana-sdk).
