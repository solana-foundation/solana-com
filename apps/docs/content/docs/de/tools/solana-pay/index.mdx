**Solana Pay ist ein Standardprotokoll und eine Reihe von
Referenzimplementierungen, die es Entwicklern ermöglichen, dezentrale Zahlungen
in ihre Apps und Dienste zu integrieren.** Die Solana-Blockchain bestätigt
Transaktionen in weniger als einer Sekunde und kostet durchschnittlich 0,0005 $,
wodurch Nutzern ein nahtloses Erlebnis ohne Zwischenhändler geboten wird.

## Warum Solana Pay?

- **Standardprotokoll**: Einheitlicher Ansatz für Zahlungen über alle
  Solana-Wallets und Apps hinweg
- **Ultraschnell**: Transaktionen werden in weniger als einer Sekunde bestätigt
- **Ultragünstig**: Durchschnittliche Transaktionskosten von 0,0005 $
- **Keine Zwischenhändler**: Direkte Peer-to-Peer-Zahlungen ohne Drittparteien
- **Universelle Wallet-Unterstützung**: Kompatibel mit Phantom, Solflare,
  Backpack und mehr

## Schnellstart

Der einfachste Einstieg erfolgt über eine Überweisungsanfrage-URL:

```
solana:recipient?amount=0.01&label=Coffee%20Shop&message=Grande%20Latte
```

Diese URL kann:

- Als QR-Code zum Scannen mit mobilen Wallets codiert werden
- Als Zahlungslink in Webanwendungen verwendet werden
- In mobilen Apps als Deep-Links eingebettet werden

## Installation

Installieren Sie das Solana Pay JavaScript SDK:

```bash
# Using pnpm (recommended)
pnpm add @solana/pay@beta @solana/kit

# Using npm
npm install @solana/pay@beta @solana/kit

# Using yarn
yarn add @solana/pay@beta @solana/kit
```

## Anwendungsfälle

Solana Pay ermöglicht eine breite Palette von Zahlungsszenarien:

- **Einzelhandelszahlungen**: Persönliche Zahlungen mit QR-Codes
- **E-Commerce**: Web-Checkout-Prozesse mit Wallet-Integration
- **Mobile Apps**: Native Zahlungserlebnisse mit Deep-Linking
- **NFT-Prägung**: Dynamische Transaktionserstellung für komplexe Interaktionen
- **Abonnements**: Wiederkehrende Zahlungen und Treueprogramme
- **DeFi-Integration**: Kombinierbare Transaktionen mit Swaps und Kreditvergabe

## Unterstützte Wallets

Solana Pay funktioniert mit allen wichtigen Solana-Wallets:

- **[Phantom](https://phantom.app/)** - Beliebteste Solana-Wallet
- **[Solflare](https://solflare.com/)** - Multiplattform-Solana-Wallet
- **[Backpack](https://backpack.app/)** - Web3-Wallet und App-Store
- **[Glow](https://glow.app/)** - Mobile-First-Solana-Wallet
- **[Ultimate](https://ultimate.app/)** - DeFi-fokussierte Wallet
- **[Decaf Wallet](https://decaf.so/)** - Wallet für globale Zahlungen
- **[Espresso Cash](https://www.espressocash.com/)** - Wallet für globale
  Zahlungen
- **[Tiplink](https://tiplink.io/)** - Die einfachste Wallet der Welt
- **[Bitget Wallet](https://bitget.com/)** - Multi-Chain-Wallet
