Starten Sie schnell mit Solana Pay durch diese schrittweisen Anleitungen.

## Installation

Richten Sie das Solana Pay SDK ein und konfigurieren Sie Ihre
Entwicklungsumgebung.

**[→ Installationsanleitung](/docs/tools/solana-pay/quickstart/installation)** -
SDK und Abhängigkeiten installieren

## Zahlungsarten

Wählen Sie die richtige Zahlungsintegration für Ihren Anwendungsfall:

### Transfer-Anfragen (Einfach)

Erstellen Sie nicht-interaktive Zahlungs-URLs für grundlegende SOL- und
SPL-Token-Transfers.

**[→ Transfer-Anfragen](/docs/tools/solana-pay/quickstart/transfer-requests)** -
Einfache Zahlungs-URLs

**Optimal für:**

- Grundlegende Zahlungen und Spenden
- QR-Code-Zahlungen
- Zahlungslinks in E-Mails

### Transaktions-Anfragen (Fortgeschritten)

Erstellen Sie interaktive Zahlungsabläufe mit serverseitiger
Transaktionszusammenstellung.

**[→ Transaktions-Anfragen](/docs/tools/solana-pay/quickstart/transaction-requests)** -
Interaktive Zahlungsabläufe

**Optimal für:**

- Dynamische Preisgestaltung
- NFT-Prägung
- Komplexe DeFi-Transaktionen
- Individuelle Geschäftslogik

## QR-Code-Integration

Generieren und personalisieren Sie QR-Codes für mobile Wallet-Zahlungen.

**[→ QR-Code-Integration](/docs/tools/solana-pay/quickstart/qr-codes)** -
Scannbare Zahlungscodes erstellen

**Optimal für:**

- Kassensysteme
- Web-zu-Mobil-Zahlungen
- Persönliche Transaktionen
