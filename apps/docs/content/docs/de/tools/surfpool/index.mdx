Surfpool ist ein Drop-in-Ersatz für `solana-test-validator`, speziell
entwickelt, um Entwicklern, die auf **Solana** aufbauen, die bestmögliche
Erfahrung zu bieten.

Sein Kernmerkmal besteht darin, dass Entwickler ihre Programme lokal simulieren
können, indem sie Mainnet-Konten just-in-time abrufen.

Surfpool führt außerdem nahtlos Infrastructure as Code in Anchor- oder
Pinocchio-basierte Projekte ein und ermöglicht reproduzierbare, nachvollziehbare
und sichere Deployments in jedes Solana-Netzwerk – privat oder öffentlich.
