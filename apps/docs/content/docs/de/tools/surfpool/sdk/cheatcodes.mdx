---
title: Cheatcodes
description:
  Transaktionen überspringen und den Surfnet-Zustand direkt ändern. SOL und
  Token bereitstellen, beliebige Konten-Daten setzen, upstream-gecachte Konten
  zurücksetzen und Live-Mainnet-Konten streamen.
---

Cheatcodes sind Zustandsänderungen, die den normalen Transaktionsablauf umgehen.
Sie werden sofort ausgeführt, ohne einen Blockhash zu verbrauchen oder Fee zu
zahlen – genau das, was man für die Testeinrichtung benötigt.

In Rust befinden sich Cheatcodes unter `Surfnet::cheatcodes()`. In JS sind sie
Methoden direkt auf der `Surfnet`-Instanz.

## SOL bereitstellen

`fundSol` / `fund_sol` setzt den lamport-Kontostand eines beliebigen Konten,
wobei das Konten erstellt wird, falls es noch nicht existiert.

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::{Pubkey, Surfnet};

let surfnet = Surfnet::start().await?;
let cheats = surfnet.cheatcodes();
let wallet = Pubkey::new_unique();

cheats.fund_sol(&wallet, 1_000_000_000)?;

// Fund several accounts in one call.
let bob = Pubkey::new_unique();
let carol = Pubkey::new_unique();
cheats.fund_sol_many(&[(&bob, 2_000_000_000), (&carol, 3_000_000_000)])?;
```

```ts !! title="TypeScript"
import { Surfnet } from "@solana/surfpool";

const surfnet = Surfnet.start();
const wallet = Surfnet.newKeypair();

surfnet.fundSol(wallet.publicKey, 1_000_000_000);

const bob = Surfnet.newKeypair();
const carol = Surfnet.newKeypair();
surfnet.fundSolMany([
  { address: bob.publicKey, lamports: 2_000_000_000 },
  { address: carol.publicKey, lamports: 3_000_000_000 }
]);
```

</CodeTabs>

## Token bereitstellen

`fundToken` / `fund_token` prägt Token in eine Wallet, indem das zugehörige
token account berechnet, bei Bedarf erstellt und der Betrag gesetzt wird.

<Callout type="info" title="Token-2022 Mints">
  Standardmäßig verwendet `fundToken` das klassische SPL Token Program. Für
  Token-2022 Mints wird die Token-2022-Programm-ID
  (`TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb`) als letztes Argument
  übergeben.
</Callout>

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::{Pubkey, Surfnet};

let surfnet = Surfnet::start().await?;
let cheats = surfnet.cheatcodes();

let wallet = Pubkey::new_unique();
let mint: Pubkey = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v"
    .parse()
    .unwrap();

// Classic SPL Token mint.
cheats.fund_token(&wallet, &mint, 5_000_000, None)?;

// Token-2022 mint — pass the program id explicitly.
let token_2022: Pubkey = "TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb"
    .parse()
    .unwrap();
cheats.fund_token(&wallet, &mint, 5_000_000, Some(&token_2022))?;

// Fund many wallets with the same mint.
let mob = vec![Pubkey::new_unique(), Pubkey::new_unique()];
cheats.fund_token_many(&mob.iter().collect::<Vec<_>>(), &mint, 1_000_000, None)?;
```

```ts !! title="TypeScript"
const surfnet = Surfnet.start();
const wallet = Surfnet.newKeypair();
const USDC = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v";

surfnet.fundToken(wallet.publicKey, USDC, 5_000_000);

// Token-2022 mint — pass the program id explicitly.
const TOKEN_2022 = "TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb";
surfnet.fundToken(wallet.publicKey, USDC, 5_000_000, TOKEN_2022);

// Fund many wallets at once.
const mob = [Surfnet.newKeypair().publicKey, Surfnet.newKeypair().publicKey];
surfnet.fundTokenMany(mob, USDC, 1_000_000);
```

</CodeTabs>

Um die ATA ohne Bereitstellung abzuleiten, verwende `getAta` / `get_ata`:

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
let ata = cheats.get_ata(&wallet, &mint, None);
```

```ts !! title="TypeScript"
const ata = surfnet.getAta(wallet.publicKey, USDC);
```

</CodeTabs>

## Beliebigen Konten-Zustand setzen

Die Methode `setAccount` schreibt lamports, Owner und rohe Datenbytes für ein
beliebiges Konten in einem einzigen Aufruf. Das Rust-SDK stellt zusätzlich einen
`SetAccount`-Builder für erweiterte Felder wie `rent_epoch` und `executable`
bereit.

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::cheatcodes::builders::SetAccount;
use surfpool_sdk::{Pubkey, Surfnet};

let surfnet = Surfnet::start().await?;
let cheats = surfnet.cheatcodes();

let address = Pubkey::new_unique();
let owner = Pubkey::new_unique();

// Direct method.
cheats.set_account(&address, 500_000, &[1, 2, 3], &owner)?;

// Builder when you need finer control.
cheats.execute(
    SetAccount::new(address)
        .lamports(500_000)
        .owner(owner)
        .data(vec![1, 2, 3])
        .rent_epoch(0)
        .executable(false),
)?;
```

```ts !! title="TypeScript"
const surfnet = Surfnet.start();
const address = Surfnet.newKeypair();
const owner = Surfnet.newKeypair();

surfnet.setAccount(
  address.publicKey,
  500_000,
  new Uint8Array([1, 2, 3]),
  owner.publicKey
);
```

</CodeTabs>

## Token-Konten-Felder mutieren

`setTokenAccount` aktualisiert die erweiterten Felder eines bestehenden token
account — Delegate, Status, Close Authority, delegierter Betrag. Verwenden Sie
es, wenn `fundToken` allein nicht ausreicht.

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::cheatcodes::builders::SetTokenAccount;
use surfpool_sdk::{Pubkey, Surfnet};

let cheats = Surfnet::start().await?.cheatcodes();

let owner = Pubkey::new_unique();
let mint: Pubkey = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v".parse()?;
let delegate = Pubkey::new_unique();

cheats.execute(
    SetTokenAccount::new(owner, mint)
        .amount(2_000_000)
        .delegate(delegate)
        .delegated_amount(500_000)
        .state("initialized"),
)?;

// Later: clear the delegation.
cheats.execute(SetTokenAccount::new(owner, mint).clear_delegate())?;
```

```ts !! title="TypeScript"
const surfnet = Surfnet.start();
const owner = Surfnet.newKeypair();
const USDC = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v";
const delegate = Surfnet.newKeypair();

surfnet.setTokenAccount(owner.publicKey, USDC, {
  amount: 2_000_000,
  delegate: delegate.publicKey,
  delegatedAmount: 500_000,
  state: "initialized"
});

// Later: clear the delegation.
surfnet.setTokenAccount(owner.publicKey, USDC, { clearDelegate: true });
```

</CodeTabs>

## Konten auf den Upstream-Zustand zurücksetzen

`resetAccount` verwirft alle lokalen Änderungen und ruft die Konten erneut vom
Upstream-RPC ab. Übergeben Sie `includeOwnedAccounts: true`, um auch alle Konten
zurückzusetzen, die dem Ziel gehören – nützlich beim Zurücksetzen eines
Programms und all seiner PDAs.

<Callout type="info" title="Benötigt `remoteRpcUrl` zum erneuten Abrufen">
  In einem Offline-Surfnet löscht `resetAccount` die lokale Kopie, hat jedoch
  keinen Upstream, von dem abgerufen werden kann. Konfigurieren Sie
  `remoteRpcUrl` beim Start, damit Zurücksetzungen den Upstream-Zustand
  wiederherstellen.
</Callout>

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::cheatcodes::builders::ResetAccount;
use surfpool_sdk::{Pubkey, Surfnet};

let cheats = Surfnet::start().await?.cheatcodes();
let token_program: Pubkey = "TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA".parse()?;

cheats.execute(
    ResetAccount::new(token_program).include_owned_accounts(true),
)?;
```

```ts !! title="TypeScript"
const surfnet = Surfnet.start();
const TOKEN_PROGRAM = "TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA";

surfnet.resetAccount(TOKEN_PROGRAM, { includeOwnedAccounts: true });
```

</CodeTabs>

## Konten live streamen

`streamAccount` registriert ein Konto für das Hintergrund-Polling vom Upstream-
RPC. Die lokale Surfnet-Kopie bleibt mit dem Mainnet synchronisiert, ohne dass
der Test explizit pollen muss. Kombinieren Sie dies mit `includeOwnedAccounts`,
um alle PDAs eines Programms zu streamen.

<Callout type="warn" title="Erfordert `remoteRpcUrl`">
  `streamAccount` funktioniert nur auf einem Surfnet, das mit `remoteRpcUrl`
  gestartet wurde (oder `remote_rpc_url` in Rust). Ohne einen Upstream-RPC gibt
  es nichts zu streamen, und der Aufruf gibt einen Fehler zurück.
</Callout>

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::cheatcodes::builders::StreamAccount;
use surfpool_sdk::{Pubkey, Surfnet};

let cheats = Surfnet::builder()
    .remote_rpc_url("https://api.mainnet-beta.solana.com")
    .start()
    .await?
    .cheatcodes();

let oracle: Pubkey = "H6ARHf6YXhGYeQfUzQNGk6rDNnLBQKrenN712K4AQJEG".parse()?;

cheats.execute(StreamAccount::new(oracle).include_owned_accounts(false))?;
```

```ts !! title="TypeScript"
const surfnet = Surfnet.startWithConfig({
  remoteRpcUrl: "https://api.mainnet-beta.solana.com"
});

surfnet.streamAccount("H6ARHf6YXhGYeQfUzQNGk6rDNnLBQKrenN712K4AQJEG");
```

</CodeTabs>

## Cheatcode-Builder (Rust)

Das Rust-SDK stellt typisierte Builder unter
`surfpool_sdk::cheatcodes::builders` bereit für Fälle, in denen die
Komfortmethoden nicht ausreichen. Jeder Builder implementiert das
`CheatcodeBuilder`-Trait und wird über `cheats.execute(builder)` ausgeführt.

| Builder           | Konstruktor                                          | Häufige Setter                                                                                                                   |
| ----------------- | ---------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| `SetAccount`      | `new(address: Pubkey)`                               | `lamports`, `data`, `owner`, `rent_epoch`, `executable`                                                                          |
| `SetTokenAccount` | `new(owner: Pubkey, mint: Pubkey)`                   | `amount`, `delegate`, `clear_delegate`, `state`, `delegated_amount`, `close_authority`, `clear_close_authority`, `token_program` |
| `ResetAccount`    | `new(address: Pubkey)`                               | `include_owned_accounts(bool)`                                                                                                   |
| `StreamAccount`   | `new(address: Pubkey)`                               | `include_owned_accounts(bool)`                                                                                                   |
| `DeployProgram`   | `new(program_id: Pubkey)`, `from_keypair_path(path)` | `so_path`, `so_bytes`, `idl_path`                                                                                                |

Die vollständige JS-Oberfläche finden Sie unter
[JS-Referenz](/docs/tools/surfpool/sdk/js-reference). Die vollständige
Rust-Oberfläche finden Sie unter
[Rust-Referenz](/docs/tools/surfpool/sdk/rust-reference).
