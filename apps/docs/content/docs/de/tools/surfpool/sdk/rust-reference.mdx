Die `surfpool-sdk`-Crate stellt eine kompakte Oberfläche bereit, die auf der
`Surfnet`-Struktur basiert. Diese Seite listet alle öffentlichen Elemente auf;
für eine erklärende Einführung siehe die Anleitungen, beginnend mit
[Übersicht](/docs/tools/surfpool/sdk/overview).

## Crate-Root-Exporte

| Element               | Art       | Hinweise                                                                  |
| --------------------- | --------- | ------------------------------------------------------------------------- |
| `Surfnet`             | Struct    | Laufende Surfnet-Instanz — RPC/WS-Endpunkte, Zahler, Cheatcodes           |
| `SurfnetBuilder`      | Struct    | Fluent-Builder, verwendet von `Surfnet::builder()`                        |
| `Cheatcodes<'_>`      | Struct    | Ausgeliehener Cheatcode-Helfer, zurückgegeben von `Surfnet::cheatcodes()` |
| `SurfnetError`        | Enum      | Alle Fehlervarianten, die das SDK zurückgibt                              |
| `SurfnetResult<T>`    | Typ-Alias | `Result<T, SurfnetError>`                                                 |
| `Keypair`             | Re-Export | `solana_keypair::Keypair`                                                 |
| `Pubkey`              | Re-Export | `solana_pubkey::Pubkey`                                                   |
| `Signer`              | Re-Export | `solana_signer::Signer`                                                   |
| `RpcClient`           | Re-Export | `solana_rpc_client::rpc_client::RpcClient`                                |
| `BlockProductionMode` | Re-Export | `surfpool_types::BlockProductionMode`                                     |
| `SimnetCommand`       | Re-Export | `surfpool_types::SimnetCommand`                                           |
| `SimnetEvent`         | Re-Export | `surfpool_types::SimnetEvent`                                             |
| `SvmFeatureConfig`    | Re-Export | `surfpool_types::SvmFeatureConfig`                                        |

### Lebenszyklus

```rust
impl Surfnet {
    pub fn stop(&mut self) -> SurfnetResult<()>;
}

impl Drop for Surfnet { /* calls stop() if not already stopped */ }
```

`stop` wartet bis zu fünf Sekunden darauf, dass die Laufzeitumgebung ihre Ports
freigibt. Ein Fehler während `Drop` wird stillschweigend ignoriert; ein Fehler
während eines expliziten `stop` gibt einen `SurfnetError::Runtime` zurück.

## SurfnetError

```rust
#[derive(Debug)]
pub enum SurfnetError {
    PortAllocation(String),
    Startup(String),
    Runtime(String),
    Cheatcode(String),
    Aborted(String),
}
```

| Variante         | Ursache                                                                          |
| ---------------- | -------------------------------------------------------------------------------- |
| `PortAllocation` | Konnte keinen freien Port für RPC oder WebSocket belegen.                        |
| `Startup`        | Die Laufzeitumgebung konnte nicht initialisiert werden.                          |
| `Runtime`        | Der Laufzeit-Thread ist abgestürzt oder unerwartet beendet worden.               |
| `Cheatcode`      | Ein Cheatcode-JSON-RPC-Aufruf hat einen Fehler zurückgegeben.                    |
| `Aborted`        | Die Laufzeitumgebung wurde während des Starts heruntergefahren oder abgebrochen. |

Das Crate implementiert `std::error::Error` und `Display` für `SurfnetError` und
funktioniert daher out of the box mit `?` und `anyhow`.
