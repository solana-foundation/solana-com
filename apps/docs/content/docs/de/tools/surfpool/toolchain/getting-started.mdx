Die surfpool CLI kann über das Installationsskript installiert oder aus dem
Quellcode erstellt werden.

## Installationsskript

Um surfpool zu installieren, öffnen Sie ein Terminal und führen Sie folgenden
Befehl aus:

```bash
curl -sL https://run.surfpool.run/ | bash
```

## Aus dem Quellcode installieren

Um die Surfpool-Binärdatei aus dem Quellcode zu kompilieren, klonen Sie das
surfpool-Repository.

```bash
git clone https://github.com/solana-foundation/surfpool.git
```

Navigieren Sie in das neue Verzeichnis.

```bash
cd surfpool
```

Kompilieren Sie anschließend die Binärdatei. Dieser Befehl kompiliert die
Binärdatei und speichert sie in `~/.cargo/bin/surfpool`.

```bash
cargo surfpool-install
```

Stellen Sie abschließend sicher, dass die Binärdatei `surfpool` in Ihrem PATH
verfügbar ist. Dieser Vorgang kann je nach Betriebssystem unterschiedlich sein.
