## ripemd160

`ripemd160` berechnet den Ripemd160-Hash eines Wertes.

### Eingaben

| Name    | Typ                     | Beschreibung                     |
| ------- | ----------------------- | -------------------------------- |
| `value` | buffer \| array[buffer] | Der hex-kodierte Wert zum Hashen |

### Ausgabe

| Name    | Typ    | Beschreibung          |
| ------- | ------ | --------------------- |
| `value` | string | Das gehashte Ergebnis |

```hcl
output "hashed_data" {
    value = ripemd160(encode_hex("hello, world"))
}
// > hashed_data: 0XA3201F82FCA034E46D10CD7B27E174976E241DA2
```


| Name    | Typ    | Beschreibung                      |
| ------- | ------ | --------------------------------- |

### Ausgabe

| Name    | Typ    | Beschreibung          |
| ------- | ------ | --------------------- |
| `value` | string | Das gehashte Ergebnis |

```hcl
output "hashed_data" {
    value = keccak256("hello, world")
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```
