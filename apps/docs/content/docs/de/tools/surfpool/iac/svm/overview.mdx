title: SVM Addon-Übersicht
description: Add-on für Solana und SVM-kompatible Blockchains (Beta)
---

Das SVM `txtx` Plugin ermöglicht die Erstellung von Runbooks, die mit Solana und
SVM-kompatiblen Blockchains interagieren. Das Plugin stellt Hilfsfunktionen
bereit, mit denen Sie Anchor-Programme deployen und Anweisungen für
Instruction-Aufrufe gemäß Programm-IDLs kodieren können. Die Aktionen können
verwendet werden, um gültige Transfer-, Programmaufruf- und
Programm-Deployment-Transaktionen zu erstellen, die über eine Mnemonic-Phrase,
einen geheimen Schlüssel oder über Ihren Browser-Signer signiert werden können.

## Schnellstart

Um das SVM-Addon zu verwenden, deklarieren Sie es in Ihrem Runbook:

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## Funktionen

- **Programm-Deployment**: Deployment von Anchor- und nativen Solana-Programmen
- **Instruction-Erstellung**: Kodierung von Instruction-Aufrufen mithilfe von
  Programm-IDLs
- **Transaktionssignierung**: Unterstützung für Mnemonics, geheime Schlüssel und
  Web-Wallets
- **Konten-Verwaltung**: Abfrage und Verwaltung von Solana-Konten
