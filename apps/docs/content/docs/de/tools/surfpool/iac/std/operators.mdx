## and_bool

`and_bool` gibt das binäre UND der links- und rechtsseitigen Argumente zurück.

### Eingaben

| Name  | Type | Beschreibung        |
| ----- | ---- | ------------------- |
| `lhs` | bool | Ein boolescher Wert |
| `rhs` | bool | Ein boolescher Wert |

### Ausgabe

| Name    | Type | Beschreibung                                                  |
| ------- | ---- | ------------------------------------------------------------- |
| `value` | bool | Das Ergebnis eines binären UND zwischen den beiden Argumenten |

```hcl
output "my_bool" {
  value = false && true
}
// > my_bool: false
```

## div

`div` gibt die ganzzahlige Division des linksseitigen Arguments durch das
rechtsseitige Argument zurück und rundet einen etwaigen Rest auf die nächste
ganze Zahl ab.

### Eingaben

| Name  | Type    | Beschreibung |
| ----- | ------- | ------------ |
| `lhs` | integer | Der Dividend |
| `rhs` | integer | Der Divisor  |

### Ausgabe

| Name    | Type    | Beschreibung                                               |
| ------- | ------- | ---------------------------------------------------------- |
| `value` | integer | Das Ergebnis der Division des Dividenden durch den Divisor |

```hcl
output "my_int" {
  value = 11 / -3
}
// > my_int: -3
```

## eq

`eq` gibt `true` zurück, wenn die Argumente auf der linken und rechten Seite
gleich sind, und `false`, wenn sie es nicht sind.

### Eingaben

| Name  | Typ | Beschreibung    |
| ----- | --- | --------------- |
| `lhs` | any | Beliebiger Wert |
| `rhs` | any | Beliebiger Wert |

### Ausgabe

| Name    | Typ  | Beschreibung                                                          |
| ------- | ---- | --------------------------------------------------------------------- |
| `value` | bool | Das Ergebnis eines Gleichheitsvergleichs zwischen den beiden Eingaben |

```hcl
output "is_eq" {
  value = "arg" == "arg"
}
// > is_eq: true
```

---

## gt

`gt` gibt `true` zurück, wenn das Argument auf der linken Seite größer ist als
das Argument auf der rechten Seite.

### Eingaben

| Name  | Typ                        | Beschreibung            |
| ----- | -------------------------- | ----------------------- |
| `lhs` | integer \| float \| string | Ein vergleichbarer Wert |
| `rhs` | integer \| float \| string | Ein vergleichbarer Wert |

### Ausgabe

| Name    | Typ  | Beschreibung                                                                                                     |
| ------- | ---- | ---------------------------------------------------------------------------------------------------------------- |
| `value` | bool | Das Ergebnis der Prüfung, ob das Argument auf der linken Seite größer ist als das Argument auf der rechten Seite |

```hcl
output "is_gt" {
  value = 2 > 1
}
// > is_gt: true
```

## neq

`neq` gibt `true` zurück, wenn das linke und das rechte Argument nicht gleich
sind, und `false` andernfalls.

### Eingaben

| Name  | Type | Beschreibung                      |
| ----- | ---- | --------------------------------- |
| `lhs` | any  | Ein zu vergleichender Wert        |
| `rhs` | any  | Ein Wert, mit dem verglichen wird |

### Ausgabe

| Name    | Typ  | Beschreibung                                                                 |
| ------- | ---- | ---------------------------------------------------------------------------- |
| `value` | bool | Das Ergebnis einer negierten Gleichheitsprüfung zwischen den beiden Eingaben |

```hcl
output "is_neq" {
  value = "arg" != "arg"
}
// > is_neq: false
```
