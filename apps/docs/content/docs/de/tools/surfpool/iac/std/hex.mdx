## encode_hex

`encode_hex` kodiert einen Buffer oder String als hexadezimalen String mit dem
Präfix 0x.

### Eingaben

| Name    | Type                  | Beschreibung                                                                                                                                   |
| ------- | --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | Der zu kodierende Buffer oder String. Strings, die mit '0x' beginnen, werden als Hex dekodiert; andernfalls werden rohe UTF-8-Bytes verwendet. |

### Ausgabe

| Name    | Type   | Beschreibung                                                     |
| ------- | ------ | ---------------------------------------------------------------- |
| `value` | string | Die Eingabe in ihrer hexadezimalen Darstellung mit dem Präfix 0x |

```hcl
output "encoded_hex" {
    value = encode_hex("hello, world")
}
// > encoded_hex: 0x68656c6c6f2c20776f726c64
```

## decode_hex

`decode_hex` dekodiert einen hexadezimalen String und gibt das Ergebnis als
Buffer zurück.

### Eingaben

| Name         | Type   | Beschreibung                                             |
| ------------ | ------ | -------------------------------------------------------- |
| `hex_string` | string | Der zu dekodierende Hex-String (mit oder ohne Präfix 0x) |

### Ausgabe

| Name    | Type   | Beschreibung                         |
| ------- | ------ | ------------------------------------ |
| `value` | buffer | Der dekodierte Hex-String als Buffer |

```hcl
output "decoded_hex" {
    value = decode_hex("0x68656c6c6f2c20776f726c64")
}
// > decoded_hex: 0x68656c6c6f2c20776f726c64
```
