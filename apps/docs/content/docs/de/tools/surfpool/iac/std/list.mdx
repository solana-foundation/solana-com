## index

`index` ruft den Eintrag aus einer Liste am angegebenen Index ab.

### Eingaben

| Name    | Typ                                              | Beschreibung                                  |
| ------- | ------------------------------------------------ | --------------------------------------------- |
| `list`  | array[string] \| array[integer] \| array[buffer] | Die Liste, aus der ein Eintrag abgerufen wird |
| `index` | integer                                          | Der Index des abzurufenden Eintrags           |

### Ausgabe

| Name    | Typ    | Beschreibung                                   |
| ------- | ------ | ---------------------------------------------- |
| `value` | string | Der Eintrag aus der Liste am angegebenen Index |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
