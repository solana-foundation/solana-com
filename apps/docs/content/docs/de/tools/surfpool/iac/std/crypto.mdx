## secp256k1_recover

`secp256k1_recover` stellt einen öffentlichen Schlüssel aus einer
secp256k1-Signatur wieder her.

### Eingaben

| Name        | Type                    | Beschreibung                                                                     |
| ----------- | ----------------------- | -------------------------------------------------------------------------------- |
| `message`   | buffer \| array[buffer] | Der Hash der ursprünglichen Nachricht, die signiert wurde                        |
| `signature` | buffer \| array[buffer] | Die Signatur, die mit dem elliptischen Kurvenalgorithmus secp256k1 erzeugt wurde |

### Ausgabe

| Name    | Type   | Beschreibung                                 |
| ------- | ------ | -------------------------------------------- |
| `value` | string | Der wiederhergestellte öffentliche Schlüssel |

```hcl
output "recovered_public_key" {
    value = secp256k1_recover(
        "0x6a2ce4b8aab1ef79aa1aa617cf6b72d7146857b83055e203b67c5177faef212c",
        "0x0165a85a1e64d7157d678d177bc8a9e6bfb8d750458d52a31c34abe1e56475b5eb62f183a5e6ddbced38fca93a8ff1c73b4ce66231e39392572af916b5303fbe12"
    )
}
// > recovered_public_key: 0x03b3e0a76b292b2c83fc0ac14ae6160d0438ebe94e14bbb5b7755153628886e08e
```
