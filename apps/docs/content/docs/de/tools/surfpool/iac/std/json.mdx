## jq

Die Funktion `jq` ermöglicht das Slicen, Filtern und Transformieren von
JSON-Daten. Weitere Details finden Sie in der
[jq-Dokumentation](https://jqlang.github.io/jq/manual/).

### Eingaben

| Name           | Typ              | Beschreibung                                                                         |
| -------------- | ---------------- | ------------------------------------------------------------------------------------ |
| `decoded_json` | string \| object | Ein JSON-Objekt                                                                      |
| `query`        | string           | Eine JSON-Abfrage. Siehe die [jq-Dokumentation](https://jqlang.github.io/jq/manual/) |

### Ausgabe

| Name    | Typ           | Beschreibung                  |
| ------- | ------------- | ----------------------------- |
| `value` | array[string] | Das Ergebnis der `jq`-Abfrage |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
