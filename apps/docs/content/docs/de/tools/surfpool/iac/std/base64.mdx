## encode_base64

`encode_base64` kodiert einen Puffer oder eine Zeichenkette als
Base64-Zeichenkette.

### Eingaben

| Name    | Typ                         | Beschreibung                                                                                                                                                                 |
| ------- | --------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | Puffer, Zeichenkette, Addon | Der zu kodierende Puffer oder die zu kodierende Zeichenkette. Zeichenketten, die mit '0x' beginnen, werden als Hex dekodiert; andernfalls werden rohe UTF-8-Bytes verwendet. |

### Ausgabe

| Name    | Typ          | Beschreibung                                 |
| ------- | ------------ | -------------------------------------------- |
| `value` | Zeichenkette | Die Eingabe, kodiert als Base64-Zeichenkette |

```hcl
output "encoded" {
    value = encode_base64("0x48656c6c6f20776f726c6421")
}
// > encoded: SGVsbG8gd29ybGQh
```

## decode_base64

`decode_base64` dekodiert eine Base64-kodierte Zeichenkette und gibt das
Ergebnis als Puffer zurück.

### Eingaben

| Name            | Typ          | Beschreibung                            |
| --------------- | ------------ | --------------------------------------- |
| `base64_string` | Zeichenkette | Die zu dekodierende Base64-Zeichenkette |

### Ausgabe

| Name    | Typ    | Beschreibung                                  |
| ------- | ------ | --------------------------------------------- |
| `value` | Puffer | Die dekodierte Base64-Zeichenkette als Puffer |

```hcl
output "decoded" {
    value = decode_base64("SGVsbG8gd29ybGQh")
}
// > decoded: 0x48656c6c6f20776f726c6421
```
