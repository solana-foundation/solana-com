title: Base58-Funktionen
description: Standardbibliotheksfunktionen für Base58-Kodierung und -Dekodierung
---

## encode_base58

`encode_base58` kodiert einen Buffer oder String als Base58-Zeichenkette.

### Eingaben

| Name    | Typ                   | Beschreibung                                                                                                                                         |
| ------- | --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | Buffer, String, Addon | Der zu kodierende Buffer oder String. Zeichenketten, die mit '0x' beginnen, werden als Hex dekodiert; andernfalls werden rohe UTF-8-Bytes verwendet. |

### Ausgabe

| Name    | Typ    | Beschreibung                                 |
| ------- | ------ | -------------------------------------------- |
| `value` | String | Die Eingabe, kodiert als Base58-Zeichenkette |

```hcl
output "encoded" {
    value = encode_base58("0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8")
}
// > encoded: CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF
```

## decode_base58

`decode_base58` dekodiert eine Base58-kodierte Zeichenkette und gibt das
Ergebnis als Buffer zurück.

### Eingaben

| Name            | Typ    | Beschreibung                            |
| --------------- | ------ | --------------------------------------- |
| `base58_string` | String | Die zu dekodierende Base58-Zeichenkette |

### Ausgabe

| Name    | Typ    | Beschreibung                                  |
| ------- | ------ | --------------------------------------------- |
| `value` | Buffer | Die dekodierte Base58-Zeichenkette als Buffer |

```hcl
output "decoded" {
    value = decode_base58("CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF")
}
// > decoded: 0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8
```
