## assert_eq

`assert_eq` überprüft, ob zwei Werte gleich sind.

### Eingaben

| Name    | Typ | Beschreibung                      |
| ------- | --- | --------------------------------- |
| `left`  | any | Ein zu vergleichender Wert        |
| `right` | any | Der Wert, mit dem verglichen wird |

### Ausgabe

| Name    | Typ                   | Beschreibung               |
| ------- | --------------------- | -------------------------- |
| `value` | addon(std::assertion) | Das Ergebnis der Assertion |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```

## assert_gt

`assert_gt` überprüft, ob der linke Wert größer als der rechte Wert ist.

### Eingaben

| Name    | Typ              | Beschreibung                                    |
| ------- | ---------------- | ----------------------------------------------- |
| `left`  | integer \| float | Ein Integer oder Float zum Vergleichen          |
| `right` | integer \| float | Ein Integer oder Float, mit dem verglichen wird |

### Ausgabe

| Name    | Typ                   | Beschreibung               |
| ------- | --------------------- | -------------------------- |
| `value` | addon(std::assertion) | Das Ergebnis der Assertion |

```hcl
output "assertion" {
    value = std::assert_gt(action.example.result, 1)
}
```

| `left`  | integer \| float | Ein Integer oder Float zum Vergleichen    |
