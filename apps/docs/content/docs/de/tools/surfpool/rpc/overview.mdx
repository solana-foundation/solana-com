Surfnet ist der lokale Solana-Netzwerksimulator von Surfpool und wurde
entwickelt, um Entwicklern eine leistungsstarke Testumgebung bereitzustellen,
die das Verhalten der Solana-Blockchain nachbildet – ohne den Overhead eines
vollständigen Validatoren-Knotens.

## Erste Schritte

Starten Sie eine Surfnet-Instanz über die Surfpool-CLI:

```bash
surfpool start
```

Dadurch wird eine lokale Surfnet-Instanz auf `http://localhost:8899` gestartet,
mit allen verfügbaren RPC-Endpunkten.

## Wichtigste Funktionen

- **Vollständige RPC-Kompatibilität**: Unterstützt die vollständige
  Solana-RPC-API
- **Schnelle Iteration**: Keine Wartezeit auf Block-Bestätigungen
- **Zustandsmanipulation**: Cheatcodes zur direkten Änderung des
  Blockchain-Zustands
- **Mainnet-Fork**: Fork vom Mainnet, um mit echten Konten und Programmen zu
  testen
- **Geyser-Plugin-Unterstützung**: Verbinden Sie Ihre Geyser-Plugins für
  Echtzeit-Daten-Streaming
