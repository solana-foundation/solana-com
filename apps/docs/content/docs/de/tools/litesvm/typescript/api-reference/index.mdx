## Kurzreferenz

### Plugin-Einrichtung

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### Kontenverwaltung (client.svm)

| Methode                                | Beschreibung                             |
| -------------------------------------- | ---------------------------------------- |
| `setAccount(account)`                  | Kontostatus direkt setzen                |
| `getAccount(address)`                  | Kontodaten abrufen                       |
| `getBalance(address)`                  | SOL-Guthaben abrufen                     |
| `airdrop(address, lamports)`           | Ein Konto aufstocken                     |
| `minimumBalanceForRentExemption(size)` | Mindestbetrag für Mietfreiheit berechnen |

### Programmverwaltung (client.svm)

| Methode                               | Beschreibung                 |
| ------------------------------------- | ---------------------------- |
| `addProgram(programId, bytes)`        | Programm aus Bytes laden     |
| `addProgramFromFile(programId, path)` | Programm aus .so-Datei laden |

### Transaktionen (client.svm)

| Methode                   | Beschreibung                       |
| ------------------------- | ---------------------------------- |
| `sendTransaction(tx)`     | Transaktion ausführen              |
| `simulateTransaction(tx)` | Simulation ohne Zustandsänderungen |
| `latestBlockhash()`       | Aktuellen Blockhash abrufen        |
| `expireBlockhash()`       | Zu neuem Blockhash wechseln        |

### Konfiguration (client.svm)

| Methode                     | Beschreibung                                  |
| --------------------------- | --------------------------------------------- |
| `withSigverify(bool)`       | Signaturprüfung aktivieren/deaktivieren       |
| `withBlockhashCheck(bool)`  | Blockhash-Validierung aktivieren/deaktivieren |
| `withSysvars()`             | Sysvar-Konten aktivieren                      |
| `withBuiltins()`            | Integrierte Programme aktivieren              |
| `withPrecompiles()`         | Vorkompilierte Programme aktivieren           |
| `withTransactionHistory(n)` | Letzte n Transaktionen speichern              |
| `withComputeBudget(budget)` | Compute-Limits setzen                         |
| `withLamports(amount)`      | Standard-lamports setzen                      |
| `withLogBytesLimit(n)`      | Ausgabelimit für Logs setzen                  |

### Zeit & Sysvars (client.svm)

| Methode                    | Beschreibung                      |
| -------------------------- | --------------------------------- |
| `warpToSlot(slot)`         | Zu einem bestimmten slot springen |
| `getClock()`               | Clock-Sysvar abrufen              |
| `getRent()`                | rent-Sysvar abrufen               |
| `getEpochSchedule()`       | epoch-Zeitplan abrufen            |
| `getLastRestartSlot()`     | Letzten Neustart-slot abrufen     |
| `setLastRestartSlot(slot)` | Letzten Neustart-slot setzen      |

### RPC (client.rpc)

| Methode                          | Beschreibung                          |
| -------------------------------- | ------------------------------------- |
| `getAccountInfo(address)`        | Einzelnes Konten abrufen (nur base64) |
| `getMultipleAccounts(addresses)` | Mehrere Konten abrufen                |
| `getLatestBlockhash()`           | Aktuellen Blockhash abrufen           |
