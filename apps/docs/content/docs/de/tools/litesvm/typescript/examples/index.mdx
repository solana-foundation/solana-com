## Beispiele ausführen

Alle Beispiele können mit `tsx` ausgeführt werden, wobei einige
Adressen/Variablen durch tatsächliche Werte ersetzt werden müssen.

```bash
# Install dependencies
pnpm add @solana/kit @solana/kit-plugin-litesvm @solana/kit-plugin-signer @solana-program/system

# Run an example
npx tsx example.ts
```
