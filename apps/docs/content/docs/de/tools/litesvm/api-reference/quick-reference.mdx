## Kurzübersicht

| Methode                                        | Beschreibung                                                                                                               |
| ---------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| `LiteSVM::new()`                               | Neue grundlegende LiteSVM-Testumgebung erstellen                                                                           |
| `LiteSVM::default()`                           | Gibt den „Standardwert“ für einen Typ zurück                                                                               |
| `airdrop(&Address, lamports)`                  | Lädt das Konten mit den angegebenen lamports auf                                                                           |
| `get_account(&Address)`                        | Gibt alle Informationen zurück, die mit dem Konten der angegebenen Adresse verknüpft sind                                  |
| `set_account(Address, Account)`                | Setzt alle Informationen, die mit dem Konten der angegebenen Adresse verknüpft sind                                        |
| `get_balance(&Address)`                        | Gibt den lamport-Saldo des angegebenen Konten zurück                                                                       |
| `minimum_balance_for_rent_exemption(data_len)` | Gibt das Mindestguthaben zurück, das erforderlich ist, um ein Konten mit der angegebenen Datenlänge rent-befreit zu machen |
| `add_program(program_id, &bytes)`              | Fügt ein SBF-Programm zur Testumgebung hinzu                                                                               |
| `add_program_from_file(program_id, path)`      | Fügt ein SBF-Programm aus der angegebenen Datei zur Testumgebung hinzu                                                     |
| `send_transaction(tx)`                         | Übermittelt eine signierte Transaktion                                                                                     |
| `simulate_transaction(tx)`                     | Simuliert eine Transaktion ohne Zustandsänderung                                                                           |
| `get_transaction(&signature)`                  | Ruft eine Transaktion aus dem Verlauf ab                                                                                   |
| `latest_blockhash()`                           | Gibt den aktuellen Blockhash zurück                                                                                        |
| `expire_blockhash()`                           | Erzwingt den Ablauf des aktuellen Blockhash                                                                                |
| `warp_to_slot(slot)`                           | Setzt die Uhr auf den angegebenen slot vor                                                                                 |
| `get_sysvar<T>()`                              | Ruft eine Sysvar aus der Testumgebung ab                                                                                   |
| `set_sysvar<T>(&value)`                        | Setzt die Sysvar in der Testumgebung                                                                                       |
| `with_feature_accounts()`                      | Fügt Feature-Gate-Konten zur Testumgebung hinzu                                                                            |
| `set_invocation_inspect_callback(callback)`    | Registriert einen Pre/Post-Aufruf-Callback (erfordert das Feature `invocation-inspect-callback`)                           |
