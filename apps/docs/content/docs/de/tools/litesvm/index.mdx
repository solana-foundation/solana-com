---
title: LiteSVM
description:
  Blitzschnelles Testen von Solana-Programmen mit einer In-Process-VM. Testen
  Sie Ihre Programme schneller – ohne Netzwerk-Overhead oder
  Validatoren-Prozesse.
---

LiteSVM ist eine leichtgewichtige Bibliothek zum Testen von Solana-Programmen.
Sie funktioniert, indem sie eine In-Process-Solana-VM erstellt, die speziell für
Programmentwickler optimiert ist. Im Gegensatz zu anderen Testansätzen, die
separate Validatoren-Prozesse starten, bettet LiteSVM die VM direkt in Ihre
Tests ein, wodurch diese erheblich schneller ausgeführt werden. Als weiterer
Bruch mit der Tradition bietet es eine ergonomische API mit sinnvollen
Standardwerten und umfangreichen Konfigurationsmöglichkeiten für alle, die diese
benötigen.
