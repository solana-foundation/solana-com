**Commerce Kit ist eine Sammlung von Tools zum Erstellen von
Commerce-Erlebnissen auf Solana.** Von Drop-in-Zahlungsschaltflächen bis zu
Headless-Primitiven bietet Commerce Kit alles, was Sie benötigen, um
Krypto-Zahlungen zu akzeptieren.

## Warum Commerce Kit?

- **Drop-in-Komponenten**: Vorgefertigter PaymentButton mit Wallet-Verbindung,
  Token- Auswahl und Transaktionsabwicklung
- **Framework-agnostisch**: Verwenden Sie Headless-Primitive in jedem
  JavaScript-Framework
- **Solana Pay**: Vollständige Solana Pay-Unterstützung mit QR-Code-Generierung
- **Typsicher**: Vollständige TypeScript-Unterstützung über alle Pakete hinweg
- **Wallet Standard**: Kompatibel mit allen Wallet Standard-Wallets (Phantom,
  Solflare, Backpack usw.)

## Schnellstart

Der schnellste Einstieg erfolgt über die `PaymentButton`-Komponente:

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## Installation

Wählen Sie die benötigten Commerce Kit-Pakete:

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
