## Cargo

Installieren Sie die Beta-Version direkt von crates.io:

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

Offizielle multiplattformfähige Docker-Images (amd64/arm64) werden in der GitHub
Container Registry veröffentlicht:

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

Verfügbare Tags:

- `latest` — Neueste stabile Version
- `beta` — Neueste Pre-Release-Version
- `v<version>` — Spezifische Version (z. B. `v2.2.0-beta.7`)

### Docker Compose

Für benutzerdefinierte Builds oder zur Einbindung von Redis für Caching und
Unterstützung von Nutzungslimits:

- [Beispiel-Dockerfile](https://github.com/solana-foundation/kora/blob/main/examples/deploy/Dockerfile)
- [Docker Compose mit Redis](https://github.com/solana-foundation/kora/blob/main/docker-compose.yml)

### Ausführung mit Docker

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## TypeScript SDK

Das Beta-SDK ist auf npm verfügbar:

```bash
pnpm add @solana/kora@beta
```

Siehe [SDK-Änderungen](/docs/tools/kora/beta/sdk-changes) für neue APIs.

## Installation überprüfen

```bash
kora --version
```
