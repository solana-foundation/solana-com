---
title: getBlockhash
description:
  Ruft den neuesten Blockhash vom Solana RPC ab, mit dem der Kora-Server
  verbunden ist.
---

## JSON-RPC-Anfrage

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getBlockhash",
  "params": []
}
```

## JSON-RPC-Antwort

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "blockhash": "base58Blockhash"
  }
}
```

## cURL-Beispiel

```bash
curl -X POST http://localhost:8080 \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"getBlockhash","params":[]}'
```

## TypeScript SDK

```typescript
const { blockhash } = await client.getBlockhash();
console.log("Current blockhash:", blockhash);
```
