Beginnen Sie mit Solana Attestations, indem Sie die erforderlichen Pakete
installieren und Ihre Entwicklungsumgebung einrichten.

## TypeScript/JavaScript

Installieren Sie die Solana Attestations-Bibliothek mit Ihrem bevorzugten
Paketmanager:

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
