## KI-Entwicklungstools

Beginnen Sie mit unserem Leitfaden zu
[wie Sie mit KI-Tools auf Solana starten](/developers/guides/getstarted/intro-to-ai).

### Solana MCP

[Solana MCP](https://mcp.solana.com/) (Machine Conversation Protocol) integriert
Solana-Wissen direkt in KI-gestützte Coding-Tools wie Claude, Cursor und
Windsurf. Es ist auf Solana-Dokumentation, Anchor-Framework, Programmbeispiele
und Stack Exchange Q&A trainiert - nützlich für die schnellere Entwicklung von
Zahlungsintegrationen.

### KI-Agent-Frameworks

KI-Agenten können autonom Zahlungstransaktionen auf Solana ausführen. Diese
Frameworks ermöglichen die Entwicklung von Agenten, die Zahlungen verarbeiten,
Transaktionen überwachen und Zahlungsworkflows automatisieren:

| Framework        | Beschreibung                                                                        | Links                                                                                     |
| ---------------- | ----------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | Tools zum Erstellen zahlungsfähiger Agenten, kompatibel mit LangChain und Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | Eine Sammlung vorgefertigter Skills zum Erstellen von Agenten auf Solana            | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | Multiplattform-Agent-Framework mit Solana-Plugin für X-, Telegram- und Discord-Bots | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Cross-Chain-Toolkit mit Solana-Zahlungsaktionen                                     | [GitHub](https://github.com/goat-sdk/goat)                                                |

### KI-Programmentwicklung

[Codigo](https://codigo.ai/) ist eine End-to-End-KI-Entwicklungsplattform für
Solana.
