## System Program

System Program (`11111111111111111111111111111111`) on ainoa ohjelma, joka voi
luoda uusia tilejä. Kaikki uudet tilit ovat aluksi System Programin
omistuksessa, vaikka omistajuus tyypillisesti siirretään luomisen yhteydessä.
System Program kuluttaa
[`DEFAULT_COMPUTE_UNITS`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L282)
(150 CU) käskyä kohden.

### System Program -käskyviittaus

| Käsky                                                                                                                  | Kuvaus                                                                                                                                                                                                                      |
| ---------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [`CreateAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L295)          | Luo uuden tilin. Varaa `space` tavua, määrittää `owner` ja siirtää `lamports` maksajalta. Uuden tilin ja maksajan on allekirjoitettava.                                                                                     |
| [`CreateAccountWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L318)  | Sama kuin `CreateAccount`, mutta johtaa uuden tilin osoitteen tilin pubkey-arvosta, seed-merkkijonosta ja omistajaohjelmasta. Tilin, jonka pubkey-arvoa käytettiin osoitteen johtamiseen, ja maksajan on allekirjoitettava. |
| [`Assign`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L343)                 | Asettaa tilin omistajan. Tilin on allekirjoitettava. Ei tee mitään, jos omistaja on jo kyseinen.                                                                                                                            |
| [`AssignWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L484)         | Sama kuin `Assign`, mutta seed-johdetulle osoitteelle. Tilin, jonka pubkey-arvoa käytettiin osoitteen johtamiseen, on allekirjoitettava.                                                                                    |
| [`Transfer`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L353)               | Siirtää lamportteja tilien välillä. Lähettäjän on allekirjoitettava.                                                                                                                                                        |
| [`TransferWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L357)       | Sama kuin `Transfer`, mutta seed-johdetusta osoitteesta. Tilin, jonka pubkey-arvoa käytettiin osoitteen johtamiseen, on allekirjoitettava.                                                                                  |
| [`Allocate`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L452)               | Asettaa tilin datan pituuden. Tilin on allekirjoitettava, sillä on oltava tyhjä data ja System Programin on omistettava se. Enintään 10 MiB.                                                                                |
| [`AllocateWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L462)       | Sama kuin `Allocate`, mutta seed-johdetulle osoitteelle. Määrittää myös omistajan. Tilin, jonka pubkey-arvoa käytettiin osoitteen johtamiseen, on allekirjoitettava.                                                        |
| [`AdvanceNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L374)    | Päivittää nonce-arvon nykyiseen lohkohajautuun. Nonce-valtuutuksen on allekirjoitettava.                                                                                                                                    |
| [`InitializeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L412) | Alustaa tilin nonce-tiliksi annetulla valtuutuksella. Tilin on oltava rent-vapautettu.                                                                                                                                      |
| [`AuthorizeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L432)  | Vaihtaa nonce-valtuutuksen. Nykyisen valtuutuksen on allekirjoitettava.                                                                                                                                                     |
| [`WithdrawNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L392)   | Nostaa lamportteja nonce-tililtä. Valtuutuksen on allekirjoitettava. Jos ei suljeta, jäljellä olevan saldon on katettava rent.                                                                                              |
| [`UpgradeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L437)    | Päivittää nonce-tilin vanhasta nykyiseen versiomuotoon. Tilin on oltava kirjoitettavissa ja sisällettävä vanhan version nonce.                                                                                              |

## Latausohjelmat

Jokainen ohjelma on latausohjelman omistuksessa. Latausohjelmat käsittelevät
omistamiensa ohjelmien käyttöönoton, päivityksen ja suorituksen. Katso
[Latausohjelmat](/docs/core/programs/program-deployment#loader-programs)
täydellinen luettelo latausohjelmista ja niiden osoitteista.
