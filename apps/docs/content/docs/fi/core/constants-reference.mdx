Tämä sivu on yksittäinen viiteopas kaikille ajonaikaisille vakioille, jotka
mainitaan ydinkonseptisivuilla. Jokainen vakio linkittää sen lähdemääritelmään
[agave](https://github.com/anza-xyz/agave/tree/v3.1.8)-validaattoriasiakkaassa
tai [solana-sdk](https://github.com/anza-xyz/solana-sdk):ssa.
