Tili on Solanan perustavanlaatuinen tietoyksikkö tilan tallentamiseen. Verkko
tallentaa kaiken tilan avain-arvo-varastoon, jossa jokainen avain on 32-tavuinen
osoite ja jokainen arvo on tili.

## Keskeiset faktat

- **Rakenne**: Jokaisella tilillä on samat
  [viisi kenttää](/docs/core/accounts/account-structure): lamports, data, owner,
  executable, rent_epoch.
- **Osoite**: Jokainen tili tunnistetaan ainutlaatuisella 32-tavuisella
  osoitteella (joko Ed25519-julkinen avain tai [PDA](/docs/core/pda)).
- **Omistajuus**: Vain tilin omistaja-ohjelma voi muokata sen dataa tai
  veloittaa lamportteja. Mikä tahansa ohjelma voi hyvittää lamportteja mille
  tahansa kirjoitettavalle tilille.
- **Rent**: Jokaisen tilin on säilytettävä vähimmäislamporttisaldo, joka on
  verrannollinen sen datakokoon pysyäkseen ketjussa.
