## Solana Client

`@solana/client` on päätön Solana-ajoympäristö, joka yhdistää RPC:n, lompakot,
transaktiot ja tilaukset yhteen säilöön.

| Paketti        | Kuvaus                     | GitHub                                                                                |
| -------------- | -------------------------- | ------------------------------------------------------------------------------------- |
| @solana/client | Päätön client-ajoympäristö | [Lähde](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [Opas: @solana/client-yleiskatsaus](/docs/frontend/client)

## Solana React Hooks

`@solana/react-hooks` lisää React-providerin ja hookit `@solana/client`-paketin
päälle, käyttäen samaa ajoympäristöä ja välimuistia.

| Paketti             | Kuvaus                                    | GitHub                                                                                     |
| ------------------- | ----------------------------------------- | ------------------------------------------------------------------------------------------ |
| @solana/react-hooks | React-provider + hookit käyttöliittymille | [Lähde](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [Opas: @solana/react-hooks](/docs/frontend/react-hooks)
