## Solana Kit

`@solana/kit` on suositeltu TypeScript SDK Solana-kehitykseen. Voit koota
asiakkaan koostettavista liitännäisistä, lisätä React-sidokset tarvittaessa ja
ottaa käyttöön generoidut ohjelma-asiakkaat kutsuttaville ketjussa oleville
ohjelmille.

| Paketti                   | Kuvaus                                                  | GitHub                                                                                |
| ------------------------- | ------------------------------------------------------- | ------------------------------------------------------------------------------------- |
| @solana/kit               | Ydin-SDK: RPC, transaktiot, koodekit, allekirjoittajat  | [Lähde](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | React-sidokset: `ClientProvider` ja hookit              | [Lähde](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | RPC-yhteys ja transaktioiden lähetys                    | [Lähde](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | Maksajan, identiteetin ja allekirjoittajan liitännäiset | [Lähde](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Wallet Standard -yhteys ja React-hookit                 | [Lähde](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

## Solana Web3.js

`@solana/web3.js` on Solanan vanha TypeScript SDK. Uusien sovellusten tulisi
käyttää `@solana/kit`:ää; katso
[Siirtyminen Kitiin](/docs/frontend/web3-compat).
