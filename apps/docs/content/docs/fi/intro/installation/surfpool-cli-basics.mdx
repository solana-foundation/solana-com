Surfpool on paikallinen kehitystyökalu, joka yksinkertaistaa Solana-kehitystä
lataamalla automaattisesti ohjelmat ja tilit klusterista (mainnet, devnet).
Käytä Surfpool CLI:tä käynnistääksesi paikallisen validatorin ohjelmien ja
transaktioiden testaamiseen.

Lisätietoja löydät [Surfpool-dokumentaatiosta](https://docs.surfpool.run/).

## Käynnistä paikallinen validator

Käynnistääksesi paikallisen validatorin, suorita seuraava komento:

```terminal
$ surfpool start
```

Oletusarvoisesti Surfpool lataa automaattisesti kaikki ohjelmat ja tilit, joista
transaktiosi ovat riippuvaisia, mainnetistä paikalliseen validatoriin.

## Surfpool Studio

Surfpool Studio tarjoaa verkkopohjaisen käyttöliittymän paikallisen validatorin
kanssa vuorovaikuttamiseen. Käytä sitä:

- Transaktioiden tietojen tarkasteluun
- Tilitietojen tutkimiseen
- SOL-tokenien ja muiden tokenien ilmapudotukseen käyttöliittymän faucetin
  avulla

Surfpool Studion käyttäminen:

1. Käynnistä paikallinen validatorisi komennolla `surfpool start`
2. Avaa selaimesi ja siirry osoitteeseen
   [http://127.0.0.1:18488](http://127.0.0.1:18488)
