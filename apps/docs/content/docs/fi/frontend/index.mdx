Frontend-kehitys Solanassa sisältää työskentelyä ohjelmien, lompakoiden ja
suosittujen JavaScript-kehysten kuten Reactin kanssa. Näiden komponenttien
kanssa vuorovaikuttaminen vaatii yhteyden käsittelyä, transaktioiden luomista ja
lukemista Solana- tileiltä.

[Solana Kit](https://github.com/anza-xyz/kit) (`@solana/kit`) on tavallinen
JavaScript SDK tähän tarkoitukseen. Kootaan asiakas pienistä, yhteen sopivista
liitännäisistä, lisätään React-sidokset tarvittaessa ja otetaan käyttöön
generoidut ohjelma- asiakkaat kutsuttavia lohkoketjuohjelmia varten.

## Tärkeimmät kirjastot

<Cards>
  <Card title="@solana/kit" href="https://github.com/anza-xyz/kit">
    - Ydinkirjasto: RPC, koodekit, allekirjoittajat ja transaktiot - Tarjoaa
    `createClient`, liitännäisisännän, jota kaikki muut kirjastot laajentavat
  </Card>
  <Card title="Kit plugins" href="https://github.com/anza-xyz/kit-plugins">
    - Kokoa asiakas `.use(...)`-liitännäisillä: RPC-yhteys, lompakko,
    allekirjoittaja ja transaktion suunnittelu - Julkaistu
    `@solana/kit-plugin-*`-paketteina
  </Card>
  <Card
    title="@solana/react"
    href="https://github.com/anza-xyz/kit/tree/main/packages/react"
  >
    - React-sidokset Kit-asiakkaalle - `ClientProvider` sekä koukut lukemiseen,
    tilauksiin ja lompakkotilaan
  </Card>
  <Card title="@solana-program/*" href="https://github.com/solana-program">
    - Generoidut asiakkaat lohkoketjuohjelmille (System, Token ja muut) -
    Tarjoavat ohjerakentajat ja asiakaslisäosat, kuten `tokenProgram()`
  </Card>
</Cards>

<Callout type="warn" title="web3.js v1 and wallet-adapter are legacy">
  Monet Solana-projektit käyttävät edelleen vanhempaa `@solana/web3.js` (v1)
  -kirjastoa ja `@solana/wallet-adapter-*`. Uusissa projekteissa suositellaan
  `@solana/kit`-kirjastoa kit- liitännäisten kanssa; Wallet Standard -etsintä
  (`@solana/kit-plugin-wallet` kautta) korvaa wallet-adapterin moderneja
  lompakkoja varten.
</Callout>

- [Kit-asiakasohjeet](/docs/frontend/client): rakenna asiakas liitännäisistä
  RPC:tä, allekirjoittamista ja transaktioiden lähettämistä varten.
- [React-ohjeet](/docs/frontend/react-hooks): `@solana/react`-sidokset ja
  lompakko- koukut samaan asiakkaaseen kerrostettuna.
- [Next.js-opas](/docs/frontend/nextjs-solana): täydellinen yhdistä-lompakko- ja
  SOL-siirtosovellus App Routerissa.
- [Siirtyminen Kitiin](/docs/frontend/web3-compat): miten web3.js- ja
  wallet-adapter- käsitteet vastaavat Kitin käsitteitä.
