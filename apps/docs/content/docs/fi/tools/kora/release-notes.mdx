## Kora v2.2.0-beta.6

_23.3.2026_ |
[GitHub-julkaisu](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.6)
| `cargo install kora-cli@2.2.0-beta.6`

Esijulkaisu Jito-bundle-tuella, uusilla RPC-metodeilla ja parannetuilla
tietoturvaominaisuuksilla.

- Lisätty Jito-bundle-tuki metodeilla `signBundle`, `signAndSendBundle` ja
  `estimateBundleFee`
- Lisätty `getVersion` RPC-metodi
- Lisätty Lighthouse-maksajan suojaus
- Lisätty reCAPTCHA v3 -bottisuojaus allekirjoituspäätepisteille
- Lisätty lompakon käyttörajoitukset Redis-tuella
- Lisätty viralliset Docker-kuvat GHCR:ssä
- Lisätty tarkat käyttörajoitussäännöt
- SDK: `KoraClient` ottaa nyt asetukset objektina; uusi `koraPlugin`
  `@solana/kit`-kirjastolle

Katso [Beta-dokumentaatio](/docs/tools/kora/beta) lisätietojen saamiseksi.

## Kora TypeScript SDK v0.2.0

_9.3.2026_ |
[GitHub-julkaisu](https://github.com/solana-foundation/kora/releases/tag/ts-sdk-v0.2.0)
| `pnpm add @solana/kora@0.2.0`

Kit-asiakasohjelman liitännäisarkkitehtuuri koostettaville
gasless-transaktioille.

- Lisätty `createKitKoraClient()` Kit-liitännäisten koostamisella (planner-,
  executor-, payer-liitännäiset)
- Mahdollistaa koostettavuuden Kit-ohjelma-liitännäisten kuten `tokenProgram()`
  kanssa
- Lisätty Token-2022-tuki `tokenProgramId`-asetusvaihtoehdon kautta
- Lisätty laskentabudjetti-ohjeiden tuki simulaatiopohjaisella CU-estimoinnilla
- Lisätty puuttuva `signature`-kenttä `signAndSendTransaction`-vastaukseen
- Palauttaa `NoopSigner` `getPaymentInstruction`-metodista Kit v6
  -allekirjoittajien deduplikointia varten
- Nostettu `@solana/kit`-peer-riippuvuus versioon `^6.1.0`

Katso [Kit Client -opas](/docs/tools/kora/guides/kit-client).

## Kora v2.0.0

_19.11.2025_ |
[GitHub-julkaisu](https://github.com/solana-foundation/kora/releases/tag/v2.0.0)
| `cargo install kora-cli@2.0.0`

Suuri julkaisu uudella arkkitehtuurilla.

- Esitelty uusi arkkitehtuuri Kora-solmulle

- Siirretty Solana SDK v3.0.x:ään ja päivitetty kaikki riippuvuudet

  ja `signTransaction`-metodeista
