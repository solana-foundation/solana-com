## Cargo

Asenna beta-julkaisu suoraan crates.io-palvelusta:

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

Viralliset monialustaisia Docker-kuvat (amd64/arm64) julkaistaan GitHub
Container Registryyn:

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

Saatavilla olevat tagit:

- `latest` — Uusin vakaa julkaisu
- `beta` — Uusin esijulkaisuversio
- `v<version>` — Tietty versio (esim. `v2.2.0-beta.7`)

### Suorittaminen Dockerilla

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## TypeScript SDK

Beta-SDK on saatavilla npm:ssä:

```bash
pnpm add @solana/kora@beta
```

Katso [SDK-muutokset](/docs/tools/kora/beta/sdk-changes) uusien rajapintojen
osalta.
