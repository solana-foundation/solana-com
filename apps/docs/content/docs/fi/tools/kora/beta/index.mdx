_Saatavilla GitHubissa:
[`v2.2.0-beta.7`](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.7)_

Kora v2.2.0-beta tuo Jito-nipputuen, uudet RPC-metodit, Docker-imagen,
parannettuja turvallisuusominaisuuksia ja SDK-parannuksia.

> **Tämä on esijulkaisu.** Tässä dokumentoidut ominaisuudet ovat saatavilla
> `v2.2.0-beta.7`-versiossa ja voivat muuttua ennen vakaata julkaisua. Vakaan
> dokumentaation löydät [Koran pääsivuilta](/docs/tools/kora).

## Pääkohdat

### Jito-nipputuki

Lähetä jopa 5 transaktiota atomisesti Jiton lohkomoottorin kautta. Uudet
RPC-metodit:

- [`signBundle`](/docs/tools/kora/beta/bundle-methods#signbundle) — Allekirjoita
  nippu lähettämättä sitä
- [`signAndSendBundle`](/docs/tools/kora/beta/bundle-methods#signandsendbundle)
  — Allekirjoita ja lähetä Jitolle
- [`estimateBundleFee`](/docs/tools/kora/beta/bundle-methods#estimatebundlefee)
  — Arvioi nippuun kokonaismaksut

Katso täydellinen ohjeistus
[Jito-nippuoppaasta](/docs/tools/kora/beta/jito-guide).

### Docker-imaget

Viralliset monialustaisen imagen (amd64/arm64) julkaistu GitHub Container
Registryyn. Katso [Asennus](/docs/tools/kora/beta/installation).

### Transaktiolaajennukset

Uusi laajennusjärjestelmä mahdollistaa Koran validoida transaktiorakenteet
allekirjoitusvirtojen aikana. Ensimmäinen sisäänrakennettu laajennus,
`gas_swap`, pakottaa tiukat säännöt kaasuttomille
token-SOL-vaihtokauppatapahtumille. Laajennukset määritetään
`kora.toml`-tiedostossa ja suoritetaan automaattisesti `signTransaction`-,
`signAndSendTransaction`-, `signBundle`- ja `signAndSendBundle`-komentojen
aikana.

Katso
[Määritykset — Transaktiolaajennukset](/docs/tools/kora/beta/configuration#transaction-plugins).

### Uudet määritysasetukset

- **Transaktiolaajennukset** transaktiorakenteen validointiin (`gas_swap`)
- **reCAPTCHA v3** -bottisuojaus allekirjoituspäätepisteille
- **Lighthouse**-maksusuorittajan tyhjennyssuojaus
- **Käyttörajoitukset** lompakkokohtainen transaktiorajoitus
- **Nippumääritykset** Jito-integrointia varten

Katso [Määritykset](/docs/tools/kora/beta/configuration).

### SDK-muutokset

- Nippumetodituki (`signBundle`, `signAndSendBundle`, `estimateBundleFee`)

Katso [SDK-muutokset](/docs/tools/kora/beta/sdk-changes).

### Lisää RPC-metodeja

- [`getVersion`](/docs/tools/kora/beta/bundle-methods#getversion) — Hae
  palvelimen versiotiedot

## Asennus

```bash
cargo install kora-cli@2.2.0-beta.7
```

Tai Dockerin kautta:

```bash
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```
