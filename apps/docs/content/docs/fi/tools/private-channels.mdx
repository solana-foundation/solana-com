**Private Channels on yritystason kerros internetin pääomamarkkinoille
Solanassa.** Se tarjoaa konfiguroitavan sovelluskerroksen yksityisiin,
välittömiin ja maksuttomiin siirtoihin maksukanavien sisällä.
