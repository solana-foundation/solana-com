---

Valitse tarvitsemasi Commerce Kit -paketit sen mukaan, mitä olet rakentamassa.
Jokainen paketti on asennettavissa itsenäisesti ja sitä voidaan käyttää yksinään
tai yhdistettynä muihin.

## Järjestelmävaatimukset

- **Node.js**: Versio 18 tai uudempi (LTS suositellaan)
- **Paketinhallinta**: pnpm, npm tai yarn
- **TypeScript**: Versio 4.5+ (suositeltu mutta ei pakollinen)

## Yleiset vertaisriippuvuudet

Solana Commerce Kit on rakennettu Solana Kit -kirjaston päälle, joten se on
yhteensopiva Solana Kitin, Gillin, Solana Programs
[Clients](https://github.com/solana-program) -asiakkaiden ja
[Codama-generoitujen JS-asiakkaiden](https://github.com/codama-idl/renderers-js)
kanssa.

```bash
pnpm add gill # or @solana/kit
pnpm add @solana-program/token # or @solana-program/target-program
pnpm add @my-program/codama-js-client
```

## TypeScript-asetukset

Jos käytät TypeScriptiä, varmista, että `tsconfig.json` sisältää:

```json
{
  "compilerOptions": {
    "module": "ESNext",
    "target": "ES2020",
    "lib": ["ES2020", "DOM", "DOM.Iterable"],
    "jsx": "react-jsx",
    "moduleResolution": "bundler",
    "resolveJsonModule": true,
    "esModuleInterop": true,
    "skipLibCheck": true
  }
}
```

## Next.js-asetukset

Jos käytät Next.js:ää, päivitä `next.config.js` transpiloimaan Commerce Kit
-paketit:

```js
/** @type {import('next').NextConfig} */
const nextConfig = {
  transpilePackages: [
    "@solana-commerce/react",
    "@solana-commerce/connector",
    "@solana-commerce/headless",
    "@solana-commerce/solana-pay",
    "@solana-commerce/sdk"
  ]
};

export default nextConfig;
```
