**Commerce Kit on kokoelma työkaluja kaupankäyntikokemuksien rakentamiseen
Solanassa.** Valmiista maksupainikkeista headless-primitiiveihin, Commerce Kit
tarjoaa kaiken tarvitsemasi kryptomaksujen vastaanottamiseen.

## Miksi Commerce Kit?

- **Valmiit komponentit**: Esirakennettu PaymentButton lompakon yhdistämisellä,
  tokenin valinnalla ja transaktioiden käsittelyllä
- **Kehysneutraali**: Käytä headless-primitiivejä missä tahansa
  JavaScript-kehyksessä
- **Solana Pay**: Täysi Solana Pay -tuki QR-koodin luomisella
- **Tyyppiturvallisuus**: Täydellinen TypeScript-tuki kaikissa paketeissa
- **Wallet Standard**: Yhteensopiva kaikkien Wallet Standard -lompakoiden kanssa
  (Phantom, Solflare, Backpack jne.)

## Pika-aloitus

Nopein tapa aloittaa on `PaymentButton`-komponentin avulla:

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## Asennus

Valitse tarvitsemasi Commerce Kit -paketit:

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
