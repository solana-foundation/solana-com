---
title: Ympäristön konfigurointi
description: Builder-mallin metodit LiteSVM-testiympäristöjen konfigurointiin
---

Kaikki konfigurointimetodit käyttävät builder-mallia `LiteSVM::default()`:n
kanssa.

## with_compute_budget

```rust
pub fn with_compute_budget(self, compute_budget: ComputeBudget) -> Self
```

Aseta laskentayksikkörajoitukset ja keon koko.

```rust
use solana_compute_budget::compute_budget::ComputeBudget;

let mut svm = LiteSVM::default()
    .with_compute_budget(ComputeBudget {
        compute_unit_limit: 1_400_000,
        heap_size: 256 * 1024,
        ..ComputeBudget::default()
    });
```

## with_sigverify

```rust
pub fn with_sigverify(self, sigverify: bool) -> Self
```

Ota allekirjoituksen varmennus käyttöön tai poista se käytöstä. Oletus: `true`.

```rust
// Disable for faster tests
let mut svm = LiteSVM::default()
    .with_sigverify(false);
```

## with_blockhash_check

```rust
pub fn with_blockhash_check(self, check: bool) -> Self
```

Ota lohkotiivisteen validointi käyttöön tai poista se käytöstä. Oletus: `true`.

```rust
// Disable to use any blockhash
let mut svm = LiteSVM::default()
    .with_blockhash_check(false);
```

## with_transaction_history

```rust
pub fn with_transaction_history(self, capacity: usize) -> Self
```

Aseta historiaan säilytettävien tapahtumien määrä. Oletus: `0`.

```rust
// Keep last 100 transactions
let mut svm = LiteSVM::default()
    .with_transaction_history(100);
```

## with_log_bytes_limit

```rust
pub fn with_log_bytes_limit(self, limit: Option<usize>) -> Self
```

Aseta tapahtumalokeille suurin sallittu tavumäärä. Oletus: `Some(10_000)`.

```rust
// Unlimited logs
let mut svm = LiteSVM::default()
    .with_log_bytes_limit(None);

// Custom limit (50KB)
let mut svm = LiteSVM::default()
    .with_log_bytes_limit(Some(50_000));
```

## with_lamports

```rust
pub fn with_lamports(self, lamports: u64) -> Self
```

Aseta alkuperäiset lamport-arvot maksujen keräystä varten. Oletus:
`1_000_000_000`.

```rust
let mut svm = LiteSVM::default()
    .with_lamports(10_000_000_000);
```

## with_default_programs

```rust
pub fn with_default_programs(self) -> Self
```

Sisällyttää oletusarvoiset SPL-ohjelmat (Token, Associated Token, jne.).

```rust
let mut svm = LiteSVM::default()
    .with_default_programs();
```

## with_sysvars

```rust
pub fn with_sysvars(self) -> Self
```

Sisällyttää oletusarvoiset sysvars-muuttujat (Clock, Rent, EpochSchedule,
SlotHashes, StakeHistory, jne.).

```rust
let mut svm = LiteSVM::default()
    .with_sysvars();
```

## with_feature_set

```rust
pub fn with_feature_set(self, feature_set: FeatureSet) -> Self
```

Määritä Solana-ominaisuusjoukko tiettyjen ajonaikaisten ominaisuuksien testausta
varten.

```rust
use agave_feature_set::FeatureSet;

let mut svm = LiteSVM::default()
    .with_feature_set(FeatureSet::all_enabled());
```

## with_precompiles

```rust
#[cfg(feature = "precompiles")]
pub fn with_precompiles(self) -> Self
```

Sisällyttää vakioesikäännösohjelmat (ed25519, secp256k1, secp256r1).

```rust
let mut svm = LiteSVM::default()
    .with_precompiles();
```

<Callout type="info">
  Vaatii `precompiles`-ominaisuusliput:

```toml
[dev-dependencies]
litesvm = { version = "0.11", features = ["precompiles"] }
```

</Callout>

## with_feature_accounts

```rust
pub fn with_feature_accounts(self) -> Self
```

Sisällyttää ominaisuusporttien tilit testiympäristöön. Tämä varmistaa, että
ominaisuusporteilla suojattu Solana-ajonaikainen toiminta heijastuu tarkasti
testeihisi.

```rust
let mut svm = LiteSVM::default()
    .with_feature_accounts();
```

## register-tracing-ominaisuus

`register-tracing`-ominaisuus mahdollistaa yksityiskohtaisen
SBF-rekisterijäljityksen tapahtuman suorituksen aikana. Se edellyttää
`invocation-inspect-callback`-ominaisuutta.

```toml
[dev-dependencies]
litesvm = { version = "0.11", features = ["register-tracing"] }
```

Käytä `LiteSVM::new_debuggable(true)` luodaksesi instanssin, jossa
rekisterijäljitys on käytössä:

```rust
#[cfg(feature = "register-tracing")]
let mut svm = LiteSVM::new_debuggable(true);
```

Kun ominaisuus on käytössä, aseta `SBF_TRACE_DISASSEMBLE`-ympäristömuuttuja myös
purkukokoonpanon tulostamiseksi rekisterijälkien rinnalle:

```bash
SBF_TRACE_DISASSEMBLE=1 cargo test
```

## set_invocation_inspect_callback

```rust
pub fn set_invocation_inspect_callback<C: InvocationInspectCallback + 'static>(&mut self, callback: C)
```

Rekisteröi takaisinkutsu, joka käynnistyy ennen ja jälkeen jokaisen ohjelman
kutsun. Edellyttää `invocation-inspect-callback`-ominaisuutta.

```toml
[dev-dependencies]
litesvm = { version = "0.11", features = ["invocation-inspect-callback"] }
```

```rust
use litesvm::{LiteSVM, types::InvocationInspectCallback};
use solana_svm::invoke_context::InvokeContext;
use solana_svm_transaction::svm_message::SVMMessage;

struct MyInspector;

impl InvocationInspectCallback for MyInspector {
    fn before_invocation(
        &self,
        _svm: &LiteSVM,
        _tx: &impl SVMMessage,
        _program_indices: &[u16],
        _invoke_context: &InvokeContext,
    ) {
        println!("Before program invocation");
    }

    fn after_invocation(
        &self,
        _svm: &LiteSVM,
        _invoke_context: &InvokeContext,
        _enable_register_tracing: bool,
    ) {
        println!("After program invocation");
    }
}

let mut svm = LiteSVM::new();
svm.set_invocation_inspect_callback(MyInspector);
```
