## Pikahakemisto

| Metodi                                         | Kuvaus                                                                                            |
| ---------------------------------------------- | ------------------------------------------------------------------------------------------------- |
| `LiteSVM::new()`                               | Luo uuden LiteSVM-perustestausympäristön                                                          |
| `LiteSVM::default()`                           | Palauttaa tyypin "oletusarvon"                                                                    |
| `airdrop(&Address, lamports)`                  | Lisää tilille määritetyn määrän lamport-yksiköitä                                                 |
| `get_account(&Address)`                        | Palauttaa kaikki annettuun osoitteeseen liittyvän tilin tiedot                                    |
| `set_account(Address, Account)`                | Asettaa kaikki annettuun osoitteeseen liittyvän tilin tiedot                                      |
| `get_balance(&Address)`                        | Hakee annetun tiliosoitteen lamport-saldon                                                        |
| `minimum_balance_for_rent_exemption(data_len)` | Palauttaa vähimmäissaldon, joka vaaditaan rent-vapaan tilin luomiseen määritetyllä datapituudella |
| `add_program(program_id, &bytes)`              | Lisää SBF-ohjelman testausympäristöön                                                             |
| `add_program_from_file(program_id, path)`      | Lisää SBF-ohjelman testausympäristöön määritetystä tiedostosta                                    |
| `send_transaction(tx)`                         | Lähettää allekirjoitetun transaktion                                                              |
| `simulate_transaction(tx)`                     | Simuloi transaktion muuttamatta tilaa                                                             |
| `get_transaction(&signature)`                  | Hakee transaktion historiasta                                                                     |
| `latest_blockhash()`                           | Hakee nykyisen lohkohashin                                                                        |
| `expire_blockhash()`                           | Pakottaa nykyisen lohkohashin vanhenemisen                                                        |
| `warp_to_slot(slot)`                           | Siirtää kellon määritettyyn slot-arvoon                                                           |
| `get_sysvar<T>()`                              | Hakee sysvarin testausympäristöstä                                                                |
| `set_sysvar<T>(&value)`                        | Asettaa sysvarin testausympäristöön                                                               |
| `with_feature_accounts()`                      | Sisällyttää feature gate -tilit testausympäristöön                                                |
| `set_invocation_inspect_callback(callback)`    | Rekisteröi kutsu-/paluukutsun (vaatii `invocation-inspect-callback`-ominaisuuden)                 |
