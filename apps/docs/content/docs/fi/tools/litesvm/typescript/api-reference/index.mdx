## Pikaviite

### Laajennuksen asennus

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### Tilin hallinta (client.svm)

| Metodi                                 | Kuvaus                   |
| -------------------------------------- | ------------------------ |
| `setAccount(account)`                  | Aseta tilin tila suoraan |
| `getAccount(address)`                  | Hae tilin tiedot         |
| `getBalance(address)`                  | Hae SOL-saldo            |
| `airdrop(address, lamports)`           | Rahoita tili             |
| `minimumBalanceForRentExemption(size)` | Laske vuokravapaa minimi |

### Ohjelmien hallinta (client.svm)

| Metodi                                | Kuvaus                        |
| ------------------------------------- | ----------------------------- |
| `addProgram(programId, bytes)`        | Lataa ohjelma tavuista        |
| `addProgramFromFile(programId, path)` | Lataa ohjelma .so-tiedostosta |

### Tapahtumat (client.svm)

| Metodi                    | Kuvaus                          |
| ------------------------- | ------------------------------- |
| `sendTransaction(tx)`     | Suorita tapahtuma               |
| `simulateTransaction(tx)` | Simuloi ilman tilamuutoksia     |
| `latestBlockhash()`       | Hae nykyinen lohkohajautus      |
| `expireBlockhash()`       | Siirry uuteen lohkohajautukseen |

### Konfigurointi (client.svm)

| Metodi                      | Kuvaus                                                   |
| --------------------------- | -------------------------------------------------------- |
| `withSigverify(bool)`       | Ota allekirjoituksen varmennus käyttöön/poista käytöstä  |
| `withBlockhashCheck(bool)`  | Ota lohkohajautuksen validointi käyttöön/poista käytöstä |
| `withSysvars()`             | Ota sysvar-tilit käyttöön                                |
| `withBuiltins()`            | Ota sisäänrakennetut ohjelmat käyttöön                   |
| `withPrecompiles()`         | Ota esikompiloidut ohjelmat käyttöön                     |
| `withTransactionHistory(n)` | Tallenna viimeiset n tapahtumaa                          |
| `withComputeBudget(budget)` | Aseta laskentarajat                                      |
| `withLamports(amount)`      | Aseta oletusarvoinen lamport-määrä                       |
| `withLogBytesLimit(n)`      | Aseta lokitulosteen raja                                 |

### Aika & Sysvars (client.svm)

| Metodi                     | Kuvaus                                   |
| -------------------------- | ---------------------------------------- |
| `warpToSlot(slot)`         | Siirry tiettyyn slot                     |
| `getClock()`               | Hae clock-sysvar                         |
| `getRent()`                | Hae rent-sysvar                          |
| `getEpochSchedule()`       | Hae epoch-aikataulu                      |
| `getLastRestartSlot()`     | Hae viimeinen uudelleenkäynnistys-slot   |
| `setLastRestartSlot(slot)` | Aseta viimeinen uudelleenkäynnistys-slot |

### RPC (client.rpc)

| Metodi                           | Kuvaus                             |
| -------------------------------- | ---------------------------------- |
| `getAccountInfo(address)`        | Hae yksittäinen tili (vain base64) |
| `getMultipleAccounts(addresses)` | Hae useita tilejä                  |
| `getLatestBlockhash()`           | Hae nykyinen lohkohajautusarvo     |
