## Esimerkkien ajaminen

Kaikki esimerkit voidaan ajaa komennolla `tsx`, mutta jotkin osoitteet/muuttujat
täytyy korvata paikkamerkit todellisilla arvoilla.

```bash
# Install dependencies
pnpm add @solana/kit @solana/kit-plugin-litesvm @solana/kit-plugin-signer @solana-program/system

# Run an example
npx tsx example.ts
```
