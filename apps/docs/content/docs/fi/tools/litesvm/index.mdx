---
title: LiteSVM
description:
  Salamannopea Solana-ohjelmien testaus prosessinsisäisellä virtuaalikoneella.
  Testaa ohjelmasi nopeammin ilman verkkoviivettä tai validator-prosesseja.
---

LiteSVM on kevyt kirjasto Solana-ohjelmien testaamiseen. Se toimii luomalla
prosessinsisäisen Solana-virtuaalikoneen, joka on optimoitu ohjelmoijille.
Toisin kuin muut testausmenetelmät, jotka käynnistävät erillisiä
validator-prosesseja, LiteSVM upottaa virtuaalikoneen suoraan testeihin, mikä
nopeuttaa niiden suoritusta huomattavasti. Perinteisestä poiketen sillä on
ergonominen API järkevillä oletusarvoilla ja laajalla konfiguroitavuudella
niille, jotka sitä tarvitsevat.
