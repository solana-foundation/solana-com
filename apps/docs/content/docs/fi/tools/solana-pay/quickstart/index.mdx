Aloita Solana Payn käyttö nopeasti näiden vaiheittaisten ohjeiden avulla.

## Asennus

Asenna Solana Pay SDK ja määritä kehitysympäristösi.

**[→ Asennusohje](/docs/tools/solana-pay/quickstart/installation)** - Asenna SDK
ja riippuvuudet

## Maksutyypit

Valitse sopiva maksuintegraatio käyttötapauksellesi:

### Siirtopyynnöt (yksinkertainen)

Luo ei-interaktiivisia maksu-URL-osoitteita perustason SOL- ja
SPL-token-siirtoihin.

**[→ Siirtopyynnöt](/docs/tools/solana-pay/quickstart/transfer-requests)** -
Yksinkertaiset maksu-URL-osoitteet

**Parhaiten sopii:**

- Perusmaksut ja lahjoitukset
- QR-koodimaksut
- Maksulinkki sähköposteissa

### Transaktiopyynnöt (edistynyt)

Rakenna interaktiivisia maksuvirtoja palvelinpuolen transaktioiden
koostamisella.

**[→ Transaktiopyynnöt](/docs/tools/solana-pay/quickstart/transaction-requests)** -
Interaktiiviset maksuvirrat

**Parhaiten sopii:**

- Dynaaminen hinnoittelu
- NFT-minttaus
- Monimutkaiset DeFi-transaktiot
- Mukautettu liiketoimintalogiikka

## QR-koodiintegraatio

Luo ja mukauta QR-koodeja mobiilikukkomaksuihin.

**[→ QR-koodiintegraatio](/docs/tools/solana-pay/quickstart/qr-codes)** - Luo
skannattavia maksukoodeja

**Parhaiten sopii:**

- Kassajärjestelmät
- Verkosta mobiiliin -maksut
- Henkilökohtaiset transaktiot
