**Solana Pay on standardiprotokolla ja joukko referenssitoteutuksia, jotka
mahdollistavat kehittäjien integroida hajautetut maksut sovelluksiinsa ja
palveluihinsa.** Solana-lohkoketju vahvistaa transaktiot alle sekunnissa ja
maksaa keskimäärin 0,0005 dollaria, tarjoten käyttäjille saumattoman kokemuksen
ilman välittäjiä.

## Miksi Solana Pay?

- **Standardiprotokolla**: Yhtenäinen lähestymistapa maksuihin kaikissa
  Solana-lompakoissa ja sovelluksissa
- **Erittäin nopea**: Transaktiot vahvistuvat alle sekunnissa
- **Erittäin edullinen**: Keskimääräinen transaktiokustannus 0,0005 dollaria
- **Ei välittäjiä**: Suorat vertaisverkkomaksut ilman kolmansia osapuolia
- **Yleinen lompakkotuki**: Yhteensopiva Phantomin, Solflaren, Backpackin ja
  muiden kanssa

## Pikaopas

Yksinkertaisin tapa aloittaa on siirron pyyntö-URL:

```
solana:recipient?amount=0.01&label=Coffee%20Shop&message=Grande%20Latte
```

Tämä URL voidaan:

- Koodata QR-koodiksi mobiililompakon skannausta varten
- Käyttää maksulinkkinä verkkosovelluksissa
- Upottaa mobiilisovelluksiin syvälinkkinä

## Asennus

Asenna Solana Pay JavaScript SDK:

```bash
# Using pnpm (recommended)
pnpm add @solana/pay@beta @solana/kit

# Using npm
npm install @solana/pay@beta @solana/kit

# Using yarn
yarn add @solana/pay@beta @solana/kit
```

## Käyttötapaukset

Solana Pay mahdollistaa laajan valikoiman maksuskenaarioita:

- **Vähittäismaksut**: Henkilökohtaiset maksut QR-koodien avulla
- **Verkkokauppa**: Verkkokassan maksuvirtoja lompakkointegraatiolla
- **Mobiilisovellukset**: Natiiveja maksukokemuksia syvälinkkien avulla
- **NFT-lyönnit**: Dynaaminen transaktioiden luominen monimutkaisiin
  vuorovaikutuksiin
- **Tilaukset**: Toistuvat maksut ja kanta-asiakasohjelmat
- **DeFi-integraatio**: Koostettavia transaktioita vaihdoilla ja lainanaotoilla

## Tuetut lompakot

Solana Pay toimii kaikkien keskeisten Solana-lompakoiden kanssa:

- **[Phantom](https://phantom.app/)** - Suosituin Solana-lompakko
- **[Solflare](https://solflare.com/)** - Monialustainen Solana-lompakko
- **[Backpack](https://backpack.app/)** - Web3-lompakko ja sovelluskauppa
- **[Glow](https://glow.app/)** - Mobiililähtöinen Solana-lompakko
- **[Ultimate](https://ultimate.app/)** - DeFi-keskittynyt lompakko
- **[Decaf Wallet](https://decaf.so/)** - Globaali maksujen lompakko
- **[Espresso Cash](https://www.espressocash.com/)** - Globaali maksujen
  lompakko
- **[Tiplink](https://tiplink.io/)** - Maailman yksinkertaisin lompakko
- **[Bitget Wallet](https://bitget.com/)** - Moniketjulompakko
