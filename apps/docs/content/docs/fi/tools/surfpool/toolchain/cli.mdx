---
title: CLI-komennot
description: Surfpool-komentorivikäyttöliittymän (CLI) käyttäminen
---

`surfpool` CLI käynnistää paikalliset Surfnetit, suorittaa Runbookit, listaa
Runbookit työtilamanifestista, luo shell-täydennykset ja käynnistää Surfpool MCP
-palvelimen.

## Komennot

| Komento                | Kuvaus                                                  |
| ---------------------- | ------------------------------------------------------- |
| `surfpool start`       | Käynnistä paikallinen Surfnet. Alias: `surfpool simnet` |
| `surfpool run`         | Suorita Runbook                                         |
| `surfpool ls`          | Listaa Runbookit nykyisessä työtilassa                  |
| `surfpool completions` | Luo shell-täydennysskriptit                             |
| `surfpool mcp`         | Käynnistä Surfpool MCP -palvelin                        |

## `surfpool start`

Käynnistää paikallisen Surfnetin. Jos suoritat sen Solana-ohjelmahakemistossa,
Surfpool voi luoda ja suorittaa Runbookeja, jotka ottavat ohjelmasi käyttöön.

```sh
surfpool start [OPTIONS]
```

### Verkko ja portit

| Lippu                 | Lyhyt     | Tyyppi       | Oletus      | Kuvaus                                                                                                                                       |
| --------------------- | --------- | ------------ | ----------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| `--port <PORT>`       | `-p`      | kokonaisluku | `8899`      | Sido JSON-RPC-palvelin tähän porttiin.                                                                                                       |
| `--ws-port <PORT>`    | `-w`      | kokonaisluku | `8900`      | Sido WebSocket-palvelin tähän porttiin.                                                                                                      |
| `--host <HOST>`       | `-o`      | merkkijono   | `127.0.0.1` | Sido RPC-, WebSocket- ja Studio-palvelut tähän isäntään.                                                                                     |
| `--rpc-url <RPC_URL>` | `-u`      | merkkijono   | ei mitään   | Haaraudu tästä tietolähteen RPC-URL-osoitteesta. Ristiriidassa `--network`:n kanssa. Voidaan myös asettaa `SURFPOOL_DATASOURCE_RPC_URL`:llä. |
| `--network <NETWORK>` | `-n`      | enum         | ei mitään   | Haaraudu ennalta määritetystä Solana-verkosta. Arvot: `mainnet`, `devnet`, `testnet`. Ristiriidassa `--rpc-url`:n kanssa.                    |
| `--offline`           | ei mitään | boolean      | `false`     | Suorita ilman etä-RPC-tietolähdettä.                                                                                                         |

### Projekti & Käyttöönotto

| Lippu                                   | Lyhyt | Tyyppi   | Oletus       | Kuvaus                                                                                               |
| --------------------------------------- | ----- | -------- | ------------ | ---------------------------------------------------------------------------------------------------- |
| `--manifest-file-path <MANIFEST_PATH>`  | `-m`  | path     | `./txtx.yml` | Polku runbook-manifestiin.                                                                           |
| `--no-deploy`                           | none  | boolean  | `false`      | Poista automaattiset ohjelmien käyttöönotot käytöstä.                                                |
| `--runbook <RUNBOOK>`                   | `-r`  | string[] | `deployment` | Runbook-tunnus, joka suoritetaan käynnistyksen jälkeen. Toistettavissa.                              |
| `--runbook-input <INPUT_PATH>`          | `-i`  | path[]   | none         | Syötetiedosto runbook-suoritukselle. Toistettavissa.                                                 |
| `--yes`                                 | `-y`  | boolean  | `false`      | Ohita runbook-luonnin kehotteet.                                                                     |
| `--watch`                               | none  | boolean  | `false`      | Ota uudelleenkäyttöön, kun ohjelma-artefaktit muuttuvat.                                             |
| `--artifacts-path <ARTIFACTS_PATH>`     | none  | path     | none         | Hakemisto, joka sisältää `.so`-ohjelma-artefaktit. Oletuksena `target/deploy`, jos jätetty tyhjäksi. |
| `--legacy-anchor-compatibility`         | none  | boolean  | `false`      | Käytä oletusasetuksia, jotka sopivat legacy Anchor -testipaketeille.                                 |

| `--snapshot <SNAPSHOT_PATH>`            | none  | path[]   | none                       | JSON-tilimomentti esilataukseen. Toistettavissa. Myöhemmät tiedostot korvaavat aiemmat tiedostot päällekkäisten avainten osalta.                                                                                                      |

### Ajoympäristö & käyttöliittymä

| Lippu                  | Lyhyt | Tyyppi  | Oletus  | Kuvaus                                                                                                            |

### Poistettu yhteensopivuus

| Lippu                         | Lyhyt | Tyyppi | Oletus     | Kuvaus                                                                                                                                   |
| ----------------------------- | ----- | ------ | ---------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| `--subgraph-db <SUBGRAPH_DB>` | `-d`  | string | `:memory:` | Vanhentunut ja piilotettu kohteesta `surfpool start --help`. Olemassa olevat skriptit voivat silti välittää sen yhteensopivuuden vuoksi. |

### Esimerkit

Käynnistä oletusasetuksilla:

```sh
surfpool start
```

Haaroita devnet:

```sh
surfpool start --network devnet
```

Käytä mukautettua tietolähdettä RPC-URL:lla:

```sh
surfpool start --rpc-url https://api.mainnet-beta.solana.com
```

Tulosta lokit TUI:n käynnistämisen sijaan:

```sh
surfpool start --no-tui
```

Käynnistä ilman etä-RPC-tietolähdettä:

```sh
surfpool start --offline
```

Airdrop SOL useille tileille:

```sh
surfpool start --airdrop <PUBKEY> --airdrop <PUBKEY>
```

Lataa tilin tila tilannekuvista:

```sh
surfpool start --snapshot ./snapshot1.json --snapshot ./snapshot2.json
```

Ota ohjelmat uudelleen käyttöön, kun `target/deploy` muuttuu:

```sh
surfpool start --watch
```

Tallenna Surfnetin tila levyllä olevaan SQLite-tietokantaan:

```sh
surfpool start --db ./surfnet.sqlite --surfnet-id local-dev
```

## `surfpool run`

Suorittaa Runbookin. Voit ajaa `.tx`-tiedoston suoraan tai viitata
`txtx.yml`-manifestissa määriteltyyn Runbookiin.

```sh
surfpool run [OPTIONS] <RUNBOOK>
```

### Argumentit

| Argumentti  | Tyyppi     | Kuvaus                                                    |
| ----------- | ---------- | --------------------------------------------------------- |
| `<RUNBOOK>` | merkkijono | Runbook-tunnus `txtx.yml`:sta tai polku `.tx`-tiedostoon. |

### Projekti

| Lippu                                  | Lyhenne   | Tyyppi     | Oletus       | Kuvaus                                  |
| -------------------------------------- | --------- | ---------- | ------------ | --------------------------------------- |
| `--manifest-file-path <MANIFEST_PATH>` | `-m`      | polku      | `./txtx.yml` | Polku runbook-manifestiin.              |
| `--env <ENVIRONMENT>`                  | ei mitään | merkkijono | ei mitään    | Ympäristö `txtx.yml`:sta käytettäväksi. |

### Suoritustila

Nämä liput ovat toisiaan poissulkevia. Jos mitään ei ole määritetty, Surfpool
käyttää selainvalvontaa.

| Lippu            | Lyhenne | Tyyppi     | Oletus     | Kuvaus                                          |
| ---------------- | ------- | ---------- | ---------- | ----------------------------------------------- |
| `--unsupervised` | `-u`    | totuusarvo | `false`    | Suorita ilman valvojan käyttöliittymää.         |
| `--browser`      | `-b`    | totuusarvo | oletustila | Suorita valvonnalla selaimen käyttöliittymässä. |
| `--terminal`     | `-t`    | totuusarvo | `false`    | Suorita valvonnalla terminaalikonsolissa.       |

### Syötteet ja tulosteet

| Lippu                          | Lyhyt     | Tyyppi            | Oletus    | Kuvaus                                                                                                                                   |
| ------------------------------ | --------- | ----------------- | --------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| `--output-json [<OUTPUT_DIR>]` | ei mitään | valinnainen polku | ei mitään | Tulosta tai kirjoita Runbook-tulosteet JSON-muodossa. Kun hakemisto annetaan, tulosteet kirjoitetaan tiedostoon kyseisessä hakemistossa. |
| `--output <OUTPUT_NAME>`       | ei mitään | merkkijono        | ei mitään | Tulosta yksi nimetty tuloste suorituksen lopussa. Ristiriidassa `--output-json`:n kanssa.                                                |
| `--input <INPUT_PATH>`         | ei mitään | polku[]           | ei mitään | Syötetiedosto eräkäsittelyä varten. Toistettavissa.                                                                                      |

### Suoritus

| Lippu       | Lyhyt     | Tyyppi     | Oletus  | Kuvaus                                       |
| ----------- | --------- | ---------- | ------- | -------------------------------------------- |
| `--explain` | ei mitään | totuusarvo | `false` | Selitä, miten Runbook suoritetaan.           |
| `--force`   | `-f`      | totuusarvo | `false` | Ohita välimuistiin tallennettu suoritustila. |

### Valvontakäyttöliittymä

Nämä liput ovat käytettävissä, kun Surfpool on rakennettu
`supervisor_ui`-ominaisuudella.

| Lippu               | Lyhyt | Tyyppi       | Oletus      | Kuvaus                           |
| ------------------- | ----- | ------------ | ----------- | -------------------------------- |
| `--port <PORT>`     | `-p`  | kokonaisluku | `8488`      | Verkkokäyttöliittymän portti.    |
| `--ip <IP_ADDRESS>` | `-i`  | merkkijono   | `127.0.0.1` | Verkkokäyttöliittymän IP-osoite. |

### Lokitus

| Lippu                  | Lyhyt     | Tyyppi | Oletus           | Kuvaus                                                                          |
| ---------------------- | --------- | ------ | ---------------- | ------------------------------------------------------------------------------- |
| `--log-level <LEVEL>`  | `-l`      | enum   | `info`           | Runbook-suorituksen lokitaso. Arvot: `trace`, `debug`, `info`, `warn`, `error`. |
| `--log-path <LOG_DIR>` | ei mitään | polku  | `.surfpool/logs` | Hakemisto Runbook-suorituslokeja varten.                                        |

### Esimerkit

Suorita manifest Runbook selaimen valvonnalla:

```sh
surfpool run deployment
```

Suorita ilman valvontaa ja kirjoita JSON-tulosteet:

```sh
surfpool run deployment --unsupervised --output-json .surfpool/runbook-outputs
```

Suorita `.tx`-tiedosto suoraan:

```sh
surfpool run ./runbooks/deployment.tx
```

## `surfpool ls`

Listaa kaikki työtilan manifestissa määritellyt Runbookit.

```sh
surfpool ls [OPTIONS]
```

| Lippu                                  | Lyhyt | Tyyppi | Oletus       | Kuvaus                     |
| -------------------------------------- | ----- | ------ | ------------ | -------------------------- |
| `--manifest-file-path <MANIFEST_PATH>` | `-m`  | polku  | `./txtx.yml` | Polku runbook-manifestiin. |

---

## `surfpool completions`

Luo shell-täydennyskomentosarjoja.

```sh
surfpool completions <SHELL>
```

| Argumentti | Tyyppi | Kuvaus                                                                                   |
| ---------- | ------ | ---------------------------------------------------------------------------------------- |
| `<SHELL>`  | enum   | Shell, jolle täydennykset luodaan. Arvot: `bash`, `elvish`, `fish`, `powershell`, `zsh`. |

## `surfpool mcp`

Käynnistää Surfpool MCP -palvelimen tekoälytyökalujen integrointia varten.

```sh
surfpool mcp
```
