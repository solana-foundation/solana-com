---
title: Aloittaminen
description: Tämä opas auttaa sinua asentamaan Surfpoolin ja pääsemään alkuun.
---

Surfpool CLI voidaan asentaa asennusskriptin avulla tai kääntämällä
lähdekoodista.

## Asennusskripti

Avaa pääte ja asenna surfpool suorittamalla:

```bash
curl -sL https://run.surfpool.run/ | bash
```

## Asenna lähdekoodista

Kääntääksesi Surfpool-binäärin lähdekoodista, kloonaa surfpool-repositorio.

```bash
git clone https://github.com/solana-foundation/surfpool.git
```

Siirry uuteen hakemistoon.

```bash
cd surfpool
```

Käännä sen jälkeen binääri. Tämä komento kääntää binäärin ja tallentaa sen
hakemistoon `~/.cargo/bin/surfpool`.

```bash
cargo surfpool-install
```

Varmista lopuksi, että `surfpool`-binääri on käytettävissä PATH-muuttujassasi.
Tämä prosessi vaihtelee käyttöjärjestelmästä riippuen.
