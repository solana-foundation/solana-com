Surfnet on Surfpoolin paikallinen Solana-verkkosimulaattori, joka on suunniteltu
tarjoamaan kehittäjille tehokkaan testausympäristön, joka jäljittelee
Solana-lohkoketjun toimintaa ilman täyden validator-solmun aiheuttamaa
kuormitusta.

## Aloittaminen

Käynnistä Surfnet-instanssi Surfpool CLI:n avulla:

```bash
surfpool start
```

Tämä käynnistää paikallisen Surfnet-instanssin osoitteessa
`http://localhost:8899`, jossa kaikki RPC-päätepisteet ovat käytettävissä.

## Keskeiset ominaisuudet

- **Täysi RPC-yhteensopivuus**: Tukee koko Solana RPC API:ta
- **Nopea iterointi**: Ei tarvetta odottaa lohkovahvistuksia
- **Tilamanipulointi**: Huijauskoodeilla voi muokata lohkoketjun tilaa suoraan
- **Mainnet-haarukka**: Haarukoidu mainnetistä testataksesi oikeilla tileillä ja
  ohjelmilla
- **Geyser Plugin -tuki**: Yhdistä Geyser-liitännäisesi reaaliaikaista
  datavirtausta varten
