Surfpool on suora korvike `solana-test-validator` -työkalulle, joka on
suunniteltu tarjoamaan paras mahdollinen kokemus **Solana**-kehittäjille.

Sen ydinominaisuus on antaa kehittäjien simuloida ohjelmiaan paikallisesti
just-in-time-menetelmällä haettujen Mainnet-tilien avulla.

Surfpool tuo myös saumattomasti Infrastructure as Code -käytännöt Anchor- tai
Pinocchio-pohjaisiin projekteihin, mahdollistaen toistettavat, tarkastettavat ja
turvalliset käyttöönotot mihin tahansa Solana-verkkoon – yksityiseen tai
julkiseen.
