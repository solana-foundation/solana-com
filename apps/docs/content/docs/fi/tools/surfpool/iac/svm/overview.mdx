---
title: SVM-lisäosan yleiskatsaus
description: Lisäosa Solanalle ja SVM-yhteensopiville lohkoketjuille (beta)
---

SVM `txtx` -liitännäisen avulla voit rakentaa Runbookeja, jotka ovat
vuorovaikutuksessa Solanan ja SVM-yhteensopivien lohkoketjujen kanssa.
Liitännäinen tarjoaa apufunktioita, joiden avulla voit ottaa käyttöön
Anchor-ohjelmia ja koodata ohjekutsuja ohjelman IDL:ien mukaisesti. Toimintojen
avulla voidaan luoda kelvollisia siirto-, ohjelma-kutsu- ja ohjelman
käyttöönottotapahtumia, jotka voidaan allekirjoittaa mnemonisella lauseella,
salaavaimella tai selaimen allekirjoittajalla.

## Pika-aloitus

Käyttääksesi SVM-lisäosaa, määritä se Runbookissasi:

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## Ominaisuudet

- **Ohjelman käyttöönotto**: Ota käyttöön Anchor- ja natiivia Solana-ohjelmia
- **Ohjeiden rakentaminen**: Koodaa ohjekutsuja ohjelman IDL:ien avulla
- **Tapahtumien allekirjoittaminen**: Tuki mnemoniikalle, salaisille avaimille
  ja verkkolompakkoille
- **Tilinhallinnointi**: Kyselyiden tekeminen ja Solana-tilien hallinta
