Nämä toiminnot ovat käytettävissä kaikissa Runbook-kirjoissa osana
vakiokirjastoa.

## send_http_request

`std::send_http_request`-toiminto tekee HTTP-pyynnön määritettyyn
URL-osoitteeseen.

### Syötteet

| Nimi         | Pakollinen  | Tyyppi  | Kuvaus                                            |
| ------------ | ----------- | ------- | ------------------------------------------------- |
| `url`        | pakollinen  | string  | URL-osoite, johon pyyntö lähetetään               |
| `method`     | valinnainen | string  | HTTP-metodi (GET, POST, PUT, DELETE). Oletus: GET |
| `headers`    | valinnainen | map     | Pyyntöön sisällytettävät HTTP-otsikot             |
| `body`       | valinnainen | string  | Pyynnön runko (POST/PUT-pyyntöjä varten)          |
| `timeout_ms` | valinnainen | integer | Pyynnön aikakatkaisu millisekunteina              |

### Tulosteet

| Nimi          | Tyyppi  | Kuvaus             |
| ------------- | ------- | ------------------ |
| `status_code` | integer | HTTP-tilakoodi     |
| `body`        | string  | Vastauksen runko   |
| `headers`     | map     | Vastauksen otsikot |

```hcl
action "api_call" "std::send_http_request" {
    description = "Fetch data from API"
    url = "https://api.example.com/data"
    method = "GET"
    headers = {
        "Authorization" = "Bearer ${variable.token}"
    }
}

output "response" {
    value = action.api_call.body
}

output "status" {
    value = action.api_call.status_code
}
```

### POST-pyyntöesimerkki

```hcl
action "create_resource" "std::send_http_request" {
    description = "Create a new resource"
    url = "https://api.example.com/resources"
    method = "POST"
    headers = {
        "Content-Type" = "application/json"
    }
    body = {
        "name" = variable.name,
        "value" = variable.value
    }
}
```
