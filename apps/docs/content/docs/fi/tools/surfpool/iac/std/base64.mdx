## encode_base64

`encode_base64` koodaa puskurin tai merkkijonon base64-merkkijonoksi.

### Syötteet

| Nimi    | Tyyppi                | Kuvaus                                                                                                                                        |
| ------- | --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | Koodattava puskuri tai merkkijono. Merkkijonot, jotka alkavat '0x':llä, dekoodataan heksadesimaalina; muutoin käytetään raakoja UTF-8-tavuja. |

### Tuloste

| Nimi    | Tyyppi | Kuvaus                               |
| ------- | ------ | ------------------------------------ |
| `value` | string | Syöte koodattuna base64-merkkijonona |

```hcl
output "encoded" {
    value = encode_base64("0x48656c6c6f20776f726c6421")
}
// > encoded: SGVsbG8gd29ybGQh
```

## decode_base64

`decode_base64` dekoodaa base64-koodatun merkkijonon ja palauttaa tuloksen
puskurina.

### Syötteet

| Nimi            | Tyyppi | Kuvaus                         |
| --------------- | ------ | ------------------------------ |
| `base64_string` | string | Dekoodattava base64-merkkijono |

### Tuloste

| Nimi    | Tyyppi | Kuvaus                                 |
| ------- | ------ | -------------------------------------- |
| `value` | buffer | Dekoodattu base64-merkkijono puskurina |

```hcl
output "decoded" {
    value = decode_base64("SGVsbG8gd29ybGQh")
}
// > decoded: 0x48656c6c6f20776f726c6421
```
