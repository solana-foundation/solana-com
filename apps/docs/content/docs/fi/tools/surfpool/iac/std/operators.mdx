## and_bool

`and_bool` palauttaa vasemman ja oikean puolen argumenttien binäärisen
AND-operaation tuloksen.

### Syötteet

| Nimi  | Tyyppi | Kuvaus     |
| ----- | ------ | ---------- |
| `lhs` | bool   | Totuusarvo |
| `rhs` | bool   | Totuusarvo |

### Tulos

| Nimi    | Tyyppi | Kuvaus                                            |
| ------- | ------ | ------------------------------------------------- |
| `value` | bool   | Kahden argumentin binäärisen AND-operaation tulos |

```hcl
output "my_bool" {
  value = false && true
}
// > my_bool: false
```

## div

`div` palauttaa vasemman puolen argumentin kokonaislukujaon oikean puolen
argumentilla, pyöristäen mahdollisen jakojäännöksen alaspäin lähimpään
kokonaislukuun.

### Syötteet

| Nimi  | Tyyppi       | Kuvaus   |
| ----- | ------------ | -------- |
| `lhs` | kokonaisluku | Jaettava |
| `rhs` | kokonaisluku | Jakaja   |

### Tulos

| Nimi    | Tyyppi       | Kuvaus                              |
| ------- | ------------ | ----------------------------------- |
| `value` | kokonaisluku | Jaettavan jakamisen tulos jakajalla |

```hcl
output "my_int" {
  value = 11 / -3
}
// > my_int: -3
```

## eq

`eq` palauttaa `true`, jos vasen ja oikea argumentti ovat yhtä suuret, ja
`false`, jos ne eivät ole.

### Syötteet

| Nimi  | Tyyppi | Kuvaus            |
| ----- | ------ | ----------------- |
| `lhs` | any    | Mikä tahansa arvo |
| `rhs` | any    | Mikä tahansa arvo |

### Tuloste

| Nimi    | Tyyppi | Kuvaus                                        |
| ------- | ------ | --------------------------------------------- |
| `value` | bool   | Tulos kahden syötteen tasa-arvotarkistuksesta |

```hcl
output "is_eq" {
  value = "arg" == "arg"
}
// > is_eq: true
```

## neq

`neq` palauttaa `true`, jos vasen ja oikea argumentti eivät ole yhtä suuret, ja
`false` muussa tapauksessa.

### Syötteet

| Nimi  | Tyyppi | Kuvaus                |
| ----- | ------ | --------------------- |
| `lhs` | any    | Vertailtava arvo      |
| `rhs` | any    | Arvo, johon verrataan |

### Tuloste

| Nimi    | Tyyppi | Kuvaus                                                        |
| ------- | ------ | ------------------------------------------------------------- |
| `value` | bool   | Tulos negoidusta tasa-arvovertailusta kahden syötteen välillä |

```hcl
output "is_neq" {
  value = "arg" != "arg"
}
// > is_neq: false
```
