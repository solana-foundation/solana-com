## assert_eq

`assert_eq` varmistaa, että kaksi arvoa ovat yhtä suuret.

### Syötteet

| Nimi    | Tyyppi | Kuvaus                |
| ------- | ------ | --------------------- |
| `left`  | any    | Vertailtava arvo      |
| `right` | any    | Arvo, johon verrataan |

### Tuloste

| Nimi    | Tyyppi                | Kuvaus         |
| ------- | --------------------- | -------------- |
| `value` | addon(std::assertion) | Väitteen tulos |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```

## assert_gt

`assert_gt` varmistaa, että vasen arvo on suurempi kuin oikea arvo.

### Syötteet

| Nimi    | Tyyppi           | Kuvaus                                      |
| ------- | ---------------- | ------------------------------------------- |
| `left`  | integer \| float | Vertailtava kokonaisluku tai liukuluku      |
| `right` | integer \| float | Kokonaisluku tai liukuluku, johon verrataan |

### Tuloste

| Nimi    | Tyyppi                | Kuvaus         |
| ------- | --------------------- | -------------- |
| `value` | addon(std::assertion) | Väitteen tulos |

```hcl
output "assertion" {
    value = std::assert_gt(action.example.result, 1)
}
```

| `right` | kokonaisluku \| liukuluku | Kokonaisluku tai liukuluku, johon verrataan |

### Tuloste

| `value` | addon(std::assertion) | Tarkistuksen tulos |
