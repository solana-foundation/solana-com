---
title: SVM-allekirjoittajat
description: Allekirjoittajat Solana- ja SVM-yhteensopiville lohkoketjuille
---

Nämä allekirjoittajat ovat käytettävissä, kun käytetään SVM-lisäosaa tapahtumien
allekirjoittamiseen.

## secret_key

`svm::secret_key`-allekirjoittajaa voidaan käyttää tapahtuman synkroniseen
allekirjoittamiseen käyttämällä salaista avainta, mnemonista lauseketta tai
keypair-tiedostoa.

### Syötteet

| Nimi              | Pakollinen  | Tyyppi     | Kuvaus                                                                     |
| ----------------- | ----------- | ---------- | -------------------------------------------------------------------------- |
| `secret_key`      | valinnainen | merkkijono | Salainen avain, jota käytetään viestien ja tapahtumien allekirjoittamiseen |
| `mnemonic`        | valinnainen | merkkijono | Mnemoninen lauseke, jota käytetään salaisen avaimen luomiseen              |
| `derivation_path` | valinnainen | merkkijono | Johdannainen polku, jota käytetään salaisen avaimen luomiseen              |
| `keypair_json`    | valinnainen | merkkijono | Polku keypair.json-tiedostoon, joka sisältää salaisen avaimen              |

### Tulosteet

| Nimi         | Tyyppi     | Kuvaus                                  |
| ------------ | ---------- | --------------------------------------- |
| `public_key` | merkkijono | Tilin pubkey                            |
| `address`    | merkkijono | SVM-osoite (alias kohteelle public_key) |

```hcl
signer "deployer" "svm::secret_key" {
    secret_key = input.secret_key
}
```

Voit myös käyttää mnemonista lauseketta:

```hcl
signer "deployer" "svm::secret_key" {
    mnemonic = input.mnemonic
    derivation_path = "m/44'/501'/0'/0'"
}
```

Tai keypair-tiedostoa:

```hcl
signer "deployer" "svm::secret_key" {
    keypair_json = "~/.config/solana/id.json"
}
```

## web_wallet

`svm::web_wallet`-allekirjoittaja sallii Runbook-operaattorin allekirjoittaa
tapahtuman valitsemallaan selaimen allekirjoittajalla.

### Syötteet

| Nimi               | Pakollinen  | Tyyppi     | Kuvaus                                                                 |
| ------------------ | ----------- | ---------- | ---------------------------------------------------------------------- |
| `expected_address` | valinnainen | merkkijono | Odotettu yhdistettävä SVM-osoite (jätä pois salliaksesi minkä tahansa) |

### Tulosteet

| Nimi         | Tyyppi     | Kuvaus                  |
| ------------ | ---------- | ----------------------- |
| `address`    | merkkijono | Yhdistetyn tilin osoite |
| `public_key` | merkkijono | Yhdistetyn tilin pubkey |

```hcl
signer "alice" "svm::web_wallet" {
    expected_address = "zbBjhHwuqyKMmz8ber5oUtJJ3ZV4B6ePmANfGyKzVGV"
}
```

Tämä allekirjoittaja kehottaa käyttäjää Web-käyttöliittymässä:

1. Yhdistämään lompakkonsa
2. Antamaan pubkey viestin allekirjoituksen kautta
3. Allekirjoittamaan kaikki tätä allekirjoittajaa vaativat tapahtumat

## squads

`svm::squads`-allekirjoittajaa voidaan käyttää tapahtumien allekirjoittamiseen
Squads-monitallekirjoitusholvilla.

### Syötteet

| Nimi      | Pakollinen  | Tyyppi | Kuvaus               |
| --------- | ----------- | ------ | -------------------- |
| `address` | valinnainen | string | Squads-holvin osoite |

### Tulosteet

| Nimi         | Tyyppi | Kuvaus                                  |
| ------------ | ------ | --------------------------------------- |
| `public_key` | string | Squads-holvin julkinen avain            |
| `address`    | string | SVM-osoite (alias kohteelle public_key) |

```hcl
signer "deployer" "svm::squads" {
    address = input.address
}
```

Squads-monitallekirjoituksen käyttäminen mahdollistaa turvallisen, usean
osapuolen tapahtumallekirjoituksen paljastamatta yksittäisiä yksityisiä avaimia.
