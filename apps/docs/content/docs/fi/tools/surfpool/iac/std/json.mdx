## jq

`jq`-funktio mahdollistaa JSON-datan osittamisen, suodattamisen ja
kartoittamisen. Katso lisätietoja
[jq-dokumentaatiosta](https://jqlang.github.io/jq/manual/).

### Syötteet

| Nimi           | Tyyppi           | Kuvaus                                                                     |
| -------------- | ---------------- | -------------------------------------------------------------------------- |
| `decoded_json` | string \| object | JSON-objekti                                                               |
| `query`        | string           | JSON-kysely. Katso [jq-dokumentaatio](https://jqlang.github.io/jq/manual/) |

### Tuloste

| Nimi    | Tyyppi        | Kuvaus             |
| ------- | ------------- | ------------------ |
| `value` | array[string] | `jq`-kyselyn tulos |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
