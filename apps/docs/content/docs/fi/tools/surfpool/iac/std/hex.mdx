## encode_hex

`encode_hex` koodaa puskurin tai merkkijonon heksadesimaaliesityksenä
0x-etuliitteen kanssa.

### Syötteet

| Nimi    | Tyyppi                | Kuvaus                                                                                                                                                    |
| ------- | --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | Koodattava puskuri tai merkkijono. Merkkijonot, jotka alkavat '0x'-etuliitteellä, dekoodataan heksana; muussa tapauksessa käytetään raakoja UTF-8-tavuja. |

### Tuloste

| Nimi    | Tyyppi | Kuvaus                                             |
| ------- | ------ | -------------------------------------------------- |
| `value` | string | Syöte heksadesimaalimuodossa 0x-etuliitteen kanssa |

```hcl
output "encoded_hex" {
    value = encode_hex("hello, world")
}
// > encoded_hex: 0x68656c6c6f2c20776f726c64
```

## decode_hex

`decode_hex` dekoodaa heksadesimaalimerkkijonon ja palauttaa tuloksen puskurina.

### Syötteet

| Nimi         | Tyyppi | Kuvaus                                                   |
| ------------ | ------ | -------------------------------------------------------- |
| `hex_string` | string | Dekoodattava hex-merkkijono (0x-etuliitteellä tai ilman) |

### Tuloste

| Nimi    | Tyyppi | Kuvaus                              |
| ------- | ------ | ----------------------------------- |
| `value` | buffer | Dekoodattu hex-merkkijono puskurina |

```hcl
output "decoded_hex" {
    value = decode_hex("0x68656c6c6f2c20776f726c64")
}
// > decoded_hex: 0x68656c6c6f2c20776f726c64
```
