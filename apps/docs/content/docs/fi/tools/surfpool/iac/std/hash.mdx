## ripemd160

`ripemd160` laskee arvon Ripemd160-tiivisteen.

### Syötteet

| Nimi    | Tyyppi                  | Kuvaus                          |
| ------- | ----------------------- | ------------------------------- |
| `value` | buffer \| array[buffer] | Tiivistettävä hex-koodattu arvo |

### Tuloste

| Nimi    | Tyyppi | Kuvaus            |
| ------- | ------ | ----------------- |
| `value` | string | Tiivistetty tulos |

```hcl
output "hashed_data" {
    value = ripemd160(encode_hex("hello, world"))
}
// > hashed_data: 0XA3201F82FCA034E46D10CD7B27E174976E241DA2
```
