## index

`index` hakee merkinnän listasta määritetyssä indeksissä.

### Syötteet

| Nimi    | Tyyppi                                           | Kuvaus                        |
| ------- | ------------------------------------------------ | ----------------------------- |
| `list`  | array[string] \| array[integer] \| array[buffer] | Lista, josta merkintä haetaan |
| `index` | integer                                          | Haettavan merkinnän indeksi   |

### Tulos

| Nimi    | Tyyppi | Kuvaus                                    |
| ------- | ------ | ----------------------------------------- |
| `value` | string | Merkintä listasta määritetyssä indeksissä |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
