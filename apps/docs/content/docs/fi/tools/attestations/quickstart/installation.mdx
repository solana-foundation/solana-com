Aloita Solana Attestations -palvelun käyttö asentamalla tarvittavat paketit ja
määrittämällä kehitysympäristösi.

## TypeScript/JavaScript

Asenna Solana Attestations -kirjasto käyttämällä haluamaasi
paketinhallintaohjelmaa:

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
