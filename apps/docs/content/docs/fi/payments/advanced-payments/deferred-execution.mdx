## Asennus: luo nonce-tili

Nonce-tilin luominen vaatii kaksi kГӨskyГӨ yhdessГӨ transaktiossa:

1. **Luo tili** kГӨyttГӨen `getCreateAccountInstruction` System Programista
2. **Alusta se nonceksi** kГӨyttГӨen `getInitializeNonceAccountInstruction`

```ts

  getNonceSize,
  getCreateAccountInstruction,
  getInitializeNonceAccountInstruction,
  SYSTEM_PROGRAM_ADDRESS
} from "@solana-program/system";

// Generate a keypair for the nonce account address
const nonceKeypair = await generateKeyPairSigner();

// Get required account size for rent calculation
const space = BigInt(getNonceSize());

// 1. Create the account (owned by System Program)
getCreateAccountInstruction({
  payer,
  newAccount: nonceKeypair,
  lamports: rent,
  space,
  programAddress: SYSTEM_PROGRAM_ADDRESS
});

// 2. Initialize as nonce account
getInitializeNonceAccountInstruction({
  nonceAccount: nonceKeypair.address,
  nonceAuthority: authorityAddress // Controls nonce advancement
});

// Assemble and send transaction to the network
```

## LykГӨtyn transaktion rakentaminen

KГӨytГӨ ГӨskettГӨisen lohkohajautuksen sijaan nonce-arvoa transaktion elinkaaren

## TuotantoympГӨristГ¶n huomioitavaa

**Nonce-tilien hallinta:**

- Luo joukko nonce-tilejГӨ rinnakkaista transaktioiden valmistelua varten
- Seuraa, mitkГӨ noncet ovat "kГӨytГ¶ssГӨ" (niillГӨ on odottavia allekirjoitettuja
  transaktioita)
- Toteuta nonce-kierrГӨtys sen jГӨlkeen, kun transaktiot on lГӨhetetty tai hylГӨtty

**Turvallisuus:**

- Nonce-valtuutus hallitsee sitГӨ, voidaanko transaktiot mitГӨtГ¶idГӨ. Harkitse
  nonce-valtuutuksen erottamista transaktioiden allekirjoittajista lisГӨhallinnan
  ja tehtГӨvien erottelun vuoksi
- _Kuka tahansa_, jolla on sarjallistetut transaktiotavut, voi lГӨhettГӨГӨ sen
  verkkoon
