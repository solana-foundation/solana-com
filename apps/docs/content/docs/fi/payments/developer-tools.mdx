## AI-kehitystyökalut

Aloita oppaamme avulla
[miten pääset alkuun AI-työkalujen kanssa Solanassa](/developers/guides/getstarted/intro-to-ai).

### Solana MCP

[Solana MCP](https://mcp.solana.com/) (Machine Conversation Protocol) integroi
Solana-tietämyksen suoraan AI-tuettuihin koodaustyökaluihin kuten Claude, Cursor
ja Windsurf. Se on koulutettu Solanan dokumentaatiolla, Anchor-frameworkilla,
ohjelmaesimerkeillä ja Stack Exchange Q&A:lla - hyödyllinen maksutoteutusten
rakentamiseen nopeammin.

### AI-agenttiframeworkit

AI-agentit voivat itsenäisesti suorittaa maksutransaktioita Solanassa. Nämä
frameworkit mahdollistavat agenttien rakentamisen, jotka käsittelevät maksuja,
seuraavat transaktioita ja automatisoivat maksutyönkulkuja:

| Framework        | Kuvaus                                                                                          | Linkit                                                                                             |
| ---------------- | ----------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------- |
| Solana Agent Kit | Työkalut maksukykyisten agenttien rakentamiseen, yhteensopiva LangChainin ja Vercel AI:n kanssa | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Dokumentaatio](https://kit.sendai.fun/) |
| Send AI Skills   | Kokoelma valmiita taitoja agenttien rakentamiseen Solanassa                                     | [GitHub](https://github.com/sendaifun/skills)                                                      |
| Eliza Framework  | Monialustainen agenttiframework Solana-pluginilla X-, Telegram- ja Discord-botteihin            | [GitHub](https://github.com/elizaOS/eliza) • [Dokumentaatio](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Ketjujen välinen toolkit Solana-maksutoiminnoilla                                               | [GitHub](https://github.com/goat-sdk/goat)                                                         |
