## AI-kehitystyökalut

Aloita oppaamme avulla
[miten pääset alkuun AI-työkalujen kanssa Solanassa](/developers/guides/getstarted/intro-to-ai).

### Tekoälyagenttikehykset

Tekoälyagentit voivat suorittaa maksutapahtumia Solanassa autonomisesti. Nämä
kehykset mahdollistavat agenttien rakentamisen, jotka käsittelevät maksuja,
seuraavat tapahtumia ja automatisoivat maksutyönkulkuja:

| Kehys            | Kuvaus                                                                                           | Linkit                                                                                    |
| ---------------- | ------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | Työkaluja maksukykyisten agenttien rakentamiseen, yhteensopiva LangChainin ja Vercel AI:n kanssa | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | Kokoelma valmiiksi rakennettuja taitoja agenttien rakentamiseen Solanassa                        | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | Monialustainen agenttikehys Solana-lisäosalla X-, Telegram- ja Discord-botteja varten            | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Ketjujen välinen työkalupakki Solana-maksutoiminnoilla                                           | [GitHub](https://github.com/goat-sdk/goat)                                                |
