SPL Token Basics -osio kattaa perustason ohjeet SPL-tokenien kanssa
vuorovaikutukseen, keskittyen toiminnallisuuteen, joka on yhteinen sekä
alkuperäiselle Token Program -ohjelmalle että Token Extensions Program
-ohjelmalle (Token 2022).

  saadaksesi lisätietoja siitä, miten SPL-tokeneille lisätään metatietoja.
