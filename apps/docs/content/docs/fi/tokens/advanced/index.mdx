Tavallisten luonti-, mint- ja siirto-operaatioiden lisäksi Token Program
kutsutaan myös ketjussa olevista ohjelmista CPI:n kautta, ja se sisältää ohjeet
lamporttien palauttamiseen:

<Cards>
  <Card
    title="Token Programin kutsuminen CPI:n kautta"
    href="/docs/tokens/advanced/cpi"
  >
    Kutsu Token Programia ketjussa olevasta ohjelmasta — siirrä, mintaa ja polta
    tokeneita, allekirjoita PDA-auktoriteetilla ja yhdistä ohjeet yhdeksi CPI:ksi.
  </Card>

<Card
  title="Ylimääräisten Lamporttien Nosto"
  href="/docs/tokens/advanced/withdraw-excess-lamports"
>
  Palauta SOL vuokravapaan minimin ylittävältä osalta token accounteista,
  minteistä ja multisig-tileiltä muuttamatta token-saldoja.
</Card>

  <Card title="Lamporttien Purkaminen" href="/docs/tokens/advanced/unwrap-lamports">
    Muunna kääritty SOL takaisin lamporteiksi, kokonaan tai osittain, sulkematta
    token accountia.
  </Card>
</Cards>

<Callout type="info">

_rs`WithdrawExcessLamports`_ ja _rs`UnwrapLamports`_ ovat saatavilla
[`@solana-program/token`](https://www.npmjs.com/package/@solana-program/token)
-asiakkaan kautta. Token Programin kutsuminen CPI:n kautta koskee ketjussa
olevia ohjelmia.

</Callout>

## Lamport-palautus

_rs`WithdrawExcessLamports`_ palauttaa SOL:n vuokravapaan minimin ylittävältä
osalta token accounteista, minteistä ja multisig-tileiltä. _rs`UnwrapLamports`_
muuntaa natiivin token accountin käärityn SOL:n takaisin lamporteiksi.
