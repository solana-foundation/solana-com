## Mitä lamportien purku tekee?

_rs`UnwrapLamports`_ siirtää SOL:in natiivista (kääritystä SOL) token account
-tilistä kohdetilille lamporteina. Voit purkaa koko saldon tai tietyn määrän, ja
token account pysyy auki.

Tämä on joustavampaa kuin käärityn SOL-tilin sulkeminen: osittainen purku
mahdollistaa osan SOL:in nostamisen samalla kun tili pidetään rahoitettuna
myöhempää käyttöä varten.

## Tilit

| Tili          | Kuvaus                                                                 |
| ------------- | ---------------------------------------------------------------------- |
| `source`      | Kirjoitettavissa. Natiivi (kääritty SOL) token account, joka puretaan. |
| `destination` | Kirjoitettavissa. Vastaanottaa puretut lamportit.                      |
| `authority`   | Tilin omistaja tai valtuutettu (tai moniallekirjoittajat).             |

<Callout type="warn">

Lähteen on oltava natiivi token account; muut token accountit palauttavat
_rs`NonNativeNotSupported`_. Valtuutettu voi purkaa enintään valtuutetun
määränsä verran.

</Callout>

## Lamportien purkaminen

Jätä _ts`amount`_ pois purkaaksesi koko saldon, tai aseta se purkaaksesi tietyn
määrän lamportteja. Esimerkki käyttää konfiguroitua
[`@solana/kit`](/docs/clients/official/javascript#solana-kit) -asiakasta — katso
[Siirrä tokeneita](/docs/tokens/basics/transfer-tokens) asiakkaan asetuksista.

```ts title="Unwrap wrapped SOL (Kit)"
import { getUnwrapLamportsInstruction } from "@solana-program/token";

// source: the native (wrapped SOL) token account to unwrap.
// destination: address that receives the unwrapped lamports.
// authority: a TransactionSigner (the account owner or a delegate).

// Unwrap a specific number of lamports.
const unwrapSome = getUnwrapLamportsInstruction({
  source,
  destination,
  authority,
  amount: 500_000_000n
});

// Or omit `amount` to unwrap the entire balance.
const unwrapAll = getUnwrapLamportsInstruction({
  source,
  destination,
  authority
});

// Send whichever instruction you need.
await client.sendTransaction([unwrapSome]);
```

Moniallekirjoitusvaltuudella välitä yhteisallekirjoittajat
_ts`multiSigners`_:llä.
