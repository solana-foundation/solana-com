z tokenami SPL, koncentrując się na funkcjonalności wspólnej zarówno dla
oryginalnego Token Program, jak i Token Extensions Program (Token 2022).

  Więcej informacji na temat dodawania metadanych do tokenów SPL znajdziesz w
  [dokumentacji Metaplex](https://developers.metaplex.com/token-metadata).
