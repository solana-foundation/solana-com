Poza typowymi operacjami tworzenia, mintowania i transferu, Token Program jest
również wywoływany z programów on-chain za pośrednictwem CPI i zawiera
instrukcje odzyskiwania lamport:

<Cards>
  <Card
    title="Wywoływanie Token Program przez CPI"
    href="/docs/tokens/advanced/cpi"
  >
    Wywołaj Token Program z programu on-chain — transferuj, mintuj i spalaj
    tokeny, podpisuj z uprawnieniami PDA i łącz instrukcje w jedno wywołanie CPI.
  </Card>

<Card
  title="Wypłata nadmiaru lamport"
  href="/docs/tokens/advanced/withdraw-excess-lamports"
>
  Odzyskuj SOL powyżej minimum zwolnionego z czynszu z token account, mintów i
  kont multisig bez zmiany sald tokenów.
</Card>

  <Card title="Rozpakowywanie lamport" href="/docs/tokens/advanced/unwrap-lamports">
    Konwertuj opakowane SOL z powrotem na lamport, w całości lub częściowo, bez zamykania
    token account.
  </Card>
</Cards>

<Callout type="info">

_rs`WithdrawExcessLamports`_ i _rs`UnwrapLamports`_ są dostępne przez klienta
[`@solana-program/token`](https://www.npmjs.com/package/@solana-program/token).
Wywoływanie Token Program przez CPI dotyczy programów on-chain.

</Callout>

## Odzyskiwanie lamport

_rs`WithdrawExcessLamports`_ odzyskuje SOL powyżej minimum zwolnionego z czynszu
z token account, mintów i kont multisig. _rs`UnwrapLamports`_ konwertuje
opakowane SOL w natywnym token account z powrotem na lamport.
