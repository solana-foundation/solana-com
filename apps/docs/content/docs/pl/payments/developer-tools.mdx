## Narzędzia do rozwoju AI

Rozpocznij pracę z naszym przewodnikiem:
[jak zacząć korzystać z narzędzi AI na Solanie](/developers/guides/getstarted/intro-to-ai).

### Solana MCP

[Solana MCP](https://mcp.solana.com/) (Machine Conversation Protocol) integruje
wiedzę o Solanie bezpośrednio z narzędziami do kodowania wspieranymi przez AI,
takimi jak Claude, Cursor i Windsurf. Model jest trenowany na dokumentacji
Solany, frameworku Anchor, przykładach programów i pytaniach i odpowiedziach ze
Stack Exchange – przydatny do szybszego budowania integracji płatności.

### Frameworki agentów AI

Agenci AI mogą autonomicznie wykonywać transakcje płatnicze na Solanie. Te
frameworki umożliwiają budowanie agentów, którzy przetwarzają płatności,
monitorują transakcje i automatyzują przepływy płatności:

| Framework        | Opis                                                                                      | Linki                                                                                     |
| ---------------- | ----------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | Narzędzia do budowy agentów obsługujących płatności, kompatybilne z LangChain i Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | Zbiór gotowych umiejętności do budowy agentów na Solanie                                  | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | Wieloplatformowy framework agentów z pluginem Solana dla botów X, Telegram, Discord       | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Toolkit cross-chain z akcjami płatności na Solanie                                        | [GitHub](https://github.com/goat-sdk/goat)                                                |
