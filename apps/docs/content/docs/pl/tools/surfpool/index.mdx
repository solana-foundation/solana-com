Surfpool to zamiennik drop-in dla `solana-test-validator`, stworzony specjalnie,
aby zapewnić jak najlepsze doświadczenie deweloperom budującym na **Solana**.

Jego kluczową funkcją jest umożliwienie deweloperom lokalnej symulacji programów
przy użyciu kont Mainnet pobieranych w czasie rzeczywistym.

Surfpool płynnie wprowadza również Infrastructure as Code do projektów opartych
na Anchor lub Pinocchio, umożliwiając odtwarzalne, audytowalne i bezpieczne
wdrożenia w dowolnej sieci Solana — prywatnej lub publicznej.
