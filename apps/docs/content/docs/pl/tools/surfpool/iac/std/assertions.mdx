## assert_eq

`assert_eq` sprawdza, czy dwie wartości są równe.

### Dane wejściowe

| Nazwa   | Typ | Opis                         |
| ------- | --- | ---------------------------- |
| `left`  | any | Wartość do porównania        |
| `right` | any | Wartość, z którą porównujemy |

### Dane wyjściowe

| Nazwa   | Typ                   | Opis          |
| ------- | --------------------- | ------------- |
| `value` | addon(std::assertion) | Wynik asercji |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```

## assert_gt

`assert_gt` sprawdza, czy lewa wartość jest większa niż prawa wartość.

### Dane wejściowe

| Nazwa   | Typ              | Opis                                                        |
| ------- | ---------------- | ----------------------------------------------------------- |
| `left`  | integer \| float | Liczba całkowita lub zmiennoprzecinkowa do porównania       |
| `right` | integer \| float | Liczba całkowita lub zmiennoprzecinkowa do porównania z nią |

### Dane wyjściowe

| Nazwa   | Typ                   | Opis          |
| ------- | --------------------- | ------------- |
| `value` | addon(std::assertion) | Wynik asercji |

```hcl
output "assertion" {
    value = std::assert_gt(action.example.result, 1)
}
```
