## ripemd160

`ripemd160` oblicza skrót Ripemd160 wartości.

### Dane wejściowe

| Nazwa   | Typ                     | Opis                                               |
| ------- | ----------------------- | -------------------------------------------------- |
| `value` | buffer \| array[buffer] | Wartość zakodowana w formacie hex do zaszyfrowania |

### Dane wyjściowe

| Nazwa   | Typ    | Opis         |
| ------- | ------ | ------------ |
| `value` | string | Wynik skrótu |

```hcl
output "hashed_data" {
    value = ripemd160(encode_hex("hello, world"))
}
// > hashed_data: 0XA3201F82FCA034E46D10CD7B27E174976E241DA2
```

| ------- | ------ | ------------ |
| `value` | string | Wynik skrótu |

```hcl

}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```

## keccak256

### Dane wejściowe

| Nazwa   | Typ    | Opis                              |
| ------- | ------ | --------------------------------- |
| `value` | string | Wartość tekstowa do zaszyfrowania |

### Dane wyjściowe

| Nazwa   | Typ    | Opis         |
| ------- | ------ | ------------ |
| `value` | string | Wynik skrótu |

```hcl
output "hashed_data" {
    value = keccak256("hello, world")
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```
