Te funkcje są dostępne podczas korzystania z dodatku SVM dla Solana i
blockchainów kompatybilnych z SVM.

## system_program_id

`svm::system_program_id` zwraca identyfikator System Program,
`11111111111111111111111111111111`.

```hcl
output "system_program_id" {
    value = svm::system_program_id()
}
// > 11111111111111111111111111111111
```

## get_instruction_data_from_idl_path

`svm::get_instruction_data_from_idl_path` tworzy zakodowane instruction data dla
wywołania programu, zapewniając sprawdzanie typów i serializację na podstawie
dostarczonego pliku IDL.

### Dane wejściowe

| Name               | Required | Type          | Description                                                     |
| ------------------ | -------- | ------------- | --------------------------------------------------------------- |
| `idl_path`         | required | string        | Ścieżka, względna do txtx.yml, do pliku IDL `.json`             |
| `instruction_name` | required | string        | Nazwa instrukcji, dla której mają zostać wygenerowane dane      |
| `arguments`        | optional | array[string] | Argumenty instrukcji, dla których mają zostać wygenerowane dane |

```hcl
output "data" {
    value = svm::get_instruction_data_from_idl("/path/to/idl.json", "my_instruction", ["arg1", "arg2"])
}
// > data: 0x95763bdcc47fa1b305000000776f726c64
```

## get_instruction_data_from_idl

`svm::get_instruction_data_from_idl` tworzy zakodowane instruction data dla
wywołania programu, zapewniając sprawdzanie typów i serializację na podstawie
dostarczonych danych IDL.

### Dane wejściowe

| Name               | Required | Type              | Description                                                                        |
| ------------------ | -------- | ----------------- | ---------------------------------------------------------------------------------- |
| `idl`              | required | string \| SVM_IDL | IDL programu                                                                       |
| `instruction_name` | required | string            | Nazwa instrukcji, dla której mają zostać wygenerowane dane, zgodnie z indeksem IDL |
| `arguments`        | optional | array[string]     | Argumenty instrukcji, dla których mają zostać wygenerowane dane                    |

```hcl
output "data" {
    value = svm::get_instruction_data_from_idl(variable.contract.idl, "my_instruction", ["arg1", "arg2"])
}
// > data: 0x95763bdcc47fa1b305000000776f726c64
```

## get_program_from_anchor_project

`svm::get_program_from_anchor_project` pobiera artefakty wdrożenia programu dla
programu w projekcie Anchor.

### Dane wejściowe

| Nazwa          | Wymagane   | Typ    | Opis                                 |
| -------------- | ---------- | ------ | ------------------------------------ |
| `program_name` | wymagane   | string | Nazwa wdrażanego programu            |
| `keypair_path` | opcjonalne | string | Lokalizacja pliku keypair programu   |
| `idl_path`     | opcjonalne | string | Lokalizacja pliku IDL programu       |
| `bin_path`     | opcjonalne | string | Lokalizacja pliku binarnego programu |

```hcl
variable "contract" {
    value = svm::get_program_from_anchor_project("my_program")
}
output "idl" {
    value = variable.contract.idl
}
```

## sol_to_lamports

`svm::sol_to_lamports` konwertuje podaną kwotę SOL na lamport.

```hcl
output "lamports" {
    value = svm::sol_to_lamports(1.1)
}
// lamports: 1100000000
```

## find_pda

`svm::find_pda` znajduje prawidłowy PDA przy użyciu podanego identyfikatora
programu i ziaren (seeds).

### Dane wejściowe

| Nazwa        | Wymagane   | Typ           | Opis                                                    |
| ------------ | ---------- | ------------- | ------------------------------------------------------- |
| `program_id` | wymagane   | string        | Adres programu, z którego pochodzi PDA                  |
| `seeds`      | opcjonalne | array[string] | Opcjonalna tablica ziaren (maks. 16, po 32 bajty każde) |

```hcl
variable "pda" {
    value = svm::find_pda("3bv3j4GvMPjvvBX9QdoX27pVoWhDSXpwKZipFF1QiVr6", ["data"])
}
output "pda" {
    value = std::encode_base58(variable.pda.pda)
}
output "bump" {
    value = variable.pda.bump_seed
}
// > pda: 4amHoWMBgLkPfM8Nq9ZP33Liq9FCuqrLoU1feejkdsUJ
// > bump: 252
```

## get_associated_token_account

`svm::get_associated_token_account` oblicza adres associated token account dla
podanych adresów portfela i mint tokena.

### Dane wejściowe

| Name                 | Required | Type   | Description       |
| -------------------- | -------- | ------ | ----------------- |
| `wallet_address`     | required | string | Adres portfela    |
| `token_mint_address` | optional | string | Adres mint tokena |

```hcl
variable "token_account" {
    value = svm::get_associated_token_account(signer.caller.address, "So11111111111111111111111111111111111111112")
}
```

## create_token_account_instruction

`svm::create_token_account_instruction` tworzy surowe bajty instrukcji do
utworzenia associated token account.

### Dane wejściowe

| Name                 | Required | Type   | Description               |
| -------------------- | -------- | ------ | ------------------------- |
| `funding_address`    | required | string | Adres konta finansującego |
| `wallet_address`     | required | string | Adres portfela            |
| `token_mint_address` | optional | string | Adres mint tokena         |
| `token_program_id`   | optional | string | Adres token program       |

```hcl
action "call" "svm::process_instructions" {
    signers = [signer.caller]

    instruction {
        raw_bytes = svm::create_token_account_instruction(
            signer.caller.address, // funding address
            signer.caller.address, // wallet address
            variable.token_mint, // token mint address
            variable.token_program // token program id
        )
    }
}
```

## u64

`svm::u64` tworzy reprezentację tablicy bajtów liczby całkowitej u64,
odpowiednią do użycia jako seed w derywacji PDA.

### Dane wejściowe

| Name    | Required | Type    | Description                                                 |
| ------- | -------- | ------- | ----------------------------------------------------------- |
| `value` | required | integer | Liczba całkowita u64 do przekonwertowania na tablicę bajtów |

```hcl
variable "pda" {
    value = svm::find_pda("3bv3j4GvMPjvvBX9QdoX27pVoWhDSXpwKZipFF1QiVr6", [svm::u64(1000000000)])
}
```
