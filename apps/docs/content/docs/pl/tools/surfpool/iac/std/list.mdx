## index

`index` pobiera wpis z listy pod podanym indeksem.

### Dane wejściowe

| Nazwa   | Typ                                              | Opis                                |
| ------- | ------------------------------------------------ | ----------------------------------- |
| `list`  | array[string] \| array[integer] \| array[buffer] | Lista, z której pobierany jest wpis |
| `index` | integer                                          | Indeks wpisu do pobrania            |

### Dane wyjściowe

| Nazwa   | Typ    | Opis                              |
| ------- | ------ | --------------------------------- |
| `value` | string | Wpis z listy pod podanym indeksem |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
