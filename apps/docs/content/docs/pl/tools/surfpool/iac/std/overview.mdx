Biblioteka standardowa udostępnia podstawowe funkcje i akcje, które są dostępne
dla wszystkich Runbooków niezależnie od tego, które dodatki blockchain są
włączone.

## Funkcje

Biblioteka standardowa zawiera funkcje narzędziowe dla:

- **Operatory**: Operacje matematyczne i porównania
- **Hash**: Funkcje skrótu (SHA256, Keccak256 itp.)
- **Hex**: Kodowanie i dekodowanie szesnastkowe
- **Crypto**: Operacje kryptograficzne
- **JSON**: Kodowanie i dekodowanie JSON
- **List**: Funkcje manipulacji listami
- **Assertions**: Asercje równości i warunków
- **Base64**: Kodowanie i dekodowanie Base64

[Zobacz wszystkie funkcje](/docs/tools/surfpool/iac/std/functions)

## Akcje

Biblioteka standardowa zawiera akcje dla:

- **HTTP**: Wysyłanie żądań HTTP
- **Operacje na plikach**: Odczyt i zapis plików

[Zobacz wszystkie akcje](/docs/tools/surfpool/iac/std/actions)
