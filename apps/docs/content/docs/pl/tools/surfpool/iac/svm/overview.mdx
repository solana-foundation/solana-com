---
title: Przegląd dodatku SVM
description: Dodatek dla Solana i blockchainów kompatybilnych z SVM (beta)
---

Wtyczka SVM `txtx` umożliwia budowanie Runbooków, które współdziałają z Solana i
blockchainem kompatybilnym z SVM. Wtyczka udostępnia funkcje narzędziowe
pozwalające na wdrażanie programów Anchor oraz kodowanie wywołań instrukcji
zgodnie z IDL programów. Akcje mogą być wykorzystywane do tworzenia poprawnych
transakcji transferu, wywołań programów oraz wdrożeń programów, które można
podpisać za pomocą frazy mnemonicznej, klucza tajnego lub przeglądarki z
podpisywarką.

## Szybki start

Aby użyć dodatku SVM, zadeklaruj go w swoim Runbooku:

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## Funkcje

- **Wdrażanie programów**: Wdrażaj programy Anchor i natywne programy Solana
- **Budowanie instrukcji**: Koduj wywołania instrukcji przy użyciu IDL programów
- **Podpisywanie transakcji**: Obsługa fraz mnemonicznych, kluczy tajnych i
  portfeli internetowych
- **Zarządzanie kontami**: Sprawdzaj i zarządzaj kontami Solana
