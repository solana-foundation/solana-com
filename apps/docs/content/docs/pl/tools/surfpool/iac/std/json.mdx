## jq

Funkcja `jq` umożliwia wycinanie, filtrowanie i mapowanie danych JSON. Więcej
informacji można znaleźć w
[dokumentacji jq](https://jqlang.github.io/jq/manual/).

### Dane wejściowe

| Nazwa          | Typ              | Opis                                                                          |
| -------------- | ---------------- | ----------------------------------------------------------------------------- |
| `decoded_json` | string \| object | Obiekt JSON                                                                   |
| `query`        | string           | Zapytanie JSON. Zobacz [dokumentację jq](https://jqlang.github.io/jq/manual/) |

### Dane wyjściowe

| Nazwa   | Typ           | Opis                 |
| ------- | ------------- | -------------------- |
| `value` | array[string] | Wynik zapytania `jq` |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
