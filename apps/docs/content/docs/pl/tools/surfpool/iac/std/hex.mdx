## encode_hex

`encode_hex` koduje bufor lub ciąg znaków jako ciąg szesnastkowy z prefiksem 0x.

### Dane wejściowe

| Nazwa   | Typ                   | Opis                                                                                                                                          |
| ------- | --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | Bufor lub ciąg znaków do zakodowania. Ciągi zaczynające się od '0x' są dekodowane jako hex; w przeciwnym razie używane są surowe bajty UTF-8. |

### Dane wyjściowe

| Nazwa   | Typ    | Opis                                                        |
| ------- | ------ | ----------------------------------------------------------- |
| `value` | string | Dane wejściowe w reprezentacji szesnastkowej z prefiksem 0x |

```hcl
output "encoded_hex" {
    value = encode_hex("hello, world")
}
// > encoded_hex: 0x68656c6c6f2c20776f726c64
```

## decode_hex

`decode_hex` dekoduje ciąg szesnastkowy i zwraca wynik jako bufor.

### Dane wejściowe

| Nazwa        | Typ    | Opis                                              |
| ------------ | ------ | ------------------------------------------------- |
| `hex_string` | string | Ciąg hex do zdekodowania (z prefiksem 0x lub bez) |

### Dane wyjściowe

| Nazwa   | Typ    | Opis                            |
| ------- | ------ | ------------------------------- |
| `value` | buffer | Zdekodowany ciąg hex jako bufor |

```hcl
output "decoded_hex" {
    value = decode_hex("0x68656c6c6f2c20776f726c64")
}
// > decoded_hex: 0x68656c6c6f2c20776f726c64
```
