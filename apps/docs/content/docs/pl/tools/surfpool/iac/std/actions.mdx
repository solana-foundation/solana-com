Te akcje są dostępne we wszystkich Runbookach jako część biblioteki
standardowej.

## send_http_request

Akcja `std::send_http_request` wysyła żądanie HTTP pod wskazany adres URL.

### Dane wejściowe

| Nazwa        | Wymagane   | Typ     | Opis                                                 |
| ------------ | ---------- | ------- | ---------------------------------------------------- |
| `url`        | wymagane   | string  | Adres URL, na który zostanie wysłane żądanie         |
| `method`     | opcjonalne | string  | Metoda HTTP (GET, POST, PUT, DELETE). Domyślnie: GET |
| `headers`    | opcjonalne | map     | Nagłówki HTTP dołączane do żądania                   |
| `body`       | opcjonalne | string  | Treść żądania (dla żądań POST/PUT)                   |
| `timeout_ms` | opcjonalne | integer | Limit czasu żądania w milisekundach                  |

### Dane wyjściowe

| Nazwa         | Typ     | Opis                |
| ------------- | ------- | ------------------- |
| `status_code` | integer | Kod statusu HTTP    |
| `body`        | string  | Treść odpowiedzi    |
| `headers`     | map     | Nagłówki odpowiedzi |

```hcl
action "api_call" "std::send_http_request" {
    description = "Fetch data from API"
    url = "https://api.example.com/data"
    method = "GET"
    headers = {
        "Authorization" = "Bearer ${variable.token}"
    }
}

output "response" {
    value = action.api_call.body
}

output "status" {
    value = action.api_call.status_code
}
```

### Przykład żądania POST

```hcl
action "create_resource" "std::send_http_request" {
    description = "Create a new resource"
    url = "https://api.example.com/resources"
    method = "POST"
    headers = {
        "Content-Type" = "application/json"
    }
    body = {
        "name" = variable.name,
        "value" = variable.value
    }
}
```
