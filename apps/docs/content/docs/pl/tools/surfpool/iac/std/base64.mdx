## encode_base64

`encode_base64` koduje bufor lub ciąg znaków jako ciąg base64.

### Dane wejściowe

| Name    | Type                  | Description                                                                                                                                   |
| ------- | --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | Bufor lub ciąg znaków do zakodowania. Ciągi zaczynające się od '0x' są dekodowane jako hex; w przeciwnym razie używane są surowe bajty UTF-8. |

### Dane wyjściowe

| Name    | Type   | Description                                |
| ------- | ------ | ------------------------------------------ |
| `value` | string | Dane wejściowe zakodowane jako ciąg base64 |

```hcl
output "encoded" {
    value = encode_base64("0x48656c6c6f20776f726c6421")
}
// > encoded: SGVsbG8gd29ybGQh
```

## decode_base64

`decode_base64` dekoduje ciąg zakodowany w base64 i zwraca wynik jako bufor.

### Dane wejściowe

| Name            | Type   | Description                 |
| --------------- | ------ | --------------------------- |
| `base64_string` | string | Ciąg base64 do zdekodowania |

### Dane wyjściowe

| Name    | Type   | Description                        |
| ------- | ------ | ---------------------------------- |
| `value` | buffer | Zdekodowany ciąg base64 jako bufor |

```hcl
output "decoded" {
    value = decode_base64("SGVsbG8gd29ybGQh")
}
// > decoded: 0x48656c6c6f20776f726c6421
```
