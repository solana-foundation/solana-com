## and_bool

`and_bool` zwraca binarne AND argumentów lewostronnego i prawostronnego.

### Wejścia

| Nazwa | Typ  | Opis             |
| ----- | ---- | ---------------- |
| `lhs` | bool | Wartość logiczna |
| `rhs` | bool | Wartość logiczna |

### Wyjście

| Nazwa   | Typ  | Opis                                         |
| ------- | ---- | -------------------------------------------- |
| `value` | bool | Wynik binarnego AND między dwoma argumentami |

```hcl
output "my_bool" {
  value = false && true
}
// > my_bool: false
```

## div

`div` zwraca wynik całkowitoliczbowego dzielenia argumentu lewostronnego przez
argument prawostronny, zaokrąglając ewentualną resztę w dół do najbliższej
liczby całkowitej.

### Wejścia

| Nazwa | Typ     | Opis     |
| ----- | ------- | -------- |
| `lhs` | integer | Dzielna  |
| `rhs` | integer | Dzielnik |

### Wyjście

| Nazwa   | Typ     | Opis                                    |
| ------- | ------- | --------------------------------------- |
| `value` | integer | Wynik dzielenia dzielnej przez dzielnik |

```hcl
output "my_int" {
  value = 11 / -3
}
// > my_int: -3
```

## eq

`eq` zwraca `true`, jeśli argumenty po lewej i prawej stronie są równe, oraz
`false`, jeśli nie są.

### Dane wejściowe

| Nazwa | Typ     | Opis            |
| ----- | ------- | --------------- |
| `lhs` | dowolny | Dowolna wartość |
| `rhs` | dowolny | Dowolna wartość |

### Dane wyjściowe

| Nazwa   | Typ  | Opis                                              |
| ------- | ---- | ------------------------------------------------- |
| `value` | bool | Wynik sprawdzenia równości między dwoma wejściami |

```hcl
output "is_eq" {
  value = "arg" == "arg"
}
// > is_eq: true
```

## gt

`gt` zwraca `true`, jeśli argument po lewej stronie jest większy niż argument po
prawej stronie.

### Dane wejściowe

| Nazwa | Typ                        | Opis                 |
| ----- | -------------------------- | -------------------- |
| `lhs` | integer \| float \| string | Wartość porównywalna |
| `rhs` | integer \| float \| string | Wartość porównywalna |

### Dane wyjściowe

| Nazwa   | Typ  | Opis                                                                                         |
| ------- | ---- | -------------------------------------------------------------------------------------------- |
| `value` | bool | Wynik sprawdzenia, czy argument po lewej stronie jest większy niż argument po prawej stronie |

```hcl
output "is_gt" {
  value = 2 > 1
}
// > is_gt: true
```

## neq

`neq` zwraca `true`, jeśli argumenty po lewej i prawej stronie nie są równe, i
`false` w przeciwnym razie.

### Dane wejściowe

| Nazwa | Typ | Opis                        |
| ----- | --- | --------------------------- |
| `lhs` | any | Wartość do porównania       |
| `rhs` | any | Wartość do porównania z nią |

### Wyjście

| Nazwa   | Typ  | Opis                                                           |
| ------- | ---- | -------------------------------------------------------------- |
| `value` | bool | Wynik zanegowanego sprawdzenia równości między dwoma wejściami |

```hcl
output "is_neq" {
  value = "arg" != "arg"
}
// > is_neq: false
```
