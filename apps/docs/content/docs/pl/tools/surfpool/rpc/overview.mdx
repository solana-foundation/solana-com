Surfnet to lokalny symulator sieci Solana firmy Surfpool, zaprojektowany w celu
zapewnienia programistom potężnego środowiska testowego, które odwzorowuje
zachowanie blockchaina Solana bez konieczności uruchamiania pełnego węzła
validator.

## Pierwsze kroki

Uruchom instancję Surfnet za pomocą CLI Surfpool:

```bash
surfpool start
```

Spowoduje to uruchomienie lokalnej instancji Surfnet na `http://localhost:8899`
z wszystkimi dostępnymi punktami końcowymi RPC.

## Główne funkcje

- **Pełna zgodność z RPC**: Obsługuje kompletne API RPC Solana
- **Szybka iteracja**: Nie trzeba czekać na potwierdzenia bloków
- **Manipulacja stanem**: Cheatcody do bezpośredniej modyfikacji stanu
  blockchaina
- **Fork mainnet**: Forkuj z mainnet, aby testować na rzeczywistych kontach i
  programach
- **Wsparcie dla wtyczek Geyser**: Podłącz swoje wtyczki Geyser do
  strumieniowania danych w czasie rzeczywistym
