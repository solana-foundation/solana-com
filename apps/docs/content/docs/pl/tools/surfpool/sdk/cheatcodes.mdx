---
title: Cheatcodes
description:
  Pomijaj transakcje i bezpośrednio modyfikuj stan Surfnet. Zasilaj SOL i
  tokeny, ustawiaj dowolne dane kont, resetuj konta buforowane z upstream i
  strumieniuj konta mainnet na żywo.
---

Cheatcodes to mutacje stanu, które omijają normalny przepływ transakcji.
Wykonują się nатychmiast, bez zużywania blockhasha ani ponoszenia opłat – to
dokładnie to, czego potrzebujesz przy konfiguracji testów.

W Rust, cheatcodes znajdują się pod `Surfnet::cheatcodes()`. W JS są to metody
bezpośrednio na instancji `Surfnet`.

## Zasilanie SOL

`fundSol` / `fund_sol` ustawia saldo w lamport na dowolnym koncie, tworząc
konto, jeśli nie istnieje.

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::{Pubkey, Surfnet};

let surfnet = Surfnet::start().await?;
let cheats = surfnet.cheatcodes();
let wallet = Pubkey::new_unique();

cheats.fund_sol(&wallet, 1_000_000_000)?;

// Fund several accounts in one call.
let bob = Pubkey::new_unique();
let carol = Pubkey::new_unique();
cheats.fund_sol_many(&[(&bob, 2_000_000_000), (&carol, 3_000_000_000)])?;
```

```ts !! title="TypeScript"
import { Surfnet } from "@solana/surfpool";

const surfnet = Surfnet.start();
const wallet = Surfnet.newKeypair();

surfnet.fundSol(wallet.publicKey, 1_000_000_000);

const bob = Surfnet.newKeypair();
const carol = Surfnet.newKeypair();
surfnet.fundSolMany([
  { address: bob.publicKey, lamports: 2_000_000_000 },
  { address: carol.publicKey, lamports: 3_000_000_000 }
]);
```

</CodeTabs>

## Zasilanie tokenów

`fundToken` / `fund_token` mintuje tokeny do portfela, obliczając powiązany
token account, tworząc go w razie potrzeby i ustawiając kwotę.

<Callout type="info" title="Minty Token-2022">
  Domyślnie `fundToken` używa klasycznego Token Program SPL. Dla mintów
  Token-2022 przekaż ID programu Token-2022
  (`TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb`) jako ostatni argument.
</Callout>

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::{Pubkey, Surfnet};

let surfnet = Surfnet::start().await?;
let cheats = surfnet.cheatcodes();

let wallet = Pubkey::new_unique();
let mint: Pubkey = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v"
    .parse()
    .unwrap();

// Classic SPL Token mint.
cheats.fund_token(&wallet, &mint, 5_000_000, None)?;

// Token-2022 mint — pass the program id explicitly.
let token_2022: Pubkey = "TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb"
    .parse()
    .unwrap();
cheats.fund_token(&wallet, &mint, 5_000_000, Some(&token_2022))?;

// Fund many wallets with the same mint.
let mob = vec![Pubkey::new_unique(), Pubkey::new_unique()];
cheats.fund_token_many(&mob.iter().collect::<Vec<_>>(), &mint, 1_000_000, None)?;
```

```ts !! title="TypeScript"
const surfnet = Surfnet.start();
const wallet = Surfnet.newKeypair();
const USDC = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v";

surfnet.fundToken(wallet.publicKey, USDC, 5_000_000);

// Token-2022 mint — pass the program id explicitly.
const TOKEN_2022 = "TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb";
surfnet.fundToken(wallet.publicKey, USDC, 5_000_000, TOKEN_2022);

// Fund many wallets at once.
const mob = [Surfnet.newKeypair().publicKey, Surfnet.newKeypair().publicKey];
surfnet.fundTokenMany(mob, USDC, 1_000_000);
```

</CodeTabs>

Aby uzyskać ATA bez zasilania, użyj `getAta` / `get_ata`:

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
let ata = cheats.get_ata(&wallet, &mint, None);
```

```ts !! title="TypeScript"
const ata = surfnet.getAta(wallet.publicKey, USDC);
```

</CodeTabs>

## Ustawianie dowolnego stanu konta

Metoda `setAccount` zapisuje lamport, właściciela i surowe bajty danych dla
dowolnego konta w jednym wywołaniu. Rust SDK dodatkowo udostępnia builder
`SetAccount` dla bardziej zaawansowanych pól, takich jak `rent_epoch` i
`executable`.

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::cheatcodes::builders::SetAccount;
use surfpool_sdk::{Pubkey, Surfnet};

let surfnet = Surfnet::start().await?;
let cheats = surfnet.cheatcodes();

let address = Pubkey::new_unique();
let owner = Pubkey::new_unique();

// Direct method.
cheats.set_account(&address, 500_000, &[1, 2, 3], &owner)?;

// Builder when you need finer control.
cheats.execute(
    SetAccount::new(address)
        .lamports(500_000)
        .owner(owner)
        .data(vec![1, 2, 3])
        .rent_epoch(0)
        .executable(false),
)?;
```

```ts !! title="TypeScript"
const surfnet = Surfnet.start();
const address = Surfnet.newKeypair();
const owner = Surfnet.newKeypair();

surfnet.setAccount(
  address.publicKey,
  500_000,
  new Uint8Array([1, 2, 3]),
  owner.publicKey
);
```

</CodeTabs>

## Mutowanie Pól Token Account

`setTokenAccount` aktualizuje zaawansowane pola istniejącego token account —
delegata, stan, uprawnienia do zamknięcia, delegowaną kwotę. Używaj tej metody,
gdy samo `fundToken` nie wystarcza.

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::cheatcodes::builders::SetTokenAccount;
use surfpool_sdk::{Pubkey, Surfnet};

let cheats = Surfnet::start().await?.cheatcodes();

let owner = Pubkey::new_unique();
let mint: Pubkey = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v".parse()?;
let delegate = Pubkey::new_unique();

cheats.execute(
    SetTokenAccount::new(owner, mint)
        .amount(2_000_000)
        .delegate(delegate)
        .delegated_amount(500_000)
        .state("initialized"),
)?;

// Later: clear the delegation.
cheats.execute(SetTokenAccount::new(owner, mint).clear_delegate())?;
```

```ts !! title="TypeScript"
const surfnet = Surfnet.start();
const owner = Surfnet.newKeypair();
const USDC = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v";
const delegate = Surfnet.newKeypair();

surfnet.setTokenAccount(owner.publicKey, USDC, {
  amount: 2_000_000,
  delegate: delegate.publicKey,
  delegatedAmount: 500_000,
  state: "initialized"
});

// Later: clear the delegation.
surfnet.setTokenAccount(owner.publicKey, USDC, { clearDelegate: true });
```

</CodeTabs>

## Resetowanie Kont do Stanu Upstream

`resetAccount` odrzuca wszelkie lokalne modyfikacje i ponownie pobiera konto z
upstream RPC. Przekaż `includeOwnedAccounts: true`, aby zresetować również każde
konto będące własnością celu — przydatne przy resetowaniu programu i wszystkich
jego PDA.

<Callout type="info" title="Wymaga `remoteRpcUrl` do ponownego pobrania">
  Na offline'owym Surfnecie, `resetAccount` usuwa lokalną kopię, ale nie ma
  źródła upstream do pobrania danych. Skonfiguruj `remoteRpcUrl` przy
  uruchomieniu, aby resetowanie przywracało stan upstream.
</Callout>

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::cheatcodes::builders::ResetAccount;
use surfpool_sdk::{Pubkey, Surfnet};

let cheats = Surfnet::start().await?.cheatcodes();
let token_program: Pubkey = "TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA".parse()?;

cheats.execute(
    ResetAccount::new(token_program).include_owned_accounts(true),
)?;
```

```ts !! title="TypeScript"
const surfnet = Surfnet.start();
const TOKEN_PROGRAM = "TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA";

surfnet.resetAccount(TOKEN_PROGRAM, { includeOwnedAccounts: true });
```

</CodeTabs>

## Strumieniowanie Kont na Żywo

`streamAccount` rejestruje konto do ciągłego odpytywania w tle z upstream RPC.
Lokalna kopia Surfnetu pozostaje zsynchronizowana z mainnecie bez konieczności
jawnego odpytywania w testach. Połącz z `includeOwnedAccounts`, aby
strumieniować wszystkie PDA będące własnością programu.

<Callout type="warn" title="Wymaga `remoteRpcUrl`">
  `streamAccount` działa tylko na Surfnecie uruchomionym z `remoteRpcUrl` (lub
  `remote_rpc_url` w Rust). Bez upstream RPC nie ma źródła do strumieniowania
  danych, a wywołanie zwróci błąd.
</Callout>

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::cheatcodes::builders::StreamAccount;
use surfpool_sdk::{Pubkey, Surfnet};

let cheats = Surfnet::builder()
    .remote_rpc_url("https://api.mainnet-beta.solana.com")
    .start()
    .await?
    .cheatcodes();

let oracle: Pubkey = "H6ARHf6YXhGYeQfUzQNGk6rDNnLBQKrenN712K4AQJEG".parse()?;

cheats.execute(StreamAccount::new(oracle).include_owned_accounts(false))?;
```

```ts !! title="TypeScript"
const surfnet = Surfnet.startWithConfig({
  remoteRpcUrl: "https://api.mainnet-beta.solana.com"
});

surfnet.streamAccount("H6ARHf6YXhGYeQfUzQNGk6rDNnLBQKrenN712K4AQJEG");
```

</CodeTabs>

## Buildery Cheatcode (Rust)

Rust SDK udostępnia typowane buildery w ramach
`surfpool_sdk::cheatcodes::builders` dla przypadków, gdy metody pomocnicze nie
są wystarczające. Każdy builder implementuje trait `CheatcodeBuilder` i jest
wykonywany przez `cheats.execute(builder)`.

| Builder           | Constructor                                          | Common setters                                                                                                                   |
| ----------------- | ---------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| `SetAccount`      | `new(address: Pubkey)`                               | `lamports`, `data`, `owner`, `rent_epoch`, `executable`                                                                          |
| `SetTokenAccount` | `new(owner: Pubkey, mint: Pubkey)`                   | `amount`, `delegate`, `clear_delegate`, `state`, `delegated_amount`, `close_authority`, `clear_close_authority`, `token_program` |
| `ResetAccount`    | `new(address: Pubkey)`                               | `include_owned_accounts(bool)`                                                                                                   |
| `StreamAccount`   | `new(address: Pubkey)`                               | `include_owned_accounts(bool)`                                                                                                   |
| `DeployProgram`   | `new(program_id: Pubkey)`, `from_keypair_path(path)` | `so_path`, `so_bytes`, `idl_path`                                                                                                |

Pełną dokumentację JS znajdziesz w
[Dokumentacji JS](/docs/tools/surfpool/sdk/js-reference). Pełną dokumentację
Rust znajdziesz w [Dokumentacji Rust](/docs/tools/surfpool/sdk/rust-reference).
