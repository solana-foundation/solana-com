CLI surfpool można zainstalować za pomocą skryptu instalacyjnego lub skompilować
ze źródeł.

## Skrypt instalacyjny

Aby zainstalować surfpool, otwórz terminal i uruchom:

```bash
curl -sL https://run.surfpool.run/ | bash
```

## Instalacja ze źródeł

Aby skompilować plik binarny Surfpool ze źródeł, sklonuj repozytorium surfpool.

```bash
git clone https://github.com/solana-foundation/surfpool.git
```

Przejdź do nowego katalogu.

```bash
cd surfpool
```

Następnie skompiluj plik binarny. To polecenie skompiluje plik binarny i zapisze
go w `~/.cargo/bin/surfpool`.

```bash
cargo surfpool-install
```

Na koniec upewnij się, że plik binarny `surfpool` jest dostępny w zmiennej PATH.
Ten proces będzie się różnić w zależności od systemu operacyjnego.
