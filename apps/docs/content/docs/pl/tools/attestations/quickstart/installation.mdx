Rozpocznij pracę z Solana Attestations, instalując niezbędne pakiety i
konfigurując środowisko programistyczne.

## TypeScript/JavaScript

Zainstaluj bibliotekę Solana Attestations używając preferowanego menedżera
pakietów:

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
