## Skrócony przewodnik

### Konfiguracja wtyczki

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### Zarządzanie kontami (client.svm)

| Metoda                                 | Opis                                       |
| -------------------------------------- | ------------------------------------------ |
| `setAccount(account)`                  | Bezpośrednie ustawienie stanu konta        |
| `getAccount(address)`                  | Pobranie danych konta                      |
| `getBalance(address)`                  | Pobierz saldo SOL                          |
| `airdrop(address, lamports)`           | Zasilenie konta                            |
| `minimumBalanceForRentExemption(size)` | Oblicz minimalną kwotę zwolnioną z czynszu |

### Zarządzanie programami (client.svm)

| Metoda                                | Opis                        |
| ------------------------------------- | --------------------------- |
| `addProgram(programId, bytes)`        | Załaduj program z bajtów    |
| `addProgramFromFile(programId, path)` | Załaduj program z pliku .so |

### Transakcje (client.svm)

| Metoda                    | Opis                         |
| ------------------------- | ---------------------------- |
| `sendTransaction(tx)`     | Wykonaj transakcję           |
| `simulateTransaction(tx)` | Symulacja bez zmiany stanu   |
| `latestBlockhash()`       | Pobierz bieżący blockhash    |
| `expireBlockhash()`       | Przejdź do nowego blockhasha |

### Konfiguracja (client.svm)

| Metoda                      | Opis                              |
| --------------------------- | --------------------------------- |
| `withSigverify(bool)`       | Włącz/wyłącz weryfikację podpisów |
| `withBlockhashCheck(bool)`  | Włącz/wyłącz walidację blockhasha |
| `withSysvars()`             | Włącz konta sysvar                |
| `withBuiltins()`            | Włącz wbudowane programy          |
| `withPrecompiles()`         | Włącz programy prekompilowane     |
| `withTransactionHistory(n)` | Przechowuj ostatnie n transakcji  |
| `withComputeBudget(budget)` | Ustaw limity obliczeniowe         |
| `withLamports(amount)`      | Ustaw domyślne lamport            |
| `withLogBytesLimit(n)`      | Ustaw limit wyjścia dziennika     |

### Czas i zmienne systemowe (client.svm)

| Metoda                     | Opis                             |
| -------------------------- | -------------------------------- |
| `warpToSlot(slot)`         | Przejdź do określonego slot      |
| `getClock()`               | Pobierz zmienną systemową zegara |
| `getRent()`                | Pobierz zmienną systemową rent   |
| `getEpochSchedule()`       | Pobierz harmonogram epoch        |
| `getLastRestartSlot()`     | Pobierz ostatni slot restartu    |
| `setLastRestartSlot(slot)` | Ustaw ostatni slot restartu      |

### RPC (client.rpc)

| Metoda                           | Opis                                    |
| -------------------------------- | --------------------------------------- |
| `getAccountInfo(address)`        | Pobierz pojedyncze konto (tylko base64) |
| `getMultipleAccounts(addresses)` | Pobierz wiele kont                      |
| `getLatestBlockhash()`           | Pobierz aktualny blockhash              |
