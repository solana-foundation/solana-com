## Podręczny przewodnik

| Metoda                                         | Opis                                                                                           |
| ---------------------------------------------- | ---------------------------------------------------------------------------------------------- |
| `LiteSVM::new()`                               | Tworzy nowe podstawowe środowisko testowe LiteSVM                                              |
| `LiteSVM::default()`                           | Zwraca „wartość domyślną“ dla danego typu                                                      |
| `airdrop(&Address, lamports)`                  | Zasila konto określoną liczbą lamport                                                          |
| `get_account(&Address)`                        | Zwraca wszystkie informacje powiązane z kontem o podanym adresie                               |
| `set_account(Address, Account)`                | Ustawia wszystkie informacje powiązane z kontem o podanym adresie                              |
| `get_balance(&Address)`                        | Pobiera saldo w lamport dla podanego adresu konta                                              |
| `minimum_balance_for_rent_exemption(data_len)` | Zwraca minimalne saldo wymagane do zwolnienia konta o określonej długości danych z rent        |
| `add_program(program_id, &bytes)`              | Dodaje program SBF do środowiska testowego                                                     |
| `add_program_from_file(program_id, path)`      | Dodaje program SBF do środowiska testowego z określonego pliku                                 |
| `send_transaction(tx)`                         | Przesyła podpisaną transakcję                                                                  |
| `simulate_transaction(tx)`                     | Symuluje transakcję bez modyfikowania stanu                                                    |
| `get_transaction(&signature)`                  | Pobiera transakcję z historii                                                                  |
| `latest_blockhash()`                           | Pobiera aktualny blockhash                                                                     |
| `expire_blockhash()`                           | Wymusza wygaśnięcie aktualnego blockhash                                                       |
| `warp_to_slot(slot)`                           | Przewija zegar do określonego slot                                                             |
| `get_sysvar<T>()`                              | Pobiera sysvar ze środowiska testowego                                                         |
| `set_sysvar<T>(&value)`                        | Ustawia sysvar w środowisku testowym                                                           |
| `with_feature_accounts()`                      | Włącza konta feature gate do środowiska testowego                                              |
| `set_invocation_inspect_callback(callback)`    | Rejestruje wywołanie zwrotne przed/po wywołaniu (wymaga funkcji `invocation-inspect-callback`) |
