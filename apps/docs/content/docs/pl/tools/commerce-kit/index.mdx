**Commerce Kit to zbiór narzędzi do tworzenia doświadczeń handlowych w
Solanie.** Od gotowych przycisków płatności po prymitywy headless, Commerce Kit
dostarcza wszystko, czego potrzebujesz do przyjmowania płatności
kryptowalutowych.

## Dlaczego Commerce Kit?

- **Gotowe komponenty**: Predefiniowany PaymentButton z połączeniem portfela,
  wyborem tokenów i obsługą transakcji
- **Niezależny od frameworka**: Używaj prymitywów headless w dowolnym frameworku
  JavaScript
- **Solana Pay**: Pełne wsparcie Solana Pay z generowaniem kodów QR
- **Bezpieczne typy**: Kompleksowe wsparcie TypeScript we wszystkich pakietach
- **Wallet Standard**: Kompatybilny ze wszystkimi portfelami Wallet Standard
  (Phantom, Solflare, Backpack itp.)

## Szybki start

Najszybszym sposobem na rozpoczęcie jest użycie komponentu `PaymentButton`:

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## Instalacja

Wybierz potrzebne pakiety Commerce Kit:

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
