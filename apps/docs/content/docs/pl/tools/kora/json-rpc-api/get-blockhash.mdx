---
title: getBlockhash
description:
  Pobiera najnowszy blockhash z Solana RPC, z którym połączony jest serwer Kora.
---

## Żądanie JSON-RPC

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getBlockhash",
  "params": []
}
```

## Odpowiedź JSON-RPC

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "blockhash": "base58Blockhash"
  }
}
```

## Przykład cURL

```bash
curl -X POST http://localhost:8080 \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"getBlockhash","params":[]}'
```

## TypeScript SDK

```typescript
const { blockhash } = await client.getBlockhash();
console.log("Current blockhash:", blockhash);
```
