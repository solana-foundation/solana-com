## Cargo

Zainstaluj wersję beta bezpośrednio z crates.io:

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

Oficjalne wieloplatformowe obrazy Docker (amd64/arm64) są publikowane w GitHub
Container Registry:

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

Dostępne tagi:

- `latest` — Najnowsze stabilne wydanie
- `beta` — Najnowsza wersja przedpremierowa
- `v<version>` — Konkretna wersja (np. `v2.2.0-beta.7`)

### Uruchamianie z Dockerem

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## TypeScript SDK

Wersja beta SDK jest dostępna na npm:

```bash
pnpm add @solana/kora@beta
```

Zobacz [Zmiany w SDK](/docs/tools/kora/beta/sdk-changes), aby poznać nowe API.

## Weryfikacja instalacji

```bash
kora --version
```
