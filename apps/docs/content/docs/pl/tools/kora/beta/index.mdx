_Dostępne na GitHub:
[`v2.2.0-beta.7`](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.7)_

Kora v2.2.0-beta wprowadza obsługę pakietów Jito, nowe metody RPC, obrazy
Docker, ulepszone funkcje bezpieczeństwa oraz usprawnienia SDK.

> **To jest wersja wstępna.** Funkcje udokumentowane tutaj są dostępne w
> `v2.2.0-beta.7` i mogą ulec zmianie przed stabilnym wydaniem. Aby zapoznać się
> ze stabilną dokumentacją, zobacz [główną dokumentację Kora](/docs/tools/kora).

## Najważniejsze zmiany

### Obsługa pakietów Jito

Przesyłaj do 5 transakcji atomowo za pośrednictwem silnika bloków Jito. Nowe
metody RPC:

- [`signBundle`](/docs/tools/kora/beta/bundle-methods#signbundle) — Podpisz
  pakiet bez transmisji
- [`signAndSendBundle`](/docs/tools/kora/beta/bundle-methods#signandsendbundle)
  — Podpisz i prześlij do Jito
- [`estimateBundleFee`](/docs/tools/kora/beta/bundle-methods#estimatebundlefee)
  — Oszacuj całkowite opłaty pakietu

Zobacz [przewodnik po pakietach Jito](/docs/tools/kora/beta/jito-guide), aby
zapoznać się z pełnym przewodnikiem.

### Obrazy Docker

Oficjalne obrazy wieloplatformowe (amd64/arm64) opublikowane w GitHub Container
Registry. Zobacz [Instalacja](/docs/tools/kora/beta/installation).

### Wtyczki transakcyjne

Nowy system wtyczek umożliwia Korze walidację struktur transakcji podczas
przepływów podpisywania. Pierwsza wbudowana wtyczka, `gas_swap`, wymusza ścisłe
zasady dla transakcji wymiany tokenów na SOL bez opłat za gaz. Wtyczki są
konfigurowane w `kora.toml` i uruchamiane automatycznie podczas
`signTransaction`, `signAndSendTransaction`, `signBundle`, oraz
`signAndSendBundle`.

Zobacz
[Konfiguracja — Wtyczki transakcyjne](/docs/tools/kora/beta/configuration#transaction-plugins).

### Nowe opcje konfiguracji

- **Wtyczki transakcyjne** do walidacji struktury transakcji (`gas_swap`)
- **reCAPTCHA v3** ochrona przed botami dla punktów końcowych podpisywania
- **Lighthouse** ochrona przed wyczerpaniem płatnika opłat
- **Limity użycia** ograniczanie liczby transakcji na portfel
- **Konfiguracja pakietów** do integracji z Jito

Zobacz [Konfiguracja](/docs/tools/kora/beta/configuration).

### Zmiany w SDK

- Obsługa metod pakietów (`signBundle`, `signAndSendBundle`,
  `estimateBundleFee`)

Zobacz [Zmiany w SDK](/docs/tools/kora/beta/sdk-changes).

### Dodatkowe metody RPC

- [`getVersion`](/docs/tools/kora/beta/bundle-methods#getversion) — Pobierz
  informacje o wersji serwera

## Instalacja

```bash
cargo install kora-cli@2.2.0-beta.7
```

Lub za pomocą Dockera:

```bash
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```
