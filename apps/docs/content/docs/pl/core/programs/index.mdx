Program Solana to [konto](/docs/core/accounts/account-types#program-accounts),
które zawiera wykonywalny bajtkod sBPF i ma ustawioną flagę `executable` na
`true`. Programy są bezstanowe. Cały stan możliwy do zmiany przechowywany jest w
osobnych kontach danych przekazywanych przez
[instrukcje](/docs/core/instructions).

## Najważniejsze informacje

- **Kompilowane do sBPF**: Programy są kompilowane do Solana Bytecode Format
  (sBPF) przez LLVM i przechowywane w kontach wykonywalnych.
- **Bezstanowe**: Cały stan możliwy do zmiany przechowywany jest w osobnych
  kontach danych, a nie w program account.
- **Aktualizowalne**: Programy wdrożone z loader-v3 (BPF Loader Upgradeable)
  mogą być aktualizowane, jeśli ustawiono uprawnienia do aktualizacji; cofnięcie
  tych uprawnień sprawia, że program staje się niezmienny.
