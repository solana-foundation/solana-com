Ta strona stanowi jedno miejsce referencyjne dla wszystkich stałych środowiska
uruchomieniowego wymienionych na stronach dotyczących kluczowych koncepcji.
Każda stała zawiera odnośnik do swojego źródłowego zdefiniowania w kliencie
validatora [agave](https://github.com/anza-xyz/agave/tree/v3.1.8) lub w
[solana-sdk](https://github.com/anza-xyz/solana-sdk).
