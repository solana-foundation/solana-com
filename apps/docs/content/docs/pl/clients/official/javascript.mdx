## Solana Kit

`@solana/kit` to zalecane SDK TypeScript do budowania na Solanie. Składasz
klienta z kompozytowalnych wtyczek, dodajesz powiązania React gdy ich
potrzebujesz i dołączasz wygenerowane klienty programów dla programów on-chain,
które wywołujesz.

| Pakiet                    | Opis                                                 | GitHub                                                                                 |
| ------------------------- | ---------------------------------------------------- | -------------------------------------------------------------------------------------- |
| @solana/kit               | Podstawowe SDK: RPC, transakcje, kodeki, podpisujący | [Źródło](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | Powiązania React: `ClientProvider` i hooki           | [Źródło](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | Połączenie RPC + wysyłanie transakcji                | [Źródło](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | Wtyczki płatnika opłat, tożsamości i podpisującego   | [Źródło](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Połączenie Wallet Standard + hooki React             | [Źródło](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

## Solana Web3.js

`@solana/web3.js` to starsze SDK TypeScript dla Solany. Nowe aplikacje powinny
używać `@solana/kit`; zobacz [Migracja do Kit](/docs/frontend/web3-compat).

| Package           | Opis                                                                     | GitHub                                                                             |
| ----------------- | ------------------------------------------------------------------------ | ---------------------------------------------------------------------------------- |
| @solana/web3.js   | Podstawowe SDK                                                           | [Źródło](https://github.com/solana-foundation/solana-web3.js)                      |
| @solana/spl-token | Interakcja z programami Token, Token-2022 oraz Associated Token programs | [Źródło](https://github.com/solana-program/token-2022/tree/main/clients/js-legacy) |
| @solana/spl-memo  | Interakcja z programem Memo                                              | [Źródło](https://github.com/solana-program/memo/tree/main/clients/js-legacy)       |
