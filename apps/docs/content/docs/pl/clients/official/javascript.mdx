## Solana Client

`@solana/client` to bezgłowowy runtime Solana, który łączy RPC, portfele,
transakcje i subskrypcje w jednym miejscu.

| Pakiet         | Opis               | GitHub                                                                                 |
| -------------- | ------------------ | -------------------------------------------------------------------------------------- |
| @solana/client | Bezgłowowy runtime | [Źródło](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [Przewodnik: przegląd @solana/client](/docs/frontend/client)

## Solana React Hooks

`@solana/react-hooks` dodaje providera React i hooki na bazie `@solana/client`,
wykorzystując ten sam runtime i cache.

| Pakiet              | Opis                         | GitHub                                                                                      |
| ------------------- | ---------------------------- | ------------------------------------------------------------------------------------------- |
| @solana/react-hooks | Provider React + hooki do UI | [Źródło](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [Przewodnik: @solana/react-hooks](/docs/frontend/react-hooks)
