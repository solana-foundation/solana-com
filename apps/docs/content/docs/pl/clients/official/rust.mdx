## Moduły programu

Użyj `solana-program`, aby tworzyć programy on-chain.

- [Przykład: Tworzenie programu Solana](/docs/programs/rust)

| Moduł          | Opis                         | Dokumentacja                             | GitHub                                                               |
| -------------- | ---------------------------- | ---------------------------------------- | -------------------------------------------------------------------- |
| solana-program | Tworzenie programów on-chain | [Zobacz](https://docs.rs/solana-program) | [Źródło](https://github.com/anza-xyz/solana-sdk/tree/master/program) |

Pinocchio to biblioteka bez zależności do tworzenia programów Solana w języku
Rust, która może być używana jako zamiennik `solana-program` do pisania
programów on-chain.

| Moduł     | Opis                             | Dokumentacja                        | GitHub                                          |
| --------- | -------------------------------- | ----------------------------------- | ----------------------------------------------- |
| pinocchio | Programy on-chain bez zależności | [Zobacz](https://docs.rs/pinocchio) | [Źródło](https://github.com/anza-xyz/pinocchio) |

<Callout type="warn">
  Pinocchio jest nadal w fazie rozwoju i prawdopodobnie wkrótce pojawią się
  zmiany wstecznie niekompatybilne.
</Callout>

Pinocchio zawiera moduły specyficzne dla programów, które umożliwiają interakcję
z programami Solana:

| Moduł                              | Opis                                  | Dokumentacja                                                  | GitHub                                                                                      |

| pinocchio-token                    | Interakcja z Token Program            | [Zobacz](https://docs.rs/pinocchio-token/)                    | [Źródło](https://github.com/anza-xyz/pinocchio/tree/main/programs/token)                    |
| pinocchio-token-2022               | Interakcja z Token-2022 Program       | [Zobacz](https://docs.rs/pinocchio-token-2022/)               | [Źródło](https://github.com/anza-xyz/pinocchio/tree/main/programs/token-2022)               |
| pinocchio-associated-token-account | Interakcja z Associated Token Program | [Zobacz](https://docs.rs/pinocchio-associated-token-account/) | [Źródło](https://github.com/anza-xyz/pinocchio/tree/main/programs/associated-token-account) |
