Surfpool to narzędzie do lokalnego rozwoju, które upraszcza pracę z Solaną
poprzez automatyczne ładowanie programów i kont z klastra (mainnet, devnet).
Użyj Surfpool CLI, aby uruchomić lokalny validator do testowania swoich
programów i transakcji.

Odwiedź [dokumentację Surfpool](https://docs.surfpool.run/), aby uzyskać więcej
informacji.

## Uruchom lokalny validator

Aby uruchomić lokalny validator, wykonaj następujące polecenie:

```terminal
$ surfpool start
```

Domyślnie Surfpool automatycznie ładuje wszystkie programy i konta, od których
twoje transakcje zależą, z mainnetu do lokalnego validatora.

## Surfpool Studio

Surfpool Studio oferuje interfejs webowy do interakcji z twoim lokalnym
validatorem. Możesz go używać do:

- Przeglądania szczegółów transakcji
- Inspekcji danych konta
- Zrzutu SOL i innych tokenów za pomocą interfejsu kranika

Aby uzyskać dostęp do Surfpool Studio:

1. Uruchom lokalny validator za pomocą `surfpool start`
2. Otwórz przeglądarkę i przejdź do
   [http://127.0.0.1:18488](http://127.0.0.1:18488)
