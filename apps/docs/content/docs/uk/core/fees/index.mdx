Кожна [транзакція](/docs/core/transactions) Solana потребує сплати комісії в
SOL. Комісія складається з двох компонентів: **базової комісії** та опціональної
**комісії пріоритизації**. Базова комісія компенсує валідаторам криптографічну
роботу з перевірки підписів. Комісія пріоритизації збільшує ймовірність того, що
поточний лідер запланує вашу транзакцію раніше за конкуруючі.

## Ключові факти

- **Базова комісія**: за підпис, розподіляється 50% спалюється / 50% валідатору.
- **Комісія пріоритизації**:
  `ceil(compute_unit_price * compute_unit_limit / 1,000,000)` lamports. 100%
  валідатору.
