Ця сторінка є єдиним довідником для всіх констант середовища виконання, згаданих
на сторінках основних концепцій. Кожна константа посилається на своє визначення
в клієнті validator [agave](https://github.com/anza-xyz/agave/tree/v3.1.8) або
[solana-sdk](https://github.com/anza-xyz/solana-sdk).
