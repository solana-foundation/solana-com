Інструкція — це запит на виконання певної функції в програмі Solana. Інструкції
є фундаментальним будівельним блоком для операцій в мережі. Кожна інструкція
визначає рівно одну програму для виклику, необхідні їй облікові записи та масив
байтів даних, який програма інтерпретує (зазвичай дискримінатор для
ідентифікації обробника інструкції, за яким слідують серіалізовані аргументи).
Логіка виконання кожної інструкції зберігається в
[програмі](/docs/core/programs), де кожна програма визначає власний набір
інструкцій. Для взаємодії з мережею Solana одна або кілька інструкцій додаються
до [транзакції](/docs/core/transactions) та надсилаються в мережу для обробки.

## Ключові факти

- **Одна програма**: кожна інструкція спрямована рівно на одну програму через
  `program_id`.
- **Метадані облікового запису**: масив `accounts` надає
  [`AccountMeta`](/docs/core/instructions/instruction-structure#account-metadata)
  для кожного облікового запису, який інструкція читає або записує.
- **Непрозорі дані**: поле `data` є масивом байтів, формат якого визначається
  цільовою програмою.
