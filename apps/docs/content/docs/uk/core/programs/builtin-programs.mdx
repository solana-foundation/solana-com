  Основні програми забезпечують фундаментальну функціональність мережі:

## System Program

System Program (`11111111111111111111111111111111`) — єдина програма, яка може
створювати нові облікові записи. Усі нові облікові записи спочатку належать
System Program, хоча право власності зазвичай переназначається під час
створення. System Program споживає
[`DEFAULT_COMPUTE_UNITS`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L282)
(150 CU) на інструкцію.

### Довідник інструкцій System Program

| Інструкція                                                                                                             | Опис                                                                                                                                                                                                                                     |
| ---------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [`CreateAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L295)          | Створює новий обліковий запис. Виділяє `space` байтів, призначає `owner` та переказує `lamports` від платника. Новий обліковий запис і платник повинні підписати.                                                                        |
| [`CreateAccountWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L318)  | Те саме, що й `CreateAccount`, але виводить адресу нового облікового запису з pubkey облікового запису, рядка seed та програми-власника. Обліковий запис, чий pubkey використовувався для виведення адреси, і платник повинні підписати. |
| [`Assign`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L343)                 | Встановлює власника облікового запису. Обліковий запис повинен підписати. Не виконує дій, якщо вже є цей власник.                                                                                                                        |
| [`AssignWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L484)         | Те саме, що й `Assign`, але для адреси, виведеної з seed. Обліковий запис, чий pubkey використовувався для виведення адреси, повинен підписати.                                                                                          |
| [`Transfer`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L353)               | Переказує lamports між обліковими записами. Відправник повинен підписати.                                                                                                                                                                |
| [`TransferWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L357)       | Те саме, що й `Transfer`, але з адреси, виведеної з seed. Обліковий запис, чий pubkey використовувався для виведення адреси, повинен підписати.                                                                                          |
| [`Allocate`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L452)               | Встановлює довжину даних облікового запису. Обліковий запис повинен підписати, мати порожні дані та належати System Program. Максимум 10 МіБ.                                                                                            |
| [`AllocateWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L462)       | Те саме, що й `Allocate`, але для адреси, виведеної з seed. Також призначає власника. Обліковий запис, чий pubkey використовувався для виведення адреси, повинен підписати.                                                              |
| [`AdvanceNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L374)    | Просуває значення nonce до поточного blockhash. Повноваження nonce повинні підписати.                                                                                                                                                    |
| [`InitializeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L412) | Ініціалізує обліковий запис як nonce-обліковий запис із заданими повноваженнями. Обліковий запис повинен бути звільнений від rent.                                                                                                       |
| [`AuthorizeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L432)  | Змінює повноваження nonce. Поточні повноваження повинні підписати.                                                                                                                                                                       |
| [`WithdrawNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L392)   | Знімає lamports з nonce-облікового запису. Повноваження повинні підписати. Якщо не закривається, залишок повинен покривати rent.                                                                                                         |
| [`UpgradeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L437)    | Оновлює nonce-обліковий запис зі старого формату до поточної версії. Обліковий запис повинен бути доступним для запису та містити застарілу версію nonce.                                                                                |

## Програми-завантажувачі

Кожна програма належить програмі-завантажувачу. Завантажувачі обробляють
розгортання, оновлення та виконання програм, якими вони володіють. Дивіться
[Програми-завантажувачі](/docs/core/programs/program-deployment#loader-programs)
для повного списку завантажувачів та їхніх адрес.
