## Solana Client

`@solana/client` — це headless середовище виконання Solana, яке об'єднує RPC,
гаманці, транзакції та підписки в одному сховищі.

| Пакет          | Опис                    | GitHub                                                                                  |
| -------------- | ----------------------- | --------------------------------------------------------------------------------------- |
| @solana/client | Headless клієнт runtime | [Джерело](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [Посібник: огляд @solana/client](/docs/frontend/client)

## Solana React Hooks

`@solana/react-hooks` додає React-провайдер та хуки поверх `@solana/client`,
використовуючи те саме середовище виконання та кеш.

| Пакет               | Опис                          | GitHub                                                                                       |
| ------------------- | ----------------------------- | -------------------------------------------------------------------------------------------- |
| @solana/react-hooks | React-провайдер + хуки для UI | [Джерело](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [Посібник: @solana/react-hooks](/docs/frontend/react-hooks)
