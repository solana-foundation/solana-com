## Solana Kit

`@solana/kit` — рекомендований TypeScript SDK для розробки на Solana. Ви
збираєте клієнт із композованих плагінів, додаєте React-байдинги за потреби та
підключаєте згенеровані клієнти програм для викликів ончейн-програм.

| Пакет                     | Опис                                               | GitHub                                                                                  |
| ------------------------- | -------------------------------------------------- | --------------------------------------------------------------------------------------- |
| @solana/kit               | Основний SDK: RPC, транзакції, кодеки, підписувачі | [Джерело](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | React-байдинги: `ClientProvider` та хуки           | [Джерело](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | RPC-з'єднання + надсилання транзакцій              | [Джерело](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | Плагіни платника комісії, ідентифікації та підпису | [Джерело](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Підключення Wallet Standard + React-хуки           | [Джерело](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

### Клієнти програм

Згенеровані клієнти, що надають будівники інструкцій та клієнтські плагіни
(наприклад, `tokenProgram()` → `client.token`).

| Пакет                          | Опис                                 | GitHub                                                                           |
| ------------------------------ | ------------------------------------ | -------------------------------------------------------------------------------- |
| @solana-program/system         | Взаємодія з System Program           | [Джерело](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Взаємодія з Token Program            | [Джерело](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Взаємодія з програмою Token-2022     | [Джерело](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Взаємодія з програмою Memo           | [Джерело](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Взаємодія з програмою Compute Budget | [Джерело](https://github.com/solana-program/compute-budget/tree/main/clients/js) |

## Підпис і керування ключами

Для сумісного з kit підпису на бекенді з підтримкою кількох бекендів керування
ключами:

| Пакет            | Опис                                                                                                                          | GitHub                                                          |
| ---------------- | ----------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------- |
| @solana/keychain | Уніфікований підпис: Memory, Vault, Privy, Turnkey, AWS KMS, Fireblocks, GCP KMS, CDP, Para, Dfns, Crossmint, Openfort, Utila | [Джерело](https://github.com/solana-foundation/solana-keychain) |

Дивіться [документацію Keychain](/docs/tools/keychain) та
[Вибір бекенду](/docs/tools/keychain/choosing-a-backend) для настанов щодо
налаштування та моделі зберігання.

## Solana Web3.js

`@solana/web3.js` — це застарілий TypeScript SDK для Solana. Нові застосунки
мають використовувати `@solana/kit`; дивіться
[Міграція до Kit](/docs/frontend/web3-compat).

| Package           | Description                                                  | GitHub                                                                             |
| ----------------- | ------------------------------------------------------------ | ---------------------------------------------------------------------------------- |
| @solana/web3.js   | Основний SDK                                                 | [Source](https://github.com/solana-foundation/solana-web3.js)                      |
| @solana/spl-token | Взаємодія з програмами Token, Token-2022 та Associated Token | [Source](https://github.com/solana-program/token-2022/tree/main/clients/js-legacy) |
| @solana/spl-memo  | Взаємодія з програмою Memo                                   | [Source](https://github.com/solana-program/memo/tree/main/clients/js-legacy)       |
