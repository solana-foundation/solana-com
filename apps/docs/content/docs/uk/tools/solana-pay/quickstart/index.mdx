Швидко почніть роботу з Solana Pay за допомогою цих покрокових посібників.

## Встановлення

Налаштуйте SDK Solana Pay та сконфігуруйте своє середовище розробки.

**[→ Посібник з встановлення](/docs/tools/solana-pay/quickstart/installation)** -
Встановлення SDK та залежностей

## Типи платежів

Оберіть правильну інтеграцію платежів для вашого випадку використання:

### Запити на переказ (Прості)

Створюйте неінтерактивні URL-адреси платежів для базових переказів SOL та SPL
токенів.

**[→ Запити на переказ](/docs/tools/solana-pay/quickstart/transfer-requests)** -
Прості URL-адреси платежів

**Найкраще підходить для:**

- Базових платежів та пожертв
- Платежів через QR-код
- Платіжних посилань в електронних листах

### Запити транзакцій (Розширені)

Створюйте інтерактивні платіжні потоки з серверною композицією транзакцій.

**[→ Запити транзакцій](/docs/tools/solana-pay/quickstart/transaction-requests)** -
Інтерактивні платіжні потоки

**Найкраще підходить для:**

- Динамічного ціноутворення
- Мінтингу NFT
- Складних DeFi транзакцій
- Власної бізнес-логіки

## Інтеграція QR-кодів

Генеруйте та налаштовуйте QR-коди для платежів через мобільні гаманці.

**[→ Інтеграція QR-кодів](/docs/tools/solana-pay/quickstart/qr-codes)** -
Генерація скануємих платіжних кодів

**Найкраще підходить для:**

- Систем торгових точок
- Платежів з веб на мобільний
- Особистих транзакцій
