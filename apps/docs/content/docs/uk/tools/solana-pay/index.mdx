**Solana Pay — це стандартний протокол та набір еталонних реалізацій, які
дозволяють розробникам інтегрувати децентралізовані платежі у свої додатки та
сервіси.** Блокчейн Solana підтверджує транзакції менш ніж за секунду з
середньою вартістю $0.0005, забезпечуючи користувачам безперебійний досвід без
посередників.

## Чому Solana Pay?

- **Стандартний протокол**: Уніфікований підхід до платежів у всіх гаманцях та
  додатках Solana
- **Надшвидкість**: Транзакції підтверджуються менш ніж за секунду
- **Мінімальна вартість**: Середня вартість транзакції $0.0005
- **Без посередників**: Прямі платежі між користувачами без третіх сторін
- **Універсальна підтримка гаманців**: Сумісність з Phantom, Solflare, Backpack
  та іншими

## Швидкий старт

Найпростіший спосіб почати — використовувати URL-адресу запиту на переказ:

```
solana:recipient?amount=0.01&label=Coffee%20Shop&message=Grande%20Latte
```

Цю URL-адресу можна:

- Закодувати у QR-код для сканування мобільним гаманцем
- Використовувати як платіжне посилання у веб-додатках
- Вбудувати у мобільні додатки як глибокі посилання

## Встановлення

Встановіть JavaScript SDK для Solana Pay:

```bash
# Using pnpm (recommended)
pnpm add @solana/pay@beta @solana/kit

# Using npm
npm install @solana/pay@beta @solana/kit

# Using yarn
yarn add @solana/pay@beta @solana/kit
```

## Сценарії використання

Solana Pay забезпечує широкий спектр платіжних сценаріїв:

- **Роздрібні платежі**: Особисті платежі за допомогою QR-кодів
- **Електронна комерція**: Веб-процеси оформлення замовлень з інтеграцією
  гаманців
- **Мобільні додатки**: Нативні платіжні рішення з глибоким зв'язуванням
- **Мінтинг NFT**: Динамічне створення транзакцій для складних взаємодій
- **Підписки**: Регулярні платежі та програми лояльності
- **Інтеграція з DeFi**: Композитні транзакції зі свопами та кредитуванням

## Підтримувані гаманці

Solana Pay працює з усіма основними гаманцями Solana:

- **[Phantom](https://phantom.app/)** - Найпопулярніший гаманець Solana
- **[Solflare](https://solflare.com/)** - Мультиплатформний гаманець Solana
- **[Backpack](https://backpack.app/)** - Web3 гаманець і магазин застосунків
- **[Glow](https://glow.app/)** - Мобільний гаманець Solana
- **[Ultimate](https://ultimate.app/)** - Гаманець для DeFi
- **[Decaf Wallet](https://decaf.so/)** - Гаманець для глобальних платежів
- **[Espresso Cash](https://www.espressocash.com/)** - Гаманець для глобальних
  платежів
- **[Tiplink](https://tiplink.io/)** - Найпростіший гаманець у світі
- **[Bitget Wallet](https://bitget.com/)** - Мультиланцюговий гаманець
