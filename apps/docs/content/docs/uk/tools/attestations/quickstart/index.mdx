Почніть роботу з Solana Attestations всього за кілька кроків. Цей посібник
ознайомить вас з основними концепціями та допоможе створити вашу першу
атестацію.

## Що ви дізнаєтесь

- Встановлення та налаштування бібліотеки Attestations
- Створення вашого першого облікового запису (орган атестації)
- Визначення схеми для ваших атестацій
- Видача вашої першої атестації
- Отримання та перевірка атестацій

## Передумови

- Базові знання розробки на Solana
- Середовище розробки Node.js 16+ або Rust
- Гаманець Solana з деякою кількістю SOL для оплати комісій за транзакції

## Кроки швидкого старту

### 1. Встановлення

Встановіть необхідні пакети та налаштуйте своє середовище розробки.

**[→ Посібник з встановлення](/docs/tools/attestations/quickstart/installation)**

### 2. Створення облікового запису

Налаштуйте орган атестації, який може видавати атестації.

**[→ Посібник зі створення облікового запису](/docs/tools/attestations/instructions/create-credential)**

### 3. Визначення схеми

Створіть схему, яка визначає структуру ваших атестацій.

**[→ Посібник зі створення схеми](/docs/tools/attestations/instructions/create-schema)**

### 4. Видача атестації

Створіть ваше перше підтверджуване твердження за допомогою облікового запису та
схеми.

**[→ Посібник зі створення атестації](/docs/tools/attestations/instructions/create-attestation)**

## Основні концепції

### Облікові записи

Обліковий запис представляє орган атестації. Він визначає, хто може видавати
атестації, та містить список авторизованих підписувачів.

### Схеми

Схеми визначають структуру та правила валідації для атестацій. Вони вказують,
які поля містить атестація та їхні типи даних.

### Атестації

Атестації — це перевірені твердження або заяви, створені авторизованими
підписувачами згідно з обліковими даними та відповідно до конкретної структури
схеми.

## Варіанти використання

Атестації можна використовувати для:

- **Перевірка особи**: Підтвердження особи користувачів і відповідності вимогам
- **Системи репутації**: Створення оцінок довіри та показників репутації
- **Сертифікація**: Видача професійних сертифікатів і кваліфікацій
- **Комплаєнс**: Забезпечення відповідності нормативним вимогам для фінансових
  послуг
- **Ланцюги постачання**: Відстеження автентичності та походження товарів
