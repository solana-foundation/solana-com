Почніть роботу з Solana Attestations, встановивши необхідні пакети та
налаштувавши середовище розробки.

## TypeScript/JavaScript

Встановіть бібліотеку Solana Attestations за допомогою вашого улюбленого
менеджера пакетів:

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
