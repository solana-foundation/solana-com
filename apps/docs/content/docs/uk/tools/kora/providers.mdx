Не хочете запускати власний вузол Kora? Наступні постачальники пропонують
хостинг інфраструктури Kora, щоб ви могли додати абстракцію комісій до вашого
застосунку без необхідності керувати власним розгортанням.

## Постачальники

### Openfort

[Openfort](https://www.openfort.io/) пропонує інфраструктуру paymaster для
Solana на базі Kora, що забезпечує транзакції без комісій через їхню платформу
вбудованих гаманців. Їхня інтеграція використовує схему спільного підписання, де
Kora виступає як платник комісій, а користувачі підписують транзакції через
вбудовані гаманці Openfort.

- **Вебсайт**: [openfort.io](https://www.openfort.io/)
- **Документація**:
  [Транзакції без комісій у Solana](https://www.openfort.io/blog/solana-gasless-transactions)
- **Можливості**: Вбудовані гаманці, транзакції без комісій, спонсорство комісій

---

## Стати оператором вузла Kora

Kora є відкритим кодом, і будь-хто може запустити вузол. Незалежно від того, чи
створюєте ви застосунок, якому потрібна абстракція комісій, чи хочете надавати
послуги paymaster іншим, ознайомтеся з
[Посібником для операторів](/docs/tools/kora/operators), щоб розпочати.

## Додати до списку

Якщо ви керуєте вузлом Kora і хочете бути доданим на цю сторінку, зв'яжіться з
нами через канали
[Спільноти та підтримки Kora](https://github.com/solana-foundation/kora?tab=readme-ov-file#community--support).
