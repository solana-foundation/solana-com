## План:

### Методи автентифікації

- Без автентифікації (розробка)
- API ключ
- HMAC підпис
- Комбінована автентифікація

### Автентифікація за допомогою API ключа

- Налаштування
- Формат заголовка
- Налаштування клієнта

### HMAC автентифікація

- Налаштування секрету
- Генерація підпису
- Валідація часової мітки
- Реалізація клієнта

### Приклади TypeScript

- Налаштування API ключа
- HMAC підписування
- Обробка помилок

### Найкращі практики безпеки

- Ротація ключів
- Зберігання секретів
- Обмеження швидкості запитів
- Список дозволених IP-адрес

### Усунення несправностей

- Поширені помилки автентифікації
- Поради щодо налагодження
- Тестування автентифікації
