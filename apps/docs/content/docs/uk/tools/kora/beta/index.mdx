_Доступно на GitHub:
[`v2.2.0-beta.7`](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.7)_

Kora v2.2.0-beta представляє підтримку пакетів Jito, нові RPC-методи,
Docker-образи, покращені функції безпеки та вдосконалення SDK.

> **Це попередній реліз.** Функції, задокументовані тут, доступні в
> `v2.2.0-beta.7` і можуть змінитися до стабільного релізу. Для стабільної
> документації дивіться [основну документацію Kora](/docs/tools/kora).

## Основні моменти

### Підтримка пакетів Jito

Надсилайте до 5 транзакцій атомарно через блок-движок Jito. Нові RPC-методи:

- [`signBundle`](/docs/tools/kora/beta/bundle-methods#signbundle) — Підписати
  пакет без трансляції
- [`signAndSendBundle`](/docs/tools/kora/beta/bundle-methods#signandsendbundle)
  — Підписати та надіслати до Jito
- [`estimateBundleFee`](/docs/tools/kora/beta/bundle-methods#estimatebundlefee)
  — Оцінити загальні комісії пакета

Дивіться [Посібник з пакетів Jito](/docs/tools/kora/beta/jito-guide) для повного
ознайомлення.

### Docker-образи

Офіційні мультиплатформні образи (amd64/arm64) опубліковані в GitHub Container
Registry. Дивіться [Встановлення](/docs/tools/kora/beta/installation).

### Плагіни транзакцій

Нова система плагінів дозволяє Kora валідувати структуру транзакцій під час
процесів підписання. Перший вбудований плагін, `gas_swap`, застосовує суворі
правила для безгазових обмінів токенів на SOL. Плагіни налаштовуються в
`kora.toml` і запускаються автоматично під час `signTransaction`,
`signAndSendTransaction`, `signBundle`, та `signAndSendBundle`.

Дивіться
[Конфігурація — Плагіни транзакцій](/docs/tools/kora/beta/configuration#transaction-plugins).

### Нові параметри конфігурації

- **Плагіни транзакцій** для валідації структури транзакцій (`gas_swap`)
- **reCAPTCHA v3** захист від ботів для кінцевих точок підписання
- **Lighthouse** захист платника комісій від виснаження
- **Ліміти використання** обмеження транзакцій на гаманець
- **Конфігурація пакетів** для інтеграції з Jito

Дивіться [Конфігурація](/docs/tools/kora/beta/configuration).

### Зміни в SDK

- Підтримка методів пакетів (`signBundle`, `signAndSendBundle`,
  `estimateBundleFee`)

Дивіться [Зміни в SDK](/docs/tools/kora/beta/sdk-changes).

### Додаткові RPC-методи

- [`getVersion`](/docs/tools/kora/beta/bundle-methods#getversion) — Отримати
  інформацію про версію сервера

## Встановлення

```bash
cargo install kora-cli@2.2.0-beta.7
```

Або через Docker:

```bash
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```
