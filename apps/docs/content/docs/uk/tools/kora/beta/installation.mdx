## Cargo

Встановіть бета-версію безпосередньо з crates.io:

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

Офіційні мультиплатформні Docker-образи (amd64/arm64) опубліковано в GitHub
Container Registry:

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

Доступні теги:

- `latest` — Останній стабільний реліз
- `beta` — Остання попередня версія
- `v<version>` — Конкретна версія (наприклад, `v2.2.0-beta.7`)

### Docker Compose

Для користувацьких збірок або для включення Redis для кешування та підтримки
лімітів використання:

- [Приклад Dockerfile](https://github.com/solana-foundation/kora/blob/main/examples/deploy/Dockerfile)
- [Docker Compose з Redis](https://github.com/solana-foundation/kora/blob/main/docker-compose.yml)

### Запуск з Docker

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## TypeScript SDK

Бета-версія SDK доступна на npm:

```bash
pnpm add @solana/kora@beta
```

Дивіться [Зміни в SDK](/docs/tools/kora/beta/sdk-changes) для нових API.

## Перевірка встановлення

```bash
kora --version
```
