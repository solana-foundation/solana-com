**Commerce Kit — це колекція інструментів для створення комерційних сервісів на
Solana.** Від готових платіжних кнопок до headless-примітивів, Commerce Kit
надає все необхідне для прийому криптоплатежів.

## Чому Commerce Kit?

- **Готові компоненти**: Готова кнопка PaymentButton з підключенням гаманця,
  вибором токена та обробкою транзакцій
- **Незалежність від фреймворків**: Використовуйте headless-примітиви в
  будь-якому JavaScript-фреймворку
- **Solana Pay**: Повна підтримка Solana Pay з генерацією QR-кодів
- **Типобезпека**: Повна підтримка TypeScript у всіх пакетах
- **Wallet Standard**: Сумісність з усіма гаманцями Wallet Standard (Phantom,
  Solflare, Backpack тощо)

## Швидкий старт

Найшвидший спосіб почати — використати компонент `PaymentButton`:

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## Встановлення

Оберіть необхідні пакети Commerce Kit:

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
