CLI Surfpool можна встановити за допомогою скрипта встановлення або зібрати з
вихідного коду.

## Скрипт встановлення

Щоб встановити surfpool, відкрийте термінал і виконайте:

```bash
curl -sL https://run.surfpool.run/ | bash
```

## Встановлення з вихідного коду

Щоб скомпілювати бінарний файл Surfpool з вихідного коду, клонуйте репозиторій
surfpool.

```bash
git clone https://github.com/solana-foundation/surfpool.git
```

Перейдіть до нового каталогу.

```bash
cd surfpool
```

Потім скомпілюйте бінарний файл. Ця команда скомпілює бінарний файл і збереже
його в `~/.cargo/bin/surfpool`.

```bash
cargo surfpool-install
```

Нарешті, переконайтеся, що бінарний файл `surfpool` доступний у вашому PATH. Цей
процес відрізнятиметься залежно від вашої операційної системи.
