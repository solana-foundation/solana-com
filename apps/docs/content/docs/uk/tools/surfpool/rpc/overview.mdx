Surfnet — це локальний симулятор мережі Solana від Surfpool, розроблений для
надання розробникам потужного середовища тестування, яке відтворює поведінку
блокчейну Solana без накладних витрат повного вузла validator.

## Початок роботи

Запустіть екземпляр Surfnet за допомогою Surfpool CLI:

```bash
surfpool start
```

Це запустить локальний екземпляр Surfnet на `http://localhost:8899` з усіма
доступними RPC-ендпоінтами.

## Ключові можливості

- **Повна сумісність із RPC**: підтримує повний Solana RPC API
- **Швидка ітерація**: не потрібно чекати на підтвердження блоків
- **Маніпулювання станом**: чіткоди для безпосередньої зміни стану блокчейну
- **Форк мейннету**: форк з мейннету для тестування з реальними акаунтами та
  програмами
- **Підтримка плагіна Geyser**: підключайте свої плагіни Geyser для потокової
  передачі даних у реальному часі
