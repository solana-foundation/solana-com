Стандартна бібліотека надає основні функції та дії, доступні всім Runbook-ам
незалежно від того, які блокчейн-розширення увімкнено.

## Функції

Стандартна бібліотека включає службові функції для:

- **Оператори**: Математичні операції та порівняння
- **Hash**: Функції хешування (SHA256, Keccak256 тощо)
- **Hex**: Шістнадцяткове кодування та декодування
- **Crypto**: Криптографічні операції
- **JSON**: Кодування та декодування JSON
- **List**: Функції роботи зі списками
- **Assertions**: Перевірки рівності та умов
- **Base64**: Кодування та декодування Base64

[Переглянути всі функції](/docs/tools/surfpool/iac/std/functions)

## Дії

Стандартна бібліотека включає дії для:

- **HTTP**: Виконання HTTP-запитів
- **File Operations**: Читання та запис файлів

[Переглянути всі дії](/docs/tools/surfpool/iac/std/actions)
