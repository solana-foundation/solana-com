## ripemd160

`ripemd160` обчислює хеш Ripemd160 від значення.

### Вхідні дані

| Ім'я    | Тип                     | Опис                                   |
| ------- | ----------------------- | -------------------------------------- |
| `value` | buffer \| array[buffer] | Значення у hex-кодуванні для хешування |

### Вихідні дані

| Ім'я    | Тип    | Опис                |
| ------- | ------ | ------------------- |
| `value` | string | Результат хешування |

```hcl
output "hashed_data" {
    value = ripemd160(encode_hex("hello, world"))
}
// > hashed_data: 0XA3201F82FCA034E46D10CD7B27E174976E241DA2
```

## sha256

`sha256` обчислює хеш sha256 від значення.

### Вхідні дані

| Ім'я    | Тип                     | Опис                                   |
| ------- | ----------------------- | -------------------------------------- |
| `value` | buffer \| array[buffer] | Значення у hex-кодуванні для хешування |

### Вихідні дані

| Ім'я    | Тип    | Опис                |
| ------- | ------ | ------------------- |
| `value` | string | Результат хешування |

```hcl
output "hashed_data" {
    value = sha256(encode_hex("hello, world"))
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```
