## and_bool

`and_bool` повертає побітове І лівого та правого аргументів.

### Вхідні дані

| Назва | Тип  | Опис            |
| ----- | ---- | --------------- |
| `lhs` | bool | Булеве значення |
| `rhs` | bool | Булеве значення |

### Вивід

| Назва   | Тип  | Опис                                         |
| ------- | ---- | -------------------------------------------- |
| `value` | bool | Результат побітового І між двома аргументами |

```hcl
output "my_bool" {
  value = false && true
}
// > my_bool: false
```

## div

`div` повертає результат цілочисельного ділення лівого аргументу на правий
аргумент, округлюючи будь-який залишок до найближчого меншого цілого числа.

### Вхідні дані

| Назва | Тип     | Опис    |
| ----- | ------- | ------- |
| `lhs` | integer | Ділене  |
| `rhs` | integer | Дільник |

### Вивід

| Назва   | Тип     | Опис                                  |
| ------- | ------- | ------------------------------------- |
| `value` | integer | Результат ділення діленого на дільник |

```hcl
output "my_int" {
  value = 11 / -3
}
// > my_int: -3
```

## eq

`eq` повертає `true`, якщо аргументи лівої та правої частини рівні, і `false`,
якщо вони не рівні.

### Вхідні дані

| Назва | Тип       | Опис              |
| ----- | --------- | ----------------- |
| `lhs` | будь-який | Будь-яке значення |
| `rhs` | будь-який | Будь-яке значення |

### Вихідні дані

| Назва   | Тип  | Опис                                              |
| ------- | ---- | ------------------------------------------------- |
| `value` | bool | Результат перевірки рівності двох вхідних значень |

```hcl
output "is_eq" {
  value = "arg" == "arg"
}
// > is_eq: true
```

## gt

`gt` повертає `true`, якщо аргумент лівої частини більший за аргумент правої
частини.

### Вхідні дані

| Назва | Тип                         | Опис                 |
| ----- | --------------------------- | -------------------- |
| `lhs` | ціле число \| дріб \| рядок | Порівнюване значення |
| `rhs` | ціле число \| дріб \| рядок | Порівнюване значення |

### Вихідні дані

| Назва   | Тип  | Опис                                                                                     |
| ------- | ---- | ---------------------------------------------------------------------------------------- |
| `value` | bool | Результат перевірки того, чи є аргумент лівої частини більшим за аргумент правої частини |

```hcl
output "is_gt" {
  value = 2 > 1
}
// > is_gt: true
```

### Вихідні дані

| Назва   | Тип  | Опис                                                                                              |

| `value` | bool | Результат перевірки того, чи є аргумент лівої частини більшим або рівним аргументу правої частини |

## neq

`neq` повертає `true`, якщо аргументи лівої та правої частин не рівні, та
`false` в іншому випадку.

### Вхідні дані

| Назва | Тип | Опис                          |
| ----- | --- | ----------------------------- |
| `lhs` | any | Значення для порівняння       |
| `rhs` | any | Значення для порівняння з ним |

### Вивід

| Назва   | Тип  | Опис                                                            |
| ------- | ---- | --------------------------------------------------------------- |
| `value` | bool | Результат перевірки на нерівність між двома вхідними значеннями |

```hcl
output "is_neq" {
  value = "arg" != "arg"
}
// > is_neq: false
```

## minus

`minus` повертає результат віднімання аргументу `integer` правої частини від
аргументу `integer` лівої частини.

### Вхідні дані

| Назва | Тип     | Опис       |
| ----- | ------- | ---------- |
| `lhs` | integer | Зменшуване |
| `rhs` | integer | Від'ємник  |

### Вивід

| Назва   | Тип     | Опис                          |
| ------- | ------- | ----------------------------- |
| `value` | integer | Результат операції віднімання |

```hcl
output "my_integer" {
  value = 10 - 6
}
// > my_integer: 4
```

| Назва | Тип     | Опис    |
| ----- | ------- | ------- |
| `lhs` | integer | Ділене  |
| `rhs` | integer | Дільник |

### Вивід

| Назва   | Тип     | Опис                        |
