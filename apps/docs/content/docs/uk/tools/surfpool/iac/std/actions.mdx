Ці дії доступні у всіх Runbooks як частина стандартної бібліотеки.

## send_http_request

Дія `std::send_http_request` виконує HTTP-запит до вказаного URL.

### Вхідні параметри

| Назва        | Обов'язковий   | Тип     | Опис                                                       |
| ------------ | -------------- | ------- | ---------------------------------------------------------- |
| `url`        | обов'язковий   | string  | URL, на який надсилається запит                            |
| `method`     | необов'язковий | string  | HTTP-метод (GET, POST, PUT, DELETE). За замовчуванням: GET |
| `headers`    | необов'язковий | map     | HTTP-заголовки для включення в запит                       |
| `body`       | необов'язковий | string  | Тіло запиту (для запитів POST/PUT)                         |
| `timeout_ms` | необов'язковий | integer | Час очікування запиту в мілісекундах                       |

### Вихідні параметри

| Назва         | Тип     | Опис                |
| ------------- | ------- | ------------------- |
| `status_code` | integer | HTTP-код статусу    |
| `body`        | string  | Тіло відповіді      |
| `headers`     | map     | Заголовки відповіді |

```hcl
action "api_call" "std::send_http_request" {
    description = "Fetch data from API"
    url = "https://api.example.com/data"
    method = "GET"
    headers = {
        "Authorization" = "Bearer ${variable.token}"
    }
}

output "response" {
    value = action.api_call.body
}

output "status" {
    value = action.api_call.status_code
}
```

### Приклад POST-запиту

```hcl
action "create_resource" "std::send_http_request" {
    description = "Create a new resource"
    url = "https://api.example.com/resources"
    method = "POST"
    headers = {
        "Content-Type" = "application/json"
    }
    body = {
        "name" = variable.name,
        "value" = variable.value
    }
}
```
