## index

`index` отримує елемент зі списку за вказаним індексом.

### Вхідні дані

| Назва   | Тип                                              | Опис                                      |
| ------- | ------------------------------------------------ | ----------------------------------------- |
| `list`  | array[string] \| array[integer] \| array[buffer] | Список, з якого потрібно отримати елемент |
| `index` | integer                                          | Індекс елемента для отримання             |

### Вихідні дані

| Назва   | Тип    | Опис                                   |
| ------- | ------ | -------------------------------------- |
| `value` | string | Елемент зі списку за вказаним індексом |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
