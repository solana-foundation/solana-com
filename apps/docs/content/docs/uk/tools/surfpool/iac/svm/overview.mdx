---
title: Огляд SVM Addon
description: Додаток для Solana та SVM-сумісних блокчейнів (бета)
---

Плагін SVM `txtx` дозволяє створювати Runbook-и, що взаємодіють із Solana та
SVM-сумісними блокчейнами. Плагін надає утилітарні функції, які дозволяють
розгортати програми Anchor та кодувати виклики інструкцій відповідно до IDL
програм. Дії можна використовувати для створення валідних транзакцій переказу,
виклику програм і розгортання програм, які можна підписати за допомогою
мнемонічної фрази, секретного ключа або браузерного підписувача.

## Швидкий старт

Щоб використовувати SVM addon, оголосіть його у своєму Runbook:

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## Можливості

- **Розгортання програм**: Розгортання програм Anchor і нативних програм Solana
- **Побудова інструкцій**: Кодування викликів інструкцій за допомогою IDL
  програм
- **Підписання транзакцій**: Підтримка мнемонік, секретних ключів і веб-гаманців
- **Керування акаунтами**: Запит і управління акаунтами Solana
