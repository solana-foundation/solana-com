## assert_eq

`assert_eq` перевіряє, що два значення є рівними.

### Вхідні дані

| Назва   | Тип | Опис                         |
| ------- | --- | ---------------------------- |
| `left`  | any | Значення для порівняння      |
| `right` | any | Значення, з яким порівнювати |

### Вихідні дані

| Назва   | Тип                   | Опис                |
| ------- | --------------------- | ------------------- |
| `value` | addon(std::assertion) | Результат перевірки |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```

## assert_gt

`assert_gt` перевіряє, що ліве значення більше за праве.

### Вхідні дані

| Назва   | Тип              | Опис                                        |
| ------- | ---------------- | ------------------------------------------- |
| `left`  | integer \| float | Ціле або дробове число для порівняння       |
| `right` | integer \| float | Ціле або дробове число для порівняння з ним |

| Назва   | Тип                   | Опис                |
| ------- | --------------------- | ------------------- |
| `value` | addon(std::assertion) | Результат перевірки |

