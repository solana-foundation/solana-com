title: Функції Base64

---

## encode_base64

`encode_base64` кодує буфер або рядок як рядок base64.

### Вхідні дані

| Назва   | Тип                   | Опис                                                                                                                                         |
| ------- | --------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | буфер, рядок, додаток | Буфер або рядок для кодування. Рядки, що починаються з '0x', декодуються як шістнадцяткові; інакше використовуються необроблені байти UTF-8. |

### Вихідні дані

| Назва   | Тип   | Опис                                    |
| ------- | ----- | --------------------------------------- |
| `value` | рядок | Вхідні дані, закодовані як рядок base64 |

```hcl
output "encoded" {
    value = encode_base64("0x48656c6c6f20776f726c6421")
}
// > encoded: SGVsbG8gd29ybGQh
```

## decode_base64

`decode_base64` декодує рядок, закодований у base64, і повертає результат як
буфер.

### Вхідні дані

| Назва           | Тип   | Опис                         |
| --------------- | ----- | ---------------------------- |
| `base64_string` | рядок | Рядок base64 для декодування |

### Вихідні дані

| Назва   | Тип   | Опис                                      |
| ------- | ----- | ----------------------------------------- |
| `value` | буфер | Декодований рядок base64 у вигляді буфера |

```hcl
output "decoded" {
    value = decode_base64("SGVsbG8gd29ybGQh")
}
// > decoded: 0x48656c6c6f20776f726c6421
```
