## encode_hex

`encode_hex` кодує буфер або рядок як шістнадцятковий рядок з префіксом 0x.

### Вхідні дані

| Назва   | Тип                   | Опис                                                                                                                                                   |
| ------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `value` | буфер, рядок, додаток | Буфер або рядок для кодування. Рядки, що починаються з '0x', декодуються як шістнадцяткові; в іншому випадку використовуються необроблені байти UTF-8. |

### Вихідні дані

| Назва   | Тип   | Опис                                                        |
| ------- | ----- | ----------------------------------------------------------- |
| `value` | рядок | Вхідні дані у шістнадцятковому представленні з префіксом 0x |

```hcl
output "encoded_hex" {
    value = encode_hex("hello, world")
}
// > encoded_hex: 0x68656c6c6f2c20776f726c64
```

## decode_hex

`decode_hex` декодує шістнадцятковий рядок і повертає результат у вигляді
буфера.

### Вхідні дані

| Назва        | Тип   | Опис                                                                 |
| ------------ | ----- | -------------------------------------------------------------------- |
| `hex_string` | рядок | Шістнадцятковий рядок для декодування (з префіксом 0x або без нього) |

### Вихідні дані

| Назва   | Тип   | Опис                                               |
| ------- | ----- | -------------------------------------------------- |
| `value` | буфер | Декодований шістнадцятковий рядок у вигляді буфера |

```hcl
output "decoded_hex" {
    value = decode_hex("0x68656c6c6f2c20776f726c64")
}
// > decoded_hex: 0x68656c6c6f2c20776f726c64
```
