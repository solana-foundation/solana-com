## jq

Функція `jq` дозволяє виконувати зрізи, фільтрацію та перетворення даних JSON.
Дивіться [документацію jq](https://jqlang.github.io/jq/manual/) для отримання
додаткової інформації.

### Вхідні дані

| Назва          | Тип              | Опис                                                                        |
| -------------- | ---------------- | --------------------------------------------------------------------------- |
| `decoded_json` | string \| object | Об'єкт JSON                                                                 |
| `query`        | string           | Запит JSON. Дивіться [документацію jq](https://jqlang.github.io/jq/manual/) |

### Вихідні дані

| Назва   | Тип           | Опис                  |
| ------- | ------------- | --------------------- |
| `value` | array[string] | Результат запиту `jq` |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
