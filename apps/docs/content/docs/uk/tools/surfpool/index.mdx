Surfpool — це заміна `solana-test-validator`, створена спеціально для
забезпечення найкращого досвіду розробникам, які будують на **Solana**.

Його основна функція — дозволяти розробникам локально симулювати свої програми,
використовуючи акаунти Mainnet, отримані саме вчасно.

Surfpool також безперешкодно впроваджує Infrastructure as Code у проєкти на
основі Anchor або Pinocchio, забезпечуючи відтворювані, перевіряємі та безпечні
розгортання в будь-якій мережі Solana — приватній чи публічній.
