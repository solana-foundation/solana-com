## Короткий довідник

| Метод                                          | Опис                                                                                                           |
| ---------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| `LiteSVM::new()`                               | Створює нове базове тестове середовище LiteSVM                                                                 |
| `LiteSVM::default()`                           | Повертає «значення за замовчуванням» для типу                                                                  |
| `airdrop(&Address, lamports)`                  | Поповнює рахунок зазначеною кількістю lamport                                                                  |
| `get_account(&Address)`                        | Повертає всю інформацію, пов'язану з обліковим записом за вказаною адресою                                     |
| `set_account(Address, Account)`                | Встановлює всю інформацію, пов'язану з обліковим записом за вказаною адресою                                   |
| `get_balance(&Address)`                        | Отримує баланс у lamport для вказаної адреси облікового запису                                                 |
| `minimum_balance_for_rent_exemption(data_len)` | Повертає мінімальний баланс, необхідний для звільнення від rent облікового запису із зазначеною довжиною даних |
| `add_program(program_id, &bytes)`              | Додає програму SBF до тестового середовища                                                                     |
| `add_program_from_file(program_id, path)`      | Додає програму SBF до тестового середовища з вказаного файлу                                                   |
| `send_transaction(tx)`                         | Надсилає підписану транзакцію                                                                                  |
| `simulate_transaction(tx)`                     | Симулює транзакцію без зміни стану                                                                             |
| `get_transaction(&signature)`                  | Отримує транзакцію з історії                                                                                   |
| `latest_blockhash()`                           | Отримує поточний blockhash                                                                                     |
| `expire_blockhash()`                           | Примусово завершує термін дії поточного blockhash                                                              |
| `warp_to_slot(slot)`                           | Переміщує годинник до вказаного slot                                                                           |
| `get_sysvar<T>()`                              | Отримує sysvar із тестового середовища                                                                         |
| `set_sysvar<T>(&value)`                        | Встановлює sysvar у тестовому середовищі                                                                       |
| `with_feature_accounts()`                      | Додає акаунти feature gate до тестового середовища                                                             |
| `set_invocation_inspect_callback(callback)`    | Реєструє зворотний виклик до/після виклику (потребує функції `invocation-inspect-callback`)                    |
