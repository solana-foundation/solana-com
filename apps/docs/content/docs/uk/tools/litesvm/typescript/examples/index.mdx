## Запуск прикладів

Усі приклади можна запустити за допомогою `tsx`, проте деякі адреси/змінні
потребують заміни заповнювачів на фактичні значення.

```bash
# Install dependencies
pnpm add @solana/kit @solana/kit-plugin-litesvm @solana/kit-plugin-signer @solana-program/system

# Run an example
npx tsx example.ts
```
