## Швидкий довідник

### Налаштування плагіна

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### Керування акаунтами (client.svm)

| Метод                                  | Опис                                     |
| -------------------------------------- | ---------------------------------------- |
| `setAccount(account)`                  | Встановити стан акаунта напряму          |
| `getAccount(address)`                  | Отримати дані акаунта                    |
| `getBalance(address)`                  | Отримати баланс SOL                      |
| `airdrop(address, lamports)`           | Поповнити акаунт                         |
| `minimumBalanceForRentExemption(size)` | Обчислити мінімум, звільнений від оренди |

### Керування програмами (client.svm)

| Метод                                 | Опис                             |
| ------------------------------------- | -------------------------------- |
| `addProgram(programId, bytes)`        | Завантажити програму з байтів    |
| `addProgramFromFile(programId, path)` | Завантажити програму з файлу .so |

### Транзакції (client.svm)

| Метод                     | Опис                        |
| ------------------------- | --------------------------- |
| `sendTransaction(tx)`     | Виконати транзакцію         |
| `simulateTransaction(tx)` | Симулювати без змін стану   |
| `latestBlockhash()`       | Отримати поточний blockhash |
| `expireBlockhash()`       | Перейти до нового blockhash |

### Конфігурація (client.svm)

| Метод                       | Опис                                   |
| --------------------------- | -------------------------------------- |
| `withSigverify(bool)`       | Увімкнути/вимкнути перевірку підпису   |
| `withBlockhashCheck(bool)`  | Увімкнути/вимкнути валідацію blockhash |
| `withSysvars()`             | Увімкнути акаунти sysvar               |
| `withBuiltins()`            | Увімкнути вбудовані програми           |
| `withPrecompiles()`         | Увімкнути прекомпільовані програми     |
| `withTransactionHistory(n)` | Зберігати останні n транзакцій         |
| `withComputeBudget(budget)` | Встановити ліміти обчислень            |
| `withLamports(amount)`      | Встановити lamport за замовчуванням    |
| `withLogBytesLimit(n)`      | Встановити ліміт виводу логів          |

### Час і системні змінні (client.svm)

| Метод                      | Опис                                 |
| -------------------------- | ------------------------------------ |
| `warpToSlot(slot)`         | Перейти до певного slot              |
| `getClock()`               | Отримати системну змінну clock       |
| `getRent()`                | Отримати системну змінну rent        |
| `getEpochSchedule()`       | Отримати розклад epoch               |
| `getLastRestartSlot()`     | Отримати останній slot перезапуску   |
| `setLastRestartSlot(slot)` | Встановити останній slot перезапуску |

### RPC (client.rpc)

| Метод                            | Опис                               |
| -------------------------------- | ---------------------------------- |
| `getAccountInfo(address)`        | Отримати один акаунт (лише base64) |
| `getMultipleAccounts(addresses)` | Отримати кілька акаунтів           |
| `getLatestBlockhash()`           | Отримати поточний blockhash        |
