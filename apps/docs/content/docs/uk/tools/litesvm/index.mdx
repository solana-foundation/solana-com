---
title: LiteSVM
description:
  Блискавично швидке тестування програм Solana за допомогою вбудованої VM.
  Тестуйте свої програми швидше без мережевих затримок або процесів validator.
---

LiteSVM — це легковагова бібліотека для тестування програм Solana, яка працює
шляхом створення вбудованої Solana VM, оптимізованої для розробників програм. На
відміну від інших підходів до тестування, що запускають окремі процеси
validator, LiteSVM вбудовує VM безпосередньо у ваші тести, завдяки чому вони
виконуються значно швидше. Як ще одна відмінність від традиційних рішень,
бібліотека має ергономічний API з розумними налаштуваннями за замовчуванням та
широкими можливостями конфігурації для тих, хто цього потребує.
