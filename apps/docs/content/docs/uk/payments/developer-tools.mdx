### Фреймворки для AI-агентів

AI-агенти можуть автономно виконувати платіжні транзакції в Solana. Ці
фреймворки дозволяють створювати агентів, які обробляють платежі, моніторять
транзакції та автоматизують платіжні процеси:

| Фреймворк        | Опис                                                                                          | Посилання                                                                                         |
| ---------------- | --------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------- |
| Solana Agent Kit | Інструменти для створення агентів із платіжними можливостями, сумісні з LangChain і Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Документація](https://kit.sendai.fun/) |
| Send AI Skills   | Колекція готових навичок для створення агентів на Solana                                      | [GitHub](https://github.com/sendaifun/skills)                                                     |
| Eliza Framework  | Мультиплатформний фреймворк для агентів із плагіном Solana для ботів X, Telegram, Discord     | [GitHub](https://github.com/elizaOS/eliza) • [Документація](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Міжланцюговий інструментарій із платіжними діями Solana                                       | [GitHub](https://github.com/goat-sdk/goat)                                                        |

### Розробка AI-програм
