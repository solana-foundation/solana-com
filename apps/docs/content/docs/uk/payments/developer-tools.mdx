## Інструменти розробки з ШІ

Почніть з нашого посібника про те,
[як розпочати роботу з інструментами ШІ на Solana](/developers/guides/getstarted/intro-to-ai).

### Solana MCP

[Solana MCP](https://mcp.solana.com/) (Machine Conversation Protocol) інтегрує
знання про Solana безпосередньо в інструменти кодування з підтримкою ШІ, такі як
Claude, Cursor та Windsurf. Він навчений на документації Solana, фреймворку
Anchor, прикладах програм та питаннях і відповідях Stack Exchange — корисний для
швидшої розробки платіжних інтеграцій.

### Фреймворки для ШІ-агентів

ШІ-агенти можуть автономно виконувати платіжні транзакції на Solana. Ці
фреймворки дозволяють створювати агентів, які обробляють платежі, відстежують
транзакції та автоматизують платіжні процеси:

| Фреймворк        | Опис                                                                                      | Посилання                                                                                         |
| ---------------- | ----------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------- |
| Solana Agent Kit | Інструменти для створення агентів з можливістю платежів, сумісні з LangChain та Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Документація](https://kit.sendai.fun/) |
| Send AI Skills   | Колекція готових навичок для створення агентів на Solana                                  | [GitHub](https://github.com/sendaifun/skills)                                                     |
| Eliza Framework  | Мультиплатформний фреймворк для агентів з плагіном Solana для ботів X, Telegram, Discord  | [GitHub](https://github.com/elizaOS/eliza) • [Документація](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Міжланцюговий інструментарій з діями для платежів Solana                                  | [GitHub](https://github.com/goat-sdk/goat)                                                        |

### Розробка програм з AI

[Codigo](https://codigo.ai/) — це комплексна платформа для розробки на Solana з
