Окрім стандартних операцій створення, мінтингу та переказу, Token Program також
викликається з ончейн-програм через CPI та включає інструкції для відновлення
lamport:

<Cards>
  <Card
    title="Виклик Token Program через CPI"
    href="/docs/tokens/advanced/cpi"
  >
    Викликайте Token Program з ончейн-програми — переказуйте, мінтіть і спалюйте
    токени, підписуйте за допомогою PDA-авторитету та об'єднуйте інструкції в один CPI.
  </Card>

<Card
  title="Вилучення надлишкових Lamport"
  href="/docs/tokens/advanced/withdraw-excess-lamports"
>
  Відновлюйте SOL вище мінімуму, звільненого від орендної плати, з token
  account, мінтів і мультипідписних акаунтів без зміни балансів токенів.
</Card>

  <Card title="Розгортання Lamport" href="/docs/tokens/advanced/unwrap-lamports">
    Конвертуйте загорнутий SOL назад у lamport — повністю або частково — без закриття
    token account.
  </Card>
</Cards>

<Callout type="info">

_rs`WithdrawExcessLamports`_ та _rs`UnwrapLamports`_ доступні через
[`@solana-program/token`](https://www.npmjs.com/package/@solana-program/token)
клієнт. Виклик Token Program через CPI застосовується до ончейн-програм.

</Callout>

## Відновлення lamport

_rs`WithdrawExcessLamports`_ відновлює SOL вище мінімуму, звільненого від
орендної плати, з token account, мінтів і мультипідписних акаунтів.
_rs`UnwrapLamports`_ конвертує загорнутий SOL у нативному token account назад у
lamport.
