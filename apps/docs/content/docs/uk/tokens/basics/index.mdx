Розділ Основи SPL Token охоплює базові інструкції для взаємодії з SPL-токенами,
зосереджуючись на функціональності, спільній для оригінального Token Program і
Token Extensions Program (Token 2022).

<Callout type="info">
  Перегляньте [документацію
  Metaplex](https://developers.metaplex.com/token-metadata) для отримання
  додаткової інформації про те, як додати метадані до SPL-токенів.
</Callout>
