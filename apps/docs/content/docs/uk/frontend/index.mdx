---

Фронтенд-розробка на Solana передбачає роботу з програмами, гаманцями та
популярними JavaScript-фреймворками, такими як React. Взаємодія з цими
компонентами вимагає обробки з'єднання, створення транзакцій та читання з
акаунтів Solana.

[Solana Kit](https://github.com/anza-xyz/kit) (`@solana/kit`) — це стандартний
JavaScript SDK для цієї роботи. Ви збираєте клієнт із невеликих, компонованих
плагінів, додаєте React-прив'язки за потреби та підключаєте згенеровані
програмні клієнти для онлайн-програм, які викликаєте.

## Основні бібліотеки

<Cards>
  <Card title="@solana/kit" href="https://github.com/anza-xyz/kit">
    - Основний SDK: RPC, кодеки, підписанти та транзакції - Надає `createClient`
    — хост плагінів, який розширює кожна інша бібліотека
  </Card>
  <Card title="Kit plugins" href="https://github.com/anza-xyz/kit-plugins">
    - Складіть клієнт за допомогою `.use(...)`: підключення RPC, гаманець,
    підписант і планування транзакцій - Публікуються як пакети
    `@solana/kit-plugin-*`
  </Card>
  <Card
    title="@solana/react"
    href="https://github.com/anza-xyz/kit/tree/main/packages/react"
  >
    - React-прив'язки для Kit-клієнта - `ClientProvider` плюс хуки для читання,
    підписок та стану гаманця
  </Card>
  <Card title="@solana-program/*" href="https://github.com/solana-program">
    - Згенеровані клієнти для онлайн-програм (System, Token та інші) - Надають
    будівельники інструкцій та плагіни клієнтів, як-от `tokenProgram()`
  </Card>
</Cards>

<Callout type="warn" title="web3.js v1 and wallet-adapter are legacy">
  Багато проєктів Solana досі покладаються на застарілий `@solana/web3.js` (v1)
  та `@solana/wallet-adapter-*`. Для нових проєктів надавайте перевагу
  `@solana/kit` із плагінами Kit нижче; виявлення через Wallet Standard (за
  допомогою `@solana/kit-plugin-wallet`) замінює wallet-adapter для сучасних
  гаманців.
</Callout>

- [Посібник із Kit-клієнта](/docs/frontend/client): побудуйте клієнт із плагінів
  для RPC, підписання та надсилання транзакцій.
- [Посібник із React](/docs/frontend/react-hooks): прив'язки `@solana/react` та
  хуки для гаманця, накладені поверх того самого клієнта.
- [Підручник із Next.js](/docs/frontend/nextjs-solana): повноцінний додаток для
  підключення гаманця та переказу SOL у App Router.
- [Міграція до Kit](/docs/frontend/web3-compat): як концепції web3.js та
  wallet-adapter відповідають поняттям Kit.
