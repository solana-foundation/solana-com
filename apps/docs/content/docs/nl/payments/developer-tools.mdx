## AI-ontwikkeltools

Ga aan de slag met onze gids over
[hoe je aan de slag gaat met AI-tools op Solana](/developers/guides/getstarted/intro-to-ai).

### Solana MCP

[Solana MCP](https://mcp.solana.com/) (Machine Conversation Protocol) integreert
Solana-kennis rechtstreeks in AI-ondersteunde codeertools zoals Claude, Cursor,
en Windsurf. Het is getraind op Solana-documentatie, het Anchor-framework,
programmavoorbeelden, en Stack Exchange Q&A - nuttig voor het sneller bouwen van
betalingsintegraties.

### AI-agentframeworks

AI-agenten kunnen autonoom betalingstransacties uitvoeren op Solana. Deze
frameworks maken het mogelijk om agenten te bouwen die betalingen verwerken,
transacties monitoren, en betalingsworkflows automatiseren:

| Framework        | Beschrijving                                                                               | Links                                                                                     |
| ---------------- | ------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | Tools voor het bouwen van betalingscapabele agenten, compatibel met LangChain en Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | Een verzameling voorgebouwde vaardigheden voor het bouwen van agenten op Solana            | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | Multi-platformagentframework met Solana-plugin voor X, Telegram, Discord-bots              | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Cross-chain-toolkit met Solana-betalingsacties                                             | [GitHub](https://github.com/goat-sdk/goat)                                                |

### AI-programmaontwikkeling

[Codigo](https://codigo.ai/) is een end-to-end Solana AI-ontwikkelplatform.
