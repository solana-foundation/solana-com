De sectie SPL Token Basics behandelt de basisinstructies voor interactie met SPL
Tokens, met focus op functionaliteit die gedeeld wordt door zowel het
oorspronkelijke Token Program als het Token Extensions Program (Token 2022).

<Callout type="info">
  Raadpleeg de [Metaplex
  documentatie](https://developers.metaplex.com/token-metadata) voor meer
  informatie over hoe je metadata kunt toevoegen aan SPL Tokens.
</Callout>
