De sectie SPL Token Basics behandelt de basisinstructies voor het werken met SPL
Tokens, met de nadruk op functionaliteit die gedeeld wordt door zowel het
oorspronkelijke Token Program als het Token Extensions Program (Token 2022).

<Callout type="info">
  Raadpleeg de [Metaplex
  documentatie](https://developers.metaplex.com/token-metadata) voor meer
  informatie over hoe je metadata toevoegt aan SPL Tokens.
</Callout>
