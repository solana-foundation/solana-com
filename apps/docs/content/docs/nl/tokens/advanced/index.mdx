Naast de veelgebruikte bewerkingen voor aanmaken, minten en overdragen, wordt
het Token Program ook aangeroepen vanuit on-chain programma's via CPI en bevat
het instructies voor het herstellen van lamports:

<Cards>
  <Card
    title="Het Token Program aanroepen via CPI"
    href="/docs/tokens/advanced/cpi"
  >
    Roep het Token Program aan vanuit een on-chain programma — draag tokens over, mint en verbrand
    tokens, onderteken met een PDA-autoriteit en combineer instructies in één enkele CPI.
  </Card>

<Card
  title="Overtollige Lamports opnemen"
  href="/docs/tokens/advanced/withdraw-excess-lamports"
>
  Herstel SOL boven het huurvrijgestelde minimum van token accounts, mints en
  multisig-accounts zonder de tokensaldi te wijzigen.
</Card>

  <Card title="Lamports uitpakken" href="/docs/tokens/advanced/unwrap-lamports">
    Converteer gewrapte SOL terug naar lamports, geheel of gedeeltelijk, zonder
    het token account te sluiten.
  </Card>
</Cards>

<Callout type="info">

_rs`WithdrawExcessLamports`_ en _rs`UnwrapLamports`_ zijn beschikbaar via de
[`@solana-program/token`](https://www.npmjs.com/package/@solana-program/token)
client. Het aanroepen van het Token Program via CPI is van toepassing op
on-chain programma's.

</Callout>

## Lamport-herstel

_rs`WithdrawExcessLamports`_ herstelt SOL boven het huurvrijgestelde minimum van
token accounts, mints en multisig-accounts. _rs`UnwrapLamports`_ converteert
gewrapte SOL in een native token account terug naar lamports.
