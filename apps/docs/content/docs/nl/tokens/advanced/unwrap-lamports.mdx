## Wat doet Lamports uitpakken?

_rs`UnwrapLamports`_ verplaatst SOL uit een native (gewrapte SOL) token account
naar een doelaccount als lamports. Je kunt het volledige saldo of een specifiek
bedrag uitpakken, en het token account blijft open.

Dit is flexibeler dan het sluiten van een gewrapt SOL-account: met gedeeltelijk
uitpakken kun je een deel van de SOL opnemen terwijl het account gefinancierd
blijft voor later gebruik.

## Accounts

| Account       | Beschrijving                                                           |
| ------------- | ---------------------------------------------------------------------- |
| `source`      | Schrijfbaar. Het native (gewrapte SOL) token account om uit te pakken. |
| `destination` | Schrijfbaar. Ontvangt de uitgepakte lamports.                          |
| `authority`   | De accounteigenaar of een gedelegeerde (of multisig-ondertekenaars).   |

<Callout type="warn">

De bron moet een native token account zijn; andere token accounts retourneren
_rs`NonNativeNotSupported`_. Een gedelegeerde kan tot zijn gedelegeerde bedrag
uitpakken.

</Callout>

## Lamports uitpakken

Laat _ts`amount`_ weg om het volledige saldo uit te pakken, of stel het in om
een specifiek aantal lamports uit te pakken. Het voorbeeld gebruikt een
geconfigureerde [`@solana/kit`](/docs/clients/official/javascript#solana-kit)
client â€” zie [Tokens overdragen](/docs/tokens/basics/transfer-tokens) voor de
clientinstelling.

```ts title="Unwrap wrapped SOL (Kit)"
import { getUnwrapLamportsInstruction } from "@solana-program/token";

// source: the native (wrapped SOL) token account to unwrap.
// destination: address that receives the unwrapped lamports.
// authority: a TransactionSigner (the account owner or a delegate).

// Unwrap a specific number of lamports.
const unwrapSome = getUnwrapLamportsInstruction({
  source,
  destination,
  authority,
  amount: 500_000_000n
});

// Or omit `amount` to unwrap the entire balance.
const unwrapAll = getUnwrapLamportsInstruction({
  source,
  destination,
  authority
});

// Send whichever instruction you need.
await client.sendTransaction([unwrapSome]);
```

Voor een multisig-autoriteit, geef de mede-ondertekenaars door met
_ts`multiSigners`_.
