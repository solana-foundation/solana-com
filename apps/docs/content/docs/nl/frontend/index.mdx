Frontend-ontwikkeling op Solana omvat het werken met programma's, wallets en
populaire JavaScript-frameworks zoals React. Interactie met deze componenten
vereist het afhandelen van verbindingen, het aanmaken van transacties en het
uitlezen van Solana-accounts.

Om bij dit werk te helpen, zijn er verschillende Solana-clientbibliotheken
beschikbaar in verschillende frameworks.

## Belangrijkste bibliotheken

<Cards>
  <Card
    title="@solana/client"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/client"
  >
    - Eenvoudige Solana-client die RPC, wallets en transacties bundelt - Bevat
    ingebouwde state store, acties, watchers en connectors
  </Card>
  <Card
    title="@solana/react-hooks"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks"
  >
    - Volledige hooks voor wallets, saldi, transfers, handtekeningen en queries
    - React provider hooks die `@solana/client` runtime state omhullen
  </Card>
  <Card
    title="@solana/web3-compat"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/web3-compat"
  >
    - Web3.js-compatibele toolkit om upgraden te vereenvoudigen. - Nieuwere
    internals die vertrouwen op een mix van web3.js en kit.
  </Card>
  <Card
    title="@solana/kit"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/kit"
  >
    - Low-level Solana SDK die de andere Solana-bibliotheken aandrijft zoals
    `@solana/react-hooks` - Volledig tree-shakable, gebruikt moderne
    webstandaarden en drijft de runtime aan
  </Card>
</Cards>

<Callout type="warn" title="@solana/web3.js is verouderd`">
  Veel Solana-ecosysteemprojecten vertrouwen nog steeds op de verouderde
  `@solana/web3.js`. Geef de voorkeur aan `@solana/web3-compat` om je
  migratiepad te vereenvoudigen.
</Callout>

- [@solana/client-gids](/docs/frontend/client): lichte, headless runtime voor
  RPC, wallets en transacties.
- [@solana/react-hooks-gids](/docs/frontend/react-hooks): React hooks gelaagd op
  dezelfde client runtime.
- [@solana/web3-compat-gids](/docs/frontend/web3-compat): compatibiliteitslaag
  om te migreren van `@solana/web3.js` naar Kit-aangedreven stacks.
