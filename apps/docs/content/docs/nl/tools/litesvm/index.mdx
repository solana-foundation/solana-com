---
title: LiteSVM
description:
  Razendsnel Solana-programma's testen met een in-process VM. Test uw
  programma's sneller zonder netwerkoverhead of validator-processen.
---

LiteSVM is een lichtgewicht bibliotheek voor het testen van Solana-programma's
die werkt door een in-process Solana VM te creëren die geoptimaliseerd is voor
programmaontwikkelaars. In tegenstelling tot andere testmethoden die aparte
validator-processen opstarten, integreert LiteSVM de VM direct in uw tests,
waardoor ze veel sneller worden uitgevoerd. Als verdere breuk met traditionele
aanpakken beschikt het over een ergonomische API met verstandige
standaardinstellingen en uitgebreide configureerbaarheid voor diegenen die dat
wensen.
