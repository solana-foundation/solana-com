## Snelreferentie

| Methode                                        | Beschrijving                                                                                                    |
| ---------------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| `LiteSVM::new()`                               | Maak een nieuwe LiteSVM basis testomgeving aan                                                                  |
| `LiteSVM::default()`                           | Geeft de "standaardwaarde" terug voor een type                                                                  |
| `airdrop(&Address, lamports)`                  | Voegt de opgegeven lamports toe aan het account                                                                 |
| `get_account(&Address)`                        | Geeft alle informatie terug die is gekoppeld aan het account van het opgegeven adres                            |
| `set_account(Address, Account)`                | Stelt alle informatie in die is gekoppeld aan het account van het opgegeven adres                               |
| `get_balance(&Address)`                        | Geeft het lamport-saldo van het opgegeven accountadres terug                                                    |
| `minimum_balance_for_rent_exemption(data_len)` | Geeft het minimale saldo terug dat vereist is om een account met de opgegeven gegevenslengte rent-vrij te maken |
| `add_program(program_id, &bytes)`              | Voegt een SBF-programma toe aan de testomgeving                                                                 |
| `add_program_from_file(program_id, path)`      | Voegt een SBF-programma toe aan de testomgeving vanuit het opgegeven bestand                                    |
| `send_transaction(tx)`                         | Verzendt een ondertekende transactie                                                                            |
| `simulate_transaction(tx)`                     | Simuleert een transactie zonder de toestand te wijzigen                                                         |
| `get_transaction(&signature)`                  | Haalt een transactie op uit de geschiedenis                                                                     |
| `latest_blockhash()`                           | Haalt de huidige blockhash op                                                                                   |
| `expire_blockhash()`                           | Dwingt het verlopen van de blockhash af voor de huidige blockhash                                               |
| `warp_to_slot(slot)`                           | Verplaatst de klok naar de opgegeven slot                                                                       |
| `get_sysvar<T>()`                              | Haalt een sysvar op uit de testomgeving                                                                         |
| `set_sysvar<T>(&value)`                        | Stelt de sysvar in voor de testomgeving                                                                         |
| `with_feature_accounts()`                      | Voegt feature gate-accounts toe aan de testomgeving                                                             |
| `set_invocation_inspect_callback(callback)`    | Registreert een pre/post-aanroep callback (vereist de functie `invocation-inspect-callback`)                    |
