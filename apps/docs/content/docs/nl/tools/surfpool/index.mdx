Surfpool is een drop-in vervanging voor `solana-test-validator`, speciaal
gebouwd om de best mogelijke ervaring te bieden aan ontwikkelaars die bouwen op
**Solana**.

De kernfunctie is dat ontwikkelaars hun programma's lokaal kunnen simuleren met
Mainnet-accounts die just-in-time worden opgehaald.

Surfpool introduceert ook naadloos Infrastructure as Code in Anchor- of
Pinocchio-gebaseerde projecten, waardoor reproduceerbare, auditeerbare en
veilige deployments naar elk Solana-netwerk mogelijk worden—privaat of publiek.
