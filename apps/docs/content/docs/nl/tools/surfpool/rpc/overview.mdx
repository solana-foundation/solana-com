Surfnet is de lokale Solana-netwerksimulator van Surfpool, ontworpen om
ontwikkelaars een krachtige testomgeving te bieden die het gedrag van de
Solana-blockchain nabootst zonder de overhead van een volledige validator-node.

## Aan de slag

Start een Surfnet-instantie met de Surfpool CLI:

```bash
surfpool start
```

Dit start een lokale Surfnet-instantie op `http://localhost:8899` met alle RPC-
eindpunten beschikbaar.

## Belangrijkste functies

- **Volledige RPC-compatibiliteit**: Ondersteunt de volledige Solana RPC API
- **Snelle iteratie**: Geen wachttijd voor blokbevestigingen
- **Statusmanipulatie**: Cheatcodes voor het rechtstreeks wijzigen van de
  blockchain-status
- **Mainnet Fork**: Fork van het mainnet om te testen met echte accounts en
  programma's
- **Geyser Plugin-ondersteuning**: Verbind je Geyser-plugins voor realtime
  datastreaming
