De `surfpool-sdk` crate biedt een beperkt oppervlak gecentreerd rondom de
`Surfnet` struct. Deze pagina geeft een overzicht van elk publiek item; voor
uitleg over het gebruik, zie de handleidingen beginnen met
[Overzicht](/docs/tools/surfpool/sdk/overview).

## Crate Root Exports

| Item                  | Soort      | Opmerkingen                                                         |
| --------------------- | ---------- | ------------------------------------------------------------------- |
| `Surfnet`             | struct     | Actieve Surfnet-instantie — RPC/WS-eindpunten, betaler, cheatcodes  |
| `SurfnetBuilder`      | struct     | Fluent builder gebruikt door `Surfnet::builder()`                   |
| `Cheatcodes<'_>`      | struct     | Geleende cheatcode-helper teruggegeven door `Surfnet::cheatcodes()` |
| `SurfnetError`        | enum       | Alle foutvarianten die de SDK retourneert                           |
| `SurfnetResult<T>`    | type alias | `Result<T, SurfnetError>`                                           |
| `Keypair`             | re-export  | `solana_keypair::Keypair`                                           |
| `Pubkey`              | re-export  | `solana_pubkey::Pubkey`                                             |
| `Signer`              | re-export  | `solana_signer::Signer`                                             |

### Levenscyclus

```rust
impl Surfnet {
    pub fn stop(&mut self) -> SurfnetResult<()>;
}

impl Drop for Surfnet { /* calls stop() if not already stopped */ }
```

`stop` wacht maximaal vijf seconden totdat de runtime zijn poorten vrijgeeft.
Een fout tijdens `Drop` is stil; een fout tijdens een expliciete `stop` geeft
een `SurfnetError::Runtime` terug.

## SurfnetError

```rust
#[derive(Debug)]
pub enum SurfnetError {
    PortAllocation(String),
    Startup(String),
    Runtime(String),
    Cheatcode(String),
    Aborted(String),
}
```

| Variant          | Oorzaak                                                          |
| ---------------- | ---------------------------------------------------------------- |
| `PortAllocation` | Kon niet binden aan een vrije poort voor RPC of WebSocket.       |
| `Startup`        | De runtime kon niet worden geïnitialiseerd.                      |
| `Runtime`        | De runtime-thread is gecrasht of onverwacht beëindigd.           |
| `Cheatcode`      | Een cheatcode JSON-RPC-aanroep heeft een fout geretourneerd.     |
| `Aborted`        | De runtime werd afgesloten of geannuleerd tijdens het opstarten. |

De crate implementeert `std::error::Error` en `Display` op `SurfnetError`, zodat
het direct werkt met `?` en `anyhow`.
