## encode_base58

`encode_base58` codeert een buffer of string als een base58-string.

### Invoer

| Name    | Type                  | Description                                                                                                                                             |
| ------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | De buffer of string die gecodeerd moet worden. Strings die beginnen met '0x' worden gedecodeerd als hex; anders worden onbewerkte UTF-8-bytes gebruikt. |

### Uitvoer

| Name    | Type   | Description                                |
| ------- | ------ | ------------------------------------------ |
| `value` | string | De invoer, gecodeerd als een base58-string |

```hcl
output "encoded" {
    value = encode_base58("0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8")
}
// > encoded: CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF
```

## decode_base58

`decode_base58` decodeert een base58-gecodeerde string en retourneert het
resultaat als een buffer.

### Invoer

| Name            | Type   | Description                      |
| --------------- | ------ | -------------------------------- |
| `base58_string` | string | De base58-string om te decoderen |

### Uitvoer

| Name    | Type   | Description                                   |
| ------- | ------ | --------------------------------------------- |
| `value` | buffer | De gedecodeerde base58-string, als een buffer |

```hcl
output "decoded" {
    value = decode_base58("CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF")
}
// > decoded: 0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8
```
