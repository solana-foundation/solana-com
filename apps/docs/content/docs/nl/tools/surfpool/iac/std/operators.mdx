## and_bool

`and_bool` geeft de binaire AND terug van de linker- en rechterargumenten.

### Invoer

| Naam  | Type | Beschrijving          |
| ----- | ---- | --------------------- |
| `lhs` | bool | Een booleaanse waarde |
| `rhs` | bool | Een booleaanse waarde |

### Uitvoer

| Naam    | Type | Beschrijving                                                |
| ------- | ---- | ----------------------------------------------------------- |
| `value` | bool | Het resultaat van een binaire AND tussen de twee argumenten |

```hcl
output "my_bool" {
  value = false && true
}
// > my_bool: false
```

## div

`div` geeft de gehele deling terug van het linkerargument door het
rechterargument, waarbij de rest naar beneden wordt afgerond op het
dichtstbijzijnde gehele getal.

### Invoer

| Naam  | Type    | Beschrijving |
| ----- | ------- | ------------ |
| `lhs` | integer | Het deeltal  |
| `rhs` | integer | De deler     |

### Uitvoer

| Naam    | Type    | Beschrijving                                              |
| ------- | ------- | --------------------------------------------------------- |
| `value` | integer | Het resultaat van het delen van het deeltal door de deler |

```hcl
output "my_int" {
  value = 11 / -3
}
// > my_int: -3
```

## eq

`eq` geeft `true` terug als de linker- en rechterargumenten gelijk zijn en
`false` als ze niet gelijk zijn.

### Invoer

| Naam  | Type | Beschrijving |
| ----- | ---- | ------------ |
| `lhs` | any  | Elke waarde  |
| `rhs` | any  | Elke waarde  |

### Uitvoer

| Naam    | Type | Beschrijving                                                           |
| ------- | ---- | ---------------------------------------------------------------------- |
| `value` | bool | Het resultaat van een gelijkheidscontrole tussen de twee invoerwaarden |

```hcl
output "is_eq" {
  value = "arg" == "arg"
}
// > is_eq: true
```

## neq

`neq` geeft `true` terug als het linker en rechter argument niet gelijk zijn en
`false` anders.

### Invoer

| Naam  | Type | Beschrijving                       |
| ----- | ---- | ---------------------------------- |
| `lhs` | any  | Een waarde om te vergelijken       |
| `rhs` | any  | Een waarde om tegen te vergelijken |

### Uitvoer

| Naam    | Type | Beschrijving                                                                      |
| ------- | ---- | --------------------------------------------------------------------------------- |
| `value` | bool | Het resultaat van een genegeerde gelijkheidscontrole tussen de twee invoerwaarden |

```hcl
output "is_neq" {
  value = "arg" != "arg"
}
// > is_neq: false
```
