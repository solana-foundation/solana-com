## index

`index` haalt het item op uit een lijst op de opgegeven index.

### Invoer

| Name    | Type                                             | Description                               |
| ------- | ------------------------------------------------ | ----------------------------------------- |
| `list`  | array[string] \| array[integer] \| array[buffer] | De lijst waaruit een item wordt opgehaald |
| `index` | integer                                          | De index van het op te halen item         |

### Uitvoer

| Name    | Type   | Description                                 |
| ------- | ------ | ------------------------------------------- |
| `value` | string | Het item uit de lijst op de opgegeven index |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
