---
title: SVM Addon Overzicht
description: Add-on voor Solana en SVM-compatibele blockchains (beta)
---

De SVM `txtx` plugin maakt het mogelijk om Runbooks te bouwen die communiceren
met Solana en SVM-compatibele blockchains. De plugin biedt hulpfuncties waarmee
je Anchor-programma's kunt deployen en instructieaanroepen kunt encoderen
volgens programma-IDL's. De acties kunnen worden gebruikt om geldige
overdrachts-, programmaoproep- en programma-deploytransacties te maken die
kunnen worden ondertekend via een mnemonische zin, geheime sleutel, of via je
browsersigner.

## Snel starten

Om de SVM-addon te gebruiken, declareer je deze in je Runbook:

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## Functies

- **Programma-deployment**: Deploy Anchor en native Solana-programma's
- **Instructieopbouw**: Encodeer instructieaanroepen met behulp van
  programma-IDL's
- **Transactieondertekening**: Ondersteuning voor mnemonics, geheime sleutels en
  webwallets
- **Accountbeheer**: Bevraag en beheer Solana-accounts
