## jq

De functie `jq` maakt het mogelijk om JSON-gegevens te segmenteren, filteren en
transformeren. Zie de [jq-documentatie](https://jqlang.github.io/jq/manual/)
voor meer informatie.

### Invoer

| Naam           | Type             | Beschrijving                                                                  |
| -------------- | ---------------- | ----------------------------------------------------------------------------- |
| `decoded_json` | string \| object | Een JSON-object                                                               |
| `query`        | string           | Een JSON-query. Zie de [jq-documentatie](https://jqlang.github.io/jq/manual/) |

### Uitvoer

| Naam    | Type          | Beschrijving                    |
| ------- | ------------- | ------------------------------- |
| `value` | array[string] | Het resultaat van de `jq`-query |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
