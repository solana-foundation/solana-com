## encode_base64

`encode_base64` codeert een buffer of string als een base64-string.

### Invoer

| Naam    | Type                  | Beschrijving                                                                                                                          |
| ------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | De buffer of string om te coderen. Strings die beginnen met '0x' worden gedecodeerd als hex; anders worden ruwe UTF-8-bytes gebruikt. |

### Uitvoer

| Naam    | Type   | Beschrijving                               |
| ------- | ------ | ------------------------------------------ |
| `value` | string | De invoer, gecodeerd als een base64-string |

```hcl
output "encoded" {
    value = encode_base64("0x48656c6c6f20776f726c6421")
}
// > encoded: SGVsbG8gd29ybGQh
```

## decode_base64

`decode_base64` decodeert een base64-gecodeerde string en retourneert het
resultaat als een buffer.

### Invoer

| Naam            | Type   | Beschrijving                  |
| --------------- | ------ | ----------------------------- |
| `base64_string` | string | De te decoderen base64-string |

### Uitvoer

| Naam    | Type   | Beschrijving                              |
| ------- | ------ | ----------------------------------------- |
| `value` | buffer | De gedecodeerde base64-string, als buffer |

```hcl
output "decoded" {
    value = decode_base64("SGVsbG8gd29ybGQh")
}
// > decoded: 0x48656c6c6f20776f726c6421
```
