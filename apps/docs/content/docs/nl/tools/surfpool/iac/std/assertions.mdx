## assert_eq

`assert_eq` controleert of twee waarden gelijk zijn.

### Invoer

| Naam    | Type | Beschrijving                    |
| ------- | ---- | ------------------------------- |
| `left`  | any  | Een waarde om te vergelijken    |
| `right` | any  | De waarde om mee te vergelijken |

### Uitvoer

| Naam    | Type                  | Beschrijving                  |
| ------- | --------------------- | ----------------------------- |
| `value` | addon(std::assertion) | Het resultaat van de assertie |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```

| `right` | integer \| float | Een geheel getal of float om mee te vergelijken |

