## ripemd160

`ripemd160` berekent de Ripemd160-hash van een waarde.

### Invoer

| Naam    | Type                    | Beschrijving                          |
| ------- | ----------------------- | ------------------------------------- |
| `value` | buffer \| array[buffer] | De hex-gecodeerde waarde om te hashen |

### Uitvoer

| Naam    | Type   | Beschrijving           |
| ------- | ------ | ---------------------- |
| `value` | string | Het gehashte resultaat |

```hcl
output "hashed_data" {
    value = ripemd160(encode_hex("hello, world"))
}
// > hashed_data: 0XA3201F82FCA034E46D10CD7B27E174976E241DA2
```

### Invoer

### Uitvoer

| Naam    | Type   | Beschrijving           |
| ------- | ------ | ---------------------- |
| `value` | string | Het gehashte resultaat |

```hcl
output "hashed_data" {
    value = sha256(encode_hex("hello, world"))
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```
