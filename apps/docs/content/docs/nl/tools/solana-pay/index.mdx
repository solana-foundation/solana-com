**Solana Pay is een standaardprotocol en set van referentie-implementaties
waarmee ontwikkelaars gedecentraliseerde betalingen kunnen integreren in hun
apps en diensten.** De Solana-blockchain bevestigt transacties in minder dan een
seconde en kost gemiddeld $0,0005, wat gebruikers een naadloze ervaring biedt
zonder tussenpersonen.

## Waarom Solana Pay?

- **Standaardprotocol**: Uniforme aanpak voor betalingen in alle Solana-wallets
  en apps
- **Ultrasnel**: Transacties worden in minder dan een seconde bevestigd
- **Ultragoedkoop**: Gemiddelde transactiekosten van $0,0005
- **Geen tussenpersonen**: Directe peer-to-peer-betalingen zonder derde partijen
- **Universele walletondersteuning**: Compatibel met Phantom, Solflare, Backpack
  en meer

## Snel aan de slag

De eenvoudigste manier om te beginnen is met een transfer request-URL:

```
solana:recipient?amount=0.01&label=Coffee%20Shop&message=Grande%20Latte
```

Deze URL kan:

- Als QR-code worden gecodeerd voor het scannen met mobiele wallets
- Als betalingslink worden gebruikt in webapplicaties
- Als deep links in mobiele apps worden ingesloten

## Installatie

Installeer de Solana Pay JavaScript SDK:

```bash
# Using pnpm (recommended)
pnpm add @solana/pay@beta @solana/kit

# Using npm
npm install @solana/pay@beta @solana/kit

# Using yarn
yarn add @solana/pay@beta @solana/kit
```

## Gebruikssituaties

Solana Pay maakt een breed scala aan betalingsscenario's mogelijk:

- **Retailbetalingen**: Persoonlijke betalingen met QR-codes
- **E-commerce**: Webcheckout-flows met walletintegratie
- **Mobiele apps**: Native betalingservaringen met deep linking
- **NFT-minting**: Dynamische transactiecreatie voor complexe interacties
- **Abonnementen**: Terugkerende betalingen en loyaliteitsprogramma's
- **DeFi-integratie**: Samenstelbare transacties met swaps en leningen
