**Commerce Kit is een verzameling tools voor het bouwen van commerce-ervaringen
op Solana.** Van plug-and-play betalingsknoppen tot headless primitieven,
Commerce Kit biedt alles wat je nodig hebt om cryptobetalingen te accepteren.

## Waarom Commerce Kit?

- **Plug-and-Play Componenten**: Voorgebouwde PaymentButton met
  wallet-verbinding, token- selectie en transactieverwerking
- **Framework Agnostisch**: Gebruik headless primitieven in elk
  JavaScript-framework
- **Solana Pay**: Volledige Solana Pay-ondersteuning met QR-code generatie
- **Type-Safe**: Volledige TypeScript-ondersteuning in alle packages
- **Wallet Standard**: Compatibel met alle Wallet Standard wallets (Phantom,
  Solflare, Backpack, etc.)

## Snel Beginnen

De snelste manier om te beginnen is met de `PaymentButton` component:

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## Installatie

Kies de Commerce Kit-packages die je nodig hebt:

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
