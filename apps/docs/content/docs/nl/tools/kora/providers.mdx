Wil je geen eigen Kora node draaien? De volgende providers bieden gehoste
Kora-infrastructuur aan, zodat je fee abstraction aan je app kunt toevoegen
zonder je eigen deployment te hoeven beheren.

## Providers

### Openfort

[Openfort](https://www.openfort.io/) biedt Solana paymaster-infrastructuur
aangedreven door Kora, waarmee transacties zonder gaskosten mogelijk zijn via
hun embedded wallet-platform. Hun integratie maakt gebruik van een co-signing
patroon waarbij Kora als fee payer fungeert terwijl gebruikers ondertekenen via
de embedded wallets van Openfort.

- **Website**: [openfort.io](https://www.openfort.io/)
- **Docs**:
  [Solana Gasless Transactions](https://www.openfort.io/blog/solana-gasless-transactions)
- **Functies**: Embedded wallets, transacties zonder gaskosten, fee sponsorship

## Word een Kora Node Operator

Kora is open source en iedereen kan een node draaien. Of je nu een app bouwt die
fee abstraction nodig heeft of paymaster-diensten aan anderen wilt aanbieden,
bekijk de [Operators Guide](/docs/tools/kora/operators) om te beginnen.
