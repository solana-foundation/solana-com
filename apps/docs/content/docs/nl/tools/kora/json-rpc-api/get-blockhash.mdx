---
title: getBlockhash
description:
  Haalt de nieuwste blockhash op van de Solana RPC waarmee de Kora-server is
  verbonden.
---

## JSON-RPC-verzoek

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getBlockhash",
  "params": []
}
```

## JSON-RPC-antwoord

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "blockhash": "base58Blockhash"
  }
}
```

## cURL-voorbeeld

```bash
curl -X POST http://localhost:8080 \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"getBlockhash","params":[]}'
```

## TypeScript SDK

```typescript
const { blockhash } = await client.getBlockhash();
console.log("Current blockhash:", blockhash);
```
