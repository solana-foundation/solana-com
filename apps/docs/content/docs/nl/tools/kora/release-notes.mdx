## Kora v2.2.0-beta.6

_23-03-2026_ |
[GitHub Release](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.6)
| `cargo install kora-cli@2.2.0-beta.6`

Pre-release met Jito-bundel ondersteuning, nieuwe RPC-methoden en verbeterde
beveiligingsfuncties.

- Jito-bundel ondersteuning toegevoegd met `signBundle`, `signAndSendBundle`, en
  `estimateBundleFee` methoden
- `getVersion` RPC-methode toegevoegd
- Lighthouse fee payer bescherming toegevoegd
- reCAPTCHA v3 botbescherming toegevoegd voor ondertekenings-endpoints
- Gebruikslimieten per wallet toegevoegd met Redis-ondersteuning
- Officiële Docker-images toegevoegd op GHCR
- Gedetailleerde gebruikslimietregels toegevoegd
- SDK: `KoraClient` accepteert nu een opties-object; nieuwe `koraPlugin` voor
  `@solana/kit`

Zie [Bètadocumentatie](/docs/tools/kora/beta) voor details.

## Kora TypeScript SDK v0.2.0

_09-03-2026_ |
[GitHub Release](https://github.com/solana-foundation/kora/releases/tag/ts-sdk-v0.2.0)
| `pnpm add @solana/kora@0.2.0`

Kit client plugin-architectuur voor composeerbare gasloze transacties.

- `createKitKoraClient()` toegevoegd met Kit plugin-compositie (planner-,
  executor-, payer-plugins)
- Maakt composeerbaarheid mogelijk met Kit-programma-plugins zoals
  `tokenProgram()`
- Token-2022 ondersteuning toegevoegd via `tokenProgramId` configuratieoptie
- Ondersteuning voor compute budget instructies toegevoegd met op simulatie
  gebaseerde CU-schatting
- Ontbrekend `signature` veld toegevoegd aan `signAndSendTransaction` respons
- Retourneert `NoopSigner` van `getPaymentInstruction` voor Kit v6 signer
  deduplicatie
- `@solana/kit` peer dependency bijgewerkt naar `^6.1.0`

Zie [Kit Client Gids](/docs/tools/kora/guides/kit-client).

## Kora v2.0.0

_19-11-2025_ |
[GitHub Release](https://github.com/solana-foundation/kora/releases/tag/v2.0.0)
| `cargo install kora-cli@2.0.0`

Grote release met nieuwe architectuur.

- Nieuwe architectuur geïntroduceerd voor de Kora-node
- TypeScript SDK `@solana/kora` v0.1.0 gepubliceerd
- Gemigreerd naar Solana SDK v3.0.x en alle afhankelijkheden bijgewerkt
- Aangepaste Kora Signer vervangen door solana-signers crate
- `sign_transaction_if_paid` en `get_all_signers` methoden verwijderd
- Responstypen gewijzigd: handtekeningen verwijderd uit `signAndSendTransaction`
  en `signTransaction`
- Verbeterde ontwikkelaarservaring, beveiliging en prestaties
