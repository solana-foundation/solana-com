## Cargo

Installeer de bètaversie rechtstreeks vanaf crates.io:

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

Officiële multi-platform Docker-images (amd64/arm64) worden gepubliceerd naar
GitHub Container Registry:

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

Beschikbare tags:

- `latest` — Nieuwste stabiele versie
- `beta` — Nieuwste pre-release versie
- `v<version>` — Specifieke versie (bijv. `v2.2.0-beta.7`)

### Uitvoeren met Docker

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## TypeScript SDK

De bèta-SDK is beschikbaar op npm:

```bash
pnpm add @solana/kora@beta
```

Zie [SDK-wijzigingen](/docs/tools/kora/beta/sdk-changes) voor nieuwe API's.

## Installatie verifiëren

```bash
kora --version
```
