_Beschikbaar op GitHub:
[`v2.2.0-beta.7`](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.7)_

Kora v2.2.0-beta introduceert ondersteuning voor Jito-bundles, nieuwe
RPC-methoden, Docker-images, verbeterde beveiligingsfuncties en
SDK-verbeteringen.

> **Dit is een pre-release.** De hier gedocumenteerde functies zijn beschikbaar
> in `v2.2.0-beta.7` en kunnen wijzigen vóór de stabiele release. Voor stabiele
> documentatie, zie de [hoofddocumentatie van Kora](/docs/tools/kora).

## Hoogtepunten

### Ondersteuning voor Jito-bundles

Verzend maximaal 5 transacties atomair via de block engine van Jito. Nieuwe
RPC-methoden:

- [`signBundle`](/docs/tools/kora/beta/bundle-methods#signbundle) — Onderteken
  een bundle zonder deze uit te zenden
- [`signAndSendBundle`](/docs/tools/kora/beta/bundle-methods#signandsendbundle)
  — Onderteken en verzend naar Jito
- [`estimateBundleFee`](/docs/tools/kora/beta/bundle-methods#estimatebundlefee)
  — Schat totale bundlekosten in

Zie de [Jito Bundle-gids](/docs/tools/kora/beta/jito-guide) voor een volledige
uitleg.

### Docker-images

Officiële multiplatform-images (amd64/arm64) gepubliceerd naar GitHub Container
Registry. Zie [Installatie](/docs/tools/kora/beta/installation).

### Transactieplugins

Een nieuw pluginsysteem stelt Kora in staat om transactievormen te valideren
tijdens ondertekeningsflows. De eerste ingebouwde plugin, `gas_swap`, handhaaft
strikte regels voor gasloze token-voor-SOL swap-transacties. Plugins worden
geconfigureerd in `kora.toml` en draaien automatisch tijdens `signTransaction`,
`signAndSendTransaction`, `signBundle`, en `signAndSendBundle`.

Zie
[Configuratie — Transactieplugins](/docs/tools/kora/beta/configuration#transaction-plugins).

### Nieuwe configuratieopties

- **Transactieplugins** voor validatie van transactievormen (`gas_swap`)
- **reCAPTCHA v3** botbescherming voor ondertekeningsendpoints
- **Lighthouse** bescherming tegen drainage van fee payer
- **Gebruikslimieten** transactielimitering per portemonnee
- **Bundleconfiguratie** voor Jito-integratie

Zie [Configuratie](/docs/tools/kora/beta/configuration).

### SDK-wijzigingen

- Ondersteuning voor bundlemethoden (`signBundle`, `signAndSendBundle`,
  `estimateBundleFee`)

Zie [SDK-wijzigingen](/docs/tools/kora/beta/sdk-changes).

### Aanvullende RPC-methoden

## Installeren

```bash
cargo install kora-cli@2.2.0-beta.7
```

Of via Docker:

```bash
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```
