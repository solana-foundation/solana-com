Ga aan de slag met Solana Attestations door de benodigde pakketten te
installeren en jouw ontwikkelomgeving in te stellen.

## TypeScript/JavaScript

Installeer de Solana Attestations-bibliotheek met jouw voorkeurspakketbeheerder:

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
