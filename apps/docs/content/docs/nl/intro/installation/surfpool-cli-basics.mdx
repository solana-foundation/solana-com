Surfpool is een lokaal ontwikkelingstool dat Solana-ontwikkeling vereenvoudigt
door automatisch programma's en accounts te laden van een cluster (mainnet,
devnet). Gebruik de Surfpool CLI om een lokale validator te starten voor het
testen van je programma's en transacties.

Bezoek de [Surfpool documentatie](https://docs.surfpool.run/) voor meer
informatie.

## Start een lokale validator

Om een lokale validator te starten, voer het volgende commando uit:

```terminal
$ surfpool start
```

Standaard laadt Surfpool automatisch alle programma's en accounts waar je
transacties van afhankelijk zijn van mainnet naar de lokale validator.

## Surfpool Studio

Surfpool Studio biedt een webgebaseerde interface voor interactie met je lokale
validator. Gebruik het om:

- Transactiedetails te bekijken
- Accountgegevens te inspecteren
- SOL en andere tokens te airdroppen via de UI-faucet

Om toegang te krijgen tot Surfpool Studio:

1. Start je lokale validator met `surfpool start`
2. Open je browser en navigeer naar
   [http://127.0.0.1:18488](http://127.0.0.1:18488)
