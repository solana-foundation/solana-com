## Solana Client

`@solana/client` is de headless Solana runtime die RPC, wallets, transacties en
subscriptions bundelt in één store.

| Package        | Beschrijving            | GitHub                                                                               |
| -------------- | ----------------------- | ------------------------------------------------------------------------------------ |
| @solana/client | Headless client runtime | [Bron](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [Handleiding: @solana/client overzicht](/docs/frontend/client)

## Solana React Hooks

`@solana/react-hooks` voegt een React provider en hooks toe bovenop
`@solana/client`, waarbij dezelfde runtime en cache worden hergebruikt.

| Package             | Beschrijving                    | GitHub                                                                                    |
| ------------------- | ------------------------------- | ----------------------------------------------------------------------------------------- |
| @solana/react-hooks | React provider + hooks voor UIs | [Bron](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [Handleiding: @solana/react-hooks](/docs/frontend/react-hooks)

`@solana/kit` is de aanbevolen TypeScript SDK voor bouwen op Solana.

| Pakket                         | Beschrijving                            | GitHub                                                                        |
| ------------------------------ | --------------------------------------- | ----------------------------------------------------------------------------- |
| @solana/kit                    | Kern-SDK                                | [Bron](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | Interactie met System Program           | [Bron](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Interactie met Token Program            | [Bron](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Interactie met Token-2022-programma     | [Bron](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Interactie met Memo-programma           | [Bron](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Interactie met Compute Budget-programma | [Bron](https://github.com/solana-program/compute-budget/tree/main/clients/js) |
| @solana/kit-plugins            | Plugins voor Kit                        | [Bron](https://github.com/anza-xyz/kit-plugins)                               |
