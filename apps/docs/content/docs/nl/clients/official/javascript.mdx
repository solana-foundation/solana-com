## Solana Kit

`@solana/kit` is de aanbevolen TypeScript SDK voor het bouwen op Solana. Je
stelt een client samen uit composeerbare plugins, voegt React-bindingen toe
wanneer je ze nodig hebt, en integreert gegenereerde programmaclients voor de
on-chain programma's die je aanroept.

| Package                   | Beschrijving                                | GitHub                                                                               |
| ------------------------- | ------------------------------------------- | ------------------------------------------------------------------------------------ |
| @solana/kit               | Kern-SDK: RPC, transacties, codecs, signers | [Bron](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | React-bindingen: `ClientProvider` en hooks  | [Bron](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | RPC-verbinding + transacties verzenden      | [Bron](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | Fee payer, identiteit en signer-plugins     | [Bron](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Wallet Standard-verbinding + React-hooks    | [Bron](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

## Solana Web3.js

`@solana/web3.js` is de verouderde TypeScript SDK voor Solana. Nieuwe apps
moeten `@solana/kit` gebruiken; zie
[Migreren naar Kit](/docs/frontend/web3-compat).
