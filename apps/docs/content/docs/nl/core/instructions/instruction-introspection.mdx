## Instructions sysvar

Programma's kunnen alle top-level instructies in de huidige transactie
inspecteren via de **Instructions sysvar**
(`Sysvar1nstructions1111111111111111111111111`). Hiermee kan een programma de
andere instructies in dezelfde transactie inspecteren, bijvoorbeeld om te
verifiëren dat een specifieke instructie zich in dezelfde transactie bevindt als
de instructie die het uitvoert.

<Callout type="info">
  De Instructions sysvar bevat alleen top-level instructies uit het
  transactiebericht. Inner instructions die via CPI worden aangeroepen, zijn
  niet toegankelijk via deze sysvar.
</Callout>

De instructions sysvar wordt niet benaderd via de normale _rs`Sysvar`_ trait. In
plaats daarvan wordt deze benaderd via vrije functies in de
[`solana_instructions_sysvar`](https://github.com/anza-xyz/solana-sdk/blob/clock%40v2.2.3/instructions-sysvar/src/lib.rs#L1)
crate.

De sysvar-data is geserialiseerd met een aangepaste binaire indeling:

| Offset          | Grootte  | Beschrijving                                  |
| --------------- | -------- | --------------------------------------------- |
| 0               | 2        | `num_instructions` (u16, little-endian)       |
| 2               | 2 \* N   | Byte-offsets voor elke instructie (u16 elk)   |
| varieert        | varieert | Geserialiseerde instructiedata                |
| laatste 2 bytes | 2        | Huidige instructie-index (u16, little-endian) |

Elke geserialiseerde instructie bevat: het aantal accounts (u16), de accounts
als 33-byte entries (1 flag-byte + 32-byte pubkey), de program ID (32 bytes), de
datalengte (u16) en de ruwe databytes.

Belangrijkste functies:

- [`load_current_index_checked(account_info)`](https://github.com/anza-xyz/solana-sdk/blob/clock%40v2.2.3/instructions-sysvar/src/lib.rs#L151):
  Retourneert de index van de momenteel uitgevoerde instructie. De runtime
  [werkt](https://github.com/anza-xyz/agave/blob/v3.1.8/transaction-context/src/lib.rs#L356-L359)
  deze waarde bij telkens wanneer een nieuwe top-level instructie begint.
- [`load_instruction_at_checked(index, account_info)`](https://github.com/anza-xyz/solana-sdk/blob/clock%40v2.2.3/instructions-sysvar/src/lib.rs#L246):
  Deserialiseert de instructie op de gegeven absolute index.
- [`get_instruction_relative(offset, account_info)`](https://github.com/anza-xyz/solana-sdk/blob/clock%40v2.2.3/instructions-sysvar/src/lib.rs#L267):
  Laadt een instructie relatief ten opzichte van de huidige (bijv. `-1` voor de
  vorige instructie, `1` voor de volgende).
