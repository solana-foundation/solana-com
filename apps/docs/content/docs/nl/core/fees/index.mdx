Elke Solana-[transactie](/docs/core/transactions) vereist een vergoeding betaald
in SOL. De vergoeding bestaat uit twee componenten: een **basisvergoeding** en
een optionele **prioriteringsvergoeding**. De basisvergoeding compenseert
validators voor het cryptografische werk van het verifiëren van handtekeningen.
De prioriteringsvergoeding vergroot de kans dat de huidige leider jouw
transactie eerder inplant dan concurrerende transacties.

## Belangrijkste feiten

- **Basisvergoeding**: per handtekening, 50% verbrand / 50% naar de validator.
- **Prioriteringsvergoeding**:
  `ceil(compute_unit_price * compute_unit_limit / 1,000,000)` lamports. 100%
  naar de validator.
