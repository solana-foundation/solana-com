Deze pagina is een enkele referentie voor alle runtime-constanten die worden
genoemd in de core conceptpagina's. Elke constante linkt naar de brondefinitie
in de [agave](https://github.com/anza-xyz/agave/tree/v3.1.8) validator client of
[solana-sdk](https://github.com/anza-xyz/solana-sdk).
