## Référence rapide

### Configuration du plugin

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### Gestion des comptes (client.svm)

| Méthode                                | Description                          |
| -------------------------------------- | ------------------------------------ |
| `setAccount(account)`                  | Définir l'état du compte directement |
| `getAccount(address)`                  | Récupérer les données du compte      |
| `getBalance(address)`                  | Obtenir le solde en SOL              |
| `airdrop(address, lamports)`           | Approvisionner un compte             |
| `minimumBalanceForRentExemption(size)` | Calculer le minimum exonéré de loyer |

### Gestion des programmes (client.svm)

| Méthode                               | Description                                |
| ------------------------------------- | ------------------------------------------ |
| `addProgram(programId, bytes)`        | Charger un programme depuis des octets     |
| `addProgramFromFile(programId, path)` | Charger un programme depuis un fichier .so |

### Transactions (client.svm)

| Méthode                   | Description                       |
| ------------------------- | --------------------------------- |
| `sendTransaction(tx)`     | Exécuter une transaction          |
| `simulateTransaction(tx)` | Simuler sans modifier l'état      |
| `latestBlockhash()`       | Obtenir le blockhash actuel       |
| `expireBlockhash()`       | Avancer vers un nouveau blockhash |

### Configuration (client.svm)

| Méthode                     | Description                                       |
| --------------------------- | ------------------------------------------------- |
| `withSigverify(bool)`       | Activer/désactiver la vérification des signatures |
| `withBlockhashCheck(bool)`  | Activer/désactiver la validation du blockhash     |
| `withSysvars()`             | Activer les comptes sysvar                        |
| `withBuiltins()`            | Activer les programmes intégrés                   |
| `withPrecompiles()`         | Activer les programmes précompilés                |
| `withTransactionHistory(n)` | Stocker les n dernières transactions              |
| `withComputeBudget(budget)` | Définir les limites de calcul                     |
| `withLamports(amount)`      | Définir les lamport par défaut                    |
| `withLogBytesLimit(n)`      | Définir la limite de sortie des journaux          |

### Temps & Sysvars (client.svm)

| Méthode                    | Description                            |
| -------------------------- | -------------------------------------- |
| `warpToSlot(slot)`         | Aller au slot spécifique               |
| `getClock()`               | Obtenir le sysvar d'horloge            |
| `getRent()`                | Obtenir le sysvar de rent              |
| `getEpochSchedule()`       | Obtenir le calendrier d'epoch          |
| `getLastRestartSlot()`     | Obtenir le dernier slot de redémarrage |
| `setLastRestartSlot(slot)` | Définir le dernier slot de redémarrage |

### RPC (client.rpc)

| Méthode                          | Description                                  |
| -------------------------------- | -------------------------------------------- |
| `getAccountInfo(address)`        | Récupérer un seul compte (base64 uniquement) |
| `getMultipleAccounts(addresses)` | Récupérer plusieurs comptes                  |
| `getLatestBlockhash()`           | Obtenir le blockhash actuel                  |
