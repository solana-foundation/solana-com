---
title: LiteSVM
description:
  Tests de programmes Solana ultra-rapides avec une VM en processus. Testez vos
  programmes plus rapidement sans surcharge réseau ni processus validator.
---

LiteSVM est une bibliothèque légère pour tester les programmes Solana qui
fonctionne en créant une VM Solana en processus, optimisée pour les développeurs
de programmes. Contrairement à d'autres approches de test qui lancent des
processus validator séparés, LiteSVM intègre la VM directement dans vos tests,
les rendant bien plus rapides à exécuter. Rompant davantage avec la tradition,
elle propose une API ergonomique avec des paramètres par défaut sensés et une
grande configurabilité pour ceux qui le souhaitent.
