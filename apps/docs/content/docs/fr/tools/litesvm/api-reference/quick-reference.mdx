## Référence rapide

| Méthode                                        | Description                                                                                                 |
| ---------------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| `LiteSVM::new()`                               | Crée un nouvel environnement de test LiteSVM de base                                                        |
| `LiteSVM::default()`                           | Retourne la « valeur par défaut » d'un type                                                                 |
| `airdrop(&Address, lamports)`                  | Approvisionne le compte avec les lamport spécifiés                                                          |
| `get_account(&Address)`                        | Retourne toutes les informations associées au compte de l'adresse fournie                                   |
| `set_account(Address, Account)`                | Définit toutes les informations associées au compte de l'adresse fournie                                    |
| `get_balance(&Address)`                        | Obtient le solde en lamport du compte à l'adresse fournie                                                   |
| `minimum_balance_for_rent_exemption(data_len)` | Retourne le solde minimum requis pour rendre un compte exempt de rent avec la longueur de données spécifiée |
| `add_program(program_id, &bytes)`              | Ajoute un programme SBF à l'environnement de test                                                           |
| `add_program_from_file(program_id, path)`      | Ajoute un programme SBF à l'environnement de test à partir du fichier spécifié                              |
| `send_transaction(tx)`                         | Soumet une transaction signée                                                                               |
| `simulate_transaction(tx)`                     | Simule une transaction sans modifier l'état                                                                 |
| `get_transaction(&signature)`                  | Récupère une transaction depuis l'historique                                                                |
| `latest_blockhash()`                           | Obtient le blockhash actuel                                                                                 |
| `expire_blockhash()`                           | Force l'expiration du blockhash actuel                                                                      |
| `warp_to_slot(slot)`                           | Déplace l'horloge vers le slot spécifié                                                                     |
| `get_sysvar<T>()`                              | Obtient une sysvar depuis l'environnement de test                                                           |
| `set_sysvar<T>(&value)`                        | Définit la sysvar dans l'environnement de test                                                              |
| `with_feature_accounts()`                      | Inclut des comptes de feature gate dans l'environnement de test                                             |
| `set_invocation_inspect_callback(callback)`    | Enregistre un callback d'invocation pré/post (nécessite la fonctionnalité `invocation-inspect-callback`)    |
