Démarrez rapidement avec Solana Pay grâce à ces guides étape par étape.

## Installation

Configurez le SDK Solana Pay et préparez votre environnement de développement.

**[→ Guide d'installation](/docs/tools/solana-pay/quickstart/installation)** -
Installez le SDK et les dépendances

## Types de paiement

Choisissez l'intégration de paiement adaptée à votre cas d'usage :

### Demandes de transfert (Simple)

Créez des URL de paiement non interactives pour les transferts basiques de SOL
et de jetons SPL.

**[→ Demandes de transfert](/docs/tools/solana-pay/quickstart/transfer-requests)** -
URL de paiement simples

**Idéal pour :**

- Paiements et dons basiques
- Paiements par QR code
- Liens de paiement dans les e-mails

### Demandes de transaction (Avancé)

Créez des flux de paiement interactifs avec composition de transactions côté
serveur.

**[→ Demandes de transaction](/docs/tools/solana-pay/quickstart/transaction-requests)** -
Flux de paiement interactifs

**Idéal pour :**

- Tarification dynamique
- Frappe de NFT
- Transactions DeFi complexes
- Logique métier personnalisée

## Intégration de QR code

Générez et personnalisez des QR codes pour les paiements par portefeuille
mobile.

**[→ Intégration de QR code](/docs/tools/solana-pay/quickstart/qr-codes)** -
Générez des codes de paiement scannables

**Idéal pour :**

- Systèmes de point de vente
- Paiements du web vers mobile
- Transactions en personne
