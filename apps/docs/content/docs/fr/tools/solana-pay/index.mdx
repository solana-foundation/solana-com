**Solana Pay est un protocole standard et un ensemble d'implémentations de
référence qui permettent aux développeurs d'intégrer des paiements décentralisés
dans leurs applications et services.** La blockchain Solana confirme les
transactions en moins d'une seconde et coûte en moyenne 0,0005 $, offrant aux
utilisateurs une expérience fluide sans intermédiaires.

## Pourquoi Solana Pay ?

- **Protocole Standard** : Approche unifiée des paiements sur tous les
  portefeuilles et applications Solana
- **Ultra Rapide** : Les transactions se confirment en moins d'une seconde
- **Ultra Économique** : Coût moyen de transaction de 0,0005 $
- **Sans Intermédiaires** : Paiements directs de pair à pair sans tiers
- **Support Universel des Portefeuilles** : Compatible avec Phantom, Solflare,
  Backpack et bien d'autres

## Démarrage Rapide

La façon la plus simple de commencer est avec une URL de demande de transfert :

```
solana:recipient?amount=0.01&label=Coffee%20Shop&message=Grande%20Latte
```

Cette URL peut être :

- Encodée sous forme de code QR pour le scan par portefeuille mobile
- Utilisée comme lien de paiement dans les applications web
- Intégrée dans les applications mobiles sous forme de liens profonds

## Installation

Installez le SDK JavaScript Solana Pay :

```bash
# Using pnpm (recommended)
pnpm add @solana/pay@beta @solana/kit

# Using npm
npm install @solana/pay@beta @solana/kit

# Using yarn
yarn add @solana/pay@beta @solana/kit
```

## Cas d'Usage

Solana Pay permet une large gamme de scénarios de paiement :

- **Paiements de Détail** : Paiements en personne utilisant des codes QR
- **E-commerce** : Flux de paiement web avec intégration de portefeuille
- **Applications Mobiles** : Expériences de paiement natives avec liaison
  profonde
- **Mint de NFT** : Création dynamique de transactions pour des interactions
  complexes
- **Abonnements** : Paiements récurrents et programmes de fidélité
- **Intégration DeFi** : Transactions composables avec swaps et prêts

## Portefeuilles compatibles

Solana Pay fonctionne avec tous les principaux portefeuilles Solana :

- **[Phantom](https://phantom.app/)** - Le portefeuille Solana le plus populaire
- **[Solflare](https://solflare.com/)** - Portefeuille Solana multi-plateforme
- **[Backpack](https://backpack.app/)** - Portefeuille Web3 et magasin
  d'applications
- **[Glow](https://glow.app/)** - Portefeuille Solana axé mobile
- **[Ultimate](https://ultimate.app/)** - Portefeuille axé sur la DeFi
- **[Decaf Wallet](https://decaf.so/)** - Portefeuille de paiements mondiaux
- **[Espresso Cash](https://www.espressocash.com/)** - Portefeuille de paiements
  mondiaux
- **[Tiplink](https://tiplink.io/)** - Le portefeuille le plus simple au monde
- **[Bitget Wallet](https://bitget.com/)** - Portefeuille multi-chaînes
