Démarrez avec Solana Attestations en installant les packages nécessaires et en
configurant votre environnement de développement.

## TypeScript/JavaScript

Installez la bibliothèque Solana Attestations en utilisant votre gestionnaire de
packages préféré :

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
