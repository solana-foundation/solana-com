## index

`index` récupère l'entrée d'une liste à l'index spécifié.

### Entrées

| Nom     | Type                                             | Description                                   |
| ------- | ------------------------------------------------ | --------------------------------------------- |
| `list`  | array[string] \| array[integer] \| array[buffer] | La liste depuis laquelle récupérer une entrée |
| `index` | integer                                          | L'index de l'entrée à récupérer               |

### Sortie

| Nom     | Type   | Description                             |
| ------- | ------ | --------------------------------------- |
| `value` | string | L'entrée de la liste à l'index spécifié |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
