Ces actions sont disponibles dans tous les Runbooks dans le cadre de la
bibliothèque standard.

## send_http_request

L'action `std::send_http_request` envoie une requête HTTP à l'URL spécifiée.

### Entrées

| Nom          | Requis    | Type    | Description                                                |
| ------------ | --------- | ------- | ---------------------------------------------------------- |
| `url`        | requis    | string  | L'URL à laquelle envoyer la requête                        |
| `method`     | optionnel | string  | La méthode HTTP (GET, POST, PUT, DELETE). Par défaut : GET |
| `headers`    | optionnel | map     | En-têtes HTTP à inclure dans la requête                    |
| `body`       | optionnel | string  | Le corps de la requête (pour les requêtes POST/PUT)        |
| `timeout_ms` | optionnel | integer | Délai d'expiration de la requête en millisecondes          |

### Sorties

| Nom           | Type    | Description                |
| ------------- | ------- | -------------------------- |
| `status_code` | integer | Le code de statut HTTP     |
| `body`        | string  | Le corps de la réponse     |
| `headers`     | map     | Les en-têtes de la réponse |

```hcl
action "api_call" "std::send_http_request" {
    description = "Fetch data from API"
    url = "https://api.example.com/data"
    method = "GET"
    headers = {
        "Authorization" = "Bearer ${variable.token}"
    }
}

output "response" {
    value = action.api_call.body
}

output "status" {
    value = action.api_call.status_code
}
```

### Exemple de requête POST

```hcl
action "create_resource" "std::send_http_request" {
    description = "Create a new resource"
    url = "https://api.example.com/resources"
    method = "POST"
    headers = {
        "Content-Type" = "application/json"
    }
    body = {
        "name" = variable.name,
        "value" = variable.value
    }
}
```
