La bibliothèque standard fournit les fonctions et actions essentielles
disponibles pour tous les Runbooks, quel que soit le module blockchain activé.

## Fonctions

La bibliothèque standard inclut des fonctions utilitaires pour :

- **Opérateurs** : Opérations mathématiques et comparaisons
- **Hash** : Fonctions de hachage (SHA256, Keccak256, etc.)
- **Hex** : Encodage et décodage hexadécimal
- **Crypto** : Opérations cryptographiques
- **JSON** : Encodage et décodage JSON
- **List** : Fonctions de manipulation de listes
- **Assertions** : Assertions d'égalité et de conditions
- **Base64** : Encodage et décodage Base64

[Voir toutes les fonctions](/docs/tools/surfpool/iac/std/functions)

## Actions

La bibliothèque standard inclut des actions pour :

- **HTTP** : Exécution de requêtes HTTP
- **Opérations sur les fichiers** : Lecture et écriture de fichiers

[Voir toutes les actions](/docs/tools/surfpool/iac/std/actions)
