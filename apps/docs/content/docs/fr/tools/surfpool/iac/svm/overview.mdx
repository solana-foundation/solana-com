---
title: Présentation de l'extension SVM
description: Extension pour Solana et les blockchains compatibles SVM (bêta)
---

Le plugin SVM `txtx` permet de créer des Runbooks qui interagissent avec Solana
et les blockchains compatibles SVM. Le plugin fournit des fonctions utilitaires
permettant de déployer des programmes Anchor et d'encoder des appels
d'instructions selon les IDL des programmes. Les actions peuvent être utilisées
pour créer des transactions valides de transfert, d'appel de programme et de
déploiement de programme, pouvant être signées via une phrase mnémonique, une
clé secrète ou via le signataire de votre navigateur.

## Démarrage rapide

Pour utiliser l'extension SVM, déclarez-la dans votre Runbook :

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## Fonctionnalités

- **Déploiement de programmes** : Déployez des programmes Anchor et des
  programmes Solana natifs
- **Construction d'instructions** : Encodez des appels d'instructions à l'aide
  des IDL de programmes
- **Signature de transactions** : Prise en charge des mnémoniques, des clés
  secrètes et des portefeuilles web
- **Gestion de comptes** : Interrogez et gérez des comptes Solana
