## ripemd160

`ripemd160` calcule le hachage Ripemd160 d'une valeur.

### Entrées

| Nom     | Type                    | Description                               |
| ------- | ----------------------- | ----------------------------------------- |
| `value` | buffer \| array[buffer] | La valeur encodée en hexadécimal à hacher |

### Sortie

| Nom     | Type   | Description       |
| ------- | ------ | ----------------- |
| `value` | string | Le résultat haché |

```hcl
output "hashed_data" {
    value = ripemd160(encode_hex("hello, world"))
}
// > hashed_data: 0XA3201F82FCA034E46D10CD7B27E174976E241DA2
```

## sha256

`sha256` calcule le hachage sha256 d'une valeur.

### Entrées

| Nom     | Type                    | Description                               |
| ------- | ----------------------- | ----------------------------------------- |
| `value` | buffer \| array[buffer] | La valeur encodée en hexadécimal à hacher |

### Sortie

| Nom     | Type   | Description       |
| ------- | ------ | ----------------- |
| `value` | string | Le résultat haché |

```hcl
output "hashed_data" {
    value = sha256(encode_hex("hello, world"))
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```

output "hashed_data" {
