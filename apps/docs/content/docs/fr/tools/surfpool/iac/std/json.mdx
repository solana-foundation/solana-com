## jq

La fonction `jq` permet de découper, filtrer et transformer des données JSON.
Consultez la [documentation jq](https://jqlang.github.io/jq/manual/) pour plus
de détails.

### Entrées

| Nom            | Type             | Description                                                                            |
| -------------- | ---------------- | -------------------------------------------------------------------------------------- |
| `decoded_json` | string \| object | Un objet JSON                                                                          |
| `query`        | string           | Une requête JSON. Consultez la [documentation jq](https://jqlang.github.io/jq/manual/) |

### Sortie

| Nom     | Type          | Description                    |
| ------- | ------------- | ------------------------------ |
| `value` | array[string] | Le résultat de la requête `jq` |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
