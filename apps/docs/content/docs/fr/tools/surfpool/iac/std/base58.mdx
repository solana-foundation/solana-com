  Fonctions de bibliothèque standard pour l'encodage et le décodage Base58

## encode_base58

`encode_base58` encode un tampon ou une chaîne de caractères en chaîne base58.

### Entrées

| Nom     | Type                  | Description                                                                                                                                   |
| ------- | --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | Le tampon ou la chaîne à encoder. Les chaînes commençant par '0x' sont décodées en hexadécimal ; sinon, les octets UTF-8 bruts sont utilisés. |

### Sortie

| Nom     | Type   | Description                                   |
| ------- | ------ | --------------------------------------------- |
| `value` | string | L'entrée, encodée sous forme de chaîne base58 |

```hcl
output "encoded" {
    value = encode_base58("0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8")
}
// > encoded: CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF
```

## decode_base58

`decode_base58` décode une chaîne encodée en base58 et retourne le résultat sous
forme de tampon.

### Entrées

| Nom             | Type   | Description                |
| --------------- | ------ | -------------------------- |
| `base58_string` | string | La chaîne base58 à décoder |

### Sortie

| Nom     | Type   | Description                                    |
| ------- | ------ | ---------------------------------------------- |
| `value` | buffer | La chaîne base58 décodée, sous forme de tampon |

```hcl
output "decoded" {
    value = decode_base58("CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF")
}
// > decoded: 0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8
```
