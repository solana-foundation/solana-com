## encode_hex

`encode_hex` encode un tampon ou une chaîne de caractères en chaîne hexadécimale
avec le préfixe 0x.

### Entrées

| Nom     | Type                  | Description                                                                                                                           |
| ------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | Le tampon ou la chaîne à encoder. Les chaînes commençant par '0x' sont décodées en hex ; sinon, les octets UTF-8 bruts sont utilisés. |

### Sortie

| Nom     | Type   | Description                                                     |
| ------- | ------ | --------------------------------------------------------------- |
| `value` | string | L'entrée dans sa représentation hexadécimale avec le préfixe 0x |

```hcl
output "encoded_hex" {
    value = encode_hex("hello, world")
}
// > encoded_hex: 0x68656c6c6f2c20776f726c64
```

## decode_hex

`decode_hex` décode une chaîne hexadécimale et retourne le résultat sous forme
de tampon.

### Entrées

| Nom          | Type   | Description                                       |
| ------------ | ------ | ------------------------------------------------- |
| `hex_string` | string | La chaîne hex à décoder (avec ou sans préfixe 0x) |

### Sortie

| Nom     | Type   | Description                                |
| ------- | ------ | ------------------------------------------ |
| `value` | buffer | La chaîne hex décodée sous forme de tampon |

```hcl
output "decoded_hex" {
    value = decode_hex("0x68656c6c6f2c20776f726c64")
}
// > decoded_hex: 0x68656c6c6f2c20776f726c64
```
