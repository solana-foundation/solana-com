## assert_eq

`assert_eq` vérifie que deux valeurs sont égales.

### Entrées

| Nom     | Type | Description            |
| ------- | ---- | ---------------------- |
| `left`  | any  | Une valeur à comparer  |
| `right` | any  | La valeur de référence |

### Sortie

| Nom     | Type                  | Description                |
| ------- | --------------------- | -------------------------- |
| `value` | addon(std::assertion) | Le résultat de l'assertion |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```

## assert_gt

`assert_gt` vérifie que la valeur de gauche est supérieure à la valeur de
droite.

### Entrées

| Nom     | Type             | Description                           |
| ------- | ---------------- | ------------------------------------- |
| `left`  | integer \| float | Un entier ou un flottant à comparer   |
| `right` | integer \| float | Un entier ou un flottant de référence |

### Sortie

| Nom     | Type                  | Description                |
| ------- | --------------------- | -------------------------- |
| `value` | addon(std::assertion) | Le résultat de l'assertion |

```hcl
output "assertion" {
    value = std::assert_gt(action.example.result, 1)
}
```
