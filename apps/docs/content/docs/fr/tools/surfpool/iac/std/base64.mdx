## encode_base64

`encode_base64` encode un tampon ou une chaîne de caractères en chaîne base64.

### Entrées

| Nom     | Type                  | Description                                                                                                                                   |
| ------- | --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | Le tampon ou la chaîne à encoder. Les chaînes commençant par '0x' sont décodées en hexadécimal ; sinon, les octets UTF-8 bruts sont utilisés. |

### Sortie

| Nom     | Type   | Description                        |
| ------- | ------ | ---------------------------------- |
| `value` | string | L'entrée, encodée en chaîne base64 |

```hcl
output "encoded" {
    value = encode_base64("0x48656c6c6f20776f726c6421")
}
// > encoded: SGVsbG8gd29ybGQh
```

## decode_base64

`decode_base64` décode une chaîne encodée en base64 et retourne le résultat sous
forme de tampon.

### Entrées

| Nom             | Type   | Description                |
| --------------- | ------ | -------------------------- |
| `base64_string` | string | La chaîne base64 à décoder |

### Sortie

| Nom     | Type   | Description                                    |
| ------- | ------ | ---------------------------------------------- |
| `value` | buffer | La chaîne base64 décodée, sous forme de tampon |

```hcl
output "decoded" {
    value = decode_base64("SGVsbG8gd29ybGQh")
}
// > decoded: 0x48656c6c6f20776f726c6421
```
