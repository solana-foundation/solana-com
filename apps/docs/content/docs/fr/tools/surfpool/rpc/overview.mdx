Surfnet est le simulateur de réseau Solana local de Surfpool, conçu pour offrir
aux développeurs un environnement de test puissant qui reproduit le comportement
de la blockchain Solana sans la surcharge d'un nœud validator complet.

## Premiers pas

Démarrez une instance Surfnet à l'aide du CLI Surfpool :

```bash
surfpool start
```

Cela démarrera une instance Surfnet locale sur `http://localhost:8899` avec tous
les points de terminaison RPC disponibles.

## Fonctionnalités clés

- **Compatibilité RPC complète** : Prend en charge l'intégralité de l'API RPC
  Solana
- **Itération rapide** : Plus besoin d'attendre les confirmations de blocs
- **Manipulation d'état** : Des cheatcodes pour modifier directement l'état de
  la blockchain
- **Fork du réseau principal** : Forkez depuis le mainnet pour tester avec de
  vrais comptes et programmes
- **Support du plugin Geyser** : Connectez vos plugins Geyser pour le streaming
  de données en temps réel
