Surfpool est un remplacement clé en main pour `solana-test-validator`, conçu
spécifiquement pour offrir la meilleure expérience possible aux développeurs qui
créent sur **Solana**.

Sa fonctionnalité principale permet aux développeurs de simuler leurs programmes
localement en utilisant des comptes Mainnet récupérés juste à temps.

Surfpool intègre également de manière transparente l'Infrastructure as Code dans
les projets basés sur Anchor ou Pinocchio, permettant des déploiements
reproductibles, auditables et sécurisés vers n'importe quel réseau Solana—privé
ou public.
