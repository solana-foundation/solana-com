Le CLI surfpool peut être installé à l'aide du script d'installation ou compilé
depuis les sources.

## Script d'installation

Pour installer surfpool, ouvrez un terminal et exécutez :

```bash
curl -sL https://run.surfpool.run/ | bash
```

## Installer depuis les sources

Pour compiler le binaire Surfpool depuis les sources, clonez le dépôt surfpool.

```bash
git clone https://github.com/solana-foundation/surfpool.git
```

Naviguez vers le nouveau répertoire.

```bash
cd surfpool
```

Ensuite, compilez le binaire. Cette commande compilera le binaire et le stockera
dans `~/.cargo/bin/surfpool`.

```bash
cargo surfpool-install
```

Enfin, assurez-vous que le binaire `surfpool` est disponible dans votre PATH. Ce
processus variera selon votre système d'exploitation.
