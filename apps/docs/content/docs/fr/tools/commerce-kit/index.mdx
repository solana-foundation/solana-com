**Commerce Kit est une collection d'outils pour créer des expériences de
commerce sur Solana.** Des boutons de paiement prêts à l'emploi aux primitives
headless, Commerce Kit fournit tout ce dont vous avez besoin pour accepter les
paiements en cryptomonnaie.

## Pourquoi Commerce Kit ?

- **Composants Prêts à l'Emploi** : PaymentButton pré-conçu avec connexion de
  portefeuille, sélection de jetons et gestion des transactions
- **Agnostique au Framework** : Utilisez des primitives headless dans n'importe
  quel framework JavaScript
- **Solana Pay** : Prise en charge complète de Solana Pay avec génération de
  code QR
- **Type-Safe** : Prise en charge complète de TypeScript pour tous les packages
- **Wallet Standard** : Compatible avec tous les portefeuilles Wallet Standard
  (Phantom, Solflare, Backpack, etc.)

## Démarrage Rapide

La façon la plus rapide de commencer est avec le composant `PaymentButton` :

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## Installation

Choisissez les packages Commerce Kit dont vous avez besoin :

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
