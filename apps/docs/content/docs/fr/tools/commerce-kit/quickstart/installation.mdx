title: Installation

---

Choisissez les packages Commerce Kit dont vous avez besoin en fonction de ce que
vous développez. Chaque package peut être installé indépendamment et utilisé
seul ou en combinaison avec d'autres.

## Configuration système requise

- **Node.js** : Version 18 ou supérieure (LTS recommandée)
- **Gestionnaire de packages** : pnpm, npm ou yarn
- **TypeScript** : Version 4.5+ (recommandé mais non obligatoire)

## Dépendances pairs communes

Solana Commerce Kit est construit sur la bibliothèque Solana Kit, il est donc
compatible avec Solana Kit, Gill, les
[Clients](https://github.com/solana-program) de programmes Solana, et les
[clients JS générés par Codama](https://github.com/codama-idl/renderers-js).

```bash
pnpm add gill # or @solana/kit
pnpm add @solana-program/token # or @solana-program/target-program
pnpm add @my-program/codama-js-client
```

## Configuration TypeScript

Si vous utilisez TypeScript, assurez-vous que votre `tsconfig.json` inclut :

```json
{
  "compilerOptions": {
    "module": "ESNext",
    "target": "ES2020",
    "lib": ["ES2020", "DOM", "DOM.Iterable"],
    "jsx": "react-jsx",
    "moduleResolution": "bundler",
    "resolveJsonModule": true,
    "esModuleInterop": true,
    "skipLibCheck": true
  }
}
```

## Configuration Next.js

Si vous utilisez Next.js, mettez à jour `next.config.js` pour transpiler les
packages Commerce Kit :

```js
/** @type {import('next').NextConfig} */
const nextConfig = {
  transpilePackages: [
    "@solana-commerce/react",
    "@solana-commerce/connector",
    "@solana-commerce/headless",
    "@solana-commerce/solana-pay",
    "@solana-commerce/sdk"
  ]
};

export default nextConfig;
```
