---
title: getBlockhash
description:
  Récupère le dernier blockhash depuis le RPC Solana auquel le serveur Kora est
  connecté.
---

## Requête JSON-RPC

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getBlockhash",
  "params": []
}
```

## Réponse JSON-RPC

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "blockhash": "base58Blockhash"
  }
}
```

## Exemple cURL

```bash
curl -X POST http://localhost:8080 \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"getBlockhash","params":[]}'
```

## SDK TypeScript

```typescript
const { blockhash } = await client.getBlockhash();
console.log("Current blockhash:", blockhash);
```
