Vous ne souhaitez pas gérer votre propre nœud Kora ? Les fournisseurs suivants
proposent une infrastructure Kora hébergée afin que vous puissiez ajouter
l'abstraction des frais à votre application sans avoir à gérer votre propre
déploiement.

## Fournisseurs

### Openfort

[Openfort](https://www.openfort.io/) propose une infrastructure paymaster Solana
alimentée par Kora, permettant des transactions sans frais via leur plateforme
de portefeuille intégré. Leur intégration utilise un modèle de co-signature où
Kora agit comme payeur de frais tandis que les utilisateurs signent via les
portefeuilles intégrés d'Openfort.

- **Site web** : [openfort.io](https://www.openfort.io/)
- **Documentation** :
  [Transactions Solana sans frais](https://www.openfort.io/blog/solana-gasless-transactions)
- **Fonctionnalités** : Portefeuilles intégrés, transactions sans frais,
  parrainage des frais

## Devenir opérateur de nœud Kora

Kora est open source et n'importe qui peut gérer un nœud. Que vous construisiez
une application nécessitant l'abstraction des frais ou que vous souhaitiez
offrir des services paymaster à d'autres, consultez le
[Guide des opérateurs](/docs/tools/kora/operators) pour démarrer.
