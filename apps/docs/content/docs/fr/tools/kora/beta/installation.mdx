## Cargo

Installez la version bêta directement depuis crates.io :

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

Les images Docker officielles multi-plateformes (amd64/arm64) sont publiées sur
le registre de conteneurs GitHub :

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

Tags disponibles :

- `latest` — Dernière version stable
- `beta` — Dernière version pré-release
- `v<version>` — Version spécifique (par exemple, `v2.2.0-beta.7`)

### Docker Compose

Pour des builds personnalisés ou pour inclure Redis pour la mise en cache et la
prise en charge des limites d'utilisation :

- [Exemple de Dockerfile](https://github.com/solana-foundation/kora/blob/main/examples/deploy/Dockerfile)
- [Docker Compose avec Redis](https://github.com/solana-foundation/kora/blob/main/docker-compose.yml)

### Exécution avec Docker

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## SDK TypeScript

Le SDK bêta est disponible sur npm :

```bash
pnpm add @solana/kora@beta
```

Consultez [Modifications du SDK](/docs/tools/kora/beta/sdk-changes) pour les
nouvelles API.

## Vérifier l'installation

```bash
kora --version
```
