## Outils de développement IA

Commencez avec notre guide sur
[comment démarrer avec les outils IA sur Solana](/developers/guides/getstarted/intro-to-ai).

### Solana MCP

[Solana MCP](https://mcp.solana.com/) (Machine Conversation Protocol) intègre
les connaissances Solana directement dans les outils de codage assistés par IA
comme Claude, Cursor, et Windsurf. Il est entraîné sur la documentation Solana,
le framework Anchor, des exemples de programmes, et les Q&R de Stack Exchange -
utile pour créer des intégrations de paiement plus rapidement.

### Frameworks d'agents IA

Les agents IA peuvent exécuter de manière autonome des transactions de paiement
sur Solana. Ces frameworks permettent de créer des agents qui traitent les
paiements, surveillent les transactions, et automatisent les flux de travail de
paiement :

| Framework        | Description                                                                                             | Liens                                                                                     |
| ---------------- | ------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | Outils pour créer des agents capables de traiter des paiements, compatibles avec LangChain et Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | Une collection de compétences pré-construites pour créer des agents sur Solana                          | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | Framework d'agents multi-plateformes avec plugin Solana pour les bots X, Telegram, Discord              | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Boîte à outils cross-chain avec actions de paiement Solana                                              | [GitHub](https://github.com/goat-sdk/goat)                                                |

### Développement de programmes IA
