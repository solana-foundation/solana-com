Cette page est une référence unique pour toutes les constantes du runtime
mentionnées dans les pages de concepts fondamentaux. Chaque constante renvoie à
sa définition source dans le client validateur
[agave](https://github.com/anza-xyz/agave/tree/v3.1.8) ou
[solana-sdk](https://github.com/anza-xyz/solana-sdk).
