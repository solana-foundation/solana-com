Bienvenue dans le guide de démarrage rapide Solana ! Ce guide pratique vous
présentera le **Solana Playground** (Solpg), où vous pourrez apprendre les
concepts fondamentaux, quel que soit votre niveau d'expérience préalable. Dans
ce tutoriel, vous découvrirez :

- **Comptes Solana** : Comment le réseau Solana stocke les données
- **Envoi de transactions** : Comment interagir avec le réseau Solana en
  envoyant des transactions
- **Construction et déploiement de programmes** : Créez votre premier programme
  Solana et déployez-le sur le réseau
- **Adresses dérivées de programme** : Apprenez à créer des adresses
  déterministes pour les comptes
- **Invocations inter-programmes** : Apprenez à appeler d'autres programmes
  depuis votre programme, permettant des interactions complexes et la
  composabilité des programmes

## Le Solana Playground

Le Solana Playground est un environnement de développement basé sur navigateur
qui vous permet de développer, déployer et tester des programmes Solana sans
rien installer localement. Suivez les instructions, copiez et collez le code, et
voyez les résultats immédiatement. Des connaissances de base en programmation
sont utiles mais non obligatoires. Il est temps de plonger et de commencer à
construire avec Solana !

En tant que nouvel utilisateur, la première étape pour interagir avec le Solana
Playground est de créer un [portefeuille](/docs/references/terminology#wallet)
Playground et d'ajouter du [SOL](/docs/references/terminology#sol). Ce
portefeuille vous permettra d'interagir avec le réseau Solana directement depuis
votre navigateur.

## Créer un portefeuille Playground

<Steps>
<Step>

### Accéder au Solana Playground

À l'aide de votre navigateur web, rendez-vous sur le
[Solana Playground](https://beta.solpg.io/).

</Step>
<Step>

### Se connecter au Playground

Cliquez sur le texte 🔴 **Not connected** dans le coin inférieur gauche de
l'écran.

![Non connecté](/assets/docs/intro/quickstart/pg-not-connected.png)

</Step>
<Step>

### Enregistrez votre keypair

Cliquez sur **Enregistrer le keypair** pour sauvegarder votre paire de clés
localement. Quand vous êtes prêt, cliquez sur **Continuer**.

![Créer un portefeuille Playground](/assets/docs/intro/quickstart/pg-create-wallet.png)

En bas de l'écran, vous verrez que vous êtes maintenant connecté au portefeuille
Playground. Vous verrez également le
[cluster connecté](/docs/references/terminology#cluster) (`devnet`),
l'[adresse du portefeuille](/docs/references/terminology#wallet) et votre solde
en [SOL](/docs/references/terminology#sol).

![Connecté](/assets/docs/intro/quickstart/pg-connected.png)

<Callout>
  Le stockage local de votre navigateur sauvegarde votre portefeuille
  Playground. Effacer le cache de votre navigateur supprimera votre portefeuille
  enregistré.
</Callout>

<Callout type="warn">
  Utilisez votre portefeuille Playground uniquement pour les tests et le
  développement. N'envoyez jamais d'actifs réels (depuis
  [`mainnet`](/docs/references/terminology#cluster)) à cette adresse.
</Callout>

Félicitations ! Vous avez créé votre premier portefeuille Solana. Vous êtes
maintenant prêt à ajouter du SOL à votre portefeuille.

</Step>
<Step>

### Ajoutez du SOL à votre portefeuille

Avant de commencer le développement, vous devez ajouter du SOL devnet à votre
portefeuille. En tant que développeur, le SOL est utilisé pour deux cas d'usage
principaux :

- Pour créer de nouveaux comptes pour stocker des données ou déployer des
  programmes
- Payer les frais de transaction lors des interactions avec le réseau Solana

Vous pouvez ajouter du SOL devnet à votre portefeuille en utilisant le terminal
Playground ou avec le Faucet Devnet. Chaque option est décrite ci-dessous :

#### Option 1 : Utiliser le terminal Playground

Exécutez la commande suivante dans le terminal Playground :

```terminal
$ solana airdrop 5
```

#### Option 2 : Utiliser le faucet Devnet

Si l'option 1 échoue, en raison de limites de taux ou d'autres erreurs, utilisez
le [Faucet Web](https://faucet.solana.com/) de Solana.

- Saisissez l'adresse de votre portefeuille
- Sélectionnez un montant
- Cliquez sur **Confirmer l'airdrop**

![Airdrop du Faucet](/assets/docs/intro/quickstart/faucet-airdrop.gif)

</Step>
</Steps>
