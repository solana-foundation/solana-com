Surfpool est un outil de développement local qui simplifie le développement sur
Solana en chargeant automatiquement les programmes et les comptes depuis un
cluster (mainnet, devnet). Utilisez la CLI Surfpool pour démarrer un validator
local afin de tester vos programmes et vos transactions.

Visitez la [documentation Surfpool](https://docs.surfpool.run/) pour plus
d'informations.

## Démarrer un validator local

Pour démarrer un validator local, exécutez la commande suivante :

```terminal
$ surfpool start
```

Par défaut, Surfpool charge automatiquement tous les programmes et comptes dont
vos transactions dépendent depuis le mainnet vers le validator local.

## Surfpool Studio

Surfpool Studio fournit une interface web pour interagir avec votre validator
local. Utilisez-le pour :

- Voir les détails des transactions
- Inspecter les données des comptes
- Airdrop de SOL et d'autres tokens via le faucet de l'interface

Pour accéder à Surfpool Studio :

1. Démarrez votre validator local avec `surfpool start`
2. Ouvrez votre navigateur et accédez à
   [http://127.0.0.1:18488](http://127.0.0.1:18488)
