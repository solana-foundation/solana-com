## Solana Client

`@solana/client` est le runtime Solana headless qui regroupe RPC, wallets,
transactions et subscriptions dans un seul store.

| Package        | Description             | GitHub                                                                                 |
| -------------- | ----------------------- | -------------------------------------------------------------------------------------- |
| @solana/client | Runtime client headless | [Source](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [Guide : présentation de @solana/client](/docs/frontend/client)

## Hooks React Solana

`@solana/react-hooks` ajoute un provider React et des hooks au-dessus de
`@solana/client`, en réutilisant le même runtime et cache.

| Package             | Description                        | GitHub                                                                                      |
| ------------------- | ---------------------------------- | ------------------------------------------------------------------------------------------- |
| @solana/react-hooks | Provider React + hooks pour les UI | [Source](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [Guide : @solana/react-hooks](/docs/frontend/react-hooks)
