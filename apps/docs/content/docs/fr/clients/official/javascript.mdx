## Solana Kit

`@solana/kit` est le SDK TypeScript recommandé pour développer sur Solana. Vous
assemblez un client à partir de plugins composables, ajoutez des liaisons React
lorsque vous en avez besoin, et intégrez des clients de programmes générés pour
les programmes on-chain que vous appelez.

| Package                   | Description                                             | GitHub                                                                                 |
| ------------------------- | ------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| @solana/kit               | SDK principal : RPC, transactions, codecs, signataires  | [Source](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | Liaisons React : `ClientProvider` et hooks              | [Source](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | Connexion RPC + envoi de transactions                   | [Source](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | Plugins de payeur de frais, d'identité et de signataire | [Source](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Connexion Wallet Standard + hooks React                 | [Source](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

## Signature et gestion des clés

Pour la signature backend compatible avec kit sur plusieurs systèmes de gestion
de clés :

| Package          | Description                                                                                                                  | GitHub                                                         |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------- |
| @solana/keychain | Signature unifiée : Memory, Vault, Privy, Turnkey, AWS KMS, Fireblocks, GCP KMS, CDP, Para, Dfns, Crossmint, Openfort, Utila | [Source](https://github.com/solana-foundation/solana-keychain) |

Consultez la [documentation Keychain](/docs/tools/keychain) et
[Choisir un backend](/docs/tools/keychain/choosing-a-backend) pour des conseils
sur la configuration et le modèle de conservation.

## Solana Web3.js

`@solana/web3.js` est le SDK TypeScript historique pour Solana. Les nouvelles
applications devraient utiliser `@solana/kit` ; consultez
[Migrer vers Kit](/docs/frontend/web3-compat).

| Package           | Description                                                         | GitHub                                                                             |
| ----------------- | ------------------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| @solana/web3.js   | SDK principal                                                       | [Source](https://github.com/solana-foundation/solana-web3.js)                      |
| @solana/spl-token | Interagir avec les programmes Token, Token-2022 et Associated Token | [Source](https://github.com/solana-program/token-2022/tree/main/clients/js-legacy) |
| @solana/spl-memo  | Interagir avec le programme Memo                                    | [Source](https://github.com/solana-program/memo/tree/main/clients/js-legacy)       |
