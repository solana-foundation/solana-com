## Que sont les frais de transfert ?

L'extension de mint _rs`TransferFeeConfig`_ du Token Extensions Program applique
des frais à chaque transfert pour ce mint.

Lors du transfert :

- Le montant transféré est réduit des frais configurés
- Les frais retenus sont suivis sur le compte de jetons de destination
- Les frais retenus peuvent être retirés directement des comptes de jetons ou
  récupérés vers le mint
- Le `withdraw_withheld_authority` peut déplacer les frais récupérés vers un
  compte de jetons récepteur de frais

<Callout type="info" title="Extensions de compte de jetons">
  Lorsqu'un compte de jetons est initialisé pour un mint avec
  _rs`TransferFeeConfig`_, le compte de jetons est initialisé avec
  _rs`TransferFeeAmount`_. Lorsque le compte de jetons est créé via
  l'[Associated Token
  Program](/docs/tokens/basics/create-token-account#how-to-create-an-associated-token-account),
  la taille requise du compte est calculée et le compte de jetons est créé avec
  la taille et les lamports exempts de loyer nécessaires.
</Callout>

## Comment créer un mint avec frais de transfert

Pour créer un mint avec frais de transfert :

1. Calculez la taille du compte mint et le loyer nécessaire pour le mint et
   l'extension _rs`TransferFeeConfig`_.
2. Créez le compte mint avec _rs`CreateAccount`_, initialisez
   _rs`TransferFeeConfig`_, et initialisez le mint avec _rs`InitializeMint`_.
3. Créez des comptes de jetons pour le mint. _rs`TransferFeeAmount`_ est
   automatiquement activé pour les comptes de jetons.
4. Utilisez _rs`TransferCheckedWithFee`_ pour transférer des jetons et vérifier
   les frais attendus. _rs`TransferChecked`_ fonctionne également et retient
   automatiquement les frais configurés sur le compte de jetons de destination.
5. Utilisez _rs`WithdrawWithheldTokensFromAccounts`_ pour déplacer les frais
   retenus directement des comptes de jetons vers un compte de jetons récepteur
   de frais, signé par le `withdraw_withheld_authority` du mint.
6. Utilisez _rs`HarvestWithheldTokensToMint`_ sans permission pour déplacer les
   frais retenus des comptes de jetons vers le compte mint, puis utilisez
   _rs`WithdrawWithheldTokensFromMint`_ pour retirer les frais retenus du compte
   mint vers un compte de jetons récepteur de frais, signé par le
   `withdraw_withheld_authority` du mint.
7. Utilisez _rs`SetTransferFee`_ pour mettre à jour la prochaine configuration
   des frais de transfert, qui prend effet à partir de deux époques plus tard.

## !!steps Calculer la taille du compte

Calculez la taille du compte mint pour le mint de base plus l'extension
_rs`TransferFeeConfig`_. C'est la taille utilisée dans _rs`CreateAccount`_.

<CodePlaceholder title="Exemple" />

```ts !! title="Example"
const client = await createLocalClient();
const mint = await generateKeyPairSigner();

// !focus(1:14)
const transferFeeConfigExtension = extension("TransferFeeConfig", {
  transferFeeConfigAuthority: client.payer.address,
  withdrawWithheldAuthority: client.payer.address,
  withheldAmount: 0n,
  olderTransferFee: {
    epoch: 0n,
    maximumFee: 10n,
    transferFeeBasisPoints: 150
  },
  newerTransferFee: {
    epoch: 0n,
    maximumFee: 10n,
    transferFeeBasisPoints: 150
  }
});

// !focus(1:1)
const mintSpace = BigInt(getMintSize([transferFeeConfigExtension]));
```
