## أدوات تطوير الذكاء الاصطناعي

ابدأ مع دليلنا حول
[كيفية البدء باستخدام أدوات الذكاء الاصطناعي على سولانا](/developers/guides/getstarted/intro-to-ai).

### Solana MCP

[Solana MCP](https://mcp.solana.com/) (بروتوكول المحادثة الآلية) يدمج معرفة
سولانا مباشرة في أدوات البرمجة المدعومة بالذكاء الاصطناعي مثل Claude وCursor
وWindsurf. تم تدريبه على مستندات سولانا وإطار عمل Anchor وأمثلة البرامج وأسئلة
وأجوبة Stack Exchange - مفيد لبناء تكاملات الدفع بشكل أسرع.

### أطر عمل وكلاء الذكاء الاصطناعي

يمكن لوكلاء الذكاء الاصطناعي تنفيذ معاملات الدفع بشكل مستقل على سولانا. تتيح هذه
الأطر بناء وكلاء يعالجون المدفوعات ويراقبون المعاملات ويؤتمتون سير عمل الدفع:

| إطار العمل       | الوصف                                                                     | الروابط                                                                                        |
| ---------------- | ------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------- |
| Solana Agent Kit | أدوات لبناء وكلاء قادرين على الدفع، متوافقة مع LangChain وVercel AI       | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [المستندات](https://kit.sendai.fun/) |
| Send AI Skills   | مجموعة من المهارات الجاهزة لبناء الوكلاء على سولانا                       | [GitHub](https://github.com/sendaifun/skills)                                                  |
| Eliza Framework  | إطار عمل وكيل متعدد المنصات مع إضافة سولانا لروبوتات X وTelegram وDiscord | [GitHub](https://github.com/elizaOS/eliza) • [المستندات](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | مجموعة أدوات متعددة السلاسل مع إجراءات دفع سولانا                         | [GitHub](https://github.com/goat-sdk/goat)                                                     |

### تطوير برامج الذكاء الاصطناعي

[Codigo](https://codigo.ai/) هي منصة تطوير شاملة للذكاء الاصطناعي على سولانا.
