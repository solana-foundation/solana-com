بالإضافة إلى عمليات الإنشاء والسك والنقل الشائعة، يُستدعى Token Program أيضًا من
البرامج على السلسلة عبر CPI، ويتضمن تعليمات لاسترجاع lamports:

<Cards>
  <Card
    title="استدعاء Token Program عبر CPI"
    href="/docs/tokens/advanced/cpi"
  >
    استدعِ Token Program من برنامج على السلسلة — انقل الرموز، اسكها، وأحرقها،
    ووقّع بصلاحية PDA، واجمع التعليمات في CPI واحدة.
  </Card>

<Card
  title="سحب lamports الزائدة"
  href="/docs/tokens/advanced/withdraw-excess-lamports"
>
  استرجع SOL الزائدة عن الحد الأدنى المعفى من الإيجار من token account والمنافذ
  وحسابات متعددي التوقيعات دون تغيير أرصدة الرموز.
</Card>

  <Card title="تحويل lamports" href="/docs/tokens/advanced/unwrap-lamports">
    حوّل SOL المُغلَّفة إلى lamports، كليًا أو جزئيًا، دون إغلاق
    token account.
  </Card>
</Cards>

<Callout type="info">

_rs`WithdrawExcessLamports`_ و _rs`UnwrapLamports`_ متاحان عبر
[`@solana-program/token`](https://www.npmjs.com/package/@solana-program/token)
العميل. استدعاء Token Program عبر CPI ينطبق على البرامج الموجودة على السلسلة.

</Callout>

## استرجاع Lamport

_rs`WithdrawExcessLamports`_ يسترجع SOL الزائدة عن الحد الأدنى المعفى من الإيجار
من token account والمنافذ وحسابات متعددي التوقيعات. _rs`UnwrapLamports`_ يحوّل
SOL المُغلَّفة في native token account إلى lamports.
