## ماذا يفعل فك تغليف Lamports؟

_rs`UnwrapLamports`_ ينقل SOL خارج token account أصلي (SOL مُغلَّف) إلى حساب
وجهة على شكل lamport. يمكنك فك تغليف الرصيد بالكامل أو مبلغ محدد، ويبقى token
account مفتوحًا.

هذا أكثر مرونةً من إغلاق حساب SOL المُغلَّف: يتيح لك فك التغليف الجزئي سحب بعض
SOL مع إبقاء الحساب ممولًا للاستخدام لاحقًا.

### مرجع المصدر

| العنصر                        | الوصف                    | المصدر                                                                                                         |
| ----------------------------- | ------------------------ | -------------------------------------------------------------------------------------------------------------- |
| _rs`UnwrapLamports`_          | التعليمة (المُميِّز 45). | [المصدر](https://github.com/solana-program/token/blob/main/pinocchio/interface/src/instruction.rs)             |
| _rs`process_unwrap_lamports`_ | منطق المعالج.            | [المصدر](https://github.com/solana-program/token/blob/main/pinocchio/program/src/processor/unwrap_lamports.rs) |

## الحسابات

| الحساب        | الوصف                                                                |
| ------------- | -------------------------------------------------------------------- |
| `source`      | قابل للكتابة. token account الأصلي (SOL المُغلَّف) المراد فك تغليفه. |
| `destination` | قابل للكتابة. يستقبل lamport المفكوك تغليفها.                        |
| `authority`   | مالك الحساب أو مندوب (أو موقّعو multisig).                           |

<Callout type="warn">

يجب أن يكون المصدر token account أصليًّا؛ وإلا ستُعيد token accounts الأخرى
_rs`NonNativeNotSupported`_. يمكن للمندوب فك تغليف ما يصل إلى المبلغ المفوَّض
له.

</Callout>

## كيفية فك تغليف Lamports

احذف _ts`amount`_ لفك تغليف الرصيد بالكامل، أو اضبطه لفك تغليف عدد محدد من
lamport. يستخدم المثال عميل
[`@solana/kit`](/docs/clients/official/javascript#solana-kit) مُهيَّأً — راجع
[نقل التوكنات](/docs/tokens/basics/transfer-tokens) لإعداد العميل.

```ts title="Unwrap wrapped SOL (Kit)"
import { getUnwrapLamportsInstruction } from "@solana-program/token";

// source: the native (wrapped SOL) token account to unwrap.
// destination: address that receives the unwrapped lamports.
// authority: a TransactionSigner (the account owner or a delegate).

// Unwrap a specific number of lamports.
const unwrapSome = getUnwrapLamportsInstruction({
  source,
  destination,
  authority,
  amount: 500_000_000n
});

// Or omit `amount` to unwrap the entire balance.
const unwrapAll = getUnwrapLamportsInstruction({
  source,
  destination,
  authority
});

// Send whichever instruction you need.
await client.sendTransaction([unwrapSome]);
```

لسلطة multisig، مرِّر المُوقِّعين المشتركين مع _ts`multiSigners`_.
