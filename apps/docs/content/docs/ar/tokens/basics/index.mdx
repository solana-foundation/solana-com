يغطي قسم أساسيات رموز SPL التعليمات الأساسية للتفاعل مع رموز SPL، مع التركيز على
الوظائف المشتركة بين Token Program الأصلي وToken Extensions Program (Token
2022).

<Callout type="info">
  راجع [وثائق Metaplex](https://developers.metaplex.com/token-metadata) لمزيد من
  المعلومات حول كيفية إضافة البيانات الوصفية إلى رموز SPL.
</Callout>
