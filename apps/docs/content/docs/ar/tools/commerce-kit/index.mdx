**مجموعة أدوات التجارة هي مجموعة من الأدوات لبناء تجارب تجارية على سولانا.** من
أزرار الدفع الجاهزة إلى المكونات الأساسية المجردة، توفر مجموعة أدوات التجارة كل
ما تحتاجه لقبول المدفوعات بالعملات المشفرة.

## لماذا مجموعة أدوات التجارة؟

- **مكونات جاهزة للاستخدام**: زر الدفع مُعد مسبقًا مع الاتصال بالمحفظة واختيار
  الرمز ومعالجة المعاملات
- **مستقل عن الإطار**: استخدم المكونات الأساسية المجردة في أي إطار عمل جافا
  سكريبت
- **Solana Pay**: دعم كامل لـ Solana Pay مع إنشاء رمز الاستجابة السريعة
- **آمن من حيث الأنواع**: دعم كامل لـ TypeScript عبر جميع الحزم
- **معيار المحفظة**: متوافق مع جميع محافظ معيار المحفظة (Phantom، Solflare،
  Backpack، إلخ.)

## البدء السريع

أسرع طريقة للبدء هي باستخدام مكون `PaymentButton`:

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## التثبيت

اختر حزم مجموعة أدوات التجارة التي تحتاجها:

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
