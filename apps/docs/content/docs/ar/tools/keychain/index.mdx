**توفر سولانا Keychain واجهة موحدة لتوقيع معاملات سولانا عبر خلفيات إدارة مفاتيح
متعددة.** طوّر محلياً باستخدام مفتاح في الذاكرة، ثم انتقل إلى إدارة مفاتيح
المؤسسات، أو وحدات أمان الأجهزة السحابية (HSM)، أو خدمات المحافظ المُدارة في
بيئة الإنتاج — دون الحاجة إلى إعادة كتابة كود التوقيع الخاص بك.

### لماذا Keychain؟

- **واجهة موحدة**: سمة `SolanaSigner` واحدة تعمل عبر جميع الخلفيات
- **تبديل الخلفيات**: تغيير موفري إدارة المفاتيح من خلال الإعدادات، لا الكود
- **تجريد بدون تكلفة**: تتضمن أعلام الميزات فقط ما تحتاجه
- **مدقَّق**: مراجعة أمنية بواسطة [Accretion](https://accretion.xyz)

### البنية المعمارية

- **اللغات**: Rust + TypeScript، بتكافؤ كامل
- **السمة**: واجهة `SolanaSigner` موحدة
- **الخلفيات**: Memory، Vault، AWS KMS، GCP KMS، Privy، Turnkey، Fireblocks،
  CDP، Crossmint، Dfns، Openfort، Para، Utila
- **التوافق**: متوافق مع `@solana/kit` و `@solana/signers` (TypeScript) | متوافق
  مع `solana-sdk` و `solana-sdk-v3` (Rust)

### الخلفيات المدعومة

تمتد الخلفيات لتشمل التطوير المحلي، وإدارة المفاتيح الذاتية والسحابية، وخدمات
المحافظ المُدارة أو MPC. راجع
[اختيار خلفية](/docs/tools/keychain/choosing-a-backend) لمقارنة نماذج الحضانة
واختيار الأنسب لك.

| الخلفية         | حالة الاستخدام               | Rust | TypeScript |
| --------------- | ---------------------------- | ---- | ---------- |
| Memory          | التطوير والاختبار            | ✓    | ✓          |
| HashiCorp Vault | إدارة مفاتيح ذاتية الاستضافة | ✓    | ✓          |
| AWS KMS         | سحابي أصيل (AWS)             | ✓    | ✓          |
| GCP KMS         | سحابي أصيل (GCP)             | ✓    | ✓          |
| Privy           | محافظ مضمّنة                 | ✓    | ✓          |
| Turnkey         | إدارة مفاتيح غير حافظة       | ✓    | ✓          |
| Fireblocks      | حضانة مؤسسية                 | ✓    | ✓          |
| CDP             | منصة مطوري Coinbase          | ✓    | ✓          |
| Crossmint       | محافظ Crossmint المُدارة     | ✓    | ✓          |
| Dfns            | بنية تحتية لمحافظ Dfns       | ✓    | ✓          |
| Openfort        | محافظ خلفية Openfort         | ✓    | ✓          |
| Para            | محافظ Para MPC               | ✓    | ✓          |

## البدء السريع

### Rust

قم بتثبيت حزمة Rust:

```bash
cargo add solana-keychain
```

الاستخدام الأساسي:

```rust
use solana_keychain::{Signer, SolanaSigner};

// Create a signer from any backend
let signer = Signer::from_memory("base58_private_key")?;

// All signers share the same interface
let pubkey = signer.pubkey();
let signature = signer.sign_transaction(&mut tx).await?;
```

- [دليل Rust](/docs/tools/keychain/getting-started/rust) - التثبيت الكامل وإعداد
  الواجهة الخلفية
- [Crates.io](https://crates.io/crates/solana-keychain) - حزمة Rust

### TypeScript

قم بتثبيت حزمة TypeScript:

```bash
pnpm add @solana/keychain
```

الاستخدام الأساسي:

```typescript
import { createKeychainSigner } from "@solana/keychain";
import { signTransactionWithSigners } from "@solana/signers";

// Create any signer via the unified factory
const signer = await createKeychainSigner({
  backend: "vault",
  vaultAddr: "https://vault.example.com:8200",
  vaultToken: "hvs.xxxxx",
  keyName: "my-solana-key",
  publicKey: "base58_public_key"
});

// Sign an already-compiled transaction
const signedTx = await signTransactionWithSigners(
  [signer],
  compiledTransaction
);
```

- [دليل TypeScript](/docs/tools/keychain/getting-started/typescript) -
  `@solana/keychain` الحزم
- [حزمة npm](https://www.npmjs.com/package/@solana/keychain) - حزمة npm
