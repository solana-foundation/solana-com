## مرجع سريع

### إعداد الإضافة

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### إدارة الحسابات (client.svm)

| الطريقة                                | الوصف                              |
| -------------------------------------- | ---------------------------------- |
| `setAccount(account)`                  | تعيين حالة الحساب مباشرةً          |
| `getAccount(address)`                  | استرداد بيانات الحساب              |
| `getBalance(address)`                  | الحصول على رصيد SOL                |
| `airdrop(address, lamports)`           | تمويل حساب                         |
| `minimumBalanceForRentExemption(size)` | حساب الحد الأدنى المعفى من الإيجار |

### إدارة البرامج (client.svm)

| الطريقة                               | الوصف                         |
| ------------------------------------- | ----------------------------- |
| `addProgram(programId, bytes)`        | تحميل برنامج من بيانات ثنائية |
| `addProgramFromFile(programId, path)` | تحميل برنامج من ملف .so       |

### المعاملات (client.svm)

| الطريقة                   | الوصف                       |
| ------------------------- | --------------------------- |
| `sendTransaction(tx)`     | تنفيذ معاملة                |
| `simulateTransaction(tx)` | محاكاة دون تغيير الحالة     |
| `latestBlockhash()`       | الحصول على blockhash الحالي |
| `expireBlockhash()`       | الانتقال إلى blockhash جديد |

### الإعداد والتكوين (client.svm)

| الطريقة                     | الوصف                           |
| --------------------------- | ------------------------------- |
| `withSigverify(bool)`       | تفعيل/تعطيل التحقق من التوقيع   |
| `withBlockhashCheck(bool)`  | تفعيل/تعطيل التحقق من blockhash |
| `withSysvars()`             | تفعيل حسابات sysvar             |
| `withBuiltins()`            | تفعيل البرامج المدمجة           |
| `withPrecompiles()`         | تفعيل البرامج المُجمَّعة مسبقاً |
| `withTransactionHistory(n)` | تخزين آخر n معاملة              |
| `withComputeBudget(budget)` | تعيين حدود الحوسبة              |
| `withLamports(amount)`      | تعيين lamports الافتراضية       |
| `withLogBytesLimit(n)`      | تعيين حد مخرجات السجل           |

### الوقت ومتغيرات النظام (client.svm)

| الطريقة                    | الوصف                              |
| -------------------------- | ---------------------------------- |
| `warpToSlot(slot)`         | الانتقال إلى slot محدد             |
| `getClock()`               | الحصول على متغير نظام الساعة       |
| `getRent()`                | الحصول على متغير نظام rent         |
| `getEpochSchedule()`       | الحصول على جدول epoch              |
| `getLastRestartSlot()`     | الحصول على آخر slot لإعادة التشغيل |
| `setLastRestartSlot(slot)` | تعيين آخر slot لإعادة التشغيل      |

### RPC (client.rpc)

| الطريقة                          | الوصف                       |
| -------------------------------- | --------------------------- |
| `getAccountInfo(address)`        | جلب حساب واحد (base64 فقط)  |
| `getMultipleAccounts(addresses)` | جلب حسابات متعددة           |
| `getLatestBlockhash()`           | الحصول على blockhash الحالي |
