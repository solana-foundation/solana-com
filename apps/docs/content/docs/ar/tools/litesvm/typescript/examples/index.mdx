## تشغيل الأمثلة

يمكن تشغيل جميع الأمثلة باستخدام `tsx`، غير أن بعض العناوين/المتغيرات تستلزم
استبدال العناصر النائبة بقيم فعلية.

```bash
# Install dependencies
pnpm add @solana/kit @solana/kit-plugin-litesvm @solana/kit-plugin-signer @solana-program/system

# Run an example
npx tsx example.ts
```
