## مرجع سريع

| الطريقة                                        | الوصف                                                                                |
| ---------------------------------------------- | ------------------------------------------------------------------------------------ |
| `LiteSVM::new()`                               | إنشاء بيئة اختبار LiteSVM أساسية جديدة                                               |
| `LiteSVM::default()`                           | يُعيد "القيمة الافتراضية" لنوع ما                                                    |
| `airdrop(&Address, lamports)`                  | يموّل الحساب بعدد lamport المحدد                                                     |
| `get_account(&Address)`                        | يُعيد جميع المعلومات المرتبطة بحساب العنوان المحدد                                   |
| `set_account(Address, Account)`                | يضبط جميع المعلومات المرتبطة بحساب العنوان المحدد                                    |
| `get_balance(&Address)`                        | يحصل على رصيد lamport لعنوان الحساب المحدد                                           |
| `minimum_balance_for_rent_exemption(data_len)` | يُعيد الحد الأدنى للرصيد المطلوب لجعل حساب بطول بيانات محدد معفيًا من rent           |
| `add_program(program_id, &bytes)`              | يضيف برنامج SBF إلى بيئة الاختبار                                                    |
| `add_program_from_file(program_id, path)`      | يضيف برنامج SBF إلى بيئة الاختبار من الملف المحدد                                    |
| `send_transaction(tx)`                         | يُرسل معاملة موقّعة                                                                  |
| `simulate_transaction(tx)`                     | يحاكي معاملة دون تعديل الحالة                                                        |
| `get_transaction(&signature)`                  | يسترجع معاملة من السجل                                                               |
| `latest_blockhash()`                           | يحصل على الـ blockhash الحالي                                                        |
| `expire_blockhash()`                           | يفرض انتهاء صلاحية الـ blockhash الحالي                                              |
| `warp_to_slot(slot)`                           | ينقل الساعة إلى slot المحدد                                                          |
| `get_sysvar<T>()`                              | يحصل على sysvar من بيئة الاختبار                                                     |
| `set_sysvar<T>(&value)`                        | يضبط sysvar في بيئة الاختبار                                                         |
| `with_feature_accounts()`                      | يُدرج حسابات بوابة الميزات في بيئة الاختبار                                          |
| `set_invocation_inspect_callback(callback)`    | يُسجّل استدعاءًا راجعًا قبل/بعد الاستدعاء (يتطلب ميزة `invocation-inspect-callback`) |
