## Kora v2.2.0-beta.6

_2026-03-23_ |
[إصدار GitHub](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.6)
| `cargo install kora-cli@2.2.0-beta.6`

إصدار تجريبي مع دعم حزم Jito، ووظائف RPC جديدة، وميزات أمان محسّنة.

- تمت إضافة دعم حزم Jito مع الوظائف `signBundle` و `signAndSendBundle` و
  `estimateBundleFee`
- تمت إضافة وظيفة RPC باسم `getVersion`
- تمت إضافة حماية رسوم Lighthouse
- تمت إضافة حماية reCAPTCHA v3 ضد الروبوتات لنقاط نهاية التوقيع
- تمت إضافة حدود استخدام لكل محفظة مع دعم Redis
- تمت إضافة صور Docker الرسمية على GHCR
- تمت إضافة قواعد حدود استخدام دقيقة
- SDK: `KoraClient` يقبل الآن كائن خيارات؛ `koraPlugin` الجديد لـ `@solana/kit`

راجع [وثائق الإصدار التجريبي](/docs/tools/kora/beta) للحصول على التفاصيل.

## Kora TypeScript SDK v0.2.0

_2026-03-09_ |
[إصدار GitHub](https://github.com/solana-foundation/kora/releases/tag/ts-sdk-v0.2.0)
| `pnpm add @solana/kora@0.2.0`

بنية إضافات عميل Kit للمعاملات القابلة للتركيب بدون رسوم غاز.

- تمت إضافة `createKitKoraClient()` مع تركيب إضافات Kit (إضافات المخطط والمنفذ
  والدافع)
- يتيح قابلية التركيب مع إضافات برنامج Kit مثل `tokenProgram()`
- تمت إضافة دعم Token-2022 عبر خيار التكوين `tokenProgramId`
- تمت إضافة دعم تعليمات ميزانية الحوسبة مع تقدير CU المعتمد على المحاكاة
- تمت إضافة حقل `signature` المفقود إلى استجابة `signAndSendTransaction`
- يُرجع `NoopSigner` من `getPaymentInstruction` لإلغاء تكرار الموقِّع في Kit v6
- تم رفع الاعتماد النظير `@solana/kit` إلى `^6.1.0`

راجع [دليل عميل Kit](/docs/tools/kora/guides/kit-client).

## Kora v2.0.0

_2025-11-19_ |
[إصدار GitHub](https://github.com/solana-foundation/kora/releases/tag/v2.0.0) |
`cargo install kora-cli@2.0.0`

إصدار رئيسي مع بنية معمارية جديدة.

- تقديم بنية معمارية جديدة لعقدة Kora
- نشر TypeScript SDK `@solana/kora` v0.1.0
- الانتقال إلى Solana SDK v3.0.x وتحديث جميع التبعيات
- استبدال Kora Signer المخصص بـ solana-signers crate
- إزالة طرق `sign_transaction_if_paid` و`get_all_signers`
- تغيير أنواع الاستجابة: إزالة التوقيعات من `signAndSendTransaction`
  و`signTransaction`
- تحسين تجربة المطورين والأمان والأداء
