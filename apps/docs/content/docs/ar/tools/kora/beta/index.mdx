_متاح على GitHub:
[`v2.2.0-beta.7`](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.7)_

يقدم Kora v2.2.0-beta دعم حزم Jito، وطرق RPC جديدة، وصور Docker، وميزات أمان
محسّنة، وتحسينات على SDK.

> **هذا إصدار تجريبي.** الميزات الموثقة هنا متاحة في `v2.2.0-beta.7` وقد تتغير
> قبل الإصدار المستقر. للحصول على التوثيق المستقر، راجع
> [مستندات Kora الرئيسية](/docs/tools/kora).

## النقاط البارزة

### دعم حزم Jito

أرسل ما يصل إلى 5 معاملات بشكل ذري عبر محرك كتل Jito. طرق RPC الجديدة:

- [`signBundle`](/docs/tools/kora/beta/bundle-methods#signbundle) — توقيع حزمة
  دون البث
- [`signAndSendBundle`](/docs/tools/kora/beta/bundle-methods#signandsendbundle)
  — التوقيع والإرسال إلى Jito
- [`estimateBundleFee`](/docs/tools/kora/beta/bundle-methods#estimatebundlefee)
  — تقدير إجمالي رسوم الحزمة

راجع [دليل حزم Jito](/docs/tools/kora/beta/jito-guide) للحصول على شرح كامل.

### صور Docker

صور رسمية متعددة المنصات (amd64/arm64) منشورة على GitHub Container Registry.
راجع [التثبيت](/docs/tools/kora/beta/installation).

### إضافات المعاملات

يسمح نظام الإضافات الجديد لـ Kora بالتحقق من صحة أشكال المعاملات أثناء تدفقات
التوقيع. أول إضافة مدمجة، `gas_swap`، تفرض قواعد صارمة لمعاملات تبادل الرموز
مقابل SOL بدون رسوم غاز. يتم تكوين الإضافات في `kora.toml` وتعمل تلقائيًا أثناء
`signTransaction` و`signAndSendTransaction` و`signBundle` و`signAndSendBundle`.

راجع
[التكوين — إضافات المعاملات](/docs/tools/kora/beta/configuration#transaction-plugins).

### خيارات التكوين الجديدة

- **إضافات المعاملات** للتحقق من صحة شكل المعاملات (`gas_swap`)
- **reCAPTCHA v3** للحماية من الروبوتات لنقاط نهاية التوقيع
- **Lighthouse** للحماية من استنزاف دافع الرسوم
- **حدود الاستخدام** للحد من المعاملات لكل محفظة
- **تكوين الحزم** لتكامل Jito

راجع [التكوين](/docs/tools/kora/beta/configuration).

### تغييرات SDK

- دعم طرق الحزم (`signBundle` و`signAndSendBundle` و`estimateBundleFee`)

راجع [تغييرات SDK](/docs/tools/kora/beta/sdk-changes).

### طرق RPC إضافية

- [`getVersion`](/docs/tools/kora/beta/bundle-methods#getversion) — استرداد
  معلومات إصدار الخادم

## التثبيت

```bash
cargo install kora-cli@2.2.0-beta.7
```

أو عبر Docker:

```bash
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```
