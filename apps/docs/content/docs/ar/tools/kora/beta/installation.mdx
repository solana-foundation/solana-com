## Cargo

قم بتثبيت الإصدار التجريبي مباشرة من crates.io:

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

يتم نشر صور Docker الرسمية متعددة المنصات (amd64/arm64) في سجل حاويات GitHub:

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

الوسوم المتاحة:

- `latest` — أحدث إصدار مستقر
- `beta` — أحدث إصدار تجريبي
- `v<version>` — إصدار محدد (على سبيل المثال، `v2.2.0-beta.7`)

### Docker Compose

للإصدارات المخصصة أو لتضمين Redis لدعم التخزين المؤقت وحدود الاستخدام:

- [نموذج Dockerfile](https://github.com/solana-foundation/kora/blob/main/examples/deploy/Dockerfile)
- [Docker Compose مع Redis](https://github.com/solana-foundation/kora/blob/main/docker-compose.yml)

### التشغيل باستخدام Docker

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## TypeScript SDK

يتوفر SDK التجريبي على npm:

```bash
pnpm add @solana/kora@beta
```

راجع [تغييرات SDK](/docs/tools/kora/beta/sdk-changes) للاطلاع على واجهات البرمجة
الجديدة.

## التحقق من التثبيت

```bash
kora --version
```
