ابدأ مع إثباتات سولانا عن طريق تثبيت الحزم الضرورية وإعداد بيئة التطوير الخاصة
بك.

## TypeScript/JavaScript

قم بتثبيت مكتبة إثباتات سولانا باستخدام مدير الحزم المفضل لديك:

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
