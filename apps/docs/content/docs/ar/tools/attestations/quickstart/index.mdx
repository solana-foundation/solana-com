ابدأ العمل مع توثيقات سولانا في بضع خطوات فقط. سيرشدك هذا الدليل عبر المفاهيم
الأساسية ويساعدك على إنشاء أول توثيق لك.

## ما ستتعلمه

- تثبيت وإعداد مكتبة التوثيقات
- إنشاء أول اعتماد لك (سلطة التوثيق)
- تعريف مخطط لتوثيقاتك
- إصدار أول توثيق لك
- جلب والتحقق من التوثيقات

## المتطلبات الأساسية

- معرفة أساسية بتطوير سولانا
- بيئة تطوير Node.js 16+ أو Rust
- محفظة سولانا تحتوي على بعض SOL لرسوم المعاملات

## خطوات البداية السريعة

### 1. التثبيت

قم بتثبيت الحزم الضرورية وإعداد بيئة التطوير الخاصة بك.

**[→ دليل التثبيت](/docs/tools/attestations/quickstart/installation)**

### 2. إنشاء اعتماد

قم بإعداد سلطة توثيق يمكنها إصدار التوثيقات.

**[→ دليل إنشاء الاعتماد](/docs/tools/attestations/instructions/create-credential)**

### 3. تعريف مخطط

أنشئ مخططاً يحدد بنية توثيقاتك.

**[→ دليل إنشاء المخطط](/docs/tools/attestations/instructions/create-schema)**

### 4. إصدار توثيق

أنشئ أول ادعاء قابل للتحقق باستخدام الاعتماد والمخطط.

**[→ دليل إنشاء التوثيق](/docs/tools/attestations/instructions/create-attestation)**

## المفاهيم الأساسية

### الاعتمادات

يمثل الاعتماد سلطة توثيق. يحدد من يمكنه إصدار التوثيقات ويحتوي على قائمة
بالموقعين المصرح لهم.

### المخططات

تحدد المخططات البنية وقواعد التحقق للتوثيقات. تحدد الحقول التي يحتويها التوثيق
وأنواع بياناتها.

### الشهادات

الشهادات هي ادعاءات أو بيانات قابلة للتحقق يتم إنشاؤها بواسطة الموقعين المعتمدين
ضمن بيانات اعتماد، وفقاً لبنية مخطط محددة.

## حالات الاستخدام

يمكن استخدام الشهادات في:

- **التحقق من الهوية**: التحقق من هويات المستخدمين والامتثال
- **أنظمة السمعة**: بناء درجات الثقة ومقاييس السمعة
- **الشهادات المهنية**: إصدار الشهادات والمؤهلات المهنية
- **الامتثال**: ضمان الامتثال التنظيمي للخدمات المالية
- **سلسلة التوريد**: تتبع صحة وأصل البضائع
