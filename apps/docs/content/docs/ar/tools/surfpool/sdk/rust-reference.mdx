يعرض الـ `surfpool-sdk` سطحاً صغيراً يتمحور حول الـ `Surfnet` البنوي. تسرد هذه
الصفحة كل عنصر عام؛ للاطلاع على الاستخدام التفصيلي، راجع الأدلة بدءاً من
[النظرة العامة](/docs/tools/surfpool/sdk/overview).

## صادرات جذر الحزمة

| العنصر                | النوع       | الملاحظات                                                        |
| --------------------- | ----------- | ---------------------------------------------------------------- |
| `Surfnet`             | بنية        | نسخة Surfnet قيد التشغيل — نقاط نهاية RPC/WS والدافع وأكواد الغش |
| `SurfnetBuilder`      | بنية        | باني سلس تستخدمه `Surfnet::builder()`                            |
| `Cheatcodes<'_>`      | بنية        | مساعد كود الغش المستعار الذي تُرجعه `Surfnet::cheatcodes()`      |
| `SurfnetError`        | تعداد       | جميع متغيرات الأخطاء التي يُرجعها SDK                            |
| `SurfnetResult<T>`    | نوع بديل    | `Result<T, SurfnetError>`                                        |
| `Keypair`             | إعادة تصدير | `solana_keypair::Keypair`                                        |
| `Pubkey`              | إعادة تصدير | `solana_pubkey::Pubkey`                                          |
| `Signer`              | إعادة تصدير | `solana_signer::Signer`                                          |
| `RpcClient`           | إعادة تصدير | `solana_rpc_client::rpc_client::RpcClient`                       |
| `BlockProductionMode` | إعادة تصدير | `surfpool_types::BlockProductionMode`                            |
| `SimnetCommand`       | إعادة تصدير | `surfpool_types::SimnetCommand`                                  |
| `SimnetEvent`         | إعادة تصدير | `surfpool_types::SimnetEvent`                                    |
| `SvmFeatureConfig`    | إعادة تصدير | `surfpool_types::SvmFeatureConfig`                               |

### دورة الحياة

```rust
impl Surfnet {
    pub fn stop(&mut self) -> SurfnetResult<()>;
}

impl Drop for Surfnet { /* calls stop() if not already stopped */ }
```

ينتظر `stop` مدةً تصل إلى خمس ثوانٍ حتى يُفرج وقت التشغيل عن منافذه. يكون الفشل
أثناء `Drop` صامتاً؛ أما الفشل أثناء استدعاء `stop` الصريح فيُعيد
`SurfnetError::Runtime`.

## SurfnetError

```rust
#[derive(Debug)]
pub enum SurfnetError {
    PortAllocation(String),
    Startup(String),
    Runtime(String),
    Cheatcode(String),
    Aborted(String),
}
```

| المتغير          | السبب                                                   |
| ---------------- | ------------------------------------------------------- |
| `PortAllocation` | تعذّر الربط بمنفذ حر لـ RPC أو WebSocket.               |
| `Startup`        | فشل تهيئة وقت التشغيل.                                  |
| `Runtime`        | تعطّل مؤشر ترابط وقت التشغيل أو خرج بشكل غير متوقع.     |
| `Cheatcode`      | أعاد استدعاء JSON-RPC لرمز الغش خطأً.                   |
| `Aborted`        | تم إيقاف تشغيل وقت التشغيل أو إلغاؤه أثناء بدء التشغيل. |

تُنفّذ الحزمة `std::error::Error` و `Display` على `SurfnetError`، لذا فهي تعمل
مع `?` و `anyhow` بشكل افتراضي.
