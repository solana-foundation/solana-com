## assert_eq

`assert_eq` تتحقق من أن قيمتين متساويتان.

### المدخلات

| الاسم   | النوع | الوصف                      |
| ------- | ----- | -------------------------- |
| `left`  | any   | قيمة للمقارنة              |
| `right` | any   | القيمة المراد المقارنة بها |

### الإخراج

| الاسم   | النوع                 | الوصف        |
| ------- | --------------------- | ------------ |
| `value` | addon(std::assertion) | نتيجة التحقق |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```

## assert_gt

`assert_gt` تتحقق من أن القيمة اليسرى أكبر من القيمة اليمنى.

### المدخلات

| الاسم   | النوع            | الوصف                        |
| ------- | ---------------- | ---------------------------- |
| `left`  | integer \| float | عدد صحيح أو عشري للمقارنة    |
| `right` | integer \| float | عدد صحيح أو عشري للمقارنة به |

### الإخراج

| الاسم   | النوع                 | الوصف        |
| ------- | --------------------- | ------------ |
| `value` | addon(std::assertion) | نتيجة التحقق |

```hcl
output "assertion" {
    value = std::assert_gt(action.example.result, 1)
}
```
