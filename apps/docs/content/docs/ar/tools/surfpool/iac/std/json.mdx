## jq

تتيح الدالة `jq` تقطيع بيانات JSON وتصفيتها وتحويلها. راجع
[توثيق jq](https://jqlang.github.io/jq/manual/) لمزيد من التفاصيل.

### المدخلات

| الاسم          | النوع            | الوصف                                                              |
| -------------- | ---------------- | ------------------------------------------------------------------ |
| `decoded_json` | string \| object | كائن JSON                                                          |
| `query`        | string           | استعلام JSON. راجع [توثيق jq](https://jqlang.github.io/jq/manual/) |

### المخرجات

| الاسم   | النوع         | الوصف              |
| ------- | ------------- | ------------------ |
| `value` | array[string] | نتيجة استعلام `jq` |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
