## and_bool

`and_bool` تُعيد العملية AND الثنائية لوسيطَي الجانب الأيسر والجانب الأيمن.

### المدخلات

| الاسم | النوع | الوصف       |
| ----- | ----- | ----------- |
| `lhs` | bool  | قيمة منطقية |
| `rhs` | bool  | قيمة منطقية |

### المخرجات

| الاسم   | النوع | الوصف                                  |
| ------- | ----- | -------------------------------------- |
| `value` | bool  | نتيجة عملية AND الثنائية بين الوسيطَين |

```hcl
output "my_bool" {
  value = false && true
}
// > my_bool: false
```

## div

`div` تُعيد القسمة الصحيحة لوسيط الجانب الأيسر على وسيط الجانب الأيمن، مع تقريب
أي باقٍ للأسفل إلى أقرب عدد صحيح.

### المدخلات

| الاسم | النوع   | الوصف        |
| ----- | ------- | ------------ |
| `lhs` | integer | المقسوم      |
| `rhs` | integer | المقسوم عليه |

### المخرجات

| الاسم   | النوع   | الوصف                               |
| ------- | ------- | ----------------------------------- |
| `value` | integer | نتيجة قسمة المقسوم على المقسوم عليه |

```hcl
output "my_int" {
  value = 11 / -3
}
// > my_int: -3
```

## eq

`eq` يُرجع `true` إذا كانت الوسيطتان اليسرى واليمنى متساويتين و `false` إذا لم
تكونا كذلك.

### المدخلات

| الاسم | النوع | الوصف   |
| ----- | ----- | ------- |
| `lhs` | any   | أي قيمة |
| `rhs` | any   | أي قيمة |

### المخرجات

| الاسم   | النوع | الوصف                            |
| ------- | ----- | -------------------------------- |
| `value` | bool  | نتيجة فحص المساواة بين المدخلَين |

```hcl
output "is_eq" {
  value = "arg" == "arg"
}
// > is_eq: true
```

| ----- | -------------------------- | ------------------- |
| `lhs` | integer \| float \| string | قيمة قابلة للمقارنة |
| `rhs` | integer \| float \| string | قيمة قابلة للمقارنة |

| ------- | ---- | ---------------------------------------------------------------------------------------------- |
| `value` | bool | نتيجة التحقق مما إذا كانت قيمة الوسيطة على الجانب الأيسر أقل من قيمة الوسيطة على الجانب الأيمن |

## neq

`neq` يُرجع `true` إذا كانت الوسيطتان على الجانبين الأيسر والأيمن غير متساويتين،
ويُرجع `false` في الحالات الأخرى.

### المدخلات

| Name  | Type | Description       |
| ----- | ---- | ----------------- |
| `lhs` | any  | قيمة للمقارنة     |
| `rhs` | any  | قيمة للمقارنة بها |

### الإخراج

| الاسم   | النوع | الوصف                                  |
| ------- | ----- | -------------------------------------- |
| `value` | bool  | نتيجة فحص المساواة المنفي بين المدخلين |

```hcl
output "is_neq" {
  value = "arg" != "arg"
}
// > is_neq: false
```
