توفر المكتبة القياسية الوظائف والإجراءات الأساسية المتاحة لجميع كتيبات التشغيل
بغض النظر عن إضافات البلوك تشين الممكّنة.

## الوظائف

تتضمن المكتبة القياسية وظائف مساعدة لـ:

- **العوامل**: العمليات الحسابية والمقارنات
- **التجزئة**: دوال التجزئة (SHA256 وKeccak256 وغيرها)
- **Hex**: ترميز وفك ترميز النظام الست عشري
- **التشفير**: العمليات التشفيرية
- **JSON**: ترميز وفك ترميز JSON
- **القوائم**: دوال معالجة القوائم
- **التأكيدات**: تأكيدات المساواة والشروط
- **Base64**: ترميز وفك ترميز Base64

[عرض جميع الوظائف](/docs/tools/surfpool/iac/std/functions)

## الإجراءات

تتضمن المكتبة القياسية إجراءات لـ:

- **HTTP**: إرسال طلبات HTTP
- **عمليات الملفات**: قراءة الملفات وكتابتها

[عرض جميع الإجراءات](/docs/tools/surfpool/iac/std/actions)
