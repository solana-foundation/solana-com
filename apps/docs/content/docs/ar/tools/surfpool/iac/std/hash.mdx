## ripemd160

`ripemd160` تحسب تجزئة Ripemd160 لقيمة ما.

### المدخلات

| الاسم   | النوع                   | الوصف                                   |
| ------- | ----------------------- | --------------------------------------- |
| `value` | buffer \| array[buffer] | القيمة المشفرة بنظام hex المراد تجزئتها |

### المخرجات

| الاسم   | النوع  | الوصف         |
| ------- | ------ | ------------- |
| `value` | string | نتيجة التجزئة |

```hcl
output "hashed_data" {
    value = ripemd160(encode_hex("hello, world"))
}
// > hashed_data: 0XA3201F82FCA034E46D10CD7B27E174976E241DA2
```
