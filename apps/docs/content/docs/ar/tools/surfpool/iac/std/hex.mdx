## encode_hex

`encode_hex` تُرمِّز مخزناً مؤقتاً أو سلسلة نصية كسلسلة ست عشرية مع بادئة 0x.

### المدخلات

| الاسم   | النوع                 | الوصف                                                                                                                                        |
| ------- | --------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer، string، addon | المخزن المؤقت أو السلسلة النصية المراد ترميزها. السلاسل التي تبدأ بـ '0x' يتم فك ترميزها كنظام ست عشري؛ وإلا يتم استخدام بايتات UTF-8 الخام. |

### المخرجات

| الاسم   | النوع  | الوصف                                |
| ------- | ------ | ------------------------------------ |
| `value` | string | المدخل بتمثيله الست عشري مع بادئة 0x |

```hcl
output "encoded_hex" {
    value = encode_hex("hello, world")
}
// > encoded_hex: 0x68656c6c6f2c20776f726c64
```

## decode_hex

`decode_hex` تفك ترميز سلسلة ست عشرية وتُعيد النتيجة كمخزن مؤقت.

### المدخلات

| الاسم        | النوع  | الوصف                                                        |
| ------------ | ------ | ------------------------------------------------------------ |
| `hex_string` | string | السلسلة الست عشرية المراد فك ترميزها (مع بادئة 0x أو بدونها) |

### المخرجات

| الاسم   | النوع  | الوصف                                         |
| ------- | ------ | --------------------------------------------- |
| `value` | buffer | السلسلة الست عشرية المفكوكة على شكل مخزن مؤقت |

```hcl
output "decoded_hex" {
    value = decode_hex("0x68656c6c6f2c20776f726c64")
}
// > decoded_hex: 0x68656c6c6f2c20776f726c64
```
