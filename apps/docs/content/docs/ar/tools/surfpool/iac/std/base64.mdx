## encode_base64

`encode_base64` يقوم بترميز مخزن مؤقت أو سلسلة نصية كسلسلة base64.

### المدخلات

| الاسم   | النوع                 | الوصف                                                                                                                                  |
| ------- | --------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | المخزن المؤقت أو السلسلة النصية المراد ترميزها. السلاسل التي تبدأ بـ '0x' يتم فك ترميزها كـ hex؛ وإلا، يتم استخدام بايتات UTF-8 الخام. |

### المخرجات

| الاسم   | النوع  | الوصف                         |
| ------- | ------ | ----------------------------- |
| `value` | string | المدخل، مُرمَّز كسلسلة base64 |

```hcl
output "encoded" {
    value = encode_base64("0x48656c6c6f20776f726c6421")
}
// > encoded: SGVsbG8gd29ybGQh
```

## decode_base64

`decode_base64` يقوم بفك ترميز سلسلة نصية مُرمَّزة بـ base64 ويعيد النتيجة كـ
مخزن مؤقت.

### المدخلات

| الاسم           | النوع  | الوصف                          |
| --------------- | ------ | ------------------------------ |
| `base64_string` | string | سلسلة base64 المراد فك ترميزها |

### المخرجات

| الاسم   | النوع  | الوصف                                    |
| ------- | ------ | ---------------------------------------- |
| `value` | buffer | سلسلة base64 المفككة الترميز، كمخزن مؤقت |

```hcl
output "decoded" {
    value = decode_base64("SGVsbG8gd29ybGQh")
}
// > decoded: 0x48656c6c6f20776f726c6421
```
