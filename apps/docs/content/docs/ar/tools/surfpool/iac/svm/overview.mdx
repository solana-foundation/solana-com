---
title: نظرة عامة على SVM Addon
description: إضافة لسولانا وBlockchains المتوافقة مع SVM (تجريبي)
---

يتيح مكوّن SVM `txtx` بناء Runbooks تتفاعل مع سولانا وشبكات البلوكشين المتوافقة
مع SVM. يوفر المكوّن دوال مساعدة تتيح لك نشر برامج Anchor وترميز استدعاءات
التعليمات وفقاً لـ IDLs الخاصة بالبرامج. يمكن استخدام الإجراءات لإنشاء معاملات
تحويل وهيكل صحيحة، واستدعاء برامج، ونشر برامج يمكن توقيعها عبر عبارة mnemonic أو
مفتاح سري أو عبر موقّع المتصفح الخاص بك.

## البداية السريعة

لاستخدام إضافة SVM، قم بتعريفها في Runbook الخاص بك:

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## المميزات

- **نشر البرامج**: نشر برامج Anchor والبرامج الأصلية لسولانا
- **بناء التعليمات**: ترميز استدعاءات التعليمات باستخدام IDLs الخاصة بالبرامج
- **توقيع المعاملات**: دعم لعبارات mnemonic والمفاتيح السرية ومحافظ الويب
- **إدارة الحسابات**: الاستعلام عن حسابات سولانا وإدارتها
