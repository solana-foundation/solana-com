## encode_base58

`encode_base58` يُرمِّز مخزناً مؤقتاً أو سلسلة نصية كسلسلة base58.

### المدخلات

| الاسم   | النوع                 | الوصف                                                                                                                                        |
| ------- | --------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer، string، addon | المخزن المؤقت أو السلسلة النصية المراد ترميزها. السلاسل التي تبدأ بـ '0x' يتم فك ترميزها كقيم hex؛ وإلا فإنه يتم استخدام بايتات UTF-8 الخام. |

### المخرجات

| الاسم   | النوع  | الوصف                          |
| ------- | ------ | ------------------------------ |
| `value` | string | المدخل، مرمَّزاً كسلسلة base58 |

```hcl
output "encoded" {
    value = encode_base58("0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8")
}
// > encoded: CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF
```

## decode_base58

`decode_base58` يفك ترميز سلسلة نصية مشفرة بـ base58 ويُعيد النتيجة كمخزن مؤقت.

### المدخلات

| الاسم           | النوع  | الوصف                          |
| --------------- | ------ | ------------------------------ |
| `base58_string` | string | سلسلة base58 المراد فك ترميزها |

### المخرجات

| الاسم   | النوع  | الوصف                                          |
| ------- | ------ | ---------------------------------------------- |
| `value` | buffer | سلسلة base58 بعد فك الترميز، على شكل مخزن مؤقت |

```hcl
output "decoded" {
    value = decode_base58("CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF")
}
// > decoded: 0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8
```
