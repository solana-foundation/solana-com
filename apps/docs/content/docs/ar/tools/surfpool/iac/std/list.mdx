## index

`index` تسترجع العنصر من قائمة عند الفهرس المحدد.

### المدخلات

| الاسم   | النوع                                            | الوصف                             |
| ------- | ------------------------------------------------ | --------------------------------- |
| `list`  | array[string] \| array[integer] \| array[buffer] | القائمة التي سيُسترجع منها العنصر |
| `index` | integer                                          | فهرس العنصر المراد استرجاعه       |

### المخرجات

| الاسم   | النوع  | الوصف                               |
| ------- | ------ | ----------------------------------- |
| `value` | string | العنصر من القائمة عند الفهرس المحدد |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
