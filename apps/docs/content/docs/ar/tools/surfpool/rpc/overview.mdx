Surfnet هو محاكي شبكة سولانا المحلي الخاص بـ Surfpool، مصمَّم لتزويد المطورين
ببيئة اختبار قوية تحاكي سلوك بلوكشين سولانا دون الحاجة إلى تحمّل أعباء تشغيل
عقدة validator كاملة.

## البدء

ابدأ تشغيل نسخة Surfnet باستخدام واجهة سطر الأوامر Surfpool CLI:

```bash
surfpool start
```

سيؤدي ذلك إلى تشغيل نسخة Surfnet محلية على `http://localhost:8899` مع توفّر جميع
نقاط نهاية RPC.

## الميزات الرئيسية

- **توافق كامل مع RPC**: يدعم واجهة برمجة تطبيقات Solana RPC بالكامل
- **تكرار سريع**: لا حاجة للانتظار لتأكيد الكتل
- **معالجة الحالة**: أكواد خاصة (Cheatcodes) لتعديل حالة البلوكشين مباشرةً
- **نسخ الشبكة الرئيسية**: قم بنسخ الشبكة الرئيسية للاختبار باستخدام حسابات
  وبرامج حقيقية
- **دعم إضافة Geyser**: قم بتوصيل إضافات Geyser الخاصة بك لبث البيانات في الوقت
  الفعلي
