يمكن تثبيت واجهة سطر الأوامر surfpool باستخدام سكريبت التثبيت أو بناؤها من
المصدر.

## سكريبت التثبيت

لتثبيت surfpool، افتح نافذة طرفية وشغّل:

```bash
curl -sL https://run.surfpool.run/ | bash
```

## التثبيت من المصدر

لتجميع ملف Surfpool التنفيذي من المصدر، استنسخ مستودع surfpool.

```bash
git clone https://github.com/solana-foundation/surfpool.git
```

انتقل إلى المجلد الجديد.

```bash
cd surfpool
```

ثم قم بتجميع الملف التنفيذي. سيقوم هذا الأمر بتجميع الملف التنفيذي وتخزينه في
`~/.cargo/bin/surfpool`.

```bash
cargo surfpool-install
```

أخيرًا، تأكد من أن الملف التنفيذي `surfpool` متاح في متغير PATH لديك. ستختلف هذه
العملية بحسب نظام التشغيل الخاص بك.
