**سولانا باي هو بروتوكول قياسي ومجموعة من التطبيقات المرجعية التي تمكّن المطورين
من دمج المدفوعات اللامركزية في تطبيقاتهم وخدماتهم.** تؤكد بلوكشين سولانا
المعاملات في أقل من ثانية وتكلف في المتوسط 0.0005 دولار، مما يوفر للمستخدمين
تجربة سلسة دون وسطاء.

## لماذا سولانا باي؟

- **بروتوكول قياسي**: نهج موحد للمدفوعات عبر جميع محافظ وتطبيقات سولانا
- **فائق السرعة**: تُؤكَّد المعاملات في أقل من ثانية
- **منخفض التكلفة للغاية**: متوسط تكلفة المعاملة 0.0005 دولار
- **بدون وسطاء**: مدفوعات مباشرة من نظير إلى نظير دون أطراف ثالثة
- **دعم شامل للمحافظ**: متوافق مع Phantom وSolflare وBackpack والمزيد

## البدء السريع

أبسط طريقة للبدء هي باستخدام رابط طلب التحويل:

```
solana:recipient?amount=0.01&label=Coffee%20Shop&message=Grande%20Latte
```

يمكن استخدام هذا الرابط:

- كرمز QR لمسحه بواسطة محافظ الهاتف المحمول
- كرابط دفع في تطبيقات الويب
- كروابط عميقة مدمجة في تطبيقات الهاتف المحمول

## التثبيت

تثبيت مجموعة أدوات تطوير JavaScript الخاصة بـ سولانا باي:

```bash
# Using pnpm (recommended)
pnpm add @solana/pay@beta @solana/kit

# Using npm
npm install @solana/pay@beta @solana/kit

# Using yarn
yarn add @solana/pay@beta @solana/kit
```

## حالات الاستخدام

تتيح سولانا باي مجموعة واسعة من سيناريوهات الدفع:

- **مدفوعات التجزئة**: المدفوعات الشخصية باستخدام رموز QR
- **التجارة الإلكترونية**: تدفقات الدفع على الويب مع تكامل المحفظة
- **تطبيقات الهاتف المحمول**: تجارب دفع أصلية بالربط العميق
- **سك NFT**: إنشاء معاملات ديناميكية للتفاعلات المعقدة
- **الاشتراكات**: المدفوعات المتكررة وبرامج الولاء
- **تكامل DeFi**: معاملات قابلة للتركيب مع المبادلات والإقراض

## المحافظ الداعمة

يعمل سولانا باي مع جميع محافظ سولانا الرئيسية:

- **[Phantom](https://phantom.app/)** - محفظة سولانا الأكثر شعبية
- **[Solflare](https://solflare.com/)** - محفظة سولانا متعددة المنصات
- **[Backpack](https://backpack.app/)** - محفظة Web3 ومتجر تطبيقات
- **[Glow](https://glow.app/)** - محفظة سولانا للهاتف المحمول
- **[Ultimate](https://ultimate.app/)** - محفظة متخصصة في التمويل اللامركزي
- **[Decaf Wallet](https://decaf.so/)** - محفظة المدفوعات العالمية
- **[Espresso Cash](https://www.espressocash.com/)** - محفظة المدفوعات العالمية
- **[Tiplink](https://tiplink.io/)** - أبسط محفظة في العالم
- **[Bitget Wallet](https://bitget.com/)** - محفظة متعددة السلاسل
