ابدأ العمل مع Solana Pay بسرعة من خلال هذه الأدلة التفصيلية.

## التثبيت

قم بإعداد Solana Pay SDK وتكوين بيئة التطوير الخاصة بك.

**[→ دليل التثبيت](/docs/tools/solana-pay/quickstart/installation)** - تثبيت SDK
والتبعيات

## أنواع الدفع

اختر تكامل الدفع المناسب لحالتك الاستخدامية:

### طلبات التحويل (بسيطة)

أنشئ روابط دفع غير تفاعلية لتحويلات SOL والرموز SPL الأساسية.

**[→ طلبات التحويل](/docs/tools/solana-pay/quickstart/transfer-requests)** -
روابط دفع بسيطة

**الأفضل لـ:**

- المدفوعات والتبرعات الأساسية
- مدفوعات رمز الاستجابة السريعة
- روابط الدفع في رسائل البريد الإلكتروني

### طلبات المعاملات (متقدمة)

قم ببناء تدفقات دفع تفاعلية مع تركيب المعاملات من جانب الخادم.

**[→ طلبات المعاملات](/docs/tools/solana-pay/quickstart/transaction-requests)** -
تدفقات دفع تفاعلية

**الأفضل لـ:**

- التسعير الديناميكي
- سك الـ NFT
- معاملات DeFi المعقدة
- منطق الأعمال المخصص

## تكامل رمز الاستجابة السريعة

قم بإنشاء وتخصيص رموز الاستجابة السريعة لمدفوعات المحفظة المحمولة.

**[→ تكامل رمز الاستجابة السريعة](/docs/tools/solana-pay/quickstart/qr-codes)** -
إنشاء رموز دفع قابلة للمسح الضوئي

**الأفضل لـ:**

- أنظمة نقاط البيع
- مدفوعات من الويب إلى الهاتف المحمول
- المعاملات الشخصية
