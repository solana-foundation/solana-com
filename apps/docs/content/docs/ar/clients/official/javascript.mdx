## عميل سولانا

`@solana/client` هو بيئة تشغيل سولانا بدون واجهة رسومية التي تجمع RPC والمحافظ
والمعاملات والاشتراكات في مخزن واحد.

| الحزمة         | الوصف                        | GitHub                                                                                 |
| -------------- | ---------------------------- | -------------------------------------------------------------------------------------- |
| @solana/client | بيئة تشغيل العميل بدون واجهة | [المصدر](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [دليل: نظرة عامة على @solana/client](/docs/frontend/client)

## خطافات React لسولانا

`@solana/react-hooks` يضيف طبقة من موفر React والخطافات فوق `@solana/client`، مع
إعادة استخدام نفس بيئة التشغيل والذاكرة المؤقتة.

| الحزمة              | الوصف                                | GitHub                                                                                      |
| ------------------- | ------------------------------------ | ------------------------------------------------------------------------------------------- |
| @solana/react-hooks | موفر React + خطافات لواجهات المستخدم | [المصدر](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [دليل: @solana/react-hooks](/docs/frontend/react-hooks)

## مجموعة أدوات سولانا

| الحزمة                         | الوصف                            | GitHub                                                                          |
| ------------------------------ | -------------------------------- | ------------------------------------------------------------------------------- |
| @solana/kit                    | SDK الأساسي                      | [المصدر](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | التفاعل مع System Program        | [المصدر](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | التفاعل مع Token Program         | [المصدر](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | التفاعل مع برنامج Token-2022     | [المصدر](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | التفاعل مع برنامج Memo           | [المصدر](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | التفاعل مع برنامج Compute Budget | [المصدر](https://github.com/solana-program/compute-budget/tree/main/clients/js) |
| @solana/kit-plugins            | الإضافات لـ Kit                  | [المصدر](https://github.com/anza-xyz/kit-plugins)                               |

- [التوثيق الرسمي لمجموعة الأدوات](https://www.solanakit.com/)
