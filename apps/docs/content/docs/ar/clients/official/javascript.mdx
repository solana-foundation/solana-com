## سولانا Kit

`@solana/kit` هو SDK TypeScript الموصى به للبناء على سولانا. يمكنك تجميع عميل من
إضافات قابلة للتركيب، وإضافة روابط React عند الحاجة إليها، واستيراد عملاء
البرامج المُولَّدة للبرامج على السلسلة التي تستدعيها.

| الحزمة                    | الوصف                                                     | GitHub                                                                                 |
| ------------------------- | --------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| @solana/kit               | SDK الأساسي: RPC، والمعاملات، وبرامج الترميز، والموقِّعون | [المصدر](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | روابط React: `ClientProvider` والخطافات                   | [المصدر](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | اتصال RPC وإرسال المعاملات                                | [المصدر](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | إضافات دافع الرسوم والهوية والتوقيع                       | [المصدر](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | اتصال Wallet Standard وخطافات React                       | [المصدر](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

`tokenProgram()` → `client.token`).

| ------------------------------ | -------------------------------- | ------------------------------------------------------------------------------- |
| @solana-program/system         | التفاعل مع System Program        | [المصدر](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | التفاعل مع Token Program         | [المصدر](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | التفاعل مع برنامج Token-2022     | [المصدر](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | التفاعل مع برنامج Memo           | [المصدر](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | التفاعل مع برنامج Compute Budget | [المصدر](https://github.com/solana-program/compute-budget/tree/main/clients/js) |

## التوقيع وإدارة المفاتيح

للتوقيع في الواجهة الخلفية المتوافق مع kit عبر خلفيات متعددة لإدارة المفاتيح:

| الحزمة           | الوصف                                                                                                                       | GitHub                                                         |
| ---------------- | --------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------- |
| @solana/keychain | التوقيع الموحَّد: الذاكرة، Vault، Privy، Turnkey، AWS KMS، Fireblocks، GCP KMS، CDP، Para، Dfns، Crossmint، Openfort، Utila | [المصدر](https://github.com/solana-foundation/solana-keychain) |

راجع [توثيق Keychain](/docs/tools/keychain) و
[اختيار الواجهة الخلفية](/docs/tools/keychain/choosing-a-backend) للحصول على
إرشادات الإعداد ونموذج الحضانة.

## سولانا Web3.js

`@solana/web3.js` هو حزمة TypeScript القديمة لـ سولانا. يجب على التطبيقات
الجديدة استخدام `@solana/kit`؛ راجع
[الترحيل إلى Kit](/docs/frontend/web3-compat).

| الحزمة            | الوصف                                                          | GitHub                                                                             |
| ----------------- | -------------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| @solana/web3.js   | حزمة SDK الأساسية                                              | [المصدر](https://github.com/solana-foundation/solana-web3.js)                      |
| @solana/spl-token | التفاعل مع Token Program وToken-2022 وAssociated Token Program | [المصدر](https://github.com/solana-program/token-2022/tree/main/clients/js-legacy) |
| @solana/spl-memo  | التفاعل مع برنامج Memo                                         | [المصدر](https://github.com/solana-program/memo/tree/main/clients/js-legacy)       |
