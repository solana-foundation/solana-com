## مكتبة تطوير البرمجيات لبايثون لسولانا

هناك حزمتان مفيدتان للتطوير باستخدام بايثون:

[`@kevinheavy/solana.py`](https://github.com/michaelhly/solana-py)

- Solana.py هي مكتبة بايثون الأساسية للتفاعل مع سولانا.
- يمكنك استخدامها لبناء المعاملات والتفاعل مع
  [واجهة برمجة تطبيقات سولانا JSON RPC](/docs/rpc)، تمامًا كما تفعل مع
  [solana-web3.js](https://github.com/solana-labs/solana-web3.js/)

- كما أنها تغطي [Token Program](https://spl.solana.com/token).

- [أحدث توثيق](https://michaelhly.github.io/solana-py/).

[`@kevinheavey/solders`](https://github.com/kevinheavey/solders)

- `solders` هي مجموعة أدوات Python عالية الأداء لـ سولانا، مكتوبة بلغة Rust.
  توفر حلولاً قوية في المجالات التالية:

- وظائف SDK الأساسية: أزواج المفاتيح، المفاتيح العامة، التوقيع وتسلسل المعاملات.
- RPC: بناء الطلبات وتحليل الاستجابات (لواجهات برمجة التطبيقات الشبكية وغير
  المتزامنة، يُوصى باستخدام
  [solana-py](https://michaelhly.github.io/solana-py/rpc/async_api/)).
- اختبار التكامل: وحدة `solders.litesvm` هي بديل لـ `solana-test-validator` أكثر
  ملاءمة و**أسرع بكثير**. تعتمد على
  [solana-program-test](https://crates.io/crates/solana-program-test).

## مكتبة تطوير البرمجيات لعميل سولانا بايثون

إذا كنت ترغب في التفاعل مع عقود سولانا باستخدام بايثون، يمكنك استخدام مكتبات
تطوير البرمجيات للعملاء لتحويل عقود سولانا المكتوبة بلغة Rust إلى عملاء بايثون،
إذا كان بإمكانك الحصول على ملف IDL للعقد.

[`Codama-py`](https://www.npmjs.com/package/codama-py)

- توحيد أي برنامج عقد على سلسلة سولانا إلى
  [`Codama IDL`](https://github.com/codama-idl/codama) وإنشاء عميل Python
  تفاعلي.
- يدعم كلاً من برامج Anchor والبرامج المخصصة.

[`Anchor-py`](https://github.com/kevinheavey/anchorpy)

- Anchor-py هو عميل بايثون لبرامج Anchor. يمكنه إنشاء عملاء من ملفات Anchor IDL،
  والتفاعل مع برامج Anchor.
