تطوير الواجهة الأمامية على سولانا يتضمن العمل مع البرامج والمحافظ وأطر عمل
JavaScript الشهيرة مثل React. التفاعل مع هذه المكونات يتطلب التعامل مع الاتصال
وإنشاء المعاملات والقراءة من حسابات سولانا.

[سولانا Kit](https://github.com/anza-xyz/kit) (`@solana/kit`) هو حزمة تطوير
البرامج (SDK) القياسية بلغة JavaScript لهذا العمل. تقوم بتجميع عميل من إضافات
صغيرة قابلة للتركيب، وتضيف ارتباطات React عند الحاجة إليها، وتستورد عملاء
البرامج المُولَّدة لبرامج السلسلة التي تستدعيها.

## المكتبات الرئيسية

<Cards>
  <Card title="@solana/kit" href="https://github.com/anza-xyz/kit">
    - حزمة SDK الأساسية: RPC، والمُرمِّزات، والموقِّعون، والمعاملات - توفر
    `createClient`، مضيف الإضافات الذي تمتد عليه كل مكتبة أخرى
  </Card>
  <Card title="Kit plugins" href="https://github.com/anza-xyz/kit-plugins">
    - إنشاء عميل باستخدام `.use(...)`: اتصال RPC، والمحفظة، والموقِّع، وتخطيط
    المعاملات - منشورة كحزم `@solana/kit-plugin-*`
  </Card>
  <Card
    title="@solana/react"
    href="https://github.com/anza-xyz/kit/tree/main/packages/react"
  >
    - ارتباطات React لعميل Kit - `ClientProvider` بالإضافة إلى خطافات للقراءة،
    والاشتراكات، وحالة المحفظة
  </Card>
  <Card title="@solana-program/*" href="https://github.com/solana-program">
    - عملاء مُولَّدون للبرامج على السلسلة (النظام، والرمز المميز، والمزيد) -
    توفر منشئي التعليمات وإضافات العملاء مثل `tokenProgram()`
  </Card>
</Cards>

<Callout type="warn" title="web3.js v1 and wallet-adapter are legacy">
  لا تزال كثير من مشاريع سولانا تعتمد على الإصدار الأقدم `@solana/web3.js` (v1)
  و `@solana/wallet-adapter-*`. للأعمال الجديدة، يُفضَّل استخدام `@solana/kit`
  مع إضافات Kit أدناه؛ إذ يحل اكتشاف Wallet Standard (عبر
  `@solana/kit-plugin-wallet`) محل wallet-adapter للمحافظ الحديثة.
</Callout>

- [دليل عميل Kit](/docs/frontend/client): بناء عميل من إضافات لـ RPC والتوقيع
  وإرسال المعاملات.
- [دليل React](/docs/frontend/react-hooks): ارتباطات `@solana/react` وخطافات
  المحفظة مُطبَّقة فوق العميل نفسه.
- [درس Next.js التعليمي](/docs/frontend/nextjs-solana): تطبيق كامل لربط المحفظة
  وتحويل SOL في App Router.
- [الانتقال إلى Kit](/docs/frontend/web3-compat): كيفية تعيين مفاهيم web3.js و
  wallet-adapter على Kit.
