تتضمن المعاملة تعليمة واحدة أو أكثر [instructions](/docs/core/instructions)،
وتوقيعات الحسابات التي تصرح بالتغييرات، وblockhash حديث. تقوم الشبكة بمعالجة
جميع التعليمات في المعاملة معًا. إذا فشلت أي تعليمة، تفشل المعاملة بأكملها ويتم
التراجع عن جميع تغييرات الحالة.

## حقائق أساسية

- **التنفيذ الذري**: تنجح جميع التعليمات أو يتم التراجع عن الجميع. لا تزال
  الرسوم تُفرض عند الفشل.
- **حد الحجم**: 1,232 بايت كحد أقصى، مشتق من الحد الأدنى لـ IPv6 MTU (1,280
  بايت) مطروحًا منه 48 بايت لرؤوس الشبكة.
- **التوقيعات**: يقدم كل موقع توقيع Ed25519 واحد بحجم 64 بايت.
- **انتهاء صلاحية blockhash**: يكون blockhash الحديث للمعاملة صالحًا لمدة 150
  فتحة.
