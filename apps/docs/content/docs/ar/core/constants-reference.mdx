هذه الصفحة هي مرجع واحد لجميع ثوابت وقت التشغيل المذكورة عبر صفحات المفاهيم
الأساسية. كل ثابت يرتبط بتعريفه المصدري في عميل المدقق
[agave](https://github.com/anza-xyz/agave/tree/v3.1.8) أو
[solana-sdk](https://github.com/anza-xyz/solana-sdk).
