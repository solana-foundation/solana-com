Solana üzerinde frontend geliştirme, programlar, cüzdanlar ve React gibi popüler
JavaScript framework'leriyle çalışmayı içerir. Bu bileşenlerle etkileşim,
bağlantı yönetimi, işlem oluşturma ve Solana hesaplarından okuma işlemlerini
gerektirir.

Bu çalışmaya yardımcı olmak için farklı framework'lerde çeşitli Solana istemci
kütüphaneleri mevcuttur.

## Ana kütüphaneler

<Cards>
  <Card
    title="@solana/client"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/client"
  >
    - RPC, cüzdanlar ve işlemleri bir araya getiren basit Solana istemcisi -
    Yerleşik durum deposu, eylemler, izleyiciler ve bağlayıcılar içerir
  </Card>
  <Card
    title="@solana/react-hooks"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks"
  >
    - Cüzdanlar, bakiyeler, transferler, imzalar ve sorgular için eksiksiz
    hook'lar - `@solana/client` çalışma zamanı durumunu sarmalayan React
    sağlayıcı hook'ları
  </Card>
  <Card
    title="@solana/web3-compat"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/web3-compat"
  >
    - Yükseltmeyi basitleştirmek için Web3.js uyumlu araç seti - Web3.js ve kit
    karışımına dayanan daha yeni iç yapı
  </Card>
  <Card
    title="@solana/kit"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/kit"
  >
    - `@solana/react-hooks` gibi diğer Solana kütüphanelerini destekleyen düşük
    seviye Solana SDK'sı - Tamamen tree-shakable, modern web standartlarını
    kullanır ve çalışma zamanını destekler
  </Card>
</Cards>

<Callout type="warn" title="@solana/web3.js kullanımdan kaldırıldı`">
  Birçok Solana ekosistem projesi hala kullanımdan kaldırılmış `@solana/web3.js`
  kütüphanesine dayanmaktadır. Geçiş yolunuzu basitleştirmek için
  `@solana/web3-compat` tercih edin.
</Callout>

- [@solana/client kılavuzu](/docs/frontend/client): RPC, cüzdanlar ve işlemler
  için yalın, headless çalışma zamanı.
- [@solana/react-hooks kılavuzu](/docs/frontend/react-hooks): aynı istemci
  çalışma zamanı üzerine katmanlı React hook'ları.
- [@solana/web3-compat kılavuzu](/docs/frontend/web3-compat): `@solana/web3.js`
  kütüphanesinden Kit destekli yığınlara geçiş için uyumluluk katmanı.
