Solana üzerinde frontend geliştirme, programlar, cüzdanlar ve React gibi popüler
JavaScript framework'leriyle çalışmayı içerir. Bu bileşenlerle etkileşim,
bağlantı yönetimi, işlem oluşturma ve Solana hesaplarından okuma işlemlerini
gerektirir.

[Solana Kit](https://github.com/anza-xyz/kit) (`@solana/kit`), bu iş için
standart JavaScript SDK'sıdır. Küçük, birleştirilebilir eklentilerden bir
istemci oluşturursunuz, ihtiyaç duyduğunuzda React bağlantıları eklersiniz ve
çağırdığınız zincir üstü programlar için oluşturulmuş program istemcilerini
dahil edersiniz.

## Ana kütüphaneler

<Cards>
  <Card title="@solana/kit" href="https://github.com/anza-xyz/kit">
    - Temel SDK: RPC, codec'ler, imzalayıcılar ve işlemler - Diğer tüm
    kütüphanelerin genişlettiği eklenti ana bilgisayarı olan `createClient`'i
    sağlar
  </Card>
  <Card title="Kit plugins" href="https://github.com/anza-xyz/kit-plugins">
    - `.use(...)` ile bir istemci oluşturun: RPC bağlantısı, cüzdan, imzalayıcı
    ve işlem planlaması - `@solana/kit-plugin-*` paketleri olarak yayımlandı
  </Card>
  <Card
    title="@solana/react"
    href="https://github.com/anza-xyz/kit/tree/main/packages/react"
  >
    - Kit istemcisi için React bağlantıları - Okumalar, abonelikler ve cüzdan
    durumu için `ClientProvider` ile kancalar
  </Card>
  <Card title="@solana-program/*" href="https://github.com/solana-program">
    - Zincir üstü programlar için oluşturulmuş istemciler (System, Token ve daha
    fazlası) - `tokenProgram()` gibi talimat oluşturucular ve istemci
    eklentileri sağlar
  </Card>
</Cards>

<Callout type="warn" title="web3.js v1 ve wallet-adapter eskidir">
  Pek çok Solana projesi hâlâ eski `@solana/web3.js` (v1) ve
  `@solana/wallet-adapter-*`'e dayanmaktadır. Yeni çalışmalar için aşağıdaki kit
  eklentileriyle birlikte `@solana/kit`'i tercih edin; Wallet Standard keşfi
  (`@solana/kit-plugin-wallet` aracılığıyla) modern cüzdanlar için
  wallet-adapter'ın yerini almaktadır.
</Callout>
