**Solana Pay, geliştiricilerin uygulamalarına ve hizmetlerine merkezi olmayan
ödemeleri entegre etmelerini sağlayan standart bir protokol ve referans
uygulamalar setidir.** Solana blok zinciri işlemleri bir saniyeden kısa sürede
onaylar ve ortalama $0,0005 maliyetle kullanıcılara aracısız sorunsuz bir
deneyim sunar.

## Neden Solana Pay?

- **Standart Protokol**: Tüm Solana cüzdanları ve uygulamaları genelinde
  birleşik ödeme yaklaşımı
- **Son Derece Hızlı**: İşlemler bir saniyeden kısa sürede onaylanır
- **Son Derece Ucuz**: Ortalama $0,0005 işlem maliyeti
- **Aracısız**: Üçüncü taraflar olmadan doğrudan eşler arası ödemeler
- **Evrensel Cüzdan Desteği**: Phantom, Solflare, Backpack ve daha fazlasıyla
  uyumlu

## Hızlı Başlangıç

Başlamanın en basit yolu bir transfer isteği URL'sidir:

```
solana:recipient?amount=0.01&label=Coffee%20Shop&message=Grande%20Latte
```

Bu URL şu şekillerde kullanılabilir:

- Mobil cüzdan taraması için QR kod olarak kodlanabilir
- Web uygulamalarında ödeme bağlantısı olarak kullanılabilir
- Mobil uygulamalara derin bağlantılar olarak gömülebilir

**[→ Transfer İsteği Hızlı Başlangıç](/docs/tools/solana-pay/quickstart/transfer-requests)** -

**[→ İşlem İsteği Hızlı Başlangıç](/docs/tools/solana-pay/quickstart/transaction-requests)** -

## Kurulum

Solana Pay JavaScript SDK'sını yükleyin:

```bash
# Using pnpm (recommended)
pnpm add @solana/pay@beta @solana/kit

# Using npm
npm install @solana/pay@beta @solana/kit

# Using yarn
yarn add @solana/pay@beta @solana/kit
```

## Kullanım Alanları

Solana Pay geniş bir ödeme senaryosu yelpazesi sunar:

- **Perakende Ödemeler**: QR kodlar kullanarak yüz yüze ödemeler
- **E-ticaret**: Cüzdan entegrasyonlu web ödeme akışları
- **Mobil Uygulamalar**: Derin bağlantılarla yerel ödeme deneyimleri
- **NFT Basımı**: Karmaşık etkileşimler için dinamik işlem oluşturma
- **Abonelikler**: Yinelenen ödemeler ve sadakat programları
- **DeFi Entegrasyonu**: Takaslar ve borç verme ile birleştirilebilir işlemler

## Desteklenen Cüzdanlar

Solana Pay tüm büyük Solana cüzdanlarıyla çalışır:

- **[Phantom](https://phantom.app/)** - En popüler Solana cüzdanı
- **[Solflare](https://solflare.com/)** - Çoklu platform Solana cüzdanı
- **[Backpack](https://backpack.app/)** - Web3 cüzdanı ve uygulama mağazası
- **[Glow](https://glow.app/)** - Mobil odaklı Solana cüzdanı
- **[Ultimate](https://ultimate.app/)** - DeFi odaklı cüzdan
- **[Decaf Wallet](https://decaf.so/)** - Küresel Ödemeler cüzdanı
- **[Espresso Cash](https://www.espressocash.com/)** - Küresel Ödemeler cüzdanı
- **[Tiplink](https://tiplink.io/)** - Dünyanın en basit cüzdanı
- **[Bitget Wallet](https://bitget.com/)** - Çoklu zincir cüzdanı
