Surfnet, Surfpool'un yerel Solana ağ simülatörüdür; geliştiricilere tam bir
validator düğümünün ek yükü olmadan Solana blok zincirinin davranışını taklit
eden güçlü bir test ortamı sunmak için tasarlanmıştır.

## Başlarken

Surfpool CLI kullanarak bir Surfnet örneği başlatın:

```bash
surfpool start
```

Bu, tüm RPC uç noktaları kullanılabilir şekilde `http://localhost:8899` üzerinde
yerel bir Surfnet örneği başlatacaktır.

## Temel Özellikler

- **Tam RPC Uyumluluğu**: Eksiksiz Solana RPC API'sini destekler
- **Hızlı Yineleme**: Blok onaylarını beklemenize gerek yok
- **Durum Manipülasyonu**: Blok zinciri durumunu doğrudan değiştirmek için hile
  kodları
- **Mainnet Fork**: Gerçek hesaplar ve programlarla test etmek için mainnet'ten
  fork alın
- **Geyser Eklenti Desteği**: Gerçek zamanlı veri akışı için Geyser
  eklentilerinizi bağlayın
