## ripemd160

`ripemd160` bir değerin Ripemd160 hash'ini hesaplar.

### Girdiler

| Ad      | Tür                     | Açıklama                     |
| ------- | ----------------------- | ---------------------------- |
| `value` | buffer \| array[buffer] | Hash'lenecek hex-kodlu değer |

### Çıktı

| Ad      | Tür    | Açıklama          |
| ------- | ------ | ----------------- |
| `value` | string | Hash'lenmiş sonuç |

```hcl
output "hashed_data" {
    value = ripemd160(encode_hex("hello, world"))
}
// > hashed_data: 0XA3201F82FCA034E46D10CD7B27E174976E241DA2
```

### Girdiler

### Çıktı

| Ad      | Tür    | Açıklama          |
| ------- | ------ | ----------------- |
| `value` | string | Hash'lenmiş sonuç |

```hcl

    value = sha256(encode_hex("hello, world"))
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```
