title: SVM Eklentisine Genel Bakış
description: Solana ve SVM Uyumlu Blok Zincirleri için Eklenti (beta)
---

SVM `txtx` eklentisi, Solana ve SVM uyumlu blok zincirleriyle etkileşime giren
Runbook'lar oluşturmayı sağlar. Eklenti, Anchor programlarını dağıtmanıza ve
program IDL'lerine göre talimat çağrılarını kodlamanıza olanak tanıyan yardımcı
işlevler sunar. Eylemler; geçerli transfer, program çağrısı ve program dağıtım
işlemleri oluşturmak için kullanılabilir; bu işlemler bir anımsatıcı ifade,
gizli anahtar veya tarayıcı imzalayıcınız aracılığıyla imzalanabilir.

## Hızlı Başlangıç

SVM eklentisini kullanmak için Runbook'unuzda bildirin:

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## Özellikler

- **Program Dağıtımı**: Anchor ve yerel Solana programlarını dağıtın
- **Talimat Oluşturma**: Program IDL'lerini kullanarak talimat çağrılarını
  kodlayın
- **İşlem İmzalama**: Anımsatıcılar, gizli anahtarlar ve web cüzdanları için
  destek
- **Hesap Yönetimi**: Solana hesaplarını sorgulayın ve yönetin
