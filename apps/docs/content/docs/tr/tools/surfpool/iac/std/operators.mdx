## and_bool

`and_bool`, sol ve sağ taraf argümanlarının ikili VE (AND) işlemini döndürür.

### Girdiler

| Ad    | Tür  | Açıklama           |
| ----- | ---- | ------------------ |
| `lhs` | bool | Bir boolean değeri |
| `rhs` | bool | Bir boolean değeri |

### Çıktı

| Ad      | Tür  | Açıklama                                               |
| ------- | ---- | ------------------------------------------------------ |
| `value` | bool | İki argüman arasındaki ikili VE (AND) işleminin sonucu |

```hcl
output "my_bool" {
  value = false && true
}
// > my_bool: false
```

## div

`div`, sol taraf argümanının sağ taraf argümanına tam sayı bölümünü döndürür;
kalan varsa en yakın tam sayıya aşağı yuvarlar.

### Girdiler

| Ad    | Tür     | Açıklama |
| ----- | ------- | -------- |
| `lhs` | integer | Bölünen  |
| `rhs` | integer | Bölen    |

### Çıktı

| Ad      | Tür     | Açıklama                             |
| ------- | ------- | ------------------------------------ |
| `value` | integer | Bölünenin bölene bölünmesinin sonucu |

```hcl
output "my_int" {
  value = 11 / -3
}
// > my_int: -3
```

## eq

`eq`, sol ve sağ taraftaki argümanlar eşitse `true`, eşit değilse `false`
döndürür.

### Girdiler

| Ad    | Tür      | Açıklama           |
| ----- | -------- | ------------------ |
| `lhs` | herhangi | Herhangi bir değer |
| `rhs` | herhangi | Herhangi bir değer |

### Çıktı

| Ad      | Tür  | Açıklama                                        |
| ------- | ---- | ----------------------------------------------- |
| `value` | bool | İki girdi arasındaki eşitlik kontrolünün sonucu |

```hcl
output "is_eq" {
  value = "arg" == "arg"
}
// > is_eq: true
```
