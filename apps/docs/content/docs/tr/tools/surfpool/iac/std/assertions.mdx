## assert_eq

`assert_eq` iki değerin eşit olduğunu doğrular.

### Girdiler

| Ad      | Tür      | Açıklama                      |
| ------- | -------- | ----------------------------- |
| `left`  | herhangi | Karşılaştırılacak bir değer   |
| `right` | herhangi | Karşılaştırma yapılacak değer |

### Çıktı

| Ad      | Tür                   | Açıklama         |
| ------- | --------------------- | ---------------- |
| `value` | addon(std::assertion) | Doğrulama sonucu |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```

### Girdiler

| Ad      | Tür                 | Açıklama                                                |
| ------- | ------------------- | ------------------------------------------------------- |
| `left`  | tam sayı \| ondalık | Karşılaştırılacak bir tam sayı veya ondalık değer       |
| `right` | tam sayı \| ondalık | Karşılaştırma yapılacak bir tam sayı veya ondalık değer |

| Ad      | Tür                   | Açıklama         |
| ------- | --------------------- | ---------------- |
| `value` | addon(std::assertion) | Doğrulama sonucu |

```hcl

```
