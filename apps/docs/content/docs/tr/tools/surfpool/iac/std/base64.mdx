## encode_base64

`encode_base64` bir tamponu veya dizeyi base64 dizesi olarak kodlar.

### Girdiler

| Ad      | Tür                   | Açıklama                                                                                                                   |
| ------- | --------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | Kodlanacak tampon veya dize. '0x' ile başlayan dizeler hex olarak çözümlenir; aksi takdirde ham UTF-8 baytları kullanılır. |

### Çıktı

| Ad      | Tür    | Açıklama                              |
| ------- | ------ | ------------------------------------- |
| `value` | string | Girdi, base64 dizesi olarak kodlanmış |

```hcl
output "encoded" {
    value = encode_base64("0x48656c6c6f20776f726c6421")
}
// > encoded: SGVsbG8gd29ybGQh
```

## decode_base64

`decode_base64` base64 ile kodlanmış bir dizeyi çözer ve sonucu bir tampon
olarak döndürür.

### Girdiler

| Ad              | Tür    | Açıklama                |
| --------------- | ------ | ----------------------- |
| `base64_string` | string | Çözülecek base64 dizesi |

### Çıktı

| Ad      | Tür    | Açıklama                                 |
| ------- | ------ | ---------------------------------------- |
| `value` | buffer | Çözümlenmiş base64 dizesi, tampon olarak |

```hcl
output "decoded" {
    value = decode_base64("SGVsbG8gd29ybGQh")
}
// > decoded: 0x48656c6c6f20776f726c6421
```
