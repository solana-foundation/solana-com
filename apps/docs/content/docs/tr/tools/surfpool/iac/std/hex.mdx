## encode_hex

`encode_hex`, bir tamponu veya dizeyi 0x önekiyle onaltılık bir dize olarak
kodlar.

### Girdiler

| Ad      | Tür                   | Açıklama                                                                                                                      |
| ------- | --------------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| `value` | tampon, dize, eklenti | Kodlanacak tampon veya dize. '0x' ile başlayan dizeler onaltılık olarak çözülür; aksi takdirde ham UTF-8 baytları kullanılır. |

### Çıktı

| Ad      | Tür  | Açıklama                                 |
| ------- | ---- | ---------------------------------------- |
| `value` | dize | Girdinin 0x önekiyle onaltılık gösterimi |

```hcl
output "encoded_hex" {
    value = encode_hex("hello, world")
}
// > encoded_hex: 0x68656c6c6f2c20776f726c64
```

## decode_hex

`decode_hex`, onaltılık bir dizeyi çözer ve sonucu bir tampon olarak döndürür.

### Girdiler

| Ad           | Tür  | Açıklama                                            |
| ------------ | ---- | --------------------------------------------------- |
| `hex_string` | dize | Çözülecek onaltılık dize (0x önekiyle veya öneksiz) |

### Çıktı

| Ad      | Tür    | Açıklama                              |
| ------- | ------ | ------------------------------------- |
| `value` | tampon | Tampon olarak çözülmüş onaltılık dize |

```hcl
output "decoded_hex" {
    value = decode_hex("0x68656c6c6f2c20776f726c64")
}
// > decoded_hex: 0x68656c6c6f2c20776f726c64
```
