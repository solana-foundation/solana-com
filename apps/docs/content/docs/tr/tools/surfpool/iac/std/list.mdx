## index

`index`, belirtilen indeksteki girdiyi bir listeden alır.

### Girdiler

| Ad      | Tür                                              | Açıklama                  |
| ------- | ------------------------------------------------ | ------------------------- |
| `list`  | array[string] \| array[integer] \| array[buffer] | Girdi alınacak liste      |
| `index` | integer                                          | Alınacak girdinin indeksi |

### Çıktı

| Ad      | Tür    | Açıklama                            |
| ------- | ------ | ----------------------------------- |
| `value` | string | Belirtilen indeksteki liste girdisi |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
