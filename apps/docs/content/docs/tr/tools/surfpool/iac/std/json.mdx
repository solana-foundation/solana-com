## jq

`jq` fonksiyonu, JSON verilerini dilimleme, filtreleme ve eşleme işlemlerine
olanak tanır. Daha fazla ayrıntı için
[jq belgelerine](https://jqlang.github.io/jq/manual/) bakın.

### Girdiler

| Ad             | Tür              | Açıklama                                                                   |
| -------------- | ---------------- | -------------------------------------------------------------------------- |
| `decoded_json` | string \| object | Bir JSON nesnesi                                                           |
| `query`        | string           | Bir JSON sorgusu. Bkz. [jq belgeleri](https://jqlang.github.io/jq/manual/) |

### Çıktı

| Ad      | Tür           | Açıklama               |
| ------- | ------------- | ---------------------- |
| `value` | array[string] | `jq` sorgusunun sonucu |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
