Surfpool CLI, kurulum betiği kullanılarak yüklenebilir veya kaynaktan
derlenebilir.

## Kurulum Betiği

Surfpool'u yüklemek için bir terminal açın ve şunu çalıştırın:

```bash
curl -sL https://run.surfpool.run/ | bash
```

## Kaynaktan Kurulum

Surfpool ikilisini kaynaktan derlemek için surfpool deposunu klonlayın.

```bash
git clone https://github.com/solana-foundation/surfpool.git
```

Yeni dizine gidin.

```bash
cd surfpool
```

Ardından, ikiliyi derleyin. Bu komut, ikiliyi derleyecek ve
`~/.cargo/bin/surfpool` konumuna kaydedecektir.

```bash
cargo surfpool-install
```

Son olarak, `surfpool` ikilisinin PATH'inizde mevcut olduğundan emin olun. Bu
işlem, işletim sisteminize bağlı olarak farklılık gösterecektir.
