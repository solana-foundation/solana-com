`surfpool-sdk` crate'i, `Surfnet` struct'ı merkezli küçük bir yüzey sunar. Bu
sayfa tüm genel öğeleri listeler; açıklayıcı kullanım için
[Genel Bakış](/docs/tools/surfpool/sdk/overview) ile başlayan kılavuzlara bakın.

## Crate Kök Dışa Aktarmaları

| Öğe                   | Tür                  | Notlar                                                                           |
| --------------------- | -------------------- | -------------------------------------------------------------------------------- |
| `Surfnet`             | struct               | Çalışan Surfnet örneği — RPC/WS uç noktaları, ödeyici, cheatcode'lar             |
| `SurfnetBuilder`      | struct               | `Surfnet::builder()` tarafından kullanılan akıcı builder                         |
| `Cheatcodes<'_>`      | struct               | `Surfnet::cheatcodes()` tarafından döndürülen ödünç alınmış cheatcode yardımcısı |
| `SurfnetError`        | enum                 | SDK'nın döndürdüğü tüm hata varyantları                                          |
| `SurfnetResult<T>`    | tür takma adı        | `Result<T, SurfnetError>`                                                        |
| `Keypair`             | yeniden dışa aktarma | `solana_keypair::Keypair`                                                        |
| `Pubkey`              | yeniden dışa aktarma | `solana_pubkey::Pubkey`                                                          |
| `Signer`              | yeniden dışa aktarma | `solana_signer::Signer`                                                          |
| `RpcClient`           | yeniden dışa aktarma | `solana_rpc_client::rpc_client::RpcClient`                                       |
| `BlockProductionMode` | yeniden dışa aktarma | `surfpool_types::BlockProductionMode`                                            |


### Yaşam Döngüsü

```rust
impl Surfnet {
    pub fn stop(&mut self) -> SurfnetResult<()>;
}

impl Drop for Surfnet { /* calls stop() if not already stopped */ }
```

`stop`, çalışma zamanının portlarını serbest bırakmasını beklerken beş saniyeye
kadar bloke eder. `Drop` sırasında oluşan hata sessizdir; açık bir `stop`
sırasında oluşan hata ise bir `SurfnetError::Runtime` döndürür.

## SurfnetError

```rust
#[derive(Debug)]
pub enum SurfnetError {
    PortAllocation(String),
    Startup(String),
    Runtime(String),
    Cheatcode(String),
    Aborted(String),
}
```

| Varyant          | Neden                                                                       |
| ---------------- | --------------------------------------------------------------------------- |
| `PortAllocation` | RPC veya WebSocket için boş bir porta bağlanılamadı.                        |
| `Startup`        | Çalışma zamanı başlatılamadı.                                               |
| `Runtime`        | Çalışma zamanı iş parçacığı panikleyerek veya beklenmedik şekilde sonlandı. |
| `Cheatcode`      | Bir cheatcode JSON-RPC çağrısı hata döndürdü.                               |
| `Aborted`        | Çalışma zamanı, başlatma sırasında kapatıldı veya iptal edildi.             |

Crate, `SurfnetError` üzerinde `std::error::Error` ve `Display` uygular;
dolayısıyla `?` ve `anyhow` ile kutudan çıktığı gibi çalışır.
