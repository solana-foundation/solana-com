**Solana Keychain, birden fazla anahtar yönetimi arka ucu genelinde Solana
işlemlerini imzalamak için birleşik bir arayüz sunar.** Bellek içi bir anahtarla
yerel ortamda geliştirin, ardından imzalama kodunuzu yeniden yazmadan üretim
ortamında kurumsal anahtar yönetimine, bulut HSM'lerine veya yönetilen cüzdan
hizmetlerine geçin.

### Neden Keychain?

- **Tek Arayüz**: Tüm arka uçlarda çalışan tek bir `SolanaSigner` trait
- **Arka Uç Değişimi**: Anahtar yönetimi sağlayıcılarını kod değil, yapılandırma
  yoluyla değiştirin
- **Sıfır Maliyetli Soyutlama**: Özellik bayrakları yalnızca ihtiyacınız
  olanları dahil eder
- **Denetlenmiş**: [Accretion](https://accretion.xyz) tarafından güvenlik
  incelemesinden geçirilmiştir

### Mimari

- **Diller**: Rust + TypeScript, tam eşdeğerlikte
- **Trait**: Birleşik `SolanaSigner` arayüzü
- **Arka Uçlar**: Memory, Vault, AWS KMS, GCP KMS, Privy, Turnkey, Fireblocks,
  CDP, Crossmint, Dfns, Openfort, Para, Utila
- **Uyumluluk**: `@solana/kit` ve `@solana/signers` uyumlu (TypeScript) |
  `solana-sdk` ve `solana-sdk-v3` uyumlu (Rust)

### Desteklenen Altyapılar

Arka uçlar; yerel geliştirme, kendi barındırmalı ve bulut tabanlı anahtar
yönetimi ile yönetilen veya MPC cüzdan hizmetlerini kapsar. Saklama modellerini
karşılaştırmak ve size uygun olanı seçmek için
[Arka uç seçimi](/docs/tools/keychain/choosing-a-backend) sayfasına bakın.

## Hızlı Başlangıç

### Rust

Rust crate'ini yükleyin:

```bash
cargo add solana-keychain
```

Temel kullanım:

```rust
use solana_keychain::{Signer, SolanaSigner};

// Create a signer from any backend
let signer = Signer::from_memory("base58_private_key")?;

// All signers share the same interface
let pubkey = signer.pubkey();
let signature = signer.sign_transaction(&mut tx).await?;
```

- [Rust Kılavuzu](/docs/tools/keychain/getting-started/rust) - Tam kurulum ve
  arka uç yapılandırması
- [Crates.io](https://crates.io/crates/solana-keychain) - Rust crate

### TypeScript

TypeScript paketini yükleyin:

```bash
pnpm add @solana/keychain
```

Temel kullanım:

```typescript
import { createKeychainSigner } from "@solana/keychain";
import { signTransactionWithSigners } from "@solana/signers";

// Create any signer via the unified factory
const signer = await createKeychainSigner({
  backend: "vault",
  vaultAddr: "https://vault.example.com:8200",
  vaultToken: "hvs.xxxxx",
  keyName: "my-solana-key",
  publicKey: "base58_public_key"
});

// Sign an already-compiled transaction
const signedTx = await signTransactionWithSigners(
  [signer],
  compiledTransaction
);
```

- [TypeScript Kılavuzu](/docs/tools/keychain/getting-started/typescript) -
  `@solana/keychain` paketleri
- [npm Paketi](https://www.npmjs.com/package/@solana/keychain) - npm paketi
