## Kora v2.2.0-beta.6

_23.03.2026_ |
[GitHub Sürümü](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.6)
| `cargo install kora-cli@2.2.0-beta.6`

Jito paket desteği, yeni RPC yöntemleri ve gelişmiş güvenlik özellikleri içeren
ön sürüm.

- `signBundle`, `signAndSendBundle` ve `estimateBundleFee` yöntemleri ile Jito
  paket desteği eklendi
- `getVersion` RPC yöntemi eklendi
- Lighthouse ücret ödeyici koruması eklendi
- İmzalama uç noktaları için reCAPTCHA v3 bot koruması eklendi
- Redis desteği ile cüzdan başına kullanım limitleri eklendi
- GHCR üzerinde resmi Docker imajları eklendi
- Ayrıntılı kullanım limiti kuralları eklendi
- SDK: `KoraClient` artık seçenekler nesnesi alıyor; `@solana/kit` için yeni
  `koraPlugin`

Ayrıntılar için [Beta Belgeleri](/docs/tools/kora/beta)'ne bakın.

## Kora TypeScript SDK v0.2.0

_09.03.2026_ |
[GitHub Sürümü](https://github.com/solana-foundation/kora/releases/tag/ts-sdk-v0.2.0)
| `pnpm add @solana/kora@0.2.0`

Birleştirilebilir gazsız işlemler için Kit istemci eklenti mimarisi.

- Kit eklenti kompozisyonu (planlayıcı, yürütücü, ödeyici eklentileri) ile
  `createKitKoraClient()` eklendi
- `tokenProgram()` gibi Kit program eklentileri ile birleştirilebilirlik sağlar
- `tokenProgramId` yapılandırma seçeneği ile Token-2022 desteği eklendi
- Simülasyon tabanlı CU tahmini ile hesaplama bütçesi talimatı desteği eklendi
- `signAndSendTransaction` yanıtına eksik `signature` alanı eklendi
- Kit v6 imzalayan tekilleştirmesi için `getPaymentInstruction`'den `NoopSigner`
  döndürülüyor
- `@solana/kit` eş bağımlılığı `^6.1.0` sürümüne yükseltildi

[Kit İstemci Rehberi](/docs/tools/kora/guides/kit-client)'ne bakın.

## Kora v2.0.0

_19.11.2025_ |
[GitHub Sürümü](https://github.com/solana-foundation/kora/releases/tag/v2.0.0) |
`cargo install kora-cli@2.0.0`

Yeni mimariyle büyük sürüm.

- Kora düğümü için yeni mimari tanıtıldı
- TypeScript SDK `@solana/kora` v0.1.0 yayınlandı
- Solana SDK v3.0.x'e geçiş yapıldı ve tüm bağımlılıklar güncellendi
- Özel Kora Signer, solana-signers crate ile değiştirildi
- `sign_transaction_if_paid` ve `get_all_signers` metodları kaldırıldı
- Yanıt türleri değişti: `signAndSendTransaction` ve `signTransaction`'den
  imzalar kaldırıldı
- Geliştirici deneyimi, güvenlik ve performans iyileştirildi
