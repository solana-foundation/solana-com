_GitHub'da mevcut:
[`v2.2.0-beta.7`](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.7)_

Kora v2.2.0-beta, Jito paket desteği, yeni RPC yöntemleri, Docker imajları,
gelişmiş güvenlik özellikleri ve SDK iyileştirmeleri sunuyor.

> **Bu bir ön sürümdür.** Burada belgelenen özellikler `v2.2.0-beta.7` sürümünde
> mevcuttur ve kararlı sürümden önce değişebilir. Kararlı dokümantasyon için
> [ana Kora belgelerine](/docs/tools/kora) bakın.

## Öne Çıkanlar

### Jito Paket Desteği

Jito'nun blok motoru aracılığıyla 5 adede kadar işlemi atomik olarak gönderin.
Yeni RPC yöntemleri:

- [`signBundle`](/docs/tools/kora/beta/bundle-methods#signbundle) — Bir paketi
  yayınlamadan imzala
- [`signAndSendBundle`](/docs/tools/kora/beta/bundle-methods#signandsendbundle)
  — İmzala ve Jito'ya gönder
- [`estimateBundleFee`](/docs/tools/kora/beta/bundle-methods#estimatebundlefee)
  — Toplam paket ücretlerini tahmin et

Tam bir anlatım için [Jito Paket Kılavuzuna](/docs/tools/kora/beta/jito-guide)
bakın.

### Docker İmajları

GitHub Container Registry'de yayınlanan resmi çok platformlu imajlar
(amd64/arm64). [Kurulum](/docs/tools/kora/beta/installation) bölümüne bakın.

### İşlem Eklentileri

Yeni bir eklenti sistemi, Kora'nın imzalama akışları sırasında işlem yapılarını
doğrulamasına olanak tanır. İlk yerleşik eklenti olan `gas_swap`, gazsız
token-SOL takası işlemleri için katı kurallar uygular. Eklentiler `kora.toml`
dosyasında yapılandırılır ve `signTransaction`, `signAndSendTransaction`,
`signBundle` ve `signAndSendBundle` sırasında otomatik olarak çalışır.

[Yapılandırma — İşlem Eklentileri](/docs/tools/kora/beta/configuration#transaction-plugins)
bölümüne bakın.

### Yeni Yapılandırma Seçenekleri

- İşlem yapısı doğrulaması için **İşlem Eklentileri** (`gas_swap`)
- İmzalama uç noktaları için **reCAPTCHA v3** bot koruması
- **Lighthouse** ücret ödeyen tüketim koruması
- Cüzdan başına işlem sınırlaması için **Kullanım Limitleri**
- Jito entegrasyonu için **Paket yapılandırması**

[Yapılandırma](/docs/tools/kora/beta/configuration) bölümüne bakın.

### SDK Değişiklikleri

- Paket yöntemi desteği (`signBundle`, `signAndSendBundle`, `estimateBundleFee`)

[SDK Değişiklikleri](/docs/tools/kora/beta/sdk-changes) bölümüne bakın.

### Ek RPC Yöntemleri

- [`getVersion`](/docs/tools/kora/beta/bundle-methods#getversion) — Sunucu sürüm
  bilgilerini al

## Kurulum

```bash
cargo install kora-cli@2.2.0-beta.7
```

Veya Docker üzerinden:

```bash
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```
