## Cargo

Beta sürümünü doğrudan crates.io'dan kurun:

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

Resmi çoklu platform Docker imajları (amd64/arm64) GitHub Container Registry'de
yayınlanmaktadır:

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

Mevcut etiketler:

- `latest` — En son kararlı sürüm
- `beta` — En son ön sürüm versiyonu
- `v<version>` — Belirli bir versiyon (örneğin, `v2.2.0-beta.7`)

### Docker Compose

Özel derlemeler veya önbellek ve kullanım limiti desteği için Redis'i dahil
etmek için:

- [Örnek Dockerfile](https://github.com/solana-foundation/kora/blob/main/examples/deploy/Dockerfile)
- [Redis ile Docker Compose](https://github.com/solana-foundation/kora/blob/main/docker-compose.yml)

### Docker ile Çalıştırma

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## TypeScript SDK

Beta SDK, npm üzerinde mevcuttur:

```bash
pnpm add @solana/kora@beta
```

Yeni API'ler için [SDK Değişiklikleri](/docs/tools/kora/beta/sdk-changes)
sayfasına bakın.

## Kurulumu Doğrulama
