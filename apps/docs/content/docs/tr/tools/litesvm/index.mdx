---
title: LiteSVM
description:
  İşlem içi VM ile yıldırım hızında Solana program testi. Ağ yükü veya validator
  süreçleri olmadan programlarınızı daha hızlı test edin.
---

LiteSVM, program geliştiricileri için optimize edilmiş işlem içi bir Solana VM
oluşturarak çalışan, Solana programlarını test etmeye yönelik hafif bir
kütüphanedir. Ayrı validator süreçleri başlatan diğer test yaklaşımlarının
aksine, LiteSVM sanal makineyi doğrudan testlerinizin içine gömer ve testlerin
çok daha hızlı çalışmasını sağlar. Gelenekten bir adım daha uzaklaşarak,
isteyenler için akıllı varsayılan değerlere ve kapsamlı yapılandırma
seçeneklerine sahip ergonomik bir API sunar.
