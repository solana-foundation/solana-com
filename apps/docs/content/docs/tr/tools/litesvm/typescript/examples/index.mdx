## Örnekleri Çalıştırma

Tüm örnekler `tsx` ile çalıştırılabilir; ancak bazı adresler/değişkenler için
yer tutucuların gerçek değerlerle değiştirilmesi gerekir.

```bash
# Install dependencies
pnpm add @solana/kit @solana/kit-plugin-litesvm @solana/kit-plugin-signer @solana-program/system

# Run an example
npx tsx example.ts
```
