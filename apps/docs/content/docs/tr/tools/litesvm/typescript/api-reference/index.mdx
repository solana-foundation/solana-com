## Hızlı Referans

### Eklenti Kurulumu

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### Hesap Yönetimi (client.svm)

| Metot                                  | Açıklama                          |
| -------------------------------------- | --------------------------------- |
| `setAccount(account)`                  | Hesap durumunu doğrudan ayarla    |
| `getAccount(address)`                  | Hesap verisini getir              |
| `getBalance(address)`                  | SOL bakiyesini al                 |
| `airdrop(address, lamports)`           | Bir hesabı fonla                  |
| `minimumBalanceForRentExemption(size)` | Kira muafiyeti minimumunu hesapla |

### Program Yönetimi (client.svm)

| Metot                                 | Açıklama                       |
| ------------------------------------- | ------------------------------ |
| `addProgram(programId, bytes)`        | Programı baytlardan yükle      |
| `addProgramFromFile(programId, path)` | Programı .so dosyasından yükle |

### İşlemler (client.svm)

| Metot                     | Açıklama                            |
| ------------------------- | ----------------------------------- |
| `sendTransaction(tx)`     | Bir işlemi gerçekleştir             |
| `simulateTransaction(tx)` | Durum değişikliği olmadan simüle et |
| `latestBlockhash()`       | Mevcut blockhash'i al               |
| `expireBlockhash()`       | Yeni bir blockhash'e geç            |

### Yapılandırma (client.svm)

| Metot                       | Açıklama                                           |
| --------------------------- | -------------------------------------------------- |
| `withSigverify(bool)`       | İmza doğrulamayı etkinleştir/devre dışı bırak      |
| `withBlockhashCheck(bool)`  | Blockhash doğrulamayı etkinleştir/devre dışı bırak |
| `withSysvars()`             | Sysvar hesaplarını etkinleştir                     |
| `withBuiltins()`            | Yerleşik programları etkinleştir                   |
| `withPrecompiles()`         | Önceden derlenmiş programları etkinleştir          |
| `withTransactionHistory(n)` | Son n işlemi sakla                                 |
| `withComputeBudget(budget)` | Hesaplama limitlerini ayarla                       |
| `withLamports(amount)`      | Varsayılan lamport değerini ayarla                 |
| `withLogBytesLimit(n)`      | Log çıktı limitini ayarla                          |

### Zaman & Sysvars (client.svm)

| Yöntem                     | Açıklama                             |
| -------------------------- | ------------------------------------ |
| `warpToSlot(slot)`         | Belirli bir slot'a atla              |
| `getClock()`               | Clock sysvar'ı al                    |
| `getRent()`                | rent sysvar'ı al                     |
| `getEpochSchedule()`       | epoch programını al                  |
| `getLastRestartSlot()`     | Son yeniden başlatma slot'unu al     |
| `setLastRestartSlot(slot)` | Son yeniden başlatma slot'unu ayarla |

### RPC (client.rpc)

| Yöntem                           | Açıklama                           |
| -------------------------------- | ---------------------------------- |
| `getAccountInfo(address)`        | Tek hesabı getir (yalnızca base64) |
| `getMultipleAccounts(addresses)` | Birden fazla hesabı getir          |
| `getLatestBlockhash()`           | Mevcut blockhash'i al              |
