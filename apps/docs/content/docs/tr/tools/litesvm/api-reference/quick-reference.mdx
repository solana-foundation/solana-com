## Hızlı Başvuru

| Yöntem                                         | Açıklama                                                                                                  |
| ---------------------------------------------- | --------------------------------------------------------------------------------------------------------- |
| `LiteSVM::new()`                               | Yeni bir LiteSVM temel test ortamı oluşturur                                                              |
| `LiteSVM::default()`                           | Bir tür için "varsayılan değer" döndürür                                                                  |
| `airdrop(&Address, lamports)`                  | Hesabı belirtilen lamport miktarıyla fonlar                                                               |
| `get_account(&Address)`                        | Sağlanan adrese ait hesapla ilişkili tüm bilgileri döndürür                                               |
| `set_account(Address, Account)`                | Sağlanan adrese ait hesapla ilişkili tüm bilgileri ayarlar                                                |
| `get_balance(&Address)`                        | Sağlanan hesap adresinin lamport bakiyesini getirir                                                       |
| `minimum_balance_for_rent_exemption(data_len)` | Belirtilen veri uzunluğuna sahip bir hesabı rent muafiyetli yapmak için gereken minimum bakiyeyi döndürür |
| `add_program(program_id, &bytes)`              | Test ortamına bir SBF programı ekler                                                                      |
| `add_program_from_file(program_id, path)`      | Belirtilen dosyadan test ortamına bir SBF programı ekler                                                  |
| `send_transaction(tx)`                         | İmzalanmış bir işlem gönderir                                                                             |
| `simulate_transaction(tx)`                     | Durumu değiştirmeden bir işlemi simüle eder                                                               |
| `get_transaction(&signature)`                  | Geçmişten bir işlem getirir                                                                               |
| `latest_blockhash()`                           | Mevcut blok karmasını getirir                                                                             |
| `expire_blockhash()`                           | Mevcut blok karmasını zorla sona erdirir                                                                  |
| `warp_to_slot(slot)`                           | Saati belirtilen slot'a ilerletir                                                                         |
| `get_sysvar<T>()`                              | Test ortamından bir sysvar getirir                                                                        |
| `set_sysvar<T>(&value)`                        | Test ortamına sysvar değerini ayarlar                                                                     |
| `with_feature_accounts()`                      | Test ortamına özellik geçidi hesaplarını dahil eder                                                       |
| `set_invocation_inspect_callback(callback)`    | Çağrı öncesi/sonrası geri çağırma işlevi kaydeder (`invocation-inspect-callback` özelliği gerektirir)     |
