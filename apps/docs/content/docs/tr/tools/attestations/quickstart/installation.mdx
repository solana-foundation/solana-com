Gerekli paketleri yükleyerek ve geliştirme ortamınızı kurarak Solana
Attestations ile çalışmaya başlayın.

## TypeScript/JavaScript

Tercih ettiğiniz paket yöneticisini kullanarak Solana Attestations kütüphanesini
yükleyin:

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
