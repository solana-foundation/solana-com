Ne geliştirdiğinize göre ihtiyacınız olan Commerce Kit paketlerini seçin. Her
paket bağımsız olarak kurulabilir ve tek başına veya diğerleriyle birlikte
kullanılabilir.

## Sistem Gereksinimleri

- **Node.js**: Sürüm 18 veya üzeri (LTS önerilir)
- **Paket Yöneticisi**: pnpm, npm veya yarn
- **TypeScript**: Sürüm 4.5+ (önerilir ancak zorunlu değildir)

## Ortak Eş Bağımlılıklar

Solana Commerce Kit, Solana Kit kütüphanesi üzerine inşa edilmiştir, bu nedenle
Solana Kit, Gill, Solana Programs
[İstemcileri](https://github.com/solana-program) ve
[Codama tarafından oluşturulan JS İstemcileri](https://github.com/codama-idl/renderers-js)
ile uyumludur.

```bash
pnpm add gill # or @solana/kit
pnpm add @solana-program/token # or @solana-program/target-program
pnpm add @my-program/codama-js-client
```

## TypeScript Yapılandırması

TypeScript kullanıyorsanız, `tsconfig.json` dosyanızın şunları içerdiğinden emin
olun:

```json
{
  "compilerOptions": {
    "module": "ESNext",
    "target": "ES2020",
    "lib": ["ES2020", "DOM", "DOM.Iterable"],
    "jsx": "react-jsx",
    "moduleResolution": "bundler",
    "resolveJsonModule": true,
    "esModuleInterop": true,
    "skipLibCheck": true
  }
}
```

## Next.js Yapılandırması

Next.js kullanıyorsanız, Commerce Kit paketlerini dönüştürmek için
`next.config.js` dosyanızı güncelleyin:

```js
/** @type {import('next').NextConfig} */
const nextConfig = {
  transpilePackages: [
    "@solana-commerce/react",
    "@solana-commerce/connector",
    "@solana-commerce/headless",
    "@solana-commerce/solana-pay",
    "@solana-commerce/sdk"
  ]
};

export default nextConfig;
```
