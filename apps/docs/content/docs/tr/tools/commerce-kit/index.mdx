**Commerce Kit, Solana üzerinde ticaret deneyimleri oluşturmak için bir araç
koleksiyonudur.** Hazır ödeme düğmelerinden başsız temel bileşenlere kadar
Commerce Kit, kripto ödemelerini kabul etmek için ihtiyacınız olan her şeyi
sağlar.

## Neden Commerce Kit?

- **Hazır Bileşenler**: Cüzdan bağlantısı, token seçimi ve işlem yönetimi içeren
  önceden oluşturulmuş PaymentButton
- **Framework Bağımsız**: Herhangi bir JavaScript framework'ünde başsız temel
  bileşenleri kullanın
- **Solana Pay**: QR kod oluşturma ile tam Solana Pay desteği
- **Tip Güvenli**: Tüm paketlerde eksiksiz TypeScript desteği
- **Wallet Standard**: Tüm Wallet Standard cüzdanlarıyla uyumlu (Phantom,
  Solflare, Backpack, vb.)

## Hızlı Başlangıç

Başlamanın en hızlı yolu `PaymentButton` bileşenidir:

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## Kurulum

İhtiyacınız olan Commerce Kit paketlerini seçin:

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
