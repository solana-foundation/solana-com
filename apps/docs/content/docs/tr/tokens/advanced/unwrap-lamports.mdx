## Lamport Çözme Ne İşe Yarar?

_rs`UnwrapLamports`_, SOL'u bir yerel (sarılı SOL) token account'ından hedef
hesaba lamport olarak taşır. Tam bakiyeyi veya belirli bir miktarı
çözebilirsiniz; token account açık kalmaya devam eder.

Bu, sarılı SOL hesabını kapatmaktan daha esnektir: kısmi bir çözme işlemi,
hesabı ileride kullanılmak üzere fonlu tutarken bir miktar SOL çekmenizi sağlar.

### Kaynak Referans

| Öğe                           | Açıklama                  | Kaynak                                                                                                         |
| ----------------------------- | ------------------------- | -------------------------------------------------------------------------------------------------------------- |
| _rs`UnwrapLamports`_          | Talimat (ayrıştırıcı 45). | [Kaynak](https://github.com/solana-program/token/blob/main/pinocchio/interface/src/instruction.rs)             |
| _rs`process_unwrap_lamports`_ | İşlemci mantığı.          | [Kaynak](https://github.com/solana-program/token/blob/main/pinocchio/program/src/processor/unwrap_lamports.rs) |

## Hesaplar

| Hesap         | Açıklama                                                 |
| ------------- | -------------------------------------------------------- |
| `source`      | Yazılabilir. Çözülecek yerel (sarılı SOL) token account. |
| `destination` | Yazılabilir. Çözülen lamport'ları alır.                  |
| `authority`   | Hesap sahibi veya bir delege (ya da çoklu imzacılar).    |

<Callout type="warn">

Kaynak, yerel bir token account olmalıdır; diğer token account'lar
_rs`NonNativeNotSupported`_ döndürür. Bir delege, kendisine devredilen miktara
kadar çözme işlemi yapabilir.

</Callout>

## Lamport Nasıl Çözülür

Tam bakiyeyi çözmek için _ts`amount`_ parametresini atlayın ya da belirli sayıda
lamport çözmek için bir değer girin. Örnekte yapılandırılmış bir
[`@solana/kit`](/docs/clients/official/javascript#solana-kit) istemcisi
kullanılmaktadır — istemci kurulumu için
[Token Transferi](/docs/tokens/basics/transfer-tokens) bölümüne bakın.

```ts title="Unwrap wrapped SOL (Kit)"
import { getUnwrapLamportsInstruction } from "@solana-program/token";

// source: the native (wrapped SOL) token account to unwrap.
// destination: address that receives the unwrapped lamports.
// authority: a TransactionSigner (the account owner or a delegate).

// Unwrap a specific number of lamports.
const unwrapSome = getUnwrapLamportsInstruction({
  source,
  destination,
  authority,
  amount: 500_000_000n
});

// Or omit `amount` to unwrap the entire balance.
const unwrapAll = getUnwrapLamportsInstruction({
  source,
  destination,
  authority
});

// Send whichever instruction you need.
await client.sendTransaction([unwrapSome]);
```

Çoklu imza yetkisi için, ortak imzacıları _ts`multiSigners`_ ile iletin.
