SPL Token Temelleri bölümü, SPL Token'larla etkileşim için temel talimatları
kapsar ve orijinal Token Program ile Token Extensions Program (Token 2022)
arasında paylaşılan işlevselliğe odaklanır.

<Callout type="info">
  SPL Token'lara nasıl metadata ekleneceği hakkında daha fazla bilgi için
  [Metaplex dokümantasyonuna](https://developers.metaplex.com/token-metadata)
  bakın.
</Callout>
