SPL Token Temelleri bölümü, orijinal Token Program ve Token Extensions Program
(Token 2022) tarafından paylaşılan işlevselliğe odaklanarak, SPL Token'larla
etkileşim için temel talimatları kapsar.

<Callout type="info">
  SPL Token'lara metadata ekleme hakkında daha fazla bilgi için [Metaplex
  belgelerine](https://developers.metaplex.com/token-metadata) başvurun.
</Callout>
