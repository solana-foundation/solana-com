---

Bir Pay sunucusu kurarak API isteklerinde Pay.sh aracılığıyla ödeme kabul
edebilirsiniz. Pay'i şununla yükleyin:

```bash
brew install pay
```

veya

```bash
npm install -g @solana/pay
```

Kurulumu şununla doğrulayın:

```bash
pay --version
```

Araç zinciri adresinizi sizin için ayarlayacaktır. Cüzdanınızı kurmak ve
yüklemek için aşağıdakini çalıştırın:

```bash
pay setup
```

## Demo sunucusunu kurma

Sunucunuzu başlatarak ödeme kabul etmeye başlayabilirsiniz. Demo sunucusunu
şununla çalışırken görebilirsiniz:

```bash
pay server demo
```

Bu iki şey yapar. HTTP isteği aracılığıyla sorgulayabileceğiniz canlı bir Pay
sunucusu başlatır ve mevcut dizininizde bir `pay-demo.yml` dosyası oluşturur.

Tüm API rotaları `endpoints` YAML yapılandırması aracılığıyla yapılandırılır.
Ödeme kabul edebilmek için bir `metering:` özniteliği eklemeniz gerekir.
`/api/v1/reports/usage` rotası sizin için örnek olarak hazırlanmıştır:

```yaml
endpoints:
  - method: GET
    path: "api/v1/reports/usage"
    resource: "reports"
    description: "Usage report — flat fee, no splits."
    metering:
      dimensions:
        - direction: usage
          unit: requests
          scale: 1
          tiers:
            - price_usd: 0.01
```

Ödemelerin çalışıp çalışmadığını başka bir terminalde istemci olarak aşağıdaki
komutu çalıştırarak test edebilirsiniz:

```bash
pay --sandbox curl http://127.0.0.1:1402/api/v1/reports/usage
```

Fiyatlandırmanın nasıl tanımlanacağı hakkında daha fazla bilgi edinmek için
[Pay.sh Building with Pay > Pricing docs](https://pay.sh/docs/building-with-pay/pricing)
sayfasını ziyaret edin.

Pay.sh'yi üretim ortamında kurma hakkında daha fazla bilgi edinmek için
[Pay.sh Configuration docs](https://pay.sh/docs/toolchain/configuration)
sayfasını ziyaret edin.
