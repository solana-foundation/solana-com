---
title: Terk Edilmiş Delegasyonları ve Abonelikleri İptal Et
description:
  Abonelik Yetkisi ortadan kalkmış bir delegasyon veya abonelikten rent geri
  kazanmak için kılavuz.
---

Delegasyonlar ve abonelikler, iptal edildiğinde normalde kayıtlı ödeyiciye geri
dönen rent tutar. Bir sponsor hesabı finanse ettiyse ve kullanıcı daha sonra
Abonelik Yetkisini kapatır veya değiştirirse, hesap askıda kalır ve rent'i
sıkışır. `RevokeAbandonedDelegation` ve `RevokeAbandonedSubscription`, kayıtlı
ödeyicinin bunu geri almasına olanak tanır.

## Ne Zaman Geçerlidir

Yalnızca hesabın Abonelik Yetkisinin kanıtlanabilir şekilde sonlandığı
durumlarda izin verilir — kapatılmış veya kapatılıp yeniden oluşturulmuş
(dolayısıyla `init_id` artık eşleşmiyor). Aktif bir delegasyon veya abonelik bu
yolla hiçbir zaman kapatılamaz.

|                                    | `RevokeAbandonedDelegation`                                                                                                  | `RevokeAbandonedSubscription`                                                                                                   |
| ---------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| Kapatır                            | Sabit veya yinelenen delegasyon PDA'sı                                                                                       | Abonelik PDA'sı                                                                                                                 |
| İmzalayan                          | Kayıtlı ödeyici                                                                                                              | Kayıtlı ödeyici                                                                                                                 |
| Hesaplar                           | Delegasyon + kayıtlı yetki                                                                                                   | Abonelik + kayıtlı yetki + plan                                                                                                 |
| Uygun olduğu durum                 | Yetki kapatıldığında veya yeniden oluşturulduğunda (`init_id` uyuşmazlığı)                                                   | Aynı durum, planın mint'i için — mint bağlı plandan okunur, bu nedenle zorla kapatmak amacıyla ilgisiz bir yetki ikame edilemez |
| Bunun yerine olağan iptal kullanın | Delegasyon [süresi dolmuş veya tamamen harcanmışsa](/docs/payments/subscriptions/recurring-delegation#revoke-the-delegation) | Abonelik [iptal edilmiş ve süresi dolmuşsa](/docs/payments/subscriptions/subscription-plan#cancel-and-revoke)                   |

`RevokeAbandonedDelegation`, tamamen harcanmamış, hiç süresi dolmayan bir
delegasyon (`expiry_ts == 0`) için kayıtlı ödeyicinin tek kurtarma yoludur:
olağan iptal, ödeyici için yalnızca süre dolumunda veya sabit bir delegasyon söz
konusuysa tam harcama gerçekleştiğinde tetiklenir.

## Terk Edilmiş Bir Delegasyonu İptal Et

Kayıtlı ödeyici imzalar. Kullanılmayan delegasyon PDA'sını ve kayıtlı Abonelik
Yetkisini (zaten kapatılmış olabilir) iletin. Rent, ödeyiciye geri döner.

<Tabs items={["TypeScript", "Rust"]}>
  <Tab value="TypeScript">

    ```ts
    import { getRevokeAbandonedDelegationInstruction } from '@solana/subscriptions';

    // Build the instruction and send it in a transaction like any other.
    const revokeAbandonedIx = getRevokeAbandonedDelegationInstruction({
      payer: payerSigner,
      delegationAccount: delegationPda,
      subscriptionAuthority: subscriptionAuthorityPda,
    });
    ```

  </Tab>

  <Tab value="Rust">

    ```rust
    use solana_address::{address, Address};
    use subscriptions::generated::instructions::*;

    let payer: Address = address!("PAYER_THAT_FUNDED_RENT_HERE");
    let delegation_pda: Address = address!("DELEGATION_PDA_ADDRESS_HERE");
    let subscription_authority: Address = address!("RECORDED_SUBSCRIPTION_AUTHORITY_HERE");

    let revoke_abandoned_ix = RevokeAbandonedDelegationBuilder::new()
        .payer(payer)
        .delegation_account(delegation_pda)
        .subscription_authority(subscription_authority)
        .instruction();
    ```

  </Tab>
</Tabs>

## Terk Edilmiş Bir Aboneliği İptal Etme

Kayıtlı ödeyici imzalar. Terk edilmiş abonelik PDA'sını, kayıtlı Abonelik
Yetkilisini (zaten kapatılmış olabilir) ve aboneliğin ait olduğu planı iletin.
Rent, ödeyiciye iade edilir.

<Tabs items={["TypeScript", "Rust"]}>
  <Tab value="TypeScript">

    ```ts
    import { getRevokeAbandonedSubscriptionInstruction } from '@solana/subscriptions';

    // Build the instruction and send it in a transaction like any other.
    const revokeAbandonedIx = getRevokeAbandonedSubscriptionInstruction({
      payer: payerSigner,
      subscriptionAccount: subscriptionPda,
      subscriptionAuthority: subscriptionAuthorityPda,
      planPda,
    });
    ```

  </Tab>

  <Tab value="Rust">

    ```rust
    use solana_address::{address, Address};
    use subscriptions::generated::instructions::*;

    let payer: Address = address!("PAYER_THAT_FUNDED_RENT_HERE");
    let subscription_pda: Address = address!("SUBSCRIPTION_PDA_ADDRESS_HERE");
    let subscription_authority: Address = address!("RECORDED_SUBSCRIPTION_AUTHORITY_HERE");
    let plan_pda: Address = address!("PLAN_PDA_ADDRESS_HERE");

    let revoke_abandoned_ix = RevokeAbandonedSubscriptionBuilder::new()
        .payer(payer)
        .subscription_account(subscription_pda)
        .subscription_authority(subscription_authority)
        .plan_pda(plan_pda)
        .instruction();
    ```

  </Tab>
</Tabs>

## Notlar

- İmzalayan, delege eden, delege edilen veya abone değil, kayıtlı ödeyicidir.
- Hesap için Abonelik Yetkilisi hâlâ aktifse her iki talimat da başarısız olur —
  bunun yerine normal iptal işlemini kullanın.
- `RevokeAbandonedDelegation` hem sabit hem de yinelenen delegasyonlar için
  çalışır.
