## AI geliştirme araçları

[Solana'da AI araçlarıyla nasıl başlanır](/developers/guides/getstarted/intro-to-ai)
kılavuzumuzla başlayın.

### Solana MCP

[Solana MCP](https://mcp.solana.com/) (Machine Conversation Protocol), Solana
bilgisini doğrudan Claude, Cursor ve Windsurf gibi AI destekli kodlama
araçlarına entegre eder. Solana dokümanları, Anchor framework, program örnekleri
ve Stack Exchange soru-cevap üzerinde eğitilmiştir - ödeme entegrasyonlarını
daha hızlı oluşturmak için kullanışlıdır.

### AI ajan çerçeveleri

AI ajanları Solana üzerinde ödeme işlemlerini özerk olarak yürütebilir. Bu
çerçeveler, ödemeleri işleyen, işlemleri izleyen ve ödeme iş akışlarını
otomatikleştiren ajanlar oluşturmayı sağlar:

| Çerçeve          | Açıklama                                                                           | Bağlantılar                                                                                     |
| ---------------- | ---------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------- |
| Solana Agent Kit | Ödeme yapabilen ajanlar oluşturmak için araçlar, LangChain ve Vercel AI ile uyumlu | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Dokümanlar](https://kit.sendai.fun/) |
| Send AI Skills   | Solana üzerinde ajan oluşturmak için önceden hazırlanmış beceriler koleksiyonu     | [GitHub](https://github.com/sendaifun/skills)                                                   |
| Eliza Framework  | X, Telegram, Discord botları için Solana eklentili çok platformlu ajan çerçevesi   | [GitHub](https://github.com/elizaOS/eliza) • [Dokümanlar](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Solana ödeme eylemleri içeren zincirler arası araç seti                            | [GitHub](https://github.com/goat-sdk/goat)                                                      |

### Yapay zeka program geliştirme
