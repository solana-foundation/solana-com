Yerel Solana geliştirme ortamınızı tek komutla kurmak için hızlı bir kurulum.
Windows (WSL), Linux ve Mac'te Rust, Solana CLI ve Anchor Framework'ü kurun.
(İsterseniz,
[her bağımlılığı ayrı ayrı kurabilirsiniz](/docs/intro/installation/dependencies).)

## Ön koşullar

<Accordions>
<Accordion title="Windows">

Windows'ta Solana programları geliştirmek için
[Windows Subsystem for Linux](https://learn.microsoft.com/en-us/windows/wsl/install)
(WSL) kullanmanız **zorunludur**. WSL'yi kurduktan sonra, diğer tüm
bağımlılıklar Linux terminali üzerinden kurulacaktır.

WSL'yi kurmak için Windows PowerShell'de aşağıdaki komutu çalıştırın:

```terminal
$ wsl --install
```

Kurulum işlemi sırasında varsayılan bir kullanıcı hesabı oluşturmanız
istenecektir.

![WSL Kurulumu](/assets/docs/intro/installation/wsl-install.png)

Varsayılan olarak WSL, Ubuntu'yu kurar. Arama çubuğunda "Ubuntu" arayarak bir
Linux terminali açabilirsiniz.

![WSL Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-search.png)

Ubuntu terminaliniz aşağıdaki resme benziyorsa, `ctrl + v` (yapıştırma klavye
kısayolu) terminalinde çalışmayan bir sorunla karşılaşabilirsiniz.

![Ubuntu Terminali](/assets/docs/intro/installation/wsl-ubuntu-terminal-1.png)

Bu sorunla karşılaşırsanız, Arama çubuğunda "Terminal" arayarak Windows
Terminal'i açın.

![Windows Terminal](/assets/docs/intro/installation/wsl-windows-terminal.png)

Ardından, Windows Terminal'i kapatın ve Ubuntu'yu tekrar arayarak bir Linux
terminali açın. Terminal artık aşağıdaki resme benzemelidir ve `ctrl + v`
(yapıştırma klavye kısayolu) çalışmalıdır.

![Ubuntu Terminali](/assets/docs/intro/installation/wsl-ubuntu-terminal-2.png)

VS Code kullanıyorsanız,
[WSL uzantısı](https://code.visualstudio.com/docs/remote/wsl-tutorial) WSL ve VS
Code'u birlikte kullanmanıza olanak tanır.

![VS Code'da WSL Kurulumu](/assets/docs/intro/installation/wsl-vscode.png)

VS Code durum çubuğunda aşağıdakileri göreceksiniz:

![WSL: Ubuntu](/assets/docs/intro/installation/wsl-vscode-ubuntu.png)

WSL kurulumunu tamamladıktan sonra şunlardan birini yapmaya hazırsınız:

- Ubuntu (Linux) terminalini kullanarak [hızlı kurulumu](#installation)
  tamamlayın
- Aşağıda açıklandığı gibi
  [Rust](/docs/intro/installation/dependencies#install-rust),
  [Solana CLI](/docs/intro/installation/dependencies#install-solana-cli) ve
  [Anchor CLI](/docs/intro/installation/dependencies#install-anchor-cli)'yi ayrı
  ayrı kurun

</Accordion>
<Accordion title="Linux">

<Tabs items={[".deb", ".rpm"]}>
<Tab value=".deb" >
Debian veya Ubuntu ya da Mint gibi Debian türevi alternatiflerdeyseniz, aşağıdaki adımları izleyin:

1. Tüm paketlerin güncel olduğundan emin olmak için paket listelerini
   güncelleyin:

```terminal
$ sudo apt-get update
```

2. Aşağıdaki bağımlılıkları kurun:

```terminal
$ sudo apt-get install -y \
    build-essential \
    pkg-config \
    libudev-dev llvm libclang-dev \
    protobuf-compiler libssl-dev
```

<Callout>
`protobuf-compiler` kurulumu sırasında aşağıdaki hatayla karşılaşırsanız, önce `sudo apt-get update` komutunu çalıştırdığınızdan emin olun

```
Package protobuf-compiler is not available, but is referred to by another package.
This may mean that the package is missing, has been obsoleted, or
is only available from another source
```

</Callout>

  </Tab>

    <Tab value=".rpm" >
    Fedora veya Red Hat Linux gibi RHEL türevleri için aşağıdaki adımları izleyin.

Paketlerin daha yeni sürümlerini almak için komutu çalıştırarak paket
listelerini güncelleyin.

```terminal
$ sudo dnf upgrade --refresh
```

Ardından, aşağıdaki bağımlılıkları kurun:

```terminal
$ sudo dnf group install c-development development-tools \
&& sudo dnf install pkg-config openssl-devel
```

## Kurulum

1. Gerekli tüm bağımlılıkları kurmak için terminalinize aşağıdaki komutu girin:

```terminal
$ curl --proto '=https' --tlsv1.2 -sSfL https://solana-install.solana.workers.dev | bash
```

Başarılı bir kurulum aşağıdaki gibi bir çıktı döndürecektir:

```txt title="Example output"
Installed Versions:
Rust: rustc 1.91.1 (ed61e7d7e 2025-11-07)
Solana CLI: solana-cli 3.0.10 (src:96c3a851; feat:3604001754, client:Agave)
Anchor CLI: anchor-cli 0.32.1
Surfpool CLI: surfpool 0.12.0
Node.js: v24.10.0
Yarn: 1.22.1
```

2. Kurulu her bağımlılığın sürümünü kontrol ederek başarılı bir kurulumu
   doğrulayın.

```terminal
$ rustc --version && solana --version && anchor --version && surfpool --version && node --version && yarn --version
```

<Callout>
  Hızlı kurulum komutu başarısız olursa, her bağımlılığı ayrı ayrı kurmak için
  aşağıdaki [Bağımlılıkları Kur](/docs/intro/installation/dependencies) bölümüne
  bakın.
</Callout>
