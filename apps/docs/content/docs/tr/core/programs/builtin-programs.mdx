  Program), konsensüs (Vote, Stake), işlem optimizasyonu (Compute Budget,

## System Program

System Program (`11111111111111111111111111111111`), yeni hesaplar oluşturabilen
tek programdır. Tüm yeni hesaplar başlangıçta System Program'a aittir, ancak
sahiplik genellikle oluşturma sırasında yeniden atanır. System Program, talimat
başına
[`DEFAULT_COMPUTE_UNITS`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L282)
(150 CU) tüketir.

## Yükleyici programlar

Her program bir yükleyici programa aittir. Yükleyiciler, sahip oldukları
programların dağıtımını, güncellemesini ve yürütülmesini yönetir. Yükleyicilerin
tam listesi ve adresleri için
[Yükleyici Programlar](/docs/core/programs/program-deployment#loader-programs)
bölümüne bakın.
