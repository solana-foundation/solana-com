## Solana Kit

`@solana/kit`, Solana üzerinde geliştirme yapmak için önerilen TypeScript
SDK'sıdır. Birleştirilebilir eklentilerden bir istemci oluşturursunuz, ihtiyaç
duyduğunuzda React bağlamalarını eklersiniz ve çağırdığınız zincir üstü
programlar için oluşturulmuş program istemcilerini dahil edersiniz.

| Paket                     | Açıklama                                             | GitHub                                                                                 |
| ------------------------- | ---------------------------------------------------- | -------------------------------------------------------------------------------------- |
| @solana/kit               | Çekirdek SDK: RPC, işlemler, codec'ler, imzalayanlar | [Kaynak](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | React bağlamaları: `ClientProvider` ve hook'lar      | [Kaynak](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | RPC bağlantısı + işlem gönderimi                     | [Kaynak](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | Ücret ödeyen, kimlik ve imzalayan eklentileri        | [Kaynak](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Wallet Standard bağlantısı + React hook'ları         | [Kaynak](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

`@solana/kit` kullanmalıdır; bkz. [Kit'e Geçiş](/docs/frontend/web3-compat).
