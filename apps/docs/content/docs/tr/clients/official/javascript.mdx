## Solana Client

`@solana/client`, RPC, cüzdanlar, işlemler ve abonelikleri tek bir depoda
toplayan başsız Solana çalışma zamanıdır.

| Paket          | Açıklama                      | GitHub                                                                                 |
| -------------- | ----------------------------- | -------------------------------------------------------------------------------------- |
| @solana/client | Başsız istemci çalışma zamanı | [Kaynak](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [Kılavuz: @solana/client genel bakış](/docs/frontend/client)

## Solana React Hooks

`@solana/react-hooks`, `@solana/client` üzerine bir React sağlayıcı ve hook'lar
ekleyerek aynı çalışma zamanını ve önbelleği yeniden kullanır.

| Paket               | Açıklama                                             | GitHub                                                                                      |
| ------------------- | ---------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| @solana/react-hooks | Kullanıcı arayüzleri için React sağlayıcı + hook'lar | [Kaynak](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [Kılavuz: @solana/react-hooks](/docs/frontend/react-hooks)
