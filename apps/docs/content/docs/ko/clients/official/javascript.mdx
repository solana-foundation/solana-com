## Solana Client

`@solana/client`는 RPC, 지갑, 트랜잭션, 구독을 하나의 스토어로 번들링하는
헤드리스 Solana 런타임입니다.

| 패키지         | 설명                       | GitHub                                                                               |
| -------------- | -------------------------- | ------------------------------------------------------------------------------------ |
| @solana/client | 헤드리스 클라이언트 런타임 | [소스](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [가이드: @solana/client 개요](/docs/frontend/client)

## Solana React Hooks

`@solana/react-hooks`는 `@solana/client` 위에 React 프로바이더와 훅을
레이어링하여 동일한 런타임과 캐시를 재사용합니다.

| 패키지              | 설명                       | GitHub                                                                                    |
| ------------------- | -------------------------- | ----------------------------------------------------------------------------------------- |
| @solana/react-hooks | UI용 React 프로바이더 + 훅 | [소스](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [가이드: @solana/react-hooks](/docs/frontend/react-hooks)
