## Solana Kit

`@solana/kit`는 Solana 위에서 개발하기 위한 공식 권장 TypeScript SDK입니다.
컴포저블 플러그인으로 클라이언트를 조립하고, 필요할 때 React 바인딩을 추가하며,
호출하는 온체인 프로그램에 맞는 생성된 프로그램 클라이언트를 가져올 수 있습니다.

| 패키지                    | 설명                                  | GitHub                                                                               |
| ------------------------- | ------------------------------------- | ------------------------------------------------------------------------------------ |
| @solana/kit               | 핵심 SDK: RPC, 트랜잭션, 코덱, 서명자 | [소스](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | React 바인딩: `ClientProvider` 및 훅  | [소스](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | RPC 연결 + 트랜잭션 전송              | [소스](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | 수수료 납부자, 신원, 서명자 플러그인  | [소스](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Wallet Standard 연결 + React 훅       | [소스](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

### 프로그램 클라이언트

`tokenProgram()` → `client.token`).

| @solana-program/token          | Token Program과 상호작용          | [소스](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Token-2022 program과 상호작용     | [소스](https://github.com/solana-program/token-2022/tree/main/clients/js)     |

| @solana-program/compute-budget | Compute Budget program과 상호작용 | [소스](https://github.com/solana-program/compute-budget/tree/main/clients/js) |

## Solana Web3.js

`@solana/web3.js`는 Solana의 레거시 TypeScript SDK입니다. 새로운 앱은
`@solana/kit`를 사용해야 합니다.
[Kit으로 마이그레이션](/docs/frontend/web3-compat)을 참조하세요.

| 패키지            | 설명                                                      | GitHub                                                                           |
| ----------------- | --------------------------------------------------------- | -------------------------------------------------------------------------------- |
| @solana/web3.js   | 핵심 SDK                                                  | [소스](https://github.com/solana-foundation/solana-web3.js)                      |
| @solana/spl-token | Token, Token-2022 및 Associated Token programs와 상호작용 | [소스](https://github.com/solana-program/token-2022/tree/main/clients/js-legacy) |
