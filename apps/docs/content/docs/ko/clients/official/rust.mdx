## 프로그램 크레이트

온체인 프로그램을 빌드하려면 `solana-program`를 사용하세요.

- [예제: Solana 프로그램 빌드](/docs/programs/rust)

| 크레이트       | 설명                 | 문서                                   | GitHub                                                             |
| -------------- | -------------------- | -------------------------------------- | ------------------------------------------------------------------ |
| solana-program | 온체인 프로그램 빌드 | [보기](https://docs.rs/solana-program) | [소스](https://github.com/anza-xyz/solana-sdk/tree/master/program) |

Pinocchio는 Rust로 Solana 프로그램을 만들기 위한 제로 디펜던시 라이브러리이며,
온체인 프로그램 작성을 위해 `solana-program`의 대체재로 사용할 수 있습니다.

| 크레이트  | 설명                          | 문서                              | GitHub                                        |
| --------- | ----------------------------- | --------------------------------- | --------------------------------------------- |
| pinocchio | 제로 디펜던시 온체인 프로그램 | [보기](https://docs.rs/pinocchio) | [소스](https://github.com/anza-xyz/pinocchio) |

<Callout type="warn">
  Pinocchio는 아직 개발 중이며 가까운 미래에 호환성을 깨는 변경 사항이 있을 수
  있습니다.
</Callout>

Pinocchio는 Solana 프로그램과 상호작용하기 위한 프로그램별 크레이트를
포함합니다:

| 크레이트                           | 설명                                | 문서                                                        | GitHub                                                                                    |
| ---------------------------------- | ----------------------------------- | ----------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| pinocchio-system                   | System Program과 상호작용           | [보기](https://docs.rs/pinocchio-system/)                   | [소스](https://github.com/anza-xyz/pinocchio/tree/main/programs/system)                   |
| pinocchio-token                    | Token Program과 상호작용            | [보기](https://docs.rs/pinocchio-token/)                    | [소스](https://github.com/anza-xyz/pinocchio/tree/main/programs/token)                    |
| pinocchio-token-2022               | Token-2022 Program과 상호작용       | [보기](https://docs.rs/pinocchio-token-2022/)               | [소스](https://github.com/anza-xyz/pinocchio/tree/main/programs/token-2022)               |
| pinocchio-associated-token-account | Associated Token Program과 상호작용 | [보기](https://docs.rs/pinocchio-associated-token-account/) | [소스](https://github.com/anza-xyz/pinocchio/tree/main/programs/associated-token-account) |
| pinocchio-memo                     | Memo Program과 상호작용             | [보기](https://docs.rs/pinocchio-memo/)                     | [소스](https://github.com/anza-xyz/pinocchio/tree/main/programs/memo)                     |
