SPL 토큰 기본 섹션에서는 SPL 토큰과 상호작용하기 위한 기본 명령어를 다루며,
원래의 Token Program과 Token Extensions Program(Token 2022) 모두에서 공유되는
기능에 중점을 둡니다.

<Callout type="info">
  SPL 토큰에 메타데이터를 추가하는 방법에 대한 자세한 내용은 [Metaplex
  문서](https://developers.metaplex.com/token-metadata)를 참조하세요.
</Callout>
