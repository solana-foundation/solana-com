프로그램에서도 호출되며 lamport 회수를 위한 명령어도 포함합니다:

  <Card

    href="/docs/tokens/advanced/cpi"

    온체인 프로그램에서 Token Program을 호출하세요 — 토큰 전송, 민트, 소각,
    PDA 권한으로 서명, 그리고 여러 명령어를 단일 CPI로 일괄 처리할 수 있습니다.

<Callout type="info">

_rs`WithdrawExcessLamports`_ 및 *rs`UnwrapLamports`*는
[`@solana-program/token`](https://www.npmjs.com/package/@solana-program/token)
클라이언트를 통해 사용할 수 있습니다. CPI를 통한 Token Program 호출은 온체인
프로그램에 적용됩니다.

</Callout>

## Lamport 회수

*rs`WithdrawExcessLamports`*는 token account, 민트, 멀티시그 계정에서 임대 면제
최솟값을 초과하는 SOL을 회수합니다. *rs`UnwrapLamports`*는 네이티브 token
account의 래핑된 SOL을 lamport로 다시 변환합니다.
