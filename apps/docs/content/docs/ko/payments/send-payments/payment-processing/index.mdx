---
title: 결제 처리
description: 가스리스 트랜잭션과 수수료 추상화로 대규모 결제를 처리하세요
---

급여, 판매자 지급, 로열티 보상과 같은 대규모 결제 처리는 개별 전송을 보내는 것
이상을 필요로 합니다.

많은 결제를 효율적으로 처리하려면:

1. **명령어 일괄 처리**로 단일 트랜잭션에 묶어 수수료와 지연 시간을 줄입니다
2. **트랜잭션 일괄 처리**로 네트워크 전체에서 처리량을 병렬화합니다
3. **수수료 추상화**로 사용자가 SOL을 보유할 필요가 없도록 합니다
