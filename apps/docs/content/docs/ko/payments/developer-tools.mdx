## AI 개발 도구

[Solana에서 AI 도구 시작하기](/developers/guides/getstarted/intro-to-ai)
가이드를 통해 시작하세요.

### Solana MCP

[Solana MCP](https://mcp.solana.com/) (Machine Conversation Protocol)는 Claude,
Cursor, Windsurf와 같은 AI 지원 코딩 도구에 Solana 지식을 직접 통합합니다.
Solana 문서, Anchor 프레임워크, 프로그램 예제, Stack Exchange Q&A로 학습되어
결제 통합을 더 빠르게 구축하는 데 유용합니다.

### AI 에이전트 프레임워크

AI 에이전트는 Solana에서 결제 트랜잭션을 자율적으로 실행할 수 있습니다. 이러한
프레임워크를 통해 결제를 처리하고, 트랜잭션을 모니터링하며, 결제 워크플로우를
자동화하는 에이전트를 구축할 수 있습니다:

| 프레임워크       | 설명                                                                                    | 링크                                                                                      |
| ---------------- | --------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | LangChain 및 Vercel AI와 호환되는 결제 가능 에이전트 구축 도구                          | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [문서](https://kit.sendai.fun/) |
| Send AI Skills   | Solana에서 에이전트를 구축하기 위한 사전 구축된 스킬 모음                               | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | X, Telegram, Discord 봇을 위한 Solana 플러그인이 포함된 멀티 플랫폼 에이전트 프레임워크 | [GitHub](https://github.com/elizaOS/eliza) • [문서](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Solana 결제 액션을 포함한 크로스체인 툴킷                                               | [GitHub](https://github.com/goat-sdk/goat)                                                |

### AI 프로그램 개발

[Codigo](https://codigo.ai/)는 엔드투엔드 Solana AI 개발 플랫폼입니다.
