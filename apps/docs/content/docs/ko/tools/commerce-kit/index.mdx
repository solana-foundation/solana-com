**Commerce Kit은 Solana에서 커머스 경험을 구축하기 위한 도구 모음입니다.**
드롭인 결제 버튼부터 헤드리스 기본 요소까지, Commerce Kit은 암호화폐 결제를
수락하는 데 필요한 모든 것을 제공합니다.

## Commerce Kit을 선택해야 하는 이유

- **드롭인 컴포넌트**: 지갑 연결, 토큰 선택 및 트랜잭션 처리 기능이 포함된 사전
  구축된 PaymentButton
- **프레임워크 독립적**: 모든 JavaScript 프레임워크에서 헤드리스 기본 요소 사용
  가능
- **Solana Pay**: QR 코드 생성을 포함한 완전한 Solana Pay 지원
- **타입 안전성**: 모든 패키지에 걸친 완전한 TypeScript 지원
- **지갑 표준**: 모든 Wallet Standard 지갑과 호환 (Phantom, Solflare, Backpack
  등)

## 빠른 시작

가장 빠르게 시작하는 방법은 `PaymentButton` 컴포넌트를 사용하는 것입니다:

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## 설치

필요한 Commerce Kit 패키지를 선택하세요:

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
