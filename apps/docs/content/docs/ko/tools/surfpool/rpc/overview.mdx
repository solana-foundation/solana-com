Surfnet은 Surfpool의 로컬 Solana 네트워크 시뮬레이터로, 전체 validator 노드의
오버헤드 없이 Solana 블록체인의 동작을 그대로 재현하는 강력한 테스트 환경을
개발자에게 제공하도록 설계되었습니다.

## 시작하기

Surfpool CLI를 사용하여 Surfnet 인스턴스를 시작하세요:

```bash
surfpool start
```

이 명령은 `http://localhost:8899`에서 모든 RPC 엔드포인트가 활성화된 로컬
Surfnet 인스턴스를 시작합니다.

## 주요 기능

- **완전한 RPC 호환성**: Solana RPC API 전체 지원
- **빠른 반복 개발**: 블록 확인을 기다릴 필요 없음
- **상태 조작**: 블록체인 상태를 직접 수정하는 치트코드
- **메인넷 포크**: 실제 계정 및 프로그램으로 테스트하기 위한 메인넷 포크
- **Geyser 플러그인 지원**: 실시간 데이터 스트리밍을 위한 Geyser 플러그인 연결
