## ripemd160

`ripemd160`는 값의 Ripemd160 해시를 계산합니다.

### 입력

| 이름    | 타입                    | 설명                    |
| ------- | ----------------------- | ----------------------- |
| `value` | buffer \| array[buffer] | 해시할 16진수 인코딩 값 |

### 출력

| 이름    | 타입   | 설명        |
| ------- | ------ | ----------- |
| `value` | string | 해시된 결과 |

```hcl
output "hashed_data" {
    value = ripemd160(encode_hex("hello, world"))
}
// > hashed_data: 0XA3201F82FCA034E46D10CD7B27E174976E241DA2
```

## sha256

`sha256`는 값의 sha256 해시를 계산합니다.

### 입력

| 이름    | 타입                    | 설명                    |
| ------- | ----------------------- | ----------------------- |
| `value` | buffer \| array[buffer] | 해시할 16진수 인코딩 값 |

### 출력

| 이름    | 타입   | 설명        |
| ------- | ------ | ----------- |
| `value` | string | 해시된 결과 |

```hcl
output "hashed_data" {
    value = sha256(encode_hex("hello, world"))
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```

## keccak256

`std::keccak256`는 값의 keccak256 해시를 계산합니다.

### 입력

| 이름    | 타입   | 설명             |
| ------- | ------ | ---------------- |
| `value` | string | 해시할 문자열 값 |

### 출력

| 이름    | 타입   | 설명        |
| ------- | ------ | ----------- |
| `value` | string | 해시된 결과 |

```hcl
output "hashed_data" {
    value = keccak256("hello, world")
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```
