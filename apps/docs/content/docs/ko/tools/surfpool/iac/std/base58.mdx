## encode_base58

`encode_base58`는 버퍼 또는 문자열을 base58 문자열로 인코딩합니다.

### 입력

| 이름    | 유형                 | 설명                                                                                                                         |
| ------- | -------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| `value` | 버퍼, 문자열, 애드온 | 인코딩할 버퍼 또는 문자열입니다. '0x'로 시작하는 문자열은 16진수로 디코딩되며, 그렇지 않으면 원시 UTF-8 바이트가 사용됩니다. |

### 출력

| 이름    | 유형   | 설명                            |
| ------- | ------ | ------------------------------- |
| `value` | 문자열 | base58 문자열로 인코딩된 입력값 |

```hcl
output "encoded" {
    value = encode_base58("0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8")
}
// > encoded: CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF
```

## decode_base58

`decode_base58`는 base58로 인코딩된 문자열을 디코딩하여 결과를 버퍼로
반환합니다.

### 입력

| 이름            | 유형   | 설명                   |
| --------------- | ------ | ---------------------- |
| `base58_string` | 문자열 | 디코딩할 base58 문자열 |

### 출력

| 이름    | 유형 | 설명                                 |
| ------- | ---- | ------------------------------------ |
| `value` | 버퍼 | 버퍼로 반환된 디코딩된 base58 문자열 |

```hcl
output "decoded" {
    value = decode_base58("CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF")
}
// > decoded: 0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8
```
