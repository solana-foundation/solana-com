## jq

`jq` 함수는 JSON 데이터의 슬라이싱, 필터링 및 매핑을 지원합니다. 자세한 내용은
[jq 문서](https://jqlang.github.io/jq/manual/)를 참조하세요.

### 입력

| 이름           | 유형             | 설명                                                           |
| -------------- | ---------------- | -------------------------------------------------------------- |
| `decoded_json` | string \| object | JSON 객체                                                      |
| `query`        | string           | JSON 쿼리. [jq 문서](https://jqlang.github.io/jq/manual/) 참조 |

### 출력

| 이름    | 유형          | 설명             |
| ------- | ------------- | ---------------- |
| `value` | array[string] | `jq` 쿼리의 결과 |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
