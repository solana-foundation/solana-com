## assert_eq

`assert_eq`는 두 값이 동일한지 확인합니다.

### 입력

| 이름    | 타입 | 설명         |
| ------- | ---- | ------------ |
| `left`  | any  | 비교할 값    |
| `right` | any  | 비교 대상 값 |

### 출력

| 이름    | 타입                  | 설명        |
| ------- | --------------------- | ----------- |
| `value` | addon(std::assertion) | 어서션 결과 |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```

## assert_gt

`assert_gt`는 왼쪽 값이 오른쪽 값보다 큰지 확인합니다.

### 입력

| 이름    | 타입             | 설명                              |
| ------- | ---------------- | --------------------------------- |
| `left`  | integer \| float | 비교할 정수 또는 부동소수점 값    |
| `right` | integer \| float | 비교 대상 정수 또는 부동소수점 값 |

### 출력

| 이름    | 타입                  | 설명        |
| ------- | --------------------- | ----------- |
| `value` | addon(std::assertion) | 어서션 결과 |

```hcl
output "assertion" {
    value = std::assert_gt(action.example.result, 1)
}
```
