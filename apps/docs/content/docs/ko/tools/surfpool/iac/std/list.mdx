## index

`index`는 지정된 인덱스에서 리스트의 항목을 가져옵니다.

### 입력

| 이름    | 타입                                             | 설명                 |
| ------- | ------------------------------------------------ | -------------------- |
| `list`  | array[string] \| array[integer] \| array[buffer] | 항목을 가져올 리스트 |
| `index` | integer                                          | 가져올 항목의 인덱스 |

### 출력

| 이름    | 타입   | 설명                               |
| ------- | ------ | ---------------------------------- |
| `value` | string | 지정된 인덱스에 있는 리스트의 항목 |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
