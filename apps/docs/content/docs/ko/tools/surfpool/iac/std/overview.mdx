표준 라이브러리는 어떤 블록체인 애드온이 활성화되어 있는지에 관계없이 모든
Runbook에서 사용할 수 있는 핵심 함수와 액션을 제공합니다.

## 함수

표준 라이브러리에는 다음을 위한 유틸리티 함수가 포함되어 있습니다:

- **연산자**: 수학적 연산 및 비교
- **Hash**: 해시 함수 (SHA256, Keccak256 등)
- **Hex**: 16진수 인코딩 및 디코딩
- **Crypto**: 암호화 연산
- **JSON**: JSON 인코딩 및 디코딩
- **List**: 리스트 조작 함수
- **Assertions**: 동등성 및 조건 어설션
- **Base64**: Base64 인코딩 및 디코딩

[모든 함수 보기](/docs/tools/surfpool/iac/std/functions)

## 액션

표준 라이브러리에는 다음을 위한 액션이 포함되어 있습니다:

- **HTTP**: HTTP 요청 보내기
- **File Operations**: 파일 읽기 및 쓰기

[모든 액션 보기](/docs/tools/surfpool/iac/std/actions)
