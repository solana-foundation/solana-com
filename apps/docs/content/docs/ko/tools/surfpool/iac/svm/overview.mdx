title: SVM 애드온 개요
description: Solana 및 SVM 호환 블록체인용 애드온 (베타)
---

SVM `txtx` 플러그인을 사용하면 Solana 및 SVM 호환 블록체인과 상호작용하는
Runbook을 구축할 수 있습니다. 이 플러그인은 Anchor 프로그램을 배포하고 프로그램
IDL에 따라 명령어 호출을 인코딩할 수 있는 유틸리티 함수를 제공합니다. 액션을
사용하여 유효한 전송, 프로그램 호출 및 프로그램 배포 트랜잭션을 생성할 수
있으며, 니모닉 구문, 시크릿 키 또는 브라우저 서명자를 통해 서명할 수 있습니다.

## 빠른 시작

SVM 애드온을 사용하려면 Runbook에서 선언하세요:

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## 기능

- **프로그램 배포**: Anchor 및 네이티브 Solana 프로그램 배포
- **명령어 빌딩**: 프로그램 IDL을 사용한 명령어 호출 인코딩
- **트랜잭션 서명**: 니모닉, 시크릿 키 및 웹 지갑 지원
- **계정 관리**: Solana 계정 조회 및 관리
