## and_bool

`and_bool`는 좌변과 우변 인수의 이진 AND를 반환합니다.

### 입력

| 이름  | 타입 | 설명      |
| ----- | ---- | --------- |
| `lhs` | bool | 불리언 값 |
| `rhs` | bool | 불리언 값 |

### 출력

| 이름    | 타입 | 설명                            |
| ------- | ---- | ------------------------------- |
| `value` | bool | 두 인수 간의 이진 AND 연산 결과 |

```hcl
output "my_bool" {
  value = false && true
}
// > my_bool: false
```

## div

`div`는 좌변 인수를 우변 인수로 나눈 정수 나눗셈을 반환하며, 나머지는 가장
가까운 정수로 내림합니다.

### 입력

| 이름  | 타입    | 설명   |
| ----- | ------- | ------ |
| `lhs` | integer | 피제수 |
| `rhs` | integer | 제수   |

### 출력

| 이름    | 타입    | 설명                      |
| ------- | ------- | ------------------------- |
| `value` | integer | 피제수를 제수로 나눈 결과 |

```hcl
output "my_int" {
  value = 11 / -3
}
// > my_int: -3
```

## eq

`eq`는 좌변과 우변 인수가 같으면 `true`를 반환하고, 같지 않으면 `false`를
반환합니다.

### 입력

| 이름  | 타입 | 설명      |
| ----- | ---- | --------- |
| `lhs` | any  | 임의의 값 |
| `rhs` | any  | 임의의 값 |

### 출력

| 이름    | 타입 | 설명                          |
| ------- | ---- | ----------------------------- |
| `value` | bool | 두 입력 간의 동등성 검사 결과 |

```hcl
output "is_eq" {
  value = "arg" == "arg"
}
// > is_eq: true
```

## gt

`gt`는 좌변 인수가 우변 인수보다 크면 `true`를 반환합니다.

### 입력

| 이름  | 타입                       | 설명           |
| ----- | -------------------------- | -------------- |
| `lhs` | integer \| float \| string | 비교 가능한 값 |
| `rhs` | integer \| float \| string | 비교 가능한 값 |

### 출력

| 이름    | 타입 | 설명                                       |
| ------- | ---- | ------------------------------------------ |
| `value` | bool | 좌변 인수가 우변 인수보다 큰지 확인한 결과 |

```hcl
output "is_gt" {
  value = 2 > 1
}
// > is_gt: true
```

## neq

`neq`는 왼쪽 인수와 오른쪽 인수가 같지 않으면 `true`를 반환하고, 그렇지 않으면
`false`를 반환합니다.

### 입력

| 이름  | 유형 | 설명         |
| ----- | ---- | ------------ |
| `lhs` | any  | 비교할 값    |
| `rhs` | any  | 비교 대상 값 |

### 출력

| 이름    | 유형 | 설명                             |
| ------- | ---- | -------------------------------- |
| `value` | bool | 두 입력 간의 부정 동등 검사 결과 |

```hcl
output "is_neq" {
  value = "arg" != "arg"
}
// > is_neq: false
```

| 이름  | 유형    | 설명   |

| `rhs` | integer | 감수   |

### 출력

| 이름    | 유형    | 설명             |
| ------- | ------- | ---------------- |
| `value` | integer | 뺄셈 연산의 결과 |

```hcl
output "my_integer" {
  value = 10 - 6
}
```

| ------- | ------- | -------------------- |
| `value` | integer | 나눗셈 연산의 나머지 |
