Surfpool은 `solana-test-validator`를 대체하는 드롭인 솔루션으로, **Solana** 기반
개발자들에게 최고의 개발 경험을 제공하기 위해 특별히 설계되었습니다.

핵심 기능은 개발자들이 실시간으로 가져온 메인넷 계정을 사용하여 로컬에서
프로그램을 시뮬레이션할 수 있도록 하는 것입니다.

또한 Surfpool은 Anchor 또는 Pinocchio 기반 프로젝트에 코드형
인프라(Infrastructure as Code)를 원활하게 도입하여, 프라이빗 또는 퍼블릭 Solana
네트워크에 재현 가능하고 감사 가능하며 안전한 배포를 지원합니다.
