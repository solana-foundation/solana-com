_GitHub에서 사용 가능:
[`v2.2.0-beta.7`](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.7)_

Kora v2.2.0-beta는 Jito 번들 지원, 새로운 RPC 메서드, Docker 이미지, 향상된 보안
기능 및 SDK 개선 사항을 제공합니다.

> **프리릴리스 버전입니다.** 여기에 문서화된 기능은 `v2.2.0-beta.7`에서 사용
> 가능하며 정식 릴리스 전에 변경될 수 있습니다. 정식 문서는
> [Kora 메인 문서](/docs/tools/kora)를 참조하세요.

## 주요 기능

### Jito 번들 지원

Jito의 블록 엔진을 통해 최대 5개의 트랜잭션을 원자적으로 제출할 수 있습니다.
새로운 RPC 메서드:

- [`signBundle`](/docs/tools/kora/beta/bundle-methods#signbundle) — 브로드캐스트
  없이 번들에 서명
- [`signAndSendBundle`](/docs/tools/kora/beta/bundle-methods#signandsendbundle)
  — Jito에 서명 및 제출
- [`estimateBundleFee`](/docs/tools/kora/beta/bundle-methods#estimatebundlefee)
  — 총 번들 수수료 추정

전체 안내는 [Jito 번들 가이드](/docs/tools/kora/beta/jito-guide)를 참조하세요.

### Docker 이미지

GitHub Container Registry에 게시된 공식 멀티플랫폼 이미지(amd64/arm64).
[설치](/docs/tools/kora/beta/installation)를 참조하세요.

### 트랜잭션 플러그인

새로운 플러그인 시스템을 통해 Kora가 서명 플로우 중에 트랜잭션 구조를 검증할 수
있습니다. 첫 번째 내장 플러그인인 `gas_swap`는 가스리스 토큰-SOL 스왑 트랜잭션에
대한 엄격한 규칙을 적용합니다. 플러그인은 `kora.toml`에서 구성되며
`signTransaction`, `signAndSendTransaction`, `signBundle`, 그리고
`signAndSendBundle` 동안 자동으로 실행됩니다.

[구성 — 트랜잭션 플러그인](/docs/tools/kora/beta/configuration#transaction-plugins)을
참조하세요.

### 새로운 구성 옵션

- **트랜잭션 플러그인** 트랜잭션 구조 검증용 (`gas_swap`)
- **reCAPTCHA v3** 서명 엔드포인트를 위한 봇 방지 기능
- **Lighthouse** 수수료 지불자 고갈 방지
- **사용 제한** 지갑당 트랜잭션 제한
- **번들 구성** Jito 통합용

[구성](/docs/tools/kora/beta/configuration)을 참조하세요.

### SDK 변경 사항

- 번들 메서드 지원 (`signBundle`, `signAndSendBundle`, `estimateBundleFee`)

[SDK 변경 사항](/docs/tools/kora/beta/sdk-changes)을 참조하세요.

### 추가 RPC 메서드

- [`getVersion`](/docs/tools/kora/beta/bundle-methods#getversion) — 서버 버전
  정보 조회

## 설치

```bash
cargo install kora-cli@2.2.0-beta.7
```

또는 Docker를 통해:

```bash
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```
