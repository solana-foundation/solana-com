## Cargo

crates.io에서 베타 릴리스를 직접 설치하세요:

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

공식 멀티 플랫폼 Docker 이미지(amd64/arm64)가 GitHub 컨테이너 레지스트리에
배포되어 있습니다:

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

사용 가능한 태그:

- `latest` — 최신 안정 릴리스
- `beta` — 최신 프리릴리스 버전
- `v<version>` — 특정 버전 (예: `v2.2.0-beta.7`)

### Docker Compose

사용자 정의 빌드 또는 캐싱 및 사용량 제한 지원을 위해 Redis를 포함하려면:

- [샘플 Dockerfile](https://github.com/solana-foundation/kora/blob/main/examples/deploy/Dockerfile)
- [Redis를 포함한 Docker Compose](https://github.com/solana-foundation/kora/blob/main/docker-compose.yml)

### Docker로 실행하기

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## TypeScript SDK

베타 SDK는 npm에서 사용할 수 있습니다:

```bash
pnpm add @solana/kora@beta
```

새로운 API는 [SDK 변경사항](/docs/tools/kora/beta/sdk-changes)을 참조하세요.
