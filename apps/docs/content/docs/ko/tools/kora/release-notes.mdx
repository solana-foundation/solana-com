## Kora v2.2.0-beta.6

_2026-03-23_ |
[GitHub 릴리스](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.6)
| `cargo install kora-cli@2.2.0-beta.6`

Jito 번들 지원, 새로운 RPC 메서드 및 강화된 보안 기능을 포함한 프리릴리스입니다.

- `signBundle`, `signAndSendBundle` 및 `estimateBundleFee` 메서드를 통한 Jito
  번들 지원 추가
- `getVersion` RPC 메서드 추가
- Lighthouse 수수료 지불자 보호 추가
- 서명 엔드포인트를 위한 reCAPTCHA v3 봇 보호 추가
- Redis 지원을 통한 지갑별 사용량 제한 추가
- GHCR의 공식 Docker 이미지 추가
- 세분화된 사용량 제한 규칙 추가
- SDK: `KoraClient`가 이제 옵션 객체를 받습니다; `@solana/kit`를 위한 새로운
  `koraPlugin`

자세한 내용은 [베타 문서](/docs/tools/kora/beta)를 참조하세요.

## Kora TypeScript SDK v0.2.0

_2026-03-09_ |
[GitHub 릴리스](https://github.com/solana-foundation/kora/releases/tag/ts-sdk-v0.2.0)
| `pnpm add @solana/kora@0.2.0`

조합 가능한 가스리스 트랜잭션을 위한 Kit 클라이언트 플러그인 아키텍처입니다.

- Kit 플러그인 구성(플래너, 실행자, 지불자 플러그인)을 포함한
  `createKitKoraClient()` 추가
- `tokenProgram()`와 같은 Kit 프로그램 플러그인과의 조합 가능성 지원
- `tokenProgramId` 구성 옵션을 통한 Token-2022 지원 추가
- 시뮬레이션 기반 CU 추정을 포함한 컴퓨팅 예산 인스트럭션 지원 추가
- `signAndSendTransaction` 응답에 누락된 `signature` 필드 추가
- Kit v6 서명자 중복 제거를 위해 `getPaymentInstruction`로부터 `NoopSigner` 반환
- `@solana/kit` 피어 의존성을 `^6.1.0`로 업그레이드

[Kit 클라이언트 가이드](/docs/tools/kora/guides/kit-client)를 참조하세요.

## Kora v2.0.0

_2025-11-19_ |
[GitHub 릴리스](https://github.com/solana-foundation/kora/releases/tag/v2.0.0) |
`cargo install kora-cli@2.0.0`

새로운 아키텍처를 적용한 주요 릴리스.

- Kora 노드를 위한 새로운 아키텍처 도입

- Solana SDK v3.0.x로 마이그레이션 및 모든 종속성 업데이트
- 사용자 정의 Kora Signer를 solana-signers crate로 교체
- `sign_transaction_if_paid` 및 `get_all_signers` 메서드 제거
- 응답 타입 변경: `signAndSendTransaction` 및 `signTransaction`에서 서명 제거
- 개발자 경험, 보안 및 성능 향상
