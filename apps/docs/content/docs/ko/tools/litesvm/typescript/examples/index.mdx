## 예시 실행

일부 주소/변수는 실제 값으로 플레이스홀더를 교체해야 하지만, 모든 예시는 `tsx`로
실행할 수 있습니다.

```bash
# Install dependencies
pnpm add @solana/kit @solana/kit-plugin-litesvm @solana/kit-plugin-signer @solana-program/system

# Run an example
npx tsx example.ts
```
