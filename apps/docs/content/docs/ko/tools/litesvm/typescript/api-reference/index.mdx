## 빠른 참조

### 플러그인 설정

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### 계정 관리 (client.svm)

| 메서드                                 | 설명                  |
| -------------------------------------- | --------------------- |
| `setAccount(account)`                  | 계정 상태 직접 설정   |
| `getAccount(address)`                  | 계정 데이터 조회      |
| `getBalance(address)`                  | SOL 잔액 조회         |
| `airdrop(address, lamports)`           | 계정에 자금 지원      |
| `minimumBalanceForRentExemption(size)` | 렌트 면제 최솟값 계산 |

### 프로그램 관리 (client.svm)

| 메서드                                | 설명                       |
| ------------------------------------- | -------------------------- |
| `addProgram(programId, bytes)`        | 바이트에서 프로그램 로드   |
| `addProgramFromFile(programId, path)` | .so 파일에서 프로그램 로드 |

### 트랜잭션 (client.svm)

| 메서드                    | 설명                      |
| ------------------------- | ------------------------- |
| `sendTransaction(tx)`     | 트랜잭션 실행             |
| `simulateTransaction(tx)` | 상태 변경 없이 시뮬레이션 |
| `latestBlockhash()`       | 현재 블록해시 조회        |
| `expireBlockhash()`       | 새 블록해시로 진행        |

### 구성 (client.svm)

| 메서드                      | 설명                                 |
| --------------------------- | ------------------------------------ |
| `withSigverify(bool)`       | 서명 검증 활성화/비활성화            |
| `withBlockhashCheck(bool)`  | 블록해시 유효성 검사 활성화/비활성화 |
| `withSysvars()`             | sysvar 계정 활성화                   |
| `withBuiltins()`            | 내장 프로그램 활성화                 |
| `withPrecompiles()`         | 사전 컴파일된 프로그램 활성화        |
| `withTransactionHistory(n)` | 마지막 n개 트랜잭션 저장             |
| `withComputeBudget(budget)` | 컴퓨팅 한도 설정                     |
| `withLamports(amount)`      | 기본 lamport 설정                    |
| `withLogBytesLimit(n)`      | 로그 출력 한도 설정                  |

### 시간 및 Sysvars (client.svm)

| 메서드                     | 설명                        |
| -------------------------- | --------------------------- |
| `warpToSlot(slot)`         | 특정 slot으로 이동          |
| `getClock()`               | clock sysvar 가져오기       |
| `getRent()`                | rent sysvar 가져오기        |
| `getEpochSchedule()`       | epoch 일정 가져오기         |
| `getLastRestartSlot()`     | 마지막 재시작 slot 가져오기 |
| `setLastRestartSlot(slot)` | 마지막 재시작 slot 설정     |

### RPC (client.rpc)

| 메서드                           | 설명                             |
| -------------------------------- | -------------------------------- |
| `getAccountInfo(address)`        | 단일 계정 가져오기 (base64 전용) |
| `getMultipleAccounts(addresses)` | 여러 계정 가져오기               |
| `getLatestBlockhash()`           | 현재 블록해시 가져오기           |
