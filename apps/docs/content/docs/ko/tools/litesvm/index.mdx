---
title: LiteSVM
description:
  인프로세스 VM으로 Solana 프로그램을 초고속으로 테스트하세요. 네트워크
  오버헤드나 validator 프로세스 없이 더 빠르게 프로그램을 테스트하세요.
---

LiteSVM은 Solana 프로그램 테스트를 위한 경량 라이브러리로, 프로그램 개발자에게
최적화된 인프로세스 Solana VM을 생성하여 동작합니다. 별도의 validator 프로세스를
실행하는 다른 테스트 방식과 달리, LiteSVM은 VM을 테스트 내부에 직접 내장하여
훨씬 빠른 실행 속도를 제공합니다. 기존 방식과의 또 다른 차별점으로, 합리적인
기본값과 폭넓은 커스터마이징 옵션을 갖춘 직관적인 API를 제공합니다.
