## 빠른 참조

| 메서드                                         | 설명                                                               |
| ---------------------------------------------- | ------------------------------------------------------------------ |
| `LiteSVM::new()`                               | 새로운 LiteSVM 기본 테스트 환경 생성                               |
| `LiteSVM::default()`                           | 타입의 "기본값" 반환                                               |
| `airdrop(&Address, lamports)`                  | 지정된 lamport로 계정에 자금 지원                                  |
| `get_account(&Address)`                        | 제공된 주소의 계정과 관련된 모든 정보 반환                         |
| `set_account(Address, Account)`                | 제공된 주소의 계정과 관련된 모든 정보 설정                         |
| `get_balance(&Address)`                        | 제공된 계정 주소의 lamport 잔액 조회                               |
| `minimum_balance_for_rent_exemption(data_len)` | 지정된 데이터 길이의 계정을 rent 면제로 만들기 위한 최소 잔액 반환 |
| `add_program(program_id, &bytes)`              | 테스트 환경에 SBF 프로그램 추가                                    |
| `add_program_from_file(program_id, path)`      | 지정된 파일에서 테스트 환경에 SBF 프로그램 추가                    |
| `send_transaction(tx)`                         | 서명된 트랜잭션 제출                                               |
| `simulate_transaction(tx)`                     | 상태 변경 없이 트랜잭션 시뮬레이션                                 |
| `get_transaction(&signature)`                  | 기록에서 트랜잭션 조회                                             |
| `latest_blockhash()`                           | 현재 블록해시 조회                                                 |
| `expire_blockhash()`                           | 현재 블록해시 만료 강제 적용                                       |
| `warp_to_slot(slot)`                           | 지정된 slot으로 클록 이동                                          |
| `get_sysvar<T>()`                              | 테스트 환경에서 sysvar 조회                                        |
| `set_sysvar<T>(&value)`                        | 테스트 환경에 sysvar 설정                                          |
| `with_feature_accounts()`                      | 테스트 환경에 기능 게이트 계정 포함                                |
| `set_invocation_inspect_callback(callback)`    | 사전/사후 호출 콜백 등록 (`invocation-inspect-callback` 기능 필요) |
