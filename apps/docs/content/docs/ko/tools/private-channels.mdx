**프라이빗 채널은 솔라나 기반 인터넷 자본 시장을 위한 엔터프라이즈
레이어입니다.** 결제 채널 내에서 비공개적이고 즉각적이며 수수료가 없는 전송을
위한 구성 가능한 앱 레이어를 제공합니다.
