**Solana Pay는 개발자가 탈중앙화 결제를 앱과 서비스에 통합할 수 있도록 하는 표준
프로토콜 및 참조 구현 세트입니다.** Solana 블록체인은 1초 이내에 거래를 확인하고
평균 $0.0005의 비용으로, 중개자 없이 사용자에게 원활한 경험을 제공합니다.

## 왜 Solana Pay인가?

- **표준 프로토콜**: 모든 Solana 지갑 및 앱에서 통합된 결제 방식
- **초고속**: 1초 이내에 거래 확인
- **초저비용**: 평균 거래 비용 $0.0005
- **중개자 없음**: 제3자 없이 직접 P2P 결제
- **범용 지갑 지원**: Phantom, Solflare, Backpack 등과 호환

## 빠른 시작

시작하는 가장 간단한 방법은 전송 요청 URL을 사용하는 것입니다:

```
solana:recipient?amount=0.01&label=Coffee%20Shop&message=Grande%20Latte
```

이 URL은 다음과 같이 사용할 수 있습니다:

- 모바일 지갑 스캔을 위한 QR 코드로 인코딩
- 웹 애플리케이션에서 결제 링크로 사용
- 모바일 앱에 딥링크로 임베드

## 설치

Solana Pay JavaScript SDK 설치:

```bash
# Using pnpm (recommended)
pnpm add @solana/pay@beta @solana/kit

# Using npm
npm install @solana/pay@beta @solana/kit

# Using yarn
yarn add @solana/pay@beta @solana/kit
```

## 사용 사례

Solana Pay는 다양한 결제 시나리오를 지원합니다:

- **소매 결제**: QR 코드를 사용한 오프라인 결제
- **전자상거래**: 지갑 통합을 통한 웹 체크아웃 플로우
- **모바일 앱**: 딥링킹을 활용한 네이티브 결제 경험
- **NFT 민팅**: 복잡한 인터랙션을 위한 동적 트랜잭션 생성
- **구독**: 반복 결제 및 로열티 프로그램
- **DeFi 통합**: 스왑 및 대출과 결합 가능한 트랜잭션

## 지원 지갑

Solana Pay는 모든 주요 Solana 지갑과 호환됩니다:

- **[Phantom](https://phantom.app/)** - 가장 인기 있는 Solana 지갑
- **[Solflare](https://solflare.com/)** - 멀티 플랫폼 Solana 지갑
- **[Backpack](https://backpack.app/)** - Web3 지갑 및 앱 스토어
- **[Glow](https://glow.app/)** - 모바일 우선 Solana 지갑
- **[Ultimate](https://ultimate.app/)** - DeFi 중심 지갑
- **[Decaf Wallet](https://decaf.so/)** - 글로벌 결제 지갑
- **[Espresso Cash](https://www.espressocash.com/)** - 글로벌 결제 지갑
- **[Tiplink](https://tiplink.io/)** - 세계에서 가장 간편한 지갑
- **[Bitget Wallet](https://bitget.com/)** - 멀티체인 지갑
