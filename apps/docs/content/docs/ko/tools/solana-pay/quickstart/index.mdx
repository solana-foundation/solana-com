단계별 가이드를 통해 Solana Pay를 빠르게 시작하고 실행하세요.

## 설치

Solana Pay SDK를 설정하고 개발 환경을 구성하세요.

**[→ 설치 가이드](/docs/tools/solana-pay/quickstart/installation)** - SDK 및
종속성 설치

## 결제 유형

사용 사례에 맞는 결제 통합을 선택하세요:

### 전송 요청 (간단함)

기본 SOL 및 SPL 토큰 전송을 위한 비대화형 결제 URL을 생성합니다.

**[→ 전송 요청](/docs/tools/solana-pay/quickstart/transfer-requests)** - 간단한
결제 URL

**적합한 경우:**

- 기본 결제 및 기부
- QR 코드 결제
- 이메일 결제 링크

### 트랜잭션 요청 (고급)

서버 측 트랜잭션 구성을 통해 대화형 결제 흐름을 구축합니다.

**[→ 트랜잭션 요청](/docs/tools/solana-pay/quickstart/transaction-requests)** -
대화형 결제 흐름

**적합한 경우:**

- 동적 가격 책정
- NFT 발행
- 복잡한 DeFi 트랜잭션
- 맞춤형 비즈니스 로직

## QR 코드 통합

모바일 지갑 결제를 위한 QR 코드를 생성하고 사용자 지정합니다.

**[→ QR 코드 통합](/docs/tools/solana-pay/quickstart/qr-codes)** - 스캔 가능한
결제 코드 생성

**적합한 경우:**

- 판매 시점 시스템
- 웹에서 모바일로의 결제
- 대면 거래
