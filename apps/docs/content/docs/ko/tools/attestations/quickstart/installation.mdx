필요한 패키지를 설치하고 개발 환경을 설정하여 Solana Attestations를 시작하세요.

## TypeScript/JavaScript

선호하는 패키지 관리자를 사용하여 Solana Attestations 라이브러리를 설치하세요:

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
