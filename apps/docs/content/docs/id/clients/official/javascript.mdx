## Solana Kit

`@solana/kit` adalah TypeScript SDK yang direkomendasikan untuk membangun di
Solana. Anda merakit klien dari plugin-plugin yang dapat dikombinasikan,
menambahkan binding React saat dibutuhkan, dan mengintegrasikan klien program
yang telah dibuat untuk program on-chain yang Anda panggil.

| Package                   | Deskripsi                                            | GitHub                                                                                 |
| ------------------------- | ---------------------------------------------------- | -------------------------------------------------------------------------------------- |
| @solana/kit               | SDK Inti: RPC, transaksi, codec, penanda tangan      | [Sumber](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | Binding React: `ClientProvider` dan hooks            | [Sumber](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | Koneksi RPC + pengiriman transaksi                   | [Sumber](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | Plugin pembayar biaya, identitas, dan penanda tangan | [Sumber](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Koneksi Wallet Standard + React hooks                | [Sumber](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

## Solana Web3.js

`@solana/web3.js` adalah SDK TypeScript lama untuk Solana. Aplikasi baru
sebaiknya menggunakan `@solana/kit`; lihat
[Migrasi ke Kit](/docs/frontend/web3-compat).

| Package           | Deskripsi                                                           | GitHub                                                                             |
