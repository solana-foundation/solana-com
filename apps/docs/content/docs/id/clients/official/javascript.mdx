## Solana Client

`@solana/client` adalah runtime Solana headless yang menggabungkan RPC, wallet,
transaksi, dan subscription dalam satu store.

| Paket          | Deskripsi               | GitHub                                                                                 |
| -------------- | ----------------------- | -------------------------------------------------------------------------------------- |
| @solana/client | Runtime client headless | [Source](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [Panduan: Ringkasan @solana/client](/docs/frontend/client)

## Solana React Hooks

`@solana/react-hooks` melapisi React provider dan hooks di atas
`@solana/client`, menggunakan kembali runtime dan cache yang sama.

| Paket               | Deskripsi                       | GitHub                                                                                      |
| ------------------- | ------------------------------- | ------------------------------------------------------------------------------------------- |
| @solana/react-hooks | React provider + hooks untuk UI | [Source](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [Panduan: @solana/react-hooks](/docs/frontend/react-hooks)

## Solana Kit

`@solana/kit` adalah SDK TypeScript yang direkomendasikan untuk membangun di
Solana.

| Paket                          | Deskripsi                                  | GitHub                                                                          |
| ------------------------------ | ------------------------------------------ | ------------------------------------------------------------------------------- |
| @solana/kit                    | SDK Inti                                   | [Source](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | Berinteraksi dengan System Program         | [Source](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Berinteraksi dengan Token Program          | [Source](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Berinteraksi dengan program Token-2022     | [Source](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Berinteraksi dengan program Memo           | [Source](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Berinteraksi dengan program Compute Budget | [Source](https://github.com/solana-program/compute-budget/tree/main/clients/js) |
| @solana/kit-plugins            | Plugin untuk Kit                           | [Source](https://github.com/anza-xyz/kit-plugins)                               |

- [Dokumentasi Kit Resmi](https://www.solanakit.com/)
- [Tutorial: Next.js + React Hooks](/docs/frontend/nextjs-solana)
