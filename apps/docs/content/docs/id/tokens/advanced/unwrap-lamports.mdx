## Apa yang Dilakukan Unwrap Lamports?

_rs`UnwrapLamports`_ memindahkan SOL keluar dari token account native (wrapped
SOL) ke akun tujuan sebagai lamport. Anda dapat unwrap seluruh saldo atau jumlah
tertentu, dan token account tetap terbuka.

Ini lebih fleksibel dibandingkan menutup akun wrapped SOL: unwrap sebagian
memungkinkan Anda menarik sebagian SOL sambil tetap menjaga akun tetap aktif
untuk digunakan nanti.

## Akun

| Akun          | Deskripsi                                                              |
| ------------- | ---------------------------------------------------------------------- |
| `source`      | Dapat ditulis. Token account native (wrapped SOL) yang akan di-unwrap. |
| `destination` | Dapat ditulis. Menerima lamport yang telah di-unwrap.                  |
| `authority`   | Pemilik akun atau delegasi (atau penanda tangan multisig).             |

<Callout type="warn">

Sumber harus berupa token account native; token account lainnya akan
mengembalikan _rs`NonNativeNotSupported`_. Delegasi dapat melakukan unwrap
hingga jumlah yang didelegasikan.

</Callout>

## Cara Melakukan Unwrap Lamports

Hilangkan _ts`amount`_ untuk unwrap seluruh saldo, atau tetapkan nilainya untuk
unwrap sejumlah lamport tertentu. Contoh ini menggunakan klien
[`@solana/kit`](/docs/clients/official/javascript#solana-kit) yang telah
dikonfigurasi — lihat [Transfer Token](/docs/tokens/basics/transfer-tokens)
untuk pengaturan klien.

```ts title="Unwrap wrapped SOL (Kit)"
import { getUnwrapLamportsInstruction } from "@solana-program/token";

// source: the native (wrapped SOL) token account to unwrap.
// destination: address that receives the unwrapped lamports.
// authority: a TransactionSigner (the account owner or a delegate).

// Unwrap a specific number of lamports.
const unwrapSome = getUnwrapLamportsInstruction({
  source,
  destination,
  authority,
  amount: 500_000_000n
});

// Or omit `amount` to unwrap the entire balance.
const unwrapAll = getUnwrapLamportsInstruction({
  source,
  destination,
  authority
});

// Send whichever instruction you need.
await client.sendTransaction([unwrapSome]);
```

Untuk otoritas multisig, teruskan penanda tangan bersama dengan
_ts`multiSigners`_.
