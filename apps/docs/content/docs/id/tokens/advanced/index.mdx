Selain operasi umum seperti membuat, minting, dan transfer, Token Program juga
dipanggil dari program on-chain melalui CPI dan mencakup instruksi untuk
memulihkan lamport:

<Cards>
  <Card
    title="Memanggil Token Program melalui CPI"
    href="/docs/tokens/advanced/cpi"
  >
    Panggil Token Program dari program on-chain — transfer, mint, dan burn
    token, tandatangani dengan otoritas PDA, dan gabungkan instruksi dalam satu CPI.
  </Card>

<Card
  title="Tarik Lamport Berlebih"
  href="/docs/tokens/advanced/withdraw-excess-lamports"
>
  Pulihkan SOL di atas minimum bebas sewa dari token account, mint, dan akun
  multisig tanpa mengubah saldo token.
</Card>

  <Card title="Unwrap Lamport" href="/docs/tokens/advanced/unwrap-lamports">
    Konversi wrapped SOL kembali menjadi lamport, sebagian atau seluruhnya, tanpa menutup
    token account.
  </Card>
</Cards>

<Callout type="info">

_rs`WithdrawExcessLamports`_ dan _rs`UnwrapLamports`_ tersedia melalui
[`@solana-program/token`](https://www.npmjs.com/package/@solana-program/token)
client. Memanggil Token Program melalui CPI berlaku untuk program on-chain.

</Callout>

## Pemulihan Lamport

_rs`WithdrawExcessLamports`_ memulihkan SOL di atas minimum bebas sewa dari
token account, mint, dan akun multisig. _rs`UnwrapLamports`_ mengonversi wrapped
SOL dalam native token account kembali menjadi lamport.
