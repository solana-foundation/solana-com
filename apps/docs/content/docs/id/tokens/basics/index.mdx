Bagian Dasar-dasar SPL Token mencakup instruksi dasar untuk berinteraksi dengan
SPL Token, dengan fokus pada fungsi yang dibagikan oleh Token Program original
dan Token Extensions Program (Token 2022).

<Callout type="info">
  Lihat [dokumentasi Metaplex](https://developers.metaplex.com/token-metadata)
  untuk informasi lebih lanjut tentang cara menambahkan metadata ke SPL Token.
</Callout>
