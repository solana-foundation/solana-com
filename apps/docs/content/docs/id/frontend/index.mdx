Pengembangan frontend di Solana melibatkan bekerja dengan program, dompet, dan
framework JavaScript populer seperti React. Berinteraksi dengan
komponen-komponen ini memerlukan penanganan koneksi, pembuatan transaksi, dan
pembacaan dari akun Solana.

[Solana Kit](https://github.com/anza-xyz/kit) (`@solana/kit`) adalah SDK
JavaScript standar untuk pekerjaan ini. Anda merakit klien dari plugin-plugin
kecil yang dapat dikomposisi, menambahkan binding React saat dibutuhkan, dan
mengintegrasikan klien program yang dihasilkan untuk program on-chain yang Anda
panggil.

## Pustaka utama

<Cards>
  <Card title="@solana/kit" href="https://github.com/anza-xyz/kit">
    - SDK inti: RPC, codec, signer, dan transaksi - Menyediakan `createClient`,
    host plugin yang diperluas oleh setiap library lainnya
  </Card>
  <Card title="Kit plugins" href="https://github.com/anza-xyz/kit-plugins">
    - Rakit klien dengan `.use(...)`: koneksi RPC, wallet, signer, dan
    perencanaan transaksi - Diterbitkan sebagai paket `@solana/kit-plugin-*`
  </Card>
  <Card
    title="@solana/react"
    href="https://github.com/anza-xyz/kit/tree/main/packages/react"
  >
    - Binding React untuk klien Kit - `ClientProvider` ditambah hook untuk
    pembacaan, langganan, dan status wallet
  </Card>
  <Card title="@solana-program/*" href="https://github.com/solana-program">
    - Klien yang dihasilkan untuk program on-chain (System, Token, dan lainnya)
    - Menyediakan pembangun instruksi dan plugin klien seperti `tokenProgram()`
  </Card>
</Cards>

<Callout type="warn" title="web3.js v1 and wallet-adapter are legacy">
  Banyak proyek Solana masih mengandalkan `@solana/web3.js` (v1) dan
  `@solana/wallet-adapter-*` yang lebih lama. Untuk pekerjaan baru, lebih baik
  gunakan `@solana/kit` dengan plugin kit di bawah; penemuan Wallet Standard
  (melalui `@solana/kit-plugin-wallet`) menggantikan wallet-adapter untuk wallet
  modern.
</Callout>
