**Commerce Kit adalah kumpulan alat untuk membangun pengalaman commerce di
Solana.** Dari tombol pembayaran siap pakai hingga primitif headless, Commerce
Kit menyediakan semua yang Anda butuhkan untuk menerima pembayaran crypto.

## Mengapa Commerce Kit?

- **Komponen Siap Pakai**: PaymentButton yang sudah jadi dengan koneksi wallet,
  pemilihan token, dan penanganan transaksi
- **Framework Agnostik**: Gunakan primitif headless di framework JavaScript apa
  pun
- **Solana Pay**: Dukungan penuh Solana Pay dengan pembuatan kode QR
- **Type-Safe**: Dukungan TypeScript lengkap di semua paket
- **Wallet Standard**: Kompatibel dengan semua wallet Wallet Standard (Phantom,
  Solflare, Backpack, dll.)

## Mulai Cepat

Cara tercepat untuk memulai adalah dengan komponen `PaymentButton`:

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## Instalasi

Pilih paket Commerce Kit yang Anda butuhkan:

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
