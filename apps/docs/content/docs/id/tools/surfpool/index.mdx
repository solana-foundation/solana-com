Surfpool adalah pengganti langsung untuk `solana-test-validator`, yang dirancang
khusus untuk memberikan pengalaman terbaik bagi pengembang yang membangun di
**Solana**.

Fitur utamanya adalah memungkinkan pengembang mensimulasikan program mereka
secara lokal menggunakan akun Mainnet yang diambil tepat waktu.

Surfpool juga mengintegrasikan Infrastructure as Code secara mulus ke dalam
proyek berbasis Anchor atau Pinocchio, memungkinkan deployment yang dapat
direproduksi, diaudit, dan aman ke jaringan Solana mana pun—privat maupun
publik.
