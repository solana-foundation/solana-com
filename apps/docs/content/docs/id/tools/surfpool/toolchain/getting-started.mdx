  Panduan ini akan mempersiapkan dan memandu Anda untuk menginstal Surfpool.
---

Surfpool CLI dapat diinstal menggunakan skrip instalasi atau dikompilasi dari
sumber.

## Skrip Instalasi

Untuk menginstal surfpool, buka terminal dan jalankan:

```bash
curl -sL https://run.surfpool.run/ | bash
```

## Instal dari Sumber

Untuk mengompilasi biner Surfpool dari sumber, clone repositori surfpool.

```bash
git clone https://github.com/solana-foundation/surfpool.git
```

Navigasi ke direktori baru.

```bash
cd surfpool
```

Kemudian, kompilasi biner tersebut. Perintah ini akan mengompilasi biner dan
menyimpannya di `~/.cargo/bin/surfpool`.

```bash
cargo surfpool-install
```

Terakhir, pastikan biner `surfpool` tersedia di PATH Anda. Proses ini akan
berbeda tergantung pada sistem operasi Anda.
