Tindakan-tindakan ini tersedia di semua Runbook sebagai bagian dari pustaka
standar.

## send_http_request

Tindakan `std::send_http_request` membuat permintaan HTTP ke URL yang
ditentukan.

### Masukan

| Nama         | Wajib    | Tipe    | Deskripsi                                          |
| ------------ | -------- | ------- | -------------------------------------------------- |
| `url`        | wajib    | string  | URL tujuan pengiriman permintaan                   |
| `method`     | opsional | string  | Metode HTTP (GET, POST, PUT, DELETE). Default: GET |
| `headers`    | opsional | map     | Header HTTP yang disertakan dalam permintaan       |
| `body`       | opsional | string  | Isi permintaan (untuk permintaan POST/PUT)         |
| `timeout_ms` | opsional | integer | Batas waktu permintaan dalam milidetik             |

### Keluaran

| Nama          | Tipe    | Deskripsi        |
| ------------- | ------- | ---------------- |
| `status_code` | integer | Kode status HTTP |
| `body`        | string  | Isi respons      |
| `headers`     | map     | Header respons   |

```hcl
action "api_call" "std::send_http_request" {
    description = "Fetch data from API"
    url = "https://api.example.com/data"
    method = "GET"
    headers = {
        "Authorization" = "Bearer ${variable.token}"
    }
}

output "response" {
    value = action.api_call.body
}

output "status" {
    value = action.api_call.status_code
}
```

### Contoh Permintaan POST

```hcl
action "create_resource" "std::send_http_request" {
    description = "Create a new resource"
    url = "https://api.example.com/resources"
    method = "POST"
    headers = {
        "Content-Type" = "application/json"
    }
    body = {
        "name" = variable.name,
        "value" = variable.value
    }
}
```
