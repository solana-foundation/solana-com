---
title: Fungsi Base58
description: Fungsi pustaka standar untuk encoding dan decoding Base58
---

## encode_base58

`encode_base58` mengenkode buffer atau string sebagai string base58.

### Input

| Nama    | Tipe                  | Deskripsi                                                                                                                           |
| ------- | --------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | Buffer atau string yang akan dienkode. String yang diawali '0x' didekode sebagai hex; jika tidak, byte UTF-8 mentah yang digunakan. |

### Output

| Nama    | Tipe   | Deskripsi                                 |
| ------- | ------ | ----------------------------------------- |
| `value` | string | Input yang dienkode sebagai string base58 |

```hcl
output "encoded" {
    value = encode_base58("0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8")
}
// > encoded: CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF
```

## decode_base58

`decode_base58` mendekode string yang dienkode dengan base58 dan mengembalikan
hasilnya sebagai buffer.

### Input

| Nama            | Tipe   | Deskripsi                        |
| --------------- | ------ | -------------------------------- |
| `base58_string` | string | String base58 yang akan didekode |

### Output

| Nama    | Tipe   | Deskripsi                                         |
| ------- | ------ | ------------------------------------------------- |
| `value` | buffer | String base58 yang telah didekode, sebagai buffer |

```hcl
output "decoded" {
    value = decode_base58("CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF")
}
// > decoded: 0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8
```
