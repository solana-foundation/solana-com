## assert_eq

`assert_eq` memastikan bahwa dua nilai adalah sama.

### Input

| Nama    | Tipe | Deskripsi                |
| ------- | ---- | ------------------------ |
| `left`  | any  | Nilai untuk dibandingkan |
| `right` | any  | Nilai pembanding         |

### Output

| Nama    | Tipe                  | Deskripsi            |
| ------- | --------------------- | -------------------- |
| `value` | addon(std::assertion) | Hasil dari assertion |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```

## assert_gt

`assert_gt` memastikan bahwa nilai kiri lebih besar dari nilai kanan.

### Input

| Nama    | Tipe             | Deskripsi                             |
| ------- | ---------------- | ------------------------------------- |
| `left`  | integer \| float | Integer atau float untuk dibandingkan |
| `right` | integer \| float | Integer atau float sebagai pembanding |

### Output

| Nama    | Tipe                  | Deskripsi            |
| ------- | --------------------- | -------------------- |
| `value` | addon(std::assertion) | Hasil dari assertion |

```hcl
output "assertion" {
    value = std::assert_gt(action.example.result, 1)
}
```

| `left`  | integer \| float | Bilangan bulat atau float untuk dibandingkan |
| `right` | integer \| float | Bilangan bulat atau float sebagai pembanding |

| Name    | Type                  | Description           |
| ------- | --------------------- | --------------------- |
| `value` | addon(std::assertion) | Hasil dari pernyataan |
