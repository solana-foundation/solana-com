## and_bool

`and_bool` mengembalikan hasil AND biner dari argumen sisi kiri dan sisi kanan.

### Input

| Name  | Type | Description   |
| ----- | ---- | ------------- |
| `lhs` | bool | Nilai boolean |
| `rhs` | bool | Nilai boolean |

### Output

| Name    | Type | Description                        |
| ------- | ---- | ---------------------------------- |
| `value` | bool | Hasil AND biner antara dua argumen |

```hcl
output "my_bool" {
  value = false && true
}
// > my_bool: false
```

## div

`div` mengembalikan hasil pembagian bilangan bulat dari argumen sisi kiri dengan
argumen sisi kanan, membulatkan sisa pembagian ke bawah menuju bilangan bulat
terdekat.

### Input

| Name  | Type    | Description |
| ----- | ------- | ----------- |
| `lhs` | integer | Dividen     |
| `rhs` | integer | Pembagi     |

### Output

| Name    | Type    | Description                            |
| ------- | ------- | -------------------------------------- |
| `value` | integer | Hasil pembagian dividen dengan pembagi |

```hcl
output "my_int" {
  value = 11 / -3
}
// > my_int: -3
```

## eq

`eq` mengembalikan `true` jika argumen kiri dan kanan sama, dan `false` jika
tidak sama.

### Input

| Nama  | Tipe | Deskripsi     |
| ----- | ---- | ------------- |
| `lhs` | any  | Nilai apa pun |
| `rhs` | any  | Nilai apa pun |

### Output

| Nama    | Tipe | Deskripsi                                   |
| ------- | ---- | ------------------------------------------- |
| `value` | bool | Hasil pemeriksaan kesamaan antara dua input |

```hcl
output "is_eq" {
  value = "arg" == "arg"
}
// > is_eq: true
```

| ------- | ---- | -------------------------------------------------------------------- |

## neq

`neq` mengembalikan `true` jika argumen di sisi kiri dan kanan tidak sama dan
`false` jika sebaliknya.

### Input

| Name  | Type | Description                     |
| ----- | ---- | ------------------------------- |
| `lhs` | any  | Nilai untuk dibandingkan        |
| `rhs` | any  | Nilai untuk dibandingkan dengan |

### Output

| Nama    | Tipe | Deskripsi                                                      |
| ------- | ---- | -------------------------------------------------------------- |
| `value` | bool | Hasil pemeriksaan kesetaraan yang dinegasikan antara dua input |

```hcl
output "is_neq" {
  value = "arg" != "arg"
}
// > is_neq: false
```
