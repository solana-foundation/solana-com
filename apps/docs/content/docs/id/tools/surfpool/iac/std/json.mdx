## jq

Fungsi `jq` memungkinkan pemotongan, penyaringan, dan pemetaan data JSON. Lihat
[dokumentasi jq](https://jqlang.github.io/jq/manual/) untuk detail lebih lanjut.

### Input

| Nama           | Tipe             | Deskripsi                                                               |
| -------------- | ---------------- | ----------------------------------------------------------------------- |
| `decoded_json` | string \| object | Objek JSON                                                              |
| `query`        | string           | Kueri JSON. Lihat [dokumentasi jq](https://jqlang.github.io/jq/manual/) |

### Output

| Nama    | Tipe          | Deskripsi        |
| ------- | ------------- | ---------------- |
| `value` | array[string] | Hasil kueri `jq` |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
