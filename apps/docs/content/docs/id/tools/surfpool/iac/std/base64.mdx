title: Fungsi Base64

---

## encode_base64

`encode_base64` mengenkode buffer atau string sebagai string base64.

### Input

| Nama    | Tipe                  | Deskripsi                                                                                                                                  |
| ------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| `value` | buffer, string, addon | Buffer atau string yang akan dienkode. String yang diawali dengan '0x' didekode sebagai hex; selain itu, byte UTF-8 mentah yang digunakan. |

### Output

| Nama    | Tipe   | Deskripsi                             |
| ------- | ------ | ------------------------------------- |
| `value` | string | Input, dienkode sebagai string base64 |

```hcl
output "encoded" {
    value = encode_base64("0x48656c6c6f20776f726c6421")
}
// > encoded: SGVsbG8gd29ybGQh
```

## decode_base64

`decode_base64` mendekode string yang dienkode base64 dan mengembalikan hasilnya
sebagai buffer.

### Input

| Nama            | Tipe   | Deskripsi                        |
| --------------- | ------ | -------------------------------- |
| `base64_string` | string | String base64 yang akan didekode |

### Output

| Nama    | Tipe   | Deskripsi                                         |
| ------- | ------ | ------------------------------------------------- |
| `value` | buffer | String base64 yang telah didekode, sebagai buffer |

```hcl
output "decoded" {
    value = decode_base64("SGVsbG8gd29ybGQh")
}
// > decoded: 0x48656c6c6f20776f726c6421
```
