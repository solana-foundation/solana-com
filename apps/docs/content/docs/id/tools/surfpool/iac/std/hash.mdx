## ripemd160

`ripemd160` menghitung hash Ripemd160 dari sebuah nilai.

### Input

| Nama    | Tipe                    | Deskripsi                              |
| ------- | ----------------------- | -------------------------------------- |
| `value` | buffer \| array[buffer] | Nilai yang dikodekan hex untuk di-hash |

### Output

| Nama    | Tipe   | Deskripsi  |
| ------- | ------ | ---------- |
| `value` | string | Hasil hash |

```hcl
output "hashed_data" {
    value = ripemd160(encode_hex("hello, world"))
}
// > hashed_data: 0XA3201F82FCA034E46D10CD7B27E174976E241DA2
```

| `value` | string | Hasil hash |

```hcl
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```

### Input

| Nama    | Tipe   | Deskripsi                  |
| ------- | ------ | -------------------------- |
| `value` | string | Nilai string untuk di-hash |

### Output

| Nama    | Tipe   | Deskripsi  |
| ------- | ------ | ---------- |
| `value` | string | Hasil hash |

```hcl
output "hashed_data" {
    value = keccak256("hello, world")
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```
