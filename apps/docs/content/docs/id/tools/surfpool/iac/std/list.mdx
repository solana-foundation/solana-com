## index

`index` mendapatkan entri dari sebuah daftar pada indeks yang ditentukan.

### Input

| Nama    | Tipe                                             | Deskripsi                            |
| ------- | ------------------------------------------------ | ------------------------------------ |
| `list`  | array[string] \| array[integer] \| array[buffer] | Daftar untuk mengambil entri darinya |
| `index` | integer                                          | Indeks entri yang akan diambil       |

### Output

| Nama    | Tipe   | Deskripsi                                     |
| ------- | ------ | --------------------------------------------- |
| `value` | string | Entri dari daftar pada indeks yang ditentukan |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
