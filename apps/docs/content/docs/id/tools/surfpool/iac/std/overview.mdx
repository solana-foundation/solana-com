Pustaka standar menyediakan fungsi dan aksi inti yang tersedia untuk semua
Runbook, terlepas dari addon blockchain mana yang diaktifkan.

## Fungsi

Pustaka standar mencakup fungsi utilitas untuk:

- **Operator**: Operasi matematika dan perbandingan
- **Hash**: Fungsi hashing (SHA256, Keccak256, dll.)
- **Hex**: Encoding dan decoding heksadesimal
- **Crypto**: Operasi kriptografi
- **JSON**: Encoding dan decoding JSON
- **List**: Fungsi manipulasi daftar
- **Assertions**: Pernyataan kesetaraan dan kondisi
- **Base64**: Encoding dan decoding Base64

[Lihat semua fungsi](/docs/tools/surfpool/iac/std/functions)

## Aksi

Pustaka standar mencakup aksi untuk:

- **HTTP**: Membuat permintaan HTTP
- **Operasi File**: Membaca dan menulis file

[Lihat semua aksi](/docs/tools/surfpool/iac/std/actions)
