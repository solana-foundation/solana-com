---
title: Ikhtisar SVM Addon
description: Add-on untuk Solana dan Blockchain Kompatibel SVM (beta)
---

Plugin SVM `txtx` memungkinkan pembuatan Runbook yang berinteraksi dengan Solana
dan blockchain kompatibel SVM. Plugin ini menyediakan fungsi utilitas yang
memungkinkan Anda men-deploy program Anchor dan mengenkode pemanggilan instruksi
sesuai dengan IDL program. Aksi-aksi ini dapat digunakan untuk membuat transaksi
transfer, pemanggilan program, dan deployment program yang valid, yang dapat
ditandatangani melalui frasa mnemonic, secret key, atau melalui penanda tangan
browser Anda.

## Mulai Cepat

Untuk menggunakan SVM addon, deklarasikan dalam Runbook Anda:

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## Fitur

- **Deployment Program**: Deploy program Anchor dan program Solana native
- **Pembuatan Instruksi**: Enkode pemanggilan instruksi menggunakan IDL program
- **Penandatanganan Transaksi**: Dukungan untuk mnemonic, secret key, dan dompet
  web
- **Manajemen Akun**: Kueri dan kelola akun Solana
