title: Fungsi Hex

  Fungsi pustaka standar untuk pengkodean dan pendekodean heksadesimal
---

## encode_hex

`encode_hex` mengkodekan buffer atau string sebagai string heksadesimal dengan
prefiks 0x.

### Masukan

| Nama    | Tipe                  | Deskripsi                                                                                                                                      |
| ------- | --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | Buffer atau string yang akan dikodekan. String yang dimulai dengan '0x' didekodekan sebagai hex; jika tidak, byte UTF-8 mentah yang digunakan. |

### Keluaran

| Nama    | Tipe   | Deskripsi                                                    |
| ------- | ------ | ------------------------------------------------------------ |
| `value` | string | Masukan dalam representasi heksadesimalnya dengan prefiks 0x |

```hcl
output "encoded_hex" {
    value = encode_hex("hello, world")
}
// > encoded_hex: 0x68656c6c6f2c20776f726c64
```

## decode_hex

`decode_hex` mendekodekan string heksadesimal dan mengembalikan hasilnya sebagai
buffer.

### Masukan

| Nama         | Tipe   | Deskripsi                                                       |
| ------------ | ------ | --------------------------------------------------------------- |
| `hex_string` | string | String hex yang akan didekodekan (dengan atau tanpa prefiks 0x) |

### Keluaran

| Nama    | Tipe   | Deskripsi                                        |
| ------- | ------ | ------------------------------------------------ |
| `value` | buffer | String hex yang telah didekodekan sebagai buffer |

```hcl
output "decoded_hex" {
    value = decode_hex("0x68656c6c6f2c20776f726c64")
}
// > decoded_hex: 0x68656c6c6f2c20776f726c64
```
