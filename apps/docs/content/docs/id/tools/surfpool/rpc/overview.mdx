Surfnet adalah simulator jaringan Solana lokal milik Surfpool, dirancang untuk
memberikan para pengembang lingkungan pengujian yang powerful yang meniru
perilaku blockchain Solana tanpa overhead dari node validator penuh.

## Memulai

Mulai instance Surfnet menggunakan Surfpool CLI:

```bash
surfpool start
```

Ini akan memulai instance Surfnet lokal di `http://localhost:8899` dengan semua
endpoint RPC tersedia.

## Fitur Utama

- **Kompatibilitas RPC Penuh**: Mendukung Solana RPC API secara lengkap
- **Iterasi Cepat**: Tidak perlu menunggu konfirmasi blok
- **Manipulasi State**: Cheatcodes untuk memodifikasi state blockchain secara
  langsung
- **Mainnet Fork**: Fork dari mainnet untuk pengujian dengan akun dan program
  nyata
- **Dukungan Geyser Plugin**: Hubungkan plugin Geyser Anda untuk streaming data
  real-time
