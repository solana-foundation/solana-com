---

## Cargo

Instal rilis beta langsung dari crates.io:

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

Image Docker multi-platform resmi (amd64/arm64) dipublikasikan ke GitHub
Container Registry:

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

Tag yang tersedia:

- `latest` — Rilis stabil terbaru
- `beta` — Versi pra-rilis terbaru
- `v<version>` — Versi spesifik (misalnya, `v2.2.0-beta.7`)

### Docker Compose

Untuk build khusus atau untuk menyertakan Redis untuk caching dan dukungan batas
penggunaan:

- [Contoh Dockerfile](https://github.com/solana-foundation/kora/blob/main/examples/deploy/Dockerfile)
- [Docker Compose dengan Redis](https://github.com/solana-foundation/kora/blob/main/docker-compose.yml)

### Menjalankan dengan Docker

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## TypeScript SDK

SDK beta tersedia di npm:

```bash
pnpm add @solana/kora@beta
```

Lihat [Perubahan SDK](/docs/tools/kora/beta/sdk-changes) untuk API baru.

## Verifikasi Instalasi

```bash
kora --version
```
