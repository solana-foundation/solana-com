_Tersedia di GitHub:
[`v2.2.0-beta.7`](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.7)_

Kora v2.2.0-beta memperkenalkan dukungan bundle Jito, metode RPC baru, image
Docker, fitur keamanan yang ditingkatkan, dan perbaikan SDK.

> **Ini adalah pra-rilis.** Fitur yang didokumentasikan di sini tersedia di
> `v2.2.0-beta.7` dan dapat berubah sebelum rilis stabil. Untuk dokumentasi
> stabil, lihat [dokumentasi utama Kora](/docs/tools/kora).

## Sorotan

### Dukungan Bundle Jito

Kirim hingga 5 transaksi secara atomik melalui block engine Jito. Metode RPC
baru:

- [`signBundle`](/docs/tools/kora/beta/bundle-methods#signbundle) — Tanda
  tangani bundle tanpa menyiarkannya
- [`signAndSendBundle`](/docs/tools/kora/beta/bundle-methods#signandsendbundle)
  — Tanda tangani dan kirim ke Jito
- [`estimateBundleFee`](/docs/tools/kora/beta/bundle-methods#estimatebundlefee)
  — Estimasi total biaya bundle

Lihat [Panduan Bundle Jito](/docs/tools/kora/beta/jito-guide) untuk panduan
lengkap.

### Image Docker

Image resmi multi-platform (amd64/arm64) dipublikasikan ke GitHub Container
Registry. Lihat [Instalasi](/docs/tools/kora/beta/installation).

### Plugin Transaksi

Sistem plugin baru memungkinkan Kora untuk memvalidasi struktur transaksi selama
proses penandatanganan. Plugin bawaan pertama, `gas_swap`, memberlakukan aturan
ketat untuk transaksi swap token-untuk-SOL tanpa gas. Plugin dikonfigurasi di
`kora.toml` dan berjalan secara otomatis selama `signTransaction`,
`signAndSendTransaction`, `signBundle`, dan `signAndSendBundle`.

Lihat
[Konfigurasi — Plugin Transaksi](/docs/tools/kora/beta/configuration#transaction-plugins).

### Opsi Konfigurasi Baru

- **Plugin Transaksi** untuk validasi struktur transaksi (`gas_swap`)
- **reCAPTCHA v3** perlindungan bot untuk endpoint penandatanganan
- **Lighthouse** perlindungan pengurasan fee payer
- **Batas Penggunaan** pembatasan transaksi per-wallet
- **Konfigurasi bundle** untuk integrasi Jito

Lihat [Konfigurasi](/docs/tools/kora/beta/configuration).

### Perubahan SDK

- Dukungan metode bundle (`signBundle`, `signAndSendBundle`,
  `estimateBundleFee`)

Lihat [Perubahan SDK](/docs/tools/kora/beta/sdk-changes).

### Metode RPC Tambahan

- [`getVersion`](/docs/tools/kora/beta/bundle-methods#getversion) — Ambil
  informasi versi server

## Instalasi

```bash
cargo install kora-cli@2.2.0-beta.7
```

Atau melalui Docker:

```bash
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```
