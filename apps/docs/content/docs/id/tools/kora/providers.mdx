Tidak ingin menjalankan node Kora sendiri? Penyedia berikut menawarkan
infrastruktur Kora terkelola sehingga Anda dapat menambahkan abstraksi biaya ke
aplikasi Anda tanpa perlu mengelola deployment sendiri.

## Penyedia

### Openfort

[Openfort](https://www.openfort.io/) menawarkan infrastruktur paymaster Solana
yang didukung oleh Kora, memungkinkan transaksi tanpa gas melalui platform
embedded wallet mereka. Integrasi mereka menggunakan pola co-signing di mana
Kora bertindak sebagai pembayar biaya sementara pengguna menandatangani melalui
embedded wallet Openfort.

- **Website**: [openfort.io](https://www.openfort.io/)
- **Dokumentasi**:
  [Transaksi Tanpa Gas Solana](https://www.openfort.io/blog/solana-gasless-transactions)
- **Fitur**: Embedded wallet, transaksi tanpa gas, sponsorship biaya

---

## Menjadi Operator Node Kora

Kora adalah open source dan siapa pun dapat menjalankan node. Baik Anda sedang
membangun aplikasi yang membutuhkan abstraksi biaya atau ingin menawarkan
layanan paymaster kepada orang lain, lihat
[Panduan Operator](/docs/tools/kora/operators) untuk memulai.
