## Kora v2.2.0-beta.6

_23-03-2026_ |
[Rilis GitHub](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.6)
| `cargo install kora-cli@2.2.0-beta.6`

Pra-rilis dengan dukungan bundle Jito, metode RPC baru, dan fitur keamanan yang
ditingkatkan.

- Ditambahkan dukungan bundle Jito dengan metode `signBundle`,
  `signAndSendBundle`, dan `estimateBundleFee`
- Ditambahkan metode RPC `getVersion`
- Ditambahkan perlindungan pembayar biaya Lighthouse
- Ditambahkan perlindungan bot reCAPTCHA v3 untuk endpoint penandatanganan
- Ditambahkan batas penggunaan per-dompet dengan dukungan Redis
- Ditambahkan image Docker resmi di GHCR
- Ditambahkan aturan batas penggunaan terperinci
- SDK: `KoraClient` sekarang menerima objek opsi; `koraPlugin` baru untuk
  `@solana/kit`

Lihat [Dokumentasi Beta](/docs/tools/kora/beta) untuk detail lebih lanjut.

## Kora TypeScript SDK v0.2.0

_09-03-2026_ |
[Rilis GitHub](https://github.com/solana-foundation/kora/releases/tag/ts-sdk-v0.2.0)
| `pnpm add @solana/kora@0.2.0`

Arsitektur plugin klien Kit untuk transaksi tanpa biaya yang dapat digabungkan.

- Ditambahkan `createKitKoraClient()` dengan komposisi plugin Kit (planner,
  executor, plugin payer)
- Memungkinkan komposabilitas dengan plugin program Kit seperti `tokenProgram()`
- Ditambahkan dukungan Token-2022 melalui opsi konfigurasi `tokenProgramId`
- Ditambahkan dukungan instruksi anggaran komputasi dengan estimasi CU berbasis
  simulasi
- Ditambahkan field `signature` yang hilang ke respons `signAndSendTransaction`
- Mengembalikan `NoopSigner` dari `getPaymentInstruction` untuk deduplikasi
  penandatangan Kit v6
- Meningkatkan dependensi peer `@solana/kit` ke `^6.1.0`

Lihat [Panduan Klien Kit](/docs/tools/kora/guides/kit-client).

## Kora v2.0.0

_2025-11-19_ |
[Rilis GitHub](https://github.com/solana-foundation/kora/releases/tag/v2.0.0) |
`cargo install kora-cli@2.0.0`

Rilis utama dengan arsitektur baru.

- Memperkenalkan arsitektur baru untuk node Kora
- Menerbitkan TypeScript SDK `@solana/kora` v0.1.0
- Migrasi ke Solana SDK v3.0.x dan memperbarui semua dependensi
- Mengganti Kora Signer khusus dengan crate solana-signers
- Menghapus metode `sign_transaction_if_paid` dan `get_all_signers`
- Mengubah tipe respons: menghapus tanda tangan dari `signAndSendTransaction`
  dan `signTransaction`
- Meningkatkan pengalaman pengembang, keamanan, dan performa
