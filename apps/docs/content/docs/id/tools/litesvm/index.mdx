---
title: LiteSVM
description:
  Pengujian program Solana yang sangat cepat dengan VM dalam proses. Uji program
  Anda lebih cepat tanpa overhead jaringan atau proses validator.
---

LiteSVM adalah pustaka ringan untuk menguji program Solana yang bekerja dengan
membuat VM Solana dalam proses yang dioptimalkan untuk pengembang program. Tidak
seperti pendekatan pengujian lain yang menjalankan proses validator terpisah,
LiteSVM menyematkan VM di dalam pengujian Anda, sehingga eksekusinya jauh lebih
cepat. Sebagai terobosan lebih lanjut dari kebiasaan yang ada, LiteSVM hadir
dengan API yang ergonomis, pengaturan default yang wajar, dan kemampuan
konfigurasi yang luas bagi yang membutuhkannya.
