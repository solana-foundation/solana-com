## Menjalankan Contoh

Semua contoh dapat dijalankan dengan `tsx`, meskipun beberapa alamat/variabel
memerlukan penggantian placeholder dengan nilai yang sebenarnya.

```bash
# Install dependencies
pnpm add @solana/kit @solana/kit-plugin-litesvm @solana/kit-plugin-signer @solana-program/system

# Run an example
npx tsx example.ts
```
