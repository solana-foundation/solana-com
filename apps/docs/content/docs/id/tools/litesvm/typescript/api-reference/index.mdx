## Referensi Cepat

### Pengaturan Plugin

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### Manajemen Akun (client.svm)

| Metode                                 | Deskripsi                        |
| -------------------------------------- | -------------------------------- |
| `setAccount(account)`                  | Atur status akun secara langsung |
| `getAccount(address)`                  | Ambil data akun                  |
| `getBalance(address)`                  | Dapatkan saldo SOL               |
| `airdrop(address, lamports)`           | Danai sebuah akun                |
| `minimumBalanceForRentExemption(size)` | Hitung minimum bebas sewa        |

### Manajemen Program (client.svm)

| Metode                                | Deskripsi                  |
| ------------------------------------- | -------------------------- |
| `addProgram(programId, bytes)`        | Muat program dari bytes    |
| `addProgramFromFile(programId, path)` | Muat program dari file .so |

### Transaksi (client.svm)

| Metode                    | Deskripsi                      |
| ------------------------- | ------------------------------ |
| `sendTransaction(tx)`     | Jalankan sebuah transaksi      |
| `simulateTransaction(tx)` | Simulasi tanpa perubahan state |
| `latestBlockhash()`       | Dapatkan blockhash saat ini    |
| `expireBlockhash()`       | Lanjutkan ke blockhash baru    |

### Konfigurasi (client.svm)

| Metode                      | Deskripsi                                    |
| --------------------------- | -------------------------------------------- |
| `withSigverify(bool)`       | Aktifkan/nonaktifkan verifikasi tanda tangan |
| `withBlockhashCheck(bool)`  | Aktifkan/nonaktifkan validasi blockhash      |
| `withSysvars()`             | Aktifkan akun sysvar                         |
| `withBuiltins()`            | Aktifkan program bawaan                      |
| `withPrecompiles()`         | Aktifkan program prakompi                    |
| `withTransactionHistory(n)` | Simpan n transaksi terakhir                  |
| `withComputeBudget(budget)` | Atur batas komputasi                         |
| `withLamports(amount)`      | Atur lamport default                         |
| `withLogBytesLimit(n)`      | Atur batas output log                        |

### Waktu & Sysvar (client.svm)

| Method                     | Deskripsi                      |
| -------------------------- | ------------------------------ |
| `warpToSlot(slot)`         | Lompat ke slot tertentu        |
| `getClock()`               | Dapatkan sysvar clock          |
| `getRent()`                | Dapatkan sysvar rent           |
| `getEpochSchedule()`       | Dapatkan jadwal epoch          |
| `getLastRestartSlot()`     | Dapatkan slot restart terakhir |
| `setLastRestartSlot(slot)` | Atur slot restart terakhir     |

### RPC (client.rpc)

| Method                           | Deskripsi                      |
| -------------------------------- | ------------------------------ |
| `getAccountInfo(address)`        | Ambil satu akun (hanya base64) |
| `getMultipleAccounts(addresses)` | Ambil beberapa akun            |
| `getLatestBlockhash()`           | Dapatkan blockhash saat ini    |
