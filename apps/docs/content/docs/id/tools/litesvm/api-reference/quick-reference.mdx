## Referensi Cepat

| Method                                         | Deskripsi                                                                                     |
| ---------------------------------------------- | --------------------------------------------------------------------------------------------- |
| `LiteSVM::new()`                               | Membuat lingkungan pengujian dasar LiteSVM baru                                               |
| `LiteSVM::default()`                           | Mengembalikan "nilai default" untuk suatu tipe                                                |
| `airdrop(&Address, lamports)`                  | Mendanai akun dengan lamport yang ditentukan                                                  |
| `get_account(&Address)`                        | Mengembalikan semua informasi yang terkait dengan akun dari alamat yang diberikan             |
| `set_account(Address, Account)`                | Menetapkan semua informasi yang terkait dengan akun dari alamat yang diberikan                |
| `get_balance(&Address)`                        | Mendapatkan saldo lamport dari alamat akun yang diberikan                                     |
| `minimum_balance_for_rent_exemption(data_len)` | Mengembalikan saldo minimum yang diperlukan agar akun dengan panjang data tertentu bebas rent |
| `add_program(program_id, &bytes)`              | Menambahkan program SBF ke lingkungan pengujian                                               |
| `add_program_from_file(program_id, path)`      | Menambahkan program SBF ke lingkungan pengujian dari file yang ditentukan                     |
| `send_transaction(tx)`                         | Mengirimkan transaksi yang telah ditandatangani                                               |
| `simulate_transaction(tx)`                     | Mensimulasikan transaksi tanpa mengubah state                                                 |
| `get_transaction(&signature)`                  | Mendapatkan transaksi dari riwayat                                                            |
| `latest_blockhash()`                           | Mendapatkan blockhash saat ini                                                                |
| `expire_blockhash()`                           | Memaksa kedaluwarsa blockhash pada blockhash saat ini                                         |
| `warp_to_slot(slot)`                           | Memundurkan jam ke slot yang ditentukan                                                       |
| `get_sysvar<T>()`                              | Mendapatkan sysvar dari lingkungan pengujian                                                  |
| `set_sysvar<T>(&value)`                        | Menetapkan sysvar ke lingkungan pengujian                                                     |
| `with_feature_accounts()`                      | Menyertakan akun feature gate dalam lingkungan pengujian                                      |
| `set_invocation_inspect_callback(callback)`    | Mendaftarkan callback invokasi pra/pasca (memerlukan fitur `invocation-inspect-callback`)     |
