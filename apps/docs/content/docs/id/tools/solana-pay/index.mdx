**Solana Pay adalah protokol standar dan seperangkat implementasi referensi yang
memungkinkan pengembang untuk mengintegrasikan pembayaran terdesentralisasi ke
dalam aplikasi dan layanan mereka.** Blockchain Solana mengonfirmasi transaksi
dalam waktu kurang dari satu detik dan biayanya rata-rata $0,0005, memberikan
pengalaman yang mulus bagi pengguna tanpa perantara.

## Mengapa Solana Pay?

- **Protokol Standar**: Pendekatan terpadu untuk pembayaran di semua dompet
  Solana dan aplikasi
- **Sangat Cepat**: Transaksi dikonfirmasi dalam waktu kurang dari satu detik
- **Sangat Murah**: Biaya transaksi rata-rata $0,0005
- **Tanpa Perantara**: Pembayaran peer-to-peer langsung tanpa pihak ketiga
- **Dukungan Dompet Universal**: Kompatibel dengan Phantom, Solflare, Backpack,
  dan banyak lagi

## Mulai Cepat

Cara termudah untuk memulai adalah dengan URL permintaan transfer:

```
solana:recipient?amount=0.01&label=Coffee%20Shop&message=Grande%20Latte
```

URL ini dapat:

- Dikodekan sebagai kode QR untuk pemindaian dompet seluler
- Digunakan sebagai tautan pembayaran di aplikasi web
- Disematkan dalam aplikasi seluler sebagai deep link

**[→ Mulai Cepat Permintaan Transfer](/docs/tools/solana-pay/quickstart/transfer-requests)** -
Buat URL pembayaran sederhana

**[→ Mulai Cepat Permintaan Transaksi](/docs/tools/solana-pay/quickstart/transaction-requests)** -
Bangun alur pembayaran interaktif

**[→ Ringkasan](/docs/tools/solana-pay/overview)** - Ringkasan detail dan

## Instalasi

Instal SDK JavaScript Solana Pay:

```bash
# Using pnpm (recommended)
pnpm add @solana/pay@beta @solana/kit

# Using npm
npm install @solana/pay@beta @solana/kit

# Using yarn
yarn add @solana/pay@beta @solana/kit
```

## Kasus Penggunaan

Solana Pay memungkinkan berbagai skenario pembayaran:

- **Pembayaran Ritel**: Pembayaran langsung menggunakan kode QR
- **E-commerce**: Alur checkout web dengan integrasi dompet
- **Aplikasi Seluler**: Pengalaman pembayaran native dengan deep linking
- **Pencetakan NFT**: Pembuatan transaksi dinamis untuk interaksi kompleks
- **Langganan**: Pembayaran berulang dan program loyalitas
- **Integrasi DeFi**: Transaksi yang dapat digabungkan dengan swap dan lending

## Dompet yang Didukung

Solana Pay berfungsi dengan semua dompet Solana utama:

- **[Phantom](https://phantom.app/)** - Dompet Solana paling populer
- **[Solflare](https://solflare.com/)** - Dompet Solana multi-platform
- **[Backpack](https://backpack.app/)** - Dompet Web3 dan toko aplikasi
- **[Glow](https://glow.app/)** - Dompet Solana berbasis mobile
- **[Ultimate](https://ultimate.app/)** - Dompet fokus DeFi
- **[Decaf Wallet](https://decaf.so/)** - Dompet Pembayaran Global
- **[Espresso Cash](https://www.espressocash.com/)** - Dompet Pembayaran Global
- **[Tiplink](https://tiplink.io/)** - Dompet paling sederhana di dunia
- **[Bitget Wallet](https://bitget.com/)** - Dompet multi-chain
