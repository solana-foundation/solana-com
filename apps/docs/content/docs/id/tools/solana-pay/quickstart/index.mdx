Mulai dengan cepat menggunakan Solana Pay dengan panduan langkah demi langkah
ini.

## Instalasi

Siapkan SDK Solana Pay dan konfigurasikan lingkungan pengembangan Anda.

**[→ Panduan Instalasi](/docs/tools/solana-pay/quickstart/installation)** -
Instal SDK dan dependensi

## Jenis Pembayaran

Pilih integrasi pembayaran yang tepat untuk kasus penggunaan Anda:

### Permintaan Transfer (Sederhana)

Buat URL pembayaran non-interaktif untuk transfer SOL dan token SPL dasar.

**[→ Permintaan Transfer](/docs/tools/solana-pay/quickstart/transfer-requests)** -
URL pembayaran sederhana

**Terbaik untuk:**

- Pembayaran dan donasi dasar
- Pembayaran kode QR
- Tautan pembayaran di email

### Permintaan Transaksi (Lanjutan)

Bangun alur pembayaran interaktif dengan komposisi transaksi sisi server.

**[→ Permintaan Transaksi](/docs/tools/solana-pay/quickstart/transaction-requests)** -
Alur pembayaran interaktif

**Terbaik untuk:**

- Penetapan harga dinamis
- Minting NFT
- Transaksi DeFi kompleks
- Logika bisnis khusus

## Integrasi Kode QR

Generate dan sesuaikan kode QR untuk pembayaran dompet mobile.

**[→ Integrasi Kode QR](/docs/tools/solana-pay/quickstart/qr-codes)** - Generate
kode pembayaran yang dapat dipindai

**Terbaik untuk:**

- Sistem point of sale
- Pembayaran dari web ke mobile
- Transaksi langsung

## Langkah Selanjutnya

Setelah menyelesaikan panduan cepat:

- **[Ringkasan](/docs/tools/solana-pay/overview)** - Ringkasan mendetail dan
  spesifikasi

  SDK dan konfigurasi
- **[Kode QR](/docs/tools/solana-pay/quickstart/qr-codes)** - Buat kode
  pembayaran yang dapat dipindai
