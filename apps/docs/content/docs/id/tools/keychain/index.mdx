**Solana Keychain menyediakan antarmuka terpadu untuk menandatangani transaksi
Solana di berbagai backend manajemen kunci.** Kembangkan secara lokal dengan
kunci in-memory, lalu beralih ke manajemen kunci enterprise, cloud HSM, atau
layanan dompet terkelola di produksi — tanpa perlu menulis ulang kode
penandatanganan Anda.

### Mengapa Keychain?

- **Antarmuka Tunggal**: Satu trait `SolanaSigner` bekerja di semua backend
- **Ganti Backend**: Ubah penyedia manajemen kunci melalui konfigurasi, bukan
  kode
- **Abstraksi Tanpa Biaya**: Feature flag hanya menyertakan apa yang Anda
  butuhkan
- **Diaudit**: Ditinjau keamanannya oleh [Accretion](https://accretion.xyz)

### Arsitektur

- **Bahasa**: Rust + TypeScript, dengan paritas penuh
- **Trait**: Antarmuka `SolanaSigner` terpadu
- **Backend**: Memory, Vault, AWS KMS, GCP KMS, Privy, Turnkey, Fireblocks, CDP,
  Crossmint, Dfns, Openfort, Para, Utila
- **Kompatibilitas**: `@solana/kit` dan `@solana/signers` kompatibel
  (TypeScript) | `solana-sdk` dan `solana-sdk-v3` kompatibel (Rust)

### Backend yang Didukung

Backend mencakup pengembangan lokal, manajemen kunci yang di-host sendiri dan
berbasis cloud, serta layanan dompet terkelola atau MPC. Lihat
[Memilih backend](/docs/tools/keychain/choosing-a-backend) untuk membandingkan
model kustodi dan memilih yang tepat.

| Backend         | Kasus Penggunaan              | Rust | TypeScript |
| --------------- | ----------------------------- | ---- | ---------- |
| Memory          | Pengembangan, pengujian       | ✓    | ✓          |
| HashiCorp Vault | Manajemen kunci self-hosted   | ✓    | ✓          |
| AWS KMS         | Cloud-native (AWS)            | ✓    | ✓          |
| GCP KMS         | Cloud-native (GCP)            | ✓    | ✓          |
| Privy           | Dompet tertanam               | ✓    | ✓          |
| Turnkey         | Manajemen kunci non-kustodial | ✓    | ✓          |
| Fireblocks      | Kustodi institusional         | ✓    | ✓          |
| CDP             | Coinbase Developer Platform   | ✓    | ✓          |
| Crossmint       | Dompet terkelola Crossmint    | ✓    | ✓          |
| Dfns            | Infrastruktur dompet Dfns     | ✓    | ✓          |
| Openfort        | Dompet backend Openfort       | ✓    | ✓          |

## Mulai Cepat

### Rust

Instal crate Rust:

```bash
cargo add solana-keychain
```

Penggunaan dasar:

```rust
use solana_keychain::{Signer, SolanaSigner};

// Create a signer from any backend
let signer = Signer::from_memory("base58_private_key")?;

// All signers share the same interface
let pubkey = signer.pubkey();
let signature = signer.sign_transaction(&mut tx).await?;
```

- [Panduan Rust](/docs/tools/keychain/getting-started/rust) - Instalasi lengkap
  dan konfigurasi backend
- [Crates.io](https://crates.io/crates/solana-keychain) - Rust crate

### TypeScript

Instal paket TypeScript:

```bash
pnpm add @solana/keychain
```

Penggunaan dasar:

```typescript
import { createKeychainSigner } from "@solana/keychain";
import { signTransactionWithSigners } from "@solana/signers";

// Create any signer via the unified factory
const signer = await createKeychainSigner({
  backend: "vault",
  vaultAddr: "https://vault.example.com:8200",
  vaultToken: "hvs.xxxxx",
  keyName: "my-solana-key",
  publicKey: "base58_public_key"
});

// Sign an already-compiled transaction
const signedTx = await signTransactionWithSigners(
  [signer],
  compiledTransaction
);
```

- [Panduan TypeScript](/docs/tools/keychain/getting-started/typescript) - paket
  `@solana/keychain`
- [Paket npm](https://www.npmjs.com/package/@solana/keychain) - paket npm
