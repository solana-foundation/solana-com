Mulai dengan Solana Attestations dengan menginstal paket-paket yang diperlukan
dan menyiapkan lingkungan pengembangan Anda.

## TypeScript/JavaScript

Instal library Solana Attestations menggunakan package manager pilihan Anda:

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
