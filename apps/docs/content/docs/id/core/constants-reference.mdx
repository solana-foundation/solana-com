Halaman ini adalah referensi tunggal untuk semua konstanta runtime yang
disebutkan di seluruh halaman konsep inti. Setiap konstanta tertaut ke definisi
sumbernya di klien validator
[agave](https://github.com/anza-xyz/agave/tree/v3.1.8) atau
[solana-sdk](https://github.com/anza-xyz/solana-sdk).
