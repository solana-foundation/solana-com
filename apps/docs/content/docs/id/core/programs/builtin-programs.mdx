## System Program

System Program (`11111111111111111111111111111111`) adalah satu-satunya program
yang dapat membuat akun baru. Semua akun baru awalnya dimiliki oleh System
Program, meskipun kepemilikan biasanya dialihkan saat pembuatan. System Program
mengonsumsi
[`DEFAULT_COMPUTE_UNITS`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L282)
(150 CU) per instruksi.

### Referensi instruksi System Program

| Instruksi                                                                                                              | Deskripsi                                                                                                                                                                                                      |
| ---------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [`CreateAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L295)          | Membuat akun baru. Mengalokasikan `space` byte, menetapkan `owner`, dan mentransfer `lamports` dari pembayar. Akun baru dan pembayar harus menandatangani.                                                     |
| [`CreateAccountWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L318)  | Sama seperti `CreateAccount`, tetapi menurunkan alamat akun baru dari pubkey akun, string seed, dan program pemilik. Akun yang pubkey-nya digunakan untuk menurunkan alamat dan pembayar harus menandatangani. |
| [`Assign`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L343)                 | Menetapkan pemilik akun. Akun harus menandatangani. Tidak ada operasi jika sudah menjadi pemilik tersebut.                                                                                                     |
| [`AssignWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L484)         | Sama seperti `Assign`, tetapi untuk alamat yang diturunkan dari seed. Akun yang pubkey-nya digunakan untuk menurunkan alamat harus menandatangani.                                                             |
| [`Transfer`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L353)               | Mentransfer lamport antar akun. Pengirim harus menandatangani.                                                                                                                                                 |
| [`TransferWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L357)       | Sama seperti `Transfer`, tetapi dari alamat yang diturunkan dari seed. Akun yang pubkey-nya digunakan untuk menurunkan alamat harus menandatangani.                                                            |
| [`Allocate`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L452)               | Menetapkan panjang data akun. Akun harus menandatangani, memiliki data kosong, dan dimiliki oleh System Program. Maksimum 10 MiB.                                                                              |
| [`AllocateWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L462)       | Sama seperti `Allocate`, tetapi untuk alamat yang diturunkan dari seed. Juga menetapkan pemilik. Akun yang pubkey-nya digunakan untuk menurunkan alamat harus menandatangani.                                  |
| [`AdvanceNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L374)    | Memajukan nilai nonce ke blockhash saat ini. Otoritas nonce harus menandatangani.                                                                                                                              |
| [`InitializeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L412) | Menginisialisasi akun sebagai akun nonce dengan otoritas yang diberikan. Akun harus bebas rent.                                                                                                                |
| [`AuthorizeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L432)  | Mengubah otoritas nonce. Otoritas saat ini harus menandatangani.                                                                                                                                               |
| [`WithdrawNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L392)   | Menarik lamport dari akun nonce. Otoritas harus menandatangani. Jika tidak menutup, saldo yang tersisa harus menutupi rent.                                                                                    |
| [`UpgradeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L437)    | Meningkatkan akun nonce dari versi lama ke format versi saat ini. Akun harus dapat ditulis dan berisi versi nonce lama.                                                                                        |

## Program loader

Setiap program dimiliki oleh program loader. Loader menangani deployment,
upgrade, dan eksekusi program yang mereka miliki. Lihat
[Program Loader](/docs/core/programs/program-deployment#loader-programs) untuk
daftar lengkap loader dan alamatnya.
