## Alat pengembangan AI

Mulai dengan panduan kami tentang
[cara memulai dengan alat AI di Solana](/docs/tools/ai).

### Framework AI Agent

AI agent dapat secara otonom mengeksekusi transaksi pembayaran di Solana.
Framework ini memungkinkan pembangunan agent yang memproses pembayaran, memantau
transaksi, dan mengotomatisasi alur kerja pembayaran:

| Framework        | Deskripsi                                                                                         | Tautan                                                                                    |
| ---------------- | ------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | Alat untuk membangun agent dengan kemampuan pembayaran, kompatibel dengan LangChain dan Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | Kumpulan skill siap pakai untuk membangun agent di Solana                                         | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | Framework agent multi-platform dengan plugin Solana untuk bot X, Telegram, dan Discord            | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Toolkit lintas rantai dengan aksi pembayaran Solana                                               | [GitHub](https://github.com/goat-sdk/goat)                                                |
