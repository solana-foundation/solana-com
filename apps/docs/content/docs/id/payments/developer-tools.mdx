## Alat pengembangan AI

Mulai dengan panduan kami tentang
[cara memulai dengan alat AI di Solana](/developers/guides/getstarted/intro-to-ai).

### Solana MCP

[Solana MCP](https://mcp.solana.com/) (Machine Conversation Protocol)
mengintegrasikan pengetahuan Solana langsung ke dalam alat coding yang didukung
AI seperti Claude, Cursor, dan Windsurf. Ini dilatih pada dokumentasi Solana,
framework Anchor, contoh program, dan Stack Exchange Q&A - berguna untuk
membangun integrasi pembayaran lebih cepat.

### Framework agen AI

Agen AI dapat secara otonom mengeksekusi transaksi pembayaran di Solana.
Framework ini memungkinkan pembuatan agen yang memproses pembayaran, memantau
transaksi, dan mengotomatisasi alur kerja pembayaran:

| Framework        | Deskripsi                                                                                            | Tautan                                                                                    |
| ---------------- | ---------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | Alat untuk membangun agen yang mampu melakukan pembayaran, kompatibel dengan LangChain dan Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | Koleksi skill yang sudah jadi untuk membangun agen di Solana                                         | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | Framework agen multi-platform dengan plugin Solana untuk bot X, Telegram, Discord                    | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Toolkit lintas rantai dengan aksi pembayaran Solana                                                  | [GitHub](https://github.com/goat-sdk/goat)                                                |

### Pengembangan program AI
