---
title: Pemrosesan pembayaran
description:
  Proses pembayaran dalam skala besar dengan transaksi tanpa gas dan abstraksi
  biaya
---

Memproses pembayaran dalam skala besar—penggajian, pembayaran merchant, hadiah
loyalitas—memerlukan lebih dari sekadar mengirim transfer individual.

Untuk memproses banyak pembayaran secara efisien:

1. **Batch instruksi** ke dalam transaksi tunggal untuk mengurangi biaya dan
   latensi
2. **Batch transaksi** untuk memparalelkan throughput di seluruh jaringan
3. **Abstraksi biaya** sehingga pengguna tidak perlu memegang SOL
