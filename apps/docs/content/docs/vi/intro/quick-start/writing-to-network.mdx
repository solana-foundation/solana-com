Trong phần trước, bạn đã học cách
[đọc dữ liệu từ mạng Solana](/docs/intro/quick-start/reading-from-network) bằng
cách truy xuất các tài khoản. Ghi dữ liệu lên mạng Solana yêu cầu một
[giao dịch](/docs/core/transactions). Một giao dịch chứa một hoặc nhiều
[lệnh](/docs/core/instructions), và mỗi lệnh gọi một chương trình.

Các chương trình xác định logic nghiệp vụ cho từng lệnh. Khi bạn gửi một giao
dịch, Solana runtime thực thi các lệnh trong giao dịch theo thứ tự. Giao dịch
mang tính nguyên tử. Hoặc là mọi lệnh trong giao dịch đều thành công, hoặc toàn
bộ giao dịch thất bại.

Các ví dụ trong phần này hướng dẫn cách:

1. Chuyển SOL giữa các tài khoản
2. Tạo một token mint mới

## Chuyển SOL

Ví dụ dưới đây chuyển SOL từ tài khoản này sang tài khoản khác. Chỉ chương trình
được chỉ định làm chủ sở hữu của tài khoản mới có thể sửa đổi dữ liệu của tài
khoản đó hoặc khấu trừ lamport từ số dư của nó. Các tài khoản ví được sở hữu bởi
[System Program](/docs/core/programs#the-system-program), vì vậy việc chuyển SOL
giữa các tài khoản ví yêu cầu một lệnh gọi đến lệnh
[transfer](https://github.com/anza-xyz/agave/blob/v2.1.11/programs/system/src/system_processor.rs#L183-L213)
của System Program. Tài khoản nguồn cũng phải ký giao dịch.

<WithNotes>

<CodeTabs flags="r">

```ts !! title="Transfer SOL"
import { createClient, generateKeyPairSigner, lamports } from "@solana/kit";
import { solanaRpc, rpcAirdrop } from "@solana/kit-plugin-rpc";
import { generatedPayer, airdropPayer } from "@solana/kit-plugin-signer";
import { getTransferSolInstruction } from "@solana-program/system";

// !tooltip[/createClient/] client
const client = await createClient()
  .use(generatedPayer())
  .use(
    solanaRpc({
      rpcUrl: "http://localhost:8899",
      rpcSubscriptionsUrl: "ws://localhost:8900"
    })
  )
  .use(rpcAirdrop())
  .use(airdropPayer(lamports(1_000_000_000n)));

// !tooltip[/receiver/] receiver
const receiver = await generateKeyPairSigner();

// !tooltip[/transferInstruction/] transferInstruction
const transferInstruction = getTransferSolInstruction({
  source: client.payer,
  destination: receiver.address,
  amount: lamports(10_000_000n)
});

// !tooltip[/sendTransaction/] sendTransaction
const result = await client.sendTransaction([transferInstruction]);

console.log("Transaction Signature:", result.context.signature);

// !tooltip[/balances/] balances
const { value: senderBalance } = await client.rpc
  .getBalance(client.payer.address)
  .send();
const { value: receiverBalance } = await client.rpc
  .getBalance(receiver.address)
  .send();

console.log("Sender Balance:", senderBalance);
console.log("Receiver Balance:", receiverBalance);
```

</CodeTabs>

### !client

Tạo một Kit client cho validator cục bộ. `generatedPayer()` thêm một payer
signer, `solanaRpc()` thêm khả năng RPC và gửi giao dịch, `rpcAirdrop()` thêm
`client.airdrop`, và `airdropPayer()` nạp SOL test vào tài khoản payer.

### !receiver

Tạo một signer cho tài khoản nhận SOL. Người nhận không cần ký giao dịch chuyển
này. Nếu địa chỉ người nhận chưa có tài khoản, việc nhận SOL sẽ tạo một tài
khoản hệ thống tại địa chỉ đó.

### !transferInstruction

Xây dựng lệnh System Program để chuyển lamport từ người thanh toán của client
đến người nhận.

### !sendTransaction

Gửi lệnh dưới dạng một giao dịch. Client Kit tải một blockhash gần đây, đặt
người thanh toán phí, ký với các người ký yêu cầu, gửi giao dịch và chờ xác
nhận.

### !balances

Lấy số dư của người gửi và người nhận sau khi giao dịch được xác nhận.

</WithNotes>

## Tạo một token

Ví dụ bên dưới tạo một mint token mới bằng cách sử dụng
[Token Extensions Program](/docs/tokens/extensions). Một mint account là tài
khoản xác định các cài đặt toàn cục của token, chẳng hạn như số thập phân, nguồn
cung, quyền phát hành và quyền đóng băng.

Tạo một mint account yêu cầu hai lệnh:

1. Gọi System Program để tạo một tài khoản mới thuộc sở hữu của Token Extensions
   Program.
2. Gọi Token Extensions Program để khởi tạo tài khoản đó như một mint.

<WithNotes>

<CodeTabs flags="r">

```ts !! title="Create mint account"
import { createClient, generateKeyPairSigner, lamports } from "@solana/kit";
import { solanaRpc, rpcAirdrop } from "@solana/kit-plugin-rpc";
import { generatedPayer, airdropPayer } from "@solana/kit-plugin-signer";
import { getCreateAccountInstruction } from "@solana-program/system";
import {
  fetchMint,
  getInitializeMintInstruction,
  getMintSize,
  TOKEN_2022_PROGRAM_ADDRESS
} from "@solana-program/token-2022";

// !tooltip[/createClient/] client
const client = await createClient()
  .use(generatedPayer())
  .use(
    solanaRpc({
      rpcUrl: "http://localhost:8899",
      rpcSubscriptionsUrl: "ws://localhost:8900"
    })
  )
  .use(rpcAirdrop())
  .use(airdropPayer(lamports(1_000_000_000n)));

// !tooltip[/mint/] mint
const mint = await generateKeyPairSigner();

// !tooltip[/space/] space
const space = BigInt(getMintSize());

// !tooltip[/rent/] rent
const rent = await client.rpc.getMinimumBalanceForRentExemption(space).send();

// !tooltip[/createAccountInstruction/] createAccountInstruction
const createAccountInstruction = getCreateAccountInstruction({
  payer: client.payer,
  newAccount: mint,
  space,
  lamports: rent,
  programAddress: TOKEN_2022_PROGRAM_ADDRESS
});

// !tooltip[/initializeMintInstruction/] initializeMintInstruction
const initializeMintInstruction = getInitializeMintInstruction({
  mint: mint.address,
  decimals: 2,
  mintAuthority: client.payer.address,
  freezeAuthority: client.payer.address,
  tokenProgram: TOKEN_2022_PROGRAM_ADDRESS
});

// !tooltip[/sendTransaction/] sendTransaction
const result = await client.sendTransaction([
  createAccountInstruction,
  initializeMintInstruction
]);

console.log("Mint Address:", mint.address);
console.log("Transaction Signature:", result.context.signature);

// !tooltip[/mintAccount/] mintAccount
const mintAccount = await fetchMint(client.rpc, mint.address);
console.log("Mint Account:", mintAccount);
```

</CodeTabs>

### !client

Tạo và nạp tiền cho một Kit client trên validator cục bộ. Người thanh toán đã
được nạp tiền sẽ chi trả rent tài khoản và phí giao dịch.

### !mint

Tạo một signer để sử dụng làm địa chỉ của mint account mới. Signer ủy quyền việc
tạo tài khoản đó.

### !space

Tính toán số byte cần thiết cho một mint account không có extension.

### !rent

Tính toán số lamport cần thiết để làm cho mint account được miễn rent.

### !createAccountInstruction

Xây dựng lệnh System Program tạo mint account và gán tài khoản cho Token
Extensions Program.

### !initializeMintInstruction

Xây dựng lệnh Token Extensions Program ghi dữ liệu mint ban đầu vào tài khoản
mới.

### !sendTransaction

Gửi cả hai lệnh trong một giao dịch. Thứ tự lệnh rất quan trọng: tài khoản phải
được tạo trước khi có thể được khởi tạo.

### !mintAccount

Lấy mint account sau khi xác nhận. Hàm trợ giúp đọc tài khoản và giải mã trường
dữ liệu của mint account thành kiểu Mint.

</WithNotes>

```ts !! title="Create mint account"
import { createClient, generateKeyPairSigner, lamports } from "@solana/kit";
import { solanaRpc, rpcAirdrop } from "@solana/kit-plugin-rpc";
import { generatedPayer, airdropPayer } from "@solana/kit-plugin-signer";
import { getCreateAccountInstruction } from "@solana-program/system";
import {
  fetchMint,
  getInitializeMintInstruction,
  getMintSize,
  TOKEN_2022_PROGRAM_ADDRESS
} from "@solana-program/token-2022";

const client = await createClient()
  .use(generatedPayer())
  .use(
    solanaRpc({
      rpcUrl: "http://localhost:8899",
      rpcSubscriptionsUrl: "ws://localhost:8900"
    })
  )
  .use(rpcAirdrop())
  .use(airdropPayer(lamports(1_000_000_000n)));

const mint = await generateKeyPairSigner();

const space = BigInt(getMintSize());

const rent = await client.rpc.getMinimumBalanceForRentExemption(space).send();

const createAccountInstruction = getCreateAccountInstruction({
  payer: client.payer,
  newAccount: mint,
  space,
  lamports: rent,
  programAddress: TOKEN_2022_PROGRAM_ADDRESS
});

const initializeMintInstruction = getInitializeMintInstruction({
  mint: mint.address,
  decimals: 2,
  mintAuthority: client.payer.address,
  freezeAuthority: client.payer.address,
  tokenProgram: TOKEN_2022_PROGRAM_ADDRESS
});

// !focus(1:4)
const result = await client.sendTransaction([
  createAccountInstruction,
  initializeMintInstruction
]);

console.log("Mint Address:", mint.address);
console.log("Transaction Signature:", result.context.signature);
```
