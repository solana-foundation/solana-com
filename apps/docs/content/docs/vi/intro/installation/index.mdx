Cài đặt nhanh để thiết lập môi trường phát triển Solana cục bộ của bạn chỉ với
một lệnh. Cài đặt Rust, Solana CLI và Anchor Framework trên Windows (WSL), Linux
và Mac. (Nếu muốn, bạn có thể
[cài đặt từng phụ thuộc riêng lẻ](/docs/intro/installation/dependencies).)

## Yêu cầu trước

<Accordions>
<Accordion title="Windows">

Để phát triển các chương trình Solana trên Windows, bạn **phải** sử dụng
[Windows Subsystem for Linux](https://learn.microsoft.com/en-us/windows/wsl/install)
(WSL). Sau khi cài đặt WSL, tất cả các phụ thuộc khác sẽ được cài đặt thông qua
terminal Linux.

Để cài đặt WSL, chạy lệnh sau trong Windows PowerShell:

```terminal
$ wsl --install
```

Quá trình cài đặt sẽ yêu cầu bạn tạo tài khoản người dùng mặc định.

![Cài đặt WSL](/assets/docs/intro/installation/wsl-install.png)

Theo mặc định, WSL cài đặt Ubuntu. Bạn có thể mở terminal Linux bằng cách tìm
kiếm "Ubuntu" trong thanh tìm kiếm.

![WSL Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-search.png)

Nếu terminal Ubuntu của bạn trông giống như hình ảnh bên dưới, bạn có thể gặp sự
cố khi `ctrl + v` (phím tắt dán) không hoạt động trong terminal.

![Terminal Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-terminal-1.png)

Nếu bạn gặp sự cố này, hãy mở Windows Terminal bằng cách tìm kiếm "Terminal"
trong thanh tìm kiếm.

![Windows Terminal](/assets/docs/intro/installation/wsl-windows-terminal.png)

Tiếp theo, đóng Windows Terminal và mở lại terminal Linux bằng cách tìm kiếm
Ubuntu một lần nữa. Terminal bây giờ sẽ trông giống như hình ảnh bên dưới, nơi
`ctrl + v` (phím tắt dán) hoạt động.

![Terminal Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-terminal-2.png)

Nếu bạn sử dụng VS Code,
[tiện ích mở rộng WSL](https://code.visualstudio.com/docs/remote/wsl-tutorial)
cho phép bạn sử dụng WSL và VS Code cùng nhau.

![Thiết lập WSL trong VS Code](/assets/docs/intro/installation/wsl-vscode.png)

Bạn sẽ thấy nội dung sau trong thanh trạng thái của VS Code:

![WSL: Ubuntu](/assets/docs/intro/installation/wsl-vscode-ubuntu.png)

Sau khi đã thiết lập WSL, bạn đã sẵn sàng để:

- Hoàn thành [cài đặt nhanh](#installation) bằng terminal Ubuntu (Linux)
- Cài đặt [Rust](/docs/intro/installation/dependencies#install-rust),
  [Solana CLI](/docs/intro/installation/dependencies#install-solana-cli) và
  [Anchor CLI](/docs/intro/installation/dependencies#install-anchor-cli) riêng
  lẻ, như mô tả bên dưới

</Accordion>

<Tabs items={[".deb", ".rpm"]}>
<Tab value=".deb" >
Nếu bạn đang sử dụng Debian hoặc các bản phái sinh từ Debian như Ubuntu hoặc Mint, hãy làm theo các bước dưới đây:

1. Cập nhật danh sách gói để đảm bảo tất cả các gói đều được cập nhật:

```terminal
$ sudo apt-get update
```

2. Cài đặt các phụ thuộc sau:

```terminal
$ sudo apt-get install -y \
    build-essential \
    pkg-config \
    libudev-dev llvm libclang-dev \
    protobuf-compiler libssl-dev
```

<Callout>
Nếu bạn gặp lỗi sau khi cài đặt `protobuf-compiler`, hãy
đảm bảo bạn chạy _shell`sudo apt-get update` trước

```
Package protobuf-compiler is not available, but is referred to by another package.
This may mean that the package is missing, has been obsoleted, or
is only available from another source
```

</Callout>

  </Tab>

    <Tab value=".rpm" >
    Đối với các bản phái sinh RHEL như Fedora hoặc Red Hat Linux, hãy làm theo các bước sau.

Cập nhật danh sách gói để lấy phiên bản mới hơn của các gói bằng cách chạy lệnh.

```terminal
$ sudo dnf upgrade --refresh
```

Tiếp theo, cài đặt các phụ thuộc sau:

```terminal
$ sudo dnf group install c-development development-tools \
&& sudo dnf install pkg-config openssl-devel
```

## Cài đặt

1. Nhập lệnh sau vào terminal của bạn để cài đặt tất cả các phụ thuộc cần thiết:

```terminal
$ curl --proto '=https' --tlsv1.2 -sSfL https://solana-install.solana.workers.dev | bash
```

Cài đặt thành công sẽ trả về kết quả như sau:

```txt title="Example output"
Installed Versions:
Rust: rustc 1.91.1 (ed61e7d7e 2025-11-07)
Solana CLI: solana-cli 3.0.10 (src:96c3a851; feat:3604001754, client:Agave)
Anchor CLI: anchor-cli 0.32.1
Surfpool CLI: surfpool 0.12.0
Node.js: v24.10.0
Yarn: 1.22.1
```

2. Xác minh cài đặt thành công bằng cách kiểm tra phiên bản của từng phụ thuộc
   đã cài đặt.

```terminal
$ rustc --version && solana --version && anchor --version && surfpool --version && node --version && yarn --version
```

<Callout>
  Nếu lệnh cài đặt nhanh thất bại, vui lòng tham khảo phần [Cài đặt các phụ
  thuộc](/docs/intro/installation/dependencies) bên dưới để biết hướng dẫn cài
  đặt từng phụ thuộc riêng lẻ.
</Callout>
