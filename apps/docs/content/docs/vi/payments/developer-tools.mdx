## Công cụ phát triển AI

Bắt đầu với hướng dẫn của chúng tôi về
[cách bắt đầu với các công cụ AI trên Solana](/developers/guides/getstarted/intro-to-ai).

### Solana MCP

[Solana MCP](https://mcp.solana.com/) (Machine Conversation Protocol) tích hợp
kiến thức Solana trực tiếp vào các công cụ lập trình hỗ trợ AI như Claude,
Cursor, và Windsurf. Nó được huấn luyện trên tài liệu Solana, framework Anchor,
các ví dụ chương trình, và Q&A Stack Exchange - hữu ích để xây dựng tích hợp
thanh toán nhanh hơn.

### Framework AI agent

AI agent có thể tự động thực thi các giao dịch thanh toán trên Solana. Các
framework này cho phép xây dựng agent xử lý thanh toán, giám sát giao dịch, và
tự động hóa quy trình thanh toán:

| Framework        | Mô tả                                                                                 | Liên kết                                                                                      |
| ---------------- | ------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| Solana Agent Kit | Công cụ xây dựng agent có khả năng thanh toán, tương thích với LangChain và Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Tài liệu](https://kit.sendai.fun/) |
| Send AI Skills   | Bộ sưu tập các kỹ năng được xây dựng sẵn để tạo agent trên Solana                     | [GitHub](https://github.com/sendaifun/skills)                                                 |
| Eliza Framework  | Framework agent đa nền tảng với plugin Solana cho bot X, Telegram, Discord            | [GitHub](https://github.com/elizaOS/eliza) • [Tài liệu](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Bộ công cụ đa chuỗi với các hành động thanh toán Solana                               | [GitHub](https://github.com/goat-sdk/goat)                                                    |

### Phát triển chương trình AI

[Codigo](https://codigo.ai/) là một nền tảng phát triển AI Solana toàn diện.
