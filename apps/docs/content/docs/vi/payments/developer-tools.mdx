## Công cụ phát triển AI

Bắt đầu với hướng dẫn của chúng tôi về

### Framework Tác Nhân AI

Các tác nhân AI có thể tự động thực hiện giao dịch thanh toán trên Solana. Những
framework này cho phép xây dựng các tác nhân xử lý thanh toán, giám sát giao
dịch, và tự động hóa quy trình thanh toán:

| Framework        | Mô Tả                                                                                    | Liên Kết                                                                                  |
| ---------------- | ---------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | Công cụ xây dựng tác nhân có khả năng thanh toán, tương thích với LangChain và Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | Bộ sưu tập các kỹ năng được xây dựng sẵn để phát triển tác nhân trên Solana              | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | Framework tác nhân đa nền tảng với plugin Solana cho bot X, Telegram, Discord            | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Bộ công cụ chuỗi chéo với các hành động thanh toán Solana                                | [GitHub](https://github.com/goat-sdk/goat)                                                |

### Phát Triển Chương Trình AI
