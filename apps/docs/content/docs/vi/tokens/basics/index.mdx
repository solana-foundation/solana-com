Phần SPL Token Cơ bản trình bày các lệnh cơ bản để tương tác với SPL Token, tập
trung vào các tính năng được chia sẻ bởi cả Token Program gốc và Token
Extensions Program (Token 2022).

<Callout type="info">
  Tham khảo [tài liệu Metaplex](https://developers.metaplex.com/token-metadata)
  để biết thêm thông tin về cách thêm metadata vào SPL Token.
</Callout>
