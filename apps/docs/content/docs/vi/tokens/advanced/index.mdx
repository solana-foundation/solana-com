Ngoài các thao tác tạo, mint và chuyển token phổ biến, Token Program còn được
gọi từ các chương trình on-chain thông qua CPI và bao gồm các lệnh để thu hồi
lamport:

<Cards>
  <Card
    title="Gọi Token Program qua CPI"
    href="/docs/tokens/advanced/cpi"
  >
    Gọi Token Program từ một chương trình on-chain — chuyển, mint và đốt
    token, ký bằng quyền PDA, và gộp các lệnh vào một CPI duy nhất.
  </Card>

<Card
  title="Rút Lamport Dư Thừa"
  href="/docs/tokens/advanced/withdraw-excess-lamports"
>
  Thu hồi SOL vượt mức tối thiểu miễn phí thuê từ token account, mint và tài
  khoản multisig mà không thay đổi số dư token.
</Card>

  <Card title="Giải Nén Lamport" href="/docs/tokens/advanced/unwrap-lamports">
    Chuyển đổi SOL đã bọc trở lại thành lamport, toàn bộ hoặc một phần, mà không đóng
    token account.
  </Card>
</Cards>

<Callout type="info">

_rs`WithdrawExcessLamports`_ và _rs`UnwrapLamports`_ có sẵn thông qua
[`@solana-program/token`](https://www.npmjs.com/package/@solana-program/token)
client. Việc gọi Token Program qua CPI áp dụng cho các chương trình on-chain.

</Callout>

## Thu hồi lamport

_rs`WithdrawExcessLamports`_ thu hồi SOL vượt mức tối thiểu miễn phí thuê từ
token account, mint và tài khoản multisig. _rs`UnwrapLamports`_ chuyển đổi SOL
đã bọc trong một native token account trở lại thành lamport.
