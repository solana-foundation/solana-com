Phát triển frontend trên Solana liên quan đến việc làm việc với các chương
trình, ví và các framework JavaScript phổ biến như React. Tương tác với những
thành phần này yêu cầu xử lý kết nối, tạo giao dịch và đọc dữ liệu từ các tài
khoản Solana.

Để hỗ trợ công việc này, nhiều thư viện client Solana có sẵn trong các framework
khác nhau.

## Thư viện chính

<Cards>
  <Card
    title="@solana/client"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/client"
  >
    - Client Solana đơn giản tích hợp RPC, ví, giao dịch - Bao gồm state store
    tích hợp sẵn, actions, watchers, connectors
  </Card>
  <Card
    title="@solana/react-hooks"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks"
  >
    - Bộ hooks đầy đủ cho ví, số dư, chuyển khoản, chữ ký, truy vấn - Hooks
    provider React bao bọc `@solana/client` runtime state
  </Card>
  <Card
    title="@solana/web3-compat"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/web3-compat"
  >
    - Bộ công cụ tương thích Web3.js để đơn giản hóa việc nâng cấp. - Cấu trúc
    nội bộ mới hơn dựa trên sự kết hợp của web3.js và kit.
  </Card>
  <Card
    title="@solana/kit"
    href="https://github.com/solana-foundation/framework-kit/tree/main/packages/kit"
  >
    - SDK Solana cấp thấp cung cấp sức mạnh cho các thư viện Solana khác như
    `@solana/react-hooks` - Hoàn toàn tree-shakable, sử dụng các tiêu chuẩn web
    hiện đại, và cung cấp sức mạnh cho runtime
  </Card>
</Cards>

<Callout type="warn" title="@solana/web3.js đã ngừng hỗ trợ`">
  Nhiều dự án trong hệ sinh thái Solana vẫn dựa vào `@solana/web3.js` đã ngừng
  hỗ trợ. Ưu tiên sử dụng `@solana/web3-compat` để đơn giản hóa lộ trình di
  chuyển của bạn.
</Callout>

- [Hướng dẫn @solana/client](/docs/frontend/client): runtime gọn nhẹ, headless
  cho RPC, ví và giao dịch.
- [Hướng dẫn @solana/react-hooks](/docs/frontend/react-hooks): các hooks React
  được xây dựng trên cùng client runtime.
- [Hướng dẫn @solana/web3-compat](/docs/frontend/web3-compat): lớp tương thích
  để di chuyển từ `@solana/web3.js` sang các stack được hỗ trợ bởi Kit.
