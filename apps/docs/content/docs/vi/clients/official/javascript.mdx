## Solana Client

`@solana/client` là runtime Solana headless tích hợp RPC, ví, giao dịch và
subscriptions trong một store duy nhất.

| Package        | Mô tả                   | GitHub                                                                                 |
| -------------- | ----------------------- | -------------------------------------------------------------------------------------- |
| @solana/client | Headless client runtime | [Source](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [Hướng dẫn: tổng quan về @solana/client](/docs/frontend/client)

## Solana React Hooks

`@solana/react-hooks` cung cấp React provider và hooks trên nền tảng
`@solana/client`, tái sử dụng cùng runtime và cache.

| Package             | Mô tả                          | GitHub                                                                                      |
| ------------------- | ------------------------------ | ------------------------------------------------------------------------------------------- |
| @solana/react-hooks | React provider + hooks cho UIs | [Source](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [Hướng dẫn: @solana/react-hooks](/docs/frontend/react-hooks)

## Solana Kit

`@solana/kit` là TypeScript SDK được khuyến nghị để xây dựng trên Solana.

| Gói                            | Mô tả                                     | GitHub                                                                            |
| ------------------------------ | ----------------------------------------- | --------------------------------------------------------------------------------- |
| @solana/kit                    | SDK cốt lõi                               | [Mã nguồn](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | Tương tác với System Program              | [Mã nguồn](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Tương tác với Token Program               | [Mã nguồn](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Tương tác với chương trình Token-2022     | [Mã nguồn](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Tương tác với chương trình Memo           | [Mã nguồn](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Tương tác với chương trình Compute Budget | [Mã nguồn](https://github.com/solana-program/compute-budget/tree/main/clients/js) |
| @solana/kit-plugins            | Plugin cho Kit                            | [Mã nguồn](https://github.com/anza-xyz/kit-plugins)                               |

- [Tài liệu chính thức về Kit](https://www.solanakit.com/)
