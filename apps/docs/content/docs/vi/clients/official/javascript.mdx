## Solana Kit

`@solana/kit` là TypeScript SDK được khuyến nghị để xây dựng trên Solana. Bạn
tập hợp một client từ các plugin có thể kết hợp, thêm các liên kết React khi
cần, và tích hợp các program client được tạo sẵn cho các on-chain program mà bạn
gọi.

| Package                   | Mô tả                                        | GitHub                                                                                   |
| ------------------------- | -------------------------------------------- | ---------------------------------------------------------------------------------------- |
| @solana/kit               | SDK cốt lõi: RPC, giao dịch, codec, signer   | [Mã nguồn](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | Liên kết React: `ClientProvider` và các hook | [Mã nguồn](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | Kết nối RPC + gửi giao dịch                  | [Mã nguồn](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | Plugin fee payer, identity và signer         | [Mã nguồn](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Kết nối Wallet Standard + React hook         | [Mã nguồn](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

`tokenProgram()` → `client.token`).

| @solana-program/token          | Tương tác với Token Program               | [Mã nguồn](https://github.com/solana-program/token/tree/main/clients/js)          |

khóa khác nhau:

| Package          | Mô tả                                                                                                                 | GitHub                                                           |
| ---------------- | --------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------- |
| @solana/keychain | Ký hợp nhất: Memory, Vault, Privy, Turnkey, AWS KMS, Fireblocks, GCP KMS, CDP, Para, Dfns, Crossmint, Openfort, Utila | [Mã nguồn](https://github.com/solana-foundation/solana-keychain) |

Xem [tài liệu Keychain](/docs/tools/keychain) và
[Chọn backend](/docs/tools/keychain/choosing-a-backend) để biết hướng dẫn thiết
lập và mô hình quản lý khóa.

## Solana Web3.js

`@solana/web3.js` là SDK TypeScript cũ cho Solana. Các ứng dụng mới nên sử dụng
`@solana/kit`; xem [Migrating to Kit](/docs/frontend/web3-compat).

| Package           | Mô tả                                                                | GitHub                                                                             |
| ----------------- | -------------------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| @solana/web3.js   | SDK cốt lõi                                                          | [Source](https://github.com/solana-foundation/solana-web3.js)                      |
| @solana/spl-token | Tương tác với các chương trình Token, Token-2022 và Associated Token | [Source](https://github.com/solana-program/token-2022/tree/main/clients/js-legacy) |
| @solana/spl-memo  | Tương tác với chương trình Memo                                      | [Source](https://github.com/solana-program/memo/tree/main/clients/js-legacy)       |
