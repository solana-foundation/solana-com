## Tham khảo nhanh

### Thiết lập Plugin

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### Quản lý tài khoản (client.svm)

| Phương thức                            | Mô tả                                 |
| -------------------------------------- | ------------------------------------- |
| `setAccount(account)`                  | Đặt trạng thái tài khoản trực tiếp    |
| `getAccount(address)`                  | Truy xuất dữ liệu tài khoản           |
| `getBalance(address)`                  | Lấy số dư SOL                         |
| `airdrop(address, lamports)`           | Nạp tiền vào tài khoản                |
| `minimumBalanceForRentExemption(size)` | Tính mức tối thiểu miễn phí tiền thuê |

### Quản lý chương trình (client.svm)

| Phương thức                           | Mô tả                       |
| ------------------------------------- | --------------------------- |
| `addProgram(programId, bytes)`        | Tải chương trình từ bytes   |
| `addProgramFromFile(programId, path)` | Tải chương trình từ tệp .so |

### Giao dịch (client.svm)

| Phương thức               | Mô tả                                 |
| ------------------------- | ------------------------------------- |
| `sendTransaction(tx)`     | Thực thi một giao dịch                |
| `simulateTransaction(tx)` | Mô phỏng mà không thay đổi trạng thái |
| `latestBlockhash()`       | Lấy blockhash hiện tại                |
| `expireBlockhash()`       | Chuyển sang blockhash mới             |

### Cấu hình (client.svm)

| Phương thức                 | Mô tả                                |
| --------------------------- | ------------------------------------ |
| `withSigverify(bool)`       | Bật/tắt xác minh chữ ký              |
| `withBlockhashCheck(bool)`  | Bật/tắt xác thực blockhash           |
| `withSysvars()`             | Bật tài khoản sysvar                 |
| `withBuiltins()`            | Bật các chương trình tích hợp sẵn    |
| `withPrecompiles()`         | Bật các chương trình biên dịch trước |
| `withTransactionHistory(n)` | Lưu trữ n giao dịch gần nhất         |
| `withComputeBudget(budget)` | Đặt giới hạn tính toán               |
| `withLamports(amount)`      | Đặt số lamport mặc định              |
| `withLogBytesLimit(n)`      | Đặt giới hạn đầu ra log              |

### Thời Gian & Sysvars (client.svm)

| Phương thức                | Mô tả                            |
| -------------------------- | -------------------------------- |
| `warpToSlot(slot)`         | Chuyển đến slot cụ thể           |
| `getClock()`               | Lấy clock sysvar                 |
| `getRent()`                | Lấy rent sysvar                  |
| `getEpochSchedule()`       | Lấy lịch epoch                   |
| `getLastRestartSlot()`     | Lấy slot khởi động lại cuối cùng |
| `setLastRestartSlot(slot)` | Đặt slot khởi động lại cuối cùng |

### RPC (client.rpc)

| Phương thức                      | Mô tả                          |
| -------------------------------- | ------------------------------ |
| `getAccountInfo(address)`        | Lấy một tài khoản (chỉ base64) |
| `getMultipleAccounts(addresses)` | Lấy nhiều tài khoản            |
| `getLatestBlockhash()`           | Lấy blockhash hiện tại         |
