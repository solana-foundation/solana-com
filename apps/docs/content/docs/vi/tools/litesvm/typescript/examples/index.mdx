## Chạy Ví dụ

Tất cả các ví dụ đều có thể chạy bằng `tsx`, tuy nhiên một số địa chỉ/biến cần
được thay thế các giá trị giữ chỗ bằng giá trị thực tế.

```bash
# Install dependencies
pnpm add @solana/kit @solana/kit-plugin-litesvm @solana/kit-plugin-signer @solana-program/system

# Run an example
npx tsx example.ts
```
