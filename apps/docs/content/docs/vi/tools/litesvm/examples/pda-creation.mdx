## Kiểm Tra Hoàn Chỉnh

```rust title="tests/pda_test.rs"
use litesvm::LiteSVM;
use solana_account::Account;
use solana_sdk::pubkey::Pubkey;

#[test]
fn test_pda_creation() {
    let mut svm = LiteSVM::new();

    // Program that owns the PDA
    let program_id = Pubkey::new_unique();

    // Derive PDA address
    let seed = b"my_pda";
    let (pda, bump) = Pubkey::find_program_address(
        &[seed],
        &program_id
    );

    println!("PDA: {}", pda);
    println!("Bump: {}", bump);

    // Create PDA account manually (for testing)
    svm.set_account(pda, Account {
        lamports: 1_000_000,
        data: vec![bump], // Store bump seed in data
        owner: program_id,
        executable: false,
        rent_epoch: 0,
    }).unwrap();

    // Verify PDA was created
    let account = svm.get_account(&pda).unwrap();
    assert_eq!(account.owner, program_id);
    assert_eq!(account.data[0], bump);

    println!("PDA created successfully!");
}
```

## Điểm Chính

1. **Dẫn Xuất PDA**: Sử dụng `Pubkey::find_program_address()` để dẫn xuất địa
   chỉ PDA
2. **Seeds**: Các PDA được dẫn xuất từ seeds và ID chương trình
3. **Bump Seed**: bump seed đảm bảo địa chỉ nằm ngoài đường cong ed25519
4. **Tạo Tài Khoản**: Sử dụng `set_account()` để tạo tài khoản thủ công cho mục
   đích kiểm thử
5. **Quyền Sở Hữu**: Các PDA phải thuộc sở hữu của chương trình đã dẫn xuất
   chúng
