Bắt đầu với Solana Attestations bằng cách cài đặt các gói cần thiết và thiết lập
môi trường phát triển của bạn.

## TypeScript/JavaScript

Cài đặt thư viện Solana Attestations bằng trình quản lý gói mà bạn ưa thích:

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
