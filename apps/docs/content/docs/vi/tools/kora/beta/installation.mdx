## Cargo

Cài đặt phiên bản beta trực tiếp từ crates.io:

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

Các image Docker chính thức đa nền tảng (amd64/arm64) được xuất bản lên GitHub
Container Registry:

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

Các tag có sẵn:

- `latest` — Phiên bản ổn định mới nhất
- `beta` — Phiên bản pre-release mới nhất
- `v<version>` — Phiên bản cụ thể (ví dụ: `v2.2.0-beta.7`)

### Docker Compose

Để build tùy chỉnh hoặc tích hợp Redis cho bộ nhớ đệm và hỗ trợ giới hạn sử
dụng:

- [Dockerfile mẫu](https://github.com/solana-foundation/kora/blob/main/examples/deploy/Dockerfile)
- [Docker Compose với Redis](https://github.com/solana-foundation/kora/blob/main/docker-compose.yml)

### Chạy với Docker

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## TypeScript SDK

SDK phiên bản beta có sẵn trên npm:

```bash
pnpm add @solana/kora@beta
```

Xem [Thay đổi SDK](/docs/tools/kora/beta/sdk-changes) để biết các API mới.

## Xác minh cài đặt

```bash
kora --version
```
