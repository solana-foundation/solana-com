_Có sẵn trên GitHub:
[`v2.2.0-beta.7`](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.7)_

Kora v2.2.0-beta giới thiệu hỗ trợ Jito bundle, các phương thức RPC mới, Docker
images, các tính năng bảo mật nâng cao và cải tiến SDK.

> **Đây là phiên bản tiền phát hành.** Các tính năng được mô tả ở đây có sẵn
> trong `v2.2.0-beta.7` và có thể thay đổi trước khi phát hành chính thức. Để
> xem tài liệu ổn định, hãy truy cập [tài liệu Kora chính](/docs/tools/kora).

## Điểm nổi bật

### Hỗ trợ Jito Bundle

Gửi tối đa 5 giao dịch một cách nguyên tử thông qua block engine của Jito. Các
phương thức RPC mới:

- [`signBundle`](/docs/tools/kora/beta/bundle-methods#signbundle) — Ký một
  bundle mà không phát sóng
- [`signAndSendBundle`](/docs/tools/kora/beta/bundle-methods#signandsendbundle)
  — Ký và gửi đến Jito
- [`estimateBundleFee`](/docs/tools/kora/beta/bundle-methods#estimatebundlefee)
  — Ước tính tổng phí bundle

Xem [Hướng dẫn Jito Bundle](/docs/tools/kora/beta/jito-guide) để biết chi tiết
đầy đủ.

### Docker Images

Các image chính thức đa nền tảng (amd64/arm64) được xuất bản lên GitHub
Container Registry. Xem [Cài đặt](/docs/tools/kora/beta/installation).

### Transaction Plugins

Một hệ thống plugin mới cho phép Kora xác thực cấu trúc giao dịch trong quá
trình ký. Plugin tích hợp đầu tiên, `gas_swap`, áp dụng các quy tắc nghiêm ngặt
cho các giao dịch hoán đổi token-lấy-SOL không tốn gas. Các plugin được cấu hình
trong `kora.toml` và chạy tự động trong `signTransaction`,
`signAndSendTransaction`, `signBundle`, và `signAndSendBundle`.

Xem
[Cấu hình — Transaction Plugins](/docs/tools/kora/beta/configuration#transaction-plugins).

### Các tùy chọn cấu hình mới

- **Transaction Plugins** để xác thực cấu trúc giao dịch (`gas_swap`)
- **reCAPTCHA v3** bảo vệ chống bot cho các endpoint ký
- **Lighthouse** bảo vệ chống rút cạn fee payer
- **Usage Limits** giới hạn giao dịch theo ví
- **Cấu hình Bundle** để tích hợp Jito

Xem [Cấu hình](/docs/tools/kora/beta/configuration).

### Thay đổi SDK

- Hỗ trợ phương thức bundle (`signBundle`, `signAndSendBundle`,
  `estimateBundleFee`)

Xem [Thay đổi SDK](/docs/tools/kora/beta/sdk-changes).

### Các phương thức RPC bổ sung

- [`getVersion`](/docs/tools/kora/beta/bundle-methods#getversion) — Lấy thông
  tin phiên bản máy chủ

## Cài đặt

```bash
cargo install kora-cli@2.2.0-beta.7
```

Hoặc qua Docker:

```bash
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```
