## Kora v2.2.0-beta.6

_23-03-2026_ |
[Phát hành trên GitHub](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.6)
| `cargo install kora-cli@2.2.0-beta.6`

Phiên bản tiền phát hành với hỗ trợ Jito bundle, các phương thức RPC mới và các
tính năng bảo mật nâng cao.

- Đã thêm hỗ trợ Jito bundle với các phương thức `signBundle`,
  `signAndSendBundle` và `estimateBundleFee`
- Đã thêm phương thức RPC `getVersion`
- Đã thêm bảo vệ fee payer của Lighthouse
- Đã thêm bảo vệ bot reCAPTCHA v3 cho các endpoint ký
- Đã thêm giới hạn sử dụng theo từng ví với hỗ trợ Redis
- Đã thêm Docker images chính thức trên GHCR
- Đã thêm quy tắc giới hạn sử dụng chi tiết
- SDK: `KoraClient` hiện nhận đối tượng options; `koraPlugin` mới cho
  `@solana/kit`

Xem [Tài liệu Beta](/docs/tools/kora/beta) để biết chi tiết.

## Kora TypeScript SDK v0.2.0

_09-03-2026_ |
[Phát hành trên GitHub](https://github.com/solana-foundation/kora/releases/tag/ts-sdk-v0.2.0)
| `pnpm add @solana/kora@0.2.0`

Kiến trúc plugin của Kit client cho các giao dịch gasless có thể kết hợp.

- Đã thêm `createKitKoraClient()` với khả năng kết hợp plugin Kit (plugin
  planner, executor, payer)
- Cho phép khả năng kết hợp với các plugin chương trình Kit như `tokenProgram()`
- Đã thêm hỗ trợ Token-2022 thông qua tùy chọn cấu hình `tokenProgramId`
- Đã thêm hỗ trợ compute budget instruction với ước tính CU dựa trên mô phỏng
- Đã thêm trường `signature` còn thiếu vào response `signAndSendTransaction`
- Trả về `NoopSigner` từ `getPaymentInstruction` cho tính năng khử trùng signer
  của Kit v6
- Đã nâng cấp peer dependency `@solana/kit` lên `^6.1.0`

Xem [Hướng dẫn Kit Client](/docs/tools/kora/guides/kit-client).

## Kora v2.0.0

_19-11-2025_ |
[Phiên bản GitHub](https://github.com/solana-foundation/kora/releases/tag/v2.0.0)
| `cargo install kora-cli@2.0.0`

Phiên bản chính với kiến trúc mới.

- Giới thiệu kiến trúc mới cho nút Kora
- Phát hành TypeScript SDK `@solana/kora` v0.1.0
- Chuyển sang Solana SDK v3.0.x và cập nhật tất cả các phụ thuộc
- Thay thế Kora Signer tùy chỉnh bằng crate solana-signers
- Loại bỏ các phương thức `sign_transaction_if_paid` và `get_all_signers`
- Thay đổi các kiểu phản hồi: loại bỏ chữ ký khỏi `signAndSendTransaction` và
  `signTransaction`
- Nâng cao trải nghiệm phát triển, bảo mật và hiệu suất
