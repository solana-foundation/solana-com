---
title: getBlockhash
description: Lấy blockhash mới nhất từ Solana RPC mà máy chủ Kora đang kết nối.
---

## Yêu cầu JSON-RPC

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getBlockhash",
  "params": []
}
```

## Phản hồi JSON-RPC

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "blockhash": "base58Blockhash"
  }
}
```

## Ví dụ cURL

```bash
curl -X POST http://localhost:8080 \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"getBlockhash","params":[]}'
```

## TypeScript SDK

```typescript
const { blockhash } = await client.getBlockhash();
console.log("Current blockhash:", blockhash);
```
