CẦN: Thiết lập thành phần

## Nội dung tổng quan:

### Phương thức xác thực

- Không có (phát triển)
- API Key
- Chữ ký HMAC
- Xác thực kết hợp

### Xác thực bằng API Key

- Cấu hình
- Định dạng header
- Thiết lập client

### Xác thực HMAC

- Cấu hình secret
- Tạo chữ ký
- Xác thực timestamp
- Triển khai client

### Ví dụ TypeScript

- Thiết lập API key
- Ký HMAC
- Xử lý lỗi

### Phương pháp bảo mật tốt nhất

- Xoay vòng key
- Lưu trữ secret
- Giới hạn tốc độ
- Danh sách IP cho phép

### Khắc phục sự cố

- Lỗi xác thực thường gặp
- Mẹo debug
- Kiểm tra xác thực
