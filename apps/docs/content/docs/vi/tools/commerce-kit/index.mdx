**Commerce Kit là bộ sưu tập các công cụ để xây dựng trải nghiệm thương mại trên
Solana.** Từ các nút thanh toán tích hợp sẵn đến các công cụ cơ bản headless,
Commerce Kit cung cấp mọi thứ bạn cần để chấp nhận thanh toán bằng tiền điện tử.

## Tại sao nên dùng Commerce Kit?

- **Các Component Tích Hợp Sẵn**: PaymentButton được xây dựng sẵn với kết nối
  ví, lựa chọn token và xử lý giao dịch
- **Độc Lập Framework**: Sử dụng các công cụ cơ bản headless trong bất kỳ
  framework JavaScript nào
- **Solana Pay**: Hỗ trợ đầy đủ Solana Pay với tạo mã QR
- **Type-Safe**: Hỗ trợ TypeScript hoàn chỉnh trên tất cả các gói
- **Wallet Standard**: Tương thích với tất cả các ví Wallet Standard (Phantom,
  Solflare, Backpack, v.v.)

## Bắt Đầu Nhanh

Cách nhanh nhất để bắt đầu là với component `PaymentButton`:

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## Cài Đặt

Chọn các gói Commerce Kit bạn cần:

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
