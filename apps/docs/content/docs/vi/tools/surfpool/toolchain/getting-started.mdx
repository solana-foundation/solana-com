---

Surfpool CLI có thể được cài đặt bằng script cài đặt hoặc được biên dịch từ mã
nguồn.

## Script Cài Đặt

Để cài đặt surfpool, mở terminal và chạy:

```bash
curl -sL https://run.surfpool.run/ | bash
```

## Cài Đặt từ Mã Nguồn

Để biên dịch tệp nhị phân Surfpool từ mã nguồn, hãy clone kho lưu trữ surfpool.

```bash
git clone https://github.com/solana-foundation/surfpool.git
```

Điều hướng đến thư mục mới.

```bash
cd surfpool
```

Sau đó, biên dịch tệp nhị phân. Lệnh này sẽ biên dịch tệp nhị phân và lưu trữ nó
trong `~/.cargo/bin/surfpool`.

```bash
cargo surfpool-install
```

Cuối cùng, hãy đảm bảo rằng tệp nhị phân `surfpool` có sẵn trong PATH của bạn.
Quá trình này sẽ khác nhau tùy thuộc vào hệ điều hành của bạn.
