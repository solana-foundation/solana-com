Surfnet là trình mô phỏng mạng Solana cục bộ của Surfpool, được thiết kế để cung
cấp cho các nhà phát triển một môi trường kiểm thử mạnh mẽ, mô phỏng hành vi của
blockchain Solana mà không cần đến chi phí vận hành của một nút validator đầy
đủ.

## Bắt đầu

Khởi động một phiên bản Surfnet bằng Surfpool CLI:

```bash
surfpool start
```

Lệnh này sẽ khởi động một phiên bản Surfnet cục bộ trên `http://localhost:8899`
với tất cả các RPC endpoint khả dụng.

## Tính năng nổi bật

- **Tương thích RPC đầy đủ**: Hỗ trợ toàn bộ Solana RPC API
- **Lặp lại nhanh chóng**: Không cần chờ xác nhận khối
- **Thao tác trạng thái**: Cheatcode để chỉnh sửa trực tiếp trạng thái
  blockchain
- **Fork Mainnet**: Fork từ mainnet để kiểm thử với các tài khoản và chương
  trình thực
- **Hỗ trợ Geyser Plugin**: Kết nối các Geyser plugin của bạn để truyền phát dữ
  liệu theo thời gian thực
