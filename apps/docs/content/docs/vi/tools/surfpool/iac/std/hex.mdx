## encode_hex

`encode_hex` mã hóa một bộ đệm hoặc chuỗi thành chuỗi thập lục phân có tiền tố
0x.

### Đầu vào

| Tên     | Kiểu                  | Mô tả                                                                                                                                |
| ------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| `value` | buffer, string, addon | Bộ đệm hoặc chuỗi cần mã hóa. Các chuỗi bắt đầu bằng '0x' được giải mã dưới dạng hex; ngược lại, các byte UTF-8 thô sẽ được sử dụng. |

### Đầu ra

| Tên     | Kiểu   | Mô tả                                                |
| ------- | ------ | ---------------------------------------------------- |
| `value` | string | Đầu vào ở dạng biểu diễn thập lục phân có tiền tố 0x |

```hcl
output "encoded_hex" {
    value = encode_hex("hello, world")
}
// > encoded_hex: 0x68656c6c6f2c20776f726c64
```

## decode_hex

`decode_hex` giải mã một chuỗi thập lục phân và trả về kết quả dưới dạng bộ đệm.

### Đầu vào

| Tên          | Kiểu   | Mô tả                                               |
| ------------ | ------ | --------------------------------------------------- |
| `hex_string` | string | Chuỗi hex cần giải mã (có hoặc không có tiền tố 0x) |

### Đầu ra

| Tên     | Kiểu   | Mô tả                                 |
| ------- | ------ | ------------------------------------- |
| `value` | buffer | Chuỗi hex đã giải mã dưới dạng bộ đệm |

```hcl
output "decoded_hex" {
    value = decode_hex("0x68656c6c6f2c20776f726c64")
}
// > decoded_hex: 0x68656c6c6f2c20776f726c64
```
