title: Tổng Quan SVM Addon
description: Add-on cho Solana và các Blockchain Tương thích SVM (beta)
---

Plugin `txtx` SVM cho phép xây dựng các Runbook tương tác với Solana và các
blockchain tương thích SVM. Plugin cung cấp các hàm tiện ích cho phép bạn triển
khai các chương trình Anchor và mã hóa các lệnh gọi theo IDL của chương trình.
Các hành động có thể được sử dụng để tạo các giao dịch chuyển tiền, gọi chương
trình và triển khai chương trình hợp lệ, có thể được ký thông qua cụm từ
mnemonic, khóa bí mật hoặc qua trình ký trên trình duyệt của bạn.

## Bắt Đầu Nhanh

Để sử dụng SVM addon, hãy khai báo nó trong Runbook của bạn:

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## Tính Năng

- **Triển Khai Chương Trình**: Triển khai các chương trình Anchor và Solana gốc
- **Xây Dựng Lệnh**: Mã hóa các lệnh gọi sử dụng IDL của chương trình
- **Ký Giao Dịch**: Hỗ trợ mnemonic, khóa bí mật và ví web
- **Quản Lý Tài Khoản**: Truy vấn và quản lý các tài khoản Solana
