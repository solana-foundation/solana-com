## ripemd160

`ripemd160` tính toán giá trị băm Ripemd160 của một giá trị.

### Đầu vào

| Name    | Type                    | Description                    |
| ------- | ----------------------- | ------------------------------ |
| `value` | buffer \| array[buffer] | Giá trị được mã hóa hex để băm |

### Đầu ra

| Name    | Type   | Description    |
| ------- | ------ | -------------- |
| `value` | string | Kết quả đã băm |

```hcl
output "hashed_data" {
    value = ripemd160(encode_hex("hello, world"))
}
// > hashed_data: 0XA3201F82FCA034E46D10CD7B27E174976E241DA2
```

| Name    | Type   | Description    |

| `value` | string | Kết quả đã băm |

```hcl

}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```
