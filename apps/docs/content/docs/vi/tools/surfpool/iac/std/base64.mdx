## encode_base64

`encode_base64` mã hóa một bộ đệm hoặc chuỗi thành chuỗi base64.

### Đầu vào

| Name    | Type                  | Description                                                                                                                             |
| ------- | --------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | Bộ đệm hoặc chuỗi cần mã hóa. Các chuỗi bắt đầu bằng '0x' sẽ được giải mã dưới dạng hex; nếu không, các byte UTF-8 thô sẽ được sử dụng. |

### Đầu ra

| Name    | Type   | Description                                 |
| ------- | ------ | ------------------------------------------- |
| `value` | string | Đầu vào, được mã hóa dưới dạng chuỗi base64 |

```hcl
output "encoded" {
    value = encode_base64("0x48656c6c6f20776f726c6421")
}
// > encoded: SGVsbG8gd29ybGQh
```

## decode_base64

`decode_base64` giải mã một chuỗi được mã hóa base64 và trả về kết quả dưới dạng
bộ đệm.

### Đầu vào

| Name            | Type   | Description              |
| --------------- | ------ | ------------------------ |
| `base64_string` | string | Chuỗi base64 cần giải mã |

### Đầu ra

| Name    | Type   | Description                               |
| ------- | ------ | ----------------------------------------- |
| `value` | buffer | Chuỗi base64 đã giải mã, dưới dạng bộ đệm |

```hcl
output "decoded" {
    value = decode_base64("SGVsbG8gd29ybGQh")
}
// > decoded: 0x48656c6c6f20776f726c6421
```
