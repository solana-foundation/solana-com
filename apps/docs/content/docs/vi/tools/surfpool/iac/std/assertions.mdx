## assert_eq

`assert_eq` kiểm định rằng hai giá trị bằng nhau.

### Đầu vào

| Tên     | Kiểu | Mô tả                  |
| ------- | ---- | ---------------------- |
| `left`  | any  | Một giá trị để so sánh |
| `right` | any  | Giá trị để so sánh với |

### Đầu ra

| Tên     | Kiểu                  | Mô tả                 |
| ------- | --------------------- | --------------------- |
| `value` | addon(std::assertion) | Kết quả của kiểm định |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```
