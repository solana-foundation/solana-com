## jq

Hàm `jq` cho phép cắt lọc, lọc và ánh xạ dữ liệu JSON. Xem
[tài liệu jq](https://jqlang.github.io/jq/manual/) để biết thêm chi tiết.

### Đầu vào

| Tên            | Kiểu             | Mô tả                                                                     |
| -------------- | ---------------- | ------------------------------------------------------------------------- |
| `decoded_json` | string \| object | Một đối tượng JSON                                                        |
| `query`        | string           | Một truy vấn JSON. Xem [tài liệu jq](https://jqlang.github.io/jq/manual/) |

### Đầu ra

| Tên     | Kiểu          | Mô tả                     |
| ------- | ------------- | ------------------------- |
| `value` | array[string] | Kết quả của truy vấn `jq` |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
