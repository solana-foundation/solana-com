## index

`index` lấy mục từ một danh sách tại chỉ mục được chỉ định.

### Đầu vào

| Name    | Type                                             | Description             |
| ------- | ------------------------------------------------ | ----------------------- |
| `list`  | array[string] \| array[integer] \| array[buffer] | Danh sách cần lấy mục   |
| `index` | integer                                          | Chỉ mục của mục cần lấy |

### Đầu ra

| Name    | Type   | Description                                |
| ------- | ------ | ------------------------------------------ |
| `value` | string | Mục từ danh sách tại chỉ mục được chỉ định |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
