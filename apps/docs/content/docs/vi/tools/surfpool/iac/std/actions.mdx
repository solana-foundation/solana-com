Các hành động này có sẵn trong tất cả các Runbook như một phần của thư viện tiêu
chuẩn.

## send_http_request

Hành động `std::send_http_request` thực hiện một yêu cầu HTTP đến URL được chỉ
định.

### Đầu vào

| Tên          | Bắt buộc | Kiểu    | Mô tả                                                    |
| ------------ | -------- | ------- | -------------------------------------------------------- |
| `url`        | bắt buộc | string  | URL để gửi yêu cầu đến                                   |
| `method`     | tùy chọn | string  | Phương thức HTTP (GET, POST, PUT, DELETE). Mặc định: GET |
| `headers`    | tùy chọn | map     | Các tiêu đề HTTP để đưa vào yêu cầu                      |
| `body`       | tùy chọn | string  | Nội dung yêu cầu (dành cho yêu cầu POST/PUT)             |
| `timeout_ms` | tùy chọn | integer | Thời gian chờ yêu cầu tính bằng mili giây                |

### Đầu ra

| Tên           | Kiểu    | Mô tả                |
| ------------- | ------- | -------------------- |
| `status_code` | integer | Mã trạng thái HTTP   |
| `body`        | string  | Nội dung phản hồi    |
| `headers`     | map     | Các tiêu đề phản hồi |

```hcl
action "api_call" "std::send_http_request" {
    description = "Fetch data from API"
    url = "https://api.example.com/data"
    method = "GET"
    headers = {
        "Authorization" = "Bearer ${variable.token}"
    }
}

output "response" {
    value = action.api_call.body
}

output "status" {
    value = action.api_call.status_code
}
```

### Ví dụ về Yêu cầu POST

```hcl
action "create_resource" "std::send_http_request" {
    description = "Create a new resource"
    url = "https://api.example.com/resources"
    method = "POST"
    headers = {
        "Content-Type" = "application/json"
    }
    body = {
        "name" = variable.name,
        "value" = variable.value
    }
}
```
