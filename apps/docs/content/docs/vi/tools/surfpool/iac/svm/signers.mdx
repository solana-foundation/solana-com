title: SVM Signers
description: Signers cho Solana và các Blockchain tương thích SVM

Các signer này khả dụng khi sử dụng addon SVM để ký giao dịch.

## secret_key

Signer `svm::secret_key` có thể được dùng để ký giao dịch đồng bộ bằng secret
key, mnemonic hoặc file keypair.

### Đầu vào

| Name              | Required | Type   | Description                                     |
| ----------------- | -------- | ------ | ----------------------------------------------- |
| `secret_key`      | optional | string | Secret key dùng để ký tin nhắn và giao dịch     |
| `mnemonic`        | optional | string | Cụm từ mnemonic dùng để tạo secret key          |
| `derivation_path` | optional | string | Đường dẫn phát sinh dùng để tạo secret key      |
| `keypair_json`    | optional | string | Đường dẫn đến file keypair.json chứa secret key |

### Đầu ra

| Name         | Type   | Description                          |
| ------------ | ------ | ------------------------------------ |
| `public_key` | string | Public key của tài khoản             |
| `address`    | string | Địa chỉ SVM (bí danh cho public_key) |

```hcl
signer "deployer" "svm::secret_key" {
    secret_key = input.secret_key
}
```

Bạn cũng có thể sử dụng mnemonic:

```hcl
signer "deployer" "svm::secret_key" {
    mnemonic = input.mnemonic
    derivation_path = "m/44'/501'/0'/0'"
}
```

Hoặc một file keypair:

```hcl
signer "deployer" "svm::secret_key" {
    keypair_json = "~/.config/solana/id.json"
}
```

## web_wallet

Signer `svm::web_wallet` cho phép người vận hành Runbook ký giao dịch bằng
browser signer tùy chọn của họ.

### Đầu vào

| Name               | Required | Type   | Description                                             |
| ------------------ | -------- | ------ | ------------------------------------------------------- |
| `expected_address` | optional | string | Địa chỉ SVM dự kiến kết nối (bỏ qua để cho phép bất kỳ) |

### Đầu ra

| Name         | Type   | Description                         |
| ------------ | ------ | ----------------------------------- |
| `address`    | string | Địa chỉ của tài khoản đã kết nối    |
| `public_key` | string | Public key của tài khoản đã kết nối |

```hcl
signer "alice" "svm::web_wallet" {
    expected_address = "zbBjhHwuqyKMmz8ber5oUtJJ3ZV4B6ePmANfGyKzVGV"
}
```

Trình ký này sẽ nhắc người dùng trong Giao diện Web để:

1. Kết nối ví của họ
2. Cung cấp pubkey thông qua chữ ký tin nhắn
3. Ký tất cả các giao dịch yêu cầu trình ký này

## squads

Trình ký `svm::squads` có thể được sử dụng để ký các giao dịch với kho tiền
multisig Squads.

### Đầu vào

| Tên       | Bắt buộc | Kiểu   | Mô tả                  |
| --------- | -------- | ------ | ---------------------- |
| `address` | tùy chọn | string | Địa chỉ kho tiền squad |

### Đầu ra

| Tên          | Kiểu   | Mô tả                                |
| ------------ | ------ | ------------------------------------ |
| `public_key` | string | pubkey của kho tiền squad            |
| `address`    | string | Địa chỉ SVM (bí danh cho public_key) |

```hcl
signer "deployer" "svm::squads" {
    address = input.address
}
```

Sử dụng multisig Squads cho phép ký giao dịch đa bên an toàn mà không cần để lộ
khóa riêng tư của từng cá nhân.
