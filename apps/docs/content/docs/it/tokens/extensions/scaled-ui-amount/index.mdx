## Cos'è l'estensione Importo UI scalato?

L'estensione dell'importo UI scalato consente agli emittenti di applicare un
moltiplicatore aggiornabile all'importo UI di un token. Questo è utile per
scenari in cui il token viene utilizzato per rappresentare un asset del mondo
reale, come un'azione o un dividendo.

Ad esempio, un frazionamento azionario raddoppia il numero di azioni per tutti i
possessori. Senza questa estensione, attualmente è poco pratico emettere nuovi
token a tutti i possessori durante un evento di frazionamento in modo efficiente
e scalabile.

Con il modello di estensione Token-2022, tuttavia, abbiamo la possibilità di
modificare come viene rappresentato l'importo UI dei token. Utilizzando
l'estensione ScaledUiAmount e l'istruzione `amount_to_ui_amount`, puoi impostare
un moltiplicatore UI sul tuo token e recuperare il suo importo UI in qualsiasi
momento.
[codice sorgente](https://github.com/solana-program/token-2022/tree/main/program/src/extension/scaled_ui_amount).

Questa funzionalità può essere utilizzata anche per dividendi o distribuzione di
rendimenti.

Nota: non vengono mai creati nuovi token, l'importo UI restituisce la quantità
grezza di token moltiplicata per il moltiplicatore corrente. La funzionalità è
puramente estetica.

<Callout type="info">
  Alcune estensioni sono incompatibili tra loro e non puoi abilitarle
  simultaneamente sullo stesso token mint o token account. Non dovresti
  abilitare sia l'estensione `ScaledUiAmount` che l'estensione
  `InterestBearingMint` sullo stesso token mint o token account.
</Callout>
