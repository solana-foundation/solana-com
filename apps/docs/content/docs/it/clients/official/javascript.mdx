## Solana Client

`@solana/client` è il runtime Solana headless che raggruppa RPC, wallet,
transazioni e sottoscrizioni in un unico store.

| Package        | Descrizione             | GitHub                                                                                 |
| -------------- | ----------------------- | -------------------------------------------------------------------------------------- |
| @solana/client | Runtime client headless | [Source](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [Guida: panoramica di @solana/client](/docs/frontend/client)

## Solana React Hooks

`@solana/react-hooks` aggiunge un provider React e degli hook sopra
`@solana/client`, riutilizzando lo stesso runtime e cache.

| Package             | Descrizione                     | GitHub                                                                                      |
| ------------------- | ------------------------------- | ------------------------------------------------------------------------------------------- |
| @solana/react-hooks | Provider React + hook per le UI | [Source](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [Guida: @solana/react-hooks](/docs/frontend/react-hooks)

## Solana Kit

`@solana/kit` è l'SDK TypeScript consigliato per sviluppare su Solana.

| Pacchetto                      | Descrizione                                 | GitHub                                                                            |
| ------------------------------ | ------------------------------------------- | --------------------------------------------------------------------------------- |
| @solana/kit                    | SDK principale                              | [Sorgente](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | Interagisci con il System Program           | [Sorgente](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Interagisci con il Token Program            | [Sorgente](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Interagisci con il programma Token-2022     | [Sorgente](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Interagisci con il programma Memo           | [Sorgente](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Interagisci con il programma Compute Budget | [Sorgente](https://github.com/solana-program/compute-budget/tree/main/clients/js) |
| @solana/kit-plugins            | Plugin per Kit                              | [Sorgente](https://github.com/anza-xyz/kit-plugins)                               |

- [Documentazione ufficiale di Kit](https://www.solanakit.com/)
- [Tutorial: Next.js + React Hooks](/docs/frontend/nextjs-solana)

## Firma e gestione delle chiavi

Per la firma backend compatibile con kit su diversi backend di gestione delle
chiavi:

| Pacchetto        | Descrizione                                                                          | GitHub                                                           |
| ---------------- | ------------------------------------------------------------------------------------ | ---------------------------------------------------------------- |
| @solana/keychain | Firma unificata: Memory, Vault, AWS KMS, Privy, Turnkey, Fireblocks, CDP, Dfns, Para | [Sorgente](https://github.com/solana-foundation/solana-keychain) |

Consulta la
[guida Aggiunta di firmatari](https://github.com/solana-foundation/solana-keychain/blob/main/docs/ADDING_SIGNERS.md)
per integrare servizi aggiuntivi di gestione delle chiavi.
