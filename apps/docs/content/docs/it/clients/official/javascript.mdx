## Solana Kit

`@solana/kit` è il TypeScript SDK consigliato per sviluppare su Solana. Puoi
assemblare un client a partire da plugin componibili, aggiungere i binding React
quando necessario e integrare i client di programma generati per i programmi
on-chain che intendi chiamare.

| Pacchetto                 | Descrizione                                     | GitHub                                                                                   |
| ------------------------- | ----------------------------------------------- | ---------------------------------------------------------------------------------------- |
| @solana/kit               | SDK principale: RPC, transazioni, codec, signer | [Sorgente](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | Binding React: `ClientProvider` e hook          | [Sorgente](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | Connessione RPC e invio di transazioni          | [Sorgente](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | Plugin per fee payer, identità e signer         | [Sorgente](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Connessione Wallet Standard e hook React        | [Sorgente](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

## Firma e gestione delle chiavi

Per la firma backend compatibile con kit su più backend di gestione delle
chiavi:

| Pacchetto        | Descrizione                                                                                                               | GitHub                                                           |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------- |
| @solana/keychain | Firma unificata: Memory, Vault, Privy, Turnkey, AWS KMS, Fireblocks, GCP KMS, CDP, Para, Dfns, Crossmint, Openfort, Utila | [Sorgente](https://github.com/solana-foundation/solana-keychain) |

Consulta la [documentazione di Keychain](/docs/tools/keychain) e
[Scegliere un backend](/docs/tools/keychain/choosing-a-backend) per la
configurazione e la guida al modello di custodia.

## Solana Web3.js

`@solana/web3.js` è il legacy TypeScript SDK per Solana. Le nuove app dovrebbero
usare `@solana/kit`; vedi [Migrazione a Kit](/docs/frontend/web3-compat).

| Package           | Description                                                      | GitHub                                                                             |
| ----------------- | ---------------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| @solana/web3.js   | SDK principale                                                   | [Source](https://github.com/solana-foundation/solana-web3.js)                      |
| @solana/spl-token | Interagisci con i programmi Token, Token-2022 e Associated Token | [Source](https://github.com/solana-program/token-2022/tree/main/clients/js-legacy) |
| @solana/spl-memo  | Interagisci con il programma Memo                                | [Source](https://github.com/solana-program/memo/tree/main/clients/js-legacy)       |
