## Strumenti di sviluppo AI

Inizia con la nostra guida su
[come iniziare con gli strumenti AI su Solana](/developers/guides/getstarted/intro-to-ai).

### Solana MCP

[Solana MCP](https://mcp.solana.com/) (Machine Conversation Protocol) integra la
conoscenza di Solana direttamente negli strumenti di coding supportati dall'AI
come Claude, Cursor, e Windsurf. È addestrato sulla documentazione di Solana, il
framework Anchor, esempi di programmi, e domande e risposte di Stack Exchange -
utile per costruire integrazioni di pagamento più velocemente.

### Framework per agenti AI

Gli agenti AI possono eseguire autonomamente transazioni di pagamento su Solana.
Questi framework consentono di costruire agenti che elaborano pagamenti,
monitorano transazioni, e automatizzano i flussi di lavoro dei pagamenti:

| Framework        | Descrizione                                                                                       | Link                                                                                      |
| ---------------- | ------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | Strumenti per costruire agenti capaci di gestire pagamenti, compatibili con LangChain e Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | Una raccolta di competenze pre-costruite per creare agenti su Solana                              | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | Framework per agenti multi-piattaforma con plugin Solana per bot X, Telegram, Discord             | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Toolkit cross-chain con azioni di pagamento Solana                                                | [GitHub](https://github.com/goat-sdk/goat)                                                |
