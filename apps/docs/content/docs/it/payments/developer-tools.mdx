### Framework per Agenti AI

Gli agenti AI possono eseguire autonomamente transazioni di pagamento su Solana.
Questi framework consentono di creare agenti che elaborano pagamenti, monitorano
transazioni e automatizzano flussi di lavoro di pagamento:

| Framework        | Descrizione                                                                                  | Link                                                                                      |
| ---------------- | -------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | Strumenti per creare agenti con capacità di pagamento, compatibili con LangChain e Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | Una raccolta di competenze predefinite per la creazione di agenti su Solana                  | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | Framework per agenti multi-piattaforma con plugin Solana per bot X, Telegram, Discord        | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Toolkit cross-chain con azioni di pagamento Solana                                           | [GitHub](https://github.com/goat-sdk/goat)                                                |
