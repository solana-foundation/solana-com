## Riferimento Rapido

| Metodo                                         | Descrizione                                                                                                   |
| ---------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| `LiteSVM::new()`                               | Crea un nuovo ambiente di test base LiteSVM                                                                   |
| `LiteSVM::default()`                           | Restituisce il "valore predefinito" per un tipo                                                               |
| `airdrop(&Address, lamports)`                  | Accredita sull'account i lamport specificati                                                                  |
| `get_account(&Address)`                        | Restituisce tutte le informazioni associate all'account dell'indirizzo fornito                                |
| `set_account(Address, Account)`                | Imposta tutte le informazioni associate all'account dell'indirizzo fornito                                    |
| `get_balance(&Address)`                        | Ottiene il saldo in lamport dell'indirizzo account fornito                                                    |
| `minimum_balance_for_rent_exemption(data_len)` | Restituisce il saldo minimo richiesto per rendere un account con la lunghezza dati specificata esente da rent |
| `add_program(program_id, &bytes)`              | Aggiunge un programma SBF all'ambiente di test                                                                |
| `add_program_from_file(program_id, path)`      | Aggiunge un programma SBF all'ambiente di test dal file specificato                                           |
| `send_transaction(tx)`                         | Invia una transazione firmata                                                                                 |
| `simulate_transaction(tx)`                     | Simula una transazione senza modificare lo stato                                                              |
| `get_transaction(&signature)`                  | Recupera una transazione dalla cronologia                                                                     |
| `latest_blockhash()`                           | Ottiene il blockhash corrente                                                                                 |
| `expire_blockhash()`                           | Forza la scadenza del blockhash corrente                                                                      |
| `warp_to_slot(slot)`                           | Sposta il clock al slot specificato                                                                           |
| `get_sysvar<T>()`                              | Ottiene una sysvar dall'ambiente di test                                                                      |
| `set_sysvar<T>(&value)`                        | Imposta la sysvar nell'ambiente di test                                                                       |
| `with_feature_accounts()`                      | Include account con feature gate nell'ambiente di test                                                        |
| `set_invocation_inspect_callback(callback)`    | Registra un callback pre/post invocazione (richiede la feature `invocation-inspect-callback`)                 |
