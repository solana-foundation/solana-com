## Riferimento Rapido

### Configurazione del Plugin

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### Gestione degli Account (client.svm)

| Metodo                                 | Descrizione                      |
| -------------------------------------- | -------------------------------- |
| `setAccount(account)`                  | Imposta lo stato dell'account    |
| `getAccount(address)`                  | Recupera i dati dell'account     |
| `getBalance(address)`                  | Ottieni il saldo in SOL          |
| `airdrop(address, lamports)`           | Finanzia un account              |
| `minimumBalanceForRentExemption(size)` | Calcola il minimo esente da rent |

### Gestione dei Programmi (client.svm)

| Metodo                                | Descrizione                     |
| ------------------------------------- | ------------------------------- |
| `addProgram(programId, bytes)`        | Carica il programma dai byte    |
| `addProgramFromFile(programId, path)` | Carica il programma da file .so |

### Transazioni (client.svm)

| Metodo                    | Descrizione                       |
| ------------------------- | --------------------------------- |
| `sendTransaction(tx)`     | Esegui una transazione            |
| `simulateTransaction(tx)` | Simula senza modifiche allo stato |
| `latestBlockhash()`       | Ottieni il blockhash corrente     |
| `expireBlockhash()`       | Avanza a un nuovo blockhash       |

### Configurazione (client.svm)

| Metodo                      | Descrizione                                     |
| --------------------------- | ----------------------------------------------- |
| `withSigverify(bool)`       | Abilita/disabilita la verifica delle firme      |
| `withBlockhashCheck(bool)`  | Abilita/disabilita la validazione del blockhash |
| `withSysvars()`             | Abilita gli account sysvar                      |
| `withBuiltins()`            | Abilita i programmi integrati                   |
| `withPrecompiles()`         | Abilita i programmi precompilati                |
| `withTransactionHistory(n)` | Memorizza le ultime n transazioni               |
| `withComputeBudget(budget)` | Imposta i limiti di calcolo                     |
| `withLamports(amount)`      | Imposta i lamport predefiniti                   |
| `withLogBytesLimit(n)`      | Imposta il limite di output dei log             |

### Tempo & Sysvars (client.svm)

| Metodo                     | Descrizione                      |
| -------------------------- | -------------------------------- |
| `warpToSlot(slot)`         | Salta a uno slot specifico       |
| `getClock()`               | Ottieni il sysvar clock          |
| `getRent()`                | Ottieni il sysvar rent           |
| `getEpochSchedule()`       | Ottieni l'epoch schedule         |
| `getLastRestartSlot()`     | Ottieni l'ultimo slot di riavvio |
| `setLastRestartSlot(slot)` | Imposta l'ultimo slot di riavvio |

### RPC (client.rpc)

| Metodo                           | Descrizione                               |
| -------------------------------- | ----------------------------------------- |
| `getAccountInfo(address)`        | Recupera un singolo account (solo base64) |
| `getMultipleAccounts(addresses)` | Recupera più account                      |
| `getLatestBlockhash()`           | Ottieni il blockhash corrente             |
