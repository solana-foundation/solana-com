# Configurazione

LiteSVM utilizza un pattern builder per la configurazione. Tutti i metodi
restituiscono `LiteSVM` per il concatenamento.

## Configurazione tipica dei test

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());

client.svm
  .withSigverify(false) // Skip signature verification (faster)
  .withBlockhashCheck(false) // Skip blockhash validation (simpler)
  .withSysvars() // Enable Clock, Rent, etc.
  .withBuiltins() // Enable ed25519, secp256k1
  .withTransactionHistory(100n); // Store last 100 transactions
```

## Firma e Blockhash

### withSigverify

```typescript
withSigverify(sigverify: boolean): LiteSVM
```

Abilita o disabilita la verifica della firma.

```typescript
// Disable for faster tests (default is enabled)
client.svm.withSigverify(false);

// Enable for production-like testing
client.svm.withSigverify(true);
```

<Callout type="info">
  Disabilitare la verifica della firma rende i test più veloci ma meno
  realistici.
</Callout>

### withBlockhashCheck

```typescript
withBlockhashCheck(check: boolean): LiteSVM
```

Abilita o disabilita il controllo della scadenza del blockhash.

```typescript
// Disable for simpler tests
client.svm.withBlockhashCheck(false);
```

## Programmi e Sysvar

### withSysvars

```typescript
withSysvars(): LiteSVM
```

Abilita gli account sysvar (Clock, rent, EpochSchedule, ecc.).

```typescript
client.svm.withSysvars();

// Now you can access sysvars
const clock = client.svm.getClock();
const rent = client.svm.getRent();
```

### withBuiltins

```typescript
withBuiltins(): LiteSVM
```

Abilita i programmi integrati (verifica della firma ed25519, secp256k1).

```typescript
client.svm.withBuiltins();
```

### withPrecompiles

```typescript
withPrecompiles(): LiteSVM
```

Abilita i programmi precompilati.

```typescript
client.svm.withPrecompiles();
```

### withDefaultPrograms

```typescript
withDefaultPrograms(): LiteSVM
```

Aggiunge i programmi predefiniti (System Program, BPF Loader, ecc.).

```typescript
client.svm.withDefaultPrograms();
```

## Limiti delle risorse

### withLamports

```typescript
withLamports(lamports: bigint): LiteSVM
```

Imposta i lamport predefiniti per gli account.

```typescript
client.svm.withLamports(1_000_000_000n); // 1 SOL
```

### withLogBytesLimit

```typescript
withLogBytesLimit(limit?: bigint): LiteSVM
```

Imposta il limite di byte per l'output del log.

```typescript
client.svm.withLogBytesLimit(10000n);
```

### withTransactionHistory

```typescript
withTransactionHistory(capacity: bigint): LiteSVM
```

Abilita l'archiviazione della cronologia delle transazioni.

```typescript
// Store last 100 transactions
client.svm.withTransactionHistory(100n);
```

## Set di funzionalità

### withFeatureSet

```typescript
withFeatureSet(featureSet: FeatureSet): LiteSVM
```

Configura il set di funzionalità per testare versioni specifiche di Solana.
