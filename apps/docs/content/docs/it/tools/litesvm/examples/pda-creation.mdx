## Test Completo

```rust title="tests/pda_test.rs"
use litesvm::LiteSVM;
use solana_account::Account;
use solana_sdk::pubkey::Pubkey;

#[test]
fn test_pda_creation() {
    let mut svm = LiteSVM::new();

    // Program that owns the PDA
    let program_id = Pubkey::new_unique();

    // Derive PDA address
    let seed = b"my_pda";
    let (pda, bump) = Pubkey::find_program_address(
        &[seed],
        &program_id
    );

    println!("PDA: {}", pda);
    println!("Bump: {}", bump);

    // Create PDA account manually (for testing)
    svm.set_account(pda, Account {
        lamports: 1_000_000,
        data: vec![bump], // Store bump seed in data
        owner: program_id,
        executable: false,
        rent_epoch: 0,
    }).unwrap();

    // Verify PDA was created
    let account = svm.get_account(&pda).unwrap();
    assert_eq!(account.owner, program_id);
    assert_eq!(account.data[0], bump);

    println!("PDA created successfully!");
}
```

## Punti Chiave

1. **Derivazione PDA**: Usa `Pubkey::find_program_address()` per derivare gli
   indirizzi PDA
2. **Seeds**: I PDA vengono derivati dai seed e dall'ID del programma
3. **Bump Seed**: Il bump seed garantisce che l'indirizzo sia fuori dalla curva
   ed25519
4. **Creazione Account**: Usa `set_account()` per creare manualmente account per
   i test
5. **Proprietà**: I PDA devono essere di proprietà del programma che li ha
   derivati
