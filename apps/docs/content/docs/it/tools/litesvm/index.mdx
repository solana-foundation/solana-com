---
title: LiteSVM
description:
  Test ultrarapidi per programmi Solana con una VM in-process. Testa i tuoi
  programmi più velocemente senza overhead di rete o processi validator.
---

LiteSVM è una libreria leggera per il testing di programmi Solana che funziona
creando una VM Solana in-process ottimizzata per gli sviluppatori di programmi.
A differenza di altri approcci di testing che avviano processi validator
separati, LiteSVM incorpora la VM direttamente nei tuoi test, rendendoli molto
più veloci da eseguire. In ulteriore rottura con la tradizione, offre un'API
ergonomica con impostazioni predefinite sensate e un'ampia configurabilità per
chi ne ha bisogno.
