**Commerce Kit è una raccolta di strumenti per la creazione di esperienze
commerciali su Solana.** Dai pulsanti di pagamento pronti all'uso ai primitive
headless, Commerce Kit fornisce tutto il necessario per accettare pagamenti in
criptovaluta.

## Perché Commerce Kit?

- **Componenti Pronti all'Uso**: PaymentButton pre-costruito con connessione
  wallet, selezione token e gestione delle transazioni
- **Indipendente dal Framework**: Utilizza primitive headless in qualsiasi
  framework JavaScript
- **Solana Pay**: Supporto completo per Solana Pay con generazione di codici QR
- **Type-Safe**: Supporto completo per TypeScript in tutti i pacchetti
- **Wallet Standard**: Compatibile con tutti i wallet Wallet Standard (Phantom,
  Solflare, Backpack, ecc.)

## Guida Rapida

Il modo più veloce per iniziare è con il componente `PaymentButton`:

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## Installazione

Scegli i pacchetti Commerce Kit di cui hai bisogno:

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
