**Solana Pay è un protocollo standard e un insieme di implementazioni di
riferimento che consentono agli sviluppatori di integrare pagamenti
decentralizzati nelle loro app e servizi.** La blockchain di Solana conferma le
transazioni in meno di un secondo e costa in media $0,0005, offrendo agli utenti
un'esperienza fluida senza intermediiari.

## Perché Solana Pay?

- **Protocollo Standard**: Approccio unificato ai pagamenti su tutti i wallet e
  le app di Solana
- **Ultra Veloce**: Le transazioni vengono confermate in meno di un secondo
- **Ultra Economico**: Costo medio di transazione di $0,0005
- **Nessun Intermediario**: Pagamenti peer-to-peer diretti senza terze parti
- **Supporto Universale dei Wallet**: Compatibile con Phantom, Solflare,
  Backpack e altri

## Guida Rapida

Il modo più semplice per iniziare è con un URL di richiesta di trasferimento:

```
solana:recipient?amount=0.01&label=Coffee%20Shop&message=Grande%20Latte
```

Questo URL può essere:

- Codificato come codice QR per la scansione con wallet mobile
- Utilizzato come link di pagamento nelle applicazioni web
- Incorporato nelle app mobile come deep link

**[→ Guida Rapida alle Richieste di Trasferimento](/docs/tools/solana-pay/quickstart/transfer-requests)** -

## Installazione

Installa l'SDK JavaScript di Solana Pay:

```bash
# Using pnpm (recommended)
pnpm add @solana/pay@beta @solana/kit

# Using npm
npm install @solana/pay@beta @solana/kit

# Using yarn
yarn add @solana/pay@beta @solana/kit
```

## Casi d'Uso

Solana Pay abilita un'ampia gamma di scenari di pagamento:

- **Pagamenti al Dettaglio**: Pagamenti in presenza tramite codici QR
- **E-commerce**: Flussi di checkout web con integrazione wallet
- **App Mobile**: Esperienze di pagamento native con deep linking
- **Conio di NFT**: Creazione dinamica di transazioni per interazioni complesse
- **Abbonamenti**: Pagamenti ricorrenti e programmi fedeltà
- **Integrazione DeFi**: Transazioni componibili con swap e prestiti

## Wallet Supportati

Solana Pay funziona con tutti i principali wallet Solana:

- **[Phantom](https://phantom.app/)** - Il wallet Solana più popolare
- **[Solflare](https://solflare.com/)** - Wallet Solana multipiattaforma
- **[Backpack](https://backpack.app/)** - Wallet Web3 e app store
- **[Glow](https://glow.app/)** - Wallet Solana mobile-first
- **[Ultimate](https://ultimate.app/)** - Wallet focalizzato su DeFi
- **[Decaf Wallet](https://decaf.so/)** - Wallet per pagamenti globali
- **[Espresso Cash](https://www.espressocash.com/)** - Wallet per pagamenti
  globali
- **[Tiplink](https://tiplink.io/)** - Il wallet più semplice al mondo
- **[Bitget Wallet](https://bitget.com/)** - Wallet multi-chain
