## Kora v2.2.0-beta.6

_23-03-2026_ |
[Rilascio su GitHub](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.6)
| `cargo install kora-cli@2.2.0-beta.6`

Pre-rilascio con supporto per bundle Jito, nuovi metodi RPC e funzionalitĂ  di
sicurezza migliorate.

- Aggiunto supporto per bundle Jito con metodi `signBundle`, `signAndSendBundle`
  e `estimateBundleFee`
- Aggiunto metodo RPC `getVersion`
- Aggiunta protezione del fee payer Lighthouse
- Aggiunta protezione bot reCAPTCHA v3 per gli endpoint di firma
- Aggiunti limiti di utilizzo per wallet con supporto Redis
- Aggiunte immagini Docker ufficiali su GHCR
- Aggiunte regole granulari per i limiti di utilizzo
- SDK: `KoraClient` ora accetta un oggetto opzioni; nuovo `koraPlugin` per
  `@solana/kit`

Consulta la [Documentazione Beta](/docs/tools/kora/beta) per i dettagli.

## Kora TypeScript SDK v0.2.0

_09-03-2026_ |
[Rilascio su GitHub](https://github.com/solana-foundation/kora/releases/tag/ts-sdk-v0.2.0)
| `pnpm add @solana/kora@0.2.0`

Architettura plugin del client Kit per transazioni gasless componibili.

- Aggiunto `createKitKoraClient()` con composizione plugin Kit (plugin planner,
  executor, payer)
- Abilita la componibilitĂ  con plugin di programmi Kit come `tokenProgram()`
- Aggiunto supporto Token-2022 tramite opzione di configurazione
  `tokenProgramId`
- Aggiunto supporto per istruzioni compute budget con stima CU basata su
  simulazione
- Aggiunto campo mancante `signature` alla risposta `signAndSendTransaction`
- Restituisce `NoopSigner` da `getPaymentInstruction` per la deduplicazione del
  signer di Kit v6
- Aggiornata la dipendenza peer `@solana/kit` a `^6.1.0`

Consulta la [Guida al client Kit](/docs/tools/kora/guides/kit-client).

## Kora v2.0.0

_19/11/2025_ |
[Release GitHub](https://github.com/solana-foundation/kora/releases/tag/v2.0.0)
| `cargo install kora-cli@2.0.0`

Release principale con nuova architettura.

- Introdotta una nuova architettura per il nodo Kora
- Pubblicato TypeScript SDK `@solana/kora` v0.1.0
- Migrazione a Solana SDK v3.0.x e aggiornamento di tutte le dipendenze
- Sostituito il Kora Signer personalizzato con il crate solana-signers
- Rimossi i metodi `sign_transaction_if_paid` e `get_all_signers`
- Modificati i tipi di risposta: rimosse le firme da `signAndSendTransaction` e
  `signTransaction`
- Migliorata l'esperienza sviluppatore, la sicurezza e le prestazioni
