Non vuoi gestire il tuo nodo Kora? I seguenti provider offrono infrastruttura
Kora ospitata, così puoi aggiungere l'astrazione delle commissioni alla tua app
senza dover gestire il tuo deployment.

## Provider

### Openfort

[Openfort](https://www.openfort.io/) offre un'infrastruttura paymaster Solana
alimentata da Kora, abilitando transazioni gasless attraverso la loro
piattaforma di wallet integrati. La loro integrazione utilizza un pattern di
co-firma in cui Kora agisce come pagatore delle commissioni mentre gli utenti
firmano tramite i wallet integrati di Openfort.

- **Sito web**: [openfort.io](https://www.openfort.io/)
- **Documentazione**:
  [Transazioni Gasless su Solana](https://www.openfort.io/blog/solana-gasless-transactions)
- **Funzionalità**: Wallet integrati, transazioni gasless, sponsorizzazione
  delle commissioni

## Diventa un Operatore di Nodi Kora

Kora è open source e chiunque può eseguire un nodo. Che tu stia costruendo
un'app che necessita di astrazione delle commissioni o voglia offrire servizi
paymaster ad altri, consulta la
[Guida per gli Operatori](/docs/tools/kora/operators) per iniziare.
