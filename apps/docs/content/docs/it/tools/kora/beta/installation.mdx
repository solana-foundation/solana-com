---

## Cargo

Installa la versione beta direttamente da crates.io:

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

Le immagini Docker ufficiali multi-piattaforma (amd64/arm64) sono pubblicate su
GitHub Container Registry:

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

Tag disponibili:

- `latest` — Ultima versione stabile
- `beta` — Ultima versione pre-release
- `v<version>` — Versione specifica (es. `v2.2.0-beta.7`)

### Docker Compose

Per build personalizzate o per includere Redis per la cache e il supporto ai
limiti di utilizzo:

- [Dockerfile di esempio](https://github.com/solana-foundation/kora/blob/main/examples/deploy/Dockerfile)
- [Docker Compose con Redis](https://github.com/solana-foundation/kora/blob/main/docker-compose.yml)

### Esecuzione con Docker

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## SDK TypeScript

L'SDK beta è disponibile su npm:

```bash
pnpm add @solana/kora@beta
```

Consulta [Modifiche all'SDK](/docs/tools/kora/beta/sdk-changes) per le nuove
API.

## Verifica dell'installazione

```bash
kora --version
```
