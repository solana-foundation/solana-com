Surfnet è il simulatore di rete Solana locale di Surfpool, progettato per
fornire agli sviluppatori un potente ambiente di test che replica il
comportamento della blockchain Solana senza il sovraccarico di un nodo validator
completo.

## Per Iniziare

Avvia un'istanza Surfnet utilizzando la CLI di Surfpool:

```bash
surfpool start
```

Questo avvierà un'istanza Surfnet locale su `http://localhost:8899` con tutti
gli endpoint RPC disponibili.

## Funzionalità Principali

- **Compatibilità RPC Completa**: Supporta l'intera API RPC di Solana
- **Iterazione Rapida**: Nessuna necessità di attendere le conferme dei blocchi
- **Manipolazione dello Stato**: Cheatcode per modificare direttamente lo stato
  della blockchain
- **Fork dalla Mainnet**: Esegui il fork dalla mainnet per testare con account e
  programmi reali
- **Supporto Plugin Geyser**: Collega i tuoi plugin Geyser per lo streaming di
  dati in tempo reale
