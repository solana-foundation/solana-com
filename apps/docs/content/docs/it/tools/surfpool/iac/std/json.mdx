## jq

La funzione `jq` consente di suddividere, filtrare e mappare dati JSON. Consulta
la [documentazione di jq](https://jqlang.github.io/jq/manual/) per ulteriori
dettagli.

### Input

| Nome           | Tipo             | Descrizione                                                                             |
| -------------- | ---------------- | --------------------------------------------------------------------------------------- |
| `decoded_json` | string \| object | Un oggetto JSON                                                                         |
| `query`        | string           | Una query JSON. Consulta la [documentazione di jq](https://jqlang.github.io/jq/manual/) |

### Output

| Nome    | Tipo          | Descrizione                   |
| ------- | ------------- | ----------------------------- |
| `value` | array[string] | Il risultato della query `jq` |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
