## index

`index` recupera l'elemento da una lista all'indice specificato.

### Input

| Nome    | Tipo                                             | Descrizione                            |
| ------- | ------------------------------------------------ | -------------------------------------- |
| `list`  | array[string] \| array[integer] \| array[buffer] | La lista da cui recuperare un elemento |
| `index` | integer                                          | L'indice dell'elemento da recuperare   |

### Output

| Nome    | Tipo   | Descrizione                                   |
| ------- | ------ | --------------------------------------------- |
| `value` | string | L'elemento della lista all'indice specificato |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
