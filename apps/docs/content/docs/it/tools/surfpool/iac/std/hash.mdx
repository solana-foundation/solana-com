## ripemd160

`ripemd160` calcola l'hash Ripemd160 di un valore.

### Input

| Nome    | Tipo                    | Descrizione                                                |
| ------- | ----------------------- | ---------------------------------------------------------- |
| `value` | buffer \| array[buffer] | Il valore codificato in esadecimale da sottoporre all'hash |

### Output

| Nome    | Tipo   | Descrizione            |
| ------- | ------ | ---------------------- |
| `value` | string | Il risultato dell'hash |

```hcl
output "hashed_data" {
    value = ripemd160(encode_hex("hello, world"))
}
// > hashed_data: 0XA3201F82FCA034E46D10CD7B27E174976E241DA2
```

## sha256

### Input

| Nome    | Tipo                    | Descrizione                                                |
| ------- | ----------------------- | ---------------------------------------------------------- |
| `value` | buffer \| array[buffer] | Il valore codificato in esadecimale da sottoporre all'hash |

### Output

| Nome    | Tipo   | Descrizione            |
| ------- | ------ | ---------------------- |
| `value` | string | Il risultato dell'hash |

```hcl
output "hashed_data" {
    value = sha256(encode_hex("hello, world"))
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```

### Input

| Nome    | Tipo   | Descrizione                              |
| ------- | ------ | ---------------------------------------- |
| `value` | string | Il valore stringa da sottoporre all'hash |

### Output

| Nome    | Tipo   | Descrizione            |
| ------- | ------ | ---------------------- |
| `value` | string | Il risultato dell'hash |

```hcl
output "hashed_data" {
    value = keccak256("hello, world")
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```
