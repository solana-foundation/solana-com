## assert_eq

`assert_eq` verifica che due valori siano uguali.

### Input

| Nome    | Tipo | Descrizione                   |
| ------- | ---- | ----------------------------- |
| `left`  | any  | Un valore da confrontare      |
| `right` | any  | Il valore con cui confrontare |

### Output

| Nome    | Tipo                  | Descrizione                  |
| ------- | --------------------- | ---------------------------- |
| `value` | addon(std::assertion) | Il risultato dell'asserzione |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```

| `right` | integer \| float | Un intero o float con cui confrontare |

### Output

| `value` | addon(std::assertion) | Il risultato dell'asserzione |
