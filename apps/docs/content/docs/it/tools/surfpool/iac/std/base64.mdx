  Funzioni della libreria standard per la codifica e decodifica Base64

## encode_base64

`encode_base64` codifica un buffer o una stringa come stringa base64.

### Input

| Nome    | Tipo                  | Descrizione                                                                                                                                                       |
| ------- | --------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | Il buffer o la stringa da codificare. Le stringhe che iniziano con '0x' vengono decodificate come esadecimali; altrimenti vengono utilizzati i byte UTF-8 grezzi. |

### Output

| Nome    | Tipo   | Descrizione                             |
| ------- | ------ | --------------------------------------- |
| `value` | string | L'input, codificato come stringa base64 |

```hcl
output "encoded" {
    value = encode_base64("0x48656c6c6f20776f726c6421")
}
// > encoded: SGVsbG8gd29ybGQh
```

## decode_base64

`decode_base64` decodifica una stringa codificata in base64 e restituisce il
risultato come buffer.

### Input

| Nome            | Tipo   | Descrizione                       |
| --------------- | ------ | --------------------------------- |
| `base64_string` | string | La stringa base64 da decodificare |

### Output

| Nome    | Tipo   | Descrizione                                 |
| ------- | ------ | ------------------------------------------- |
| `value` | buffer | La stringa base64 decodificata, come buffer |

```hcl
output "decoded" {
    value = decode_base64("SGVsbG8gd29ybGQh")
}
// > decoded: 0x48656c6c6f20776f726c6421
```
