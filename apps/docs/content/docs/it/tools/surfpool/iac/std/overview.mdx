La libreria standard fornisce funzioni e azioni principali disponibili per tutti
i Runbook, indipendentemente dagli addon blockchain abilitati.

## Funzioni

La libreria standard include funzioni di utilità per:

- **Operatori**: Operazioni matematiche e confronti
- **Hash**: Funzioni di hashing (SHA256, Keccak256, ecc.)
- **Hex**: Codifica e decodifica esadecimale
- **Crypto**: Operazioni crittografiche
- **JSON**: Codifica e decodifica JSON
- **List**: Funzioni di manipolazione delle liste
- **Assertions**: Asserzioni di uguaglianza e condizioni
- **Base64**: Codifica e decodifica Base64

[Visualizza tutte le funzioni](/docs/tools/surfpool/iac/std/functions)

## Azioni

La libreria standard include azioni per:

- **HTTP**: Esecuzione di richieste HTTP
- **Operazioni sui file**: Lettura e scrittura di file

[Visualizza tutte le azioni](/docs/tools/surfpool/iac/std/actions)
