## and_bool

`and_bool` restituisce l'AND binario degli argomenti a sinistra e a destra.

### Input

| Nome  | Tipo | Descrizione        |
| ----- | ---- | ------------------ |
| `lhs` | bool | Un valore booleano |
| `rhs` | bool | Un valore booleano |

### Output

| Nome    | Tipo | Descrizione                                        |
| ------- | ---- | -------------------------------------------------- |
| `value` | bool | Il risultato di un AND binario tra i due argomenti |

```hcl
output "my_bool" {
  value = false && true
}
// > my_bool: false
```

## div

`div` restituisce la divisione intera dell'argomento a sinistra per l'argomento
a destra, arrotondando qualsiasi resto verso il basso all'intero più vicino.

### Input

| Nome  | Tipo    | Descrizione  |
| ----- | ------- | ------------ |
| `lhs` | integer | Il dividendo |
| `rhs` | integer | Il divisore  |

### Output

| Nome    | Tipo    | Descrizione                                                |
| ------- | ------- | ---------------------------------------------------------- |
| `value` | integer | Il risultato della divisione del dividendo per il divisore |

```hcl
output "my_int" {
  value = 11 / -3
}
// > my_int: -3
```

## eq

`eq` restituisce `true` se gli argomenti a sinistra e a destra sono uguali e
`false` se non lo sono.

### Input

| Nome  | Tipo | Descrizione      |
| ----- | ---- | ---------------- |
| `lhs` | any  | Qualsiasi valore |
| `rhs` | any  | Qualsiasi valore |

### Output

| Nome    | Tipo | Descrizione                                                 |
| ------- | ---- | ----------------------------------------------------------- |
| `value` | bool | Il risultato di un controllo di uguaglianza tra i due input |

```hcl
output "is_eq" {
  value = "arg" == "arg"
}
// > is_eq: true
```
