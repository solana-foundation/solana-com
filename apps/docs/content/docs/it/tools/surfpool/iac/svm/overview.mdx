---
title: Panoramica del Plugin SVM
description: Add-on per Solana e Blockchain Compatibili con SVM (beta)
---

Il plugin SVM `txtx` consente di creare Runbook che interagiscono con Solana e
le blockchain compatibili con SVM. Il plugin fornisce funzioni di utilità che
permettono di distribuire programmi Anchor e codificare chiamate alle istruzioni
in base agli IDL dei programmi. Le azioni possono essere utilizzate per creare
transazioni valide di trasferimento, chiamata a programmi e distribuzione di
programmi, che possono essere firmate tramite una frase mnemonica, una chiave
segreta o tramite il firmatario del browser.

## Avvio Rapido

Per utilizzare il plugin SVM, dichiaralo nel tuo Runbook:

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## Funzionalità

- **Distribuzione di Programmi**: Distribuisci programmi Anchor e programmi
  Solana nativi
- **Costruzione delle Istruzioni**: Codifica le chiamate alle istruzioni
  utilizzando gli IDL dei programmi
- **Firma delle Transazioni**: Supporto per mnemonici, chiavi segrete e wallet
  web
- **Gestione degli Account**: Interroga e gestisci gli account Solana
