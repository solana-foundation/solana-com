Il crate `surfpool-sdk` espone una superficie ridotta incentrata sulla struct
`Surfnet`. Questa pagina elenca ogni elemento pubblico; per una guida narrativa
all'utilizzo, consulta le guide a partire da
[Panoramica](/docs/tools/surfpool/sdk/overview).

## Esportazioni della radice del crate

| Elemento              | Tipo          | Note                                                               |
| --------------------- | ------------- | ------------------------------------------------------------------ |
| `Surfnet`             | struct        | Istanza Surfnet in esecuzione — endpoint RPC/WS, payer, cheatcode  |
| `SurfnetBuilder`      | struct        | Builder fluente utilizzato da `Surfnet::builder()`                 |
| `Cheatcodes<'_>`      | struct        | Helper cheatcode in prestito restituito da `Surfnet::cheatcodes()` |
| `SurfnetError`        | enum          | Tutte le varianti di errore restituite dall'SDK                    |
| `SurfnetResult<T>`    | alias di tipo | `Result<T, SurfnetError>`                                          |

### Ciclo di vita

```rust
impl Surfnet {
    pub fn stop(&mut self) -> SurfnetResult<()>;
}

impl Drop for Surfnet { /* calls stop() if not already stopped */ }
```

`stop` attende fino a cinque secondi affinché il runtime rilasci le proprie
porte. Un errore durante `Drop` è silenzioso; un errore durante una chiamata
esplicita a `stop` restituisce un `SurfnetError::Runtime`.

## SurfnetError

```rust
#[derive(Debug)]
pub enum SurfnetError {
    PortAllocation(String),
    Startup(String),
    Runtime(String),
    Cheatcode(String),
    Aborted(String),
}
```

| Variante         | Causa                                                                    |
| ---------------- | ------------------------------------------------------------------------ |
| `PortAllocation` | Impossibile eseguire il binding a una porta libera per RPC o WebSocket.  |
| `Startup`        | Il runtime non è riuscito a inizializzarsi.                              |
| `Runtime`        | Il thread del runtime è andato in panico o è terminato inaspettatamente. |
| `Cheatcode`      | Una chiamata JSON-RPC di un cheatcode ha restituito un errore.           |
| `Aborted`        | Il runtime è stato arrestato o annullato durante l'avvio.                |

Il crate implementa `std::error::Error` e `Display` su `SurfnetError`, quindi
funziona con `?` e `anyhow` senza configurazioni aggiuntive.
