La CLI di surfpool può essere installata tramite lo script di installazione
oppure compilata dal codice sorgente.

## Script di Installazione

Per installare surfpool, apri un terminale ed esegui:

```bash
curl -sL https://run.surfpool.run/ | bash
```

## Installazione dal Codice Sorgente

Per compilare il binario di Surfpool dal codice sorgente, clona il repository di
surfpool.

```bash
git clone https://github.com/solana-foundation/surfpool.git
```

Naviga nella nuova directory.

```bash
cd surfpool
```

Quindi, compila il binario. Questo comando compilerà il binario e lo salverà in
`~/.cargo/bin/surfpool`.

```bash
cargo surfpool-install
```

Infine, assicurati che il binario `surfpool` sia disponibile nel tuo PATH.
Questo processo varierà a seconda del sistema operativo in uso.
