Inizia a utilizzare Solana Attestations installando i pacchetti necessari e
configurando il tuo ambiente di sviluppo.

## TypeScript/JavaScript

Installa la libreria Solana Attestations utilizzando il gestore di pacchetti che
preferisci:

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
