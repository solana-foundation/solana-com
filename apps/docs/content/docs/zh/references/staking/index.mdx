通过质押您的 SOL 代币，您可以帮助保障网络安全，并在此过程中[获得奖励](https://docs.anza.xyz/implemented-proposals/staking-rewards)。

您可以通过将代币委托给处理交易并运行网络的 validator 来进行质押。

委托质押是一种风险共担、收益共享的金融模式，长期委托代币的持有者有可能获得回报。这是通过将代币持有者（委托人）与其所委托的 validator 的经济激励保持一致来实现的。

委托给某个 validator 的质押越多，该 validator 被选中将新交易写入账本的频率就越高。validator 写入的交易越多，validator 及其委托人获得的奖励也就越多。那些将系统配置为能够处理更多交易的 validator 会按比例获得更多奖励，同时也能让网络保持高速和稳定运行。

validator 需要承担运行和维护系统的成本，这部分成本会以奖励的一定比例作为费用转嫁给委托人，这种费用被称为
_佣金_。由于委托给 validator 的质押越多，其获得的奖励也越多，因此 validator 之间可能会相互竞争，争取为其服务提供更低的佣金。

虽然目前 Solana 协议尚未实现此功能，但未来委托人可能会因质押而面临代币损失的风险，这一过程被称为
_惩罚_（slashing）。惩罚是指当 validator 存在恶意行为（如制造无效交易、审查特定类型的交易或网络参与者）时，系统会销毁并移除其部分 SOL。

目前协议中尚未实现惩罚机制。如需了解更多有关惩罚的信息，请参阅[惩罚机制路线图](https://docs.anza.xyz/proposals/optimistic-confirmation-and-slashing#slashing-roadmap)。

## 如何质押我的 SOL 代币？

您可以将 SOL 转入支持质押的钱包进行质押。钱包会提供创建质押账户和进行委托的操作步骤。

### 支持的钱包

许多网页和移动端钱包都支持 Solana 质押操作。请向您常用钱包的维护者咨询相关支持情况。

### Solana 命令行工具

- Solana 命令行工具可配合 CLI 生成的 keypair 文件钱包、纸钱包或已连接的 Ledger
  Nano，完成所有质押操作。
  [使用 Solana 命令行工具进行质押操作](https://docs.anza.xyz/cli/examples/delegate-stake)。

### 创建质押账户

请按照钱包的指引创建质押账户。该账户类型与普通收发代币的账户不同。

### 选择 validator

请根据钱包的指引选择 validator。您可以通过下方链接获取潜在高性能 validator 的信息。Solana 基金会不推荐任何特定的 validator。

网站 solanabeach.io 由我们的 validator 之一 Staking
Facilities 构建和维护。该网站提供了网络整体的高层次图形信息，以及每个 validator 的列表和部分近期性能统计数据。

- https://solanabeach.io

如需查看区块生产统计信息，请使用 Solana 命令行工具：

- `solana validators`
- `solana block-production`

Solana 团队不会对如何解读这些信息提供建议。请自行进行尽职调查。

### 委托你的质押

请按照钱包的指引，将你的质押委托给你选择的 validator 。
