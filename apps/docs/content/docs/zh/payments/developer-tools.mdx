## AI 开发工具

请参考我们的指南，了解[如何在 Solana 上入门 AI 工具](/developers/guides/getstarted/intro-to-ai)。

### AI 代理框架

AI 代理可以在 Solana 上自主执行支付交易。这些框架使构建能够处理支付、监控交易和自动化支付工作流的代理成为可能：

| 框架             | 描述                                                                 | 链接                                                                                      |
| ---------------- | -------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | 用于构建具备支付能力的代理的工具，兼容 LangChain 和 Vercel AI        | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [文档](https://kit.sendai.fun/) |
| Send AI Skills   | 用于在 Solana 上构建代理的预构建技能集合                             | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | 多平台代理框架，带有适用于 X、Telegram、Discord 机器人的 Solana 插件 | [GitHub](https://github.com/elizaOS/eliza) • [文档](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | 具有 Solana 支付操作的跨链工具包                                     | [GitHub](https://github.com/goat-sdk/goat)                                                |

### AI 程序开发

[Codigo](https://codigo.ai/) 是一个端到端的 Solana AI 开发平台。
