## AI 开发工具

请参考我们的指南，了解[如何在 Solana 上入门 AI 工具](/developers/guides/getstarted/intro-to-ai)。

### Solana MCP

[Solana MCP](https://mcp.solana.com/)（机器对话协议）将 Solana 知识直接集成到 Claude、Cursor、Windsurf 等 AI 编程工具中。它基于 Solana 文档、Anchor 框架、程序示例和 Stack
Exchange 问答训练而成，有助于更快集成支付功能。

### AI Agent 框架

AI
agent 可以在 Solana 上自主执行支付交易。这些框架支持构建能够处理支付、监控交易和自动化支付流程的 agent：

| 框架             | 描述                                                        | 链接                                                                                      |
| ---------------- | ----------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | 构建具备支付能力的 agent 工具，兼容 LangChain 和 Vercel AI  | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | 构建 Solana agent 的预置技能集合                            | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | 多平台 agent 框架，支持 X、Telegram、Discord 的 Solana 插件 | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | 跨链工具包，支持 Solana 支付操作                            | [GitHub](https://github.com/goat-sdk/goat)                                                |

### AI 程序开发

[Codigo](https://codigo.ai/) 是一个端到端的 Solana AI 开发平台。
