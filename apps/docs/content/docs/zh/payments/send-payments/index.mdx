---
title: 发送付款
description: 使用稳定币进行付款，享受秒级结算和低于 0.001 美元的手续费
---

无论你在开发工资发放、供应商付款，还是跨境汇款，选择在 Solana 上发送付款，都能实现亚秒级结算、低于一美分的手续费，并且没有中间商抽成。你可以添加备注以便对账，将数百笔付款批量处理为一笔交易，或为用户代付手续费，让他们无需持有 SOL。

本节介绍稳定币转账的核心模式——从单笔转账到大批量支付处理。
