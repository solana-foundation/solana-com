通过即插即用组件、Solana Pay 协议或直接 RPC 集成，在 Solana 上接受支付。

## 收款地址规范

在 Solana 上，**请分享你的钱包地址，而不是你的 token
account**。通常由发送方根据所发送的 token 推导出相应的
[Associated Token Account (ATA)](/docs/payments/how-payments-work#token-accounts)（详见
[发送支付](/docs/payments/send-payments) 了解更多信息）。

**示例：** 分享 `7EcDhS...`（你的钱包），而不是 `9xKpQ...`（你的 USDC token
account）。

<Callout>
  如果用于收银台或电商场景，请参阅 [Solana
  Pay](/docs/payments/accept-payments/solana-pay) 获取二维码和支付链接，或使用
  [Payment Button](/docs/payments/accept-payments/payment-button) 快速集成 React
  组件。
</Callout>

## 集成选项

| 方式                                                            | 最适合场景            | 复杂度 |
| --------------------------------------------------------------- | --------------------- | ------ |
| [Payment Button](/docs/payments/accept-payments/payment-button) | React 应用、快速集成  | 低     |
| [Solana Pay](/docs/payments/accept-payments/solana-pay)         | 二维码、支付链接、POS | 中     |
| [Direct RPC](/docs/payments/interacting-with-solana)            | 自定义流程、完全控制  | 高     |

## 验证

收到支付后，请使用
[Verification Tools](/docs/payments/accept-payments/verification-tools)
验证款项是否到账。对于高并发应用，请参阅
[Indexing](/docs/payments/accept-payments/indexing) 以实现生产级交易监控。
