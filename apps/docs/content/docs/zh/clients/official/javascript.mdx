## Solana 客户端

`@solana/client`
是无界面的 Solana 运行时，将 RPC、钱包、交易和订阅集成在一个存储中。

| Package        | Description        | GitHub                                                                                 |
| -------------- | ------------------ | -------------------------------------------------------------------------------------- |
| @solana/client | 无界面客户端运行时 | [Source](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [指南：@solana/client 概览](/docs/frontend/client)

## Solana React Hooks

`@solana/react-hooks` 在 `@solana/client` 之上，增加了 React
provider 和 hooks，复用相同的运行时和缓存。

| Package             | Description                       | GitHub                                                                                      |
| ------------------- | --------------------------------- | ------------------------------------------------------------------------------------------- |
| @solana/react-hooks | 用于 UI 的 React provider + hooks | [Source](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [指南：@solana/react-hooks](/docs/frontend/react-hooks)

## Solana Kit

`@solana/kit` 是推荐用于在 Solana 上开发的 TypeScript SDK。

| 包                             | 描述                       | GitHub                                                                          |
| ------------------------------ | -------------------------- | ------------------------------------------------------------------------------- |
| @solana/kit                    | 核心 SDK                   | [源代码](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | 与 System Program 交互     | [源代码](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | 与 Token Program 交互      | [源代码](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | 与 Token-2022 程序交互     | [源代码](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | 与 Memo 程序交互           | [源代码](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | 与 Compute Budget 程序交互 | [源代码](https://github.com/solana-program/compute-budget/tree/main/clients/js) |
| @solana/kit-plugins            | Kit 插件                   | [源代码](https://github.com/anza-xyz/kit-plugins)                               |

- [官方 Kit 文档](https://www.solanakit.com/)
- [教程：Next.js + React Hooks](/docs/frontend/nextjs-solana)

## 签名与密钥管理

适用于多个密钥管理后端的 kit 兼容后端签名：

| 包               | 描述                                                                          | GitHub                                                         |
| ---------------- | ----------------------------------------------------------------------------- | -------------------------------------------------------------- |
| @solana/keychain | 统一签名：Memory、Vault、AWS KMS、Privy、Turnkey、Fireblocks、CDP、Dfns、Para | [源代码](https://github.com/solana-foundation/solana-keychain) |

请参阅[添加签名者指南](https://github.com/solana-foundation/solana-keychain/blob/main/docs/ADDING_SIGNERS.md)以集成其他密钥管理服务。

| @solana/spl-memo  | 与 Memo 程序交互                                              | [源代码](https://github.com/solana-program/memo/tree/main/clients/js-legacy)       |
