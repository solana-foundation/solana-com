## Solana Kit

`@solana/kit` 是在 Solana 上构建应用的推荐 TypeScript
SDK。您可以通过组合式插件来组装客户端，在需要时添加 React 绑定，并引入生成的程序客户端来调用链上程序。

| 包                        | 描述                                      | GitHub                                                                               |
| ------------------------- | ----------------------------------------- | ------------------------------------------------------------------------------------ |
| @solana/kit               | 核心 SDK：RPC、交易、编解码器、签名者     | [源码](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | React 绑定：`ClientProvider` 及相关 Hooks | [源码](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | RPC 连接 + 交易发送                       | [源码](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | 付费账户、身份标识与签名者插件            | [源码](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Wallet Standard 连接 + React Hooks        | [源码](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

### 程序客户端

生成的客户端提供指令构建器和客户端插件（例如 `tokenProgram()` →
`client.token`）。

| 包                             | 描述                       | GitHub                                                                        |
| ------------------------------ | -------------------------- | ----------------------------------------------------------------------------- |
| @solana-program/system         | 与 System Program 交互     | [源码](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | 与 Token Program 交互      | [源码](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | 与 Token-2022 程序交互     | [源码](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | 与 Memo 程序交互           | [源码](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | 与 Compute Budget 程序交互 | [源码](https://github.com/solana-program/compute-budget/tree/main/clients/js) |

## 签名与密钥管理

用于在多种密钥管理后端之间进行兼容 Kit 的后端签名：

| 包               | 描述                                                                                                                   | GitHub                                                       |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------ |
| @solana/keychain | 统一签名方案：Memory、Vault、Privy、Turnkey、AWS KMS、Fireblocks、GCP KMS、CDP、Para、Dfns、Crossmint、Openfort、Utila | [源码](https://github.com/solana-foundation/solana-keychain) |

请参阅 [Keychain 文档](/docs/tools/keychain) 和
[选择后端](/docs/tools/keychain/choosing-a-backend)，获取设置和托管模型的指导。

## Solana Web3.js

`@solana/web3.js` 是 Solana 的旧版 TypeScript SDK。新应用应使用
`@solana/kit`；请参阅 [迁移至 Kit](/docs/frontend/web3-compat)。

| Package           | Description                                            | GitHub                                                                             |
| ----------------- | ------------------------------------------------------ | ---------------------------------------------------------------------------------- |
| @solana/web3.js   | 核心 SDK                                               | [Source](https://github.com/solana-foundation/solana-web3.js)                      |
| @solana/spl-token | 与 Token、Token-2022 及 Associated Token programs 交互 | [Source](https://github.com/solana-program/token-2022/tree/main/clients/js-legacy) |
| @solana/spl-memo  | 与 Memo program 交互                                   | [Source](https://github.com/solana-program/memo/tree/main/clients/js-legacy)       |
