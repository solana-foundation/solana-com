每笔 Solana
[交易](/docs/core/transactions)都需要支付 SOL 作为手续费。手续费包含两个部分：**基础费用**
和可选的
**优先级费用**。基础费用用于补偿 validator 进行签名验证的加密工作。优先级费用则可以提升当前 leader 将你的交易排在其他竞争交易之前的概率。

## 关键信息

- **基础费用**：按签名计费，50% 销毁 / 50% 分配给 validator。
- **优先级费用**： `ceil(compute_unit_price * compute_unit_limit / 1,000,000)`
  lamport。100% 分配给 validator。
