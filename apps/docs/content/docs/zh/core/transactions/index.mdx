一笔交易包含一个或多个
[指令](/docs/core/instructions)、授权更改的账户签名，以及一个最新区块哈希。网络会将交易中的所有指令一起处理。如果任何指令失败，整个交易将失败，所有状态更改都会被回滚。

## 关键要点

- **原子执行**：所有指令全部成功或全部回滚。即使失败也会收取手续费。
- **大小限制**：最大 1,232 字节，来源于 IPv6 最小 MTU（1,280 字节）减去 48 字节网络头部。
- **签名**：每个签名者提供一个 64 字节的 Ed25519 签名。
- **区块哈希过期**：交易的最新区块哈希在 150 个 slot 内有效。

| 区块哈希过期         | 150 个 slot        | [`MAX_PROCESSING_AGE`](https://github.com/anza-xyz/solana-sdk/blob/clock%40v2.2.3/clock/src/lib.rs#L133)                                                                                                                          |
