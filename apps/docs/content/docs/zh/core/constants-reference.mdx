本页面集中列出了核心概念页面中提到的所有运行时常量。每个常量都链接到其在
[agave](https://github.com/anza-xyz/agave/tree/v3.1.8) validator 客户端或
[solana-sdk](https://github.com/anza-xyz/solana-sdk) 中的源定义。
