## 快速参考

### 插件设置

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### 账户管理 (client.svm)

| 方法                                   | 描述               |
| -------------------------------------- | ------------------ |
| `setAccount(account)`                  | 直接设置账户状态   |
| `getAccount(address)`                  | 获取账户数据       |
| `getBalance(address)`                  | 获取 SOL 余额      |
| `airdrop(address, lamports)`           | 为账户充值         |
| `minimumBalanceForRentExemption(size)` | 计算免租金最低金额 |

### 程序管理 (client.svm)

| 方法                                  | 描述                |
| ------------------------------------- | ------------------- |
| `addProgram(programId, bytes)`        | 从字节加载程序      |
| `addProgramFromFile(programId, path)` | 从 .so 文件加载程序 |

### 交易 (client.svm)

| 方法                      | 描述                   |
| ------------------------- | ---------------------- |
| `sendTransaction(tx)`     | 执行交易               |
| `simulateTransaction(tx)` | 模拟执行（不改变状态） |
| `latestBlockhash()`       | 获取当前 blockhash     |
| `expireBlockhash()`       | 推进至新 blockhash     |

### 配置 (client.svm)

| 方法                        | 描述                     |
| --------------------------- | ------------------------ |
| `withSigverify(bool)`       | 启用/禁用签名验证        |
| `withBlockhashCheck(bool)`  | 启用/禁用 blockhash 验证 |
| `withSysvars()`             | 启用 sysvar 账户         |
| `withBuiltins()`            | 启用内置程序             |
| `withPrecompiles()`         | 启用预编译程序           |
| `withTransactionHistory(n)` | 存储最近 n 笔交易        |
| `withComputeBudget(budget)` | 设置计算限制             |
| `withLamports(amount)`      | 设置默认 lamport 数量    |
| `withLogBytesLimit(n)`      | 设置日志输出上限         |

### 时间与系统变量 (client.svm)

| 方法                       | 描述               |
| -------------------------- | ------------------ |
| `warpToSlot(slot)`         | 跳转到指定 slot    |
| `getClock()`               | 获取时钟系统变量   |
| `getRent()`                | 获取 rent 系统变量 |
| `getEpochSchedule()`       | 获取 epoch 调度表  |
| `getLastRestartSlot()`     | 获取最后重启 slot  |
| `setLastRestartSlot(slot)` | 设置最后重启 slot  |

### 远程过程调用 (client.rpc)

| 方法                             | 描述                      |
| -------------------------------- | ------------------------- |
| `getAccountInfo(address)`        | 获取单个账户（仅 base64） |
| `getMultipleAccounts(addresses)` | 获取多个账户              |
| `getLatestBlockhash()`           | 获取当前区块哈希          |
