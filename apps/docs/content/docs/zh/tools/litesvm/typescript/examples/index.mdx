## 运行示例

所有示例均可使用 `tsx` 运行，但部分地址/变量需要将占位符替换为实际值。

```bash
# Install dependencies
pnpm add @solana/kit @solana/kit-plugin-litesvm @solana/kit-plugin-signer @solana-program/system

# Run an example
npx tsx example.ts
```
