---
title: LiteSVM
description:
  使用进程内虚拟机，极速测试 Solana 程序。无需网络开销或 validator
  进程，让您的程序测试更快速。
---

LiteSVM 是一个轻量级的 Solana 程序测试库，通过创建一个专为程序开发者优化的进程内 Solana 虚拟机来运行。与其他需要启动独立 validator 进程的测试方案不同，LiteSVM 将虚拟机直接嵌入到您的测试中，使测试执行速度大幅提升。此外，它还提供了符合人体工学的 API，具有合理的默认配置，并为有需要的用户提供了丰富的自定义选项。
