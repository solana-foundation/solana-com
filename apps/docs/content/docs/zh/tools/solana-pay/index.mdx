**Solana
Pay 是一套标准协议和参考实现，使开发者能够将去中心化支付集成到他们的应用程序和服务中。**
Solana 区块链可在不到一秒的时间内确认交易，平均成本仅为 $0.0005，为用户提供无需中介的无缝体验。

## 为什么选择 Solana Pay？

- **标准协议**：在所有 Solana 钱包和应用程序中统一的支付方式
- **超快速度**：交易在不到一秒内确认
- **超低成本**：平均交易成本 $0.0005
- **无中介**：无需第三方的点对点直接支付
- **通用钱包支持**：兼容 Phantom、Solflare、Backpack 等钱包

## 快速入门

最简单的入门方式是使用转账请求 URL：

```
solana:recipient?amount=0.01&label=Coffee%20Shop&message=Grande%20Latte
```

此 URL 可以：

- 编码为二维码供移动钱包扫描
- 在 Web 应用程序中用作支付链接
- 作为深层链接嵌入到移动应用程序中

**[→ 转账请求快速入门](/docs/tools/solana-pay/quickstart/transfer-requests)** - 创建简单的支付 URL

## 安装

安装 Solana Pay JavaScript SDK：

```bash
# Using pnpm (recommended)
pnpm add @solana/pay@beta @solana/kit

# Using npm
npm install @solana/pay@beta @solana/kit

# Using yarn
yarn add @solana/pay@beta @solana/kit
```

## 应用场景

Solana Pay 支持广泛的支付场景：

- **零售支付**：使用二维码进行现场支付
- **电子商务**：集成钱包的 Web 结账流程
- **移动应用**：通过深层链接实现原生支付体验
- **NFT 铸造**：为复杂交互动态创建交易
- **订阅服务**：定期支付和会员计划
- **DeFi 集成**：与兑换和借贷的可组合交易

## 支持的钱包

Solana Pay 兼容所有主流 Solana 钱包:

- **[Phantom](https://phantom.app/)** - 最受欢迎的 Solana 钱包
- **[Solflare](https://solflare.com/)** - 多平台 Solana 钱包
- **[Backpack](https://backpack.app/)** - Web3 钱包和应用商店
- **[Glow](https://glow.app/)** - 移动优先的 Solana 钱包
- **[Ultimate](https://ultimate.app/)** - 专注 DeFi 的钱包
- **[Decaf Wallet](https://decaf.so/)** - 全球支付钱包
- **[Espresso Cash](https://www.espressocash.com/)** - 全球支付钱包
- **[Tiplink](https://tiplink.io/)** - 全球最简单的钱包
- **[Bitget Wallet](https://bitget.com/)** - 多链钱包
