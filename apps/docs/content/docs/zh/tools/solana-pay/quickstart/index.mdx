## 安装

设置 Solana Pay SDK 并配置您的开发环境。

**[→ 安装指南](/docs/tools/solana-pay/quickstart/installation)** - 安装 SDK 和依赖项

## 支付类型

根据您的使用场景选择合适的支付集成方式：

### 转账请求（简单）

为基本的 SOL 和 SPL 代币转账创建非交互式支付 URL。

**[→ 转账请求](/docs/tools/solana-pay/quickstart/transfer-requests)** - 简单的支付 URL

**适用于：**

- 基本支付和捐赠
- 二维码支付
- 电子邮件中的支付链接

### 交易请求（高级）

通过服务器端交易组合构建交互式支付流程。

**[→ 交易请求](/docs/tools/solana-pay/quickstart/transaction-requests)** - 交互式支付流程

**适用于：**

- 动态定价
- NFT 铸造
- 复杂的 DeFi 交易
- 自定义业务逻辑

## 二维码集成

为移动钱包支付生成和自定义二维码。

**[→ 二维码集成](/docs/tools/solana-pay/quickstart/qr-codes)** - 生成可扫描的支付码

**适用于：**

- 销售点系统
- 网页到移动端支付
- 面对面交易
