## Kora v2.2.0-beta.6

_2026-03-23_ |
[GitHub 发布](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.6)
| `cargo install kora-cli@2.2.0-beta.6`

预发布版本，支持 Jito 捆绑包、新增 RPC 方法以及增强的安全功能。

- 通过 `signBundle`、`signAndSendBundle` 和 `estimateBundleFee`
  方法新增 Jito 捆绑包支持
- 新增 `getVersion` RPC 方法
- 新增 Lighthouse 费用支付方保护
- 为签名端点新增 reCAPTCHA v3 机器人保护
- 新增基于 Redis 支持的单钱包使用限制
- 在 GHCR 上新增官方 Docker 镜像
- 新增细粒度使用限制规则
- SDK：`KoraClient` 现在接受选项对象；新增 `koraPlugin` 用于 `@solana/kit`

详见[测试版文档](/docs/tools/kora/beta)。

## Kora TypeScript SDK v0.2.0

_2026-03-09_ |
[GitHub 发布](https://github.com/solana-foundation/kora/releases/tag/ts-sdk-v0.2.0)
| `pnpm add @solana/kora@0.2.0`

Kit 客户端插件架构，用于可组合的免 Gas 交易。

- 新增 `createKitKoraClient()`，支持 Kit 插件组合（规划器、执行器、支付方插件）
- 实现与 Kit 程序插件（如 `tokenProgram()`）的可组合性
- 通过 `tokenProgramId` 配置选项新增 Token-2022 支持
- 新增计算预算指令支持，包含基于模拟的 CU 估算
- 在 `signAndSendTransaction` 响应中新增缺失的 `signature` 字段
- 从 `getPaymentInstruction` 返回 `NoopSigner`，用于 Kit v6 签名者去重
- 将 `@solana/kit` 对等依赖项升级到 `^6.1.0`

详见 [Kit 客户端指南](/docs/tools/kora/guides/kit-client)。

## Kora v2.0.0

_2025-11-19_ |
[GitHub 发布](https://github.com/solana-foundation/kora/releases/tag/v2.0.0) |
`cargo install kora-cli@2.0.0`

采用新架构的主要版本。

- 为 Kora 节点引入了全新架构
- 发布了 TypeScript SDK `@solana/kora` v0.1.0
- 迁移到 Solana SDK v3.0.x 并更新了所有依赖项
- 用 solana-signers crate 替换了自定义 Kora Signer
- 移除了 `sign_transaction_if_paid` 和 `get_all_signers` 方法
- 更改了响应类型：从 `signAndSendTransaction` 和 `signTransaction` 中移除了签名
- 增强了开发体验、安全性和性能
