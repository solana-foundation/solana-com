---

## Cargo

直接从 crates.io 安装测试版：

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

官方多平台 Docker 镜像（amd64/arm64）已发布到 GitHub Container Registry：

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

可用标签：

- `latest` — 最新稳定版本
- `beta` — 最新预发布版本
- `v<version>` — 特定版本（例如：`v2.2.0-beta.7`）

### Docker Compose

对于自定义构建或包含 Redis 以支持缓存和使用限制：

- [示例 Dockerfile](https://github.com/solana-foundation/kora/blob/main/examples/deploy/Dockerfile)
- [包含 Redis 的 Docker Compose](https://github.com/solana-foundation/kora/blob/main/docker-compose.yml)

### 使用 Docker 运行

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## TypeScript SDK

测试版 SDK 可在 npm 上获取：

```bash
pnpm add @solana/kora@beta
```

查看 [SDK 变更](/docs/tools/kora/beta/sdk-changes) 了解新的 API。

## 验证安装

```bash
kora --version
```
