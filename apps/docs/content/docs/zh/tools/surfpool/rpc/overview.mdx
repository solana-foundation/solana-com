Surfnet 是 Surfpool 的本地 Solana 网络模拟器，旨在为开发者提供一个强大的测试环境，能够模拟 Solana 区块链的行为，同时无需运行完整的 validator 节点。

## 快速入门

使用 Surfpool CLI 启动 Surfnet 实例：

```bash
surfpool start
```

这将在 `http://localhost:8899`
上启动一个本地 Surfnet 实例，并提供所有 RPC 端点。

## 核心功能

- **完整 RPC 兼容性**：支持完整的 Solana RPC API
- **快速迭代**：无需等待区块确认
- **状态操控**：通过作弊码直接修改区块链状态
- **主网分叉**：从主网分叉，使用真实账户和程序进行测试
- **Geyser 插件支持**：连接您的 Geyser 插件以实现实时数据流
