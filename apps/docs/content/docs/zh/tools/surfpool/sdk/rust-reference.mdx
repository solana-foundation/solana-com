`surfpool-sdk` crate 提供了以 `Surfnet`
结构体为核心的精简接口。本页列出所有公共项目；如需了解使用说明，请参阅从
[概述](/docs/tools/surfpool/sdk/overview) 开始的指南。

## Crate 根导出

| 项目                  | 类型     | 说明                                                |
| --------------------- | -------- | --------------------------------------------------- |
| `Surfnet`             | 结构体   | 运行中的 Surfnet 实例 — RPC/WS 端点、付款方、作弊码 |
| `SurfnetBuilder`      | 结构体   | 供 `Surfnet::builder()` 使用的链式构建器            |
| `Cheatcodes<'_>`      | 结构体   | 由 `Surfnet::cheatcodes()` 返回的借用作弊码辅助器   |
| `SurfnetError`        | 枚举     | SDK 返回的所有错误变体                              |
| `SurfnetResult<T>`    | 类型别名 | `Result<T, SurfnetError>`                           |
| `Keypair`             | 重导出   | `solana_keypair::Keypair`                           |
| `Pubkey`              | 重导出   | `solana_pubkey::Pubkey`                             |
| `Signer`              | 重导出   | `solana_signer::Signer`                             |
| `RpcClient`           | 重导出   | `solana_rpc_client::rpc_client::RpcClient`          |
| `BlockProductionMode` | 重导出   | `surfpool_types::BlockProductionMode`               |
| `SimnetCommand`       | 重导出   | `surfpool_types::SimnetCommand`                     |
| `SimnetEvent`         | 重导出   | `surfpool_types::SimnetEvent`                       |
| `SvmFeatureConfig`    | 重导出   | `surfpool_types::SvmFeatureConfig`                  |

### 生命周期

```rust
impl Surfnet {
    pub fn stop(&mut self) -> SurfnetResult<()>;
}

impl Drop for Surfnet { /* calls stop() if not already stopped */ }
```

`stop` 最多等待五秒，等待运行时释放其端口。`Drop`
期间的失败会被静默忽略；显式调用 `stop` 时的失败则返回 `SurfnetError::Runtime`。

## SurfnetError

```rust
#[derive(Debug)]
pub enum SurfnetError {
    PortAllocation(String),
    Startup(String),
    Runtime(String),
    Cheatcode(String),
    Aborted(String),
}
```

| 变体             | 原因                                     |
| ---------------- | ---------------------------------------- |
| `PortAllocation` | 无法绑定到可用的 RPC 或 WebSocket 端口。 |
| `Startup`        | 运行时初始化失败。                       |
| `Runtime`        | 运行时线程发生 panic 或意外退出。        |
| `Cheatcode`      | 作弊码 JSON-RPC 调用返回错误。           |
| `Aborted`        | 运行时在启动期间被关闭或取消。           |

该 crate 在 `SurfnetError` 上实现了 `std::error::Error` 和
`Display`，因此可开箱即用地与 `?` 和 `anyhow` 配合使用。
