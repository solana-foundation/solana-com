Surfpool 是 `solana-test-validator` 的直接替代品，专为在 **Solana**
上构建的开发者提供最佳体验而设计。

其核心功能是让开发者使用即时获取的主网账户在本地模拟其程序。

Surfpool 还将基础设施即代码无缝引入基于 Anchor 或 Pinocchio 的项目，实现对任何 Solana 网络（私有或公有）的可复现、可审计且安全的部署。
