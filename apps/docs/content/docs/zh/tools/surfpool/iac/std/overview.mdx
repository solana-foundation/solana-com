标准库提供了所有 Runbook 可用的核心函数和操作，无论启用了哪些区块链插件。

## 函数

标准库包含以下实用函数：

- **运算符**：数学运算与比较
- **哈希**：哈希函数（SHA256、Keccak256 等）
- **十六进制**：十六进制编码与解码
- **加密**：加密操作
- **JSON**：JSON 编码与解码
- **列表**：列表操作函数
- **断言**：相等与条件断言
- **Base64**：Base64 编码与解码

[查看所有函数](/docs/tools/surfpool/iac/std/functions)

## 操作

标准库包含以下操作：

- **HTTP**：发起 HTTP 请求
- **文件操作**：读取和写入文件

[查看所有操作](/docs/tools/surfpool/iac/std/actions)
