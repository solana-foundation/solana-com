## assert_eq

`assert_eq` 断言两个值相等。

### 输入

| 名称    | 类型 | 描述           |
| ------- | ---- | -------------- |
| `left`  | any  | 要比较的值     |
| `right` | any  | 要与之比较的值 |

### 输出

| 名称    | 类型                  | 描述       |
| ------- | --------------------- | ---------- |
| `value` | addon(std::assertion) | 断言的结果 |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```

## assert_gt

`assert_gt` 断言左侧值大于右侧值。

### 输入

| 名称    | 类型             | 描述                       |
| ------- | ---------------- | -------------------------- |
| `left`  | integer \| float | 用于比较的整数或浮点数     |
| `right` | integer \| float | 用于与之比较的整数或浮点数 |

### 输出

| 名称    | 类型                  | 描述       |
| ------- | --------------------- | ---------- |
| `value` | addon(std::assertion) | 断言的结果 |

```hcl

```
