---
title: SVM 签名者
description: 适用于 Solana 和 SVM 兼容区块链的签名者
---

这些签名者在使用 SVM 插件进行交易签名时可用。

## secret_key

`svm::secret_key` 签名者可用于使用密钥、助记词或 keypair 文件同步签署交易。

### 输入

| 名称              | 是否必填 | 类型   | 描述                             |
| ----------------- | -------- | ------ | -------------------------------- |
| `secret_key`      | 可选     | string | 用于签署消息和交易的密钥         |
| `mnemonic`        | 可选     | string | 用于生成密钥的助记词短语         |
| `derivation_path` | 可选     | string | 用于生成密钥的派生路径           |
| `keypair_json`    | 可选     | string | 包含密钥的 keypair.json 文件路径 |

### 输出

| 名称         | 类型   | 描述                          |
| ------------ | ------ | ----------------------------- |
| `public_key` | string | 账户的 pubkey                 |
| `address`    | string | SVM 地址（public_key 的别名） |

```hcl
signer "deployer" "svm::secret_key" {
    secret_key = input.secret_key
}
```

您也可以使用助记词：

```hcl
signer "deployer" "svm::secret_key" {
    mnemonic = input.mnemonic
    derivation_path = "m/44'/501'/0'/0'"
}
```

或使用 keypair 文件：

```hcl
signer "deployer" "svm::secret_key" {
    keypair_json = "~/.config/solana/id.json"
}
```

## web_wallet

`svm::web_wallet`
签名者将允许 Runbook 操作者使用其选择的浏览器签名者对交易进行签名。

### 输入

| 名称               | 是否必填 | 类型   | 描述                                      |
| ------------------ | -------- | ------ | ----------------------------------------- |
| `expected_address` | 可选     | string | 预期连接的 SVM 地址（省略则允许任意地址） |

### 输出

| 名称         | 类型   | 描述                |
| ------------ | ------ | ------------------- |
| `address`    | string | 已连接账户的地址    |
| `public_key` | string | 已连接账户的 pubkey |

```hcl
signer "alice" "svm::web_wallet" {
    expected_address = "zbBjhHwuqyKMmz8ber5oUtJJ3ZV4B6ePmANfGyKzVGV"
}
```

此签名者将在 Web UI 中提示用户：

1. 连接钱包
2. 通过消息签名提供 pubkey
3. 对所有需要此签名者的交易进行签名

## squads

`svm::squads` 签名者可用于通过 Squads 多签金库对交易进行签名。

### 输入

| 名称      | 是否必填 | 类型   | 描述           |
| --------- | -------- | ------ | -------------- |
| `address` | 可选     | string | Squad 金库地址 |

### 输出

| 名称         | 类型   | 描述                          |
| ------------ | ------ | ----------------------------- |
| `public_key` | string | Squad 金库的 pubkey           |
| `address`    | string | SVM 地址（public_key 的别名） |

```hcl
signer "deployer" "svm::squads" {
    address = input.address
}
```

使用 Squads 多签可实现安全的多方交易签名，无需暴露个人私钥。
