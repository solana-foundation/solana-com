## and_bool

`and_bool` 返回左侧参数和右侧参数的二进制 AND 结果。

### 输入

| 名称  | 类型 | 描述   |
| ----- | ---- | ------ |
| `lhs` | bool | 布尔值 |
| `rhs` | bool | 布尔值 |

### 输出

| 名称    | 类型 | 描述                              |
| ------- | ---- | --------------------------------- |
| `value` | bool | 两个参数进行二进制 AND 运算的结果 |

```hcl
output "my_bool" {
  value = false && true
}
// > my_bool: false
```

## div

`div` 返回左侧参数除以右侧参数的整数除法结果，任何余数均向下舍入至最近的整数。

### 输入

| 名称  | 类型    | 描述   |
| ----- | ------- | ------ |
| `lhs` | integer | 被除数 |
| `rhs` | integer | 除数   |

### 输出

| 名称    | 类型    | 描述                 |
| ------- | ------- | -------------------- |
| `value` | integer | 被除数除以除数的结果 |

```hcl
output "my_int" {
  value = 11 / -3
}
// > my_int: -3
```

## eq

`eq` 如果左侧参数与右侧参数相等则返回 `true`，如果不相等则返回 `false`。

### 输入

| 名称  | 类型 | 描述   |
| ----- | ---- | ------ |
| `lhs` | any  | 任意值 |
| `rhs` | any  | 任意值 |

### 输出

| 名称    | 类型 | 描述                       |
| ------- | ---- | -------------------------- |
| `value` | bool | 两个输入之间等值检查的结果 |

```hcl
output "is_eq" {
  value = "arg" == "arg"
}
// > is_eq: true
```

| `lhs` | integer \| float \| string | 可比较的值 |
| `rhs` | integer \| float \| string | 可比较的值 |

### 输出

| ------- | ---- | ---------------------------------- |
