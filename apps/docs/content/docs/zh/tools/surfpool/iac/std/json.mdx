## jq

`jq` 函数支持对 JSON 数据进行切片、过滤和映射。更多详情请参阅
[jq 文档](https://jqlang.github.io/jq/manual/)。

### 输入

| 名称           | 类型             | 描述                                                                 |
| -------------- | ---------------- | -------------------------------------------------------------------- |
| `decoded_json` | string \| object | JSON 对象                                                            |
| `query`        | string           | JSON 查询语句。请参阅 [jq 文档](https://jqlang.github.io/jq/manual/) |

### 输出

| 名称    | 类型          | 描述            |
| ------- | ------------- | --------------- |
| `value` | array[string] | `jq` 查询的结果 |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
