SVM `txtx`
插件支持构建与 Solana 及 SVM 兼容区块链交互的 Runbook。该插件提供实用函数，允许您部署 Anchor 程序，并根据程序 IDL 对指令调用进行编码。这些操作可用于创建有效的转账、程序调用和程序部署交易，并可通过助记词、私钥或浏览器签名器进行签名。

## 快速开始

要使用 SVM 插件，请在您的 Runbook 中声明：

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## 功能特性

- **程序部署**：部署 Anchor 和原生 Solana 程序
- **指令构建**：使用程序 IDL 对指令调用进行编码
- **交易签名**：支持助记词、私钥和 Web 钱包
- **账户管理**：查询和管理 Solana 账户
