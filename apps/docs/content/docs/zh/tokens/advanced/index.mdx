description:
  使用 Token Program 将多个代币操作合并为单条指令，并从 token
  account、铸币账户和多签账户中回收 SOL。
url: /docs/tokens/advanced
---

除了常见的创建、铸造和转账操作外，Token
Program 还可通过 CPI 从链上程序调用，并包含用于回收 lamport 的指令：

<Cards>
  <Card
    title="通过 CPI 调用 Token Program"
    href="/docs/tokens/advanced/cpi"
  >
    从链上程序调用 Token Program —— 转账、铸造和销毁代币，使用 PDA 权限签名，并将多条指令合并为单次 CPI。
  </Card>

<Card
  title="提取多余 Lamport"
  href="/docs/tokens/advanced/withdraw-excess-lamports"
>
  从 token account、铸币账户和多签账户中回收超出免租最低额的
  SOL，且不改变代币余额。
</Card>

  <Card title="解包 Lamport" href="/docs/tokens/advanced/unwrap-lamports">
    将已包装的 SOL 全部或部分转换回 lamport，且无需关闭 token account。
  </Card>
</Cards>

<Callout type="info">

_rs`WithdrawExcessLamports`_ 和 _rs`UnwrapLamports`_ 可通过
[`@solana-program/token`](https://www.npmjs.com/package/@solana-program/token)
客户端使用。通过 CPI 调用 Token Program 适用于链上程序。

</Callout>

## Lamport 回收

_rs`WithdrawExcessLamports`_ 可从 token
account、铸币账户和多签账户中回收超出免租最低额的 SOL。_rs`UnwrapLamports`_
可将原生 token account 中已包装的 SOL 转换回 lamport。
