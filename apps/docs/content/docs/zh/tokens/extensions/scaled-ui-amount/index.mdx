## 什么是缩放 UI 数量扩展？

缩放 UI 数量扩展允许发行方为代币的 UI 数量应用一个可更新的倍数。这在代币用于表示现实世界资产（如股票或股息）的场景中非常有用。

例如，股票分割会使所有持有者的股票数量翻倍。如果没有此扩展，在分割事件中以高效且可扩展的方式为所有持有者铸造新代币是不可行的。

然而，借助 Token-2022 扩展模型，我们可以改变 token 的 UI 数量显示方式。通过使用 ScaledUiAmount 扩展和
`amount_to_ui_amount`
指令，你可以为你的 token 设置 UI 乘数，并随时获取其 UI 数量。
[源代码](https://github.com/solana-program/token-2022/tree/main/program/src/extension/scaled_ui_amount)。

此功能还可用于股息或收益分配。

注意：不会创建任何新代币，UI 数量返回的是代币的原始数量乘以当前倍数。此功能完全是外观上的。

<Callout type="info">
  某些扩展之间不兼容，不能在同一个 token mint 或 token account
  上同时启用。你不应在同一个 token mint 或 token account 上同时启用
  `ScaledUiAmount` 扩展和 `InterestBearingMint` 扩展。
</Callout>
