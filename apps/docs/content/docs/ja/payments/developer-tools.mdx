## AI開発ツール

[SolanaでAIツールを始める方法](/docs/tools/ai)のガイドで始めましょう。

### AIエージェントフレームワーク

AIエージェントは、Solana上で決済トランザクションを自律的に実行できます。これらのフレームワークにより、決済処理、トランザクション監視、決済ワークフローの自動化を行うエージェントを構築できます：

| フレームワーク   | 説明                                                                                                      | リンク                                                                                            |
| ---------------- | --------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------- |
| Solana Agent Kit | 決済機能を持つエージェントを構築するためのツール。LangChainおよびVercel AIと互換性あり                    | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [ドキュメント](https://kit.sendai.fun/) |
| Send AI Skills   | Solana上でエージェントを構築するための事前構築されたスキル集                                              | [GitHub](https://github.com/sendaifun/skills)                                                     |
| Eliza Framework  | X、Telegram、Discord ボット向けのSolanaプラグインを備えたマルチプラットフォームエージェントフレームワーク | [GitHub](https://github.com/elizaOS/eliza) • [ドキュメント](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Solana決済アクションを備えたクロスチェーンツールキット                                                    | [GitHub](https://github.com/goat-sdk/goat)                                                        |

### AIプログラム開発
