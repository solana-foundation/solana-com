## AI開発ツール

[SolanaでAIツールを始める方法](/developers/guides/getstarted/intro-to-ai)のガイドで始めましょう。

### Solana MCP

[Solana MCP](https://mcp.solana.com/) (Machine Conversation
Protocol) は、Solanaの知識をClaude、Cursor、Windsurfなどのサポート付きコーディングツールに直接統合します。Solanaドキュメント、Anchorフレームワーク、プログラム例、Stack
ExchangeのQ&Aでトレーニングされており、決済統合をより迅速に構築するのに役立ちます。

### AIエージェントフレームワーク

AIエージェントはSolana上で決済トランザクションを自律的に実行できます。これらのフレームワークにより、決済を処理し、トランザクションを監視し、決済ワークフローを自動化するエージェントを構築できます。

| フレームワーク   | 説明                                                                                             | リンク                                                                                            |
| ---------------- | ------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------- |
| Solana Agent Kit | 決済機能を持つエージェントを構築するためのツール、LangChainおよびVercel AIと互換性あり           | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [ドキュメント](https://kit.sendai.fun/) |
| Send AI Skills   | Solana上でエージェントを構築するための事前構築済みスキルのコレクション                           | [GitHub](https://github.com/sendaifun/skills)                                                     |
| Eliza Framework  | X、Telegram、Discord bot用のSolanaプラグイン付きマルチプラットフォームエージェントフレームワーク | [GitHub](https://github.com/elizaOS/eliza) • [ドキュメント](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Solana決済アクション付きクロスチェーンツールキット                                               | [GitHub](https://github.com/goat-sdk/goat)                                                        |

### AIプログラム開発
