---
title: 決済処理
description: ガスレストランザクションと手数料抽象化により、大規模な決済処理を実現
---

給与支払い、加盟店への支払い、ロイヤルティ報酬など、大規模な決済処理には個別の送金以上のものが必要です。

多数の決済を効率的に処理するには:

1. **命令をバッチ処理**して単一のトランザクションにまとめ、手数料とレイテンシを削減
2. **トランザクションをバッチ処理**してネットワーク全体でスループットを並列化
3. **手数料抽象化**によりユーザーがSOLを保有する必要をなくす
