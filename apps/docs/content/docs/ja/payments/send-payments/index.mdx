---
title: 支払いの送信
description: 即時決済と0.001ドル未満の手数料でステーブルコイン決済を送信
---

給与支払い、ベンダーへの支払い、国際送金のいずれを構築する場合でも、Solanaでの支払い送信は、1秒未満の決済、1セント未満の手数料、そして仲介者による手数料徴収がないことを意味します。照合用のメモを添付したり、数百件の支払いを単一のトランザクションにまとめたり、ユーザーがSOLを保有する必要がないように手数料をスポンサーすることができます。

このセクションでは、単一の送金から大量の決済処理まで、ステーブルコインを移動するための中核的なパターンを説明します。
