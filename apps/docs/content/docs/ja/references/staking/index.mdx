SOLトークンをステーキングすることで、ネットワークのセキュリティ確保に貢献し、同時に[報酬を獲得](https://docs.anza.xyz/implemented-proposals/staking-rewards)できます。

トランザクションを処理しネットワークを運営するバリデーターにトークンを委任することで、ステーキングできます。

ステークの委任は、リスクと報酬を共有する金融モデルであり、長期間委任されたトークンの保有者に利益をもたらす可能性があります。これは、トークン保有者(委任者)と委任先のバリデーターの金銭的インセンティブを一致させることで実現されます。

バリデーターに委任されるステークが多いほど、このバリデーターが台帳に新しいトランザクションを書き込むために選ばれる頻度が高くなります。バリデーターが書き込むトランザクションが多いほど、バリデーターとその委任者が獲得する報酬も多くなります。より多くのトランザクションを処理できるようにシステムを構成したバリデーターは、比例してより多くの報酬を獲得し、ネットワークを可能な限り高速かつスムーズに稼働させ続けます。

バリデーターはシステムの運用と維持にコストを負担しており、これは獲得した報酬の一定割合として徴収される手数料の形で委任者に転嫁されます。この手数料は*コミッション*として知られています。バリデーターは委任されるステークが多いほど多くの報酬を獲得するため、サービスに対して最も低いコミッションを提供するために互いに競争する可能性があります。

これは現在Solanaプロトコルには実装されていませんが、将来的には、委任者が*スラッシング*として知られるプロセスを通じてステーキング時にトークンを失うリスクを負う可能性があります。スラッシングは、無効なトランザクションの作成や特定の種類のトランザクションまたはネットワーク参加者の検閲など、意図的な悪意のある行動に対応して、バリデーターのSOLの一部を削除および破棄することを含みます。

現在、プロトコル内にスラッシングの実装はありません。スラッシングの詳細については、[スラッシングロードマップ](https://docs.anza.xyz/proposals/optimistic-confirmation-and-slashing#slashing-roadmap)を参照してください。

## SOLトークンをステーキングするにはどうすればよいですか?

ステーキングをサポートするウォレットにトークンを移動することで、SOLをステーキングできます。ウォレットは、ステークアカウントの作成と委任を行うための手順を提供します。

### 対応ウォレット

多くのウェブおよびモバイルウォレットがSolanaのステーキング操作をサポートしています。ステータスについては、お気に入りのウォレットのメンテナーに確認してください。

### Solanaコマンドラインツール

- SolanaコマンドラインツールはCLIで生成されたkeypairファイルウォレット、ペーパーウォレット、または接続されたLedger
  Nanoと組み合わせて、すべてのステーク操作を実行できます。[Solanaコマンドラインツールを使用したステーキングコマンド](https://docs.anza.xyz/cli/examples/delegate-stake)。

### ステークアカウントの作成

ステーキングアカウントの作成については、ウォレットの指示に従ってください。このアカウントは、単にトークンを送受信するために使用されるものとは異なるタイプになります。

### validatorの選択

validatorの選択については、ウォレットの指示に従ってください。パフォーマンスの高い可能性のあるvalidatorに関する情報は、以下のリンクから入手できます。Solana
Foundationは特定のvalidatorを推奨していません。

solanabeach.ioサイトは、validatorの1つであるStaking
Facilitiesによって構築および維持されています。ネットワーク全体に関する高レベルのグラフィカル情報、および各validatorのリストと各validatorに関する最近のパフォーマンス統計を提供しています。

- https://solanabeach.io

ブロック生成統計を表示するには、Solanaコマンドラインツールを使用します。

- `solana validators`
- `solana block-production`

Solanaチームは、この情報の解釈方法について推奨事項を提供していません。ご自身でデューデリジェンスを行ってください。

### ステークを委任する

選択したvalidatorにステークを委任するには、ウォレットの指示に従ってください。
