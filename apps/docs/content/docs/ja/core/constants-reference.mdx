このページは、コアコンセプトページ全体で言及されているすべてのランタイム定数の単一リファレンスです。各定数は、[agave](https://github.com/anza-xyz/agave/tree/v3.1.8)
validatorクライアントまたは[solana-sdk](https://github.com/anza-xyz/solana-sdk)のソース定義にリンクしています。
