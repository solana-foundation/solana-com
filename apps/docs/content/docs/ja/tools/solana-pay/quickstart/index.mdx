これらのステップバイステップガイドで、Solana Payをすぐに使い始めましょう。

## インストール

Solana Pay SDKをセットアップし、開発環境を構成します。

**[→ インストールガイド](/docs/tools/solana-pay/quickstart/installation)** -
SDKと依存関係をインストール

## 決済タイプ

ユースケースに適した決済統合を選択してください：

### 転送リクエスト（シンプル）

基本的なSOLおよびSPLトークン転送用の非インタラクティブな決済URLを作成します。

**[→ 転送リクエスト](/docs/tools/solana-pay/quickstart/transfer-requests)** - シンプルな決済URL

**最適な用途：**

- 基本的な支払いと寄付
- QRコード決済
- メール内の決済リンク

### トランザクションリクエスト（高度）

サーバーサイドのトランザクション構成によるインタラクティブな決済フローを構築します。

**[→ トランザクションリクエスト](/docs/tools/solana-pay/quickstart/transaction-requests)** - インタラクティブな決済フロー

**最適な用途：**

- 動的価格設定
- NFTミント
- 複雑なDeFiトランザクション
- カスタムビジネスロジック

## QRコード統合

モバイルウォレット決済用のQRコードを生成およびカスタマイズします。

**[→ QRコード統合](/docs/tools/solana-pay/quickstart/qr-codes)** - スキャン可能な決済コードを生成

**最適な用途：**

- POSシステム
- ウェブからモバイルへの決済
- 対面取引
