**Solana
Payは、開発者が分散型決済をアプリやサービスに組み込むことを可能にする標準プロトコルおよびリファレンス実装のセットです。**
Solanaブロックチェーンは1秒未満でトランザクションを承認し、平均コストは$0.0005であり、仲介者なしでシームレスなユーザー体験を提供します。

## なぜSolana Payなのか？

- **標準プロトコル**: すべてのSolanaウォレットとアプリ全体で統一された決済アプローチ
- **超高速**: トランザクションは1秒未満で承認
- **超低コスト**: 平均トランザクションコスト$0.0005
- **仲介者不要**: 第三者を介さない直接のピアツーピア決済
- **ユニバーサルウォレット対応**: Phantom、Solflare、Backpackなどに対応

## クイックスタート

最も簡単な始め方は、転送リクエストURLを使用することです：

```
solana:recipient?amount=0.01&label=Coffee%20Shop&message=Grande%20Latte
```

このURLは次のように利用できます：

- モバイルウォレットでスキャンするためのQRコードとしてエンコード
- Webアプリケーションでの決済リンクとして使用
- モバイルアプリにディープリンクとして埋め込み

## インストール

Solana Pay JavaScript SDKをインストール：

```bash
# Using pnpm (recommended)
pnpm add @solana/pay@beta @solana/kit

# Using npm
npm install @solana/pay@beta @solana/kit

# Using yarn
yarn add @solana/pay@beta @solana/kit
```

## ユースケース

Solana Payは幅広い決済シナリオを実現します：

- **小売決済**: QRコードを使用した対面決済
- **Eコマース**: ウォレット統合によるWebチェックアウトフロー
- **モバイルアプリ**: ディープリンクによるネイティブ決済体験
- **NFTミント**: 複雑なインタラクションのための動的トランザクション作成
- **サブスクリプション**: 定期決済とロイヤルティプログラム
- **DeFi統合**: スワップや貸付との組み合わせ可能なトランザクション

## 対応ウォレット

Solana Payは、主要なSolanaウォレットすべてに対応しています：

- **[Phantom](https://phantom.app/)** - 最も人気のあるSolanaウォレット
- **[Solflare](https://solflare.com/)** - マルチプラットフォーム対応のSolanaウォレット
- **[Backpack](https://backpack.app/)** - Web3ウォレット＆アプリストア
- **[Glow](https://glow.app/)** - モバイルファーストのSolanaウォレット
- **[Ultimate](https://ultimate.app/)** - DeFi特化型ウォレット
- **[Decaf Wallet](https://decaf.so/)** - グローバル決済対応ウォレット
- **[Espresso Cash](https://www.espressocash.com/)** - グローバル決済対応ウォレット
- **[Tiplink](https://tiplink.io/)** - 世界一シンプルなウォレット
- **[Bitget Wallet](https://bitget.com/)** - マルチチェーン対応ウォレット
