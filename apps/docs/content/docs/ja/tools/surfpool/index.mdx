Surfpoolは `solana-test-validator`
のドロップイン代替ツールであり、**Solana**上で開発するデベロッパーに最高の体験を提供することを目的として設計されています。

その中核機能は、デベロッパーがメインネットのアカウントをジャストインタイムで取得し、ローカル環境でプログラムをシミュレートできることです。

SurfpoolはさらにAnchorまたはPinocchioベースのプロジェクトにInfrastructure as
Codeをシームレスに導入し、あらゆるSolanaネットワーク（プライベート・パブリックを問わず）への再現性・監査可能・セキュアなデプロイを実現します。
