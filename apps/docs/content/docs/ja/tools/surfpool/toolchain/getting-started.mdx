Surfpool
CLIは、インストールスクリプトを使用するか、ソースからビルドしてインストールできます。

## インストールスクリプト

Surfpoolをインストールするには、ターミナルを開いて次のコマンドを実行してください：

```bash
curl -sL https://run.surfpool.run/ | bash
```

## ソースからインストール

ソースからSurfpoolバイナリをコンパイルするには、Surfpoolリポジトリをクローンしてください。

```bash
git clone https://github.com/solana-foundation/surfpool.git
```

新しいディレクトリに移動します。

```bash
cd surfpool
```

次に、バイナリをコンパイルします。このコマンドはバイナリをコンパイルし、`~/.cargo/bin/surfpool`に保存します。

```bash
cargo surfpool-install
```

最後に、`surfpool`バイナリがPATHで利用可能であることを確認してください。この手順はオペレーティングシステムによって異なります。
