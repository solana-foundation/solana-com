`surfpool-sdk` クレートは、`Surfnet`
構造体を中心とした小さなサーフェスを公開しています。このページではすべてのパブリックアイテムを一覧表示しています。使用方法のガイドについては、[概要](/docs/tools/surfpool/sdk/overview)から始まるガイドを参照してください。

## クレートルートのエクスポート

| アイテム              | 種類           | 備考                                                                       |
| --------------------- | -------------- | -------------------------------------------------------------------------- |
| `Surfnet`             | 構造体         | 実行中のSurfnetインスタンス — RPC/WSエンドポイント、ペイヤー、チートコード |
| `SurfnetBuilder`      | 構造体         | `Surfnet::builder()` が使用するフルエントビルダー                          |
| `Cheatcodes<'_>`      | 構造体         | `Surfnet::cheatcodes()` が返す借用チートコードヘルパー                     |
| `SurfnetError`        | 列挙型         | SDKが返すすべてのエラーバリアント                                          |
| `SurfnetResult<T>`    | 型エイリアス   | `Result<T, SurfnetError>`                                                  |
| `Keypair`             | 再エクスポート | `solana_keypair::Keypair`                                                  |
| `Pubkey`              | 再エクスポート | `solana_pubkey::Pubkey`                                                    |
| `Signer`              | 再エクスポート | `solana_signer::Signer`                                                    |
| `RpcClient`           | 再エクスポート | `solana_rpc_client::rpc_client::RpcClient`                                 |
| `BlockProductionMode` | 再エクスポート | `surfpool_types::BlockProductionMode`                                      |
| `SimnetCommand`       | 再エクスポート | `surfpool_types::SimnetCommand`                                            |
| `SimnetEvent`         | 再エクスポート | `surfpool_types::SimnetEvent`                                              |
| `SvmFeatureConfig`    | 再エクスポート | `surfpool_types::SvmFeatureConfig`                                         |

```rust

    pub fn rpc_url(&self) -> &str;
    pub fn ws_url(&self) -> &str;

    pub fn instance_id(&self) -> &str;
    pub fn rpc_client(&self) -> RpcClient;
    pub fn cheatcodes(&self) -> Cheatcodes<'_>;

```

### ライフサイクル

```rust
impl Surfnet {
    pub fn stop(&mut self) -> SurfnetResult<()>;
}

impl Drop for Surfnet { /* calls stop() if not already stopped */ }
```

`stop` はランタイムがポートを解放するまで最大5秒間待機します。`Drop`
中の失敗はサイレントです。明示的な `stop` 中の失敗は `SurfnetError::Runtime`
を返します。

## SurfnetError

```rust
#[derive(Debug)]
pub enum SurfnetError {
    PortAllocation(String),
    Startup(String),
    Runtime(String),
    Cheatcode(String),
    Aborted(String),
}
```

| バリアント       | 原因                                                           |
| ---------------- | -------------------------------------------------------------- |
| `PortAllocation` | RPC または WebSocket の空きポートにバインドできませんでした。  |
| `Startup`        | ランタイムの初期化に失敗しました。                             |
| `Runtime`        | ランタイムスレッドがパニックまたは予期せず終了しました。       |
| `Cheatcode`      | チートコードの JSON-RPC 呼び出しがエラーを返しました。         |
| `Aborted`        | 起動中にランタイムがシャットダウンまたはキャンセルされました。 |

このクレートは `std::error::Error` および `Display` を `SurfnetError`
に実装しているため、`?` および `anyhow` とすぐに連携できます。
