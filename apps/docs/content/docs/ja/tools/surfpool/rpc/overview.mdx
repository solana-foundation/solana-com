SurfnetはSurfpoolのローカルSolanaネットワークシミュレーターです。完全なvalidatorノードのオーバーヘッドなしにSolanaブロックチェーンの動作を再現する、強力なテスト環境を開発者に提供するよう設計されています。

## はじめに

Surfpool CLIを使用してSurfnetインスタンスを起動します：

```bash
surfpool start
```

これにより、`http://localhost:8899`ですべてのRPCエンドポイントが利用可能なローカルSurfnetインスタンスが起動します。

## 主な機能

- **完全なRPC互換性**：Solana RPC APIの全機能をサポート
- **高速なイテレーション**：ブロック確認を待つ必要なし
- **ステート操作**：ブロックチェーンの状態を直接変更するチートコード
- **メインネットフォーク**：実際のアカウントとプログラムでテストするためにメインネットからフォーク
- **Geyserプラグインサポート**：リアルタイムデータストリーミングのためにGeyserプラグインを接続
