SVM `txtx`
プラグインを使用すると、SolanaおよびSVM対応ブロックチェーンと連携するRunbookを構築できます。このプラグインは、Anchorプログラムのデプロイや、プログラムIDLに基づくinstructionsコールのエンコードを可能にするユーティリティ関数を提供します。これらのアクションを使用して、有効な送金、プログラム呼び出し、およびプログラムデプロイのトランザクションを作成し、ニーモニックフレーズ、シークレットキー、またはブラウザの署名者を通じて署名することができます。

## クイックスタート

SVMアドオンを使用するには、Runbook内で宣言してください：

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## 機能

- **プログラムのデプロイ**: AnchorおよびネイティブのSolanaプログラムをデプロイ
- **instructions構築**: プログラムIDLを使用してinstructionsコールをエンコード
- **トランザクション署名**: ニーモニック、シークレットキー、Webウォレットに対応
- **アカウント管理**: Solanaアカウントのクエリと管理
