## encode_hex

`encode_hex`
は、バッファまたは文字列を0xプレフィックス付きの16進数文字列としてエンコードします。

### 入力

| 名前    | 型                    | 説明                                                                                                                                  |
| ------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer、string、addon | エンコードするバッファまたは文字列。'0x'で始まる文字列は16進数としてデコードされます。それ以外の場合は生のUTF-8バイトが使用されます。 |

### 出力

| 名前    | 型     | 説明                                         |
| ------- | ------ | -------------------------------------------- |
| `value` | string | 0xプレフィックス付きの16進数表現による入力値 |

```hcl
output "encoded_hex" {
    value = encode_hex("hello, world")
}
// > encoded_hex: 0x68656c6c6f2c20776f726c64
```

## decode_hex

`decode_hex` は、16進数文字列をデコードし、結果をバッファとして返します。

### 入力

| 名前         | 型     | 説明                                                           |
| ------------ | ------ | -------------------------------------------------------------- |
| `hex_string` | string | デコードする16進数文字列（0xプレフィックスあり・なし両方対応） |

### 出力

| 名前    | 型     | 説明                                     |
| ------- | ------ | ---------------------------------------- |
| `value` | buffer | バッファとしてデコードされた16進数文字列 |

```hcl
output "decoded_hex" {
    value = decode_hex("0x68656c6c6f2c20776f726c64")
}
// > decoded_hex: 0x68656c6c6f2c20776f726c64
```
