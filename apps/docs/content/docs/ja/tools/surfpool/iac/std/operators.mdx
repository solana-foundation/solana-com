## and_bool

`and_bool` は、左辺と右辺の引数のビット AND を返します。

### 入力

| 名前  | 型   | 説明     |
| ----- | ---- | -------- |
| `lhs` | bool | ブール値 |
| `rhs` | bool | ブール値 |

### 出力

| 名前    | 型   | 説明                         |
| ------- | ---- | ---------------------------- |
| `value` | bool | 2つの引数のビット AND の結果 |

```hcl
output "my_bool" {
  value = false && true
}
// > my_bool: false
```

## div

`div`
は、左辺の引数を右辺の引数で整数除算した結果を返します。余りは最も近い整数に切り捨てられます。

### 入力

| 名前  | 型      | 説明   |
| ----- | ------- | ------ |
| `lhs` | integer | 被除数 |
| `rhs` | integer | 除数   |

### 出力

| 名前    | 型      | 説明                     |
| ------- | ------- | ------------------------ |
| `value` | integer | 被除数を除数で割った結果 |

```hcl
output "my_int" {
  value = 11 / -3
}
// > my_int: -3
```

## eq

`eq` は、左辺と右辺の引数が等しい場合に `true` を返し、等しくない場合に `false`
を返します。

### 入力

| 名前  | 型  | 説明     |
| ----- | --- | -------- |
| `lhs` | any | 任意の値 |
| `rhs` | any | 任意の値 |

### 出力

| 名前    | 型   | 説明                            |
| ------- | ---- | ------------------------------- |
| `value` | bool | 2つの入力間の等値チェックの結果 |

```hcl
output "is_eq" {
  value = "arg" == "arg"
}
// > is_eq: true
```

| 名前  | 型                         | 説明         |
| ----- | -------------------------- | ------------ |
| `lhs` | integer \| float \| string | 比較可能な値 |
| `rhs` | integer \| float \| string | 比較可能な値 |

### 出力

| 名前    | 型   | 説明                                                   |
| ------- | ---- | ------------------------------------------------------ |
| `value` | bool | 左辺の引数が右辺の引数より大きいかどうかを確認した結果 |

```hcl
output "is_gt" {
  value = 2 > 1
}
// > is_gt: true
```

## neq

`neq` は、左辺と右辺の引数が等しくない場合に `true` を返し、そうでない場合は

### 入力

| 名前  | 型  | 説明         |
| ----- | --- | ------------ |
| `lhs` | any | 比較する値   |
| `rhs` | any | 比較対象の値 |

### 出力

| 名前    | 型   | 説明                                |
| ------- | ---- | ----------------------------------- |
| `value` | bool | 2つの入力間の否定等価チェックの結果 |

```hcl
output "is_neq" {
  value = "arg" != "arg"
}
// > is_neq: false
```
