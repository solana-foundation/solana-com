## encode_base58

`encode_base58` は、バッファまたは文字列をBase58文字列としてエンコードします。

### 入力

| 名前    | 型                    | 説明                                                                                                                                    |
| ------- | --------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer、string、addon | エンコードするバッファまたは文字列。'0x'で始まる文字列は16進数としてデコードされます。それ以外の場合は、生のUTF-8バイトが使用されます。 |

### 出力

| 名前    | 型     | 説明                                     |
| ------- | ------ | ---------------------------------------- |
| `value` | string | Base58文字列としてエンコードされた入力値 |

```hcl
output "encoded" {
    value = encode_base58("0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8")
}
// > encoded: CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF
```

## decode_base58

`decode_base58`
は、Base58エンコードされた文字列をデコードし、結果をバッファとして返します。

### 入力

| 名前            | 型     | 説明                     |
| --------------- | ------ | ------------------------ |
| `base58_string` | string | デコードするBase58文字列 |

### 出力

| 名前    | 型     | 説明                                     |
| ------- | ------ | ---------------------------------------- |
| `value` | buffer | バッファとしてデコードされたBase58文字列 |

```hcl
output "decoded" {
    value = decode_base58("CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF")
}
// > decoded: 0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8
```
