標準ライブラリは、有効になっているブロックチェーンアドオンに関わらず、すべてのRunbookで使用できるコア関数とアクションを提供します。

## 関数

標準ライブラリには、以下のユーティリティ関数が含まれています：

- **演算子**: 数学的演算と比較
- **Hash**: ハッシュ関数（SHA256、Keccak256など）
- **Hex**: 16進数のエンコードとデコード
- **Crypto**: 暗号化操作
- **JSON**: JSONのエンコードとデコード
- **List**: リスト操作関数
- **Assertions**: 等値条件アサーション
- **Base64**: Base64のエンコードとデコード

[すべての関数を表示](/docs/tools/surfpool/iac/std/functions)

## アクション

標準ライブラリには、以下のアクションが含まれています：

- **HTTP**: HTTPリクエストの実行
- **File Operations**: ファイルの読み書き

[すべてのアクションを表示](/docs/tools/surfpool/iac/std/actions)
