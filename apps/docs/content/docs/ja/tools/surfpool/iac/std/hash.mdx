## ripemd160

`ripemd160` は値のRipemd160ハッシュを計算します。

### 入力

| 名前    | 型                      | 説明                                   |
| ------- | ----------------------- | -------------------------------------- |
| `value` | buffer \| array[buffer] | ハッシュ化する16進数エンコードされた値 |

### 出力

| 名前    | 型     | 説明                 |
| ------- | ------ | -------------------- |
| `value` | string | ハッシュ化された結果 |

```hcl
output "hashed_data" {
    value = ripemd160(encode_hex("hello, world"))
}
// > hashed_data: 0XA3201F82FCA034E46D10CD7B27E174976E241DA2
```

## sha256

`sha256` は値のsha256ハッシュを計算します。

### 入力

| 名前    | 型                      | 説明                                   |
| ------- | ----------------------- | -------------------------------------- |
| `value` | buffer \| array[buffer] | ハッシュ化する16進数エンコードされた値 |

### 出力

| 名前    | 型     | 説明                 |
| ------- | ------ | -------------------- |
| `value` | string | ハッシュ化された結果 |

```hcl
output "hashed_data" {
    value = sha256(encode_hex("hello, world"))
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```

## keccak256

`std::keccak256` は値のkeccak256ハッシュを計算します。

### 入力

| 名前    | 型     | 説明                     |
| ------- | ------ | ------------------------ |
| `value` | string | ハッシュ化する文字列の値 |

### 出力

| 名前    | 型     | 説明                 |
| ------- | ------ | -------------------- |
| `value` | string | ハッシュ化された結果 |

```hcl
output "hashed_data" {
    value = keccak256("hello, world")
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```
