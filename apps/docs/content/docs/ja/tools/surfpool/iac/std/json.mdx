## jq

`jq`関数を使用すると、JSONデータのスライス、フィルタリング、およびマッピングが可能です。詳細については、
[jqドキュメント](https://jqlang.github.io/jq/manual/)を参照してください。

### 入力

| 名前           | 型               | 説明                                                                                  |
| -------------- | ---------------- | ------------------------------------------------------------------------------------- |
| `decoded_json` | string \| object | JSONオブジェクト                                                                      |
| `query`        | string           | JSONクエリ。[jqドキュメント](https://jqlang.github.io/jq/manual/)を参照してください。 |

### 出力

| 名前    | 型            | 説明             |
| ------- | ------------- | ---------------- |
| `value` | array[string] | `jq`クエリの結果 |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
