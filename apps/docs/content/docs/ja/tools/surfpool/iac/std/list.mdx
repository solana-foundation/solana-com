## index

`index` は、指定されたインデックスのリストからエントリを取得します。

### 入力

| 名前    | 型                                               | 説明                           |
| ------- | ------------------------------------------------ | ------------------------------ |
| `list`  | array[string] \| array[integer] \| array[buffer] | エントリを取得するリスト       |
| `index` | integer                                          | 取得するエントリのインデックス |

### 出力

| 名前    | 型     | 説明                                         |
| ------- | ------ | -------------------------------------------- |
| `value` | string | 指定されたインデックスのリストからのエントリ |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
