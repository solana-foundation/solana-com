## assert_eq

`assert_eq` は、2つの値が等しいことをアサートします。

### 入力

| 名前    | 型  | 説明         |
| ------- | --- | ------------ |
| `left`  | any | 比較する値   |
| `right` | any | 比較対象の値 |

### 出力

| 名前    | 型                    | 説明               |
| ------- | --------------------- | ------------------ |
| `value` | addon(std::assertion) | アサーションの結果 |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```
