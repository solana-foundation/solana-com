## encode_base64

`encode_base64` は、バッファまたは文字列をBase64文字列としてエンコードします。

### 入力

| 名前    | 型                    | 説明                                                                                                                                    |
| ------- | --------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer、string、addon | エンコードするバッファまたは文字列。'0x'で始まる文字列は16進数としてデコードされます。それ以外の場合は、生のUTF-8バイトが使用されます。 |

### 出力

| 名前    | 型     | 説明                                     |
| ------- | ------ | ---------------------------------------- |
| `value` | string | Base64文字列としてエンコードされた入力値 |

```hcl
output "encoded" {
    value = encode_base64("0x48656c6c6f20776f726c6421")
}
// > encoded: SGVsbG8gd29ybGQh
```

## decode_base64

`decode_base64`
は、Base64エンコードされた文字列をデコードし、結果をバッファとして返します。

### 入力

| 名前            | 型     | 説明                     |
| --------------- | ------ | ------------------------ |
| `base64_string` | string | デコードするBase64文字列 |

### 出力

| 名前    | 型     | 説明                                     |
| ------- | ------ | ---------------------------------------- |
| `value` | buffer | バッファとして返されたBase64デコード結果 |

```hcl
output "decoded" {
    value = decode_base64("SGVsbG8gd29ybGQh")
}
// > decoded: 0x48656c6c6f20776f726c6421
```
