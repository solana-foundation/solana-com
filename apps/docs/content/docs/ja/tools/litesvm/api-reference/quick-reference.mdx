## クイックリファレンス

| メソッド                                       | 説明                                                                                     |
| ---------------------------------------------- | ---------------------------------------------------------------------------------------- |
| `LiteSVM::new()`                               | LiteSVMの基本テスト環境を新規作成する                                                    |
| `LiteSVM::default()`                           | 型の「デフォルト値」を返す                                                               |
| `airdrop(&Address, lamports)`                  | 指定したlamportでアカウントに資金を供給する                                              |
| `get_account(&Address)`                        | 指定したアドレスのアカウントに関連するすべての情報を返す                                 |
| `set_account(Address, Account)`                | 指定したアドレスのアカウントに関連するすべての情報を設定する                             |
| `get_balance(&Address)`                        | 指定したアカウントアドレスのlamport残高を取得する                                        |
| `minimum_balance_for_rent_exemption(data_len)` | 指定したデータ長のアカウントをrent免除にするために必要な最低残高を返す                   |
| `add_program(program_id, &bytes)`              | テスト環境にSBFプログラムを追加する                                                      |
| `add_program_from_file(program_id, path)`      | 指定したファイルからテスト環境にSBFプログラムを追加する                                  |
| `send_transaction(tx)`                         | 署名済みトランザクションを送信する                                                       |
| `simulate_transaction(tx)`                     | 状態を変更せずにトランザクションをシミュレートする                                       |
| `get_transaction(&signature)`                  | 履歴からトランザクションを取得する                                                       |
| `latest_blockhash()`                           | 現在のブロックハッシュを取得する                                                         |
| `expire_blockhash()`                           | 現在のブロックハッシュを強制的に期限切れにする                                           |
| `warp_to_slot(slot)`                           | クロックを指定したslotに移動する                                                         |
| `get_sysvar<T>()`                              | テスト環境からsysvarを取得する                                                           |
| `set_sysvar<T>(&value)`                        | テスト環境にsysvarを設定する                                                             |
| `with_feature_accounts()`                      | テスト環境にフィーチャーゲートアカウントを含める                                         |
| `set_invocation_inspect_callback(callback)`    | 呼び出し前後のコールバックを登録する（`invocation-inspect-callback` フィーチャーが必要） |
