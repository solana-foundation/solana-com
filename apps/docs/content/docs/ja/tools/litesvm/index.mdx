---
title: LiteSVM
description: インプロセスVMによる超高速Solanaプログラムテスト。ネットワークオーバーヘッドやvalidatorプロセスなしで、プログラムをより速くテストできます。
---

LiteSVMは、Solanaプログラムをテストするための軽量ライブラリです。プログラム開発者向けに最適化されたインプロセスのSolana
VMを内部に作成することで動作します。別のvalidatorプロセスを起動する他のテスト手法とは異なり、LiteSVMはVMをテスト内部に組み込むため、実行が大幅に高速化されます。さらに従来の慣習から一歩踏み込み、使いやすいAPIと合理的なデフォルト設定を備えており、必要に応じて広範なカスタマイズも可能です。
