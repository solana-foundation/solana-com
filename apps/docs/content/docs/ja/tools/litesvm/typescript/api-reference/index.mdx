## クイックリファレンス

### プラグインのセットアップ

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### アカウント管理 (client.svm)

| メソッド                               | 説明                           |
| -------------------------------------- | ------------------------------ |
| `setAccount(account)`                  | アカウントの状態を直接設定する |
| `getAccount(address)`                  | アカウントデータを取得する     |
| `getBalance(address)`                  | SOL残高を取得する              |
| `airdrop(address, lamports)`           | アカウントに資金を送る         |
| `minimumBalanceForRentExemption(size)` | 家賃免除の最低額を計算する     |

### プログラム管理 (client.svm)

| メソッド                              | 説明                                |
| ------------------------------------- | ----------------------------------- |
| `addProgram(programId, bytes)`        | バイトからプログラムを読み込む      |
| `addProgramFromFile(programId, path)` | .soファイルからプログラムを読み込む |

### トランザクション (client.svm)

| メソッド                  | 説明                             |
| ------------------------- | -------------------------------- |
| `sendTransaction(tx)`     | トランザクションを実行する       |
| `simulateTransaction(tx)` | 状態変更なしにシミュレートする   |
| `latestBlockhash()`       | 現在のブロックハッシュを取得する |
| `expireBlockhash()`       | 新しいブロックハッシュに進める   |

### 設定 (client.svm)

| メソッド                    | 説明                                 |
| --------------------------- | ------------------------------------ |
| `withSigverify(bool)`       | 署名検証の有効化／無効化             |
| `withBlockhashCheck(bool)`  | ブロックハッシュ検証の有効化／無効化 |
| `withSysvars()`             | sysvarアカウントの有効化             |
| `withBuiltins()`            | 組み込みプログラムの有効化           |
| `withPrecompiles()`         | プリコンパイル済みプログラムの有効化 |
| `withTransactionHistory(n)` | 最後のnトランザクションを保存する    |
| `withComputeBudget(budget)` | コンピュート制限を設定する           |
| `withLamports(amount)`      | デフォルトのlamportを設定する        |
| `withLogBytesLimit(n)`      | ログ出力制限を設定する               |

### 時間とSysvar（client.svm）

| メソッド                   | 説明                    |
| -------------------------- | ----------------------- |
| `warpToSlot(slot)`         | 特定のslotにジャンプ    |
| `getClock()`               | クロックsysvarを取得    |
| `getRent()`                | rent sysvarを取得       |
| `getEpochSchedule()`       | epochスケジュールを取得 |
| `getLastRestartSlot()`     | 最後の再起動slotを取得  |
| `setLastRestartSlot(slot)` | 最後の再起動slotを設定  |

### RPC（client.rpc）

| メソッド                         | 説明                               |
| -------------------------------- | ---------------------------------- |
| `getAccountInfo(address)`        | 単一アカウントを取得（base64のみ） |
| `getMultipleAccounts(addresses)` | 複数アカウントを取得               |
| `getLatestBlockhash()`           | 現在のブロックハッシュを取得       |
