## Kora v2.2.0-beta.6

_2026年3月23日_ |
[GitHub リリース](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.6)
| `cargo install kora-cli@2.2.0-beta.6`

Jito バンドルサポート、新しい RPC メソッド、強化されたセキュリティ機能を含むプレリリース。

- `signBundle`、`signAndSendBundle`、 `estimateBundleFee`
  メソッドによる Jito バンドルサポートを追加
- `getVersion` RPC メソッドを追加
- Lighthouse 手数料支払者保護を追加
- 署名エンドポイント用の reCAPTCHA v3 ボット保護を追加
- Redis サポート付きのウォレット別使用制限を追加
- GHCR での公式 Docker イメージを追加
- 詳細な使用制限ルールを追加
- SDK: `KoraClient` がオプションオブジェクトを受け取るように変更、`@solana/kit`
  用の新しい `koraPlugin` を追加

詳細は [ベータドキュメント](/docs/tools/kora/beta) をご覧ください。

## Kora TypeScript SDK v0.2.0

_2026年3月9日_ |
[GitHub リリース](https://github.com/solana-foundation/kora/releases/tag/ts-sdk-v0.2.0)
| `pnpm add @solana/kora@0.2.0`

合成可能なガスレストランザクションのための Kit クライアントプラグインアーキテクチャ。

- Kit プラグイン合成（プランナー、エグゼキューター、ペイヤープラグイン）による
  `createKitKoraClient()` を追加
- `tokenProgram()` などの Kit プログラムプラグインとの合成可能性を実現
- `tokenProgramId` 設定オプションによる Token-2022 サポートを追加
- シミュレーションベースの CU 推定による計算バジェット命令サポートを追加
- `signAndSendTransaction` レスポンスに不足していた `signature` フィールドを追加
- Kit v6 署名者の重複排除のために `getPaymentInstruction` から `NoopSigner`
  を返却
- `@solana/kit` ピア依存関係を `^6.1.0` にアップデート

詳細は [Kit クライアントガイド](/docs/tools/kora/guides/kit-client)
をご覧ください。

## Kora v2.0.0

_2025-11-19_ |
[GitHub リリース](https://github.com/solana-foundation/kora/releases/tag/v2.0.0)
| `cargo install kora-cli@2.0.0`

新しいアーキテクチャを含むメジャーリリース。

- Kora ノードの新しいアーキテクチャを導入
- TypeScript SDK `@solana/kora` v0.1.0 を公開
- Solana SDK v3.0.x に移行し、すべての依存関係を更新
- カスタム Kora Signer を solana-signers クレートに置き換え
- `sign_transaction_if_paid` および `get_all_signers` メソッドを削除
- レスポンスタイプを変更：`signAndSendTransaction` および `signTransaction`
  から署名を削除
- 開発者体験、セキュリティ、パフォーマンスを強化
