## Cargo

crates.ioから直接ベータ版リリースをインストールします：

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

公式のマルチプラットフォームDockerイメージ（amd64/arm64）がGitHub Container
Registryに公開されています：

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

利用可能なタグ：

- `latest` — 最新の安定版リリース
- `beta` — 最新のプレリリースバージョン
- `v<version>` — 特定のバージョン（例：`v2.2.0-beta.7`）

### Docker Compose

カスタムビルドや、キャッシングと使用制限サポートのためにRedisを含める場合：

- [サンプルDockerfile](https://github.com/solana-foundation/kora/blob/main/examples/deploy/Dockerfile)
- [RedisとのDocker Compose](https://github.com/solana-foundation/kora/blob/main/docker-compose.yml)

### Dockerでの実行

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## TypeScript SDK

ベータ版SDKはnpmで利用可能です：

```bash
pnpm add @solana/kora@beta
```

新しいAPIについては、[SDK変更点](/docs/tools/kora/beta/sdk-changes)をご覧ください。

## インストールの確認

```bash
kora --version
```
