description: プロジェクトでSolana Attestationsを始める
---

必要なパッケージをインストールし、開発環境をセットアップして、Solana
Attestationsを始めましょう。

## TypeScript/JavaScript

お好みのパッケージマネージャーを使用して、Solana
Attestationsライブラリをインストールします:

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
