**Solana
Keychainは、複数のキー管理バックエンドにわたってSolanaトランザクションに署名するための統一インターフェースを提供します。**
インメモリキーを使用してローカルで開発し、署名コードを書き直すことなく、本番環境ではエンタープライズキー管理、クラウドHSM、またはマネージドウォレットサービスに切り替えることができます。

### なぜKeychainなのか？

- **単一インターフェース**: `SolanaSigner`
  トレイト1つですべてのバックエンドに対応
- **バックエンドの切り替え**: コードではなく設定によってキー管理プロバイダーを変更
- **ゼロコスト抽象化**: フィーチャーフラグにより必要なものだけを組み込み
- **監査済み**: [Accretion](https://accretion.xyz)
  によるセキュリティレビュー済み

### アーキテクチャ

- **言語**: RustとTypeScript、完全な機能同等性
- **トレイト**: 統一 `SolanaSigner` インターフェース
- **バックエンド**: Memory、Vault、AWS KMS、GCP
  KMS、Privy、Turnkey、Fireblocks、CDP、Crossmint、Dfns、Openfort、Para、Utila
- **互換性**: `@solana/kit` および `@solana/signers` 対応（TypeScript）|
  `solana-sdk` および `solana-sdk-v3` 対応（Rust）

### サポートされているバックエンド

バックエンドは、ローカル開発、セルフホストおよびクラウドキー管理、マネージドまたはMPCウォレットサービスにわたります。カストディモデルの比較と最適な選択については、[バックエンドの選び方](/docs/tools/keychain/choosing-a-backend)をご覧ください。

| Memory          | 開発・テスト                   | ✓    | ✓          |
| HashiCorp Vault | セルフホストキー管理           | ✓    | ✓          |
| AWS KMS         | クラウドネイティブ（AWS）      | ✓    | ✓          |
| GCP KMS         | クラウドネイティブ（GCP）      | ✓    | ✓          |

## クイックスタート

### Rust

Rust クレートをインストールします：

```bash
cargo add solana-keychain
```

基本的な使い方：

```rust
use solana_keychain::{Signer, SolanaSigner};

// Create a signer from any backend
let signer = Signer::from_memory("base58_private_key")?;

// All signers share the same interface
let pubkey = signer.pubkey();
let signature = signer.sign_transaction(&mut tx).await?;
```

- [Rust ガイド](/docs/tools/keychain/getting-started/rust) - 完全なインストールとバックエンド設定
- [Crates.io](https://crates.io/crates/solana-keychain) - Rust クレート

### TypeScript

TypeScript パッケージをインストールします：

```bash
pnpm add @solana/keychain
```

基本的な使い方：

```typescript
import { createKeychainSigner } from "@solana/keychain";
import { signTransactionWithSigners } from "@solana/signers";

// Create any signer via the unified factory
const signer = await createKeychainSigner({
  backend: "vault",
  vaultAddr: "https://vault.example.com:8200",
  vaultToken: "hvs.xxxxx",
  keyName: "my-solana-key",
  publicKey: "base58_public_key"
});

// Sign an already-compiled transaction
const signedTx = await signTransactionWithSigners(
  [signer],
  compiledTransaction
);
```

- [TypeScript ガイド](/docs/tools/keychain/getting-started/typescript) -
  `@solana/keychain` パッケージ
- [npm パッケージ](https://www.npmjs.com/package/@solana/keychain) -
  npm パッケージ
