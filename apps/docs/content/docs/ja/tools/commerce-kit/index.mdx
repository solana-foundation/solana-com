**Commerce
Kitは、Solana上でコマース体験を構築するためのツール集です。**ドロップイン型の決済ボタンからヘッドレスプリミティブまで、Commerce
Kitは暗号通貨決済を受け入れるために必要なすべてを提供します。

## なぜCommerce Kitなのか？

- **ドロップイン型コンポーネント**: ウォレット接続、トークン選択、トランザクション処理を備えた事前構築済みのPaymentButton
- **フレームワーク非依存**: あらゆるJavaScriptフレームワークでヘッドレスプリミティブを使用可能
- **Solana Pay**: QRコード生成を備えた完全なSolana Payサポート
- **型安全**: すべてのパッケージで完全なTypeScriptサポート
- **ウォレット標準**: すべてのWallet
  Standard準拠ウォレット（Phantom、Solflare、Backpackなど）と互換性あり

## クイックスタート

最も早く始める方法は、`PaymentButton`コンポーネントを使用することです：

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## インストール

必要なCommerce Kitパッケージを選択してください：

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
