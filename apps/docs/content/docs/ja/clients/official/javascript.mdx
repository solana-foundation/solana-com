## Solana Kit

`@solana/kit` は、Solana 上で開発するための推奨 TypeScript
SDK です。コンポーザブルなプラグインからクライアントを組み立て、必要に応じて React バインディングを追加し、呼び出すオンチェーンプログラム用に生成されたプログラムクライアントを取り込みます。

| パッケージ                | 説明                                                | GitHub                                                                                 |
| ------------------------- | --------------------------------------------------- | -------------------------------------------------------------------------------------- |
| @solana/kit               | コア SDK：RPC、トランザクション、コーデック、署名者 | [ソース](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | React バインディング：`ClientProvider` とフック     | [ソース](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | RPC 接続 + トランザクション送信                     | [ソース](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | 手数料支払者、ID、署名者プラグイン                  | [ソース](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Wallet Standard 接続 + React フック                 | [ソース](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

## 署名と鍵管理

複数の鍵管理バックエンドにわたる Kit 互換のバックエンド署名に対応：

| パッケージ       | 説明                                                                                                               | GitHub                                                         |
| ---------------- | ------------------------------------------------------------------------------------------------------------------ | -------------------------------------------------------------- |
| @solana/keychain | 統合署名：Memory、Vault、Privy、Turnkey、AWS KMS、Fireblocks、GCP KMS、CDP、Para、Dfns、Crossmint、Openfort、Utila | [ソース](https://github.com/solana-foundation/solana-keychain) |

[Keychain ドキュメント](/docs/tools/keychain)と[バックエンドの選択](/docs/tools/keychain/choosing-a-backend)を参照して、セットアップおよびカストディモデルのガイダンスをご確認ください。

## Solana Web3.js

`@solana/web3.js` はSolanaのレガシーTypeScript
SDKです。新しいアプリには`@solana/kit`を使用してください。詳細は[Kitへの移行](/docs/frontend/web3-compat)をご覧ください。

| パッケージ        | 説明                                                         | GitHub                                                                             |

| @solana/spl-token | Token、Token-2022、および Associated Token programs との連携 | [ソース](https://github.com/solana-program/token-2022/tree/main/clients/js-legacy) |
| @solana/spl-memo  | Memoプログラムとの連携                                       | [ソース](https://github.com/solana-program/memo/tree/main/clients/js-legacy)       |
