## Solana Client

`@solana/client`は、RPC、ウォレット、トランザクション、サブスクリプションを1つのストアにまとめたヘッドレスSolanaランタイムです。

| パッケージ     | 説明                             | GitHub                                                                                 |
| -------------- | -------------------------------- | -------------------------------------------------------------------------------------- |
| @solana/client | ヘッドレスクライアントランタイム | [ソース](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [ガイド: @solana/client概要](/docs/frontend/client)

## Solana React Hooks

`@solana/react-hooks`は、`@solana/client`の上にReactプロバイダーとフックを重ね、同じランタイムとキャッシュを再利用します。

| パッケージ          | 説明                         | GitHub                                                                                      |
| ------------------- | ---------------------------- | ------------------------------------------------------------------------------------------- |
| @solana/react-hooks | UI用Reactプロバイダー+フック | [ソース](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [ガイド: @solana/react-hooks](/docs/frontend/react-hooks)

| パッケージ                     | 説明                                         | GitHub                                                                          |
| ------------------------------ | -------------------------------------------- | ------------------------------------------------------------------------------- |
| @solana/kit                    | コアSDK                                      | [ソース](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | System Programとのインタラクション           | [ソース](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Token Programとのインタラクション            | [ソース](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Token-2022プログラムとのインタラクション     | [ソース](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Memoプログラムとのインタラクション           | [ソース](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Compute Budgetプログラムとのインタラクション | [ソース](https://github.com/solana-program/compute-budget/tree/main/clients/js) |
| @solana/kit-plugins            | Kit用プラグイン                              | [ソース](https://github.com/anza-xyz/kit-plugins)                               |

- [公式Kitドキュメント](https://www.solanakit.com/)

## 署名と鍵管理

複数の鍵管理バックエンドに対応したkit互換のバックエンド署名には:

| パッケージ       | 説明                                                                          | GitHub                                                         |
| ---------------- | ----------------------------------------------------------------------------- | -------------------------------------------------------------- |
| @solana/keychain | 統合署名: Memory、Vault、AWS KMS、Privy、Turnkey、Fireblocks、CDP、Dfns、Para | [ソース](https://github.com/solana-foundation/solana-keychain) |

追加の鍵管理サービスを統合するには、
[署名者の追加ガイド](https://github.com/solana-foundation/solana-keychain/blob/main/docs/ADDING_SIGNERS.md)
をご覧ください。
