## 転送手数料とは？

Token Extensions
Programの*rs`TransferFeeConfig`*ミント拡張機能は、そのミントの各転送に手数料を適用します。

転送時：

- 転送金額は設定された手数料分減額されます
- 保留された手数料は送信先トークンアカウントで追跡されます
- 保留された手数料はトークンアカウントから直接引き出すか、ミントに回収できます
- `withdraw_withheld_authority`は回収された手数料を手数料受取人トークンアカウントに移動できます

<Callout type="info" title="トークンアカウント拡張機能">
  _rs`TransferFeeConfig`_を持つミント用にトークンアカウントが初期化されると、トークンアカウントは_rs`TransferFeeAmount`_で初期化されます。トークンアカウントが[Associated
  Token
  Program](/docs/tokens/basics/create-token-account#how-to-create-an-associated-token-account)を通じて作成される場合、必要なアカウントサイズが計算され、トークンアカウントは必要なサイズとrentを免除するlamportで作成されます。
</Callout>

## 転送手数料ミントの作成方法

転送手数料ミントを作成するには：

1. ミントアカウントのサイズと*rs`TransferFeeConfig`*拡張機能に必要なrentを計算します。
2. *rs`CreateAccount`*でミントアカウントを作成し、*rs`TransferFeeConfig`*を初期化し、*rs`InitializeMint`*でミントを初期化します。
3. ミント用のトークンアカウントを作成します。*rs`TransferFeeAmount`*はトークンアカウントに対して自動的に有効になります。
4. *rs`TransferCheckedWithFee`*を使用してトークンを転送し、期待される手数料を確認します。*rs`TransferChecked`*も機能し、送信先トークンアカウントで設定された手数料を自動的に保留します。
5. *rs`WithdrawWithheldTokensFromAccounts`*を使用して、保留された手数料をトークンアカウントから手数料受取人トークンアカウントに直接移動します。これはミントの`withdraw_withheld_authority`によって署名されます。
6. パーミッションレスの*rs`HarvestWithheldTokensToMint`*を使用して、保留された手数料をトークンアカウントからミントアカウントに移動し、次に*rs`WithdrawWithheldTokensFromMint`*を使用してミントアカウントの保留された手数料を手数料受取人トークンアカウントに引き出します。これはミントの`withdraw_withheld_authority`によって署名されます。
7. *rs`SetTransferFee`*を使用して次の転送手数料設定を更新します。これは2epoch後から有効になります。

## !!steps アカウントサイズの計算

ベースミント用のミントアカウントサイズに*rs`TransferFeeConfig`*拡張を加えたサイズを計算します。これは*rs`CreateAccount`*で使用されるサイズです。

<CodePlaceholder title="例" />

```ts !! title="Example"
const client = await createLocalClient();
const mint = await generateKeyPairSigner();

// !focus(1:14)
const transferFeeConfigExtension = extension("TransferFeeConfig", {
  transferFeeConfigAuthority: client.payer.address,
  withdrawWithheldAuthority: client.payer.address,
  withheldAmount: 0n,
  olderTransferFee: {
    epoch: 0n,
    maximumFee: 10n,
    transferFeeBasisPoints: 150
  },
  newerTransferFee: {
    epoch: 0n,
    maximumFee: 10n,
    transferFeeBasisPoints: 150
  }
});

// !focus(1:1)
const mintSpace = BigInt(getMintSize([transferFeeConfigExtension]));
```
