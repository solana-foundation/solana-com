  Token
  Programを使用して、複数のトークン操作を1つのinstructionにまとめたり、token

一般的な作成、ミント、転送操作に加えて、Token
ProgramはCPIを介してオンチェーンプログラムからも呼び出され、lamportを回収するためのinstructionsも含まれています：

<Cards>
  <Card
    title="CPIを介したToken Programの呼び出し"
    href="/docs/tokens/advanced/cpi"
  >
    オンチェーンプログラムからToken Programを呼び出し、トークンの転送・ミント・バーンを行い、PDAオーソリティで署名し、複数のinstructionsを1つのCPIにまとめます。
  </Card>

<Card
  title="余剰Lamportの引き出し"
  href="/docs/tokens/advanced/withdraw-excess-lamports"
>
  token
  account、ミント、マルチシグアカウントから、トークン残高を変更せずに家賃免除最低額を超えるSOLを回収します。
</Card>

  <Card title="Lamportのアンラップ" href="/docs/tokens/advanced/unwrap-lamports">
    ラップされたSOLをlamportに、全額または一部、token accountを閉じることなく変換します。
  </Card>
</Cards>

<Callout type="info">

_rs`WithdrawExcessLamports`_ と _rs`UnwrapLamports`_ は
[`@solana-program/token`](https://www.npmjs.com/package/@solana-program/token)
クライアントを通じて利用できます。CPIを介したToken
Programの呼び出しはオンチェーンプログラムに適用されます。

</Callout>

## Lamportの回収

_rs`WithdrawExcessLamports`_ は、token
account、ミント、マルチシグアカウントから家賃免除最低額を超えるSOLを回収します。_rs`UnwrapLamports`_
は、ネイティブtoken account内のラップされたSOLをlamportに変換します。
