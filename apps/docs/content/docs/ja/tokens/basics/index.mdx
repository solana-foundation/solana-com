SPL
Token基本セクションでは、SPLトークンとのやり取りに関する基本的なinstructionsを扱っており、オリジナルのToken
ProgramとToken Extensions Program
(Token 2022)の両方に共通する機能に焦点を当てています。

  SPLトークンにメタデータを追加する方法については、[Metaplexドキュメント](https://developers.metaplex.com/token-metadata)を参照してください。
</Callout>
