Surfpoolは、クラスター（メインネット、デブネット）からプログラムとアカウントを自動的にロードすることで、Solana開発を簡素化するローカル開発ツールです。Surfpool
CLIを使用して、プログラムやトランザクションをテストするためのローカルvalidatorを起動しましょう。

詳細については、[Surfpoolのドキュメント](https://docs.surfpool.run/)をご覧ください。

## ローカルvalidatorの起動

ローカルvalidatorを起動するには、次のコマンドを実行します：

```terminal
$ surfpool start
```

デフォルトでは、Surfpoolはトランザクションが依存するプログラムやアカウントをメインネットからローカルvalidatorに自動的にロードします。

## Surfpool Studio

Surfpool
Studioは、ローカルvalidatorとやり取りするためのウェブベースのインターフェースを提供します。以下のことができます：

- トランザクションの詳細を表示
- アカウントデータを検査
- UIフォーセットを使用してSOLやその他のトークンをエアドロップ

Surfpool Studioにアクセスするには：

1. `surfpool start`でローカルvalidatorを起動します
2. ブラウザを開き、[http://127.0.0.1:18488](http://127.0.0.1:18488)に移動します
