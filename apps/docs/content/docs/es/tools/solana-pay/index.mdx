**Solana Pay es un protocolo estándar y un conjunto de implementaciones de
referencia que permiten a los desarrolladores incorporar pagos descentralizados
en sus aplicaciones y servicios.** La blockchain de Solana confirma
transacciones en menos de un segundo y cuesta en promedio $0.0005,
proporcionando a los usuarios una experiencia fluida sin intermediarios.

## ¿Por qué Solana Pay?

- **Protocolo Estándar**: Enfoque unificado para pagos en todas las billeteras y
  aplicaciones de Solana
- **Ultra Rápido**: Las transacciones se confirman en menos de un segundo
- **Ultra Económico**: Costo promedio de transacción de $0.0005
- **Sin Intermediarios**: Pagos directos entre pares sin terceros
- **Soporte Universal de Billeteras**: Compatible con Phantom, Solflare,
  Backpack y más

## Inicio Rápido

La forma más sencilla de comenzar es con una URL de solicitud de transferencia:

```
solana:recipient?amount=0.01&label=Coffee%20Shop&message=Grande%20Latte
```

Esta URL puede ser:

- Codificada como un código QR para escaneo con billeteras móviles
- Usada como un enlace de pago en aplicaciones web
- Incrustada en aplicaciones móviles como enlaces profundos

**[→ Inicio Rápido de Solicitud de Transferencia](/docs/tools/solana-pay/quickstart/transfer-requests)** -

## Instalación

Instala el SDK de JavaScript de Solana Pay:

```bash
# Using pnpm (recommended)
pnpm add @solana/pay@beta @solana/kit

# Using npm
npm install @solana/pay@beta @solana/kit

# Using yarn
yarn add @solana/pay@beta @solana/kit
```

## Casos de Uso

Solana Pay habilita una amplia gama de escenarios de pago:

- **Pagos Minoristas**: Pagos en persona usando códigos QR
- **Comercio Electrónico**: Flujos de pago web con integración de billeteras
- **Aplicaciones Móviles**: Experiencias de pago nativas con enlazado profundo
- **Acuñación de NFT**: Creación dinámica de transacciones para interacciones
  complejas
- **Suscripciones**: Pagos recurrentes y programas de fidelidad
- **Integración DeFi**: Transacciones componibles con intercambios y préstamos

## Wallets Compatibles

Solana Pay funciona con todas las wallets principales de Solana:

- **[Phantom](https://phantom.app/)** - La wallet de Solana más popular
- **[Solflare](https://solflare.com/)** - Wallet de Solana multiplataforma
- **[Backpack](https://backpack.app/)** - Wallet Web3 y tienda de apps
- **[Glow](https://glow.app/)** - Wallet de Solana enfocada en móvil
- **[Ultimate](https://ultimate.app/)** - Wallet enfocada en DeFi
- **[Decaf Wallet](https://decaf.so/)** - Wallet de pagos globales
- **[Espresso Cash](https://www.espressocash.com/)** - Wallet de pagos globales
- **[Tiplink](https://tiplink.io/)** - La wallet más sencilla del mundo
- **[Bitget Wallet](https://bitget.com/)** - Wallet multicadena
