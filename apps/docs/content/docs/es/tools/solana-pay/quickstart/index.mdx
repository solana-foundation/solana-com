Ponte en marcha rápidamente con Solana Pay siguiendo estas guías paso a paso.

## Instalación

Configura el SDK de Solana Pay y prepara tu entorno de desarrollo.

**[→ Guía de Instalación](/docs/tools/solana-pay/quickstart/installation)** -
Instala el SDK y las dependencias

## Tipos de Pago

Elige la integración de pago adecuada para tu caso de uso:

### Solicitudes de Transferencia (Simple)

Crea URL de pago no interactivas para transferencias básicas de SOL y tokens
SPL.

**[→ Solicitudes de Transferencia](/docs/tools/solana-pay/quickstart/transfer-requests)** -
URL de pago simples

**Ideal para:**

- Pagos básicos y donaciones
- Pagos con código QR
- Enlaces de pago en correos electrónicos

### Solicitudes de Transacción (Avanzado)

Construye flujos de pago interactivos con composición de transacciones del lado
del servidor.

**[→ Solicitudes de Transacción](/docs/tools/solana-pay/quickstart/transaction-requests)** -
Flujos de pago interactivos

**Ideal para:**

- Precios dinámicos
- Acuñación de NFT
- Transacciones DeFi complejas
- Lógica de negocio personalizada

## Integración de Código QR

Genera y personaliza códigos QR para pagos con billeteras móviles.

**[→ Integración de Código QR](/docs/tools/solana-pay/quickstart/qr-codes)** -
Genera códigos de pago escaneables

**Ideal para:**

- Sistemas de punto de venta
- Pagos de web a móvil
- Transacciones en persona
