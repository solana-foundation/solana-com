**Commerce Kit es una colección de herramientas para construir experiencias de
comercio en Solana.** Desde botones de pago listos para usar hasta primitivas
sin interfaz, Commerce Kit proporciona todo lo que necesitas para aceptar pagos
en criptomonedas.

## ¿Por qué Commerce Kit?

- **Componentes Listos para Usar**: PaymentButton preconstruido con conexión de
  billetera, selección de tokens y gestión de transacciones
- **Agnóstico de Framework**: Usa primitivas sin interfaz en cualquier framework
  de JavaScript
- **Solana Pay**: Soporte completo de Solana Pay con generación de códigos QR
- **Type-Safe**: Soporte completo de TypeScript en todos los paquetes
- **Wallet Standard**: Compatible con todas las billeteras Wallet Standard
  (Phantom, Solflare, Backpack, etc.)

## Inicio Rápido

La forma más rápida de comenzar es con el componente `PaymentButton`:

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## Instalación

Elige los paquetes de Commerce Kit que necesitas:

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
