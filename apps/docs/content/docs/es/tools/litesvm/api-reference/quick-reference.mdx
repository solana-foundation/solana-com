## Referencia Rápida

| Método                                         | Descripción                                                                                                      |
| ---------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| `LiteSVM::new()`                               | Crea un nuevo entorno de prueba básico de LiteSVM                                                                |
| `LiteSVM::default()`                           | Devuelve el "valor predeterminado" para un tipo                                                                  |
| `airdrop(&Address, lamports)`                  | Financia la cuenta con los lamport especificados                                                                 |
| `get_account(&Address)`                        | Devuelve toda la información asociada con la cuenta de la dirección proporcionada                                |
| `set_account(Address, Account)`                | Establece toda la información asociada con la cuenta de la dirección proporcionada                               |
| `get_balance(&Address)`                        | Obtiene el saldo en lamport de la dirección de cuenta proporcionada                                              |
| `minimum_balance_for_rent_exemption(data_len)` | Devuelve el saldo mínimo requerido para que una cuenta con la longitud de datos especificada esté exenta de rent |
| `add_program(program_id, &bytes)`              | Agrega un programa SBF al entorno de prueba                                                                      |
| `add_program_from_file(program_id, path)`      | Agrega un programa SBF al entorno de prueba desde el archivo especificado                                        |
| `send_transaction(tx)`                         | Envía una transacción firmada                                                                                    |
| `simulate_transaction(tx)`                     | Simula una transacción sin modificar el estado                                                                   |
| `get_transaction(&signature)`                  | Obtiene una transacción del historial                                                                            |
| `latest_blockhash()`                           | Obtiene el blockhash actual                                                                                      |
| `expire_blockhash()`                           | Fuerza la expiración del blockhash actual                                                                        |
| `warp_to_slot(slot)`                           | Mueve el reloj al slot especificado                                                                              |
| `get_sysvar<T>()`                              | Obtiene una sysvar del entorno de prueba                                                                         |
| `set_sysvar<T>(&value)`                        | Establece la sysvar en el entorno de prueba                                                                      |
| `with_feature_accounts()`                      | Incluye cuentas de feature gate en el entorno de prueba                                                          |
| `set_invocation_inspect_callback(callback)`    | Registra un callback de invocación previo/posterior (requiere la función `invocation-inspect-callback`)          |
