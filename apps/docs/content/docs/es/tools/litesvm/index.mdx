---
title: LiteSVM
description:
  Pruebas ultrarrápidas de programas Solana con una VM en proceso. Prueba tus
  programas más rápido sin sobrecarga de red ni procesos de validator.
---

LiteSVM es una biblioteca ligera para probar programas de Solana que funciona
creando una VM de Solana en proceso, optimizada para desarrolladores de
programas. A diferencia de otros enfoques de prueba que lanzan procesos de
validator independientes, LiteSVM integra la VM directamente en tus pruebas, lo
que las hace ejecutar mucho más rápido. Rompiendo aún más con la tradición,
ofrece una API ergonómica con valores predeterminados sensatos y una amplia
capacidad de configuración para quienes la necesiten.
