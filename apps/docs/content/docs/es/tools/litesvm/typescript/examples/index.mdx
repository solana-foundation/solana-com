## Ejecución de Ejemplos

Todos los ejemplos se pueden ejecutar con `tsx`, aunque algunas
direcciones/variables requieren reemplazar los marcadores de posición con
valores reales.

```bash
# Install dependencies
pnpm add @solana/kit @solana/kit-plugin-litesvm @solana/kit-plugin-signer @solana-program/system

# Run an example
npx tsx example.ts
```
