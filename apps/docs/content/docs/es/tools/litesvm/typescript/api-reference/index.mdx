## Referencia rápida

### Configuración del plugin

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### Gestión de cuentas (client.svm)

| Método                                 | Descripción                                    |
| -------------------------------------- | ---------------------------------------------- |
| `setAccount(account)`                  | Establecer el estado de la cuenta directamente |
| `getAccount(address)`                  | Recuperar datos de la cuenta                   |
| `getBalance(address)`                  | Obtener saldo de SOL                           |
| `airdrop(address, lamports)`           | Financiar una cuenta                           |
| `minimumBalanceForRentExemption(size)` | Calcular el mínimo exento de renta             |

### Gestión de programas (client.svm)

| Método                                | Descripción                       |
| ------------------------------------- | --------------------------------- |
| `addProgram(programId, bytes)`        | Cargar programa desde bytes       |
| `addProgramFromFile(programId, path)` | Cargar programa desde archivo .so |

### Transacciones (client.svm)

| Método                    | Descripción                   |
| ------------------------- | ----------------------------- |
| `sendTransaction(tx)`     | Ejecutar una transacción      |
| `simulateTransaction(tx)` | Simular sin cambios de estado |
| `latestBlockhash()`       | Obtener el blockhash actual   |
| `expireBlockhash()`       | Avanzar a un nuevo blockhash  |

### Configuración (client.svm)

| Método                      | Descripción                                   |
| --------------------------- | --------------------------------------------- |
| `withSigverify(bool)`       | Activar/desactivar la verificación de firmas  |
| `withBlockhashCheck(bool)`  | Activar/desactivar la validación de blockhash |
| `withSysvars()`             | Activar cuentas sysvar                        |
| `withBuiltins()`            | Activar programas integrados                  |
| `withPrecompiles()`         | Activar programas precompilados               |
| `withTransactionHistory(n)` | Almacenar las últimas n transacciones         |
| `withComputeBudget(budget)` | Establecer límites de cómputo                 |
| `withLamports(amount)`      | Establecer lamports predeterminados           |
| `withLogBytesLimit(n)`      | Establecer límite de salida de registros      |

### Tiempo y Sysvars (client.svm)

| Método                     | Descripción                           |
| -------------------------- | ------------------------------------- |
| `warpToSlot(slot)`         | Saltar a un slot específico           |
| `getClock()`               | Obtener sysvar de reloj               |
| `getRent()`                | Obtener sysvar de rent                |
| `getEpochSchedule()`       | Obtener programación de epoch         |
| `getLastRestartSlot()`     | Obtener el último slot de reinicio    |
| `setLastRestartSlot(slot)` | Establecer el último slot de reinicio |

### RPC (client.rpc)

| Método                           | Descripción                           |
| -------------------------------- | ------------------------------------- |
| `getAccountInfo(address)`        | Obtener una sola cuenta (solo base64) |
| `getMultipleAccounts(addresses)` | Obtener múltiples cuentas             |
| `getLatestBlockhash()`           | Obtener el blockhash actual           |
