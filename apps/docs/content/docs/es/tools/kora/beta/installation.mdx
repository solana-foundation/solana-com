title: Instalación (Beta)
description: Instala Kora v2.2.0-beta mediante Cargo o Docker.
---

## Cargo

Instala la versión beta directamente desde crates.io:

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

Las imágenes oficiales de Docker multiplataforma (amd64/arm64) se publican en
GitHub Container Registry:

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

Etiquetas disponibles:

- `latest` — Última versión estable
- `beta` — Última versión preliminar
- `v<version>` — Versión específica (ej., `v2.2.0-beta.7`)

### Docker Compose

Para compilaciones personalizadas o para incluir Redis como soporte de caché y
límites de uso:

- [Ejemplo de Dockerfile](https://github.com/solana-foundation/kora/blob/main/examples/deploy/Dockerfile)
- [Docker Compose con Redis](https://github.com/solana-foundation/kora/blob/main/docker-compose.yml)

### Ejecutar con Docker

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## SDK de TypeScript

El SDK beta está disponible en npm:

```bash
pnpm add @solana/kora@beta
```

Consulta [Cambios del SDK](/docs/tools/kora/beta/sdk-changes) para conocer las
nuevas APIs.

## Verificar la Instalación

```bash
kora --version
```
