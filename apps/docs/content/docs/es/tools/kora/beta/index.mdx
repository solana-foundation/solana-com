_Disponible en GitHub:
[`v2.2.0-beta.7`](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.7)_

Kora v2.2.0-beta introduce soporte para paquetes de Jito, nuevos métodos RPC,
imágenes Docker, funciones de seguridad mejoradas y mejoras en el SDK.

> **Esta es una versión preliminar.** Las funcionalidades documentadas aquí
> están disponibles en `v2.2.0-beta.7` y pueden cambiar antes de la versión
> estable. Para documentación estable, consulta la
> [documentación principal de Kora](/docs/tools/kora).

## Aspectos destacados

### Soporte para paquetes de Jito

Envía hasta 5 transacciones de forma atómica a través del motor de bloques de
Jito. Nuevos métodos RPC:

- [`signBundle`](/docs/tools/kora/beta/bundle-methods#signbundle) — Firma un
  paquete sin transmitirlo
- [`signAndSendBundle`](/docs/tools/kora/beta/bundle-methods#signandsendbundle)
  — Firma y envía a Jito
- [`estimateBundleFee`](/docs/tools/kora/beta/bundle-methods#estimatebundlefee)
  — Estima las tarifas totales del paquete

Consulta la [Guía de paquetes de Jito](/docs/tools/kora/beta/jito-guide) para un
tutorial completo.

### Imágenes Docker

Imágenes oficiales multiplataforma (amd64/arm64) publicadas en GitHub Container
Registry. Consulta [Instalación](/docs/tools/kora/beta/installation).

### Complementos de transacciones

Un nuevo sistema de complementos permite que Kora valide la estructura de las
transacciones durante los flujos de firma. El primer complemento integrado,
`gas_swap`, aplica reglas estrictas para transacciones de intercambio de tokens
por SOL sin comisiones. Los complementos se configuran en `kora.toml` y se
ejecutan automáticamente durante `signTransaction`, `signAndSendTransaction`,
`signBundle`, y `signAndSendBundle`.

Consulta
[Configuración — Complementos de transacciones](/docs/tools/kora/beta/configuration#transaction-plugins).

### Nuevas opciones de configuración

- **Complementos de transacciones** para validación de la estructura de
  transacciones (`gas_swap`)
- **reCAPTCHA v3** protección contra bots para endpoints de firma
- **Lighthouse** protección contra drenaje del pagador de tarifas
- **Límites de uso** limitación de transacciones por billetera
- **Configuración de paquetes** para integración con Jito

Consulta [Configuración](/docs/tools/kora/beta/configuration).

### Cambios en el SDK

- Soporte para métodos de paquetes (`signBundle`, `signAndSendBundle`,
  `estimateBundleFee`)

Consulta [Cambios en el SDK](/docs/tools/kora/beta/sdk-changes).

### Métodos RPC adicionales

- [`getVersion`](/docs/tools/kora/beta/bundle-methods#getversion) — Obtiene
  información sobre la versión del servidor

## Instalar

```bash
cargo install kora-cli@2.2.0-beta.7
```

O mediante Docker:

```bash
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```
