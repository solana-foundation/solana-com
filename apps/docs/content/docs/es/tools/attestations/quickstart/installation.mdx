Comienza con Solana Attestations instalando los paquetes necesarios y
configurando tu entorno de desarrollo.

## TypeScript/JavaScript

Instala la biblioteca Solana Attestations usando tu gestor de paquetes
preferido:

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
