title: Primeros pasos
description: Esta guía le ayudará a configurar todo e instalar Surfpool.
---

El CLI de surfpool se puede instalar usando el script de instalación o
compilarse desde el código fuente.

## Script de instalación

Para instalar surfpool, abre una terminal y ejecuta:

```bash
curl -sL https://run.surfpool.run/ | bash
```

## Instalar desde el código fuente

Para compilar el binario de Surfpool desde el código fuente, clona el
repositorio de surfpool.

```bash
git clone https://github.com/solana-foundation/surfpool.git
```

Navega al nuevo directorio.

```bash
cd surfpool
```

Luego, compila el binario. Este comando compilará el binario y lo almacenará en
`~/.cargo/bin/surfpool`.

```bash
cargo surfpool-install
```

Finalmente, asegúrate de que el binario `surfpool` esté disponible en tu PATH.
Este proceso variará según tu sistema operativo.
