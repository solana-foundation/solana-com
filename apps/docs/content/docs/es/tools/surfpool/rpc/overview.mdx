Surfnet es el simulador de red Solana local de Surfpool, diseñado para
proporcionar a los desarrolladores un potente entorno de pruebas que imita el
comportamiento de la blockchain de Solana sin la sobrecarga de un nodo validator
completo.

## Primeros pasos

Inicia una instancia de Surfnet usando la CLI de Surfpool:

```bash
surfpool start
```

Esto iniciará una instancia local de Surfnet en `http://localhost:8899` con
todos los endpoints RPC disponibles.

## Características principales

- **Compatibilidad RPC completa**: Compatible con la API RPC de Solana completa
- **Iteración rápida**: Sin necesidad de esperar confirmaciones de bloque
- **Manipulación de estado**: Cheatcodes para modificar directamente el estado
  de la blockchain
- **Fork de Mainnet**: Haz un fork desde mainnet para probar con cuentas y
  programas reales
- **Soporte para plugins Geyser**: Conecta tus plugins Geyser para streaming de
  datos en tiempo real
