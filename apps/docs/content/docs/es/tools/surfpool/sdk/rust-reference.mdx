El crate `surfpool-sdk` expone una superficie reducida centrada en la estructura
`Surfnet`. Esta página lista todos los elementos públicos; para una descripción
narrativa de uso, consulta las guías que comienzan con
[Descripción general](/docs/tools/surfpool/sdk/overview).

## Exportaciones de la raíz del crate

| Elemento              | Tipo           | Notas                                                                     |
| --------------------- | -------------- | ------------------------------------------------------------------------- |
| `Surfnet`             | struct         | Instancia de Surfnet en ejecución — endpoints RPC/WS, pagador, cheatcodes |
| `SurfnetBuilder`      | struct         | Builder fluido utilizado por `Surfnet::builder()`                         |
| `Cheatcodes<'_>`      | struct         | Auxiliar de cheatcode prestado devuelto por `Surfnet::cheatcodes()`       |
| `SurfnetError`        | enum           | Todas las variantes de error que devuelve el SDK                          |
| `SurfnetResult<T>`    | alias de tipo  | `Result<T, SurfnetError>`                                                 |
| `Keypair`             | re-exportación | `solana_keypair::Keypair`                                                 |
| `Pubkey`              | re-exportación | `solana_pubkey::Pubkey`                                                   |
| `Signer`              | re-exportación | `solana_signer::Signer`                                                   |
| `RpcClient`           | re-exportación | `solana_rpc_client::rpc_client::RpcClient`                                |
| `BlockProductionMode` | re-exportación | `surfpool_types::BlockProductionMode`                                     |
| `SimnetCommand`       | re-exportación | `surfpool_types::SimnetCommand`                                           |
| `SimnetEvent`         | re-exportación | `surfpool_types::SimnetEvent`                                             |
| `SvmFeatureConfig`    | re-exportación | `surfpool_types::SvmFeatureConfig`                                        |

### Ciclo de vida

```rust
impl Surfnet {
    pub fn stop(&mut self) -> SurfnetResult<()>;
}

impl Drop for Surfnet { /* calls stop() if not already stopped */ }
```

`stop` espera hasta cinco segundos para que el runtime libere sus puertos. Un
fallo durante `Drop` es silencioso; un fallo durante un `stop` explícito
devuelve un `SurfnetError::Runtime`.

## SurfnetError

```rust
#[derive(Debug)]
pub enum SurfnetError {
    PortAllocation(String),
    Startup(String),
    Runtime(String),
    Cheatcode(String),
    Aborted(String),
}
```

| Variante         | Causa                                                              |
| ---------------- | ------------------------------------------------------------------ |
| `PortAllocation` | No se pudo vincular a un puerto libre para RPC o WebSocket.        |
| `Startup`        | El tiempo de ejecución no pudo inicializarse.                      |
| `Runtime`        | El hilo de ejecución entró en pánico o finalizó inesperadamente.   |
| `Cheatcode`      | Una llamada JSON-RPC de cheatcode devolvió un error.               |
| `Aborted`        | El tiempo de ejecución fue detenido o cancelado durante el inicio. |

El crate implementa `std::error::Error` y `Display` en `SurfnetError`, por lo
que funciona con `?` y `anyhow` de forma nativa.
