## assert_eq

`assert_eq` verifica que dos valores sean iguales.

### Entradas

| Nombre  | Tipo | Descripción                  |
| ------- | ---- | ---------------------------- |
| `left`  | any  | Un valor a comparar          |
| `right` | any  | El valor con el que comparar |

### Salida

| Nombre  | Tipo                  | Descripción                 |
| ------- | --------------------- | --------------------------- |
| `value` | addon(std::assertion) | El resultado de la aserción |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```

## assert_gt

`assert_gt` verifica que el valor izquierdo sea mayor que el valor derecho.

### Entradas

| Nombre  | Tipo             | Descripción                              |
| ------- | ---------------- | ---------------------------------------- |
| `left`  | integer \| float | Un entero o flotante a comparar          |
| `right` | integer \| float | Un entero o flotante con el que comparar |

### Salida

| Nombre  | Tipo                  | Descripción                 |
| ------- | --------------------- | --------------------------- |
| `value` | addon(std::assertion) | El resultado de la aserción |

```hcl
output "assertion" {
    value = std::assert_gt(action.example.result, 1)
}
```
