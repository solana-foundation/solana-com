## encode_hex

`encode_hex` codifica un búfer o cadena como una cadena hexadecimal con prefijo
0x.

### Entradas

| Nombre  | Tipo                  | Descripción                                                                                                                                       |
| ------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | El búfer o cadena a codificar. Las cadenas que comienzan con '0x' se decodifican como hex; de lo contrario, se utilizan bytes UTF-8 sin procesar. |

### Salida

| Nombre  | Tipo   | Descripción                                                |
| ------- | ------ | ---------------------------------------------------------- |
| `value` | string | La entrada en su representación hexadecimal con prefijo 0x |

```hcl
output "encoded_hex" {
    value = encode_hex("hello, world")
}
// > encoded_hex: 0x68656c6c6f2c20776f726c64
```

## decode_hex

`decode_hex` decodifica una cadena hexadecimal y devuelve el resultado como un
búfer.

### Entradas

| Nombre       | Tipo   | Descripción                                        |
| ------------ | ------ | -------------------------------------------------- |
| `hex_string` | string | La cadena hex a decodificar (con o sin prefijo 0x) |

### Salida

| Nombre  | Tipo   | Descripción                              |
| ------- | ------ | ---------------------------------------- |
| `value` | buffer | La cadena hex decodificada como un búfer |

```hcl
output "decoded_hex" {
    value = decode_hex("0x68656c6c6f2c20776f726c64")
}
// > decoded_hex: 0x68656c6c6f2c20776f726c64
```
