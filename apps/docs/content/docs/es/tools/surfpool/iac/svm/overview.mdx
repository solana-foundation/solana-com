---
title: Descripción general del complemento SVM
description: Complemento para Solana y blockchains compatibles con SVM (beta)
---

El complemento `txtx` de SVM permite crear Runbooks que interactúan con Solana y
blockchains compatibles con SVM. El complemento proporciona funciones
utilitarias que permiten desplegar programas Anchor y codificar llamadas de
instrucciones según los IDL de los programas. Las acciones pueden usarse para
crear transacciones válidas de transferencia, llamadas a programas y despliegue
de programas, que pueden firmarse mediante una frase mnemónica, clave secreta o
a través del firmante de tu navegador.

## Inicio rápido

Para usar el complemento SVM, decláraloo en tu Runbook:

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## Características

- **Despliegue de programas**: Despliega programas Anchor y programas nativos de
  Solana
- **Construcción de instrucciones**: Codifica llamadas de instrucciones usando
  IDL de programas
- **Firma de transacciones**: Compatibilidad con frases mnemónicas, claves
  secretas y carteras web
- **Gestión de cuentas**: Consulta y administra cuentas de Solana
