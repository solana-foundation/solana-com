## jq

La función `jq` permite segmentar, filtrar y mapear datos JSON. Consulta la
[documentación de jq](https://jqlang.github.io/jq/manual/) para más detalles.

### Entradas

| Nombre         | Tipo             | Descripción                                                                               |
| -------------- | ---------------- | ----------------------------------------------------------------------------------------- |
| `decoded_json` | string \| object | Un objeto JSON                                                                            |
| `query`        | string           | Una consulta JSON. Consulta la [documentación de jq](https://jqlang.github.io/jq/manual/) |

### Salida

| Nombre  | Tipo          | Descripción                      |
| ------- | ------------- | -------------------------------- |
| `value` | array[string] | El resultado de la consulta `jq` |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
