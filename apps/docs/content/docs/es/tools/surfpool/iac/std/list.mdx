## index

`index` obtiene la entrada de una lista en el índice especificado.

### Entradas

| Nombre  | Tipo                                             | Descripción                               |
| ------- | ------------------------------------------------ | ----------------------------------------- |
| `list`  | array[string] \| array[integer] \| array[buffer] | La lista de la que se obtiene una entrada |
| `index` | integer                                          | El índice de la entrada a obtener         |

### Salida

| Nombre  | Tipo   | Descripción                                      |
| ------- | ------ | ------------------------------------------------ |
| `value` | string | La entrada de la lista en el índice especificado |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
