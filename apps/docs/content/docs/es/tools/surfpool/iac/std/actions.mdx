Estas acciones están disponibles en todos los Runbooks como parte de la
biblioteca estándar.

## send_http_request

La acción `std::send_http_request` realiza una solicitud HTTP a la URL
especificada.

### Entradas

| Nombre       | Requerido | Tipo    | Descripción                                               |
| ------------ | --------- | ------- | --------------------------------------------------------- |
| `url`        | requerido | string  | La URL a la que se enviará la solicitud                   |
| `method`     | opcional  | string  | El método HTTP (GET, POST, PUT, DELETE). Por defecto: GET |
| `headers`    | opcional  | map     | Encabezados HTTP a incluir en la solicitud                |
| `body`       | opcional  | string  | El cuerpo de la solicitud (para solicitudes POST/PUT)     |
| `timeout_ms` | opcional  | integer | Tiempo de espera de la solicitud en milisegundos          |

### Salidas

| Nombre        | Tipo    | Descripción                     |
| ------------- | ------- | ------------------------------- |
| `status_code` | integer | El código de estado HTTP        |
| `body`        | string  | El cuerpo de la respuesta       |
| `headers`     | map     | Los encabezados de la respuesta |

```hcl
action "api_call" "std::send_http_request" {
    description = "Fetch data from API"
    url = "https://api.example.com/data"
    method = "GET"
    headers = {
        "Authorization" = "Bearer ${variable.token}"
    }
}

output "response" {
    value = action.api_call.body
}

output "status" {
    value = action.api_call.status_code
}
```

### Ejemplo de Solicitud POST

```hcl
action "create_resource" "std::send_http_request" {
    description = "Create a new resource"
    url = "https://api.example.com/resources"
    method = "POST"
    headers = {
        "Content-Type" = "application/json"
    }
    body = {
        "name" = variable.name,
        "value" = variable.value
    }
}
```
