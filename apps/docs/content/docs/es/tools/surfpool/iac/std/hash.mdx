## ripemd160

`ripemd160` calcula el hash Ripemd160 de un valor.

### Entradas

| Nombre  | Tipo                    | Descripción                          |
| ------- | ----------------------- | ------------------------------------ |
| `value` | buffer \| array[buffer] | El valor codificado en hex a hashear |

### Salida

| Nombre  | Tipo   | Descripción           |
| ------- | ------ | --------------------- |
| `value` | string | El resultado hasheado |

```hcl
output "hashed_data" {
    value = ripemd160(encode_hex("hello, world"))
}
// > hashed_data: 0XA3201F82FCA034E46D10CD7B27E174976E241DA2
```

| ------- | ------ | --------------------- |
| `value` | string | El resultado hasheado |

```hcl

}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```
