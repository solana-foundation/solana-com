  Funciones de biblioteca estándar para codificación y decodificación en Base64
---

## encode_base64

`encode_base64` codifica un buffer o cadena de texto como una cadena base64.

### Entradas

| Nombre  | Tipo                  | Descripción                                                                                                                                            |
| ------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `value` | buffer, string, addon | El buffer o cadena a codificar. Las cadenas que comienzan con '0x' se decodifican como hexadecimal; de lo contrario, se usan bytes UTF-8 sin procesar. |

### Salida

| Nombre  | Tipo   | Descripción                                   |
| ------- | ------ | --------------------------------------------- |
| `value` | string | La entrada, codificada como una cadena base64 |

```hcl
output "encoded" {
    value = encode_base64("0x48656c6c6f20776f726c6421")
}
// > encoded: SGVsbG8gd29ybGQh
```

## decode_base64

`decode_base64` decodifica una cadena codificada en base64 y devuelve el
resultado como un buffer.

### Entradas

| Nombre          | Tipo   | Descripción                    |
| --------------- | ------ | ------------------------------ |
| `base64_string` | string | La cadena base64 a decodificar |

### Salida

| Nombre  | Tipo   | Descripción                                   |
| ------- | ------ | --------------------------------------------- |
| `value` | buffer | La cadena base64 decodificada, como un buffer |

```hcl
output "decoded" {
    value = decode_base64("SGVsbG8gd29ybGQh")
}
// > decoded: 0x48656c6c6f20776f726c6421
```
