## encode_base58

`encode_base58` codifica un buffer o cadena de texto como una cadena base58.

### Entradas

| Name    | Type                  | Description                                                                                                                                                         |
| ------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | El buffer o cadena de texto a codificar. Las cadenas que comienzan con '0x' se decodifican como hexadecimal; de lo contrario, se utilizan bytes UTF-8 sin procesar. |

### Salida

| Name    | Type   | Description                                   |
| ------- | ------ | --------------------------------------------- |
| `value` | string | La entrada, codificada como una cadena base58 |

```hcl
output "encoded" {
    value = encode_base58("0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8")
}
// > encoded: CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF
```

## decode_base58

`decode_base58` decodifica una cadena codificada en base58 y devuelve el
resultado como un buffer.

### Entradas

| Name            | Type   | Description                    |
| --------------- | ------ | ------------------------------ |
| `base58_string` | string | La cadena base58 a decodificar |

### Salida

| Name    | Type   | Description                                   |
| ------- | ------ | --------------------------------------------- |
| `value` | buffer | La cadena base58 decodificada, como un buffer |

```hcl
output "decoded" {
    value = decode_base58("CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF")
}
// > decoded: 0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8
```
