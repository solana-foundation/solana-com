La biblioteca estándar proporciona funciones y acciones principales que están
disponibles para todos los Runbooks, independientemente de los complementos de
blockchain habilitados.

## Funciones

La biblioteca estándar incluye funciones de utilidad para:

- **Operadores**: Operaciones matemáticas y comparaciones
- **Hash**: Funciones de hash (SHA256, Keccak256, etc.)
- **Hex**: Codificación y decodificación hexadecimal
- **Crypto**: Operaciones criptográficas
- **JSON**: Codificación y decodificación JSON
- **List**: Funciones de manipulación de listas
- **Assertions**: Afirmaciones de igualdad y condiciones
- **Base64**: Codificación y decodificación en Base64

[Ver todas las funciones](/docs/tools/surfpool/iac/std/functions)

## Acciones

La biblioteca estándar incluye acciones para:

- **HTTP**: Realización de solicitudes HTTP
- **Operaciones de Archivos**: Lectura y escritura de archivos

[Ver todas las acciones](/docs/tools/surfpool/iac/std/actions)
