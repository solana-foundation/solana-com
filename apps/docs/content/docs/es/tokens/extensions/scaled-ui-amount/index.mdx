## ¿Qué es la extensión Cantidad escalada de la interfaz de usuario?

La extensión de cantidad escalada de la interfaz de usuario permite a los
emisores aplicar un multiplicador actualizable a la cantidad mostrada de un
token. Esto es útil para escenarios donde el token se utiliza para representar
un activo del mundo real, como una acción o un dividendo.

Por ejemplo, un desdoblamiento de acciones duplica el número de acciones para
todos los titulares. Sin esta extensión, actualmente resulta poco práctico
emitir nuevos tokens a todos los titulares durante un evento de desdoblamiento
de manera eficiente y escalable.

Sin embargo, con el modelo de extensión Token-2022, tenemos la posibilidad de
cambiar cómo se representan las cantidades de UI de los tokens. Usando la
extensión ScaledUiAmount y la instrucción `amount_to_ui_amount`, puedes
establecer un multiplicador de UI en tu token y obtener su cantidad de UI en
cualquier momento.
[código fuente](https://github.com/solana-program/token-2022/tree/main/program/src/extension/scaled_ui_amount).

Esta función también puede utilizarse para dividendos o distribución de
rendimientos.

Nota: Nunca se crean nuevos tokens, la cantidad mostrada en la interfaz devuelve
la cantidad bruta de tokens multiplicada por el multiplicador actual. La función
es completamente cosmética.

<Callout type="info">
  Algunas extensiones son incompatibles entre sí y no puedes habilitarlas
  simultáneamente en el mismo mint de token o token account. No debes habilitar
  tanto la extensión `ScaledUiAmount` como la extensión `InterestBearingMint` en
  el mismo mint de token o token account.
</Callout>
