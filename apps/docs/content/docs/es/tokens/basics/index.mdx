La sección Conceptos Básicos de Tokens SPL cubre las instrucciones básicas para
interactuar con Tokens SPL, enfocándose en la funcionalidad compartida tanto por
el Token Program original como por el Token Extensions Program (Token 2022).

<Callout type="info">
  Consulta la [documentación de
  Metaplex](https://developers.metaplex.com/token-metadata) para obtener más
  información sobre cómo añadir metadatos a los Tokens SPL.
</Callout>
