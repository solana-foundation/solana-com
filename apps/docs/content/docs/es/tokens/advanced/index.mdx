Más allá de las operaciones comunes de creación, acuñación y transferencia, el
Token Program también es invocado desde programas on-chain mediante CPI e
incluye instrucciones para recuperar lamports:

<Cards>
  <Card
    title="Llamar al Token Program mediante CPI"
    href="/docs/tokens/advanced/cpi"
  >
    Invoca el Token Program desde un programa on-chain: transfiere, acuña y quema
    tokens, firma con una autoridad PDA y agrupa instrucciones en una sola CPI.
  </Card>

<Card
  title="Retirar Lamports Excedentes"
  href="/docs/tokens/advanced/withdraw-excess-lamports"
>
  Recupera SOL por encima del mínimo exento de renta desde token accounts, mints
  y cuentas multisig sin modificar los saldos de tokens.
</Card>

  <Card title="Desenvolver Lamports" href="/docs/tokens/advanced/unwrap-lamports">
    Convierte SOL envuelto de vuelta en lamports, total o parcialmente, sin cerrar
    el token account.
  </Card>
</Cards>

<Callout type="info">

_rs`WithdrawExcessLamports`_ y _rs`UnwrapLamports`_ están disponibles a través
del cliente
[`@solana-program/token`](https://www.npmjs.com/package/@solana-program/token).
Llamar al Token Program mediante CPI aplica a programas on-chain.

</Callout>

## Recuperación de lamports

_rs`WithdrawExcessLamports`_ recupera SOL por encima del mínimo exento de renta
desde token accounts, mints y cuentas multisig. _rs`UnwrapLamports`_ convierte
SOL envuelto en un native token account de vuelta en lamports.
