Esta página es una referencia única para todas las constantes del runtime
mencionadas en las páginas de conceptos principales. Cada constante enlaza a su
definición fuente en el cliente validador
[agave](https://github.com/anza-xyz/agave/tree/v3.1.8) o
[solana-sdk](https://github.com/anza-xyz/solana-sdk).
