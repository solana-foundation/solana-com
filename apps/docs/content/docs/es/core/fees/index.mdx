Cada [transacción](/docs/core/transactions) de Solana requiere una tarifa pagada
en SOL. La tarifa tiene dos componentes: una **tarifa base** y una **tarifa de
priorización** opcional. La tarifa base compensa a los validadores por el
trabajo criptográfico de verificar firmas. La tarifa de priorización aumenta la
probabilidad de que el líder actual programe tu transacción antes que las
competidoras.

## Datos clave

- **Tarifa base**: por firma, dividida 50% quemada / 50% al validator.
- **Tarifa de priorización**:
  `ceil(compute_unit_price * compute_unit_limit / 1,000,000)` lamports. 100% al
  validator.
