El desarrollo frontend en Solana implica trabajar con programas, wallets y
frameworks populares de JavaScript como React. Interactuar con estos componentes
requiere manejar la conexión, la creación de transacciones y la lectura de
cuentas de Solana.

[Solana Kit](https://github.com/anza-xyz/kit) (`@solana/kit`) es el SDK estándar
de JavaScript para este trabajo. Ensamblas un cliente a partir de pequeños
plugins componibles, añades enlaces con React cuando los necesitas, y agregas
los clientes de programa generados para los programas en cadena que utilizas.

## Bibliotecas principales

<Cards>
  <Card title="@solana/kit" href="https://github.com/anza-xyz/kit">
    - SDK principal: RPC, codecs, firmantes y transacciones - Proporciona
    `createClient`, el host de plugins que todas las demás bibliotecas extienden
  </Card>
  <Card title="Kit plugins" href="https://github.com/anza-xyz/kit-plugins">
    - Construye un cliente con `.use(...)`: conexión RPC, billetera, firmante y
    planificación de transacciones - Publicados como paquetes
    `@solana/kit-plugin-*`
  </Card>
  <Card
    title="@solana/react"
    href="https://github.com/anza-xyz/kit/tree/main/packages/react"
  >
    - Enlaces de React para un cliente Kit - `ClientProvider` más hooks para
    lecturas, suscripciones y estado de la billetera
  </Card>
  <Card title="@solana-program/*" href="https://github.com/solana-program">
    - Clientes generados para programas en cadena (System, Token y más) -
    Proporcionan constructores de instrucciones y plugins de cliente como
    `tokenProgram()`
  </Card>
</Cards>

<Callout
  type="warn"
  title="web3.js v1 y wallet-adapter son versiones heredadas"
>
  Muchos proyectos de Solana todavía dependen del antiguo `@solana/web3.js` (v1)
  y `@solana/wallet-adapter-*`. Para nuevos desarrollos, prefiere `@solana/kit`
  con los plugins de kit que se muestran a continuación; la detección de Wallet
  Standard (a través de `@solana/kit-plugin-wallet`) reemplaza a wallet-adapter
  para las billeteras modernas.
</Callout>
