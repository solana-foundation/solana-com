## Crates de programas

Usa `solana-program` para construir programas on-chain.

- [Ejemplo: Construir un programa Solana](/docs/programs/rust)

| Crate          | Descripción                  | Documentación                         | GitHub                                                                      |
| -------------- | ---------------------------- | ------------------------------------- | --------------------------------------------------------------------------- |
| solana-program | Construir programas on-chain | [Ver](https://docs.rs/solana-program) | [Código fuente](https://github.com/anza-xyz/solana-sdk/tree/master/program) |

Pinocchio es una biblioteca sin dependencias para crear programas de Solana en
Rust y puede usarse como reemplazo de `solana-program` para escribir programas
on-chain.

| Crate     | Descripción                         | Documentación                    | GitHub                                                 |
| --------- | ----------------------------------- | -------------------------------- | ------------------------------------------------------ |
| pinocchio | Programas on-chain sin dependencias | [Ver](https://docs.rs/pinocchio) | [Código fuente](https://github.com/anza-xyz/pinocchio) |

<Callout type="warn">
  Pinocchio todavía está en desarrollo y probablemente tendrá cambios
  importantes en un futuro próximo.
</Callout>

Pinocchio incluye crates específicos para interactuar con programas Solana:

| Crate                              | Descripción                              | Documentación                                              | GitHub                                                                                             |
| ---------------------------------- | ---------------------------------------- | ---------------------------------------------------------- | -------------------------------------------------------------------------------------------------- |
| pinocchio-system                   | Interactuar con System Program           | [Ver](https://docs.rs/pinocchio-system/)                   | [Código fuente](https://github.com/anza-xyz/pinocchio/tree/main/programs/system)                   |
| pinocchio-token                    | Interactuar con Token Program            | [Ver](https://docs.rs/pinocchio-token/)                    | [Código fuente](https://github.com/anza-xyz/pinocchio/tree/main/programs/token)                    |
| pinocchio-token-2022               | Interactuar con Token-2022 Program       | [Ver](https://docs.rs/pinocchio-token-2022/)               | [Código fuente](https://github.com/anza-xyz/pinocchio/tree/main/programs/token-2022)               |
| pinocchio-associated-token-account | Interactuar con Associated Token Program | [Ver](https://docs.rs/pinocchio-associated-token-account/) | [Código fuente](https://github.com/anza-xyz/pinocchio/tree/main/programs/associated-token-account) |
| pinocchio-memo                     | Interactuar con Memo Program             | [Ver](https://docs.rs/pinocchio-memo/)                     | [Código fuente](https://github.com/anza-xyz/pinocchio/tree/main/programs/memo)                     |
