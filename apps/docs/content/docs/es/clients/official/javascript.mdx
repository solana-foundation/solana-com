## Solana Client

`@solana/client` es el runtime de Solana sin interfaz que agrupa RPC, wallets,
transacciones y suscripciones en un único store.

| Paquete        | Descripción                     | GitHub                                                                                        |
| -------------- | ------------------------------- | --------------------------------------------------------------------------------------------- |
| @solana/client | Runtime de cliente sin interfaz | [Código fuente](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [Guía: descripción general de @solana/client](/docs/frontend/client)

## Solana React Hooks

`@solana/react-hooks` añade un proveedor de React y hooks sobre
`@solana/client`, reutilizando el mismo runtime y caché.

| Paquete             | Descripción                         | GitHub                                                                                             |
| ------------------- | ----------------------------------- | -------------------------------------------------------------------------------------------------- |
| @solana/react-hooks | Proveedor de React + hooks para UIs | [Código fuente](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [Guía: @solana/react-hooks](/docs/frontend/react-hooks)

## Solana Kit

`@solana/kit` es el SDK de TypeScript recomendado para desarrollar en Solana.

| Paquete                        | Descripción                                | GitHub                                                                          |
| ------------------------------ | ------------------------------------------ | ------------------------------------------------------------------------------- |
| @solana/kit                    | SDK principal                              | [Fuente](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | Interactuar con el System Program          | [Fuente](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Interactuar con el Token Program           | [Fuente](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Interactuar con el programa Token-2022     | [Fuente](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Interactuar con el programa Memo           | [Fuente](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Interactuar con el programa Compute Budget | [Fuente](https://github.com/solana-program/compute-budget/tree/main/clients/js) |
| @solana/kit-plugins            | Plugins para Kit                           | [Fuente](https://github.com/anza-xyz/kit-plugins)                               |

- [Documentación oficial de Kit](https://www.solanakit.com/)
- [Tutorial: Next.js + React Hooks](/docs/frontend/nextjs-solana)

## Firma y gestión de claves

Para firma compatible con kit en múltiples backends de gestión de claves:

| Paquete          | Descripción                                                                          | GitHub                                                         |
| ---------------- | ------------------------------------------------------------------------------------ | -------------------------------------------------------------- |
| @solana/keychain | Firma unificada: Memory, Vault, AWS KMS, Privy, Turnkey, Fireblocks, CDP, Dfns, Para | [Fuente](https://github.com/solana-foundation/solana-keychain) |

Consulta la
[guía para agregar firmantes](https://github.com/solana-foundation/solana-keychain/blob/main/docs/ADDING_SIGNERS.md)
para integrar servicios adicionales de gestión de claves.
