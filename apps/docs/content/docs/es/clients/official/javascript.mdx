## Solana Kit

`@solana/kit` es el SDK de TypeScript recomendado para desarrollar en Solana.
Puedes ensamblar un cliente a partir de plugins componibles, agregar enlaces de
React cuando los necesites e incorporar clientes de programa generados para los
programas en cadena que invoques.

| Paquete                   | Descripción                                          | GitHub                                                                                 |
| ------------------------- | ---------------------------------------------------- | -------------------------------------------------------------------------------------- |
| @solana/kit               | SDK principal: RPC, transacciones, codecs, firmantes | [Fuente](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | Enlaces de React: `ClientProvider` y hooks           | [Fuente](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | Conexión RPC + envío de transacciones                | [Fuente](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | Plugins de pagador de tarifas, identidad y firmante  | [Fuente](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Conexión Wallet Standard + hooks de React            | [Fuente](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

## Firma y gestión de claves

Para firma en backend compatible con kit a través de múltiples backends de
gestión de claves:

| Paquete          | Descripción                                                                                                               | GitHub                                                         |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------- |
| @solana/keychain | Firma unificada: Memory, Vault, Privy, Turnkey, AWS KMS, Fireblocks, GCP KMS, CDP, Para, Dfns, Crossmint, Openfort, Utila | [Fuente](https://github.com/solana-foundation/solana-keychain) |

Consulta la [documentación de Keychain](/docs/tools/keychain) y
[Cómo elegir un backend](/docs/tools/keychain/choosing-a-backend) para obtener
orientación sobre la configuración y el modelo de custodia.

## Solana Web3.js

`@solana/web3.js` es el SDK de TypeScript heredado para Solana. Las nuevas
aplicaciones deben usar `@solana/kit`; consulta
[Migración a Kit](/docs/frontend/web3-compat).

| Package           | Description                                                       | GitHub                                                                             |
| ----------------- | ----------------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| @solana/web3.js   | SDK principal                                                     | [Source](https://github.com/solana-foundation/solana-web3.js)                      |
| @solana/spl-token | Interactúa con los programas Token, Token-2022 y Associated Token | [Source](https://github.com/solana-program/token-2022/tree/main/clients/js-legacy) |
| @solana/spl-memo  | Interactúa con el programa Memo                                   | [Source](https://github.com/solana-program/memo/tree/main/clients/js-legacy)       |
