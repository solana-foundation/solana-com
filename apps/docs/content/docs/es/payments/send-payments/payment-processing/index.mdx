---
title: Procesamiento de pagos
description:
  Procesa pagos a escala con transacciones sin gas y abstracción de comisiones
---

Procesar pagos a escala (nóminas, pagos a comerciantes, recompensas de
fidelización) requiere más que enviar transferencias individuales.

Para procesar muchos pagos de manera eficiente:

1. **Agrupa instrucciones** en transacciones únicas para reducir comisiones y
   latencia
2. **Agrupa transacciones** para paralelizar el rendimiento en toda la red
3. **Abstracción de comisiones** para que los usuarios nunca necesiten tener SOL
