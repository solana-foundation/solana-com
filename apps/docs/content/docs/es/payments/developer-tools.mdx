## Herramientas de desarrollo de IA

Comienza con nuestra guía sobre
[cómo empezar con herramientas de IA en Solana](/developers/guides/getstarted/intro-to-ai).

### Solana MCP

[Solana MCP](https://mcp.solana.com/) (Machine Conversation Protocol) integra el
conocimiento de Solana directamente en herramientas de codificación asistidas
por IA como Claude, Cursor, y Windsurf. Está entrenado en documentación de
Solana, el framework Anchor, ejemplos de programas, y preguntas y respuestas de
Stack Exchange - útil para construir integraciones de pago más rápido.

### Frameworks de agentes de IA

Los agentes de IA pueden ejecutar transacciones de pago de forma autónoma en
Solana. Estos frameworks permiten construir agentes que procesan pagos,
monitorean transacciones, y automatizan flujos de trabajo de pago:

| Framework        | Descripción                                                                                         | Enlaces                                                                                   |
| ---------------- | --------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | Herramientas para construir agentes capaces de procesar pagos, compatible con LangChain y Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | Una colección de habilidades preconstruidas para crear agentes en Solana                            | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | Framework de agentes multiplataforma con plugin de Solana para bots de X, Telegram, Discord         | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Toolkit multicadena con acciones de pago de Solana                                                  | [GitHub](https://github.com/goat-sdk/goat)                                                |

### Desarrollo de programas con IA

de extremo a extremo.
