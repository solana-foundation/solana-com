<Callout type="info" title="Summary">
  Programs interact with the runtime through ~30 syscalls. Each deducts compute
  units from the transaction's shared budget. Costs range from 10 CUs for memory
  operations to 25,000 CUs for secp256k1 recovery.
</Callout>

## Syscalls

Programs interact with the runtime through syscalls registered in
[`create_program_runtime_environment_v1()`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/lib.rs#L282).
Each syscall deducts compute units from the transaction's compute budget.

### Available syscalls

The complete set of syscalls available to sBPF programs, as registered in
[`create_program_runtime_environment_v1()`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/lib.rs#L282):

| Category | Syscalls |
|---|---|
| **Control flow** | `abort`, `sol_panic_` |
| **Logging** | `sol_log_`, `sol_log_64_`, `sol_log_pubkey`, `sol_log_compute_units_`, `sol_log_data` |
| **PDA** | `sol_create_program_address`, `sol_try_find_program_address` |
| **Hashing** | `sol_sha256`, `sol_keccak256`, `sol_blake3`, `sol_poseidon` |
| **Cryptography** | `sol_secp256k1_recover`, `sol_curve_validate_point`, `sol_curve_group_op`, `sol_curve_multiscalar_mul`, `sol_alt_bn128_group_op`, `sol_alt_bn128_compression`, `sol_big_mod_exp` |
| **CPI** | `sol_invoke_signed_rust`, `sol_invoke_signed_c` |
| **Sysvars** | `sol_get_clock_sysvar`, `sol_get_epoch_schedule_sysvar`, `sol_get_rent_sysvar`, `sol_get_epoch_rewards_sysvar`, `sol_get_last_restart_slot_sysvar`, `sol_get_sysvar` |
| **Memory** | `sol_memcpy_`, `sol_memmove_`, `sol_memset_`, `sol_memcmp_` |
| **Program data** | `sol_set_return_data`, `sol_get_return_data`, `sol_get_processed_sibling_instruction`, `sol_get_stack_height` |
| **Compute** | `sol_remaining_compute_units` |
| **Epoch stake** | `sol_get_epoch_stake` |
| **Memory allocation** | `sol_alloc_free_` (legacy, deployment disabled) |
