<Callout type="info" title="Summary">
  Programs interact with the runtime through ~40 syscalls covering logging,
  hashing, cryptography, CPI, sysvars, memory operations, and more. Each deducts
  compute units from the transaction's shared budget. Costs range from 10 CUs for
  memory operations to 25,000 CUs for secp256k1 recovery.
</Callout>

## Syscalls

Programs interact with the runtime through syscalls registered in
[`create_program_runtime_environment_v1()`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/lib.rs#L282).
Each syscall deducts compute units from the transaction's compute budget.

### Available syscalls

The complete set of syscalls available to sBPF programs, as registered in
[`create_program_runtime_environment_v1()`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/lib.rs#L282).
Feature-gated syscalls are only available when the corresponding feature is
active on the cluster.

#### Control flow

| Syscall | Description | Source |
|---|---|---|
| `abort` | Halts the program immediately. Inserted by LLVM for untenable conditions; not intended for explicit use. | [`SyscallAbort`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/lib.rs#L691) |
| `sol_panic_` | Halts the program with a file name, line, and column number for diagnostics. | [`SyscallPanic`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/lib.rs#L710) |

#### Logging

| Syscall | Description | Source |
|---|---|---|
| `sol_log_` | Logs a UTF-8 string message to the program log. Cost: max(syscall_base_cost, len) CUs. | [`SyscallLog`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/logging.rs#L5) |
| `sol_log_64_` | Logs five `u64` values as hex to the program log. Cost: 100 CUs. | [`SyscallLogU64`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/logging.rs#L37) |
| `sol_log_pubkey` | Logs a `Pubkey` as a base58 string to the program log. Cost: 100 CUs. | [`SyscallLogPubkey`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/logging.rs#L84) |
| `sol_log_compute_units_` | Logs the remaining compute units to the program log. Cost: 100 CUs. | [`SyscallLogBpfComputeUnits`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/logging.rs#L60) |
| `sol_log_data` | Logs arbitrary byte slices as base64-encoded data. Costs: base cost per field + total byte length. | [`SyscallLogData`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/logging.rs#L109) |

#### PDA

| Syscall | Description | Source |
|---|---|---|
| `sol_create_program_address` | Derives a program address from seeds and a program ID. Returns 1 if the derived address is on the Ed25519 curve (invalid PDA). Cost: 1,500 CUs. | [`SyscallCreateProgramAddress`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/lib.rs#L798) |
| `sol_try_find_program_address` | Finds a valid PDA by iterating bump seeds from 255 down to 1. Returns the address and bump. Cost: 1,500 CUs per iteration. | [`SyscallTryFindProgramAddress`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/lib.rs#L836) |

| `sol_keccak256` | Computes a Keccak-256 hash over one or more byte slices. Cost: 85 + 1/byte CUs. | [`SyscallHash<Keccak256Hasher>`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/lib.rs#L1912) |

|---|---|---|

| `sol_invoke_signed_c` | Invokes a cross-program instruction from a C program. Uses the C `SolInstruction` ABI for serialization. Cost: 1,000 CUs (946 with SIMD-0339) + serialization costs. | [`SyscallInvokeSignedC`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/cpi.rs#L78) |

| Syscall | Description | Source |
|---|---|---|

| `sol_memcmp_` | Compares `n` bytes of two memory regions. Writes the result (first differing byte comparison) to a result address. Cost: max(10, n/250) CUs. | [`SyscallMemcmp`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/mem_ops.rs#L67) |
| `sol_memset_` | Fills `n` bytes of a memory region with a given byte value. Cost: max(10, n/250) CUs. | [`SyscallMemset`](https://github.com/anza-xyz/agave/blob/v3.1.8/syscalls/src/mem_ops.rs#L113) |
