---
title: "Quickstart: get a swap quote"
description:
  Pull a live, routed swap quote from a DEX aggregator in about five minutes -
  the first building block of any trading system on Solana.
---

You'll end up with live market data: a real quote for swapping SOL to USDC,
routed across Solana's DEX liquidity, with price impact and the exact route. You
need Node.js 20+ (or just `curl`) and about five minutes. Quoting is read-only -
no wallet or SOL required.

## 1. Request a quote

[Jupiter](https://dev.jup.ag)'s quote endpoint routes across major Solana DEXs
for best execution. The free tier needs no API key:

```bash
curl -s "https://lite-api.jup.ag/swap/v1/quote?inputMint=So11111111111111111111111111111111111111112&outputMint=EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v&amount=100000000&slippageBps=50"
```

The parameters: `inputMint` is wrapped SOL, `outputMint` is USDC, `amount` is
0.1 SOL in lamports, and `slippageBps=50` caps deviation from the quote at 0.5%.

## 2. Read the response

The fields that matter:

- **`outAmount`** - the USDC you'd receive (6 decimals, so `18234567` means
  ~18.23 USDC)
- **`priceImpactPct`** - how much your trade moves the price; large for thin
  liquidity
- **`routePlan`** - which venues the trade routes through and in what
  proportions
- **`otherAmountThreshold`** - the worst outcome your slippage setting allows

## 3. The same call from code

```ts
const params = new URLSearchParams({
  inputMint: "So11111111111111111111111111111111111111112",
  outputMint: "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v",
  amount: "100000000", // 0.1 SOL in lamports
  slippageBps: "50"
});

const quote = await fetch(
  `https://lite-api.jup.ag/swap/v1/quote?${params}`
).then((res) => res.json());

console.log("out:", quote.outAmount, "impact:", quote.priceImpactPct);
```

## 4. From quote to trade

Executing is one more API call - the aggregator returns a serialized transaction
for your wallet to sign. The full flow, including slippage strategy, inspecting
the transaction before signing, and landing it reliably, is in
[Swap tokens](/docs/defi/swaps).

<Callout>
  Aggregator quotes reflect live mainnet liquidity. To test execution without
  real funds, run [surfpool](/docs/intro/installation/surfpool-cli-basics) - it
  forks mainnet locally, so swap transactions execute against real pool state
  with test SOL.
</Callout>
