Cancel an existing [signatureSubscribe](/docs/rpc/websocket/signaturesubscribe)
subscription.

<Callout type="info" title="Source">
  [`signature_unsubscribe`](https://github.com/anza-xyz/agave/blob/v3.1.8/rpc/src/rpc_pubsub.rs#L525)
</Callout>

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "signatureUnsubscribe",
  "params": [0]
}
```

### !params

#### !! subscription id

!type integer  
!required

Subscription id returned by `signatureSubscribe`.

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  "result": true,
  "id": 1
}
```

!type boolean

Returns `true` if the subscription was removed. Returns an error if the
subscription id is unknown.

</APIMethod>
