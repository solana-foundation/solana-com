Subscribe to notifications when the validator sets a new root slot.

<Callout type="info" title="Source">
  [`root_subscribe`](https://github.com/anza-xyz/agave/blob/v3.1.8/rpc/src/rpc_pubsub.rs#L596)
</Callout>

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

```ts !!request title="Kit"
import { createSolanaRpcSubscriptions } from "@solana/kit";

const rpc = createSolanaRpcSubscriptions("wss://api.devnet.solana.com");

const subscription = await rpc
  .rootNotifications()
  .subscribe({ abortSignal: AbortSignal.timeout(5_000) });

for await (const notification of subscription) {
  console.log(notification);
}
```

```ts !!request title="web3.js"
import { Connection, clusterApiUrl } from "@solana/web3.js";

const connection = new Connection(clusterApiUrl("devnet"), "confirmed");

const subscriptionId = connection.onRootChange((root) => {
  console.log("New root:", root);
});
```

```rs !!request title="Rust"
use anyhow::Result;
use futures::StreamExt;
use solana_client::nonblocking::pubsub_client::PubsubClient;

#[tokio::main]
async fn main() -> Result<()> {
    let pubsub_client = PubsubClient::new("wss://api.devnet.solana.com/").await?;

    let (mut notifications, unsubscribe) = pubsub_client.root_subscribe().await?;

    while let Some(notification) = notifications.next().await {
        println!("{:?}", notification);
    }

    unsubscribe().await;

    Ok(())
}
```

### !params

**None**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

Subscription id. Pass this to
[rootUnsubscribe](/docs/rpc/websocket/rootunsubscribe).

</APIMethod>

### Notification format

Notifications are delivered as `rootNotification`.

<CodeReference>

```jsonc !!
{
  // !hover jsonrpc
  "jsonrpc": "2.0",
  // !hover method
  "method": "rootNotification",
  // !hover(1:4) params
  "params": {
    // !hover params.result
    "result": 324,
    // !hover params.subscription
    "subscription": 3
  }
}
```

## !reference

### !! jsonrpc

!type string

Always `"2.0"`.

### !! method

!type string

Always `"rootNotification"`.

### !! params

!type object

Notification wrapper with the latest root and the subscription id.

#### !! result

!type u64

Latest rooted slot observed by the validator.

#### !! subscription

!type integer

Subscription id that produced this notification.

</CodeReference>
