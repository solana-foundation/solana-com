Subscribe to notifications when the validator sets a new root slot.

<Callout type="info" title="Source">
  [`root_subscribe`](https://github.com/anza-xyz/agave/blob/v3.1.8/rpc/src/rpc_pubsub.rs#L596)
</Callout>

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**None**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

Subscription id. Pass this to
[rootUnsubscribe](/docs/rpc/websocket/rootunsubscribe).

</APIMethod>

### Notification format

Notifications are delivered as `rootNotification`.

<CodeReference>

```jsonc !!
{
  // !hover jsonrpc
  "jsonrpc": "2.0",
  // !hover method
  "method": "rootNotification",
  // !hover(1:4) params
  "params": {
    // !hover params.result
    "result": 324,
    // !hover params.subscription
    "subscription": 3
  }
}
```

## !reference

### !! jsonrpc

!type string

Always `"2.0"`.

### !! method

!type string

Always `"rootNotification"`.

### !! params

!type object

Notification wrapper with the latest root and the subscription id.

#### !! result

!type u64

Latest rooted slot observed by the validator.

#### !! subscription

!type integer

Subscription id that produced this notification.

</CodeReference>
