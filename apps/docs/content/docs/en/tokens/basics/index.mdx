SPL Tokens, focusing on functionality shared by both the original Token Program
and the Token Extension Program (Token 2022).

<Callout type="info">
  Refer to the [Metaplex
  documentation](https://developers.metaplex.com/token-metadata) for more
  information on how to add metadata to SPL Tokens.
