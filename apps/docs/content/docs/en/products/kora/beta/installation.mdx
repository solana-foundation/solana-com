## Cargo

Install the beta release directly from crates.io:

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

Official multi-platform Docker images (amd64/arm64) are published to GitHub
Container Registry:

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

Available tags:

- `latest` — Latest stable release
- `beta` — Latest pre-release version
- `v<version>` — Specific version (e.g., `v2.2.0-beta.7`)

### Docker Compose

### Running with Docker

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## TypeScript SDK

The beta SDK is available on npm:

```bash
pnpm add @solana/kora@beta
```

See [SDK Changes](/docs/products/kora/beta/sdk-changes) for new APIs.

## Verify Installation

```bash
kora --version
```
