This quickstart walks through the full Private Channels flow on Solana devnet:
set up the client environment, deposit SPL tokens into the escrow, send an
off-chain transfer between two wallets, and withdraw funds back to your Solana
wallet.

By the end you will understand the full deposit -> off-chain transfer ->
withdrawal cycle and have working TypeScript code for each step.

<Callout type="info">
  This quickstart assumes a Private Channels instance is already deployed and
  running on devnet with your mint on the allowlist. If you need to deploy one
  first, the [Operators guide](/docs/tools/private-channels/operators) covers
  the full setup - expect 30-60 minutes for the initial Docker build.
</Callout>

## Steps

1. **[Installation](/docs/tools/private-channels/quickstart/installation)** -
   clone the repository and generate the TypeScript clients (5 minutes)
2. **[Deposit Tokens](/docs/tools/private-channels/quickstart/deposit)** -
   deposit SPL tokens into the escrow; allow ~15 seconds for the indexer to
   credit your channel balance
3. **[Send a Transfer](/docs/tools/private-channels/quickstart/transfer)** -
   send an off-chain transfer through the gateway; confirms in under a second,
   never touches Mainnet
4. **[Withdraw Funds](/docs/tools/private-channels/quickstart/withdraw)** - burn
   your channel-side balance; the operator settles funds to your Mainnet wallet
   within a few seconds
