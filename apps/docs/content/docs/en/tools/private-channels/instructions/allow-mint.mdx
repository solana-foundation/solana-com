  Permits an SPL token mint on the instance and creates the escrow ATA for that

Permits an SPL token mint on the instance and creates the escrow's associated
token account for that mint. Deposits for this mint become possible after this
call.

## Parameters

| Parameter | Type | Description                                                                                        |
| --------- | ---- | -------------------------------------------------------------------------------------------------- |
| `bump`    | `u8` | Bump seed from `Pubkey::find_program_address(&[b"allowed_mint", instance_pda, mint], &PROGRAM_ID)` |

> The IDL contains a `discriminator: u8` argument (value `1`) that is auto-set
> by the generated client and should not be passed manually.

## Important Notes

- Also creates `instance_ata` - the escrow's ATA for this mint
- Mints with a transfer hook extension are explicitly blocked - attempting to
  allow such a mint throws `TransferHookNotAllowed`
- Emits an event via `EmitEvent` CPI
