Solana settles more stablecoin volume than any other use case on the network —
over [$1 trillion in 2025](https://app.artemisanalytics.com/asset/solana) — and
the Token-2022 program gives issuers the controls regulators expect (freeze,
pause, seize, allowlists) as native token features rather than custom contract
code.

## The issuance lifecycle

1. **Design** — pick extensions and authority structure. Start with
   [Choose a token design](/docs/tokenization/token-design).
2. **Issue** — create the mint with metadata and compliance controls:
   [Issue a stablecoin](/docs/tokenization/stablecoins/issue-a-stablecoin).
3. **Operate** — mint against deposits, burn against redemptions, respond to
   compliance events (freeze/seize), and publish attestations.
4. **Monitor** — track supply, transfers, and holders:
   [Index token activity](/docs/tokenization/index-token-activity).

## Compliance controls at a glance

| Requirement                             | Mechanism                                                                                                    |
| --------------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| Block sanctioned addresses              | Freeze authority, or blocklists via [Token ACL](/docs/tokenization/token-acl)                                |
| Approve holders before they can receive | [Default account state](/docs/tokens/extensions/default-state) (frozen until thawed) or Token ACL allowlists |
| Emergency stop                          | [Pausable](/docs/tokens/extensions/pausable)                                                                 |
| Seize funds under legal order           | [Permanent delegate](/docs/tokens/extensions/permanent-delegate)                                             |
| Burn from frozen accounts               | [Permissioned burn](/docs/tokens/extensions/permissioned-burn)                                               |

## Distribution

Once issued, a stablecoin is a standard SPL token: it works with every wallet,
the [payments stack](/docs/payments), fee abstraction via
[Kora](/docs/tools/kora) (users pay gas in your stablecoin), and
[confidential transfers](/docs/tokenization/private-transfers) where amounts
need to stay private.
