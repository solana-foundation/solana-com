[Solana Pay](https://solanapay.com) — это стандартный протокол для кодирования
платёжных запросов в виде URL. Эти URL можно отправлять как ссылки, кодировать в
QR-коды для мобильных кошельков или использовать для программного создания
платёжных транзакций.

Пакет `@solana-commerce/solana-pay` предоставляет функции
кодирования/декодирования URL, генерации QR-кодов и создания транзакций —
совместим с [@solana/kit](https://github.com/anza-xyz/kit).

## Установка

```bash
pnpm add @solana-commerce/solana-pay
```

## Создание платёжных URL

Используйте `encodeURL`, чтобы создать протокольный URL `solana:`, который
смогут распознать кошельки:

```typescript
import { encodeURL } from "@solana-commerce/solana-pay";

const url = encodeURL({
  recipient: address("merchant-wallet-address"),
  amount: 100000000n, // 0.1 SOL in lamports
  label: "Coffee Shop",
  message: "Thanks for your order!"
});

console.log(url.toString());
// solana:merchant...?amount=0.1&label=Coffee%20Shop&message=...
```

### Параметры URL

| Параметр    | Тип         | Описание                                       |
| ----------- | ----------- | ---------------------------------------------- |
| `recipient` | `Address`   | Адрес кошелька продавца (обязательно)          |
| `amount`    | `bigint`    | Сумма в лампортах или минорных единицах токена |
| `splToken`  | `Address`   | Адрес токена SPL (не указывать для SOL)        |
| `reference` | `Address[]` | Уникальные ссылки для отслеживания платежей    |
| `label`     | `string`    | Имя продавца, отображаемое в кошельке          |
| `message`   | `string`    | Сообщение, показываемое перед оплатой          |
| `memo`      | `string`    | On-chain memo, прикреплённое к транзакции      |

### Платежи в SPL-токенах

Для платежей в стейблкоинах укажите mint токена:

```typescript
import { encodeURL } from "@solana-commerce/solana-pay";

const url = encodeURL({
  recipient: address("merchant-wallet"),
  amount: 25000000n, // 25 USDC (6 decimals)
  splToken: USDC_MINT,
  label: "My Store"
});
```

### Отслеживание платежей по ссылкам

Создайте уникальный идентификатор для отслеживания конкретных платежей:

```typescript
import { encodeURL } from "@solana-commerce/solana-pay";
import { address, generateKeyPairSigner } from "@solana/kit";

const orderReference = (await generateKeyPairSigner()).address;

const url = encodeURL({
  recipient: address(MERCHANT_WALLET_ADDRESS),
  amount: 500000000n, // 0.5 SOL
  reference: orderReference,
  memo: `Order-${Date.now()}`
});

// Query the blockchain later for transactions containing this reference
```

## Генерация QR-кодов

Используйте `createQR`, чтобы создавать сканируемые QR-коды для мобильных
кошельков:

```typescript
import { createQR, encodeURL } from "@solana-commerce/solana-pay";
import { address } from "@solana/kit";

const url = encodeURL({
  recipient: address(MERCHANT_WALLET_ADDRESS),
  amount: 100000000n,
  label: "Coffee Shop"
});

const qrCode = await createQR(
  url.toString(),
  400, // size in pixels
  "white", // background
  "black" // foreground
);

// Use as image source
document.getElementById("qr").src = qrCode;
```

### Стилизованные QR-коды

Для брендированных QR-кодов с логотипами и индивидуальным оформлением:

```typescript
import { createStyledQRCode, encodeURL } from "@solana-commerce/solana-pay";

const qr = await createStyledQRCode(url.toString(), {
  width: 600,
  margin: 3,
  color: {
    dark: "#9945FF", // Solana purple
    light: "#FFFFFF"
  },
  errorCorrectionLevel: "H",
  dotStyle: "dots",
  cornerStyle: "extra-rounded",
  logo: "/your-logo.png",
  logoSize: 120
});
```

## Формирование транзакций

Используйте `createTransfer`, чтобы сформировать полную транзакцию для подписи:

```typescript
import { createTransfer } from "@solana-commerce/solana-pay";

const transaction = await createTransfer(
  client.rpc,
  address(SENDER_WALLET_ADDRESS),
  {
    recipient: address(MERCHANT_WALLET_ADDRESS),
    amount: 100000000n, // 0.1 SOL
    memo: "Coffee purchase"
  }
);
```

Для SPL-токенов укажите mint-адрес:

```typescript
const USDC_MINT = address("EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v");

const transaction = await createTransfer(
  client.rpc,
  address(SENDER_WALLET_ADDRESS),
  {
    recipient: address(MERCHANT_WALLET_ADDRESS),
    amount: 25000000n, // 25 USDC
    splToken: USDC_MINT,
    reference: [orderReference],
    memo: "Order #12345"
  }
);
```

## Разбор URL-адресов

Проверьте и извлеките данные из Solana Pay URL-адресов:

```typescript
import { parseURL, ParseURLError } from "@solana-commerce/solana-pay";

try {
  const parsed = parseURL("solana:merchant...?amount=1.5&label=Store");

  console.log(parsed.recipient); // Address
  console.log(parsed.amount); // 1500000000n (lamports)
  console.log(parsed.label); // "Store"
} catch (error) {
  if (error instanceof ParseURLError) {
    console.error("Invalid URL:", error.message);
  }
}
```
