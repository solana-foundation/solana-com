---
title: Обработка платежей
description:
  Обрабатывайте платежи в масштабе с помощью транзакций без газа и абстракции
  комиссий
---

Обработка массовых платежей — зарплаты, выплаты мерчантам, бонусы по программам
лояльности — требует большего, чем просто отправка отдельных переводов.

Чтобы эффективно обрабатывать множество платежей:

1. **Группируйте инструкции** в одну транзакцию для снижения комиссий и задержек
2. **Группируйте транзакции** для параллельной обработки в сети
3. **Абстрагируйте комиссии**, чтобы пользователям не нужно было держать SOL
