## Инструменты для разработки ИИ

Начните с нашего гайда по
[работе с инструментами ИИ на Solana](/developers/guides/getstarted/intro-to-ai).

### Solana MCP

[Solana MCP](https://mcp.solana.com/) (Machine Conversation Protocol)
интегрирует знания о Solana напрямую в инструменты программирования с поддержкой
ИИ, такие как Claude, Cursor и Windsurf. Обучен на документации Solana,
фреймворке Anchor, примерах программ и вопросах-ответах Stack Exchange — полезен
для быстрой интеграции платежей.

### Фреймворки для ИИ-агентов

ИИ-агенты могут автономно выполнять платёжные транзакции в Solana. Эти
фреймворки позволяют создавать агентов для обработки платежей, мониторинга
транзакций и автоматизации платёжных процессов:

| Фреймворк        | Описание                                                                                  | Ссылки                                                                                    |
| ---------------- | ----------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | Инструменты для создания агентов с поддержкой платежей, совместим с LangChain и Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | Коллекция готовых навыков для создания агентов на Solana                                  | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | Мультиплатформенный фреймворк агентов с плагином Solana для ботов X, Telegram, Discord    | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Кроссчейн-инструментарий с действиями для платежей в Solana                               | [GitHub](https://github.com/goat-sdk/goat)                                                |

### Разработка программ с ИИ

[Codigo](https://codigo.ai/) — это комплексная платформа для разработки ИИ на
Solana.
