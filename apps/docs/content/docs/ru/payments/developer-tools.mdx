### Фреймворки для AI-агентов

AI-агенты могут автономно выполнять платёжные транзакции в Solana. Эти
фреймворки позволяют создавать агентов, которые обрабатывают платежи,
отслеживают транзакции и автоматизируют платёжные процессы:

| Фреймворк        | Описание                                                                                    | Ссылки                                                                                    |
| ---------------- | ------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | Инструменты для создания агентов с поддержкой платежей, совместимые с LangChain и Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | Коллекция готовых навыков для создания агентов на Solana                                    | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | Мультиплатформенный фреймворк для агентов с плагином Solana для ботов X, Telegram и Discord | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Кроссчейн-инструментарий с действиями для платежей в Solana                                 | [GitHub](https://github.com/goat-sdk/goat)                                                |

### Разработка AI-программ

[Codigo](https://codigo.ai/) — это комплексная платформа для разработки на
Solana с использованием AI.
