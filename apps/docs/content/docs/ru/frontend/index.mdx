Разработка фронтенда на Solana включает работу с программами, кошельками и
популярными JavaScript-фреймворками, такими как React. Для взаимодействия с
этими компонентами необходимо уметь устанавливать соединение, создавать
транзакции и читать данные из аккаунтов Solana.

[Solana Kit](https://github.com/anza-xyz/kit) (`@solana/kit`) — это стандартный
JavaScript SDK для этих задач. Вы собираете клиент из небольших, компонуемых
плагинов, добавляете привязки React при необходимости и подключаете
сгенерированные клиенты программ для вызова on-chain программ.

## Основные библиотеки

<Cards>
  <Card title="@solana/kit" href="https://github.com/anza-xyz/kit">
    - Основной SDK: RPC, кодеки, подписанты и транзакции - Предоставляет
    `createClient` — хост плагинов, который расширяет каждая другая библиотека
  </Card>
  <Card title="Kit plugins" href="https://github.com/anza-xyz/kit-plugins">
    - Собирайте клиент с помощью `.use(...)`: подключение RPC, кошелёк,
    подписант и планирование транзакций - Публикуется в виде пакетов
    `@solana/kit-plugin-*`
  </Card>
  <Card
    title="@solana/react"
    href="https://github.com/anza-xyz/kit/tree/main/packages/react"
  >
    - Привязки React для Kit-клиента - `ClientProvider` плюс хуки для чтения,
    подписок и состояния кошелька
  </Card>
  <Card title="@solana-program/*" href="https://github.com/solana-program">
    - Сгенерированные клиенты для on-chain программ (System, Token и другие) -
    Предоставляют построители инструкций и плагины клиента, такие как
    `tokenProgram()`
  </Card>
</Cards>

<Callout type="warn" title="web3.js v1 and wallet-adapter are legacy">
  Многие проекты Solana по-прежнему используют устаревшие `@solana/web3.js` (v1)
  и `@solana/wallet-adapter-*`. Для новых проектов предпочтительнее использовать
  `@solana/kit` с плагинами kit, перечисленными ниже; обнаружение Wallet
  Standard (через `@solana/kit-plugin-wallet`) заменяет wallet-adapter для
  современных кошельков.
</Callout>

- [Руководство по Kit-клиенту](/docs/frontend/client): сборка клиента из
  плагинов для RPC, подписания и отправки транзакций.
- [Руководство по React](/docs/frontend/react-hooks): привязки `@solana/react` и
  хуки кошелька поверх того же клиента.
- [Туториал Next.js](/docs/frontend/nextjs-solana): полноценное приложение с

- [Миграция на Kit](/docs/frontend/web3-compat): как концепции web3.js и
  wallet-adapter соответствуют концепциям Kit.
