Начните работу с Solana Attestations, установив необходимые пакеты и настроив
среду разработки.

## TypeScript/JavaScript

Установите библиотеку Solana Attestations, используя предпочитаемый менеджер
пакетов:

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
