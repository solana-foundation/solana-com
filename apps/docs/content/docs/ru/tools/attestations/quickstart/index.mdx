Начните работу с Solana Attestations всего за несколько шагов. Это руководство
ознакомит вас с основными концепциями и поможет создать вашу первую аттестацию.

## Что вы изучите

- Установка и настройка библиотеки Attestations
- Создание вашего первого учётного документа (центра аттестации)
- Определение схемы для ваших аттестаций
- Выдача вашей первой аттестации
- Получение и проверка аттестаций

## Предварительные требования

- Базовые знания разработки на Solana
- Node.js 16+ или среда разработки Rust
- Кошелёк Solana с небольшим количеством SOL для оплаты транзакций

## Основные концепции

### Учётные документы

Учётный документ представляет собой центр аттестации. Он определяет, кто может
выдавать аттестации, и содержит список авторизованных подписантов.

### Схемы

Схемы определяют структуру и правила валидации для аттестаций. Они указывают,
какие поля содержит аттестация и их типы данных.

### Аттестации

Аттестации — это проверяемые утверждения или заявления, созданные
авторизованными подписантами в рамках учетных данных, следующие определенной
структуре схемы.

## Варианты применения

Аттестации могут использоваться для:

- **Верификация личности**: Проверка личности пользователей и соответствия
  требованиям
- **Репутационные системы**: Формирование рейтингов доверия и репутационных
  метрик
- **Сертификация**: Выдача профессиональных сертификатов и квалификаций
- **Соответствие требованиям**: Обеспечение соблюдения нормативных требований
  для финансовых услуг
- **Цепочка поставок**: Отслеживание подлинности и происхождения товаров
