**Solana Pay — это стандартный протокол и набор эталонных реализаций, которые
позволяют разработчикам интегрировать децентрализованные платежи в свои
приложения и сервисы.** Блокчейн Solana подтверждает транзакции менее чем за
секунду и стоит в среднем $0.0005, обеспечивая пользователям бесшовный опыт без
посредников.

## Почему Solana Pay?

- **Стандартный протокол**: Единый подход к платежам во всех кошельках и
  приложениях Solana
- **Сверхбыстро**: Транзакции подтверждаются менее чем за секунду
- **Сверхдёшево**: Средняя стоимость транзакции $0.0005
- **Без посредников**: Прямые одноранговые платежи без третьих сторон
- **Универсальная поддержка кошельков**: Совместимость с Phantom, Solflare,
  Backpack и другими

## Быстрый старт

Самый простой способ начать — использовать URL-адрес запроса на перевод:

```
solana:recipient?amount=0.01&label=Coffee%20Shop&message=Grande%20Latte
```

Этот URL можно:

- Закодировать в QR-код для сканирования мобильным кошельком
- Использовать как платёжную ссылку в веб-приложениях
- Встроить в мобильные приложения в виде deep links

## Установка

Установите JavaScript SDK Solana Pay:

```bash
# Using pnpm (recommended)
pnpm add @solana/pay@beta @solana/kit

# Using npm
npm install @solana/pay@beta @solana/kit

# Using yarn
yarn add @solana/pay@beta @solana/kit
```

## Варианты использования

Solana Pay поддерживает широкий спектр платёжных сценариев:

- **Розничные платежи**: Платежи офлайн с использованием QR-кодов
- **Электронная коммерция**: Веб-процессы оформления заказов с интеграцией
  кошельков
- **Мобильные приложения**: Нативный платёжный опыт с deep linking
- **Минтинг NFT**: Динамическое создание транзакций для сложных взаимодействий
- **Подписки**: Регулярные платежи и программы лояльности
- **Интеграция с DeFi**: Составные транзакции со свопами и кредитованием

## Поддерживаемые кошельки

Solana Pay работает со всеми основными кошельками Solana:

- **[Phantom](https://phantom.app/)** - самый популярный кошелек Solana
- **[Solflare](https://solflare.com/)** - мультиплатформенный кошелек Solana
- **[Backpack](https://backpack.app/)** - Web3-кошелек и магазин приложений
- **[Glow](https://glow.app/)** - мобильный кошелек Solana
- **[Ultimate](https://ultimate.app/)** - кошелек для DeFi
- **[Decaf Wallet](https://decaf.so/)** - кошелек для глобальных платежей
- **[Espresso Cash](https://www.espressocash.com/)** - кошелек для глобальных
  платежей
- **[Tiplink](https://tiplink.io/)** - самый простой кошелек в мире
- **[Bitget Wallet](https://bitget.com/)** - мультичейн-кошелек
