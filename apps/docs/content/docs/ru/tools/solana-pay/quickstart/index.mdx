Быстро начните работу с Solana Pay, следуя этим пошаговым руководствам.

## Установка

Настройте Solana Pay SDK и сконфигурируйте среду разработки.

**[→ Руководство по установке](/docs/tools/solana-pay/quickstart/installation)** -
Установите SDK и зависимости

## Типы платежей

Выберите подходящий способ интеграции платежей для вашего случая использования:

### Запросы на перевод (простой способ)

Создавайте неинтерактивные URL-адреса для базовых переводов SOL и SPL-токенов.

**[→ Запросы на перевод](/docs/tools/solana-pay/quickstart/transfer-requests)** -
Простые URL-адреса для платежей

**Лучше всего подходит для:**

- Базовых платежей и пожертвований
- Платежей по QR-кодам
- Платёжных ссылок в электронных письмах

### Запросы транзакций (продвинутый способ)

Создавайте интерактивные процессы оплаты с серверной композицией транзакций.

**[→ Запросы транзакций](/docs/tools/solana-pay/quickstart/transaction-requests)** -
Интерактивные процессы оплаты

**Лучше всего подходит для:**

- Динамического ценообразования
- Минтинга NFT
- Сложных DeFi-транзакций
- Пользовательской бизнес-логики

## Интеграция QR-кодов

Генерируйте и настраивайте QR-коды для платежей через мобильные кошельки.

**[→ Интеграция QR-кодов](/docs/tools/solana-pay/quickstart/qr-codes)** -
Создавайте сканируемые платёжные коды

**Лучше всего подходит для:**

- Систем точек продаж
- Платежей с веб-сайта на мобильное устройство
- Личных транзакций
