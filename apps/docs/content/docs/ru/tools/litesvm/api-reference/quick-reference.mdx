## Краткий справочник

| Метод                                          | Описание                                                                                               |
| ---------------------------------------------- | ------------------------------------------------------------------------------------------------------ |
| `LiteSVM::new()`                               | Создать новую базовую тестовую среду LiteSVM                                                           |
| `LiteSVM::default()`                           | Возвращает «значение по умолчанию» для типа                                                            |
| `airdrop(&Address, lamports)`                  | Пополняет аккаунт указанным количеством lamport                                                        |
| `get_account(&Address)`                        | Возвращает всю информацию, связанную с аккаунтом по указанному адресу                                  |
| `set_account(Address, Account)`                | Устанавливает всю информацию, связанную с аккаунтом по указанному адресу                               |
| `get_balance(&Address)`                        | Возвращает баланс в lamport для аккаунта по указанному адресу                                          |
| `minimum_balance_for_rent_exemption(data_len)` | Возвращает минимальный баланс, необходимый для освобождения аккаунта с указанной длиной данных от rent |
| `add_program(program_id, &bytes)`              | Добавляет SBF-программу в тестовую среду                                                               |
| `add_program_from_file(program_id, path)`      | Добавляет SBF-программу в тестовую среду из указанного файла                                           |
| `send_transaction(tx)`                         | Отправляет подписанную транзакцию                                                                      |
| `simulate_transaction(tx)`                     | Симулирует транзакцию без изменения состояния                                                          |
| `get_transaction(&signature)`                  | Получает транзакцию из истории                                                                         |
| `latest_blockhash()`                           | Возвращает текущий blockhash                                                                           |
| `expire_blockhash()`                           | Принудительно завершает срок действия текущего blockhash                                               |
| `warp_to_slot(slot)`                           | Переводит часы на указанный slot                                                                       |
| `get_sysvar<T>()`                              | Получает sysvar из тестовой среды                                                                      |
| `set_sysvar<T>(&value)`                        | Устанавливает sysvar в тестовой среде                                                                  |
| `with_feature_accounts()`                      | Включает аккаунты feature gate в тестовую среду                                                        |
| `set_invocation_inspect_callback(callback)`    | Регистрирует обратный вызов до/после вызова (требует функции `invocation-inspect-callback`)            |
