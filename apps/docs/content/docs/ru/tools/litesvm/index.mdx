  Молниеносное тестирование программ Solana с виртуальной машиной внутри
  процесса. Тестируйте ваши программы быстрее без сетевых задержек или процессов
  validator.
---

LiteSVM — это легковесная библиотека для тестирования программ Solana, которая
создаёт виртуальную машину Solana внутри процесса, оптимизированную для
разработчиков программ. В отличие от других подходов к тестированию, запускающих
отдельные процессы validator, LiteSVM встраивает виртуальную машину
непосредственно в ваши тесты, что значительно ускоряет их выполнение. Помимо
этого, библиотека предоставляет удобный API с разумными настройками по умолчанию
и широкими возможностями конфигурации для тех, кому это необходимо.
