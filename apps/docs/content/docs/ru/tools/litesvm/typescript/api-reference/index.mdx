## Краткий справочник

### Настройка плагина

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### Управление аккаунтами (client.svm)

| Метод                                  | Описание                                     |
| -------------------------------------- | -------------------------------------------- |
| `setAccount(account)`                  | Напрямую задать состояние аккаунта           |
| `getAccount(address)`                  | Получить данные аккаунта                     |
| `getBalance(address)`                  | Получить баланс SOL                          |
| `airdrop(address, lamports)`           | Пополнить аккаунт                            |
| `minimumBalanceForRentExemption(size)` | Рассчитать минимум для освобождения от ренты |

### Управление программами (client.svm)

| Метод                                 | Описание                         |
| ------------------------------------- | -------------------------------- |
| `addProgram(programId, bytes)`        | Загрузить программу из байтов    |
| `addProgramFromFile(programId, path)` | Загрузить программу из .so файла |

### Транзакции (client.svm)

| Метод                     | Описание                             |
| ------------------------- | ------------------------------------ |
| `sendTransaction(tx)`     | Выполнить транзакцию                 |
| `simulateTransaction(tx)` | Симулировать без изменения состояния |
| `latestBlockhash()`       | Получить текущий blockhash           |
| `expireBlockhash()`       | Перейти к новому blockhash           |

### Конфигурация (client.svm)

| Метод                       | Описание                               |
| --------------------------- | -------------------------------------- |
| `withSigverify(bool)`       | Включить/отключить проверку подписей   |
| `withBlockhashCheck(bool)`  | Включить/отключить проверку blockhash  |
| `withSysvars()`             | Включить аккаунты sysvar               |
| `withBuiltins()`            | Включить встроенные программы          |
| `withPrecompiles()`         | Включить предкомпилированные программы |
| `withTransactionHistory(n)` | Хранить последние n транзакций         |
| `withComputeBudget(budget)` | Установить лимиты вычислений           |
| `withLamports(amount)`      | Установить lamport по умолчанию        |
| `withLogBytesLimit(n)`      | Установить лимит вывода логов          |

### Время и Sysvar-ы (client.svm)

| Метод                      | Описание                              |
| -------------------------- | ------------------------------------- |
| `warpToSlot(slot)`         | Перейти к определённому slot          |
| `getClock()`               | Получить clock sysvar                 |
| `getRent()`                | Получить rent sysvar                  |
| `getEpochSchedule()`       | Получить расписание epoch             |
| `getLastRestartSlot()`     | Получить последний slot перезапуска   |
| `setLastRestartSlot(slot)` | Установить последний slot перезапуска |

### RPC (client.rpc)

| Метод                            | Описание                              |
| -------------------------------- | ------------------------------------- |
| `getAccountInfo(address)`        | Получить один аккаунт (только base64) |
| `getMultipleAccounts(addresses)` | Получить несколько аккаунтов          |
| `getLatestBlockhash()`           | Получить текущий blockhash            |
