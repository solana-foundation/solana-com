Не хотите запускать собственную ноду Kora? Следующие провайдеры предлагают
хостинговую инфраструктуру Kora, чтобы вы могли добавить абстракцию комиссий в
своё приложение без необходимости управлять собственным развёртыванием.

## Провайдеры

### Openfort

[Openfort](https://www.openfort.io/) предлагает инфраструктуру paymaster для
Solana на основе Kora, обеспечивая транзакции без комиссий через платформу
встроенных кошельков. Их интеграция использует паттерн совместного подписания,
где Kora выступает в роли плательщика комиссий, а пользователи подписывают
транзакции через встроенные кошельки Openfort.

- **Веб-сайт**: [openfort.io](https://www.openfort.io/)
- **Документация**:
  [Транзакции без комиссий в Solana](https://www.openfort.io/blog/solana-gasless-transactions)
- **Возможности**: Встроенные кошельки, транзакции без комиссий, спонсирование
  комиссий

## Станьте оператором ноды Kora

Kora — проект с открытым исходным кодом, и любой может запустить ноду.
Независимо от того, создаёте ли вы приложение, которому требуется абстракция
комиссий, или хотите предлагать услуги paymaster другим, ознакомьтесь с
[Руководством для операторов](/docs/tools/kora/operators), чтобы начать работу.
