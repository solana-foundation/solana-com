_Доступно на GitHub:
[`v2.2.0-beta.7`](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.7)_

Kora v2.2.0-beta включает поддержку пакетов Jito, новые методы RPC,
Docker-образы, улучшенные функции безопасности и усовершенствования SDK.

> **Это предварительная версия.** Функции, описанные здесь, доступны в
> `v2.2.0-beta.7` и могут измениться до стабильного релиза. Для стабильной
> документации см. [основную документацию Kora](/docs/tools/kora).

## Основные возможности

### Поддержка пакетов Jito

Отправляйте до 5 транзакций атомарно через движок блоков Jito. Новые методы RPC:

- [`signBundle`](/docs/tools/kora/beta/bundle-methods#signbundle) — Подписать
  пакет без передачи
- [`signAndSendBundle`](/docs/tools/kora/beta/bundle-methods#signandsendbundle)
  — Подписать и отправить в Jito
- [`estimateBundleFee`](/docs/tools/kora/beta/bundle-methods#estimatebundlefee)
  — Оценить общие комиссии пакета

См. [Руководство по пакетам Jito](/docs/tools/kora/beta/jito-guide) для полного
пошагового руководства.

### Docker-образы

Официальные мультиплатформенные образы (amd64/arm64) опубликованы в GitHub
Container Registry. См. [Установка](/docs/tools/kora/beta/installation).

### Плагины транзакций

Новая система плагинов позволяет Kora проверять структуру транзакций в процессе
подписания. Первый встроенный плагин, `gas_swap`, применяет строгие правила для
беспошлинных транзакций обмена токенов на SOL. Плагины настраиваются в
`kora.toml` и запускаются автоматически во время `signTransaction`,
`signAndSendTransaction`, `signBundle` и `signAndSendBundle`.

См.
[Конфигурация — Плагины транзакций](/docs/tools/kora/beta/configuration#transaction-plugins).

### Новые параметры конфигурации

- **Плагины транзакций** для проверки структуры транзакций (`gas_swap`)
- **reCAPTCHA v3** защита от ботов для конечных точек подписания
- **Lighthouse** защита от истощения плательщика комиссий
- **Лимиты использования** ограничение транзакций для каждого кошелька
- **Конфигурация пакетов** для интеграции с Jito

См. [Конфигурация](/docs/tools/kora/beta/configuration).

### Изменения в SDK

- Поддержка методов пакетов (`signBundle`, `signAndSendBundle`,
  `estimateBundleFee`)

См. [Изменения в SDK](/docs/tools/kora/beta/sdk-changes).

### Дополнительные методы RPC

- [`getVersion`](/docs/tools/kora/beta/bundle-methods#getversion) — Получить
  информацию о версии сервера

## Установка

```bash
cargo install kora-cli@2.2.0-beta.7
```

Или через Docker:

```bash
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```
