## Cargo

Установите бета-версию напрямую с crates.io:

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

Официальные мультиплатформенные Docker-образы (amd64/arm64) публикуются в GitHub
Container Registry:

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

Доступные теги:

- `latest` — последний стабильный релиз
- `beta` — последняя предварительная версия
- `v<version>` — конкретная версия (например, `v2.2.0-beta.7`)

### Docker Compose

Для кастомных сборок или для включения Redis для кеширования и поддержки лимитов
использования:

- [Пример Dockerfile](https://github.com/solana-foundation/kora/blob/main/examples/deploy/Dockerfile)
- [Docker Compose с Redis](https://github.com/solana-foundation/kora/blob/main/docker-compose.yml)

### Запуск с Docker

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## TypeScript SDK

Бета-версия SDK доступна в npm:

```bash
pnpm add @solana/kora@beta
```

См. [Изменения SDK](/docs/tools/kora/beta/sdk-changes) для ознакомления с новыми
API.

## Проверка установки

```bash
kora --version
```
