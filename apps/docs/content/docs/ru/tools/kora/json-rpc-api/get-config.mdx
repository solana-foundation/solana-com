---
title: getConfig
description: Получает текущую конфигурацию сервера Kora.
---

## JSON-RPC запрос

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getConfig",
  "params": []
}
```

## JSON-RPC ответ

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "enabled_methods": {},
    "fee_payers": ["3Z1Ef7YaxK8oUMoi6exf7wYZjZKWJJsrzJXSt1c3qrDE"],
    "validation_config": {}
  }
}
```

## Пример cURL

```bash
curl -X POST http://localhost:8080 \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"getConfig","params":[]}'
```

## TypeScript SDK

```typescript
const config = await client.getConfig();
console.log("Fee payer:", config.fee_payer);
console.log(
  "Validation config:",
  JSON.stringify(config.validation_config, null, 2)
);
```
