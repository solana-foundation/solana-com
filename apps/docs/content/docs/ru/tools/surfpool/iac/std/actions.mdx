Эти действия доступны во всех Runbooks в рамках стандартной библиотеки.

## send_http_request

Действие `std::send_http_request` выполняет HTTP-запрос к указанному URL.

### Входные параметры

| Название     | Обязательный   | Тип     | Описание                                               |
| ------------ | -------------- | ------- | ------------------------------------------------------ |
| `url`        | обязательный   | string  | URL-адрес для отправки запроса                         |
| `method`     | необязательный | string  | HTTP-метод (GET, POST, PUT, DELETE). По умолчанию: GET |
| `headers`    | необязательный | map     | HTTP-заголовки для включения в запрос                  |
| `body`       | необязательный | string  | Тело запроса (для POST/PUT-запросов)                   |
| `timeout_ms` | необязательный | integer | Тайм-аут запроса в миллисекундах                       |

### Выходные параметры

| Название      | Тип     | Описание           |
| ------------- | ------- | ------------------ |
| `status_code` | integer | HTTP-код состояния |
| `body`        | string  | Тело ответа        |
| `headers`     | map     | Заголовки ответа   |

```hcl
action "api_call" "std::send_http_request" {
    description = "Fetch data from API"
    url = "https://api.example.com/data"
    method = "GET"
    headers = {
        "Authorization" = "Bearer ${variable.token}"
    }
}

output "response" {
    value = action.api_call.body
}

output "status" {
    value = action.api_call.status_code
}
```

### Пример POST-запроса

```hcl
action "create_resource" "std::send_http_request" {
    description = "Create a new resource"
    url = "https://api.example.com/resources"
    method = "POST"
    headers = {
        "Content-Type" = "application/json"
    }
    body = {
        "name" = variable.name,
        "value" = variable.value
    }
}
```
