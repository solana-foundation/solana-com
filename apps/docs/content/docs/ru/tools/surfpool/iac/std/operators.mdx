## and_bool

`and_bool` возвращает побитовое И левого и правого аргументов.

### Входные данные

| Имя   | Тип  | Описание        |
| ----- | ---- | --------------- |
| `lhs` | bool | Булево значение |
| `rhs` | bool | Булево значение |

### Выходные данные

| Имя     | Тип  | Описание                               |
| ------- | ---- | -------------------------------------- |
| `value` | bool | Результат побитового И двух аргументов |

```hcl
output "my_bool" {
  value = false && true
}
// > my_bool: false
```

## div

`div` возвращает результат целочисленного деления левого аргумента на правый
аргумент, округляя остаток вниз до ближайшего целого числа.

### Входные данные

| Имя   | Тип     | Описание |
| ----- | ------- | -------- |
| `lhs` | integer | Делимое  |
| `rhs` | integer | Делитель |

### Выходные данные

| Имя     | Тип     | Описание                               |
| ------- | ------- | -------------------------------------- |
| `value` | integer | Результат деления делимого на делитель |

```hcl
output "my_int" {
  value = 11 / -3
}
// > my_int: -3
```

## eq

`eq` возвращает `true`, если аргументы левой и правой части равны, и `false`,
если они не равны.

### Входные данные

| Имя   | Тип | Описание       |
| ----- | --- | -------------- |
| `lhs` | any | Любое значение |
| `rhs` | any | Любое значение |

### Выходные данные

| Имя     | Тип  | Описание                                           |
| ------- | ---- | -------------------------------------------------- |
| `value` | bool | Результат проверки равенства двух входных значений |

```hcl
output "is_eq" {
  value = "arg" == "arg"
}
// > is_eq: true
```

## gt

### Входные данные

| Имя   | Тип                        | Описание              |
| ----- | -------------------------- | --------------------- |
| `lhs` | integer \| float \| string | Сравниваемое значение |
| `rhs` | integer \| float \| string | Сравниваемое значение |

### Выходные данные

| Имя     | Тип  | Описание                                                                        |
| ------- | ---- | ------------------------------------------------------------------------------- |
| `value` | bool | Результат проверки того, что аргумент левой части больше аргумента правой части |

```hcl
output "is_gt" {
  value = 2 > 1
}
// > is_gt: true
```

| ----- | --- | ---------------------------- |
| `lhs` | any | Значение для сравнения       |
| `rhs` | any | Значение для сравнения с ним |

### Выходные данные

| Название | Тип  | Описание                                                |
| -------- | ---- | ------------------------------------------------------- |
| `value`  | bool | Результат проверки на неравенство двух входных значений |

```hcl

// > is_neq: false
```

## minus

`minus` возвращает результат вычитания аргумента `integer` правой части из
аргумента `integer` левой части.

### Входные данные

| Название | Тип     | Описание    |
| -------- | ------- | ----------- |
| `lhs`    | integer | Уменьшаемое |
| `rhs`    | integer | Вычитаемое  |

### Выходные данные

| Название | Тип     | Описание                     |
| -------- | ------- | ---------------------------- |
| `value`  | integer | Результат операции вычитания |

```hcl
output "my_integer" {
  value = 10 - 6
}
// > my_integer: 4
```
