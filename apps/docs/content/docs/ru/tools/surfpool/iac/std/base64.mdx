## encode_base64

`encode_base64` кодирует буфер или строку в строку base64.

### Входные данные

| Имя     | Тип                       | Описание                                                                                                                                                       |
| ------- | ------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | буфер, строка, дополнение | Буфер или строка для кодирования. Строки, начинающиеся с '0x', декодируются как шестнадцатеричные; в противном случае используются необработанные байты UTF-8. |

### Выходные данные

| Имя     | Тип    | Описание                                       |
| ------- | ------ | ---------------------------------------------- |
| `value` | строка | Входные данные, закодированные в строку base64 |

```hcl
output "encoded" {
    value = encode_base64("0x48656c6c6f20776f726c6421")
}
// > encoded: SGVsbG8gd29ybGQh
```

## decode_base64

`decode_base64` декодирует строку в кодировке base64 и возвращает результат в
виде буфера.

### Входные данные

| Имя             | Тип    | Описание                        |
| --------------- | ------ | ------------------------------- |
| `base64_string` | строка | Строка base64 для декодирования |

### Выходные данные

| Имя     | Тип   | Описание                                   |
| ------- | ----- | ------------------------------------------ |
| `value` | буфер | Декодированная строка base64 в виде буфера |

```hcl
output "decoded" {
    value = decode_base64("SGVsbG8gd29ybGQh")
}
// > decoded: 0x48656c6c6f20776f726c6421
```
