---
title: Обзор SVM Addon
description: Дополнение для Solana и SVM-совместимых блокчейнов (бета)
---

Плагин SVM `txtx` позволяет создавать Runbook'и, взаимодействующие с Solana и
SVM-совместимыми блокчейнами. Плагин предоставляет вспомогательные функции,
позволяющие разворачивать программы Anchor и кодировать вызовы инструкций в
соответствии с IDL программ. Действия можно использовать для создания корректных
транзакций перевода, вызова программ и развёртывания программ, которые можно
подписать с помощью мнемонической фразы, секретного ключа или браузерного
подписанта.

## Быстрый старт

Чтобы использовать SVM addon, объявите его в своём Runbook:

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## Возможности

- **Развёртывание программ**: Развёртывание программ Anchor и нативных программ
  Solana
- **Построение инструкций**: Кодирование вызовов инструкций с использованием IDL
  программ
- **Подпись транзакций**: Поддержка мнемоник, секретных ключей и веб-кошельков
- **Управление аккаунтами**: Запрос и управление аккаунтами Solana
