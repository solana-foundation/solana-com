## encode_hex

`encode_hex` кодирует буфер или строку как шестнадцатеричную строку с префиксом
0x.

### Входные данные

| Name    | Type                  | Description                                                                                                                                                    |
| ------- | --------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | Буфер или строка для кодирования. Строки, начинающиеся с «0x», декодируются как шестнадцатеричные; в противном случае используются необработанные байты UTF-8. |

### Выходные данные

| Name    | Type   | Description                                                     |
| ------- | ------ | --------------------------------------------------------------- |
| `value` | string | Входные данные в шестнадцатеричном представлении с префиксом 0x |

```hcl
output "encoded_hex" {
    value = encode_hex("hello, world")
}
// > encoded_hex: 0x68656c6c6f2c20776f726c64
```

## decode_hex

`decode_hex` декодирует шестнадцатеричную строку и возвращает результат в виде
буфера.

### Входные данные

| Name         | Type   | Description                                                              |
| ------------ | ------ | ------------------------------------------------------------------------ |
| `hex_string` | string | Шестнадцатеричная строка для декодирования (с префиксом 0x или без него) |

### Выходные данные

| Name    | Type   | Description                                           |
| ------- | ------ | ----------------------------------------------------- |
| `value` | buffer | Декодированная шестнадцатеричная строка в виде буфера |

```hcl
output "decoded_hex" {
    value = decode_hex("0x68656c6c6f2c20776f726c64")
}
// > decoded_hex: 0x68656c6c6f2c20776f726c64
```
