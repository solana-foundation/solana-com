## assert_eq

`assert_eq` проверяет равенство двух значений.

### Входные параметры

| Имя     | Тип | Описание                                   |
| ------- | --- | ------------------------------------------ |
| `left`  | any | Значение для сравнения                     |
| `right` | any | Значение, с которым производится сравнение |

### Выходные данные

| Имя     | Тип                   | Описание              |
| ------- | --------------------- | --------------------- |
| `value` | addon(std::assertion) | Результат утверждения |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```

## assert_gt

`assert_gt` проверяет, что левое значение больше правого.

### Входные параметры

| Имя     | Тип              | Описание                                                     |
| ------- | ---------------- | ------------------------------------------------------------ |
| `left`  | integer \| float | Целое число или число с плавающей точкой для сравнения       |
| `right` | integer \| float | Целое число или число с плавающей точкой для сравнения с ним |

### Выходные данные

| Имя     | Тип                   | Описание              |
| ------- | --------------------- | --------------------- |
| `value` | addon(std::assertion) | Результат утверждения |

```hcl
output "assertion" {
    value = std::assert_gt(action.example.result, 1)
}
```
