## index

`index` возвращает элемент списка по указанному индексу.

### Входные данные

| Имя     | Тип                                              | Описание                                |
| ------- | ------------------------------------------------ | --------------------------------------- |
| `list`  | array[string] \| array[integer] \| array[buffer] | Список, из которого извлекается элемент |
| `index` | integer                                          | Индекс извлекаемого элемента            |

### Выходные данные

| Имя     | Тип    | Описание                             |
| ------- | ------ | ------------------------------------ |
| `value` | string | Элемент списка по указанному индексу |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
