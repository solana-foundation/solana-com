## jq

Функция `jq` позволяет выполнять срезы, фильтрацию и преобразование данных JSON.
Подробности см. в [документации jq](https://jqlang.github.io/jq/manual/).

### Входные данные

| Имя            | Тип              | Описание                                                                |
| -------------- | ---------------- | ----------------------------------------------------------------------- |
| `decoded_json` | string \| object | Объект JSON                                                             |
| `query`        | string           | Запрос JSON. См. [документацию jq](https://jqlang.github.io/jq/manual/) |

### Выходные данные

| Имя     | Тип           | Описание               |
| ------- | ------------- | ---------------------- |
| `value` | array[string] | Результат запроса `jq` |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
