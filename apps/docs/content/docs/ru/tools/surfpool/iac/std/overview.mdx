Стандартная библиотека предоставляет основные функции и действия, доступные во
всех Runbook'ах вне зависимости от того, какие блокчейн-аддоны подключены.

## Функции

Стандартная библиотека включает вспомогательные функции для:

- **Операторы**: Математические операции и сравнения
- **Hash**: Функции хеширования (SHA256, Keccak256 и др.)
- **Hex**: Кодирование и декодирование в шестнадцатеричный формат
- **Crypto**: Криптографические операции
- **JSON**: Кодирование и декодирование JSON
- **List**: Функции работы со списками
- **Утверждения**: Проверки равенства и условий
- **Base64**: Кодирование и декодирование Base64

[Просмотреть все функции](/docs/tools/surfpool/iac/std/functions)

## Действия

Стандартная библиотека включает действия для:

- **HTTP**: Выполнение HTTP-запросов
- **Файловые операции**: Чтение и запись файлов

[Просмотреть все действия](/docs/tools/surfpool/iac/std/actions)
