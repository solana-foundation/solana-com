## ripemd160

`ripemd160` вычисляет хеш Ripemd160 для заданного значения.

### Входные данные

| Имя     | Тип                     | Описание                                 |
| ------- | ----------------------- | ---------------------------------------- |
| `value` | buffer \| array[buffer] | Значение в hex-кодировке для хеширования |

### Выходные данные

| Имя     | Тип    | Описание              |
| ------- | ------ | --------------------- |
| `value` | string | Результат хеширования |

```hcl
output "hashed_data" {
    value = ripemd160(encode_hex("hello, world"))
}
// > hashed_data: 0XA3201F82FCA034E46D10CD7B27E174976E241DA2
```

## sha256

`sha256` вычисляет хеш sha256 для заданного значения.

### Входные данные

| Имя     | Тип                     | Описание                                 |
| ------- | ----------------------- | ---------------------------------------- |
| `value` | buffer \| array[buffer] | Значение в hex-кодировке для хеширования |

### Выходные данные

| Имя     | Тип    | Описание              |
| ------- | ------ | --------------------- |
| `value` | string | Результат хеширования |

```hcl
output "hashed_data" {
    value = sha256(encode_hex("hello, world"))
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```

### Входные данные

| Имя     | Тип    | Описание                           |
| ------- | ------ | ---------------------------------- |
| `value` | string | Строковое значение для хеширования |

### Выходные данные

| Имя     | Тип    | Описание              |
| ------- | ------ | --------------------- |
| `value` | string | Результат хеширования |

```hcl
output "hashed_data" {
    value = keccak256("hello, world")
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```
