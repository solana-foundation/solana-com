CLI Surfpool можно установить с помощью скрипта установки или собрать из
исходного кода.

## Скрипт установки

Чтобы установить surfpool, откройте терминал и выполните:

```bash
curl -sL https://run.surfpool.run/ | bash
```

## Установка из исходного кода

Чтобы скомпилировать бинарный файл Surfpool из исходного кода, клонируйте
репозиторий surfpool.

```bash
git clone https://github.com/solana-foundation/surfpool.git
```

Перейдите в новый каталог.

```bash
cd surfpool
```

Затем скомпилируйте бинарный файл. Эта команда скомпилирует бинарный файл и
сохранит его в `~/.cargo/bin/surfpool`.

```bash
cargo surfpool-install
```

Наконец, убедитесь, что бинарный файл `surfpool` доступен в вашем PATH. Этот
процесс будет отличаться в зависимости от вашей операционной системы.
