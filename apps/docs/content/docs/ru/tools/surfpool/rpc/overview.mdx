Surfnet — это локальный симулятор сети Solana от Surfpool, разработанный для
того, чтобы предоставить разработчикам мощную среду тестирования,
воспроизводящую поведение блокчейна Solana без необходимости запускать
полноценный узел validator.

## Начало работы

Запустите экземпляр Surfnet с помощью Surfpool CLI:

```bash
surfpool start
```

Это запустит локальный экземпляр Surfnet на `http://localhost:8899` со всеми
доступными конечными точками RPC.

## Ключевые возможности

- **Полная совместимость с RPC**: поддерживает полный RPC API Solana
- **Быстрая итерация**: не нужно ждать подтверждения блоков
- **Управление состоянием**: чит-коды для прямого изменения состояния блокчейна
- **Форк мейннета**: форк из мейннета для тестирования с реальными аккаунтами и
  программами
- **Поддержка плагина Geyser**: подключайте ваши плагины Geyser для потоковой
  передачи данных в реальном времени
