Крейт `surfpool-sdk` предоставляет небольшую поверхность, центром которой
является структура `Surfnet`. На этой странице перечислены все публичные
элементы; для ознакомления с практическим использованием см. руководства,
начиная с [Обзора](/docs/tools/surfpool/sdk/overview).

## Корневые экспорты крейта

| Элемент               | Тип        | Примечания                                                             |
| --------------------- | ---------- | ---------------------------------------------------------------------- |
| `Surfnet`             | struct     | Запущенный экземпляр Surfnet — RPC/WS эндпоинты, плательщик, читкоды   |
| `SurfnetBuilder`      | struct     | Fluent builder, используемый в `Surfnet::builder()`                    |
| `Cheatcodes<'_>`      | struct     | Заимствованный помощник читкодов, возвращаемый `Surfnet::cheatcodes()` |
| `SurfnetError`        | enum       | Все варианты ошибок, возвращаемых SDK                                  |
| `SurfnetResult<T>`    | type alias | `Result<T, SurfnetError>`                                              |
| `Keypair`             | re-export  | `solana_keypair::Keypair`                                              |
| `Pubkey`              | re-export  | `solana_pubkey::Pubkey`                                                |
| `Signer`              | re-export  | `solana_signer::Signer`                                                |
| `RpcClient`           | re-export  | `solana_rpc_client::rpc_client::RpcClient`                             |
| `BlockProductionMode` | re-export  | `surfpool_types::BlockProductionMode`                                  |
| `SimnetCommand`       | re-export  | `surfpool_types::SimnetCommand`                                        |
| `SimnetEvent`         | re-export  | `surfpool_types::SimnetEvent`                                          |
| `SvmFeatureConfig`    | re-export  | `surfpool_types::SvmFeatureConfig`                                     |

### Жизненный цикл

```rust
impl Surfnet {
    pub fn stop(&mut self) -> SurfnetResult<()>;
}

impl Drop for Surfnet { /* calls stop() if not already stopped */ }
```

`stop` ожидает до пяти секунд, пока среда выполнения не освободит свои порты.
Сбой во время `Drop` происходит беззвучно; сбой во время явного вызова `stop`
возвращает `SurfnetError::Runtime`.


Каждый сеттер возвращает `Self`, что позволяет выстраивать цепочки вызовов.

## SurfnetError

```rust
#[derive(Debug)]
pub enum SurfnetError {
    PortAllocation(String),
    Startup(String),
    Runtime(String),
    Cheatcode(String),
    Aborted(String),
}
```

| Вариант          | Причина                                                                    |
| ---------------- | -------------------------------------------------------------------------- |
| `PortAllocation` | Не удалось привязаться к свободному порту для RPC или WebSocket.           |
| `Startup`        | Не удалось инициализировать среду выполнения.                              |
| `Runtime`        | Поток среды выполнения завершился аварийно или неожиданно завершил работу. |
| `Cheatcode`      | Вызов чит-кода JSON-RPC вернул ошибку.                                     |
| `Aborted`        | Среда выполнения была остановлена или отменена в процессе запуска.         |

Крейт реализует `std::error::Error` и `Display` для `SurfnetError`, поэтому он
работает с `?` и `anyhow` из коробки.
