---
title: Читкоды
description:
  Пропускайте транзакции и напрямую изменяйте состояние Surfnet. Пополняйте SOL
  и токены, задавайте произвольные данные аккаунта, сбрасывайте кешированные
  upstream-аккаунты и транслируйте живые аккаунты mainnet в реальном времени.
---

Читкоды — это мутации состояния, которые обходят стандартный поток транзакций.
Они выполняются мгновенно, не потребляя blockhash и не требуя оплаты комиссий —
именно то, что нужно для подготовки тестовой среды.

В Rust читкоды находятся в `Surfnet::cheatcodes()`. В JS они являются методами
непосредственно на экземпляре `Surfnet`.

## Пополнение SOL

`fundSol` / `fund_sol` устанавливает баланс в lamport для любого аккаунта,
создавая аккаунт, если он не существует.

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::{Pubkey, Surfnet};

let surfnet = Surfnet::start().await?;
let cheats = surfnet.cheatcodes();
let wallet = Pubkey::new_unique();

cheats.fund_sol(&wallet, 1_000_000_000)?;

// Fund several accounts in one call.
let bob = Pubkey::new_unique();
let carol = Pubkey::new_unique();
cheats.fund_sol_many(&[(&bob, 2_000_000_000), (&carol, 3_000_000_000)])?;
```

```ts !! title="TypeScript"
import { Surfnet } from "@solana/surfpool";

const surfnet = Surfnet.start();
const wallet = Surfnet.newKeypair();

surfnet.fundSol(wallet.publicKey, 1_000_000_000);

const bob = Surfnet.newKeypair();
const carol = Surfnet.newKeypair();
surfnet.fundSolMany([
  { address: bob.publicKey, lamports: 2_000_000_000 },
  { address: carol.publicKey, lamports: 3_000_000_000 }
]);
```

</CodeTabs>

## Пополнение токенов

`fundToken` / `fund_token` выпускает токены на кошелёк, вычисляя associated
token account, создавая его при необходимости и устанавливая сумму.

<Callout type="info" title="Минты Token-2022">
  По умолчанию `fundToken` использует классический SPL Token program. Для минтов
  Token-2022 передайте ID программы Token-2022
  (`TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb`) в качестве последнего
  аргумента.
</Callout>

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::{Pubkey, Surfnet};

let surfnet = Surfnet::start().await?;
let cheats = surfnet.cheatcodes();

let wallet = Pubkey::new_unique();
let mint: Pubkey = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v"
    .parse()
    .unwrap();

// Classic SPL Token mint.
cheats.fund_token(&wallet, &mint, 5_000_000, None)?;

// Token-2022 mint — pass the program id explicitly.
let token_2022: Pubkey = "TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb"
    .parse()
    .unwrap();
cheats.fund_token(&wallet, &mint, 5_000_000, Some(&token_2022))?;

// Fund many wallets with the same mint.
let mob = vec![Pubkey::new_unique(), Pubkey::new_unique()];
cheats.fund_token_many(&mob.iter().collect::<Vec<_>>(), &mint, 1_000_000, None)?;
```

```ts !! title="TypeScript"
const surfnet = Surfnet.start();
const wallet = Surfnet.newKeypair();
const USDC = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v";

surfnet.fundToken(wallet.publicKey, USDC, 5_000_000);

// Token-2022 mint — pass the program id explicitly.
const TOKEN_2022 = "TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb";
surfnet.fundToken(wallet.publicKey, USDC, 5_000_000, TOKEN_2022);

// Fund many wallets at once.
const mob = [Surfnet.newKeypair().publicKey, Surfnet.newKeypair().publicKey];
surfnet.fundTokenMany(mob, USDC, 1_000_000);
```

</CodeTabs>

Чтобы получить ATA без пополнения, используйте `getAta` / `get_ata`:

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
let ata = cheats.get_ata(&wallet, &mint, None);
```

```ts !! title="TypeScript"
const ata = surfnet.getAta(wallet.publicKey, USDC);
```

</CodeTabs>

## Установка произвольного состояния аккаунта

Метод `setAccount` записывает lamport, владельца и необработанные байты данных
для любого аккаунта за один вызов. Rust SDK дополнительно предоставляет билдер
`SetAccount` для более сложных полей, таких как `rent_epoch` и `executable`.

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::cheatcodes::builders::SetAccount;
use surfpool_sdk::{Pubkey, Surfnet};

let surfnet = Surfnet::start().await?;
let cheats = surfnet.cheatcodes();

let address = Pubkey::new_unique();
let owner = Pubkey::new_unique();

// Direct method.
cheats.set_account(&address, 500_000, &[1, 2, 3], &owner)?;

// Builder when you need finer control.
cheats.execute(
    SetAccount::new(address)
        .lamports(500_000)
        .owner(owner)
        .data(vec![1, 2, 3])
        .rent_epoch(0)
        .executable(false),
)?;
```

```ts !! title="TypeScript"
const surfnet = Surfnet.start();
const address = Surfnet.newKeypair();
const owner = Surfnet.newKeypair();

surfnet.setAccount(
  address.publicKey,
  500_000,
  new Uint8Array([1, 2, 3]),
  owner.publicKey
);
```

</CodeTabs>

## Изменение полей token account

`setTokenAccount` обновляет расширенные поля существующего token account —
делегат, состояние, полномочия на закрытие, делегированный объём. Используйте
его, когда одного `fundToken` недостаточно.

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::cheatcodes::builders::SetTokenAccount;
use surfpool_sdk::{Pubkey, Surfnet};

let cheats = Surfnet::start().await?.cheatcodes();

let owner = Pubkey::new_unique();
let mint: Pubkey = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v".parse()?;
let delegate = Pubkey::new_unique();

cheats.execute(
    SetTokenAccount::new(owner, mint)
        .amount(2_000_000)
        .delegate(delegate)
        .delegated_amount(500_000)
        .state("initialized"),
)?;

// Later: clear the delegation.
cheats.execute(SetTokenAccount::new(owner, mint).clear_delegate())?;
```

```ts !! title="TypeScript"
const surfnet = Surfnet.start();
const owner = Surfnet.newKeypair();
const USDC = "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v";
const delegate = Surfnet.newKeypair();

surfnet.setTokenAccount(owner.publicKey, USDC, {
  amount: 2_000_000,
  delegate: delegate.publicKey,
  delegatedAmount: 500_000,
  state: "initialized"
});

// Later: clear the delegation.
surfnet.setTokenAccount(owner.publicKey, USDC, { clearDelegate: true });
```

</CodeTabs>

## Сброс аккаунтов к состоянию upstream

`resetAccount` отменяет все локальные изменения и повторно получает аккаунт из
upstream RPC. Передайте `includeOwnedAccounts: true`, чтобы также сбросить
каждый аккаунт, принадлежащий целевому объекту, — это удобно при сбросе
программы и всех её PDA.

<Callout
  type="info"
  title="Требуется `remoteRpcUrl` для повторного получения данных"
>
  На офлайн-Surfnet `resetAccount` очищает локальную копию, но не имеет upstream
  для получения данных. Настройте `remoteRpcUrl` при запуске, чтобы сброс
  восстанавливал состояние upstream.
</Callout>

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::cheatcodes::builders::ResetAccount;
use surfpool_sdk::{Pubkey, Surfnet};

let cheats = Surfnet::start().await?.cheatcodes();
let token_program: Pubkey = "TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA".parse()?;

cheats.execute(
    ResetAccount::new(token_program).include_owned_accounts(true),
)?;
```

```ts !! title="TypeScript"
const surfnet = Surfnet.start();
const TOKEN_PROGRAM = "TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA";

surfnet.resetAccount(TOKEN_PROGRAM, { includeOwnedAccounts: true });
```

</CodeTabs>

## Потоковая передача аккаунтов в реальном времени

`streamAccount` регистрирует аккаунт для фонового опроса из upstream RPC.
Локальная копия в Surfnet остаётся синхронизированной с mainnet без
необходимости явного опроса в тесте. Используйте в сочетании с
`includeOwnedAccounts` для потоковой передачи всех PDA, принадлежащих программе.

<Callout type="warn" title="Требуется `remoteRpcUrl`">
  `streamAccount` работает только на Surfnet, запущенном с `remoteRpcUrl` (или
  `remote_rpc_url` в Rust). Без upstream RPC передавать данные неоткуда, и вызов
  вернёт ошибку.
</Callout>

<CodeTabs storage="surfpool-lang">

```rust !! title="Rust"
use surfpool_sdk::cheatcodes::builders::StreamAccount;
use surfpool_sdk::{Pubkey, Surfnet};

let cheats = Surfnet::builder()
    .remote_rpc_url("https://api.mainnet-beta.solana.com")
    .start()
    .await?
    .cheatcodes();

let oracle: Pubkey = "H6ARHf6YXhGYeQfUzQNGk6rDNnLBQKrenN712K4AQJEG".parse()?;

cheats.execute(StreamAccount::new(oracle).include_owned_accounts(false))?;
```

```ts !! title="TypeScript"
const surfnet = Surfnet.startWithConfig({
  remoteRpcUrl: "https://api.mainnet-beta.solana.com"
});

surfnet.streamAccount("H6ARHf6YXhGYeQfUzQNGk6rDNnLBQKrenN712K4AQJEG");
```

</CodeTabs>

## Построители чит-кодов (Rust)

Rust SDK предоставляет типизированные построители в
`surfpool_sdk::cheatcodes::builders` для случаев, когда удобных методов
недостаточно. Каждый построитель реализует трейт `CheatcodeBuilder` и
выполняется через `cheats.execute(builder)`.

| Строитель         | Конструктор                                          | Общие сеттеры                                                                                                                    |
| ----------------- | ---------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| `SetAccount`      | `new(address: Pubkey)`                               | `lamports`, `data`, `owner`, `rent_epoch`, `executable`                                                                          |
| `SetTokenAccount` | `new(owner: Pubkey, mint: Pubkey)`                   | `amount`, `delegate`, `clear_delegate`, `state`, `delegated_amount`, `close_authority`, `clear_close_authority`, `token_program` |
| `ResetAccount`    | `new(address: Pubkey)`                               | `include_owned_accounts(bool)`                                                                                                   |
| `StreamAccount`   | `new(address: Pubkey)`                               | `include_owned_accounts(bool)`                                                                                                   |
| `DeployProgram`   | `new(program_id: Pubkey)`, `from_keypair_path(path)` | `so_path`, `so_bytes`, `idl_path`                                                                                                |

Для полного описания JS API см.
[Справочник по JS](/docs/tools/surfpool/sdk/js-reference). Для полного описания
Rust API см. [Справочник по Rust](/docs/tools/surfpool/sdk/rust-reference).
