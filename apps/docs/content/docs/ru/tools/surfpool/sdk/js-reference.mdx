Пакет `@solana/surfpool` предоставляет один класс — `Surfnet` — а также
вспомогательные типы, необходимые для типобезопасного вызова его методов. На
этой странице перечислены все экспортируемые символы; для описания сценариев
использования обратитесь к руководствам, начиная с
[Обзора](/docs/tools/surfpool/sdk/overview).

## Экспорты пакета

| Символ                  | Тип   | Примечания                                                |
| ----------------------- | ----- | --------------------------------------------------------- |
| `Surfnet`               | класс | Запущенный экземпляр Surfnet с методами-читкодами         |
| `SurfnetConfig`         | тип   | Аргумент для `Surfnet.startWithConfig`                    |
| `DeployOptions`         | тип   | Аргумент для `Surfnet.deploy`                             |
| `SetTokenAccountUpdate` | тип   | Аргумент для `Surfnet.setTokenAccount`                    |
| `ResetAccountOptions`   | тип   | Аргумент для `Surfnet.resetAccount`                       |
| `StreamAccountOptions`  | тип   | Аргумент для `Surfnet.streamAccount`                      |
| `SolAccountFunding`     | тип   | Элемент массива, передаваемого в `Surfnet.fundSolMany`    |
| `SimnetEventValue`      | тип   | Элемент, возвращаемый `Surfnet.drainEvents`               |
| `KeypairInfo`           | тип   | Возвращается `Surfnet.newKeypair`                         |
| `ClockValue`            | тип   | Поле объекта `SimnetEventValue`                           |
| `EpochInfoValue`        | тип   | Возвращается вспомогательными функциями перемотки времени |
| `ByteArrayLike`         | тип   | `Uint8Array \| number[]` — гибкий ввод байтов             |

## Класс Surfnet

### Статические методы

```ts
class Surfnet {
  static start(): Surfnet;
  static startWithConfig(config: SurfnetConfig): Surfnet;
  static newKeypair(): KeypairInfo;
}
```

### Свойства экземпляра

```ts
class Surfnet {
  readonly rpcUrl: string; // http://127.0.0.1:<port>
  readonly wsUrl: string; // ws://127.0.0.1:<port>
  readonly payer: string; // base58 public key
  readonly payerSecretKey: Uint8Array; // 64-byte private key
  readonly instanceId: string;
}
```

### Жизненный цикл

```ts
class Surfnet {
  stop(): void; // idempotent; safe to call from `finally` / test teardown
}
```

### Читкоды SOL

```ts
class Surfnet {
  fundSol(address: string, lamports: number): void;
  fundSolMany(accounts: SolAccountFunding[]): void;
  setAccount(
    address: string,
    lamports: number,
    data: Uint8Array,
    owner: string
  ): void;
  resetAccount(address: string, options?: ResetAccountOptions): void;
}
```

### Читкоды токенов

```ts
class Surfnet {
  fundToken(
    owner: string,
    mint: string,
    amount: number,
    tokenProgram?: string
  ): void;
  fundTokenMany(
    owners: string[],
    mint: string,
    amount: number,
    tokenProgram?: string
  ): void;
  setTokenBalance(
    owner: string,
    mint: string,
    amount: number,
    tokenProgram?: string
  ): void;
  setTokenAccount(
    owner: string,
    mint: string,
    update: SetTokenAccountUpdate,
    tokenProgram?: string
  ): void;
  getAta(owner: string, mint: string, tokenProgram?: string): string;
}
```

### Перемотка времени

```ts
class Surfnet {
  timeTravelToSlot(slot: number): EpochInfoValue;
  timeTravelToEpoch(epoch: number): EpochInfoValue;
  timeTravelToTimestamp(timestampMs: number): EpochInfoValue;
}
```

### Развёртывание программ

```ts
class Surfnet {
  deployProgram(programName: string): string; // returns program id
  deploy(options: DeployOptions): string; // returns program id
}
```

### Стриминг и события

```ts
class Surfnet {
  streamAccount(address: string, options?: StreamAccountOptions): void;
  drainEvents(): SimnetEventValue[];
}
```

## Типы конфигурации

### SurfnetConfig

```ts
interface SurfnetConfig {
  offline?: boolean; // default: true
  remoteRpcUrl?: string; // setting this implies offline=false
  blockProductionMode?: "manual" | "clock" | "transaction"; // default: "transaction"
  slotTimeMs?: number; // default: 1
  airdropSol?: number; // lamports; default: 10_000_000_000
  airdropAddresses?: string[];
  payerSecretKey?: ByteArrayLike;
  enableFeatures?: string[]; // feature IDs (base58 or kebab name)
  disableFeatures?: string[];
  allFeatures?: boolean;
}
```

### DeployOptions

```ts
interface DeployOptions {
  programId: string; // base58 address
  soPath?: string; // mutually exclusive with soBytes
  soBytes?: ByteArrayLike;
  idlPath?: string; // optional Anchor IDL JSON
}
```

### SetTokenAccountUpdate

```ts
interface SetTokenAccountUpdate {
  amount?: number;
  delegate?: string; // base58 pubkey
  clearDelegate?: boolean;
  state?: string; // e.g. "initialized", "frozen"
  delegatedAmount?: number;
  closeAuthority?: string;
  clearCloseAuthority?: boolean;
}
```

### ResetAccountOptions

```ts
interface ResetAccountOptions {
  includeOwnedAccounts?: boolean;
}
```

### StreamAccountOptions

```ts
interface StreamAccountOptions {
  includeOwnedAccounts?: boolean;
}
```

### SolAccountFunding

```ts
interface SolAccountFunding {
  address: string;
  lamports: number;
}
```

### KeypairInfo

```ts
interface KeypairInfo {
  publicKey: string; // base58
  secretKey: number[]; // 64 bytes
}
```

### ByteArrayLike

```ts
type ByteArrayLike = Uint8Array | number[];
```

Там, где метод принимает `ByteArrayLike`, вы можете передать либо `Uint8Array`,
либо обычный массив чисел. Обёртка нормализует оба варианта в формат, ожидаемый
нативным биндингом.

## Типы событий

### SimnetEventValue

Плоский объект с дискриминатором `kind` и необязательными полями. Каталог
вариантов см. в разделе [События](/docs/tools/surfpool/sdk/events).

```ts
interface SimnetEventValue {
  kind: string;
  message?: string;
  timestamp?: string;
  initialTransactionCount?: number;
  clock?: ClockValue;
  epochInfo?: EpochInfoValue;
  accountPubkey?: string;
  clockCommand?: string;
  slotIntervalMs?: number;
  transactionSignature?: string;
  logs?: string[];
  computeUnitsConsumed?: number;
  fee?: number;
  errorMessage?: string;
  tag?: string;
  profileKey?: string;
  profileSlot?: number;
  runbookId?: string;
  runbookErrors?: string[];
}
```

### ClockValue

```ts
interface ClockValue {
  slot: number;
  epoch: number;
  leaderScheduleEpoch: number;
  unixTimestamp: number;
  epochStartTimestamp: number;
}
```

### EpochInfoValue

```ts
interface EpochInfoValue {
  absoluteSlot: number;
  slotIndex: number;
  slotsInEpoch: number;

  blockHeight: number;
  transactionCount?: number;
}
```

## Ошибки

Каждый метод выбрасывает стандартный `Error`, сообщение которого оборачивает
исходную Rust-ошибку `SurfnetError`. Используйте `error.message` для различения
случаев.

```ts
try {
  surfnet.deploy({ programId, soPath: "missing.so" });
} catch (err) {
  // err.message starts with "Cheatcode:" / "Startup:" / etc.
  console.error(err);
}
```

## Размер нативного модуля

Пакет зависит от платформо-зависимых нативных бинарных файлов, опубликованных
как `optionalDependencies`:

| Тройка                     | Пакет                            |
| -------------------------- | -------------------------------- |
| `aarch64-apple-darwin`     | `@solana/surfpool-darwin-arm64`  |
| `x86_64-apple-darwin`      | `@solana/surfpool-darwin-x64`    |
| `x86_64-unknown-linux-gnu` | `@solana/surfpool-linux-x64-gnu` |

Если ни один из платформенных пакетов не устанавливается на вашем компьютере,
`Surfnet.start()` выдаёт при первом вызове понятную ошибку «нативный модуль не
найден». Смотрите [Установку](/docs/tools/surfpool/sdk/installation) для
получения информации о матрице поддержки и устранении неполадок.
