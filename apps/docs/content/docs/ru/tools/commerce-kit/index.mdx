**Commerce Kit — это набор инструментов для создания коммерческих решений на
Solana.** От готовых платежных кнопок до headless-примитивов, Commerce Kit
предоставляет всё необходимое для приёма криптоплатежей.

## Почему Commerce Kit?

- **Готовые компоненты**: Предварительно собранный PaymentButton с подключением
  кошелька, выбором токена и обработкой транзакций
- **Независимость от фреймворков**: Используйте headless-примитивы в любом
  JavaScript-фреймворке
- **Solana Pay**: Полная поддержка Solana Pay с генерацией QR-кодов
- **Типобезопасность**: Полная поддержка TypeScript во всех пакетах
- **Wallet Standard**: Совместимость со всеми кошельками стандарта Wallet
  Standard (Phantom, Solflare, Backpack и др.)

## Быстрый старт

Самый быстрый способ начать — использовать компонент `PaymentButton`:

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## Установка

Выберите необходимые пакеты Commerce Kit:

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
