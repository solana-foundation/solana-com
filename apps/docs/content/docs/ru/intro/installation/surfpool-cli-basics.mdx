Surfpool — это инструмент для локальной разработки, который упрощает разработку
на Solana, автоматически загружая программы и аккаунты из кластера (mainnet,
devnet). Используйте Surfpool CLI для запуска локального валидатора, чтобы
тестировать свои программы и транзакции.

Посетите [документацию Surfpool](https://docs.surfpool.run/) для получения
дополнительной информации.

## Запуск локального валидатора

Чтобы запустить локальный валидатор, выполните следующую команду:

```terminal
$ surfpool start
```

По умолчанию Surfpool автоматически загружает любые программы и аккаунты, от
которых зависят ваши транзакции, из mainnet в локальный валидатор.

## Surfpool Studio

Surfpool Studio предоставляет веб-интерфейс для взаимодействия с вашим локальным
валидатором. Используйте его для:

- Просмотра деталей транзакций
- Изучения данных аккаунтов
- Получения SOL и других токенов через UI-кран

Чтобы получить доступ к Surfpool Studio:

1. Запустите локальный валидатор с помощью `surfpool start`
2. Откройте браузер и перейдите по адресу
   [http://127.0.0.1:18488](http://127.0.0.1:18488)
