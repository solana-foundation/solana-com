Solana — это высокопроизводительный блокчейн с низкими комиссиями и быстрыми
транзакциями, вокруг которого сформировалось отличное сообщество разработчиков
игр.

Здесь вы найдете информацию о доступных в экосистеме SDK для разработки игр на
Solana. Все SDK являются open source и созданы сообществом. Если вы работаете
над собственным SDK, не стесняйтесь открыть PR на этой странице и добавить его.

## Unity SDK

Игровой движок Unity известен своим дружелюбным подходом для начинающих и
поддержкой кроссплатформенности, включая WebGL, iOS и Android. Создавайте один
раз, экспортируйте везде.

- [Документация](https://docs.magicblock.gg/introduction)
- [Исходный код](https://github.com/magicblock-labs/Solana.Unity-SDK)
- [Примеры игр](https://github.com/solana-developers/solana-game-examples)
- [Крестики-нолики](https://blog.magicblock.gg/bolt-tic-tac-toe/)

Unity SDK для Solana поддерживается [Magicblock](https://www.magicblock.gg/) и
включает:

- Поддержку NFT
- Интеграцию с Candy Machine
- Транзакции
- RPC-функции
- Глубокие ссылки Phantom
- Коннектор WebGL
- Надежную поддержку WebSocket-соединений
- Адаптер мобильного кошелька
- Ключи сессий для автоматического одобрения транзакций
- Генерацию клиентского кода для Anchor и многое другое.

## Godot SDK

[Движок Godot](https://godotengine.org/) — это open source игровой движок,
который получил большую поддержку за последние несколько лет. SDK для Solana в
Godot поддерживается [ZenRepublic](https://twitter.com/ZenRepublicNDM) и
[Virus-Axel](https://twitter.com/AxelBenjam). Он включает интеграцию с адаптером
кошелька, транзакции, RPC-функции и генерацию клиентского кода для Anchor. Также
доступна полностью функциональная интеграция Metaplex
[candy machine](https://zenwiki.gitbook.io/solana-godot-sdk-docs/guides/setup-candy-machine).

- [Solana Godot Engine SDK](https://github.com/Virus-Axel/godot-solana-sdk)
- [Документация](https://zenwiki.gitbook.io/solana-godot-sdk-docs)
- [Демо](https://github.com/ZenRepublic/GodotSolanaSDKDemoPackage)
- [Учебник](https://www.youtube.com/watch?v=tszFPInYmXQ)
- [Поддержка](https://twitter.com/ZenRepublicNDM)

## Turbo.Computer - игровой движок на Rust

Turbo — это игровой движок на языке Rust, созданный с нуля с акцентом на Solana.
Он обладает легковесной архитектурой и обеспечивает быструю итерацию. Движок
подходит для начинающих и полностью поддерживает RPC Solana. Вы даже можете
использовать его инструменты ИИ для создания полноценных игр.

- [Веб-сайт](https://turbo.computer/)
- [Документация](https://docs.turbo.computer/)
- [Аккаунт в Twitter](https://twitter.com/makegamesfast)
- [Мейнтейнер](https://twitter.com/jozanza)

# Протокол Honeycomb

Honeycomb предоставляет набор ончейн-программ и инструментов для сжатия
состояния, которые обрабатывают все ключевые функции жизненного цикла игры и
обеспечивают совместимость внутри виртуальной машины Solana (SVM). Он
поддерживает множество функций NFT, таких как стейкинг, миссии, лутбоксы,
профили игроков, сжатие состояния, автоматическое одобрение транзакций и многое
другое.

- [Аккаунт в Twitter](https://twitter.com/honeycomb_prtcl)
- [Документация](https://docs.honeycombprotocol.com/)

# SDK для Unreal

Unreal Engine известен своими великолепными визуальными эффектами и фреймворком
сценариев на основе узлов. Существует несколько SDK для Solana, поддерживаемых
различными компаниями.

### Varmeta Unreal SDK

VAR META предоставляет открытый SDK для Unreal, который интегрирует
функциональность блокчейна Solana в проекты Unreal Engine. Он позволяет
разработчикам взаимодействовать с умными контрактами Solana, управлять
кошельками и выполнять транзакции непосредственно в Unreal Engine.

- [Unreal SDK от Varmeta](https://github.com/VAR-META-Tech/UnrealSolanaSDK)
- [Мейнтейнер](https://www.var-meta.com/)

### Bitfrost Unreal SDK

Bitfrost работает над SDK для Unreal, который уже был использован в нескольких
играх, созданных на недавних Solana game jams. В комплекте поддержка C# `solnet`
в C++ и Blueprints, минтинг Metaplex NFT, примеры интеграции платежного
процессора в игровой кошелёк и многое другое.

- [Solana Unreal SDK от Bitfrost](https://github.com/Bifrost-Technologies/Solana-Unreal-SDK)
- [Учебное пособие](https://www.youtube.com/watch?v=S8fm8mFeUkk)
- [Мейнтейнер](https://twitter.com/BifrostTitan)

### Thugz Unreal SDK

Thugz — это технологическая студия и проект NFT, который также поддерживает
открытый SDK для Unreal. Он включает множество функций, ориентированных на NFT,
а плагин совместим с iOS и Android.

- [Плагин Thugz Blockchain](https://www.fab.com/listings/a7e220ba-93e5-4a3b-99e7-17b13da76f05)
- [Репозиторий](https://github.com/ThugzLabs/Thugz-BC-Plugin-Packaged-for-UE5.4)
- [Документация](https://thugzblockchainplugin-docs.vercel.app/)
- [Видеоурок](https://www.youtube.com/watch?v=dS7sTZd_E9U&ab_channel=ThugzNFT)

### Набор инструментов Star Atlas Foundation

Команда [Star Atlas](https://staratlas.com/) открыла исходный код части своего

## Next.js/React + Anchor

Один из самых простых способов разработки на Solana — использование
JavaScript-фреймворка Web3js в сочетании с фреймворком Solana Anchor. Для более
сложных игр мы рекомендуем рассмотреть использование игровых движков, таких как
[Unity](#unity-sdk) или [Unreal](#unreal-sdks).

Самый быстрый способ настроить игру на основе Next.js:

```shell
npx create-solana-game your-game-name
```

Это сгенерирует отличное стартовое приложение с поддержкой `wallet-adapter`,
Anchor-программой, React-приложением и Unity-клиентом, уже настроенными для
совместной работы. Преимущество использования Next.js в том, что один и тот же
код можно использовать и на бэкенде, и на фронтенде, что ускоряет разработку.

- [Web3.js](https://solana.com/docs/clients/javascript)
- [Видеоурок](https://www.youtube.com/watch?v=fnhivg_pemI&t=1s&ab_channel=Solana)

## Phaser

Быстрый, бесплатный и увлекательный фреймворк с открытым исходным кодом для
браузерных игр на основе Canvas и WebGL. [Phaser](https://phaser.io/) — отличный
способ начать разработку игр на Solana. Шаблон Phaser Solana Platformer Template
— это отличная отправная точка для вашей игры.

- [Исходный код](https://github.com/Bread-Heads-NFT/solana-phaser-template)
- [Поддержка](https://twitter.com/blockiosaurus)

## Flutter

Flutter — это фреймворк с открытым исходным кодом, поддерживаемый Google, для
создания красивых, нативно компилируемых многоплатформенных приложений из единой
кодовой базы. SDK для Solana на Flutter поддерживается командой
[Espresso Cash](https://www.espressocash.com/).

- [Исходный код](https://github.com/espresso-cash/espresso-cash-public)

## Python

Python — это простой в изучении и широко используемый язык программирования
(часто применяется в приложениях машинного обучения). Фреймворк Seahorse
позволяет разработчикам писать программы для Solana на Python. Seahorse построен
поверх фреймворка Anchor и преобразует код на Python в код на Rust, основанный
на Anchor. В настоящее время Seahorse находится в стадии бета-тестирования.

- [Документация](https://www.seahorse.dev/)
- [Репозиторий на Github](https://github.com/solana-developers/seahorse)
- [Пример в Anchor Playground](https://beta.solpg.io/tutorials/hello-seahorse)

## Родной C#

Оригинальный порт Web3js на C#. Он включает множество функций, таких как
транзакции, RPC-функции и генерация клиентского кода для Anchor.

- [Исходный код и документация](https://github.com/bmresearch/Solnet/blob/master/docs/articles/getting_started.md)
