## Solana Kit

`@solana/kit` — рекомендуемый TypeScript SDK для разработки на Solana. Вы
собираете клиент из компонуемых плагинов, добавляете React-биндинги по мере
необходимости и подключаете сгенерированные программные клиенты для вызываемых
on-chain программ.

| Пакет                     | Описание                                                  | GitHub                                                                                       |
| ------------------------- | --------------------------------------------------------- | -------------------------------------------------------------------------------------------- |
| @solana/kit               | Основной SDK: RPC, транзакции, кодеки, подписанты         | [Исходный код](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | React-биндинги: `ClientProvider` и хуки                   | [Исходный код](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | RPC-подключение + отправка транзакций                     | [Исходный код](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | Плагины для плательщика комиссий, идентификации и подписи | [Исходный код](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Подключение Wallet Standard + React-хуки                  | [Исходный код](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

Для совместимого с kit подписания на бэкенде с поддержкой нескольких систем
управления ключами:

| Пакет            | Описание                                                                                                                          | GitHub                                                               |
| ---------------- | --------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------- |
| @solana/keychain | Унифицированная подпись: Memory, Vault, Privy, Turnkey, AWS KMS, Fireblocks, GCP KMS, CDP, Para, Dfns, Crossmint, Openfort, Utila | [Исходный код](https://github.com/solana-foundation/solana-keychain) |

Смотрите [документацию Keychain](/docs/tools/keychain) и

настройке и модели хранения.

## Solana Web3.js

`@solana/web3.js` — это устаревший TypeScript SDK для Solana. Новым приложениям
следует использовать `@solana/kit`; см.
[Миграция на Kit](/docs/frontend/web3-compat).

| Package           | Description                                                    | GitHub                                                                             |
| ----------------- | -------------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| @solana/web3.js   | Основной SDK                                                   | [Source](https://github.com/solana-foundation/solana-web3.js)                      |
| @solana/spl-token | Взаимодействие с Token, Token-2022 и Associated Token programs | [Source](https://github.com/solana-program/token-2022/tree/main/clients/js-legacy) |
| @solana/spl-memo  | Взаимодействие с программой Memo                               | [Source](https://github.com/solana-program/memo/tree/main/clients/js-legacy)       |
