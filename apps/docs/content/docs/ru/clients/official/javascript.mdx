## Solana Client

`@solana/client` — это безинтерфейсный рантайм Solana, который объединяет RPC,
кошельки, транзакции и подписки в одном хранилище.

| Пакет          | Описание                | GitHub                                                                                 |
| -------------- | ----------------------- | -------------------------------------------------------------------------------------- |
| @solana/client | Безинтерфейсный рантайм | [Source](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [Гайд: обзор @solana/client](/docs/frontend/client)

## Solana React Hooks

`@solana/react-hooks` добавляет React-провайдер и хуки поверх `@solana/client`,
используя тот же рантайм и кэш.

| Пакет               | Описание                      | GitHub                                                                                      |
| ------------------- | ----------------------------- | ------------------------------------------------------------------------------------------- |
| @solana/react-hooks | React-провайдер + хуки для UI | [Source](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [Гайд: @solana/react-hooks](/docs/frontend/react-hooks)

## Solana Kit

`@solana/kit` — это рекомендуемый TypeScript SDK для разработки на Solana.

| Пакет                          | Описание                                   | GitHub                                                                                |
| ------------------------------ | ------------------------------------------ | ------------------------------------------------------------------------------------- |
| @solana/kit                    | Основной SDK                               | [Исходный код](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | Взаимодействие с System Program            | [Исходный код](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Взаимодействие с Token Program             | [Исходный код](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Взаимодействие с программой Token-2022     | [Исходный код](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Взаимодействие с программой Memo           | [Исходный код](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Взаимодействие с программой Compute Budget | [Исходный код](https://github.com/solana-program/compute-budget/tree/main/clients/js) |
| @solana/kit-plugins            | Плагины для Kit                            | [Исходный код](https://github.com/anza-xyz/kit-plugins)                               |

- [Официальная документация Kit](https://www.solanakit.com/)
- [Туториал: Next.js + React Hooks](/docs/frontend/nextjs-solana)

## Подписание и управление ключами

Для совместимого с kit подписания на бэкенде с использованием нескольких систем
управления ключами:

| Пакет            | Описание                                                                                        | GitHub                                                               |
| ---------------- | ----------------------------------------------------------------------------------------------- | -------------------------------------------------------------------- |
| @solana/keychain | Унифицированное подписание: Memory, Vault, AWS KMS, Privy, Turnkey, Fireblocks, CDP, Dfns, Para | [Исходный код](https://github.com/solana-foundation/solana-keychain) |

Смотрите
[руководство по добавлению подписантов](https://github.com/solana-foundation/solana-keychain/blob/main/docs/ADDING_SIGNERS.md)
для интеграции дополнительных сервисов управления ключами.
