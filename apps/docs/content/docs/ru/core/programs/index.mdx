Программа Solana — это
[аккаунт](/docs/core/accounts/account-types#program-accounts), который содержит
исполняемый sBPF-байткод и имеет установленный флаг `executable` в значение
`true`. Программы не хранят состояние. Все изменяемые данные размещаются в
отдельных аккаунтах данных, которые передаются через
[инструкции](/docs/core/instructions).

## Основные факты

- **Компилируются в sBPF**: Программы компилируются в Solana Bytecode Format
  (sBPF) через LLVM и хранятся в исполняемых аккаунтах.
- **Без состояния**: Все изменяемые данные хранятся в отдельных аккаунтах
  данных, а не в аккаунте программы.
- **Обновляемые**: Программы, развернутые с помощью loader-v3 (BPF Loader
  Upgradeable), могут быть обновлены при наличии полномочий на обновление; если
  эти полномочия отозваны, программа становится неизменяемой.
