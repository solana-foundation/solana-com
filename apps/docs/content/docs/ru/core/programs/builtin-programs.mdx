## System Program

System Program (`11111111111111111111111111111111`) — единственная программа,
которая может создавать новые аккаунты. Все новые аккаунты изначально
принадлежат System Program, однако право собственности обычно передаётся при
создании. System Program потребляет
[`DEFAULT_COMPUTE_UNITS`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L282)
(150 CU) за инструкцию.

### Справочник инструкций System Program

| Инструкция                                                                                                             | Описание                                                                                                                                                                                                                    |
| ---------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [`CreateAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L295)          | Создаёт новый аккаунт. Выделяет `space` байт, назначает `owner` и переводит `lamports` от плательщика. Новый аккаунт и плательщик должны подписать.                                                                         |
| [`CreateAccountWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L318)  | То же, что и `CreateAccount`, но адрес нового аккаунта вычисляется на основе pubkey аккаунта, seed-строки и программы-владельца. Аккаунт, pubkey которого использован для вычисления адреса, и плательщик должны подписать. |
| [`Assign`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L343)                 | Устанавливает владельца аккаунта. Аккаунт должен подписать. Если уже установлен этот владелец — ничего не происходит.                                                                                                       |
| [`AssignWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L484)         | То же, что и `Assign`, но для адреса, полученного через seed. Аккаунт, pubkey которого использован для вычисления адреса, должен подписать.                                                                                 |
| [`Transfer`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L353)               | Переводит лампорты между аккаунтами. Отправитель должен подписать.                                                                                                                                                          |
| [`TransferWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L357)       | То же, что и `Transfer`, но с адреса, полученного через seed. Аккаунт, pubkey которого использован для вычисления адреса, должен подписать.                                                                                 |
| [`Allocate`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L452)               | Устанавливает длину данных аккаунта. Аккаунт должен подписать, иметь пустые данные и принадлежать System Program. Максимум 10 МБ.                                                                                           |
| [`AllocateWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L462)       | То же, что и `Allocate`, но для адреса, полученного через seed. Также назначает владельца. Аккаунт, pubkey которого использован для вычисления адреса, должен подписать.                                                    |
| [`AdvanceNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L374)    | Продвигает значение nonce до текущего blockhash. Владелец nonce должен подписать.                                                                                                                                           |
| [`InitializeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L412) | Инициализирует аккаунт как nonce-аккаунт с указанным владельцем. Аккаунт должен быть освобождён от rent.                                                                                                                    |
| [`AuthorizeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L432)  | Меняет владельца nonce. Текущий владелец должен подписать.                                                                                                                                                                  |
| [`WithdrawNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L392)   | Выводит лампорты с nonce-аккаунта. Владелец должен подписать. Если аккаунт не закрывается, остаток должен покрывать rent.                                                                                                   |
| [`UpgradeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L437)    | Обновляет nonce-аккаунт с устаревшей версии до текущего формата. Аккаунт должен быть доступен для записи и содержать устаревшую версию nonce.                                                                               |

## Загрузочные программы

Каждая программа принадлежит загрузочной программе. Загрузчики отвечают за
развёртывание, обновление и выполнение принадлежащих им программ. Полный список
загрузчиков и их адресов смотрите в разделе
[Загрузочные программы](/docs/core/programs/program-deployment#loader-programs).
