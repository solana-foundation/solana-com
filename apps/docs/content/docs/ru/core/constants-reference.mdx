На этой странице собраны все константы среды выполнения, упомянутые на страницах
с основными концепциями. Каждая константа содержит ссылку на исходное
определение в клиенте validator
[agave](https://github.com/anza-xyz/agave/tree/v3.1.8) или в
[solana-sdk](https://github.com/anza-xyz/solana-sdk).
