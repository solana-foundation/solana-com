Раздел «Основы SPL-токенов» охватывает базовые инструкции для работы с
SPL-токенами, уделяя внимание функциональности, общей как для оригинального
Token Program, так и для Token Extensions Program (Token 2022).

<Callout type="info">
  Обратитесь к [документации
  Metaplex](https://developers.metaplex.com/token-metadata) для получения
  дополнительной информации о том, как добавить метаданные к SPL-токенам.
</Callout>
