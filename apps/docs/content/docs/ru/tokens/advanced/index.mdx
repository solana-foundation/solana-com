Помимо стандартных операций создания, минтинга и переноса, Token Program также
вызывается из ончейн-программ через CPI и включает инструкции для восстановления
lamport:

<Cards>
  <Card
    title="Вызов Token Program через CPI"
    href="/docs/tokens/advanced/cpi"
  >
    Вызывайте Token Program из ончейн-программы — переносите, минтите и сжигайте
    токены, подписывайте с помощью PDA-авторитета и объединяйте инструкции в один CPI.
  </Card>

<Card
  title="Вывод избыточных Lamport"
  href="/docs/tokens/advanced/withdraw-excess-lamports"
>
  Восстанавливайте SOL сверх минимума, освобождённого от арендной платы, из
  token account, минтов и мультисиг-аккаунтов без изменения балансов токенов.
</Card>

  <Card title="Развёртывание Lamport" href="/docs/tokens/advanced/unwrap-lamports">
    Конвертируйте обёрнутый SOL обратно в lamport — полностью или частично — без закрытия
    token account.
  </Card>
</Cards>

<Callout type="info">

_rs`WithdrawExcessLamports`_ и _rs`UnwrapLamports`_ доступны через клиент
[`@solana-program/token`](https://www.npmjs.com/package/@solana-program/token).
Вызов Token Program через CPI применяется к ончейн-программам.

</Callout>

## Восстановление lamport

_rs`WithdrawExcessLamports`_ восстанавливает SOL сверх минимума, освобождённого
от арендной платы, из token account, минтов и мультисиг-аккаунтов.
_rs`UnwrapLamports`_ конвертирует обёрнутый SOL в нативном token account обратно
в lamport.
