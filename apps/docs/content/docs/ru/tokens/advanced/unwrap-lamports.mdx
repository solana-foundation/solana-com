## Что делает развёртывание lamport?

_rs`UnwrapLamports`_ перемещает SOL из нативного (wrapped SOL) token account в
целевой аккаунт в виде lamport. Можно развернуть весь баланс или указать
конкретную сумму — token account при этом остаётся открытым.

Это более гибкий подход по сравнению с закрытием wrapped SOL аккаунта: частичное
развёртывание позволяет вывести часть SOL, сохранив аккаунт пополненным для
дальнейшего использования.

## Аккаунты

| Account       | Description                                                                  |
| ------------- | ---------------------------------------------------------------------------- |
| `source`      | Доступен для записи. Нативный (wrapped SOL) token account для развёртывания. |
| `destination` | Доступен для записи. Получает развёрнутые lamport.                           |
| `authority`   | Владелец аккаунта или делегат (либо подписанты мультисига).                  |

<Callout type="warn">

Источником должен быть нативный token account; для других token account
возвращается _rs`NonNativeNotSupported`_. Делегат может развернуть сумму не
более делегированной ему.

</Callout>

## Как развернуть lamport

Не указывайте _ts`amount`_, чтобы развернуть весь баланс, или задайте конкретное
количество lamport. В примере используется настроенный клиент
[`@solana/kit`](/docs/clients/official/javascript#solana-kit) — см. раздел
[Перевод токенов](/docs/tokens/basics/transfer-tokens) для настройки клиента.

```ts title="Unwrap wrapped SOL (Kit)"
import { getUnwrapLamportsInstruction } from "@solana-program/token";

// source: the native (wrapped SOL) token account to unwrap.
// destination: address that receives the unwrapped lamports.
// authority: a TransactionSigner (the account owner or a delegate).

// Unwrap a specific number of lamports.
const unwrapSome = getUnwrapLamportsInstruction({
  source,
  destination,
  authority,
  amount: 500_000_000n
});

// Or omit `amount` to unwrap the entire balance.
const unwrapAll = getUnwrapLamportsInstruction({
  source,
  destination,
  authority
});

// Send whichever instruction you need.
await client.sendTransaction([unwrapSome]);
```

Для мультисиг-авторитета передайте со-подписантов через _ts`multiSigners`_.
