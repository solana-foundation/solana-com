## Что такое расширение Scaled UI Amount?

Расширение Scaled UI Amount позволяет эмитентам применять обновляемый множитель
к количеству UI токена. Это полезно в сценариях, когда токен используется для
представления реального актива, такого как акция или дивиденд.

Например, при дроблении акций количество акций удваивается для всех держателей.
Без этого расширения в настоящее время невозможно эффективно и масштабируемо
выпустить новые токены всем держателям во время события дробления.

Однако с моделью расширения Token-2022 появляется возможность изменить способ
отображения UI-количества токенов. Используя расширение ScaledUiAmount и
инструкцию `amount_to_ui_amount`, вы можете установить UI-множитель для своего
токена и получать его UI-количество в любой момент.
[исходный код](https://github.com/solana-program/token-2022/tree/main/program/src/extension/scaled_ui_amount).

Эта функция также может быть использована для дивидендов или распределения
доходности.

Примечание: Новые токены никогда не создаются, количество UI возвращает исходное
количество токенов, умноженное на текущий множитель. Эта функция является
исключительно косметической.

<Callout type="info">
  Некоторые расширения несовместимы друг с другом, и вы не можете включить их
  одновременно для одного и того же токен-минта или token account. Не следует
  включать одновременно расширения `ScaledUiAmount` и `InterestBearingMint` для
  одного и того же токен-минта или token account.
</Callout>
