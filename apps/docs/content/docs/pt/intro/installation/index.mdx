Uma instalação rápida para configurar o seu ambiente de desenvolvimento local da
Solana com um comando. Instale o Rust, a CLI da Solana e o Anchor Framework no
Windows (WSL), Linux e Mac. (Se preferir, pode
[instalar cada dependência individualmente](/docs/intro/installation/dependencies).)

## Pré-requisitos

<Accordions>
<Accordion title="Windows">

Para desenvolver programas Solana no Windows, **deve** usar o
[Windows Subsystem for Linux](https://learn.microsoft.com/en-us/windows/wsl/install)
(WSL). Após instalar o WSL, todas as outras dependências serão instaladas
através do terminal Linux.

Para instalar o WSL, execute o seguinte comando no Windows PowerShell:

```terminal
$ wsl --install
```

O processo de instalação irá solicitar que crie uma conta de utilizador
predefinida.

![Instalação do WSL](/assets/docs/intro/installation/wsl-install.png)

Por predefinição, o WSL instala o Ubuntu. Pode abrir um terminal Linux
procurando "Ubuntu" na barra de pesquisa.

![WSL Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-search.png)

Se o seu terminal Ubuntu se parecer com a imagem abaixo, poderá encontrar um
problema em que `ctrl + v` (atalho de teclado para colar) não funciona no
terminal.

![Terminal Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-terminal-1.png)

Se encontrar este problema, abra o Windows Terminal procurando "Terminal" na
barra de pesquisa.

![Windows Terminal](/assets/docs/intro/installation/wsl-windows-terminal.png)

De seguida, feche o Windows Terminal e reabra um terminal Linux procurando
Ubuntu novamente. O terminal deverá agora parecer-se com a imagem abaixo, onde
`ctrl + v` (atalho de teclado para colar) funciona.

![Terminal Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-terminal-2.png)

Se usar o VS Code, a
[extensão WSL](https://code.visualstudio.com/docs/remote/wsl-tutorial)
permite-lhe usar o WSL e o VS Code em conjunto.

![Configuração do WSL no VS Code](/assets/docs/intro/installation/wsl-vscode.png)

Você verá o seguinte na barra de status do VS Code:

![WSL: Ubuntu](/assets/docs/intro/installation/wsl-vscode-ubuntu.png)

Depois de configurar o WSL, você está pronto para:

- Concluir a [instalação rápida](#installation) usando o terminal Ubuntu (Linux)
- Instalar [Rust](/docs/intro/installation/dependencies#install-rust), a
  [CLI da Solana](/docs/intro/installation/dependencies#install-solana-cli) e a
  [CLI do Anchor](/docs/intro/installation/dependencies#install-anchor-cli)
  individualmente, conforme descrito abaixo

</Accordion>
<Accordion title="Linux">

<Tabs items={[".deb", ".rpm"]}>
<Tab value=".deb" >
Se estiver no Debian ou em alternativas derivadas do Debian como Ubuntu ou Mint, siga os passos abaixo:

1. Atualize as listas de pacotes para garantir que todos os pacotes estejam
   atualizados:

```terminal
$ sudo apt-get update
```

2. Instale as seguintes dependências:

```terminal
$ sudo apt-get install -y \
    build-essential \
    pkg-config \
    libudev-dev llvm libclang-dev \
    protobuf-compiler libssl-dev
```

<Callout>
Se você encontrar o seguinte erro ao instalar `protobuf-compiler`, certifique-se
de executar primeiro _shell`sudo apt-get update`

```
Package protobuf-compiler is not available, but is referred to by another package.
This may mean that the package is missing, has been obsoleted, or
is only available from another source
```

</Callout>

  </Tab>

    <Tab value=".rpm" >
    Para derivados do RHEL como Fedora ou Red Hat Linux, siga os seguintes passos.

Atualize as listas de pacotes para obter as versões mais recentes dos pacotes
executando o comando.

```terminal
$ sudo dnf upgrade --refresh
```

Em seguida, instale as seguintes dependências:

```terminal
$ sudo dnf group install c-development development-tools \
&& sudo dnf install pkg-config openssl-devel
```

## Instalação

1. Digite o seguinte comando no seu terminal para instalar todas as dependências
   necessárias:

```terminal
$ curl --proto '=https' --tlsv1.2 -sSfL https://solana-install.solana.workers.dev | bash
```

Uma instalação bem-sucedida retornará uma saída como a seguinte:

```txt title="Example output"
Installed Versions:
Rust: rustc 1.91.1 (ed61e7d7e 2025-11-07)
Solana CLI: solana-cli 3.0.10 (src:96c3a851; feat:3604001754, client:Agave)
Anchor CLI: anchor-cli 0.32.1
Surfpool CLI: surfpool 0.12.0
Node.js: v24.10.0
Yarn: 1.22.1
```

2. Verifique uma instalação bem-sucedida verificando a versão de cada
   dependência instalada.

```terminal
$ rustc --version && solana --version && anchor --version && surfpool --version && node --version && yarn --version
```

<Callout>
  Se o comando de instalação rápida falhar, consulte a seção [Instalar
  Dependências](/docs/intro/installation/dependencies) abaixo para instruções
  sobre como instalar cada dependência individualmente.
</Callout>
