O desenvolvimento frontend na Solana envolve trabalhar com programas, carteiras
e frameworks JavaScript populares como React. Interagir com esses componentes
requer lidar com conexão, criação de transações e leitura de contas Solana.

[Solana Kit](https://github.com/anza-xyz/kit) (`@solana/kit`) é o SDK JavaScript
padrão para este trabalho. Você monta um cliente a partir de pequenos plugins
combináveis, adiciona bindings React quando necessário e importa clientes de
programa gerados para os programas on-chain que você chama.

## Bibliotecas principais

<Cards>
  <Card title="@solana/kit" href="https://github.com/anza-xyz/kit">
    - SDK principal: RPC, codecs, signatários e transações - Fornece
    `createClient`, o host de plugins que todas as outras bibliotecas estendem
  </Card>
  <Card title="Kit plugins" href="https://github.com/anza-xyz/kit-plugins">
    - Monte um cliente com `.use(...)`: conexão RPC, carteira, signatário e
    planejamento de transações - Publicado como pacotes `@solana/kit-plugin-*`
  </Card>
  <Card
    title="@solana/react"
    href="https://github.com/anza-xyz/kit/tree/main/packages/react"
  >
    - Bindings React para um cliente Kit - `ClientProvider` mais hooks para
    leituras, assinaturas e estado da carteira
  </Card>
  <Card title="@solana-program/*" href="https://github.com/solana-program">
    - Clientes gerados para programas on-chain (System, Token e outros) -
    Fornecem construtores de instruções e plugins de cliente como
    `tokenProgram()`
  </Card>
</Cards>

<Callout type="warn" title="web3.js v1 and wallet-adapter are legacy">
  Muitos projetos Solana ainda dependem do antigo `@solana/web3.js` (v1) e do
  `@solana/wallet-adapter-*`. Para novos projetos, prefira `@solana/kit` com os
  plugins kit abaixo; a descoberta via Wallet Standard (por meio de
  `@solana/kit-plugin-wallet`) substitui o wallet-adapter para carteiras
  modernas.
</Callout>

- [Tutorial Next.js](/docs/frontend/nextjs-solana): um app completo de conexão

- [Migrando para o Kit](/docs/frontend/web3-compat): como os conceitos do
