## Ferramentas de desenvolvimento de IA

Comece com o nosso guia sobre
[como começar com ferramentas de IA na Solana](/developers/guides/getstarted/intro-to-ai).

### Solana MCP

[Solana MCP](https://mcp.solana.com/) (Machine Conversation Protocol) integra
conhecimento sobre Solana diretamente em ferramentas de programação assistidas
por IA como Claude, Cursor, e Windsurf. É treinado em documentação Solana,
framework Anchor, exemplos de programas, e perguntas e respostas do Stack
Exchange - útil para construir integrações de pagamento mais rapidamente.

### Frameworks de agentes de IA

Agentes de IA podem executar transações de pagamento de forma autónoma na
Solana. Estes frameworks permitem construir agentes que processam pagamentos,
monitorizam transações, e automatizam fluxos de trabalho de pagamento:

| Framework        | Descrição                                                                                                | Links                                                                                     |
| ---------------- | -------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | Ferramentas para construir agentes capazes de processar pagamentos, compatível com LangChain e Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | Uma coleção de competências pré-construídas para criar agentes na Solana                                 | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | Framework de agentes multiplataforma com plugin Solana para bots X, Telegram, Discord                    | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Toolkit cross-chain com ações de pagamento Solana                                                        | [GitHub](https://github.com/goat-sdk/goat)                                                |

### Desenvolvimento de programas com IA

[Codigo](https://codigo.ai/) é uma plataforma de desenvolvimento Solana com IA
de ponta a ponta.
