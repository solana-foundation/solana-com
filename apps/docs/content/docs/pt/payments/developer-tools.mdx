### Frameworks de Agentes de IA

Agentes de IA podem executar transações de pagamento de forma autônoma na
Solana. Estes frameworks permitem construir agentes que processam pagamentos,
monitoram transações e automatizam fluxos de trabalho de pagamento:

| Framework        | Descrição                                                                                            | Links                                                                                     |
| ---------------- | ---------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | Ferramentas para construir agentes com capacidade de pagamento, compatível com LangChain e Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | Uma coleção de habilidades pré-construídas para criar agentes na Solana                              | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | Framework de agentes multiplataforma com plugin Solana para bots de X, Telegram e Discord            | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Toolkit cross-chain com ações de pagamento Solana                                                    | [GitHub](https://github.com/goat-sdk/goat)                                                |

### Desenvolvimento de Programas de IA

[Codigo](https://codigo.ai/) é uma plataforma de desenvolvimento de IA para
Solana completa e integrada.
