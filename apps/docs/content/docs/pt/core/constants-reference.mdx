Esta página é uma referência única para todas as constantes de runtime
mencionadas nas páginas de conceitos principais. Cada constante liga à sua
definição de origem no cliente validator
[agave](https://github.com/anza-xyz/agave/tree/v3.1.8) ou
[solana-sdk](https://github.com/anza-xyz/solana-sdk).
