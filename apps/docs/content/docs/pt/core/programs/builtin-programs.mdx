  Os programas principais fornecem funcionalidades fundamentais da rede: gestão
  de contas (System Program), consenso (Vote, Stake), otimização de transações
  (Compute Budget, Address Lookup Table) e privacidade (ZK ElGamal Proof).

  todos os 7 programas.
</Callout>

## O System Program

O System Program (`11111111111111111111111111111111`) é o único programa que
pode criar novas contas. Todas as novas contas são inicialmente propriedade do
System Program, embora a propriedade seja normalmente reatribuída após a
criação. O System Program consome
[`DEFAULT_COMPUTE_UNITS`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L282)
(150 CUs) por instrução.

### Referência de instruções do System Program

| Instrução                                                                                                              | Descrição                                                                                                                                                                                                                 |
| ---------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [`CreateAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L295)          | Cria uma nova conta. Aloca `space` bytes, atribui `owner` e transfere `lamports` do pagador. A nova conta e o pagador devem assinar.                                                                                      |
| [`CreateAccountWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L318)  | Igual a `CreateAccount`, mas deriva o endereço da nova conta a partir da pubkey de uma conta, uma string seed e o programa proprietário. A conta cuja pubkey foi usada para derivar o endereço e o pagador devem assinar. |
| [`Assign`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L343)                 | Define o proprietário da conta. A conta deve assinar. Sem efeito se já for esse proprietário.                                                                                                                             |
| [`AssignWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L484)         | Igual a `Assign`, mas para um endereço derivado de seed. A conta cuja pubkey foi usada para derivar o endereço deve assinar.                                                                                              |
| [`Transfer`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L353)               | Transfere lamports entre contas. O remetente deve assinar.                                                                                                                                                                |
| [`TransferWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L357)       | Igual a `Transfer`, mas a partir de um endereço derivado de seed. A conta cuja pubkey foi usada para derivar o endereço deve assinar.                                                                                     |
| [`Allocate`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L452)               | Define o comprimento dos dados da conta. A conta deve assinar, ter dados vazios e ser propriedade do System Program. Máximo de 10 MiB.                                                                                    |
| [`AllocateWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L462)       | Igual a `Allocate`, mas para um endereço derivado de seed. Também atribui o proprietário. A conta cuja pubkey foi usada para derivar o endereço deve assinar.                                                             |
| [`AdvanceNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L374)    | Avança o valor do nonce para o blockhash atual. A autoridade do nonce deve assinar.                                                                                                                                       |
| [`InitializeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L412) | Inicializa uma conta como conta nonce com a autoridade fornecida. A conta deve estar isenta de rent.                                                                                                                      |
| [`AuthorizeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L432)  | Altera a autoridade do nonce. A autoridade atual deve assinar.                                                                                                                                                            |
| [`WithdrawNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L392)   | Retira lamports de uma conta nonce. A autoridade deve assinar. Se não estiver a fechar, o saldo restante deve cobrir o rent.                                                                                              |
| [`UpgradeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L437)    | Atualiza uma conta nonce do formato de versão legada para a versão atual. A conta deve ser gravável e conter uma versão nonce legada.                                                                                     |

| **Vote**                 | `Vote111111111111111111111111111111111111111` | Cria e gere contas que rastreiam o estado de votação e recompensas dos validadores  | [Fonte](https://github.com/anza-xyz/agave/tree/v3.1.8/programs/vote)             |

| **Address Lookup Table** | `AddressLookupTab1e1111111111111111111111111` | Gere tabelas de pesquisa de endereços para transações que referenciam muitas contas | [Fonte](https://github.com/solana-program/address-lookup-table)                  |

## Programas de carregamento

Cada programa é propriedade de um programa de carregamento. Os carregadores
tratam da implementação, atualização e execução dos programas que possuem.
Consulte
[Programas de carregamento](/docs/core/programs/program-deployment#loader-programs)
para a lista completa de carregadores e os respetivos endereços.
