## Solana Kit

`@solana/kit` é o SDK TypeScript recomendado para desenvolver na Solana. Você
monta um cliente a partir de plugins combináveis, adiciona bindings React quando
necessário e importa clientes de programa gerados para os programas on-chain que
você utiliza.

| Pacote                    | Descrição                                            | GitHub                                                                                |
| ------------------------- | ---------------------------------------------------- | ------------------------------------------------------------------------------------- |
| @solana/kit               | SDK principal: RPC, transações, codecs, signatários  | [Fonte](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | Bindings React: `ClientProvider` e hooks             | [Fonte](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | Conexão RPC + envio de transações                    | [Fonte](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | Plugins de pagador de taxas, identidade e signatário | [Fonte](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Conexão Wallet Standard + hooks React                | [Fonte](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

chaves:

## Solana Web3.js

`@solana/web3.js` é o SDK TypeScript legado para Solana. Novos aplicativos devem
usar `@solana/kit`; consulte [Migrando para o Kit](/docs/frontend/web3-compat).

| Package           | Descrição                                                      | GitHub                                                                            |
| ----------------- | -------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| @solana/web3.js   | SDK principal                                                  | [Fonte](https://github.com/solana-foundation/solana-web3.js)                      |
| @solana/spl-token | Interaja com os programas Token, Token-2022 e Associated Token | [Fonte](https://github.com/solana-program/token-2022/tree/main/clients/js-legacy) |
| @solana/spl-memo  | Interaja com o programa Memo                                   | [Fonte](https://github.com/solana-program/memo/tree/main/clients/js-legacy)       |
