## Solana Client

`@solana/client` é o runtime Solana headless que agrupa RPC, carteiras,
transações e subscrições numa única store.

| Pacote         | Descrição                | GitHub                                                                                |
| -------------- | ------------------------ | ------------------------------------------------------------------------------------- |
| @solana/client | Runtime cliente headless | [Fonte](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [Guia: visão geral do @solana/client](/docs/frontend/client)

## Solana React Hooks

`@solana/react-hooks` adiciona um provider React e hooks sobre o
`@solana/client`, reutilizando o mesmo runtime e cache.

| Pacote              | Descrição                       | GitHub                                                                                     |
| ------------------- | ------------------------------- | ------------------------------------------------------------------------------------------ |
| @solana/react-hooks | Provider React + hooks para UIs | [Fonte](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [Guia: @solana/react-hooks](/docs/frontend/react-hooks)

## Solana Kit

`@solana/kit` é o SDK TypeScript recomendado para desenvolver na Solana.

| Pacote                         | Descrição                               | GitHub                                                                         |
| ------------------------------ | --------------------------------------- | ------------------------------------------------------------------------------ |
| @solana/kit                    | SDK Principal                           | [Fonte](https://github.com/anza-xyz/kit)                                       |
| @solana-program/system         | Interagir com o System Program          | [Fonte](https://github.com/solana-program/system/tree/main/clients/js)         |
| @solana-program/token          | Interagir com o Token Program           | [Fonte](https://github.com/solana-program/token/tree/main/clients/js)          |
| @solana-program/token-2022     | Interagir com o programa Token-2022     | [Fonte](https://github.com/solana-program/token-2022/tree/main/clients/js)     |
| @solana-program/memo           | Interagir com o programa Memo           | [Fonte](https://github.com/solana-program/memo/tree/main/clients/js)           |
| @solana-program/compute-budget | Interagir com o programa Compute Budget | [Fonte](https://github.com/solana-program/compute-budget/tree/main/clients/js) |
| @solana/kit-plugins            | Plugins para o Kit                      | [Fonte](https://github.com/anza-xyz/kit-plugins)                               |

## Assinatura e Gestão de Chaves

Para assinatura backend compatível com kit em múltiplos backends de gestão de
chaves:

| Pacote           | Descrição                                                                                 | GitHub                                                        |
| ---------------- | ----------------------------------------------------------------------------------------- | ------------------------------------------------------------- |
| @solana/keychain | Assinatura unificada: Memory, Vault, AWS KMS, Privy, Turnkey, Fireblocks, CDP, Dfns, Para | [Fonte](https://github.com/solana-foundation/solana-keychain) |

Consulte o
[guia Adicionando Signatários](https://github.com/solana-foundation/solana-keychain/blob/main/docs/ADDING_SIGNERS.md)
para integrar serviços adicionais de gestão de chaves.
