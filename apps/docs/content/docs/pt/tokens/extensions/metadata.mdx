## O Que SÃ£o as ExtensÃµes Ponteiro de Metadados e Metadados de Token?

<Callout type="info" title="URI de Metadados Off-chain">

As duas extensÃµes resolvem problemas diferentes:

  _rs`TokenMetadata`_ Ã© uma extensÃ£o TLV de comprimento variÃ¡vel.

## Como Armazenar Metadados de Token na Conta de Cunhagem

Para armazenar metadados na conta de cunhagem:
