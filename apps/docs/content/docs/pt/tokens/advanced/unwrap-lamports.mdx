## O Que Faz o Desempacotamento de Lamports?

_rs`UnwrapLamports`_ move SOL para fora de um token account nativo (SOL
empacotado) para uma conta de destino como lamports. Você pode desempacotar o
saldo completo ou um valor específico, e o token account permanece aberto.

Isso é mais flexível do que fechar um token account de SOL empacotado: um
desempacotamento parcial permite retirar algum SOL enquanto mantém a conta
financiada para uso posterior.

## Contas

| Conta         | Descrição                                                              |
| ------------- | ---------------------------------------------------------------------- |
| `source`      | Gravável. O token account nativo (SOL empacotado) a ser desempacotado. |
| `destination` | Gravável. Recebe os lamports desempacotados.                           |
| `authority`   | O proprietário da conta ou um delegado (ou signatários multisig).      |

<Callout type="warn">

A origem deve ser um token account nativo; outros token accounts retornam
_rs`NonNativeNotSupported`_. Um delegado pode desempacotar até o valor delegado
a ele.

</Callout>

## Como Desempacotar Lamports

Omita _ts`amount`_ para desempacotar o saldo completo, ou defina-o para
desempacotar um número específico de lamports. O exemplo usa um cliente
[`@solana/kit`](/docs/clients/official/javascript#solana-kit) configurado —
consulte [Transferir Tokens](/docs/tokens/basics/transfer-tokens) para
configuração do cliente.

```ts title="Unwrap wrapped SOL (Kit)"
import { getUnwrapLamportsInstruction } from "@solana-program/token";

// source: the native (wrapped SOL) token account to unwrap.
// destination: address that receives the unwrapped lamports.
// authority: a TransactionSigner (the account owner or a delegate).

// Unwrap a specific number of lamports.
const unwrapSome = getUnwrapLamportsInstruction({
  source,
  destination,
  authority,
  amount: 500_000_000n
});

// Or omit `amount` to unwrap the entire balance.
const unwrapAll = getUnwrapLamportsInstruction({
  source,
  destination,
  authority
});

// Send whichever instruction you need.
await client.sendTransaction([unwrapSome]);
```

Para uma autoridade multisig, passe os co-signatários com _ts`multiSigners`_.
