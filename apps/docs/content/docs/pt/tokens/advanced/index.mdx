Além das operações comuns de criação, emissão e transferência, o Token Program
também é chamado por programas on-chain via CPI e inclui instruções para
recuperar lamports:

<Cards>
  <Card
    title="Chamando o Token Program via CPI"
    href="/docs/tokens/advanced/cpi"
  >
    Invoque o Token Program a partir de um programa on-chain — transfira, emita e queime
    tokens, assine com uma autoridade PDA e agrupe instruções em uma única CPI.
  </Card>

<Card
  title="Retirar Lamports Excedentes"
  href="/docs/tokens/advanced/withdraw-excess-lamports"
>
  Recupere SOL acima do mínimo isento de aluguel de token accounts, mints e
  contas multisig sem alterar os saldos de tokens.
</Card>

  <Card title="Desempacotar Lamports" href="/docs/tokens/advanced/unwrap-lamports">
    Converta SOL embrulhado de volta em lamports, total ou parcialmente, sem fechar
    o token account.
  </Card>
</Cards>

<Callout type="info">

_rs`WithdrawExcessLamports`_ e _rs`UnwrapLamports`_ estão disponíveis através do
[`@solana-program/token`](https://www.npmjs.com/package/@solana-program/token)
client. Chamar o Token Program via CPI aplica-se a programas on-chain.

</Callout>

## Recuperação de lamports

_rs`WithdrawExcessLamports`_ recupera SOL acima do mínimo isento de aluguel de
token accounts, mints e contas multisig. _rs`UnwrapLamports`_ converte SOL
embrulhado em um native token account de volta em lamports.
