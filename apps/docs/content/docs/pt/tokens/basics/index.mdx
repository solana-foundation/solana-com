A seção Fundamentos de SPL Token cobre as instruções básicas para interagir com
SPL Tokens, focando em funcionalidades compartilhadas tanto pelo Token Program
original quanto pelo Token Extensions Program (Token 2022).

<Callout type="info">
  Consulte a [documentação da
  Metaplex](https://developers.metaplex.com/token-metadata) para mais
  informações sobre como adicionar metadados aos SPL Tokens.
</Callout>
