Comece a usar Solana Attestations instalando os pacotes necessários e
configurando seu ambiente de desenvolvimento.

## TypeScript/JavaScript

Instale a biblioteca Solana Attestations usando seu gerenciador de pacotes
preferido:

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
