_Disponível no GitHub:
[`v2.2.0-beta.7`](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.7)_

O Kora v2.2.0-beta apresenta suporte a bundles Jito, novos métodos RPC, imagens
Docker, recursos de segurança aprimorados e melhorias no SDK.

> **Este é um pré-lançamento.** Os recursos documentados aqui estão disponíveis
> no `v2.2.0-beta.7` e podem mudar antes do lançamento estável. Para
> documentação estável, consulte a
> [documentação principal do Kora](/docs/tools/kora).

## Destaques

### Suporte a Bundles Jito

Envie até 5 transações atomicamente através do block engine do Jito. Novos
métodos RPC:

- [`signBundle`](/docs/tools/kora/beta/bundle-methods#signbundle) — Assina um
  bundle sem transmitir
- [`signAndSendBundle`](/docs/tools/kora/beta/bundle-methods#signandsendbundle)
  — Assina e envia para o Jito
- [`estimateBundleFee`](/docs/tools/kora/beta/bundle-methods#estimatebundlefee)
  — Estima as taxas totais do bundle

Consulte o [Guia de Bundles Jito](/docs/tools/kora/beta/jito-guide) para um
passo a passo completo.

### Imagens Docker

Imagens oficiais multiplataforma (amd64/arm64) publicadas no GitHub Container
Registry. Consulte [Instalação](/docs/tools/kora/beta/installation).

### Plugins de Transação

Um novo sistema de plugins permite que o Kora valide a estrutura das transações
durante os fluxos de assinatura. O primeiro plugin integrado, `gas_swap`, aplica
regras rigorosas para transações de troca de token por SOL sem taxas de gas. Os
plugins são configurados em `kora.toml` e executados automaticamente durante
`signTransaction`, `signAndSendTransaction`, `signBundle` e `signAndSendBundle`.

Consulte
[Configuração — Plugins de Transação](/docs/tools/kora/beta/configuration#transaction-plugins).

### Novas Opções de Configuração

- **Plugins de Transação** para validação da estrutura de transações
  (`gas_swap`)
- **reCAPTCHA v3** proteção contra bots para endpoints de assinatura
- **Lighthouse** proteção contra drenagem do pagador de taxas
- **Limites de Uso** limitação de transações por carteira
- **Configuração de bundle** para integração com Jito

Consulte [Configuração](/docs/tools/kora/beta/configuration).

### Alterações no SDK

- Suporte a métodos de bundle (`signBundle`, `signAndSendBundle`,
  `estimateBundleFee`)

Consulte [Alterações no SDK](/docs/tools/kora/beta/sdk-changes).

### Métodos RPC Adicionais

- [`getVersion`](/docs/tools/kora/beta/bundle-methods#getversion) — Obtém
  informações sobre a versão do servidor

## Instalação

```bash
cargo install kora-cli@2.2.0-beta.7
```

Ou via Docker:

```bash
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```
