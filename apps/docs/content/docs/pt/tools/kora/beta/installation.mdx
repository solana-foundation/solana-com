---

## Cargo

Instale a versão beta diretamente do crates.io:

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

Imagens Docker oficiais multiplataforma (amd64/arm64) são publicadas no GitHub
Container Registry:

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

Tags disponíveis:

- `latest` — Versão estável mais recente
- `beta` — Versão de pré-lançamento mais recente
- `v<version>` — Versão específica (ex: `v2.2.0-beta.7`)

### Docker Compose

Para builds personalizadas ou para incluir Redis para cache e suporte a limites
de uso:

- [Exemplo de Dockerfile](https://github.com/solana-foundation/kora/blob/main/examples/deploy/Dockerfile)
- [Docker Compose com Redis](https://github.com/solana-foundation/kora/blob/main/docker-compose.yml)

### Executando com Docker

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## SDK TypeScript

O SDK beta está disponível no npm:

```bash
pnpm add @solana/kora@beta
```

Consulte [Mudanças no SDK](/docs/tools/kora/beta/sdk-changes) para novas APIs.

## Verificar Instalação

```bash
kora --version
```
