---
title: LiteSVM
description:
  Testes ultrarrápidos de programas Solana com uma VM em processo. Teste seus
  programas mais rapidamente sem sobrecarga de rede ou processos de validator.
---

LiteSVM é uma biblioteca leve para testar programas Solana que funciona criando
uma VM Solana em processo, otimizada para desenvolvedores de programas. Ao
contrário de outras abordagens de teste que iniciam processos de validator
separados, o LiteSVM incorpora a VM diretamente nos seus testes, fazendo-os
executar muito mais rapidamente. Em uma ruptura adicional com a tradição, possui
uma API ergonômica com padrões sensatos e ampla configurabilidade para quem a
desejar.
