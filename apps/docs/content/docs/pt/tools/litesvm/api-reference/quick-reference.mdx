## Referência Rápida

| Método                                         | Descrição                                                                                                  |
| ---------------------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| `LiteSVM::new()`                               | Cria um novo ambiente de teste básico LiteSVM                                                              |
| `LiteSVM::default()`                           | Retorna o "valor padrão" para um tipo                                                                      |
| `airdrop(&Address, lamports)`                  | Financia a conta com os lamports especificados                                                             |
| `get_account(&Address)`                        | Retorna todas as informações associadas à conta do endereço fornecido                                      |
| `set_account(Address, Account)`                | Define todas as informações associadas à conta do endereço fornecido                                       |
| `get_balance(&Address)`                        | Obtém o saldo em lamport da conta do endereço fornecido                                                    |
| `minimum_balance_for_rent_exemption(data_len)` | Retorna o saldo mínimo necessário para tornar uma conta com o tamanho de dados especificado isenta de rent |
| `add_program(program_id, &bytes)`              | Adiciona um programa SBF ao ambiente de teste                                                              |
| `add_program_from_file(program_id, path)`      | Adiciona um programa SBF ao ambiente de teste a partir do arquivo especificado                             |
| `send_transaction(tx)`                         | Envia uma transação assinada                                                                               |
| `simulate_transaction(tx)`                     | Simula uma transação sem modificar o estado                                                                |
| `get_transaction(&signature)`                  | Obtém uma transação do histórico                                                                           |
| `latest_blockhash()`                           | Obtém o blockhash atual                                                                                    |
| `expire_blockhash()`                           | Força a expiração do blockhash atual                                                                       |
| `warp_to_slot(slot)`                           | Avança o relógio para o slot especificado                                                                  |
| `get_sysvar<T>()`                              | Obtém uma sysvar do ambiente de teste                                                                      |
| `set_sysvar<T>(&value)`                        | Define a sysvar no ambiente de teste                                                                       |
| `with_feature_accounts()`                      | Inclui contas de feature gate no ambiente de teste                                                         |
| `set_invocation_inspect_callback(callback)`    | Registra um callback de invocação pré/pós (requer a funcionalidade `invocation-inspect-callback`)          |
