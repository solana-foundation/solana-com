## Referência Rápida

### Configuração do Plugin

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### Gestão de Contas (client.svm)

| Método                                 | Descrição                           |
| -------------------------------------- | ----------------------------------- |
| `setAccount(account)`                  | Definir estado da conta diretamente |
| `getAccount(address)`                  | Recuperar dados da conta            |
| `getBalance(address)`                  | Obter saldo de SOL                  |
| `airdrop(address, lamports)`           | Financiar uma conta                 |
| `minimumBalanceForRentExemption(size)` | Calcular mínimo isento de aluguel   |

### Gestão de Programas (client.svm)

| Método                                | Descrição                                 |
| ------------------------------------- | ----------------------------------------- |
| `addProgram(programId, bytes)`        | Carregar programa a partir de bytes       |
| `addProgramFromFile(programId, path)` | Carregar programa a partir de arquivo .so |

### Transações (client.svm)

| Método                    | Descrição                        |
| ------------------------- | -------------------------------- |
| `sendTransaction(tx)`     | Executar uma transação           |
| `simulateTransaction(tx)` | Simular sem alterações de estado |
| `latestBlockhash()`       | Obter blockhash atual            |
| `expireBlockhash()`       | Avançar para novo blockhash      |

### Configuração (client.svm)

| Método                      | Descrição                                  |
| --------------------------- | ------------------------------------------ |
| `withSigverify(bool)`       | Ativar/desativar verificação de assinatura |
| `withBlockhashCheck(bool)`  | Ativar/desativar validação de blockhash    |
| `withSysvars()`             | Ativar contas sysvar                       |
| `withBuiltins()`            | Ativar programas integrados                |
| `withPrecompiles()`         | Ativar programas pré-compilados            |
| `withTransactionHistory(n)` | Armazenar as últimas n transações          |
| `withComputeBudget(budget)` | Definir limites de computação              |
| `withLamports(amount)`      | Definir lamports padrão                    |
| `withLogBytesLimit(n)`      | Definir limite de saída de log             |

### Tempo & Sysvars (client.svm)

| Método                     | Descrição                       |
| -------------------------- | ------------------------------- |
| `warpToSlot(slot)`         | Ir para um slot específico      |
| `getClock()`               | Obter sysvar do clock           |
| `getRent()`                | Obter sysvar de rent            |
| `getEpochSchedule()`       | Obter agendamento de epoch      |
| `getLastRestartSlot()`     | Obter último slot de reinício   |
| `setLastRestartSlot(slot)` | Definir último slot de reinício |

### RPC (client.rpc)

| Método                           | Descrição                           |
| -------------------------------- | ----------------------------------- |
| `getAccountInfo(address)`        | Buscar conta única (somente base64) |
| `getMultipleAccounts(addresses)` | Buscar múltiplas contas             |
| `getLatestBlockhash()`           | Obter blockhash atual               |
