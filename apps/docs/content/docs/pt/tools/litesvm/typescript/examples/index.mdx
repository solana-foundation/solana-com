## Executando Exemplos

Todos os exemplos podem ser executados com `tsx`, embora alguns
endereços/variáveis exijam a substituição dos marcadores por valores reais.

```bash
# Install dependencies
pnpm add @solana/kit @solana/kit-plugin-litesvm @solana/kit-plugin-signer @solana-program/system

# Run an example
npx tsx example.ts
```
