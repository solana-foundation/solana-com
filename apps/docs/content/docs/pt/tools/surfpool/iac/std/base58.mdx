## encode_base58

`encode_base58` codifica um buffer ou string como uma string base58.

### Entradas

| Nome    | Tipo                  | Descrição                                                                                                                             |
| ------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | O buffer ou string a codificar. Strings começando com '0x' são decodificadas como hex; caso contrário, são usados bytes UTF-8 brutos. |

### Saída

| Nome    | Tipo   | Descrição                                    |
| ------- | ------ | -------------------------------------------- |
| `value` | string | A entrada, codificada como uma string base58 |

```hcl
output "encoded" {
    value = encode_base58("0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8")
}
// > encoded: CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF
```

## decode_base58

`decode_base58` decodifica uma string codificada em base58 e retorna o resultado
como um buffer.

### Entradas

| Nome            | Tipo   | Descrição                     |
| --------------- | ------ | ----------------------------- |
| `base58_string` | string | A string base58 a decodificar |

### Saída

| Nome    | Tipo   | Descrição                                    |
| ------- | ------ | -------------------------------------------- |
| `value` | buffer | A string base58 decodificada, como um buffer |

```hcl
output "decoded" {
    value = decode_base58("CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF")
}
// > decoded: 0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8
```
