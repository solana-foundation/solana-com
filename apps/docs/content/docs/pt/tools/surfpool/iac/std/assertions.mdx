## assert_eq

`assert_eq` verifica se dois valores são iguais.

### Entradas

| Nome    | Tipo | Descrição                   |
| ------- | ---- | --------------------------- |
| `left`  | any  | Um valor para comparar      |
| `right` | any  | O valor com o qual comparar |

### Saída

| Nome    | Tipo                  | Descrição               |
| ------- | --------------------- | ----------------------- |
| `value` | addon(std::assertion) | O resultado da asserção |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```

## assert_gt

`assert_gt` verifica se o valor à esquerda é maior que o valor à direita.

### Entradas

| Nome    | Tipo             | Descrição                               |
| ------- | ---------------- | --------------------------------------- |
| `left`  | integer \| float | Um inteiro ou float para comparar       |
| `right` | integer \| float | Um inteiro ou float com o qual comparar |

### Saída

| Nome    | Tipo                  | Descrição               |
| ------- | --------------------- | ----------------------- |
| `value` | addon(std::assertion) | O resultado da asserção |

```hcl
output "assertion" {
    value = std::assert_gt(action.example.result, 1)
}
```
