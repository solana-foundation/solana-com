A biblioteca padrão fornece funções e ações essenciais que estão disponíveis
para todos os Runbooks, independentemente dos addons de blockchain habilitados.

## Funções

A biblioteca padrão inclui funções utilitárias para:

- **Operadores**: Operações matemáticas e comparações
- **Hash**: Funções de hashing (SHA256, Keccak256, etc.)
- **Hex**: Codificação e decodificação hexadecimal
- **Crypto**: Operações criptográficas
- **JSON**: Codificação e decodificação JSON
- **List**: Funções de manipulação de listas
- **Assertions**: Asserções de igualdade e condições
- **Base64**: Codificação e decodificação Base64

[Ver todas as funções](/docs/tools/surfpool/iac/std/functions)

## Ações

A biblioteca padrão inclui ações para:

- **HTTP**: Realização de requisições HTTP
- **Operações de Arquivo**: Leitura e escrita de arquivos

[Ver todas as ações](/docs/tools/surfpool/iac/std/actions)
