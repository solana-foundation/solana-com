## jq

A função `jq` permite fatiamento, filtragem e mapeamento de dados JSON. Consulte
a [documentação do jq](https://jqlang.github.io/jq/manual/) para mais detalhes.

### Entradas

| Nome           | Tipo             | Descrição                                                                               |
| -------------- | ---------------- | --------------------------------------------------------------------------------------- |
| `decoded_json` | string \| object | Um objeto JSON                                                                          |
| `query`        | string           | Uma consulta JSON. Consulte a [documentação do jq](https://jqlang.github.io/jq/manual/) |

### Saída

| Nome    | Tipo          | Descrição                    |
| ------- | ------------- | ---------------------------- |
| `value` | array[string] | O resultado da consulta `jq` |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
