## encode_base64

`encode_base64` codifica um buffer ou string como uma string base64.

### Entradas

| Nome    | Tipo                  | Descrição                                                                                                                             |
| ------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | O buffer ou string a codificar. Strings começando com '0x' são decodificadas como hex; caso contrário, bytes UTF-8 brutos são usados. |

### Saída

| Nome    | Tipo   | Descrição                                    |
| ------- | ------ | -------------------------------------------- |
| `value` | string | A entrada, codificada como uma string base64 |

```hcl
output "encoded" {
    value = encode_base64("0x48656c6c6f20776f726c6421")
}
// > encoded: SGVsbG8gd29ybGQh
```

## decode_base64

`decode_base64` decodifica uma string codificada em base64 e retorna o resultado
como um buffer.

### Entradas

| Nome            | Tipo   | Descrição                     |
| --------------- | ------ | ----------------------------- |
| `base64_string` | string | A string base64 a decodificar |

### Saída

| Nome    | Tipo   | Descrição                                    |
| ------- | ------ | -------------------------------------------- |
| `value` | buffer | A string base64 decodificada, como um buffer |

```hcl
output "decoded" {
    value = decode_base64("SGVsbG8gd29ybGQh")
}
// > decoded: 0x48656c6c6f20776f726c6421
```
