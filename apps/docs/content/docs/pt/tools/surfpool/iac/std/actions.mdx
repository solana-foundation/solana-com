Estas ações estão disponíveis em todos os Runbooks como parte da biblioteca
padrão.

## send_http_request

A ação `std::send_http_request` faz uma requisição HTTP para a URL especificada.

### Entradas

| Nome         | Obrigatório | Tipo    | Descrição                                           |
| ------------ | ----------- | ------- | --------------------------------------------------- |
| `url`        | obrigatório | string  | A URL para onde enviar a requisição                 |
| `method`     | opcional    | string  | O método HTTP (GET, POST, PUT, DELETE). Padrão: GET |
| `headers`    | opcional    | map     | Cabeçalhos HTTP a incluir na requisição             |
| `body`       | opcional    | string  | O corpo da requisição (para requisições POST/PUT)   |
| `timeout_ms` | opcional    | integer | Tempo limite da requisição em milissegundos         |

### Saídas

| Nome          | Tipo    | Descrição                 |
| ------------- | ------- | ------------------------- |
| `status_code` | integer | O código de status HTTP   |
| `body`        | string  | O corpo da resposta       |
| `headers`     | map     | Os cabeçalhos da resposta |

```hcl
action "api_call" "std::send_http_request" {
    description = "Fetch data from API"
    url = "https://api.example.com/data"
    method = "GET"
    headers = {
        "Authorization" = "Bearer ${variable.token}"
    }
}

output "response" {
    value = action.api_call.body
}

output "status" {
    value = action.api_call.status_code
}
```

### Exemplo de Requisição POST

```hcl
action "create_resource" "std::send_http_request" {
    description = "Create a new resource"
    url = "https://api.example.com/resources"
    method = "POST"
    headers = {
        "Content-Type" = "application/json"
    }
    body = {
        "name" = variable.name,
        "value" = variable.value
    }
}
```
