## ripemd160

`ripemd160` calcula o hash Ripemd160 de um valor.

### Entradas

| Nome    | Tipo                    | Descrição                           |
| ------- | ----------------------- | ----------------------------------- |
| `value` | buffer \| array[buffer] | O valor codificado em hex para hash |

### Saída

| Nome    | Tipo   | Descrição           |
| ------- | ------ | ------------------- |
| `value` | string | O resultado do hash |

```hcl
output "hashed_data" {
    value = ripemd160(encode_hex("hello, world"))
}
// > hashed_data: 0XA3201F82FCA034E46D10CD7B27E174976E241DA2
```

## sha256

### Entradas

| Nome    | Tipo                    | Descrição                           |
| ------- | ----------------------- | ----------------------------------- |
| `value` | buffer \| array[buffer] | O valor codificado em hex para hash |

### Saída

| Nome    | Tipo   | Descrição           |
| ------- | ------ | ------------------- |
| `value` | string | O resultado do hash |

```hcl
output "hashed_data" {
    value = sha256(encode_hex("hello, world"))
}
// > hashed_data: 0x09ca7e4eaa6e8ae9c7d261167129184883644d07dfba7cbfbc4c8a2e08360d5b
```
