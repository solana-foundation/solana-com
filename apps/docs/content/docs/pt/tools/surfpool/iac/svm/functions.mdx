Estas funções estão disponíveis ao utilizar o addon SVM para Solana e
blockchains compatíveis com SVM.

## system_program_id

`svm::system_program_id` retorna o id do System Program,
`11111111111111111111111111111111`.

```hcl
output "system_program_id" {
    value = svm::system_program_id()
}
// > 11111111111111111111111111111111
```

## get_instruction_data_from_idl_path

`svm::get_instruction_data_from_idl_path` cria instruction data codificada para
uma invocação de programa, fornecendo verificação de tipos e serialização com
base no arquivo IDL fornecido.

### Entradas

| Name               | Required | Type          | Description                                                 |
| ------------------ | -------- | ------------- | ----------------------------------------------------------- |
| `idl_path`         | required | string        | O caminho, relativo ao txtx.yml, para o arquivo IDL `.json` |
| `instruction_name` | required | string        | O nome da instrução para a qual gerar dados                 |
| `arguments`        | optional | array[string] | Os argumentos da instrução para os quais gerar dados        |

```hcl
output "data" {
    value = svm::get_instruction_data_from_idl("/path/to/idl.json", "my_instruction", ["arg1", "arg2"])
}
// > data: 0x95763bdcc47fa1b305000000776f726c64
```

## get_instruction_data_from_idl

`svm::get_instruction_data_from_idl` cria instruction data codificada para uma
invocação de programa, fornecendo verificação de tipos e serialização com base
nos dados IDL fornecidos.

### Entradas

| Name               | Required | Type              | Description                                                             |
| ------------------ | -------- | ----------------- | ----------------------------------------------------------------------- |
| `idl`              | required | string \| SVM_IDL | O IDL do programa                                                       |
| `instruction_name` | required | string            | O nome da instrução para a qual gerar dados, conforme indexado pelo IDL |
| `arguments`        | optional | array[string]     | Os argumentos da instrução para os quais gerar dados                    |

```hcl
output "data" {
    value = svm::get_instruction_data_from_idl(variable.contract.idl, "my_instruction", ["arg1", "arg2"])
}
// > data: 0x95763bdcc47fa1b305000000776f726c64
```

## get_program_from_anchor_project

`svm::get_program_from_anchor_project` recupera os artefatos de implantação do
programa para um programa em um projeto Anchor.

### Entradas

| Nome           | Obrigatório | Tipo   | Descrição                              |
| -------------- | ----------- | ------ | -------------------------------------- |
| `program_name` | obrigatório | string | O nome do programa sendo implantado    |
| `keypair_path` | opcional    | string | O local do arquivo keypair do programa |
| `idl_path`     | opcional    | string | O local do arquivo IDL do programa     |
| `bin_path`     | opcional    | string | O local do arquivo binário do programa |

```hcl
variable "contract" {
    value = svm::get_program_from_anchor_project("my_program")
}
output "idl" {
    value = variable.contract.idl
}
```

## sol_to_lamports

`svm::sol_to_lamports` converte a quantidade de SOL fornecida para lamports.

```hcl
output "lamports" {
    value = svm::sol_to_lamports(1.1)
}
// lamports: 1100000000
```

---

## lamports_to_sol

`svm::lamports_to_sol` converte a quantidade de lamports fornecida para SOL.

```hcl
output "sol" {
    value = svm::lamports_to_sol(1100000000)
}
// sol: 1.1
```

## find_pda

`svm::find_pda` encontra um PDA válido usando o ID do programa e as seeds
fornecidas.

### Entradas

| Nome         | Obrigatório | Tipo          | Descrição                                           |
| ------------ | ----------- | ------------- | --------------------------------------------------- |
| `program_id` | obrigatório | string        | O endereço do programa do qual o PDA é derivado     |
| `seeds`      | opcional    | array[string] | Um array opcional de seeds (máx. 16, 32 bytes cada) |

```hcl
variable "pda" {
    value = svm::find_pda("3bv3j4GvMPjvvBX9QdoX27pVoWhDSXpwKZipFF1QiVr6", ["data"])
}
output "pda" {
    value = std::encode_base58(variable.pda.pda)
}
output "bump" {
    value = variable.pda.bump_seed
}
// > pda: 4amHoWMBgLkPfM8Nq9ZP33Liq9FCuqrLoU1feejkdsUJ
// > bump: 252
```

## get_associated_token_account

`svm::get_associated_token_account` calcula o endereço do associated token
account para os endereços de carteira e token mint fornecidos.

### Entradas

| Name                 | Required | Type   | Description              |
| -------------------- | -------- | ------ | ------------------------ |
| `wallet_address`     | required | string | O endereço da carteira   |
| `token_mint_address` | optional | string | O endereço do token mint |

```hcl
variable "token_account" {
    value = svm::get_associated_token_account(signer.caller.address, "So11111111111111111111111111111111111111112")
}
```

## create_token_account_instruction

`svm::create_token_account_instruction` cria bytes de instrução brutos para
criar um associated token account.

### Entradas

| Name                 | Required | Type   | Description                          |
| -------------------- | -------- | ------ | ------------------------------------ |
| `funding_address`    | required | string | O endereço da conta de financiamento |
| `wallet_address`     | required | string | O endereço da carteira               |
| `token_mint_address` | optional | string | O endereço do token mint             |
| `token_program_id`   | optional | string | O endereço do Token Program          |

```hcl
action "call" "svm::process_instructions" {
    signers = [signer.caller]

    instruction {
        raw_bytes = svm::create_token_account_instruction(
            signer.caller.address, // funding address
            signer.caller.address, // wallet address
            variable.token_mint, // token mint address
            variable.token_program // token program id
        )
    }
}
```

## u64

`svm::u64` cria uma representação em array de bytes de um inteiro u64, adequada
para uso como seed na derivação de PDA.

### Entradas

| Name    | Required | Type    | Description                                 |
| ------- | -------- | ------- | ------------------------------------------- |
| `value` | required | integer | O inteiro u64 a converter em array de bytes |

```hcl
variable "pda" {
    value = svm::find_pda("3bv3j4GvMPjvvBX9QdoX27pVoWhDSXpwKZipFF1QiVr6", [svm::u64(1000000000)])
}
```
