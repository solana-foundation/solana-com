## index

`index` obtém a entrada de uma lista no índice especificado.

### Entradas

| Nome    | Tipo                                             | Descrição                             |
| ------- | ------------------------------------------------ | ------------------------------------- |
| `list`  | array[string] \| array[integer] \| array[buffer] | A lista da qual recuperar uma entrada |
| `index` | integer                                          | O índice da entrada a ser recuperada  |

### Saída

| Nome    | Tipo   | Descrição                                 |
| ------- | ------ | ----------------------------------------- |
| `value` | string | A entrada da lista no índice especificado |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
