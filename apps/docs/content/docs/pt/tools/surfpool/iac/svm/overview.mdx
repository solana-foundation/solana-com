---
title: Visão Geral do Addon SVM
description: Add-on para Solana e Blockchains Compatíveis com SVM (beta)
---

O plugin SVM `txtx` permite criar Runbooks que interagem com Solana e
blockchains compatíveis com SVM. O plugin fornece funções utilitárias que
permitem implantar programas Anchor e codificar chamadas de instrução de acordo
com os IDLs dos programas. As ações podem ser usadas para criar transações
válidas de transferência, chamada de programa e implantação de programa, que
podem ser assinadas via frase mnemônica, chave secreta ou pelo assinador do seu
navegador.

## Início Rápido

Para usar o addon SVM, declare-o no seu Runbook:

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## Funcionalidades

- **Implantação de Programas**: Implante programas Anchor e programas nativos
  Solana
- **Construção de Instruções**: Codifique chamadas de instrução usando IDLs de
  programas
- **Assinatura de Transações**: Suporte para mnemônicos, chaves secretas e
  carteiras web
- **Gerenciamento de Contas**: Consulte e gerencie contas Solana
