O Surfnet é o simulador de rede Solana local do Surfpool, projetado para
fornecer aos desenvolvedores um poderoso ambiente de testes que imita o
comportamento da blockchain Solana sem a sobrecarga de um nó validator completo.

## Primeiros Passos

Inicie uma instância do Surfnet usando o CLI do Surfpool:

```bash
surfpool start
```

Isso iniciará uma instância local do Surfnet em `http://localhost:8899` com
todos os endpoints RPC disponíveis.

## Principais Recursos

- **Compatibilidade Total com RPC**: Suporta a API RPC completa da Solana
- **Iteração Rápida**: Sem necessidade de aguardar confirmações de blocos
- **Manipulação de Estado**: Cheatcodes para modificar diretamente o estado da
  blockchain
- **Fork da Mainnet**: Faça um fork da mainnet para testar com contas e
  programas reais
- **Suporte a Plugin Geyser**: Conecte seus plugins Geyser para streaming de
  dados em tempo real
