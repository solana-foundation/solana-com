O CLI do surfpool pode ser instalado usando o script de instalação ou compilado
a partir do código-fonte.

## Script de Instalação

Para instalar o surfpool, abra um terminal e execute:

```bash
curl -sL https://run.surfpool.run/ | bash
```

## Instalar a Partir do Código-Fonte

Para compilar o binário do Surfpool a partir do código-fonte, clone o
repositório do surfpool.

```bash
git clone https://github.com/solana-foundation/surfpool.git
```

Navegue até o novo diretório.

```bash
cd surfpool
```

Em seguida, compile o binário. Este comando irá compilar o binário e armazená-lo
em `~/.cargo/bin/surfpool`.

```bash
cargo surfpool-install
```

Por fim, certifique-se de que o binário `surfpool` está disponível no seu PATH.
Este processo irá variar dependendo do seu sistema operacional.
