Escolha os pacotes do Commerce Kit que você precisa com base no que está
construindo. Cada pacote pode ser instalado independentemente e pode ser usado
sozinho ou combinado com outros.

## Requisitos do Sistema

- **Node.js**: Versão 18 ou superior (LTS recomendado)
- **Gerenciador de Pacotes**: pnpm, npm ou yarn
- **TypeScript**: Versão 4.5+ (recomendado, mas não obrigatório)

## Dependências Peer Comuns

O Solana Commerce Kit é construído sobre a biblioteca Solana Kit, portanto é
compatível com Solana Kit, Gill, Solana Programs
[Clients](https://github.com/solana-program) e
[Clientes JS gerados por Codama](https://github.com/codama-idl/renderers-js).

```bash
pnpm add gill # or @solana/kit
pnpm add @solana-program/token # or @solana-program/target-program
pnpm add @my-program/codama-js-client
```

## Configuração do TypeScript

Se estiver usando TypeScript, certifique-se de que seu `tsconfig.json` inclui:

```json
{
  "compilerOptions": {
    "module": "ESNext",
    "target": "ES2020",
    "lib": ["ES2020", "DOM", "DOM.Iterable"],
    "jsx": "react-jsx",
    "moduleResolution": "bundler",
    "resolveJsonModule": true,
    "esModuleInterop": true,
    "skipLibCheck": true
  }
}
```

## Configuração do Next.js

Se estiver usando Next.js, atualize `next.config.js` para transpilar os pacotes
do Commerce Kit:

```js
/** @type {import('next').NextConfig} */
const nextConfig = {
  transpilePackages: [
    "@solana-commerce/react",
    "@solana-commerce/connector",
    "@solana-commerce/headless",
    "@solana-commerce/solana-pay",
    "@solana-commerce/sdk"
  ]
};

export default nextConfig;
```
