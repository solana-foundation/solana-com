**Commerce Kit é uma coleção de ferramentas para construir experiências de
comércio na Solana.** De botões de pagamento prontos a primitivos headless, o
Commerce Kit fornece tudo o que você precisa para aceitar pagamentos em
criptomoedas.

## Por que Commerce Kit?

- **Componentes Prontos**: PaymentButton pré-construído com conexão de carteira,
  seleção de token e gestão de transações
- **Agnóstico a Framework**: Use primitivos headless em qualquer framework
  JavaScript
- **Solana Pay**: Suporte completo ao Solana Pay com geração de código QR
- **Tipagem Segura**: Suporte completo a TypeScript em todos os pacotes
- **Wallet Standard**: Compatível com todas as carteiras Wallet Standard
  (Phantom, Solflare, Backpack, etc.)

## Início Rápido

A forma mais rápida de começar é com o componente `PaymentButton`:

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## Instalação

Escolha os pacotes do Commerce Kit que você precisa:

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
