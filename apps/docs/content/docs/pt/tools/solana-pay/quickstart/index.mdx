Comece a usar o Solana Pay rapidamente com estes guias passo a passo.

## Instalação

Configure o SDK do Solana Pay e configure seu ambiente de desenvolvimento.

**[→ Guia de Instalação](/docs/tools/solana-pay/quickstart/installation)** -
Instale o SDK e as dependências

## Tipos de Pagamento

Escolha a integração de pagamento certa para o seu caso de uso:

### Solicitações de Transferência (Simples)

Crie URLs de pagamento não interativas para transferências básicas de SOL e
tokens SPL.

**[→ Solicitações de Transferência](/docs/tools/solana-pay/quickstart/transfer-requests)** -
URLs de pagamento simples

**Melhor para:**

- Pagamentos básicos e doações
- Pagamentos por QR code
- Links de pagamento em e-mails

### Solicitações de Transação (Avançado)

Crie fluxos de pagamento interativos com composição de transações do lado do
servidor.

**[→ Solicitações de Transação](/docs/tools/solana-pay/quickstart/transaction-requests)** -
Fluxos de pagamento interativos

**Melhor para:**

- Preços dinâmicos
- Cunhagem de NFTs
- Transações DeFi complexas
- Lógica de negócios personalizada

## Integração de QR Code

Gere e personalize QR codes para pagamentos em carteiras móveis.

**[→ Integração de QR Code](/docs/tools/solana-pay/quickstart/qr-codes)** - Gere
códigos de pagamento escaneáveis

**Melhor para:**

- Sistemas de ponto de venda
- Pagamentos da web para celular
- Transações presenciais
