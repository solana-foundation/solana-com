**O Solana Pay é um protocolo padrão e um conjunto de implementações de
referência que permitem aos desenvolvedores incorporar pagamentos
descentralizados em suas aplicações e serviços.** A blockchain Solana confirma
transações em menos de um segundo e custa em média $0,0005, proporcionando aos
usuários uma experiência perfeita sem intermediários.

## Por que Solana Pay?

- **Protocolo Padrão**: Abordagem unificada para pagamentos em todas as
  carteiras e aplicações Solana
- **Ultra Rápido**: Transações confirmadas em menos de um segundo
- **Ultra Barato**: Custo médio de transação de $0,0005
- **Sem Intermediários**: Pagamentos diretos peer-to-peer sem terceiros
- **Suporte Universal de Carteiras**: Compatível com Phantom, Solflare, Backpack
  e mais

## Início Rápido

A maneira mais simples de começar é com uma URL de solicitação de transferência:

```
solana:recipient?amount=0.01&label=Coffee%20Shop&message=Grande%20Latte
```

Esta URL pode ser:

- Codificada como um código QR para leitura por carteiras móveis
- Usada como um link de pagamento em aplicações web
- Incorporada em aplicações móveis como deep links

## Instalação

Instale o SDK JavaScript do Solana Pay:

```bash
# Using pnpm (recommended)
pnpm add @solana/pay@beta @solana/kit

# Using npm
npm install @solana/pay@beta @solana/kit

# Using yarn
yarn add @solana/pay@beta @solana/kit
```

## Casos de Uso

O Solana Pay possibilita uma ampla gama de cenários de pagamento:

- **Pagamentos no Varejo**: Pagamentos presenciais usando códigos QR
- **E-commerce**: Fluxos de checkout web com integração de carteira
- **Aplicações Móveis**: Experiências de pagamento nativas com deep linking
- **Cunhagem de NFT**: Criação dinâmica de transações para interações complexas
- **Assinaturas**: Pagamentos recorrentes e programas de fidelidade
- **Integração DeFi**: Transações componíveis com swaps e empréstimos

## Carteiras Compatíveis

Solana Pay funciona com todas as principais carteiras Solana:

- **[Phantom](https://phantom.app/)** - Carteira Solana mais popular
- **[Solflare](https://solflare.com/)** - Carteira Solana multiplataforma
- **[Backpack](https://backpack.app/)** - Carteira Web3 e loja de aplicativos
- **[Glow](https://glow.app/)** - Carteira Solana mobile-first
- **[Ultimate](https://ultimate.app/)** - Carteira focada em DeFi
- **[Decaf Wallet](https://decaf.so/)** - Carteira de Pagamentos Globais
- **[Espresso Cash](https://www.espressocash.com/)** - Carteira de Pagamentos
  Globais
- **[Tiplink](https://tiplink.io/)** - A carteira mais simples do mundo
- **[Bitget Wallet](https://bitget.com/)** - Carteira multi-chain
