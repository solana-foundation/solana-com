Η ανάπτυξη frontend στο Solana περιλαμβάνει εργασία με προγράμματα, πορτοφόλια
και δημοφιλή JavaScript frameworks όπως το React. Η αλληλεπίδραση με αυτά τα
στοιχεία απαιτεί χειρισμό σύνδεσης, δημιουργία συναλλαγών και ανάγνωση από
λογαριασμούς Solana.

[Solana Kit](https://github.com/anza-xyz/kit) (`@solana/kit`) είναι το τυπικό
JavaScript SDK για αυτή τη δουλειά. Συναρμολογείτε έναν client από μικρά,
συνθέσιμα plugins, προσθέτετε React bindings όταν τα χρειάζεστε, και φορτώνετε
generated program clients για τα on-chain programs που καλείτε.

## Κύριες βιβλιοθήκες

<Cards>
  <Card title="@solana/kit" href="https://github.com/anza-xyz/kit">
    - Core SDK: RPC, codecs, signers και transactions - Παρέχει `createClient`,
    τον plugin host που επεκτείνει κάθε άλλη βιβλιοθήκη
  </Card>
  <Card title="Kit plugins" href="https://github.com/anza-xyz/kit-plugins">
    - Συνθέστε έναν client με `.use(...)`: σύνδεση RPC, wallet, signer και
    transaction planning - Δημοσιεύονται ως πακέτα `@solana/kit-plugin-*`
  </Card>
  <Card
    title="@solana/react"
    href="https://github.com/anza-xyz/kit/tree/main/packages/react"
  >
    - React bindings για έναν Kit client - `ClientProvider` και hooks για reads,
    subscriptions και κατάσταση wallet
  </Card>
  <Card title="@solana-program/*" href="https://github.com/solana-program">
    - Generated clients για on-chain programs (System, Token και άλλα) -
    Παρέχουν instruction builders και client plugins όπως το `tokenProgram()`
  </Card>
</Cards>

<Callout type="warn" title="web3.js v1 and wallet-adapter are legacy">
  Πολλά έργα Solana εξακολουθούν να βασίζονται στο παλαιότερο `@solana/web3.js`
  (v1) και `@solana/wallet-adapter-*`. Για νέες εργασίες, προτιμήστε το
  `@solana/kit` με τα kit plugins παρακάτω· η ανακάλυψη Wallet Standard (μέσω
  `@solana/kit-plugin-wallet`) αντικαθιστά το wallet-adapter για σύγχρονα
  wallets.
</Callout>

- [Οδηγός Kit client](/docs/frontend/client): δημιουργήστε έναν client από
  plugins για RPC, signing και αποστολή transactions.
- [Οδηγός React](/docs/frontend/react-hooks): bindings `@solana/react` και hooks
  wallet επιπέδου πάνω στον ίδιο client.
- [Εκμάθηση Next.js](/docs/frontend/nextjs-solana): μια πλήρης εφαρμογή σύνδεσης
  wallet και μεταφοράς SOL στο App Router.
- [Μετάβαση στο Kit](/docs/frontend/web3-compat): πώς οι έννοιες web3.js και
  wallet-adapter αντιστοιχίζονται στο Kit.
