## Solana Kit

`@solana/kit` είναι το συνιστώμενο TypeScript SDK για ανάπτυξη στο Solana.
Συναρμολογείτε έναν client από συνθέσιμα plugins, προσθέτετε React bindings όταν
τα χρειάζεστε, και ενσωματώνετε παραγόμενους program clients για τα on-chain
programs που καλείτε.

| Package                   | Περιγραφή                                    | GitHub                                                                               |
| ------------------------- | -------------------------------------------- | ------------------------------------------------------------------------------------ |
| @solana/kit               | Core SDK: RPC, transactions, codecs, signers | [Πηγή](https://github.com/anza-xyz/kit)                                              |
| @solana/react             | React bindings: `ClientProvider` και hooks   | [Πηγή](https://github.com/anza-xyz/kit/tree/main/packages/react)                     |
| @solana/kit-plugin-rpc    | Σύνδεση RPC + αποστολή συναλλαγών            | [Πηγή](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-rpc)    |
| @solana/kit-plugin-signer | Plugins για fee payer, identity και signer   | [Πηγή](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-signer) |
| @solana/kit-plugin-wallet | Σύνδεση Wallet Standard + React hooks        | [Πηγή](https://github.com/anza-xyz/kit-plugins/tree/main/packages/kit-plugin-wallet) |

εφαρμογές θα πρέπει να χρησιμοποιούν `@solana/kit`· δείτε
[Μετάβαση στο Kit](/docs/frontend/web3-compat).
