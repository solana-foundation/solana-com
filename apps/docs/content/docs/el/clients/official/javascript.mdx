## Solana Client

Το `@solana/client` είναι το headless Solana runtime που ενσωματώνει RPC,
πορτοφόλια, συναλλαγές και subscriptions σε ένα store.

| Πακέτο         | Περιγραφή               | GitHub                                                                               |
| -------------- | ----------------------- | ------------------------------------------------------------------------------------ |
| @solana/client | Headless client runtime | [Πηγή](https://github.com/solana-foundation/framework-kit/tree/main/packages/client) |

- [Οδηγός: επισκόπηση @solana/client](/docs/frontend/client)

## Solana React Hooks

Το `@solana/react-hooks` προσθέτει ένα React provider και hooks πάνω από το
`@solana/client`, επαναχρησιμοποιώντας το ίδιο runtime και cache.

| Πακέτο              | Περιγραφή                      | GitHub                                                                                    |
| ------------------- | ------------------------------ | ----------------------------------------------------------------------------------------- |
| @solana/react-hooks | React provider + hooks για UIs | [Πηγή](https://github.com/solana-foundation/framework-kit/tree/main/packages/react-hooks) |

- [Οδηγός: @solana/react-hooks](/docs/frontend/react-hooks)
