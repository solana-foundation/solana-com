Με το Solana να είναι ένα blockchain υψηλής απόδοσης με χαμηλές χρεώσεις και
γρήγορες συναλλαγές, έχει σχηματιστεί γύρω του μια εξαιρετική κοινότητα
προγραμματιστών παιχνιδιών.

Εδώ μπορείτε να βρείτε λεπτομέρειες σχετικά με τα SDKs ανάπτυξης παιχνιδιών
Solana που είναι διαθέσιμα στο οικοσύστημα. Όλα τα SDKs είναι ανοιχτού κώδικα
και κατασκευασμένα από την κοινότητα. Αν εργάζεστε σε ένα SDK εσείς οι ίδιοι, μη
διστάσετε να ανοίξετε ένα PR σε αυτή τη σελίδα και να το προσθέσετε.

## Unity SDK

Η μηχανή παιχνιδιών Unity είναι γνωστή για τη φιλική της προσέγγιση προς τους
αρχάριους και την υποστήριξη πολλαπλών πλατφορμών, συμπεριλαμβανομένων των
WebGL, iOS και Android. Αναπτύξτε μία φορά, εξάγετε παντού.

- [Τεκμηρίωση](https://docs.magicblock.gg/introduction)
- [Πηγαίος κώδικας](https://github.com/magicblock-labs/Solana.Unity-SDK)
- [Παραδείγματα παιχνιδιών](https://github.com/solana-developers/solana-game-examples)
- [Τρίλιζα](https://blog.magicblock.gg/bolt-tic-tac-toe/)

Το Solana Unity SDK συντηρείται από το [Magicblock](https://www.magicblock.gg/)
και περιλαμβάνει:

- Υποστήριξη NFT
- Ενσωμάτωση candy machine
- Συναλλαγές
- Λειτουργίες RPC
- Βαθιούς συνδέσμους Phantom
- Σύνδεση WebGL
- Αξιόπιστη υποστήριξη σύνδεσης WebSocket
- Προσαρμογέα πορτοφολιού για κινητά
- Κλειδιά συνεδρίας για αυτόματη έγκριση συναλλαγών
- Δημιουργία κώδικα πελάτη Anchor και άλλα.

## Godot SDK

Η [Μηχανή Godot](https://godotengine.org/) είναι μια μηχανή παιχνιδιών ανοιχτού
κώδικα που έχει κερδίσει μεγάλη υποστήριξη τα τελευταία χρόνια. Το SDK Solana
για Godot συντηρείται από τη [ZenRepublic](https://twitter.com/ZenRepublicNDM)
και τον [Virus-Axel](https://twitter.com/AxelBenjam). Περιλαμβάνει ενσωμάτωση
προσαρμογέα πορτοφολιού, συναλλαγές, λειτουργίες RPC και δημιουργία κώδικα
πελάτη Anchor. Ακόμη και μια πλήρως λειτουργική ενσωμάτωση Metaplex
[candy machine](https://zenwiki.gitbook.io/solana-godot-sdk-docs/guides/setup-candy-machine).

- [Solana Godot Engine SDK](https://github.com/Virus-Axel/godot-solana-sdk)
- [Τεκμηρίωση](https://zenwiki.gitbook.io/solana-godot-sdk-docs)
- [Επίδειξη](https://github.com/ZenRepublic/GodotSolanaSDKDemoPackage)
- [Οδηγός](https://www.youtube.com/watch?v=tszFPInYmXQ)
- [Συντηρητής](https://twitter.com/ZenRepublicNDM)

## Turbo.Computer - Μηχανή παιχνιδιών Rust

Το Turbo είναι μια μηχανή παιχνιδιών γραμμένη σε rust από το μηδέν με επίκεντρο
το Solana, με ελαφριά αρχιτεκτονική και γρήγορους χρόνους επανάληψης. Είναι
φιλικό προς τους αρχάριους και έρχεται με πλήρη υποστήριξη Solana RPC. Μπορείτε
ακόμη να χρησιμοποιήσετε τα εργαλεία τεχνητής νοημοσύνης για να δημιουργήσετε
ολοκληρωμένα παιχνίδια.

- [Ιστοσελίδα](https://turbo.computer/)
- [Τεκμηρίωση](https://docs.turbo.computer/)
- [Λογαριασμός Twitter](https://twitter.com/makegamesfast)
- [Συντηρητής](https://twitter.com/jozanza)

# Honeycomb Protocol

Το Honeycomb παρέχει μια σειρά από προγράμματα onchain και εργαλεία συμπίεσης
κατάστασης που χειρίζονται όλες τις κρίσιμες λειτουργίες κύκλου ζωής παιχνιδιών
και τη συνθεσιμότητα εντός του Solana Virtual Machine (SVM). Υποστηρίζει πολλές
λειτουργίες NFT όπως staking, αποστολές, loot boxes, προφίλ παικτών, συμπίεση
κατάστασης, αυτόματη έγκριση συναλλαγών και άλλα.

- [Λογαριασμός Twitter](https://twitter.com/honeycomb_prtcl)
- [Τεκμηρίωση](https://docs.honeycombprotocol.com/)

# Unreal SDKs

Η Unreal Engine είναι γνωστή για τα εξαιρετικά οπτικά της και το πλαίσιο
προγραμματισμού βασισμένο σε κόμβους. Υπάρχουν πολλαπλά Solana SDKs που
συντηρούνται από διαφορετικές εταιρείες.

### Varmeta Unreal SDK

Η VAR META παρέχει ένα SDK ανοιχτού κώδικα για την Unreal για την ενσωμάτωση
λειτουργιών του blockchain Solana σε έργα Unreal Engine. Επιτρέπει στους
προγραμματιστές να αλληλεπιδρούν με τα έξυπνα συμβόλαια του Solana, να
διαχειρίζονται πορτοφόλια και να εκτελούν συναλλαγές απευθείας μέσα στην Unreal
Engine.

- [Unreal SDK από την Varmeta](https://github.com/VAR-META-Tech/UnrealSolanaSDK)
- [Συντηρητής](https://www.var-meta.com/)

### Bitfrost Unreal SDK

Το Bitfrost αναπτύσσει ένα Unreal SDK που έχει ήδη χρησιμοποιηθεί σε πολλαπλά
παιχνίδια που δημιουργήθηκαν κατά τη διάρκεια πρόσφατων game jams του Solana.
Περιλαμβάνει υποστήριξη `solnet` σε C++ και Blueprints, minting metaplex NFTs,
παραδείγματα payment processor σε game wallet και πολλά άλλα.

- [Solana Unreal SDK από την Bitfrost](https://github.com/Bifrost-Technologies/Solana-Unreal-SDK)
- [Οδηγός](https://www.youtube.com/watch?v=S8fm8mFeUkk)
- [Συντηρητής](https://twitter.com/BifrostTitan)

### Thugz Unreal SDK

Το Thugz είναι ένα τεχνολογικό στούντιο και έργο NFT που επίσης συντηρεί ως
ανοιχτού κώδικα ένα Unreal SDK για το Solana. Έρχεται με πολλές λειτουργίες
εστιασμένες στα NFT και το plugin είναι συμβατό και με iOS και Android.

- [Thugz Blockchain plugin](https://www.fab.com/listings/a7e220ba-93e5-4a3b-99e7-17b13da76f05)
- [Repository](https://github.com/ThugzLabs/Thugz-BC-Plugin-Packaged-for-UE5.4)
- [Documentation](https://thugzblockchainplugin-docs.vercel.app/)
- [Video Tutorial](https://www.youtube.com/watch?v=dS7sTZd_E9U&ab_channel=ThugzNFT)

### Star Atlas Foundation Kit

Η ομάδα του [Star Atlas](https://staratlas.com/) έκανε διαθέσιμο ως ανοιχτό
κώδικα ένα μέρος του SDK τους με την ονομασία _Foundation Kit_. Δεν συντηρείται
ενεργά αλλά μπορεί να αποτελέσει ένα σημείο εκκίνησης για το έργο σας.

- [Star Atlas Unreal SDK](https://github.com/staratlasmeta/FoundationKit)
- [Tutorial](https://www.youtube.com/watch?v=S8fm8mFeUkk)

## Next.js/React + Anchor

Ένας από τους ευκολότερους τρόπους για να αναπτύξετε στο Solana είναι
χρησιμοποιώντας το JavaScript framework Web3js σε συνδυασμό με το framework
Solana Anchor. Για πιο περίπλοκα παιχνίδια, συνιστούμε να εξετάσετε τη χρήση
μιας Game Engine όπως το [Unity](#unity-sdk) ή το [Unreal](#unreal-sdks).

Ο ταχύτερος τρόπος για να δημιουργήσετε ένα παιχνίδι βασισμένο σε Next.js:

```shell
npx create-solana-game your-game-name
```

Αυτό θα δημιουργήσει μια εξαιρετική εφαρμογή εκκίνησης με υποστήριξη
`wallet-adapter`, ένα πρόγραμμα Anchor, μια εφαρμογή React και ένα Unity client
ήδη διαμορφωμένα να λειτουργούν μαζί. Ένα πλεονέκτημα της χρήσης του Next.js
είναι ότι μπορείτε να χρησιμοποιήσετε τον ίδιο κώδικα στο backend και στο
frontend, επιταχύνοντας την ανάπτυξη.

- [Web3.js](https://solana.com/docs/clients/javascript)
- [Video Tutorial](https://www.youtube.com/watch?v=fnhivg_pemI&t=1s&ab_channel=Solana)

> Αν εργάζεστε σε ένα Solana SDK και θέλετε να προσθέσετε το preset σας στο
> preset μπορείτε να ανοίξετε ένα PR εδώ:
> [Solana games preset](https://github.com/solana-developers/solana-game-preset)

## Phaser

Ένα γρήγορο, δωρεάν και διασκεδαστικό framework ανοιχτού κώδικα για παιχνίδια
που λειτουργούν σε browser με Canvas και WebGL. Το [Phaser](https://phaser.io/)
είναι ένας εξαιρετικός τρόπος για να ξεκινήσετε με την ανάπτυξη παιχνιδιών στο
Solana. Το Phaser Solana Platformer Template είναι ένα εξαιρετικό σημείο
εκκίνησης για το παιχνίδι σας.

- [Source Code](https://github.com/Bread-Heads-NFT/solana-phaser-template)
- [Maintainer](https://twitter.com/blockiosaurus)

## Flutter

Το Flutter είναι ένα framework ανοιχτού κώδικα, που συντηρείται από την Google,
για τη δημιουργία όμορφων, εγγενώς μεταγλωττισμένων εφαρμογών πολλαπλών
πλατφορμών από μία ενιαία βάση κώδικα. Το Solana Flutter SDK συντηρείται από την
ομάδα [Espresso Cash](https://www.espressocash.com/).

- [Πηγαίος Κώδικας](https://github.com/espresso-cash/espresso-cash-public)

## Python

Η Python είναι μια εύκολη στην εκμάθηση και ευρέως χρησιμοποιούμενη γλώσσα
προγραμματισμού (συχνά χρησιμοποιείται σε εφαρμογές μηχανικής μάθησης). Το
πλαίσιο Seahorse επιτρέπει στους προγραμματιστές να γράφουν προγράμματα Solana
σε Python. Το Seahorse είναι χτισμένο πάνω στο πλαίσιο Anchor και μετατρέπει τον
κώδικα Python σε κώδικα rust βασισμένο στο Anchor. Το Seahorse βρίσκεται αυτή τη
στιγμή σε έκδοση beta.

- [Τεκμηρίωση](https://www.seahorse.dev/)
- [Github Repo](https://github.com/solana-developers/seahorse)
- [Παράδειγμα Anchor Playground](https://beta.solpg.io/tutorials/hello-seahorse)

## Native C#

Η αρχική μεταφορά του Web3js σε C#. Διαθέτει πολλές λειτουργίες όπως συναλλαγές,
λειτουργίες RPC και δημιουργία κώδικα πελάτη Anchor.

- [Πηγή και Τεκμηρίωση](https://github.com/bmresearch/Solnet/blob/master/docs/articles/getting_started.md)
