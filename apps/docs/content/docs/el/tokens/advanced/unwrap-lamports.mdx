## Τι Κάνει η Αναδίπλωση Lamports;

_rs`UnwrapLamports`_ μεταφέρει SOL έξω από ένα native (wrapped SOL) token
account σε έναν λογαριασμό προορισμού ως lamports. Μπορείτε να αναδιπλώσετε το
πλήρες υπόλοιπο ή ένα συγκεκριμένο ποσό, και το token account παραμένει ανοιχτό.

Αυτό είναι πιο ευέλικτο από το κλείσιμο ενός wrapped SOL λογαριασμού: μια μερική
αναδίπλωση σάς επιτρέπει να αναλάβετε κάποιο SOL διατηρώντας τον λογαριασμό
χρηματοδοτημένο για μελλοντική χρήση.

### Αναφορά Πηγής

| Στοιχείο                      | Περιγραφή                    | Πηγή                                                                                                         |
| ----------------------------- | ---------------------------- | ------------------------------------------------------------------------------------------------------------ |
| _rs`UnwrapLamports`_          | Η εντολή (discriminator 45). | [Πηγή](https://github.com/solana-program/token/blob/main/pinocchio/interface/src/instruction.rs)             |
| _rs`process_unwrap_lamports`_ | Λογική επεξεργαστή.          | [Πηγή](https://github.com/solana-program/token/blob/main/pinocchio/program/src/processor/unwrap_lamports.rs) |

## Λογαριασμοί

| Λογαριασμός   | Περιγραφή                                                              |
| ------------- | ---------------------------------------------------------------------- |
| `source`      | Εγγράψιμο. Το native (wrapped SOL) token account προς αναδίπλωση.      |
| `destination` | Εγγράψιμο. Λαμβάνει τα αναδιπλωμένα lamports.                          |
| `authority`   | Ο κάτοχος του λογαριασμού ή ένας εκπρόσωπος (ή υπογράφοντες multisig). |

<Callout type="warn">

Η πηγή πρέπει να είναι native token account· άλλα token accounts επιστρέφουν
_rs`NonNativeNotSupported`_. Ένας εκπρόσωπος μπορεί να αναδιπλώσει έως το
εξουσιοδοτημένο ποσό του.

</Callout>

## Πώς να Αναδιπλώσετε Lamports

Παραλείψτε το _ts`amount`_ για να αναδιπλώσετε το πλήρες υπόλοιπο, ή ορίστε το
για να αναδιπλώσετε συγκεκριμένο αριθμό lamports. Το παράδειγμα χρησιμοποιεί
έναν διαμορφωμένο [`@solana/kit`](/docs/clients/official/javascript#solana-kit)
client — δείτε [Μεταφορά Tokens](/docs/tokens/basics/transfer-tokens) για τη
ρύθμιση του client.

```ts title="Unwrap wrapped SOL (Kit)"
import { getUnwrapLamportsInstruction } from "@solana-program/token";

// source: the native (wrapped SOL) token account to unwrap.
// destination: address that receives the unwrapped lamports.
// authority: a TransactionSigner (the account owner or a delegate).

// Unwrap a specific number of lamports.
const unwrapSome = getUnwrapLamportsInstruction({
  source,
  destination,
  authority,
  amount: 500_000_000n
});

// Or omit `amount` to unwrap the entire balance.
const unwrapAll = getUnwrapLamportsInstruction({
  source,
  destination,
  authority
});

// Send whichever instruction you need.
await client.sendTransaction([unwrapSome]);
```

Για αρχή multisig, περάστε τους συν-υπογράφοντες με _ts`multiSigners`_.
