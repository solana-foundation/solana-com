Πέρα από τις κοινές λειτουργίες δημιουργίας, έκδοσης και μεταφοράς, το Token
Program καλείται επίσης από on-chain προγράμματα μέσω CPI και περιλαμβάνει
εντολές για την ανάκτηση lamports:

<Cards>
  <Card
    title="Κλήση του Token Program μέσω CPI"
    href="/docs/tokens/advanced/cpi"
  >
    Καλέστε το Token Program από ένα on-chain πρόγραμμα — μεταφέρετε, εκδώστε και καταστρέψτε
    tokens, υπογράψτε με μια αρχή PDA και ομαδοποιήστε εντολές σε ένα μόνο CPI.
  </Card>

<Card
  title="Ανάκτηση Πλεονάζοντων Lamports"
  href="/docs/tokens/advanced/withdraw-excess-lamports"
>
  Ανακτήστε SOL πάνω από το ελάχιστο όριο απαλλαγής ενοικίου από token accounts,
  mints και multisig accounts χωρίς να αλλάξετε τα υπόλοιπα token.
</Card>

  <Card title="Μετατροπή Lamports" href="/docs/tokens/advanced/unwrap-lamports">
    Μετατρέψτε το wrapped SOL ξανά σε lamports, εν μέρει ή εξ ολοκλήρου, χωρίς να κλείσετε
    το token account.
  </Card>
</Cards>

<Callout type="info">

_rs`WithdrawExcessLamports`_ και _rs`UnwrapLamports`_ είναι διαθέσιμα μέσω του
[`@solana-program/token`](https://www.npmjs.com/package/@solana-program/token)
client. Η κλήση του Token Program μέσω CPI ισχύει για on-chain προγράμματα.

</Callout>

## Ανάκτηση lamport

_rs`WithdrawExcessLamports`_ ανακτά SOL πάνω από το ελάχιστο όριο απαλλαγής
ενοικίου από token accounts, mints και multisig accounts. _rs`UnwrapLamports`_
μετατρέπει wrapped SOL σε ένα native token account ξανά σε lamports.
