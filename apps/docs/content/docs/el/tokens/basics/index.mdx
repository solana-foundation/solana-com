Η ενότητα Βασικά SPL Token καλύπτει τις βασικές οδηγίες για την αλληλεπίδραση με
τα SPL Tokens, εστιάζοντας σε λειτουργικότητα που είναι κοινή τόσο στο αρχικό
Token Program όσο και στο Token Extensions Program (Token 2022).

<Callout type="info">
  Ανατρέξτε στην [τεκμηρίωση
  Metaplex](https://developers.metaplex.com/token-metadata) για περισσότερες
  πληροφορίες σχετικά με το πώς να προσθέσετε metadata στα SPL Tokens.
</Callout>
