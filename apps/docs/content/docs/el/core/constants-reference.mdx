Αυτή η σελίδα αποτελεί μια ενιαία αναφορά για όλες τις σταθερές runtime που
αναφέρονται στις σελίδες βασικών εννοιών. Κάθε σταθερά συνδέεται με τον ορισμό
της στον πηγαίο κώδικα του πελάτη validator
[agave](https://github.com/anza-xyz/agave/tree/v3.1.8) ή του
[solana-sdk](https://github.com/anza-xyz/solana-sdk).

## Λογαριασμοί

| Σταθερά                                   | Τιμή                      | Περιγραφή                                                                   | Πηγή                                                                                                                          |
| ----------------------------------------- | ------------------------- | --------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| `MAX_ACCOUNT_DATA_LEN`                    | 10 MiB (10.485.760 bytes) | Μέγιστο μέγεθος δεδομένων λογαριασμού                                       | [`MAX_ACCOUNT_DATA_LEN`](https://github.com/anza-xyz/agave/blob/v3.1.8/transaction-context/src/lib.rs#L34)                    |
