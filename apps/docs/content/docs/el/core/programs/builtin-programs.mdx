## Το System Program

Το System Program (`11111111111111111111111111111111`) είναι το μόνο πρόγραμμα
που μπορεί να δημιουργήσει νέους λογαριασμούς. Όλοι οι νέοι λογαριασμοί ανήκουν
αρχικά στο System Program, αν και η κυριότητα συνήθως ανατίθεται εκ νέου κατά τη
δημιουργία. Το System Program καταναλώνει
[`DEFAULT_COMPUTE_UNITS`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L282)
(150 CUs) ανά εντολή.

### Αναφορά εντολών του System Program

| Εντολή                                                                                                                 | Περιγραφή                                                                                                                                                                                                                                                                 |
| ---------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [`CreateAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L295)          | Δημιουργεί νέο λογαριασμό. Εκχωρεί `space` bytes, αναθέτει `owner` και μεταφέρει `lamports` από τον πληρωτή. Ο νέος λογαριασμός και ο πληρωτής πρέπει να υπογράψουν.                                                                                                      |
| [`CreateAccountWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L318)  | Όπως το `CreateAccount`, αλλά παράγει τη διεύθυνση του νέου λογαριασμού από το pubkey ενός λογαριασμού, ένα seed string και το πρόγραμμα κατόχου. Ο λογαριασμός του οποίου το pubkey χρησιμοποιήθηκε για την παραγωγή της διεύθυνσης και ο πληρωτής πρέπει να υπογράψουν. |
| [`Assign`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L343)                 | Ορίζει τον κάτοχο του λογαριασμού. Ο λογαριασμός πρέπει να υπογράψει. Καμία ενέργεια αν είναι ήδη αυτός ο κάτοχος.                                                                                                                                                        |
| [`AssignWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L484)         | Όπως το `Assign`, αλλά για διεύθυνση που παράγεται από seed. Ο λογαριασμός του οποίου το pubkey χρησιμοποιήθηκε για την παραγωγή της διεύθυνσης πρέπει να υπογράψει.                                                                                                      |
| [`Transfer`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L353)               | Μεταφέρει lamports μεταξύ λογαριασμών. Ο αποστολέας πρέπει να υπογράψει.                                                                                                                                                                                                  |
| [`TransferWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L357)       | Όπως το `Transfer`, αλλά από διεύθυνση που παράγεται από seed. Ο λογαριασμός του οποίου το pubkey χρησιμοποιήθηκε για την παραγωγή της διεύθυνσης πρέπει να υπογράψει.                                                                                                    |
| [`Allocate`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L452)               | Ορίζει το μήκος δεδομένων του λογαριασμού. Ο λογαριασμός πρέπει να υπογράψει, να έχει κενά δεδομένα και να ανήκει στο System Program. Μέγιστο 10 MiB.                                                                                                                     |
| [`AllocateWithSeed`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L462)       | Όπως το `Allocate`, αλλά για διεύθυνση που παράγεται από seed. Επίσης αναθέτει τον κάτοχο. Ο λογαριασμός του οποίου το pubkey χρησιμοποιήθηκε για την παραγωγή της διεύθυνσης πρέπει να υπογράψει.                                                                        |
| [`AdvanceNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L374)    | Προωθεί την τιμή nonce στο τρέχον blockhash. Η εξουσία nonce πρέπει να υπογράψει.                                                                                                                                                                                         |
| [`InitializeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L412) | Αρχικοποιεί έναν λογαριασμό ως λογαριασμό nonce με τη δεδομένη εξουσία. Ο λογαριασμός πρέπει να είναι απαλλαγμένος από rent.                                                                                                                                              |
| [`AuthorizeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L432)  | Αλλάζει την εξουσία nonce. Η τρέχουσα εξουσία πρέπει να υπογράψει.                                                                                                                                                                                                        |
| [`WithdrawNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L392)   | Αποσύρει lamports από λογαριασμό nonce. Η εξουσία πρέπει να υπογράψει. Αν δεν κλείνει, το υπόλοιπο που απομένει πρέπει να καλύπτει το rent.                                                                                                                               |
| [`UpgradeNonceAccount`](https://github.com/anza-xyz/agave/blob/v3.1.8/programs/system/src/system_processor.rs#L437)    | Αναβαθμίζει έναν λογαριασμό nonce από παλαιού τύπου σε τρέχουσα μορφή έκδοσης. Ο λογαριασμός πρέπει να είναι εγγράψιμος και να περιέχει παλαιού τύπου έκδοση nonce.                                                                                                       |

## Προγράμματα φόρτωσης

Κάθε πρόγραμμα ανήκει σε ένα πρόγραμμα φόρτωσης. Τα προγράμματα φόρτωσης
χειρίζονται την ανάπτυξη, την αναβάθμιση και την εκτέλεση των προγραμμάτων που
κατέχουν. Δείτε
[Προγράμματα φόρτωσης](/docs/core/programs/program-deployment#loader-programs)
για την πλήρη λίστα των προγραμμάτων φόρτωσης και των διευθύνσεών τους.
