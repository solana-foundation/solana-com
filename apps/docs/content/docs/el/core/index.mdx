Επτά έννοιες αποτελούν τη βάση της ανάπτυξης στο Solana. Διαβάστε τα Accounts,
Programs, Instructions, Transactions και Fees με αυτή τη σειρά, καθώς το καθένα
βασίζεται στο προηγούμενο. Διαβάστε τα Accounts και Programs πριν από τα PDAs.
Διαβάστε τα Programs, Instructions και PDAs πριν από τα CPIs.

<Cards>
  <Card title="Accounts" href="/docs/core/accounts">
    Το Solana αποθηκεύει όλη την κατάσταση σε λογαριασμούς, ένα key-value store
    με κλειδιά διευθύνσεων 32 bytes. Κάθε λογαριασμός έχει πέντε πεδία:
    lamports, data, owner, executable και rent_epoch.
  </Card>
  <Card title="Programs" href="/docs/core/programs">
    Τα smart contracts του Solana, μεταγλωττισμένα σε sBPF. Τα προγράμματα είναι
    stateless και όλη η μεταβλητή κατάσταση βρίσκεται σε ξεχωριστούς
    λογαριασμούς δεδομένων που περνούν μέσω εντολών.
  </Card>
  <Card title="Instructions" href="/docs/core/instructions">
    Ένα αίτημα για την εκτέλεση ενός προγράμματος, που περιέχει ένα program ID,
    λίστα λογαριασμών και αδιαφανή bytes δεδομένων. Πολλαπλές εντολές
    συντίθενται σε μία μόνο συναλλαγή.
  </Card>
  <Card title="Transactions" href="/docs/core/transactions">
    Η ατομική μονάδα εκτέλεσης στο Solana. Περιέχει υπογραφές και ένα μήνυμα
    (διευθύνσεις λογαριασμών, πρόσφατο blockhash και μεταγλωττισμένες εντολές).
    Μέγιστο 1232 bytes.
  </Card>
  <Card title="Fees" href="/docs/core/fees">
    Βασική χρέωση 5.000 lamports ανά υπογραφή, συν μία προαιρετική χρέωση
    προτεραιότητας βάσει των compute units που ζητούνται. Μέγιστο 1,4M compute
    units ανά συναλλαγή.
  </Card>
  <Card title="Program Derived Addresses (PDAs)" href="/docs/core/pda">
    Ντετερμινιστικές διευθύνσεις που προέρχονται από ένα program ID και seeds.
    Μόνο το πρόγραμμα που τις κατέχει μπορεί να υπογράψει εκ μέρους ενός PDA
    μέσω invoke_signed.
  </Card>
  <Card title="Cross-Program Invocations (CPIs)" href="/docs/core/cpi">
    Ένα πρόγραμμα που καλεί ένα άλλο μέσω invoke ή invoke_signed. Τα δικαιώματα
    signer και writable επεκτείνονται στον καλούμενο. Μέγιστο βάθος στοίβας
    εντολών 5 (top-level συν CPIs).
  </Card>
</Cards>
