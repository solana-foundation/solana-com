Το Surfpool είναι μια άμεση αντικατάσταση του `solana-test-validator`,
σχεδιασμένο ειδικά για να προσφέρει την καλύτερη δυνατή εμπειρία στους
προγραμματιστές που αναπτύσσουν εφαρμογές στο **Solana**.

Το βασικό του χαρακτηριστικό είναι ότι επιτρέπει στους προγραμματιστές να
προσομοιώνουν τοπικά τα προγράμματά τους χρησιμοποιώντας λογαριασμούς του
Mainnet που ανακτώνται εν ώρα ανάγκης.

Το Surfpool εισάγει επίσης απρόσκοπτα το Infrastructure as Code σε έργα
βασισμένα σε Anchor ή Pinocchio, επιτρέποντας αναπαραγώγιμες, ελέγξιμες και
ασφαλείς αναπτύξεις σε οποιοδήποτε δίκτυο Solana—ιδιωτικό ή δημόσιο.
