Το Surfnet είναι ο τοπικός προσομοιωτής δικτύου Solana του Surfpool,
σχεδιασμένος να παρέχει στους developers ένα ισχυρό περιβάλλον δοκιμών που
μιμείται τη συμπεριφορά του blockchain Solana χωρίς την επιβάρυνση ενός πλήρους
κόμβου validator.

## Ξεκινώντας

Εκκινήστε μια παρουσία Surfnet χρησιμοποιώντας το Surfpool CLI:

```bash
surfpool start
```

Αυτό θα εκκινήσει μια τοπική παρουσία Surfnet στο `http://localhost:8899` με όλα
τα endpoints RPC διαθέσιμα.

## Βασικά Χαρακτηριστικά

- **Πλήρης Συμβατότητα RPC**: Υποστηρίζει το πλήρες Solana RPC API
- **Γρήγορη Επανάληψη**: Δεν χρειάζεται να περιμένετε για επιβεβαιώσεις block
- **Χειρισμός Κατάστασης**: Cheatcodes για άμεση τροποποίηση της κατάστασης του
  blockchain
- **Mainnet Fork**: Fork από το mainnet για δοκιμές με πραγματικούς λογαριασμούς
  και προγράμματα
- **Υποστήριξη Plugin Geyser**: Συνδέστε τα plugins Geyser σας για ροή δεδομένων
  σε πραγματικό χρόνο
