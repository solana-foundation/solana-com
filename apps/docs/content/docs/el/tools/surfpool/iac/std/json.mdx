## jq

Η συνάρτηση `jq` επιτρέπει την τεμαχίωση, το φιλτράρισμα και την αντιστοίχιση
δεδομένων JSON. Δείτε την [τεκμηρίωση jq](https://jqlang.github.io/jq/manual/)
για περισσότερες λεπτομέρειες.

### Είσοδοι

| Όνομα          | Τύπος            | Περιγραφή                                                                        |
| -------------- | ---------------- | -------------------------------------------------------------------------------- |
| `decoded_json` | string \| object | Ένα αντικείμενο JSON                                                             |
| `query`        | string           | Ένα ερώτημα JSON. Δείτε την [τεκμηρίωση jq](https://jqlang.github.io/jq/manual/) |

### Έξοδος

| Όνομα   | Τύπος         | Περιγραφή                         |
| ------- | ------------- | --------------------------------- |
| `value` | array[string] | Το αποτέλεσμα του ερωτήματος `jq` |

```hcl
output "message" {
    value = jq("{ \"message\": \"Hello world!\" }", ".message")
}
// > message: Hello world!
```
