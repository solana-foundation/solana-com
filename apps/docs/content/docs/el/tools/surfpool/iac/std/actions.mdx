Αυτές οι ενέργειες είναι διαθέσιμες σε όλα τα Runbooks ως μέρος της τυπικής
βιβλιοθήκης.

## send_http_request

Η ενέργεια `std::send_http_request` πραγματοποιεί ένα αίτημα HTTP στην
καθορισμένη διεύθυνση URL.

### Είσοδοι

| Όνομα        | Απαιτείται  | Τύπος   | Περιγραφή                                                |
| ------------ | ----------- | ------- | -------------------------------------------------------- |
| `url`        | απαιτείται  | string  | Η διεύθυνση URL για αποστολή του αιτήματος               |
| `method`     | προαιρετικό | string  | Η μέθοδος HTTP (GET, POST, PUT, DELETE). Προεπιλογή: GET |
| `headers`    | προαιρετικό | map     | Κεφαλίδες HTTP για συμπερίληψη στο αίτημα                |
| `body`       | προαιρετικό | string  | Το σώμα του αιτήματος (για αιτήματα POST/PUT)            |
| `timeout_ms` | προαιρετικό | integer | Χρονικό όριο αιτήματος σε χιλιοστά του δευτερολέπτου     |

### Έξοδοι

| Όνομα         | Τύπος   | Περιγραφή                 |
| ------------- | ------- | ------------------------- |
| `status_code` | integer | Ο κωδικός κατάστασης HTTP |
| `body`        | string  | Το σώμα της απόκρισης     |
| `headers`     | map     | Οι κεφαλίδες απόκρισης    |

```hcl
action "api_call" "std::send_http_request" {
    description = "Fetch data from API"
    url = "https://api.example.com/data"
    method = "GET"
    headers = {
        "Authorization" = "Bearer ${variable.token}"
    }
}

output "response" {
    value = action.api_call.body
}

output "status" {
    value = action.api_call.status_code
}
```

### Παράδειγμα Αιτήματος POST

```hcl
action "create_resource" "std::send_http_request" {
    description = "Create a new resource"
    url = "https://api.example.com/resources"
    method = "POST"
    headers = {
        "Content-Type" = "application/json"
    }
    body = {
        "name" = variable.name,
        "value" = variable.value
    }
}
```
