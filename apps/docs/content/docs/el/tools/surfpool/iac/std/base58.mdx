## encode_base58

`encode_base58` κωδικοποιεί ένα buffer ή αλφαριθμητικό ως συμβολοσειρά base58.

### Είσοδοι

| Όνομα   | Τύπος                 | Περιγραφή                                                                                                                                                            |
| ------- | --------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | Το buffer ή αλφαριθμητικό προς κωδικοποίηση. Τα αλφαριθμητικά που αρχίζουν με '0x' αποκωδικοποιούνται ως hex· διαφορετικά, χρησιμοποιούνται ακατέργαστα bytes UTF-8. |

### Έξοδος

| Όνομα   | Τύπος  | Περιγραφή                                        |
| ------- | ------ | ------------------------------------------------ |
| `value` | string | Η είσοδος, κωδικοποιημένη ως συμβολοσειρά base58 |

```hcl
output "encoded" {
    value = encode_base58("0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8")
}
// > encoded: CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF
```

## decode_base58

`decode_base58` αποκωδικοποιεί μια κωδικοποιημένη συμβολοσειρά base58 και
επιστρέφει το αποτέλεσμα ως buffer.

### Είσοδοι

| Όνομα           | Τύπος  | Περιγραφή                                  |
| --------------- | ------ | ------------------------------------------ |
| `base58_string` | string | Η συμβολοσειρά base58 προς αποκωδικοποίηση |

### Έξοδος

| Όνομα   | Τύπος  | Περιγραφή                                          |
| ------- | ------ | -------------------------------------------------- |
| `value` | buffer | Η αποκωδικοποιημένη συμβολοσειρά base58, ως buffer |

```hcl
output "decoded" {
    value = decode_base58("CctJBuDbaFtojUWfQ3iEcq77eFDjojCtoS4Q59f6bUtF")
}
// > decoded: 0xaca1e2ae0c54a9a8f12da5dde27a93bb5ff94aeef722b1e474a16318234f83c8
```
