## encode_hex

`encode_hex` κωδικοποιεί ένα buffer ή μια συμβολοσειρά ως δεκαεξαδική
συμβολοσειρά με πρόθεμα 0x.

### Είσοδοι

| Όνομα   | Τύπος                 | Περιγραφή                                                                                                                                                          |
| ------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `value` | buffer, string, addon | Το buffer ή η συμβολοσειρά προς κωδικοποίηση. Συμβολοσειρές που αρχίζουν με '0x' αποκωδικοποιούνται ως hex· διαφορετικά, χρησιμοποιούνται ακατέργαστα bytes UTF-8. |

### Έξοδος

| Όνομα   | Τύπος  | Περιγραφή                                                |
| ------- | ------ | -------------------------------------------------------- |
| `value` | string | Η είσοδος στη δεκαεξαδική της αναπαράσταση με πρόθεμα 0x |

```hcl
output "encoded_hex" {
    value = encode_hex("hello, world")
}
// > encoded_hex: 0x68656c6c6f2c20776f726c64
```

## decode_hex

`decode_hex` αποκωδικοποιεί μια δεκαεξαδική συμβολοσειρά και επιστρέφει το
αποτέλεσμα ως buffer.

### Είσοδοι

| Όνομα        | Τύπος  | Περιγραφή                                                               |
| ------------ | ------ | ----------------------------------------------------------------------- |
| `hex_string` | string | Η δεκαεξαδική συμβολοσειρά προς αποκωδικοποίηση (με ή χωρίς πρόθεμα 0x) |

### Έξοδος

| Όνομα   | Τύπος  | Περιγραφή                                              |
| ------- | ------ | ------------------------------------------------------ |
| `value` | buffer | Η αποκωδικοποιημένη δεκαεξαδική συμβολοσειρά ως buffer |

```hcl
output "decoded_hex" {
    value = decode_hex("0x68656c6c6f2c20776f726c64")
}
// > decoded_hex: 0x68656c6c6f2c20776f726c64
```
