Η τυπική βιβλιοθήκη παρέχει βασικές συναρτήσεις και ενέργειες που είναι
διαθέσιμες σε όλα τα Runbooks, ανεξάρτητα από το ποια πρόσθετα blockchain είναι
ενεργοποιημένα.

## Συναρτήσεις

Η τυπική βιβλιοθήκη περιλαμβάνει βοηθητικές συναρτήσεις για:

- **Τελεστές**: Μαθηματικές πράξεις και συγκρίσεις
- **Hash**: Συναρτήσεις κατακερματισμού (SHA256, Keccak256, κ.λπ.)
- **Hex**: Κωδικοποίηση και αποκωδικοποίηση δεκαεξαδικών
- **Crypto**: Κρυπτογραφικές λειτουργίες
- **JSON**: Κωδικοποίηση και αποκωδικοποίηση JSON
- **List**: Συναρτήσεις διαχείρισης λιστών
- **Assertions**: Ισότητα και ελέγχοι συνθηκών
- **Base64**: Κωδικοποίηση και αποκωδικοποίηση Base64

[Προβολή όλων των συναρτήσεων](/docs/tools/surfpool/iac/std/functions)

## Ενέργειες

Η τυπική βιβλιοθήκη περιλαμβάνει ενέργειες για:

- **HTTP**: Εκτέλεση αιτημάτων HTTP
- **Λειτουργίες Αρχείων**: Ανάγνωση και εγγραφή αρχείων

[Προβολή όλων των ενεργειών](/docs/tools/surfpool/iac/std/actions)
