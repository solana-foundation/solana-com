## index

`index` λαμβάνει την καταχώρηση από μια λίστα στον καθορισμένο δείκτη.

### Είσοδοι

| Όνομα   | Τύπος                                            | Περιγραφή                                         |
| ------- | ------------------------------------------------ | ------------------------------------------------- |
| `list`  | array[string] \| array[integer] \| array[buffer] | Η λίστα από την οποία θα ανακτηθεί μια καταχώρηση |
| `index` | integer                                          | Ο δείκτης της καταχώρησης προς ανάκτηση           |

### Έξοδος

| Όνομα   | Τύπος  | Περιγραφή                                         |
| ------- | ------ | ------------------------------------------------- |
| `value` | string | Η καταχώρηση από τη λίστα στον καθορισμένο δείκτη |

```hcl
output "entry" {
    value = index(['a', 'b', 'c'], 1)
}
// > entry: b
```
