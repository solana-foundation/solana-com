---
title: Επισκόπηση SVM Addon
description: Add-on για Solana και SVM Συμβατά Blockchains (beta)
---

Το πρόσθετο SVM `txtx` επιτρέπει τη δημιουργία Runbooks που αλληλεπιδρούν με το
Solana και συμβατά blockchains SVM. Το πρόσθετο παρέχει βοηθητικές συναρτήσεις
που σας επιτρέπουν να αναπτύσσετε προγράμματα Anchor και να κωδικοποιείτε
κλήσεις εντολών σύμφωνα με τα IDL προγραμμάτων. Οι ενέργειες μπορούν να
χρησιμοποιηθούν για τη δημιουργία έγκυρων συναλλαγών μεταφοράς, κλήσης
προγράμματος και ανάπτυξης προγράμματος, οι οποίες μπορούν να υπογραφούν μέσω
φράσης mnemonic, μυστικού κλειδιού ή μέσω του υπογράφοντα του προγράμματος
περιήγησής σας.

## Γρήγορη Εκκίνηση

Για να χρησιμοποιήσετε το SVM addon, δηλώστε το στο Runbook σας:

```hcl
addon "svm" {
  network_id = "devnet"
  rpc_api_url = "https://api.devnet.solana.com"
}
```

## Δυνατότητες

- **Ανάπτυξη Προγράμματος**: Ανάπτυξη προγραμμάτων Anchor και εγγενών
  προγραμμάτων Solana
- **Δημιουργία Εντολών**: Κωδικοποίηση κλήσεων εντολών χρησιμοποιώντας IDL
  προγραμμάτων
- **Υπογραφή Συναλλαγών**: Υποστήριξη για mnemonics, μυστικά κλειδιά και web
  wallets
- **Διαχείριση Λογαριασμών**: Ερώτηση και διαχείριση λογαριασμών Solana
