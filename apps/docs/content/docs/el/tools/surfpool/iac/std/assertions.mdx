## assert_eq

`assert_eq` ελέγχει ότι δύο τιμές είναι ίσες.

### Είσοδοι

| Όνομα   | Τύπος | Περιγραφή                    |
| ------- | ----- | ---------------------------- |
| `left`  | any   | Μια τιμή για σύγκριση        |
| `right` | any   | Η τιμή για σύγκριση με αυτήν |

### Έξοδος

| Όνομα   | Τύπος                 | Περιγραφή                     |
| ------- | --------------------- | ----------------------------- |
| `value` | addon(std::assertion) | Το αποτέλεσμα της διεκδίκησης |

```hcl
output "assertion" {
    value = std::assert_eq(action.example.result, 1)
}
```

| ------- | ---------------- | ---------------------------------------------------------- |
| `left`  | integer \| float | Ένας ακέραιος ή κινητής υποδιαστολής για σύγκριση          |
