## and_bool

`and_bool` επιστρέφει το δυαδικό AND των ορισμάτων αριστερής και δεξιάς πλευράς.

### Είσοδοι

| Όνομα | Τύπος | Περιγραφή       |
| ----- | ----- | --------------- |
| `lhs` | bool  | Μια λογική τιμή |
| `rhs` | bool  | Μια λογική τιμή |

### Έξοδος

| Όνομα   | Τύπος | Περιγραφή                                           |
| ------- | ----- | --------------------------------------------------- |
| `value` | bool  | Το αποτέλεσμα δυαδικού AND μεταξύ των δύο ορισμάτων |

```hcl
output "my_bool" {
  value = false && true
}
// > my_bool: false
```

## div

`div` επιστρέφει το ακέραιο πηλίκο της διαίρεσης του ορίσματος αριστερής πλευράς
με το όρισμα δεξιάς πλευράς, στρογγυλοποιώντας οποιοδήποτε υπόλοιπο προς τα κάτω
στον πλησιέστερο ακέραιο.

### Είσοδοι

| Όνομα | Τύπος   | Περιγραφή    |
| ----- | ------- | ------------ |
| `lhs` | integer | Ο διαιρετέος |
| `rhs` | integer | Ο διαιρέτης  |

### Έξοδος

| Όνομα   | Τύπος   | Περιγραφή                                                  |
| ------- | ------- | ---------------------------------------------------------- |
| `value` | integer | Το αποτέλεσμα της διαίρεσης του διαιρετέου με τον διαιρέτη |

```hcl
output "my_int" {
  value = 11 / -3
}
// > my_int: -3
```

## eq

`eq` επιστρέφει `true` αν τα ορίσματα αριστερής και δεξιάς πλευράς είναι ίσα και
`false` αν δεν είναι.

### Είσοδοι

| Όνομα | Τύπος | Περιγραφή        |
| ----- | ----- | ---------------- |
| `lhs` | any   | Οποιαδήποτε τιμή |
| `rhs` | any   | Οποιαδήποτε τιμή |

### Έξοδος

| Όνομα   | Τύπος | Περιγραφή                                             |
| ------- | ----- | ----------------------------------------------------- |
| `value` | bool  | Το αποτέλεσμα ελέγχου ισότητας μεταξύ των δύο εισόδων |

```hcl

  value = "arg" == "arg"
}

```
