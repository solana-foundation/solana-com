## encode_base64

`encode_base64` κωδικοποιεί ένα buffer ή μια συμβολοσειρά ως συμβολοσειρά
base64.

### Είσοδοι

| Όνομα   | Τύπος                 | Περιγραφή                                                                                                                                                             |
| ------- | --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `value` | buffer, string, addon | Το buffer ή η συμβολοσειρά προς κωδικοποίηση. Οι συμβολοσειρές που ξεκινούν με '0x' αποκωδικοποιούνται ως hex· διαφορετικά, χρησιμοποιούνται ακατέργαστα bytes UTF-8. |

### Έξοδος

| Όνομα   | Τύπος  | Περιγραφή                                        |
| ------- | ------ | ------------------------------------------------ |
| `value` | string | Η είσοδος, κωδικοποιημένη ως συμβολοσειρά base64 |

```hcl
output "encoded" {
    value = encode_base64("0x48656c6c6f20776f726c6421")
}
// > encoded: SGVsbG8gd29ybGQh
```

## decode_base64

`decode_base64` αποκωδικοποιεί μια κωδικοποιημένη συμβολοσειρά base64 και
επιστρέφει το αποτέλεσμα ως buffer.

### Είσοδοι

| Όνομα           | Τύπος  | Περιγραφή                                  |
| --------------- | ------ | ------------------------------------------ |
| `base64_string` | string | Η συμβολοσειρά base64 προς αποκωδικοποίηση |

### Έξοδος

| Όνομα   | Τύπος  | Περιγραφή                                          |
| ------- | ------ | -------------------------------------------------- |
| `value` | buffer | Η αποκωδικοποιημένη συμβολοσειρά base64, ως buffer |

```hcl
output "decoded" {
    value = decode_base64("SGVsbG8gd29ybGQh")
}
// > decoded: 0x48656c6c6f20776f726c6421
```
