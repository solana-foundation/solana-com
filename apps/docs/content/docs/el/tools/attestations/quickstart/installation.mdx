Ξεκινήστε με τα Solana Attestations εγκαθιστώντας τα απαραίτητα πακέτα και
ρυθμίζοντας το περιβάλλον ανάπτυξής σας.

## TypeScript/JavaScript

Εγκαταστήστε τη βιβλιοθήκη Solana Attestations χρησιμοποιώντας τον προτιμώμενο
διαχειριστή πακέτων σας:

```bash
# Using pnpm (recommended)
pnpm add sas-lib

# Using npm
npm install sas-lib

# Using yarn
yarn add sas-lib
```
