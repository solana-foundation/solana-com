Ξεκινήστε με τις Solana Attestations σε λίγα μόνο βήματα. Αυτός ο οδηγός θα σας
καθοδηγήσει στις βασικές έννοιες και θα σας βοηθήσει να δημιουργήσετε την πρώτη
σας βεβαίωση.

## Τι Θα Μάθετε

- Εγκατάσταση και ρύθμιση της βιβλιοθήκης Attestations
- Δημιουργία του πρώτου σας διαπιστευτηρίου (αρχή βεβαίωσης)
- Ορισμός σχήματος για τις βεβαιώσεις σας
- Έκδοση της πρώτης σας βεβαίωσης
- Ανάκτηση και επαλήθευση βεβαιώσεων

## Προαπαιτούμενα

- Βασικές γνώσεις ανάπτυξης Solana
- Node.js 16+ ή περιβάλλον ανάπτυξης Rust
- Πορτοφόλι Solana με λίγο SOL για τα τέλη συναλλαγών

## Βήματα Γρήγορης Έναρξης

### 1. Εγκατάσταση

Εγκαταστήστε τα απαραίτητα πακέτα και ρυθμίστε το περιβάλλον ανάπτυξής σας.

**[→ Οδηγός Εγκατάστασης](/docs/tools/attestations/quickstart/installation)**

### 2. Δημιουργία Διαπιστευτηρίου

Ρυθμίστε μια αρχή βεβαίωσης που μπορεί να εκδίδει βεβαιώσεις.

**[→ Οδηγός Δημιουργίας Διαπιστευτηρίου](/docs/tools/attestations/instructions/create-credential)**

### 3. Ορισμός Σχήματος

Δημιουργήστε ένα σχήμα που ορίζει τη δομή των βεβαιώσεών σας.

**[→ Οδηγός Δημιουργίας Σχήματος](/docs/tools/attestations/instructions/create-schema)**

### 4. Έκδοση Βεβαίωσης

Δημιουργήστε τον πρώτο σας επαληθεύσιμο ισχυρισμό χρησιμοποιώντας το
διαπιστευτήριο και το σχήμα.

**[→ Οδηγός Δημιουργίας Βεβαίωσης](/docs/tools/attestations/instructions/create-attestation)**

## Βασικές Έννοιες

### Διαπιστευτήρια

Ένα διαπιστευτήριο αντιπροσωπεύει μια αρχή βεβαίωσης. Καθορίζει ποιος μπορεί να
εκδίδει βεβαιώσεις και περιέχει έναν κατάλογο εξουσιοδοτημένων υπογραφόντων.

### Σχήματα

Τα σχήματα ορίζουν τη δομή και τους κανόνες επικύρωσης για τις βεβαιώσεις.
Καθορίζουν ποια πεδία περιέχει μια βεβαίωση και τους τύπους δεδομένων τους.

### Βεβαιώσεις

Οι βεβαιώσεις είναι επαληθεύσιμοι ισχυρισμοί ή δηλώσεις που δημιουργούνται από
εξουσιοδοτημένους υπογράφοντες κάτω από ένα διαπιστευτήριο, ακολουθώντας μια
συγκεκριμένη δομή σχήματος.

## Περιπτώσεις Χρήσης

Οι βεβαιώσεις μπορούν να χρησιμοποιηθούν για:

- **Επαλήθευση Ταυτότητας**: Επαληθεύστε τις ταυτότητες χρηστών και τη
  συμμόρφωση
- **Συστήματα Φήμης**: Δημιουργήστε βαθμολογίες εμπιστοσύνης και μετρικές φήμης
- **Πιστοποιήσεις**: Εκδώστε επαγγελματικά πιστοποιητικά και προσόντα
- **Συμμόρφωση**: Διασφαλίστε τη συμμόρφωση με κανονισμούς για
  χρηματοοικονομικές υπηρεσίες
- **Εφοδιαστική Αλυσίδα**: Παρακολουθήστε την αυθεντικότητα και την προέλευση
  των αγαθών
