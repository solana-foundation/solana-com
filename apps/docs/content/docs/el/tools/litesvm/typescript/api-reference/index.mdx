## Γρήγορη Αναφορά

### Ρύθμιση Plugin

```typescript
import { createClient, generateKeyPairSigner } from "@solana/kit";
import { litesvm } from "@solana/kit-plugin-litesvm";
import { signer } from "@solana/kit-plugin-signer";

const mySigner = await generateKeyPairSigner();
const client = createClient().use(signer(mySigner)).use(litesvm());
// client.svm: LiteSVM instance
// client.rpc: RPC compatibility layer
```

### Διαχείριση Λογαριασμών (client.svm)

| Μέθοδος                                | Περιγραφή                                |
| -------------------------------------- | ---------------------------------------- |
| `setAccount(account)`                  | Άμεσος ορισμός κατάστασης λογαριασμού    |
| `getAccount(address)`                  | Ανάκτηση δεδομένων λογαριασμού           |
| `getBalance(address)`                  | Λήψη υπολοίπου SOL                       |
| `airdrop(address, lamports)`           | Χρηματοδότηση λογαριασμού                |
| `minimumBalanceForRentExemption(size)` | Υπολογισμός ελάχιστου απαλλαγής ενοικίου |

### Διαχείριση Προγραμμάτων (client.svm)

| Μέθοδος                               | Περιγραφή                           |
| ------------------------------------- | ----------------------------------- |
| `addProgram(programId, bytes)`        | Φόρτωση προγράμματος από bytes      |
| `addProgramFromFile(programId, path)` | Φόρτωση προγράμματος από αρχείο .so |

### Συναλλαγές (client.svm)

| Μέθοδος                   | Περιγραφή                            |
| ------------------------- | ------------------------------------ |
| `sendTransaction(tx)`     | Εκτέλεση συναλλαγής                  |
| `simulateTransaction(tx)` | Προσομοίωση χωρίς αλλαγές κατάστασης |
| `latestBlockhash()`       | Λήψη τρέχοντος blockhash             |
| `expireBlockhash()`       | Μετάβαση σε νέο blockhash            |

### Διαμόρφωση (client.svm)

| Μέθοδος                     | Περιγραφή                                         |
| --------------------------- | ------------------------------------------------- |
| `withSigverify(bool)`       | Ενεργοποίηση/απενεργοποίηση επαλήθευσης υπογραφής |
| `withBlockhashCheck(bool)`  | Ενεργοποίηση/απενεργοποίηση επικύρωσης blockhash  |
| `withSysvars()`             | Ενεργοποίηση λογαριασμών sysvar                   |
| `withBuiltins()`            | Ενεργοποίηση ενσωματωμένων προγραμμάτων           |
| `withPrecompiles()`         | Ενεργοποίηση προμεταγλωττισμένων προγραμμάτων     |
| `withTransactionHistory(n)` | Αποθήκευση των τελευταίων n συναλλαγών            |
| `withComputeBudget(budget)` | Ορισμός ορίων υπολογισμού                         |
| `withLamports(amount)`      | Ορισμός προεπιλεγμένων lamport                    |
| `withLogBytesLimit(n)`      | Ορισμός ορίου εξόδου καταγραφής                   |

### Χρόνος & Sysvars (client.svm)

| Μέθοδος                    | Περιγραφή                             |
| -------------------------- | ------------------------------------- |
| `warpToSlot(slot)`         | Μετάβαση σε συγκεκριμένο slot         |
| `getClock()`               | Λήψη clock sysvar                     |
| `getRent()`                | Λήψη rent sysvar                      |
| `getEpochSchedule()`       | Λήψη χρονοδιαγράμματος epoch          |
| `getLastRestartSlot()`     | Λήψη τελευταίου slot επανεκκίνησης    |
| `setLastRestartSlot(slot)` | Ορισμός τελευταίου slot επανεκκίνησης |

### RPC (client.rpc)

| Μέθοδος                          | Περιγραφή                                      |
| -------------------------------- | ---------------------------------------------- |
| `getAccountInfo(address)`        | Ανάκτηση μεμονωμένου λογαριασμού (μόνο base64) |
| `getMultipleAccounts(addresses)` | Ανάκτηση πολλαπλών λογαριασμών                 |
| `getLatestBlockhash()`           | Λήψη τρέχοντος blockhash                       |
