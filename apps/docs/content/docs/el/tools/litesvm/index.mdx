  τα προγράμματά σας πιο γρήγορα χωρίς επιβάρυνση δικτύου ή διεργασίες
  validator.
---

Το LiteSVM είναι μια ελαφριά βιβλιοθήκη για τη δοκιμή προγραμμάτων Solana που
λειτουργεί δημιουργώντας μια ενσωματωμένη Solana VM βελτιστοποιημένη για
προγραμματιστές. Σε αντίθεση με άλλες προσεγγίσεις δοκιμών που εκκινούν
ξεχωριστές διεργασίες validator, το LiteSVM ενσωματώνει την VM μέσα στις δοκιμές
σας, κάνοντάς τες να εκτελούνται πολύ πιο γρήγορα. Σε μια περαιτέρω απομάκρυνση
από την παράδοση, διαθέτει ένα εργονομικό API με λογικές προεπιλογές και
εκτεταμένες δυνατότητες διαμόρφωσης για όσους το επιθυμούν.
