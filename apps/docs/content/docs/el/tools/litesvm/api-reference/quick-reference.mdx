## Γρήγορη Αναφορά

| Μέθοδος                                        | Περιγραφή                                                                                                                         |
| ---------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| `LiteSVM::new()`                               | Δημιουργεί νέο βασικό περιβάλλον δοκιμών LiteSVM                                                                                  |
| `LiteSVM::default()`                           | Επιστρέφει την «προεπιλεγμένη τιμή» για έναν τύπο                                                                                 |
| `airdrop(&Address, lamports)`                  | Χρηματοδοτεί τον λογαριασμό με τα lamport που ορίζονται                                                                           |
| `get_account(&Address)`                        | Επιστρέφει όλες τις πληροφορίες που σχετίζονται με τον λογαριασμό της παρεχόμενης διεύθυνσης                                      |
| `set_account(Address, Account)`                | Ορίζει όλες τις πληροφορίες που σχετίζονται με τον λογαριασμό της παρεχόμενης διεύθυνσης                                          |
| `get_balance(&Address)`                        | Λαμβάνει το υπόλοιπο lamport του παρεχόμενου λογαριασμού                                                                          |
| `minimum_balance_for_rent_exemption(data_len)` | Επιστρέφει το ελάχιστο υπόλοιπο που απαιτείται για τη δημιουργία λογαριασμού με καθορισμένο μήκος δεδομένων απαλλαγμένου από rent |
| `add_program(program_id, &bytes)`              | Προσθέτει ένα πρόγραμμα SBF στο περιβάλλον δοκιμών                                                                                |
| `add_program_from_file(program_id, path)`      | Προσθέτει ένα πρόγραμμα SBF στο περιβάλλον δοκιμών από το καθορισμένο αρχείο                                                      |
| `send_transaction(tx)`                         | Υποβάλλει μια υπογεγραμμένη συναλλαγή                                                                                             |
| `simulate_transaction(tx)`                     | Προσομοιώνει μια συναλλαγή χωρίς τροποποίηση της κατάστασης                                                                       |
| `get_transaction(&signature)`                  | Λαμβάνει μια συναλλαγή από το ιστορικό                                                                                            |
| `latest_blockhash()`                           | Λαμβάνει το τρέχον blockhash                                                                                                      |
| `expire_blockhash()`                           | Επιβάλλει τη λήξη του τρέχοντος blockhash                                                                                         |
| `warp_to_slot(slot)`                           | Μεταφέρει το ρολόι στο καθορισμένο slot                                                                                           |
| `get_sysvar<T>()`                              | Λαμβάνει ένα sysvar από το περιβάλλον δοκιμών                                                                                     |
| `set_sysvar<T>(&value)`                        | Ορίζει το sysvar στο περιβάλλον δοκιμών                                                                                           |
| `with_feature_accounts()`                      | Συμπεριλαμβάνει λογαριασμούς feature gate στο περιβάλλον δοκιμών                                                                  |
| `set_invocation_inspect_callback(callback)`    | Καταχωρεί μια επανάκληση πριν/μετά την κλήση (απαιτεί τη λειτουργία `invocation-inspect-callback`)                                |
