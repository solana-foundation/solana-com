_Διαθέσιμο στο GitHub:
[`v2.2.0-beta.7`](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.7)_

Το Kora v2.2.0-beta εισάγει υποστήριξη δεσμών Jito, νέες μεθόδους RPC, εικόνες
Docker, βελτιωμένες δυνατότητες ασφαλείας και βελτιώσεις SDK.

> **Αυτή είναι μια προ-έκδοση.** Οι λειτουργίες που τεκμηριώνονται εδώ είναι
> διαθέσιμες στο `v2.2.0-beta.7` και ενδέχεται να αλλάξουν πριν την σταθερή
> έκδοση. Για σταθερή τεκμηρίωση, δείτε την
> [κύρια τεκμηρίωση του Kora](/docs/tools/kora).

## Σημαντικά Σημεία

### Υποστήριξη Δεσμών Jito

Υποβάλετε έως 5 συναλλαγές ατομικά μέσω του block engine του Jito. Νέες μέθοδοι
RPC:

- [`signBundle`](/docs/tools/kora/beta/bundle-methods#signbundle) — Υπογραφή
  δέσμης χωρίς μετάδοση
- [`signAndSendBundle`](/docs/tools/kora/beta/bundle-methods#signandsendbundle)
  — Υπογραφή και υποβολή στο Jito
- [`estimateBundleFee`](/docs/tools/kora/beta/bundle-methods#estimatebundlefee)
  — Εκτίμηση συνολικών τελών δέσμης

Δείτε τον [Οδηγό Δεσμών Jito](/docs/tools/kora/beta/jito-guide) για πλήρη
αναλυτική παρουσίαση.

### Εικόνες Docker

Επίσημες εικόνες πολλαπλών πλατφορμών (amd64/arm64) δημοσιευμένες στο GitHub
Container Registry. Δείτε την [Εγκατάσταση](/docs/tools/kora/beta/installation).

### Πρόσθετα Συναλλαγών

Ένα νέο σύστημα προσθέτων επιτρέπει στο Kora να επικυρώνει τη δομή συναλλαγών
κατά τη διάρκεια ροών υπογραφής. Το πρώτο ενσωματωμένο πρόσθετο, `gas_swap`,
επιβάλλει αυστηρούς κανόνες για συναλλαγές ανταλλαγής token-για-SOL χωρίς gas.
Τα πρόσθετα διαμορφώνονται στο `kora.toml` και εκτελούνται αυτόματα κατά τη
διάρκεια των `signTransaction`, `signAndSendTransaction`, `signBundle`, και
`signAndSendBundle`.

Δείτε
[Διαμόρφωση — Πρόσθετα Συναλλαγών](/docs/tools/kora/beta/configuration#transaction-plugins).

### Νέες Επιλογές Διαμόρφωσης

- **Πρόσθετα Συναλλαγών** για επικύρωση δομής συναλλαγών (`gas_swap`)
- **reCAPTCHA v3** προστασία από bots για σημεία υπογραφής
- **Lighthouse** προστασία από εξάντληση πληρωτή τελών
- **Όρια Χρήσης** περιορισμός συναλλαγών ανά πορτοφόλι
- **Διαμόρφωση δέσμης** για ενσωμάτωση Jito

Δείτε τη [Διαμόρφωση](/docs/tools/kora/beta/configuration).

### Αλλαγές SDK

- Υποστήριξη μεθόδων δέσμης (`signBundle`, `signAndSendBundle`,
  `estimateBundleFee`)

Δείτε τις [Αλλαγές SDK](/docs/tools/kora/beta/sdk-changes).

### Πρόσθετες Μέθοδοι RPC

- [`getVersion`](/docs/tools/kora/beta/bundle-methods#getversion) — Ανάκτηση
  πληροφοριών έκδοσης διακομιστή

## Εγκατάσταση

```bash
cargo install kora-cli@2.2.0-beta.7
```

Ή μέσω Docker:

```bash
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```
