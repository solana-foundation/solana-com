## Cargo

Εγκαταστήστε την έκδοση beta απευθείας από το crates.io:

```bash
cargo install kora-cli@2.2.0-beta.7
```

## Docker

Επίσημες εικόνες Docker πολλαπλών πλατφορμών (amd64/arm64) δημοσιεύονται στο
GitHub Container Registry:

```bash
# Pull the latest beta release
docker pull ghcr.io/solana-foundation/kora:beta

# Pull a specific beta version
docker pull ghcr.io/solana-foundation/kora:v2.2.0-beta.7
```

Διαθέσιμες ετικέτες:

- `latest` — Τελευταία σταθερή έκδοση
- `beta` — Τελευταία προ-έκδοση
- `v<version>` — Συγκεκριμένη έκδοση (π.χ., `v2.2.0-beta.7`)

### Docker Compose

Για προσαρμοσμένες εγκαταστάσεις ή για να συμπεριλάβετε το Redis για υποστήριξη
caching και ορίων χρήσης:

- [Δείγμα Dockerfile](https://github.com/solana-foundation/kora/blob/main/examples/deploy/Dockerfile)
- [Docker Compose με Redis](https://github.com/solana-foundation/kora/blob/main/docker-compose.yml)

### Εκτέλεση με Docker

```bash
docker run -d \
  -p 8080:8080 \
  -v $(pwd)/kora.toml:/app/kora.toml \
  -v $(pwd)/signers.toml:/app/signers.toml \
  -e KORA_PRIVATE_KEY="your-private-key" \
  ghcr.io/solana-foundation/kora:v2.2.0-beta.7 \
  --config /app/kora.toml rpc start --signers-config /app/signers.toml
```

## TypeScript SDK

Το SDK beta είναι διαθέσιμο στο npm:

```bash
pnpm add @solana/kora@beta
```

Δείτε [Αλλαγές SDK](/docs/tools/kora/beta/sdk-changes) για τα νέα APIs.

## Επαλήθευση Εγκατάστασης

```bash
kora --version
```
