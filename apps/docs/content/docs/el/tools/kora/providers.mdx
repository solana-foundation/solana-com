Δεν θέλετε να εκτελέσετε τον δικό σας κόμβο Kora; Οι παρακάτω πάροχοι προσφέρουν
φιλοξενούμενη υποδομή Kora ώστε να μπορείτε να προσθέσετε αφαίρεση προμηθειών
στην εφαρμογή σας χωρίς να διαχειρίζεστε τη δική σας ανάπτυξη.

## Πάροχοι

### Openfort

Η [Openfort](https://www.openfort.io/) προσφέρει υποδομή paymaster για Solana
βασισμένη στο Kora, επιτρέποντας συναλλαγές χωρίς προμήθειες μέσω της πλατφόρmas
ενσωματωμένου πορτοφολιού τους. Η ενσωμάτωσή τους χρησιμοποιεί ένα μοτίvo
συν-υπογραφής όπου το Kora ενεργεί ως πληρωτής προμηθειών ενώ οι χρήστες
υπογράφουν μέσω των ενσωματωμένων πορτοφολιών της Openfort.

- **Ιστότοπος**: [openfort.io](https://www.openfort.io/)
- **Τεκμηρίωση**:
  [Συναλλαγές Solana Χωρίς Προμήθειες](https://www.openfort.io/blog/solana-gasless-transactions)
- **Χαρακτηριστικά**: Ενσωματωμένα πορτοφόλια, συναλλαγές χωρίς προμήθειες,
  χορηγία προμηθειών

## Γίνετε Διαχειριστής Κόμβου Kora

Το Kora είναι ανοιχτού κώδικα και ο καθένας μπορεί να εκτελέσει έναν κόμβο. Είτε
δημιουργείτε μια εφαρμογή που χρειάζεται αφαίρεση προμηθειών είτε θέλετε να
προσφέρετε υπηρεσίες paymaster σε άλλους, δείτε τον
[Οδηγό Διαχειριστών](/docs/tools/kora/operators) για να ξεκινήσετε.

## Καταχωρηθείτε

Εάν διαχειρίζεστε έναν κόμβο Kora και θέλετε να προστεθείτε σε αυτή τη σελίδα,
επικοινωνήστε μέσω των καναλιών
[Κοινότητα & Υποστήριξη Kora](https://github.com/solana-foundation/kora?tab=readme-ov-file#community--support).
