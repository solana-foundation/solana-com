## Kora v2.2.0-beta.6

_23-03-2026_ |
[Έκδοση GitHub](https://github.com/solana-foundation/kora/releases/tag/v2.2.0-beta.6)
| `cargo install kora-cli@2.2.0-beta.6`

Προ-έκδοση με υποστήριξη Jito bundle, νέες μεθόδους RPC και βελτιωμένα
χαρακτηριστικά ασφαλείας.

- Προστέθηκε υποστήριξη Jito bundle με τις μεθόδους `signBundle`,
  `signAndSendBundle` και `estimateBundleFee`
- Προστέθηκε η μέθοδος RPC `getVersion`
- Προστέθηκε προστασία fee payer του Lighthouse
- Προστέθηκε προστασία από bots reCAPTCHA v3 για τα endpoints υπογραφής
- Προστέθηκαν όρια χρήσης ανά πορτοφόλι με υποστήριξη Redis
- Προστέθηκαν επίσημες εικόνες Docker στο GHCR
- Προστέθηκαν λεπτομερείς κανόνες ορίων χρήσης
- SDK: Το `KoraClient` δέχεται πλέον αντικείμενο επιλογών· νέο `koraPlugin` για
  `@solana/kit`

Δείτε την [Τεκμηρίωση Beta](/docs/tools/kora/beta) για λεπτομέρειες.

## Kora TypeScript SDK v0.2.0

_09-03-2026_ |
[Έκδοση GitHub](https://github.com/solana-foundation/kora/releases/tag/ts-sdk-v0.2.0)
| `pnpm add @solana/kora@0.2.0`

Αρχιτεκτονική plugin πελάτη Kit για συνθέσιμες συναλλαγές χωρίς gas.

- Προστέθηκε `createKitKoraClient()` με σύνθεση plugin Kit (planner, executor,
  payer plugins)
- Επιτρέπει τη συνθεσιμότητα με plugins προγράμματος Kit όπως το
  `tokenProgram()`
- Προστέθηκε υποστήριξη Token-2022 μέσω της επιλογής διαμόρφωσης
  `tokenProgramId`
- Προστέθηκε υποστήριξη οδηγιών compute budget με εκτίμηση CU βασισμένη σε
  προσομοίωση
- Προστέθηκε το πεδίο `signature` που έλειπε στην απάντηση
  `signAndSendTransaction`
- Επιστρέφει `NoopSigner` από το `getPaymentInstruction` για αφαίρεση
  διπλοεγγραφών υπογραφέα Kit v6
- Η εξάρτηση peer `@solana/kit` αναβαθμίστηκε σε `^6.1.0`

Δείτε τον [Οδηγό Kit Client](/docs/tools/kora/guides/kit-client).
