**Το Solana Pay είναι ένα τυποποιημένο πρωτόκολλο και ένα σύνολο υλοποιήσεων
αναφοράς που δίνουν τη δυνατότητα στους προγραμματιστές να ενσωματώσουν
αποκεντρωμένες πληρωμές στις εφαρμογές και τις υπηρεσίες τους.** Το blockchain
του Solana επιβεβαιώνει συναλλαγές σε λιγότερο από ένα δευτερόλεπτο και κοστίζει
κατά μέσο όρο $0.0005, παρέχοντας στους χρήστες μια απρόσκοπτη εμπειρία χωρίς
διαμεσολαβητές.

## Γιατί Solana Pay;

- **Τυποποιημένο Πρωτόκολλο**: Ενιαία προσέγγιση για πληρωμές σε όλα τα
  πορτοφόλια και τις εφαρμογές του Solana
- **Εξαιρετικά Γρήγορο**: Οι συναλλαγές επιβεβαιώνονται σε λιγότερο από ένα
  δευτερόλεπτο
- **Εξαιρετικά Οικονομικό**: Μέσο κόστος συναλλαγής $0.0005
- **Χωρίς Διαμεσολαβητές**: Άμεσες peer-to-peer πληρωμές χωρίς τρίτους
- **Καθολική Υποστήριξη Πορτοφολιού**: Συμβατό με Phantom, Solflare, Backpack
  και άλλα

## Γρήγορη Εκκίνηση

Ο απλούστερος τρόπος για να ξεκινήσετε είναι με ένα URL αιτήματος μεταφοράς:

```
solana:recipient?amount=0.01&label=Coffee%20Shop&message=Grande%20Latte
```

Αυτό το URL μπορεί να:

- Κωδικοποιηθεί ως κωδικός QR για σάρωση από πορτοφόλια κινητών
- Χρησιμοποιηθεί ως σύνδεσμος πληρωμής σε διαδικτυακές εφαρμογές
- Ενσωματωθεί σε εφαρμογές κινητών ως συνδέσμους βαθιάς σύνδεσης

**[→ Γρήγορη Εκκίνηση Αιτήματος Μεταφοράς](/docs/tools/solana-pay/quickstart/transfer-requests)** -

## Εγκατάσταση

Εγκαταστήστε το Solana Pay JavaScript SDK:

```bash
# Using pnpm (recommended)
pnpm add @solana/pay@beta @solana/kit

# Using npm
npm install @solana/pay@beta @solana/kit

# Using yarn
yarn add @solana/pay@beta @solana/kit
```

## Περιπτώσεις Χρήσης

Το Solana Pay καθιστά δυνατό ένα ευρύ φάσμα σεναρίων πληρωμών:

- **Πληρωμές Λιανικής**: Πληρωμές αυτοπροσώπως με χρήση κωδικών QR
- **Ηλεκτρονικό Εμπόριο**: Ροές ολοκλήρωσης αγοράς στο διαδίκτυο με ενσωμάτωση
  πορτοφολιού
- **Εφαρμογές Κινητών**: Εγγενείς εμπειρίες πληρωμών με βαθιά σύνδεση
- **Δημιουργία NFT**: Δυναμική δημιουργία συναλλαγών για σύνθετες
  αλληλεπιδράσεις
- **Συνδρομές**: Επαναλαμβανόμενες πληρωμές και προγράμματα επιβράβευσης
- **Ενσωμάτωση DeFi**: Συνθέσιμες συναλλαγές με ανταλλαγές και δανεισμό

## Υποστηριζόμενα Πορτοφόλια

Το Solana Pay λειτουργεί με όλα τα κύρια πορτοφόλια Solana:

- **[Phantom](https://phantom.app/)** - Το πιο δημοφιλές πορτοφόλι Solana
- **[Solflare](https://solflare.com/)** - Πορτοφόλι Solana για πολλαπλές
  πλατφόρμες
- **[Backpack](https://backpack.app/)** - Πορτοφόλι Web3 και κατάστημα εφαρμογών
- **[Glow](https://glow.app/)** - Πορτοφόλι Solana με έμφαση στα κινητά
- **[Ultimate](https://ultimate.app/)** - Πορτοφόλι με έμφαση στο DeFi
- **[Decaf Wallet](https://decaf.so/)** - Πορτοφόλι παγκόσμιων πληρωμών
- **[Espresso Cash](https://www.espressocash.com/)** - Πορτοφόλι παγκόσμιων
  πληρωμών
- **[Tiplink](https://tiplink.io/)** - Το πιο απλό πορτοφόλι στον κόσμο
- **[Bitget Wallet](https://bitget.com/)** - Πορτοφόλι πολλαπλών αλυσίδων
