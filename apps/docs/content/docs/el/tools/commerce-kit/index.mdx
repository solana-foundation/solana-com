**Το Commerce Kit είναι μια συλλογή εργαλείων για τη δημιουργία εμπειριών
εμπορίου στο Solana.** Από κουμπιά πληρωμής plug-and-play έως headless
primitives, το Commerce Kit παρέχει όλα όσα χρειάζεστε για να δέχεστε
κρυπτονομισματικές πληρωμές.

## Γιατί Commerce Kit;

- **Έτοιμα Στοιχεία**: Προκατασκευασμένο PaymentButton με σύνδεση πορτοφολιού,
  επιλογή token και διαχείριση συναλλαγών
- **Ανεξάρτητο από Framework**: Χρησιμοποιήστε headless primitives σε
  οποιοδήποτε JavaScript framework
- **Solana Pay**: Πλήρης υποστήριξη Solana Pay με δημιουργία κωδικού QR
- **Type-Safe**: Πλήρης υποστήριξη TypeScript σε όλα τα πακέτα
- **Wallet Standard**: Συμβατό με όλα τα πορτοφόλια Wallet Standard (Phantom,
  Solflare, Backpack, κ.λπ.)

## Γρήγορη Έναρξη

Ο ταχύτερος τρόπος για να ξεκινήσετε είναι με το στοιχείο `PaymentButton`:

```tsx
import { PaymentButton } from "@solana-commerce/kit";

function App() {
  return (
    <PaymentButton
      config={{
        merchant: {
          name: "My Store",
          wallet: "your-solana-wallet-address"
        },
        mode: "buyNow"
      }}
      onPaymentSuccess={(signature) => {
        console.log("Payment successful:", signature);
      }}
    />
  );
}
```

## Εγκατάσταση

Επιλέξτε τα πακέτα Commerce Kit που χρειάζεστε:

```bash
# Unified Commerce Meta Package
pnpm add @solana-commerce/kit

# Drop-in payment button (most common)
pnpm add @solana-commerce/react

# Wallet connection only
pnpm add @solana-commerce/connector

# Headless commerce primitives
pnpm add @solana-commerce/headless

# Solana Pay QR codes
pnpm add @solana-commerce/solana-pay

# React hooks for custom UIs
pnpm add @solana-commerce/sdk
```
