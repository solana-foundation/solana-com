Μια γρήγορη εγκατάσταση για να ρυθμίσετε το τοπικό σας περιβάλλον ανάπτυξης
Solana με μία εντολή. Εγκαταστήστε το Rust, το Solana CLI και το Anchor
Framework σε Windows (WSL), Linux και Mac. (Εάν προτιμάτε, μπορείτε να
[εγκαταστήσετε κάθε εξάρτηση ξεχωριστά](/docs/intro/installation/dependencies).)

## Προαπαιτούμενα

<Accordions>
<Accordion title="Windows">

Για να αναπτύξετε προγράμματα Solana σε Windows **πρέπει** να χρησιμοποιήσετε το
[Windows Subsystem for Linux](https://learn.microsoft.com/en-us/windows/wsl/install)
(WSL). Μετά την εγκατάσταση του WSL, όλες οι άλλες εξαρτήσεις θα εγκατασταθούν
μέσω του τερματικού Linux.

Για να εγκαταστήσετε το WSL, εκτελέστε την ακόλουθη εντολή στο Windows
PowerShell:

```terminal
$ wsl --install
```

Η διαδικασία εγκατάστασης θα σας ζητήσει να δημιουργήσετε έναν προεπιλεγμένο
λογαριασμό χρήστη.

![Εγκατάσταση WSL](/assets/docs/intro/installation/wsl-install.png)

Από προεπιλογή, το WSL εγκαθιστά το Ubuntu. Μπορείτε να ανοίξετε ένα τερματικό
Linux αναζητώντας "Ubuntu" στη γραμμή αναζήτησης.

![WSL Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-search.png)

Εάν το τερματικό Ubuntu σας μοιάζει με την παρακάτω εικόνα, ενδέχεται να
αντιμετωπίσετε ένα πρόβλημα όπου το `ctrl + v` (συντόμευση πληκτρολογίου
επικόλλησης) δεν λειτουργεί στο τερματικό.

![Τερματικό Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-terminal-1.png)

Εάν αντιμετωπίσετε αυτό το πρόβλημα, ανοίξτε το Windows Terminal αναζητώντας
"Terminal" στη γραμμή αναζήτησης.

![Windows Terminal](/assets/docs/intro/installation/wsl-windows-terminal.png)

Στη συνέχεια, κλείστε το Windows Terminal και ανοίξτε ξανά ένα τερματικό Linux
αναζητώντας το Ubuntu ξανά. Το τερματικό θα πρέπει τώρα να μοιάζει με την
παρακάτω εικόνα, όπου το `ctrl + v` (συντόμευση πληκτρολογίου επικόλλησης)
λειτουργεί.

![Τερματικό Ubuntu](/assets/docs/intro/installation/wsl-ubuntu-terminal-2.png)

Εάν χρησιμοποιείτε το VS Code, η
[επέκταση WSL](https://code.visualstudio.com/docs/remote/wsl-tutorial) σας
επιτρέπει να χρησιμοποιείτε το WSL και το VS Code μαζί.

![Ρύθμιση WSL στο VS Code](/assets/docs/intro/installation/wsl-vscode.png)

Θα δείτε τα ακόλουθα στη γραμμή κατάστασης του VS Code:

![WSL: Ubuntu](/assets/docs/intro/installation/wsl-vscode-ubuntu.png)

Αφού ρυθμίσετε το WSL, είστε έτοιμοι είτε να:

- Ολοκληρώσετε τη [γρήγορη εγκατάσταση](#installation) χρησιμοποιώντας το
  τερματικό Ubuntu (Linux)
- Εγκαταστήσετε [Rust](/docs/intro/installation/dependencies#install-rust), το
  [Solana CLI](/docs/intro/installation/dependencies#install-solana-cli) και το
  [Anchor CLI](/docs/intro/installation/dependencies#install-anchor-cli)
  ξεχωριστά, όπως περιγράφεται παρακάτω

Αν χρησιμοποιείτε Debian ή παράγωγα του Debian όπως Ubuntu ή Mint, ακολουθήστε τα παρακάτω βήματα:

1. Ενημερώστε τις λίστες πακέτων για να βεβαιωθείτε ότι όλα τα πακέτα είναι
   ενημερωμένα:

```terminal
$ sudo apt-get update
```

2. Εγκαταστήστε τις ακόλουθες εξαρτήσεις:

```terminal
$ sudo apt-get install -y \
    build-essential \
    pkg-config \
    libudev-dev llvm libclang-dev \
    protobuf-compiler libssl-dev
```

ότι εκτελείτε πρώτα _shell`sudo apt-get update`

```
is only available from another source
```

```terminal
$ sudo dnf group install c-development development-tools \
```

## Εγκατάσταση

1. Εισαγάγετε την ακόλουθη εντολή στο τερματικό σας για να εγκαταστήσετε όλες
   τις απαραίτητες εξαρτήσεις:

```terminal
$ curl --proto '=https' --tlsv1.2 -sSfL https://solana-install.solana.workers.dev | bash
```

Μια επιτυχημένη εγκατάσταση θα επιστρέψει αποτέλεσμα όπως το ακόλουθο:

```txt title="Example output"
Installed Versions:
Rust: rustc 1.91.1 (ed61e7d7e 2025-11-07)
Solana CLI: solana-cli 3.0.10 (src:96c3a851; feat:3604001754, client:Agave)
Anchor CLI: anchor-cli 0.32.1
Surfpool CLI: surfpool 0.12.0
Node.js: v24.10.0
Yarn: 1.22.1
```

2. Επαληθεύστε μια επιτυχημένη εγκατάσταση ελέγχοντας την έκδοση κάθε
   εγκατεστημένης εξάρτησης.

```terminal
$ rustc --version && solana --version && anchor --version && surfpool --version && node --version && yarn --version
```

<Callout>
  Εάν η εντολή γρήγορης εγκατάστασης αποτύχει, ανατρέξτε στην ενότητα
  [Εγκατάσταση εξαρτήσεων](/docs/intro/installation/dependencies) παρακάτω για
  οδηγίες εγκατάστασης κάθε εξάρτησης ξεχωριστά.
</Callout>
