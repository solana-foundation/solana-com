---
title: Προγραμματισμός staking
---

Για να μεγιστοποιηθεί η κατανομή των stakes, η αποκέντρωση και η αντίσταση στη
λογοκρισία στο δίκτυο Solana, το staking μπορεί να πραγματοποιηθεί
προγραμματιστικά. Η ομάδα και η κοινότητα έχουν αναπτύξει αρκετά onchain και
offchain προγράμματα για να διευκολύνουν τη διαχείριση των stakes.

### Πρόγραμμα ανάθεσης του Solana Foundation

Το
[Πρόγραμμα ανάθεσης του Solana Foundation](https://solana.org/delegation-program)
είναι σχεδιασμένο να εξυπηρετεί τις ανάγκες του δικτύου στοχεύοντας στους
ακόλουθους στόχους:

1. Μεγιστοποίηση της αποκέντρωσης, αξιοπιστίας και απόδοσης του Solana
2. Μεγιστοποίηση του αριθμού των validators που έχουν διαφορετικές πηγές stake
3. Διατήρηση ενός μεγάλου και αντιπροσωπευτικού testnet

### Stake Pools

Αυτό το onchain πρόγραμμα συγκεντρώνει SOL για να τεθεί σε staking από έναν
διαχειριστή, επιτρέποντας στους κατόχους SOL να κάνουν stake και να κερδίζουν
ανταμοιβές χωρίς να διαχειρίζονται stakes. Οι χρήστες καταθέτουν SOL σε
αντάλλαγμα για SPL tokens (παράγωγα staking) που αντιπροσωπεύουν την κυριότητά
τους στο stake pool. Ο διαχειριστής του pool θέτει σε stake το κατατεθειμένο SOL
σύμφωνα με τη στρατηγική του, ίσως χρησιμοποιώντας μια παραλλαγή ενός bot
αυτόματης ανάθεσης όπως περιγράφεται παραπάνω. Καθώς τα stakes αποφέρουν
ανταμοιβές, το pool και τα tokens του pool αυξάνονται αναλογικά σε αξία. Τέλος,
οι κάτοχοι pool tokens μπορούν να στείλουν SPL tokens πίσω στο stake pool για να
εξαργυρώσουν SOL, συμμετέχοντας έτσι στην αποκέντρωση με πολύ λιγότερη
απαιτούμενη προσπάθεια. Περισσότερες πληροφορίες μπορείτε να βρείτε στην
[τεκμηρίωση του SPL stake pool](https://spl.solana.com/stake-pool).
