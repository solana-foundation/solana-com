_Σημείωση πριν τη διάβαση: Όλες οι αναφορές σε αυξήσεις αξιών αφορούν απόλυτους
όρους σε σχέση με το υπόλοιπο SOL. Το παρόν έγγραφο δεν προβαίνει σε καμία
υπόδειξη σχετικά με τη νομισματική αξία του SOL σε οποιαδήποτε χρονική στιγμή._

Κάνοντας staking τα SOL tokens σας, συμβάλλετε στην ασφάλεια του δικτύου και
[κερδίζετε ανταμοιβές](https://docs.anza.xyz/implemented-proposals/staking-rewards)
ταυτόχρονα.

Μπορείτε να κάνετε staking αναθέτοντας τα tokens σας σε validators που
επεξεργάζονται συναλλαγές και λειτουργούν το δίκτυο.

Η ανάθεση stake είναι ένα οικονομικό μοντέλο κοινού κινδύνου και κοινής
ανταμοιβής που μπορεί να παρέχει αποδόσεις στους κατόχους tokens που έχουν
ανατεθεί για μεγάλο χρονικό διάστημα. Αυτό επιτυγχάνεται ευθυγραμμίζοντας τα
οικονομικά κίνητρα των κατόχων tokens (delegators) και των validators στους
οποίους αναθέτουν.

Όσο περισσότερο stake ανατίθεται σε έναν validator, τόσο πιο συχνά επιλέγεται
αυτός ο validator να γράψει νέες συναλλαγές στο ledger. Όσο περισσότερες
συναλλαγές γράφει ο validator, τόσο περισσότερες ανταμοιβές κερδίζουν ο
validator και οι delegators του. Οι validators που διαμορφώνουν τα συστήματά
τους ώστε να μπορούν να επεξεργάζονται περισσότερες συναλλαγές κερδίζουν
αναλογικά περισσότερες ανταμοιβές και επειδή διατηρούν το δίκτυο να λειτουργεί
όσο πιο γρήγορα και όσο πιο ομαλά γίνεται.

Οι validators επιβαρύνονται με κόστη λειτουργίας και συντήρησης των συστημάτων
τους, και αυτό μεταφέρεται στους delegators με τη μορφή χρέωσης που εισπράττεται
ως ποσοστό των κερδισμένων ανταμοιβών. Αυτή η χρέωση είναι γνωστή ως
_προμήθεια_. Δεδομένου ότι οι validators κερδίζουν περισσότερες ανταμοιβές όσο
περισσότερο stake τους ανατίθεται, μπορεί να ανταγωνίζονται μεταξύ τους για να
προσφέρουν τη χαμηλότερη προμήθεια για τις υπηρεσίες τους.

Αν και αυτό δεν έχει υλοποιηθεί στο πρωτόκολλο Solana σήμερα, στο μέλλον, οι
delegators θα μπορούσαν να διακινδυνεύσουν να χάσουν tokens κάνοντας staking
μέσω μιας διαδικασίας γνωστής ως _slashing_. Το slashing περιλαμβάνει την
αφαίρεση και καταστροφή ενός τμήματος των SOL ενός validator ως απάντηση σε
σκόπιμη κακόβουλη συμπεριφορά, όπως η δημιουργία άκυρων συναλλαγών ή η
λογοκρισία ορισμένων τύπων συναλλαγών ή συμμετεχόντων του δικτύου.

Δεν υπάρχει υλοποίηση slashing εντός του πρωτοκόλλου αυτή τη στιγμή. Για
περισσότερες πληροφορίες σχετικά με το slashing δείτε το
[slashing roadmap](https://docs.anza.xyz/proposals/optimistic-confirmation-and-slashing#slashing-roadmap).

## Πώς μπορώ να κάνω stake τα SOL tokens μου;

Μπορείτε να κάνετε stake SOL μεταφέροντας τα tokens σας σε ένα πορτοφόλι που
υποστηρίζει staking. Το πορτοφόλι παρέχει βήματα για τη δημιουργία λογαριασμού
stake και την ανάθεση.

### Υποστηριζόμενα πορτοφόλια

Πολλά web και mobile πορτοφόλια υποστηρίζουν λειτουργίες staking στο Solana.
Παρακαλούμε ελέγξτε με τους υπεύθυνους του αγαπημένου σας πορτοφολιού σχετικά με
την κατάσταση

### Εργαλεία γραμμής εντολών Solana

- Τα εργαλεία γραμμής εντολών Solana μπορούν να εκτελέσουν όλες τις λειτουργίες
  stake σε συνδυασμό με ένα αρχείο πορτοφολιού keypair που δημιουργήθηκε από
  CLI, ένα χάρτινο πορτοφόλι ή με συνδεδεμένο Ledger Nano.
  [Εντολές staking χρησιμοποιώντας τα εργαλεία γραμμής εντολών Solana](https://docs.anza.xyz/cli/examples/delegate-stake).

### Δημιουργία λογαριασμού stake

Ακολουθήστε τις οδηγίες του πορτοφολιού για τη δημιουργία λογαριασμού staking.
Αυτός ο λογαριασμός θα είναι διαφορετικού τύπου από έναν που χρησιμοποιείται
απλώς για αποστολή και λήψη tokens.

### Επιλογή validator

Ακολουθήστε τις οδηγίες του πορτοφολιού για την επιλογή validator. Μπορείτε να
λάβετε πληροφορίες σχετικά με πιθανώς αποδοτικούς validators από τους παρακάτω
συνδέσμους. Το Solana Foundation δεν συνιστά κάποιον συγκεκριμένο validator.

Ο ιστότοπος solanabeach.io έχει κατασκευαστεί και συντηρείται από έναν από τους
validators μας, το Staking Facilities. Παρέχει κάποιες γραφικές πληροφορίες
υψηλού επιπέδου για το δίκτυο συνολικά, καθώς και μια λίστα κάθε validator και
κάποια πρόσφατα στατιστικά απόδοσης για τον καθένα.

- https://solanabeach.io

Για να δείτε στατιστικά παραγωγής blocks, χρησιμοποιήστε τα εργαλεία γραμμής
εντολών Solana:

- `solana validators`
- `solana block-production`

Η ομάδα Solana δεν κάνει συστάσεις σχετικά με τον τρόπο ερμηνείας αυτών των
πληροφοριών. Κάντε τη δική σας έρευνα.

### Εκχώρηση του stake σας

Ακολουθήστε τις οδηγίες του πορτοφολιού για την εκχώρηση του stake σας στον
validator που επιλέξατε.
