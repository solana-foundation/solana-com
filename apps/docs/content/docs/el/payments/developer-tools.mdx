## Εργαλεία ανάπτυξης AI

Ξεκινήστε με τον οδηγό μας για το
[πώς να ξεκινήσετε με εργαλεία AI στο Solana](/developers/guides/getstarted/intro-to-ai).

### Solana MCP

Το [Solana MCP](https://mcp.solana.com/) (Machine Conversation Protocol)
ενσωματώνει τη γνώση του Solana απευθείας σε εργαλεία κωδικοποίησης με
υποστήριξη AI όπως το Claude, Cursor, και Windsurf. Είναι εκπαιδευμένο σε
έγγραφα Solana, το framework Anchor, παραδείγματα προγραμμάτων, και
ερωτήσεις-απαντήσεις Stack Exchange - χρήσιμο για την ταχύτερη δημιουργία
ενσωματώσεων πληρωμών.

### Frameworks AI agent

Τα AI agents μπορούν να εκτελούν αυτόνομα συναλλαγές πληρωμών στο Solana. Αυτά
τα frameworks επιτρέπουν τη δημιουργία agents που επεξεργάζονται πληρωμές,
παρακολουθούν συναλλαγές, και αυτοματοποιούν ροές εργασίας πληρωμών:

| Framework        | Περιγραφή                                                                                    | Σύνδεσμοι                                                                                 |
| ---------------- | -------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Solana Agent Kit | Εργαλεία για τη δημιουργία agents με δυνατότητα πληρωμών, συμβατά με LangChain και Vercel AI | [GitHub](https://github.com/sendaifun/solana-agent-kit) • [Docs](https://kit.sendai.fun/) |
| Send AI Skills   | Μια συλλογή προκατασκευασμένων δεξιοτήτων για τη δημιουργία agents στο Solana                | [GitHub](https://github.com/sendaifun/skills)                                             |
| Eliza Framework  | Framework agent πολλαπλών πλατφορμών με plugin Solana για bots X, Telegram, Discord          | [GitHub](https://github.com/elizaOS/eliza) • [Docs](https://docs.elizaos.ai/)             |
| GOAT Toolkit     | Toolkit πολλαπλών αλυσίδων με ενέργειες πληρωμών Solana                                      | [GitHub](https://github.com/goat-sdk/goat)                                                |

### Ανάπτυξη προγραμμάτων AI

Το [Codigo](https://codigo.ai/) είναι μια ολοκληρωμένη πλατφόρμα ανάπτυξης AI
για το Solana.
