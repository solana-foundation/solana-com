Αποδεχτείτε πληρωμές στο Solana χρησιμοποιώντας έτοιμα components, το πρωτόκολλο
Solana Pay ή άμεση ενσωμάτωση RPC.

## Σύμβαση διεύθυνσης λήψης

Στο Solana, **μοιραστείτε τη διεύθυνση του πορτοφολιού σας, όχι το token account
σας**. Είναι τυπικό ο αποστολέας να παράγει το κατάλληλο
[associated token account (ATA)](/docs/payments/how-payments-work#token-accounts)
για οποιοδήποτε token στέλνει (δείτε
[Αποστολή πληρωμών](/docs/payments/send-payments) για περισσότερες
λεπτομέρειες).

**Παράδειγμα:** Μοιραστείτε `7EcDhS...` (το πορτοφόλι σας), όχι `9xKpQ...` (το
USDC token account σας).

<Callout>
  Για point-of-sale και e-commerce, δείτε το [Solana
  Pay](/docs/payments/accept-payments/solana-pay) για QR codes και URLs
  πληρωμών, ή το [Payment Button](/docs/payments/accept-payments/payment-button)
  για έτοιμα React components.
</Callout>

## Επιλογές ενσωμάτωσης

| Προσέγγιση                                                      | Ιδανική για                         | Πολυπλοκότητα |
| --------------------------------------------------------------- | ----------------------------------- | ------------- |
| [Payment Button](/docs/payments/accept-payments/payment-button) | Εφαρμογές React, γρήγορη ενσωμάτωση | Χαμηλή        |
| [Solana Pay](/docs/payments/accept-payments/solana-pay)         | QR codes, links πληρωμών, POS       | Μέτρια        |
| [Direct RPC](/docs/payments/interacting-with-solana)            | Προσαρμοσμένες ροές, πλήρης έλεγχος | Υψηλότερη     |

## Επαλήθευση

Μετά τη λήψη μιας πληρωμής, επαληθεύστε ότι ολοκληρώθηκε χρησιμοποιώντας
[εργαλεία επαλήθευσης](/docs/payments/accept-payments/verification-tools). Για
εφαρμογές υψηλού όγκου, δείτε το
[Indexing](/docs/payments/accept-payments/indexing) για παρακολούθηση συναλλαγών
επιπέδου παραγωγής.
