---
title: Επεξεργασία πληρωμών
description:
  Επεξεργαστείτε πληρωμές σε κλίμακα με συναλλαγές χωρίς gas και αφαίρεση
  προμηθειών
---

Η επεξεργασία πληρωμών σε κλίμακα—μισθοδοσία, πληρωμές εμπόρων, ανταμοιβές
πιστότητας—απαιτεί περισσότερα από την αποστολή μεμονωμένων μεταφορών.

Για να επεξεργαστείτε πολλές πληρωμές αποτελεσματικά:

1. **Ομαδοποιήστε εντολές** σε μεμονωμένες συναλλαγές για να μειώσετε τις
   προμήθειες και την καθυστέρηση
2. **Ομαδοποιήστε συναλλαγές** για να παραλληλοποιήσετε τη διεκπεραίωση σε
   ολόκληρο το δίκτυο
3. **Αφαίρεση προμηθειών** ώστε οι χρήστες να μην χρειάζεται ποτέ να κατέχουν
   SOL
