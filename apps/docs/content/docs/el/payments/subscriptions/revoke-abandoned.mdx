---
title: Ανάκληση Εγκαταλελειμμένων Εκχωρήσεων και Συνδρομών
description:
  Οδηγός για την ανάκτηση rent από μια εκχώρηση ή συνδρομή της οποίας η
  Subscription Authority έχει εξαφανιστεί.
---

Οι εκχωρήσεις και οι συνδρομές διατηρούν rent που κανονικά επιστρέφει στον
καταχωρημένο πληρωτή κατά την ανάκληση. Εάν ένας χορηγός χρηματοδότησε τον
λογαριασμό και ο χρήστης αργότερα κλείσει ή αντικαταστήσει την Subscription
Authority του, ο λογαριασμός εγκαταλείπεται και το rent του παραμένει
δεσμευμένο. Τα `RevokeAbandonedDelegation` και `RevokeAbandonedSubscription`
επιτρέπουν στον καταχωρημένο πληρωτή να το ανακτήσει.

## Πότε Εφαρμόζεται

Επιτρέπεται μόνο όταν η Subscription Authority του λογαριασμού είναι
αποδεδειγμένα τερματισμένη — κλειστή, ή κλειστή και αναδημιουργημένη (οπότε το
`init_id` δεν ταιριάζει πλέον). Μια ενεργή εκχώρηση ή συνδρομή δεν μπορεί ποτέ
να κλείσει με αυτόν τον τρόπο.

|                                                      | `RevokeAbandonedDelegation`                                                                                              | `RevokeAbandonedSubscription`                                                                                                                                   |
| ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Κλείνει                                              | PDA σταθερής ή επαναλαμβανόμενης εκχώρησης                                                                               | PDA συνδρομής                                                                                                                                                   |
| Υπογράφων                                            | Καταχωρημένος πληρωτής                                                                                                   | Καταχωρημένος πληρωτής                                                                                                                                          |
| Λογαριασμοί                                          | Εκχώρηση + καταχωρημένη authority                                                                                        | Συνδρομή + καταχωρημένη authority + πλάνο                                                                                                                       |
| Επιλέξιμο όταν                                       | Η authority είναι κλειστή ή αναδημιουργημένη (αναντιστοιχία `init_id`)                                                   | Το ίδιο, για το mint του πλάνου — το mint διαβάζεται από το συνδεδεμένο πλάνο, οπότε δεν μπορεί να υποκατασταθεί μη σχετική authority για να επιβληθεί κλείσιμο |
| Χρησιμοποιήστε αντ' αυτού την κανονική ανάκληση όταν | Η εκχώρηση είναι [ληγμένη ή πλήρως εξαντλημένη](/docs/payments/subscriptions/recurring-delegation#revoke-the-delegation) | Η συνδρομή είναι [ακυρωμένη και ληγμένη](/docs/payments/subscriptions/subscription-plan#cancel-and-revoke)                                                      |

Το `RevokeAbandonedDelegation` είναι επίσης η μοναδική οδός ανάκτησης για τον
καταχωρημένο πληρωτή σε περίπτωση εκχώρησης χωρίς λήξη (`expiry_ts == 0`) που
δεν έχει εξαντληθεί πλήρως: η κανονική ανάκληση ενεργοποιείται για τον πληρωτή
μόνο κατά τη λήξη ή, για σταθερή εκχώρηση, μετά την πλήρη χρήση.

## Ανάκληση Εγκαταλελειμμένης Εκχώρησης

Υπογράφει ο καταχωρημένος πληρωτής. Περάστε το PDA της ανενεργής εκχώρησης και
την καταχωρημένη Subscription Authority της (η οποία μπορεί να είναι ήδη
κλειστή). Το rent επιστρέφει στον πληρωτή.

<Tabs items={["TypeScript", "Rust"]}>
  <Tab value="TypeScript">

    ```ts
    import { getRevokeAbandonedDelegationInstruction } from '@solana/subscriptions';

    // Build the instruction and send it in a transaction like any other.
    const revokeAbandonedIx = getRevokeAbandonedDelegationInstruction({
      payer: payerSigner,
      delegationAccount: delegationPda,
      subscriptionAuthority: subscriptionAuthorityPda,
    });
    ```

  </Tab>

  <Tab value="Rust">

    ```rust
    use solana_address::{address, Address};
    use subscriptions::generated::instructions::*;

    let payer: Address = address!("PAYER_THAT_FUNDED_RENT_HERE");
    let delegation_pda: Address = address!("DELEGATION_PDA_ADDRESS_HERE");
    let subscription_authority: Address = address!("RECORDED_SUBSCRIPTION_AUTHORITY_HERE");

    let revoke_abandoned_ix = RevokeAbandonedDelegationBuilder::new()
        .payer(payer)
        .delegation_account(delegation_pda)
        .subscription_authority(subscription_authority)
        .instruction();
    ```

  </Tab>
</Tabs>

## Ανάκληση Εγκαταλελειμμένης Συνδρομής

Ο καταγεγραμμένος πληρωτής υπογράφει. Περάστε το PDA της εγκαταλελειμμένης
συνδρομής, την καταγεγραμμένη Αρχή Συνδρομής (η οποία μπορεί να έχει ήδη
κλείσει) και το πλάνο στο οποίο ανήκει η συνδρομή. Το rent επιστρέφεται στον
πληρωτή.

<Tabs items={["TypeScript", "Rust"]}>
  <Tab value="TypeScript">

    ```ts
    import { getRevokeAbandonedSubscriptionInstruction } from '@solana/subscriptions';

    // Build the instruction and send it in a transaction like any other.
    const revokeAbandonedIx = getRevokeAbandonedSubscriptionInstruction({
      payer: payerSigner,
      subscriptionAccount: subscriptionPda,
      subscriptionAuthority: subscriptionAuthorityPda,
      planPda,
    });
    ```

  </Tab>

  <Tab value="Rust">

    ```rust
    use solana_address::{address, Address};
    use subscriptions::generated::instructions::*;

    let payer: Address = address!("PAYER_THAT_FUNDED_RENT_HERE");
    let subscription_pda: Address = address!("SUBSCRIPTION_PDA_ADDRESS_HERE");
    let subscription_authority: Address = address!("RECORDED_SUBSCRIPTION_AUTHORITY_HERE");
    let plan_pda: Address = address!("PLAN_PDA_ADDRESS_HERE");

    let revoke_abandoned_ix = RevokeAbandonedSubscriptionBuilder::new()
        .payer(payer)
        .subscription_account(subscription_pda)
        .subscription_authority(subscription_authority)
        .plan_pda(plan_pda)
        .instruction();
    ```

  </Tab>
</Tabs>

## Σημειώσεις

- Ο καταγεγραμμένος πληρωτής υπογράφει, όχι ο εκχωρητής, ο αποδέκτης εκχώρησης ή
  ο συνδρομητής.
- Και οι δύο εντολές αποτυγχάνουν εάν η Αρχή Συνδρομής είναι ακόμα ενεργή για
  τον λογαριασμό — χρησιμοποιήστε αντ' αυτού την κανονική ανάκληση.
- `RevokeAbandonedDelegation` λειτουργεί τόσο για σταθερές όσο και για
  επαναλαμβανόμενες εκχωρήσεις.
