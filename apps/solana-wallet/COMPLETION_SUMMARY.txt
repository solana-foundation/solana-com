Create New Wallet
1. Open http://localhost:3001
2. Click "Create New Wallet"
3. View generated keys
4. Keys auto-saved to localStorage

Import Existing Wallet
1. Click "Import Wallet"
2. Paste secret key (base58)
3. Click "Import"
4. Wallet restored from key

Send SOL
1. Enter recipient public key
2. Enter amount in SOL
3. Click "Send Transaction"
4. Confirm transaction
5. Balance updates automatically

Request Airdrop
1. Ensure you're on devnet/testnet
2. Click "Request 2 SOL Airdrop"
3. Wait 1-2 minutes
4. Check updated balance

Switch Networks
1. Click network button (devnet/testnet/mainnet)
2. Balance automatically refreshes
3. Data updated for new network

View Tokens
1. Click "Refresh Tokens"
2. See all SPL token accounts
3. Copy mint addresses as needed
