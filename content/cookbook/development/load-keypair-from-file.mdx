---
title: Load a local json file keypair
description: "Learn how to load a keypair from file."
---

When running your local project and scripts, you may want to use a json file
keypair. This can be very useful for all the cookbook examples as well.

If you do not already have a local keypair file created, use the Solana CLI to
create one:

```shell
solana-keygen new
```

You can also grind for a vanity keypair with the command below and load it into
your scripts:

```shell
solana-keygen grind --starts-with a23:1
```

### Load a keypair file

<Tabs persist groupId="language" items={['gill', 'web3.js v1']}>

<Tab>

To load the default Solana CLI keypair file (`~/.config/solana/id.json`):

```typescript title="TypeScript"
import { loadKeypairSignerFromFile } from "gill/node";

const signer = await loadKeypairSignerFromFile();
```

To load a different keypair file at a specific file path:

```typescript title="TypeScript"
import { loadKeypairSignerFromFile } from "gill/node";

const signer = await loadKeypairSignerFromFile(
  "/custom/file/path/to/keypair.json",
);
```

</Tab>

<Tab>

To load the default Solana CLI keypair file (`~/.config/solana/id.json`):

```typescript title="TypeScript"
import { getKeypairFromFile } from "@solana-developers/helpers";

const keypair = await getKeypairFromFile();
```

To load a different keypair file at a specific file path:

```typescript title="TypeScript"
import { getKeypairFromFile } from "@solana-developers/helpers";

const keypair = await getKeypairFromFile("/custom/file/path/to/keypair.json");
```

</Tab>
