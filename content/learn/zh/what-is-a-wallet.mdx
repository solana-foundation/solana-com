# 什么是钱包？

钱包是您使用 Solana 的主要工具。它存储您的数字资产，让您可以连接到应用程序，并使您能够发送和接收代币。本章将帮助您了解钱包并安全地设置您的第一个钱包。

## 您将学到什么

在本章结束时，您将了解：

- 钱包的功能及其工作原理
- 不同类型钱包的优缺点
- 如何创建您的第一个钱包
- 安全最佳实践
- 常见的钱包功能和用途

## 了解钱包

钱包有以下几个功能：

- **存储您的密钥**：证明您拥有资产的加密密钥
- **持有您的资产**：显示您的代币、NFT 和其他数字物品
- **签署交易**：代表您批准操作
- **连接到应用程序**：您登录 Solana 应用程序的方法

可以将其视为以下功能的结合：

- 密码管理器（存储您的密钥）
- 银行账户查看器（显示您的余额）
- 数字签名工具（批准交易）

## 钱包的工作原理

每个钱包都基于加密密钥构建：

### 种子短语

由 12-24 个单词组成的短语，可以用来重新创建您的钱包。这是您的最终备份——请妥善保管并保持私密。

### 私钥

从您的种子短语派生而来，此密钥用于签署交易。您的钱包会为您管理这一切——您无需直接查看或处理它。

### 公钥/地址

从您的私钥派生而来，这是您用来接收资金的地址。公开分享它是安全的。

## 钱包类型

### 软件钱包

**浏览器扩展**

- 适用于网页应用
- 使用方便
- 适合日常交易
- 免费使用

**移动应用**

- 随时随地访问您的钱包
- 支持二维码扫描支付
- 生物识别安全（指纹/面部识别）
- 免费使用

**桌面应用**

- 功能齐全的界面
- 适合活跃交易者
- 提供高级功能
- 免费使用

### 硬件钱包

**物理设备**

- 最高级别的安全性
- 离线密钥存储
- 最适合大额资产
- 价格：50-200 美元

### 选择您的第一个钱包

对于初学者，我们建议从浏览器扩展钱包开始。它们：

- 用户友好
- 支持良好
- 对日常使用安全
- 设置简单

## 创建您的第一个钱包

{/* 插图建议：逐步展示钱包创建过程的视觉指南 */}

让我们一起了解设置钱包的一般流程：

### 第一步：选择并安装一个钱包

1. 研究流行的 Solana 钱包
2. 访问官方网站（始终验证网址）
3. 下载适合您平台的钱包
4. 按照官方说明进行安装

### 第二步：创建一个新钱包

1. 打开您的钱包应用程序
2. 选择“创建新钱包”选项
3. 创建一个强密码以保护设备访问
4. **重要**：写下您的种子短语

### 第 3 步：保护您的种子短语

您的种子短语至关重要。以下是安全存储的方法：

**请务必：**

- 将其写在纸上
- 存放在安全的地方
- 考虑在不同地点保存多份副本
- 如果可能，使用防火保险箱

**切勿：**

- 截图保存
- 存储在电子邮件或云服务中
- 与任何人分享
- 在不可信的网站上输入

### 第 4 步：验证并完成

1. 按提示确认您的种子短语
2. 您的钱包现在可以使用了
3. 您将在顶部看到您的钱包地址
4. 您的初始余额将为 0 SOL

## 基本钱包功能

现代钱包包括：

### 资产管理

- 查看代币余额
- 显示 NFT 收藏
- 跟踪交易历史
- 监控投资组合价值

### 交易工具

- 发送和接收代币
- 不同代币之间的兑换
- 连接到应用程序
- 批准或拒绝交易

### 安全功能

- 密码保护
- 生物识别认证（移动端）
- 交易警告
- 钓鱼检测

## 您的第一步

现在您已经有了一个钱包，接下来可以这样做：

1. **保存您的地址**：将其复制到安全的地方以备参考
2. **获取一些 SOL**：您需要少量 SOL 用于支付交易费用
3. **练习接收**：让朋友发送少量资金给您
4. **探索界面**：熟悉钱包的布局

## 重要的安全提醒

- **切勿分享您的种子短语**：任何合法服务都不会要求提供它
- **验证网站**：骗子会创建假的钱包网站
- **从小额开始**：在学习过程中使用小额资金
- **保持软件更新**：及时安装钱包更新
- **保持警惕**：如果某事看起来好得不真实，那可能就是骗局

## 常见问题

### 我需要为钱包付费吗？

大多数软件钱包是免费的。您只需为硬件钱包或高级功能付费。

### 我可以拥有多个钱包吗？

可以，许多用户会根据不同用途（如日常使用、储蓄、特定应用）拥有多个钱包。

### 如果我忘记密码怎么办？

您可以使用种子短语重置设备密码。这就是种子短语如此重要的原因。

### 我的资产存储在钱包里吗？

不，资产存储在 Solana 上。您的钱包存储的是证明所有权的密钥。
