# Pengenalan Token Solana

Token adalah aset digital yang dibangun di Solana. Seperti halnya dolar AS yang
merupakan mata uang asli Amerika Serikat, SOL adalah mata uang asli Solana.
Token mewakili semua aset lainnya - dari stablecoin seperti USDC hingga token
proyek dan NFT. Bab ini menjelaskan bagaimana token bekerja dan apa yang dapat
Anda lakukan dengan token tersebut.

## Apa yang Akan Anda Pelajari

Pada akhir bab ini, Anda akan memahami:

- Apa itu token dan bagaimana cara kerjanya di Solana
- Berbagai jenis token dan penggunaannya
- Cara mengelola token di dompet Anda
- Token account dan biaya
- Pertimbangan keamanan saat berurusan dengan token

## Memahami Token di Solana

Solana memiliki dua jenis utama aset digital:

- **SOL**: Mata uang asli yang digunakan untuk transaksi dan biaya
- **Token**: Semua aset digital lainnya di Solana

Sama seperti aplikasi di ponsel Anda mengikuti standar tertentu agar berfungsi
dengan baik, token di Solana mengikuti standar umum yang membuatnya kompatibel
dengan dompet dan aplikasi.

## Jenis-jenis Token

{/* Illustration recommendation: Grid showing different token categories with examples and use cases */}

### Stablecoin

Versi digital dari mata uang tradisional yang mempertahankan nilai stabil.

**Contoh:**

- **USDC**: Dolar AS digital
- **USDT**: Tether, stablecoin USD lainnya
- **EURC**: Euro digital

**Kegunaan:** Menabung, pembayaran, trading, menghindari volatilitas

### Aset Wrapped

Token yang mewakili aset dari jaringan lain.

**Contoh:**

- **Wrapped Bitcoin (BTC)**: Bitcoin yang dapat digunakan di Solana
- **Wrapped Ethereum (ETH)**: Ethereum di Solana
- **Wrapped SOL**: Digunakan dalam beberapa protokol DeFi

**Kegunaan:** Akses cryptocurrency lain dengan kecepatan dan biaya rendah Solana

### Token Proyek

Token yang dibuat oleh aplikasi dan protokol.

**Contoh:**

- Token tata kelola untuk voting pada keputusan protokol
- Token bursa yang menawarkan diskon biaya
- Token protokol untuk partisipasi ekosistem

**Kegunaan:** Voting tata kelola, diskon biaya, rewards

### Token Utilitas

Token dengan fungsi spesifik dalam aplikasi.

**Contoh:**

- Token game untuk ekonomi dalam game
- Token sosial untuk komunitas kreator
- Token reward dari protokol

**Kegunaan:** Akses fitur, dapatkan rewards, berpartisipasi dalam ekosistem

## Cara Kerja Token Account

{/* Illustration recommendation: Wallet diagram showing main SOL account and separate token accounts as sub-compartments */}

Tidak seperti beberapa jaringan di mana semua aset berbagi satu alamat, Solana
menggunakan sistem token account:

1. **Dompet Utama**: Menyimpan SOL anda
2. **Token Account**: Akun terpisah untuk setiap jenis token

### Mengapa Token Account?

- Performa dan organisasi yang lebih baik
- Pemisahan yang jelas antara aset berbeda
- Penyimpanan jaringan yang efisien

### Rent Akun

- Setiap token account memerlukan deposit sekitar 0,002 SOL
- "rent" ini dapat dikembalikan saat menutup akun
- Mencegah spam dan mendanai penyimpanan jaringan
- Biasanya biayanya $0,20-0,50 per jenis token

## Mengelola Token Anda

### Menerima Token

1. Bagikan alamat dompet anda
2. Pengirim memulai transfer
3. Dompet membuat token account jika diperlukan
4. Token muncul di dompet anda

### Melihat Token Anda

Dompet modern menampilkan:

- Simbol dan logo token
- Saldo saat ini
- Nilai USD
- Transaksi terbaru

### Mengirim Token

1. Pilih token yang akan dikirim
2. Masukkan alamat penerima
3. Pilih jumlah
4. Konfirmasi transaksi
5. Bayar biaya SOL kecil (~$0.0005)

### Menutup Token Account

Ketika Anda tidak lagi membutuhkan token:

1. Kirim semua token keluar
2. Tutup akun
3. Dapatkan kembali deposit rent

## Dasar-dasar Pembuatan Token

Siapa pun dapat membuat token di Solana:

### Persyaratan

- Sejumlah kecil SOL untuk biaya (~0.5 SOL)
- Informasi dasar token (nama, simbol, jumlah)
- Opsional: logo dan metadata

### Penggunaan Umum

- Token komunitas
- Sistem hadiah
- Penggalangan dana
- Ekonomi game
- Proyek eksperimental

### Catatan Penting

Membuat token tidak memberikannya nilai. Nilai berasal dari utilitas, adopsi,
dan dinamika pasar.

## Keamanan Token

{/* Illustration recommendation: Warning signs and safety checklist */}

### Verifikasi Sebelum Mempercayai

- Periksa situs web resmi proyek
- Verifikasi alamat token
- Cari laporan audit
- Teliti tim pengembang

### Penipuan Umum yang Harus Dihindari

- **Token palsu**: Meniru proyek populer
- **Rug pulls**: Pembuat meninggalkan proyek
- **Honeypots**: Token yang bisa dibeli tapi tidak bisa dijual
- **Pump and dumps**: Manipulasi harga secara artifisial

### Praktik Terbaik

1. **Teliti secara menyeluruh** sebelum membeli
2. **Mulai dengan jumlah kecil** untuk token baru
3. **Gunakan sumber terpercaya** untuk informasi token
4. **Bersikap skeptis** terhadap jaminan keuntungan
5. **Periksa likuiditas** sebelum pembelian besar

## Bekerja dengan Token

### Menukar Token

Gunakan agregator DEX untuk mendapatkan rate terbaik:

1. Hubungkan dompet
2. Pilih token untuk ditukar
3. Tinjau rate dan biaya
4. Konfirmasi penukaran

### Memeriksa Informasi Token

Sumber daya untuk penelitian:

- Block explorer: Lihat detail token dan pemegangnya
- Agregator harga: Periksa harga dan likuiditas
- Situs resmi proyek: Verifikasi keaslian

### Mengelola Beberapa Token

- Atur dengan menyembunyikan saldo nol
- Favoritkan token yang sering digunakan
- Gunakan pelacak portofolio
- Tutup token account yang tidak digunakan

## Pertanyaan Umum

### Apakah saya memerlukan SOL untuk mengirim token?

Ya, semua transaksi memerlukan biaya SOL kecil, bahkan saat mengirim token lain.

### Mengapa beberapa token muncul secara otomatis?

Proyek dapat melakukan airdrop token ke dompet. Berhati-hatilah terhadap token
yang tidak diminta.

### Bisakah saya memulihkan token yang telah dikirim?

Tidak, transaksi di Solana bersifat permanen. Selalu periksa kembali alamat
tujuan.

### Apa yang menentukan nilai token?

Penawaran dan permintaan, utilitas, pengembangan proyek, dan sentimen pasar
semuanya memengaruhi harga token.
