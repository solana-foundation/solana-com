## Как использовать расширение Scaled UI Amount

Чтобы использовать расширение Scaled UI Amount, необходимо включить его для
токенового минта или токенового аккаунта.

### Включение расширения Scaled UI Amount для токенового минта

Чтобы включить расширение Scaled UI Amount для токенового минта, необходимо
установить поле `scaled_ui_amount_extension` в значение `true` в аккаунте
`Mint`. Вот пример создания токена с включенным расширением Scaled UI Amount с
использованием CLI `spl-token`:

```terminal title="Create Token"
$ spl-token --program-id TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb create-token --ui-amount-multiplier 1.5
Creating token 66EV4CaihdqyQ1fbsr51wBsoqKLgAG5KiYz7r5XNrxUM under program TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb

Address:  66EV4CaihdqyQ1fbsr51wBsoqKLgAG5KiYz7r5XNrxUM
Decimals:  9

Signature: 2sPziXu9M3duTCvsDvxQE9UKC9nBiLayi8muDvnjhA2qYvfXSZuaUieoq39MFjg4kf8xFrw6crmYSkPyV59dvudF
```

### Обновление множителя UI-значения

Чтобы обновить множитель UI-значения, необходимо использовать команду
`update-ui-amount-multiplier`. Таймстамп является необязательным и может быть
использован для установки пользовательского времени начала действия нового
множителя. Если таймстамп не указан, будет использован текущий таймстамп.

```terminal title="Update UI Amount Multiplier"
$ spl-token update-ui-amount-multiplier 66EV4CaihdqyQ1fbsr51wBsoqKLgAG5KiYz7r5XNrxUM 1.5 -- 1746471400
```

### Получение баланса

Чтобы получить баланс, необходимо использовать команду `balance`.

```terminal title="Fetch Balance"
$ spl-token balance 66EV4CaihdqyQ1fbsr51wBsoqKLgAG5KiYz7r5XNrxUM
```
