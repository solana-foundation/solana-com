**Ключ функции**:
[`C9oAhLxDBm3ssWtJx1yBGzPY55r2rArHmN1pbQn6HogH`](https://explorer.solana.com/address/C9oAhLxDBm3ssWtJx1yBGzPY55r2rArHmN1pbQn6HogH)

Этот флаг функции уменьшает стандартное выделение CU для инструкций встроенных
программ с 200,000 до 3,000 CUs, чтобы улучшить производительность валидаторов.

## Влияние

Менее 0.00001% транзакций будут затронуты этим изменением.

**Разработчикам** следует всегда использовать лимиты вычислительного бюджета для
явного выделения CUs для своих транзакций. Это изменение не повлияет на
транзакции, которые явно выделяют CUs.

Следующие встроенные программы будут иметь стандартное выделение CUs,
уменьшенное до 3,000 CUs в результате этого изменения:
