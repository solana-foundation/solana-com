Поскольку Solana является высокопроизводительным блокчейном с низкими комиссиями
и быстрыми транзакциями, вокруг него сформировалось отличное сообщество
разработчиков игр.

Здесь вы можете найти подробную информацию о SDK для разработки игр на Solana,
доступных в экосистеме. Все SDK имеют открытый исходный код и созданы
сообществом. Если вы сами работаете над SDK, не стесняйтесь открыть PR к этой
странице и добавить его.

## Unity SDK

Игровой движок Unity известен своим дружелюбным подходом для начинающих и
поддержкой различных платформ, включая WebGL, iOS и Android. Создайте один раз,
экспортируйте куда угодно.

- [Документация](https://docs.magicblock.gg/introduction)
- [Исходный код](https://github.com/magicblock-labs/Solana.Unity-SDK)
- [Примеры игр](https://github.com/solana-developers/solana-game-examples)
- [Крестики-нолики](https://blog.magicblock.gg/bolt-tic-tac-toe/)

SDK Solana для Unity поддерживается компанией
[Magicblock](https://www.magicblock.gg/) и включает в себя:

- Поддержку NFT
- Интеграцию с Candy Machine
- Транзакции
- Функции RPC
- Глубокие ссылки Phantom
- Коннектор WebGL
- Поддержку надежного WebSocket-соединения
- Адаптер для мобильных кошельков
- Сессионные ключи для автоматического подтверждения транзакций
- Генерацию клиентского кода Anchor и многое другое.

## Godot SDK

[Godot Engine](https://godotengine.org/) — это игровой движок с открытым
исходным кодом, который приобрел большую популярность за последние несколько
лет. SDK Solana для Godot поддерживается
[ZenRepublic](https://twitter.com/ZenRepublicNDM) и
[Virus-Axel](https://twitter.com/AxelBenjam). Он включает в себя интеграцию с
адаптером кошелька, транзакции, функции RPC и генерацию клиентского кода Anchor.
Даже полностью функциональную интеграцию с Metaplex
[Candy Machine](https://zenwiki.gitbook.io/solana-godot-sdk-docs/guides/setup-candy-machine).

- [SDK Solana для Godot Engine](https://github.com/Virus-Axel/godot-solana-sdk)
- [Документация](https://zenwiki.gitbook.io/solana-godot-sdk-docs)
- [Демо](https://github.com/ZenRepublic/GodotSolanaSDKDemoPackage)
- [Руководство](https://www.youtube.com/watch?v=tszFPInYmXQ)
- [Разработчик](https://twitter.com/ZenRepublicNDM)

## Turbo.Computer - игровой движок на Rust

Turbo — это игровой движок на языке Rust, созданный с нуля с фокусом на Solana,
с легкой архитектурой и быстрым циклом разработки. Он дружелюбен к начинающим и
поставляется с полной поддержкой Solana RPC. Вы даже можете использовать его
инструменты ИИ для генерации полноценных игр.

- [Веб-сайт](https://turbo.computer/)
- [Документация](https://docs.turbo.computer/)
- [Аккаунт в Twitter](https://twitter.com/makegamesfast)
- [Разработчик](https://twitter.com/jozanza)

# Honeycomb Protocol

Honeycomb предоставляет набор онchain-программ и инструментов сжатия состояния,
которые обрабатывают все ключевые функции жизненного цикла игры и обеспечивают
композицию внутри Solana Virtual Machine (SVM). Он поддерживает множество
функций NFT, таких как стейкинг, миссии, лутбоксы, профили игроков, сжатие
состояния, автоматическое подтверждение транзакций и многое другое.

- [Аккаунт в Twitter](https://twitter.com/honeycomb_prtcl)
- [Документация](https://docs.honeycombprotocol.com/)

# Unreal SDKs

Unreal Engine известен своей отличной графикой и системой программирования на
основе узлов. Существует несколько SDK для Solana, поддерживаемых разными
компаниями.

### Varmeta Unreal SDK

VAR META предоставляет SDK с открытым исходным кодом для Unreal, позволяющий
интегрировать функциональность блокчейна Solana в проекты Unreal Engine. Он
позволяет разработчикам взаимодействовать со смарт-контрактами Solana, управлять
кошельками и выполнять транзакции непосредственно в Unreal Engine.

- [Unreal SDK от Varmeta](https://github.com/VAR-META-Tech/UnrealSolanaSDK)
- [Разработчик](https://www.var-meta.com/)

### Bitfrost Unreal SDK

Bitfrost работает над SDK для Unreal, который уже использовался в нескольких
играх, созданных во время недавних игровых джемов Solana. Он поставляется с
поддержкой C# в C++ и Blueprints, созданием NFT на основе Metaplex, примерами
процессора платежей в игровом кошельке и многим другим.

- [Solana Unreal SDK от Bitfrost](https://github.com/Bifrost-Technologies/Solana-Unreal-SDK)
- [Руководство](https://www.youtube.com/watch?v=S8fm8mFeUkk)
- [Разработчик](https://twitter.com/BifrostTitan)

### Thugz Unreal SDK

Thugz — это технологическая студия и NFT-проект, который также поддерживает SDK
с открытым исходным кодом для Unreal на Solana. Он поставляется с множеством
функций, ориентированных на NFT, и плагин совместим с iOS и Android.

- [Плагин Thugz Blockchain](https://www.fab.com/listings/a7e220ba-93e5-4a3b-99e7-17b13da76f05)
- [Репозиторий](https://github.com/ThugzLabs/Thugz-BC-Plugin-Packaged-for-UE5.4)
- [Документация](https://thugzblockchainplugin-docs.vercel.app/)
- [Видеоурок](https://www.youtube.com/watch?v=dS7sTZd_E9U&ab_channel=ThugzNFT)

### Star Atlas Foundation Kit

Команда [Star Atlas](https://staratlas.com/) сделала открытым исходный код части

- [Star Atlas Unreal SDK](https://github.com/staratlasmeta/FoundationKit)

## Next.js/React + Anchor

Один из самых простых способов разработки на Solana — использование
JavaScript-фреймворка Web3js в сочетании с фреймворком Solana Anchor. Однако для
более сложных игр мы рекомендуем обратить внимание на игровые движки, такие как
[Unity](#unity-sdk) или [Unreal](#unreal-sdks).

Самый быстрый способ настроить игру на базе Next.js:

```shell
npx create-solana-game your-game-name
```

Это создаст отличное стартовое приложение с поддержкой `wallet-adapter`,
программой Anchor, приложением React и клиентом Unity, уже настроенными для
совместной работы. Преимущество использования Next.js заключается в том, что вы
можете использовать один и тот же код как на бэкенде, так и на фронтенде,
ускоряя разработку.

- [Web3.js](https://solana.com/docs/clients/javascript)
- [Видеоурок](https://www.youtube.com/watch?v=fnhivg_pemI&t=1s&ab_channel=Solana)

## Phaser

Быстрый, бесплатный и увлекательный фреймворк с открытым исходным кодом для
браузерных игр на основе Canvas и WebGL. [Phaser](https://phaser.io/) — отличный
способ начать разработку игр на Solana. Шаблон Phaser Solana Platformer —
прекрасная отправная точка для вашей игры.

- [Исходный код](https://github.com/Bread-Heads-NFT/phaser-solana-platformer-template)
- [Сопровождающий](https://twitter.com/blockiosaurus)

## Flutter

Flutter — это фреймворк с открытым исходным кодом, поддерживаемый Google, для
создания красивых, нативно компилируемых мультиплатформенных приложений из
единой кодовой базы. SDK Solana для Flutter поддерживается командой
[Espresso Cash](https://www.espressocash.com/).

- [Исходный код](https://github.com/espresso-cash/espresso-cash-public)

## Python

Python — это простой в освоении и широко используемый язык программирования
(часто применяемый в приложениях машинного обучения). Фреймворк Seahorse
позволяет разработчикам писать программы для Solana на Python. Seahorse построен
на основе фреймворка Anchor и преобразует код Python в код на Rust, основанный
на Anchor. В настоящее время Seahorse находится в бета-версии.

- [Документация](https://www.seahorse.dev/)
- [Репозиторий GitHub](https://github.com/solana-developers/seahorse)
- [Пример в Anchor Playground](https://beta.solpg.io/tutorials/hello-seahorse)

## Нативный C#

Оригинальный порт Web3js на C#. Он включает множество функций, таких как
транзакции, функции RPC и генерацию клиентского кода Anchor.

- [Исходный код и документация](https://github.com/bmresearch/Solnet/blob/master/docs/articles/getting_started.md)
