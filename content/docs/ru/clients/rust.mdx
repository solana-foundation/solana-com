Пакеты Rust от Solana
[опубликованы на crates.io](https://crates.io/search?q=solana-) и их можно найти
[на docs.rs](https://docs.rs/releases/search?query=solana-) с префиксом
`solana-`.

## Пакеты Rust

Ниже представлены наиболее важные и часто используемые пакеты Rust для
разработки на Solana:

- [`solana-program`] &mdash; Импортируется программами, работающими на Solana,
  компилируется в SBF. Этот пакет содержит множество фундаментальных типов
  данных и реэкспортируется из [`solana-sdk`], который не может быть
  импортирован из программы Solana.

- [`solana-sdk`] &mdash; Базовый SDK для офчейн-разработки, реэкспортирует
  [`solana-program`] и добавляет больше API поверх него. Большинство программ
  Solana, которые не работают в блокчейне, будут импортировать этот пакет.

- [`solana-client`] &mdash; Для взаимодействия с узлом Solana через
  [JSON RPC API](/docs/rpc).

- [`solana-cli-config`] &mdash; Загрузка и сохранение конфигурационного файла
  CLI Solana.

- [`solana-clap-utils`] &mdash; Функции для настройки CLI с использованием
  [`clap`], как в основном CLI Solana. Включает функции для загрузки всех типов
  подписывающих устройств, поддерживаемых CLI.

## Установка

Для отправки транзакций и взаимодействия с Solana:

```shell title="Terminal"
cargo add solana-sdk solana-client
```

Для создания программ Solana установите:

```shell title="Terminal"
cargo add solana-program
```
