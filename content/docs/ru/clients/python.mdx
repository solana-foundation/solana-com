---

Существует [Python SDK](https://github.com/michaelhly/solana-py), который
поддерживает большинство функций для взаимодействия с Solana. Если вы
используете Anchor для своих программ Solana, вы сможете использовать
[anchor-py](https://github.com/kevinheavey/anchorpy) с вашим idl для создания
клиента, совместимого с Python.

## Установка

```bash title="Terminal"
pip install solders
pip install solana
```
