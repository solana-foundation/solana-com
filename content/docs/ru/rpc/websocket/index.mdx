После подключения к RPC PubSub WebSocket по адресу `ws://<ADDRESS>/`:

- Отправляйте запросы на подписку в WebSocket, используя методы ниже
- Одновременно может быть активным несколько подписок
- Многие подписки принимают необязательный
  [параметр `commitment`](/docs/rpc/#configuring-state-commitment),
  определяющий, насколько завершённым должно быть изменение, чтобы вызвать
  уведомление. Для подписок, если параметр завершённости не указан, используется
  значение по умолчанию `finalized`.

## RPC PubSub WebSocket Endpoint

Порт по умолчанию: `8900`

- ws://localhost:8900
- http://192.168.1.88:8900
