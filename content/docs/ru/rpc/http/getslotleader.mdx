h1: Метод RPC getSlotLeader
---

Возвращает текущего лидера слота

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getSlotLeader",
  "params": [
    // !hover(1:3) 0
    {
      // !hover commitment
      "commitment": "finalized"
    }
  ]
}
```

```ts !!request title="Kit"
import { createSolanaRpc } from "@solana/kit";

const rpc_url = "https://api.devnet.solana.com";
const rpc = createSolanaRpc(rpc_url);

let slotLeader = await rpc.getSlotLeader().send();

console.log(slotLeader);
```

```ts !!request title="web3.js"
import { Connection, clusterApiUrl } from "@solana/web3.js";

const connection = new Connection(clusterApiUrl("devnet"), "confirmed");

let slotLeader = await connection.getSlotLeader();

console.log(slotLeader);
```

### !params

#### !! 0

!type object

Объект конфигурации, содержащий следующие поля:

##### !! commitment

!type string

Параметр commitment описывает, насколько завершённым является блок в данный
момент времени. См.
[Настройка завершённости состояния](/docs/rpc#configuring-state-commitment).

##### !! minContextSlot

!type number

Минимальный слот, на котором может быть выполнен запрос

### !!result

