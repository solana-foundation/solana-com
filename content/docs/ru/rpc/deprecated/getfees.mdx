h1: Метод RPC getFees
---

Возвращает недавний хэш блока из реестра, таблицу комиссий, которая может быть
использована для расчета стоимости отправки транзакции с его использованием, и
последний слот, в котором хэш блока будет действителен.

<Callout type="warn" title={"Устаревший метод"}>
  Ожидается, что этот метод будет удален в версии `solana-core` v2.0.
  Пожалуйста, используйте [getFeeForMessage](/docs/rpc/http/getfeeformessage)
  вместо него.
</Callout>
