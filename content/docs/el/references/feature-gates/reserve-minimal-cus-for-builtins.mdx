Αυτή η πύλη λειτουργίας μειώνει την προεπιλεγμένη κατανομή CU για τις εντολές
των ενσωματωμένων προγραμμάτων από 200.000 σε 3.000 CUs για τη βελτίωση της
απόδοσης του επικυρωτή.

## Επίδραση

Λιγότερο από το 0,00001% των συναλλαγών θα επηρεαστεί από αυτή την αλλαγή.

Οι **προγραμματιστές** θα πρέπει πάντα να χρησιμοποιούν όρια υπολογιστικού
προϋπολογισμού για να κατανέμουν ρητά CUs για τις συναλλαγές τους. Αυτή η αλλαγή
δεν θα επηρεάσει τις συναλλαγές που κατανέμουν ρητά CUs.

Τα ακόλουθα ενσωματωμένα προγράμματα θα έχουν την προεπιλεγμένη κατανομή CU τους
μειωμένη σε 3.000 CUs με αυτή την αλλαγή:
