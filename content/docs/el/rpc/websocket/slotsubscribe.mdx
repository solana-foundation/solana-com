h1: Μέθοδος RPC slotSubscribe
---

Εγγραφείτε για να λαμβάνετε ειδοποιήσεις κάθε φορά που ένα slot υποβάλλεται σε
επεξεργασία από τον validator

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "slotSubscribe"
}
```

### !params

**Κανένα**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

Αναγνωριστικό συνδρομής (απαραίτητο για την κατάργηση της εγγραφής)

</APIMethod>

#### Μορφή ειδοποίησης:

Η ειδοποίηση θα είναι ένα αντικείμενο με τα ακόλουθα πεδία:

- `parent: <u64>` - Το γονικό slot
- `root: <u64>` - Το τρέχον ριζικό slot
- `slot: <u64>` - Η νέα τιμή του slot

Παράδειγμα:

```json
{
  "jsonrpc": "2.0",
  "method": "slotNotification",
  "params": {
    "result": {
      "parent": 75,
      "root": 44,
      "slot": 76
    },
    "subscription": 0
  }
}
```
