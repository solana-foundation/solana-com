h1: Μέθοδος RPC programUnsubscribe
---

Κατάργηση εγγραφής από ειδοποιήσεις αλλαγών λογαριασμών που ανήκουν σε program

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "programUnsubscribe",
  "params": [
    // !hover 0
    0
  ]
}
```

### !params

#### !! 0

!type number  
!required

Αναγνωριστικό της συνδρομής λογαριασμού προς ακύρωση

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": true,
  "id": 1
}
```

!type boolean

Μήνυμα επιτυχίας κατάργησης εγγραφής

</APIMethod>
