## Τι είναι η επέκταση Κλιμακωμένου ποσού UI;

Η επέκταση κλιμακωμένου ποσού UI επιτρέπει στους εκδότες να εφαρμόζουν έναν
ενημερώσιμο πολλαπλασιαστή στο ποσό UI ενός token. Αυτό είναι χρήσιμο για
σενάρια όπου το token χρησιμοποιείται για να αντιπροσωπεύσει ένα περιουσιακό
στοιχείο του πραγματικού κόσμου, όπως μια μετοχή ή μέρισμα.

Για παράδειγμα, ένα split μετοχών διπλασιάζει τον αριθμό των μετοχών για όλους
τους κατόχους. Χωρίς αυτή την επέκταση, είναι επί του παρόντος μη πρακτικό να
εκδοθούν νέα tokens σε όλους τους κατόχους κατά τη διάρκεια ενός γεγονότος split
με αποδοτικό και κλιμακούμενο τρόπο.

Με το μοντέλο επέκτασης Token-2022, ωστόσο, έχουμε τη δυνατότητα να αλλάξουμε
τον τρόπο αναπαράστασης των ποσών UI των tokens. Χρησιμοποιώντας την επέκταση
ScaledUiAmount και την εντολή `amount_to_ui_amount`, μπορείτε να ορίσετε έναν
πολλαπλασιαστή UI στο token σας και να λάβετε το ποσό UI του ανά πάσα στιγμή.
[πηγαίος κώδικας](https://github.com/solana-program/token-2022/tree/main/program/src/extension/scaled_ui_amount).

Αυτή η λειτουργία μπορεί επίσης να χρησιμοποιηθεί για μερίσματα ή διανομή
αποδόσεων.

Σημείωση: Δεν δημιουργούνται ποτέ νέα tokens, το ποσό UI επιστρέφει την
ακατέργαστη ποσότητα των tokens πολλαπλασιασμένη με τον τρέχοντα πολλαπλασιαστή.
Η λειτουργία είναι εντελώς διακοσμητική.

<Callout type="info">
  Ορισμένες επεκτάσεις είναι ασύμβατες μεταξύ τους και δεν μπορείτε να τις
  ενεργοποιήσετε ταυτόχρονα στο ίδιο token mint ή token account. Δεν πρέπει να
  ενεργοποιήσετε ταυτόχρονα την επέκταση `ScaledUiAmount` και την επέκταση
  `InterestBearingMint` στο ίδιο token mint ή token account.
</Callout>
