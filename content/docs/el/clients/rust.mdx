Τα Rust crates της Solana είναι
[δημοσιευμένα στο crates.io](https://crates.io/search?q=solana-) και μπορούν να
βρεθούν [στο docs.rs](https://docs.rs/releases/search?query=solana-) με το
πρόθεμα `solana-`.

## Rust Crates

Τα ακόλουθα είναι τα πιο σημαντικά και συχνά χρησιμοποιούμενα Rust crates για
την ανάπτυξη στη Solana:

- [`solana-program`] &mdash; Εισάγεται από προγράμματα που εκτελούνται στη
  Solana, μεταγλωττισμένα σε SBF. Αυτό το crate περιέχει πολλούς θεμελιώδεις
  τύπους δεδομένων και επανεξάγεται από το [`solana-sdk`], το οποίο δεν μπορεί
  να εισαχθεί από ένα πρόγραμμα Solana.

- [`solana-sdk`] &mdash; Το βασικό offchain SDK, επανεξάγει το
  [`solana-program`] και προσθέτει περισσότερα APIs πάνω σε αυτό. Τα περισσότερα
  προγράμματα Solana που δεν εκτελούνται on-chain θα εισάγουν αυτό.

- [`solana-client`] &mdash; Για αλληλεπίδραση με έναν κόμβο Solana μέσω του
  [JSON RPC API](/docs/rpc).

- [`solana-cli-config`] &mdash; Φόρτωση και αποθήκευση του αρχείου ρυθμίσεων του
  Solana CLI.

- [`solana-clap-utils`] &mdash; Ρουτίνες για τη ρύθμιση ενός CLI,
  χρησιμοποιώντας το [`clap`], όπως χρησιμοποιείται από το κύριο Solana CLI.
  Περιλαμβάνει συναρτήσεις για τη φόρτωση όλων των τύπων υπογραφόντων που
  υποστηρίζονται από το CLI.

## Εγκατάσταση

Για αποστολή συναλλαγών και αλληλεπίδραση με τη Solana:

```shell title="Terminal"
cargo add solana-sdk solana-client
```

Για την ανάπτυξη προγραμμάτων Solana, εγκαταστήστε:

```shell title="Terminal"
cargo add solana-program
```
