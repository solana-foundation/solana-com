h1: Go SDK για το Solana
---

Υπάρχει ένα [Go SDK](https://github.com/gagliardetto/solana-go) που υποστηρίζει
τις περισσότερες λειτουργίες για αλληλεπίδραση με το Solana. Αν χρησιμοποιείτε
το Anchor για τα προγράμματα Solana σας, θα μπορείτε να χρησιμοποιήσετε το
[anchor-go](https://github.com/gagliardetto/anchor-go) με το idl σας για να
δημιουργήσετε ένα συμβατό πελάτη με τη Golang.

## Εγκατάσταση

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```

## Περισσότερη τεκμηρίωση

Παρακαλούμε επισκεφθείτε την
[τεκμηρίωση του Go SDK](https://github.com/gagliardetto/solana-go?tab=readme-ov-file#installation)
για περισσότερες πληροφορίες σχετικά με τη χρήση αυτού του Go SDK με το Solana.
