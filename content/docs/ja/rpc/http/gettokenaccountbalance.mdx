h1: getTokenAccountBalance RPCメソッド
---

SPLトークンアカウントのトークン残高を返します。

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getTokenAccountBalance",
  "params": [
    // !hover pubkey
    "7fUAJdStEuGbc3sM84cKRL6yYaaSstyLSU4ve5oovLS7",
    {
      // !hover commitment
      "commitment": "finalized"
    }
  ]
}
```

```ts !!request title="Kit"
import { address, createSolanaRpc } from "@solana/kit";

const rpc_url = "https://api.devnet.solana.com";
const rpc = createSolanaRpc(rpc_url);

// !hover pubkey
let tokenAddress = address("48gpnn8nsmkvkgso7462Z1nFhUrprGQ71u1YLBPzizbY");

let tokenBalance = await rpc.getTokenAccountBalance(tokenAddress).send();

console.log(tokenBalance);
```

```ts !!request title="web3.js"
import { Connection, PublicKey, clusterApiUrl } from "@solana/web3.js";

const connection = new Connection(clusterApiUrl("devnet"), "confirmed");

// !hover pubkey
let tokenAddress = new PublicKey(
  "48gpnn8nsmkvkgso7462Z1nFhUrprGQ71u1YLBPzizbY"
);

let tokenBalance = await connection.getTokenAccountBalance(tokenAddress);

console.log(tokenBalance);
```

```rs !!request title="Rust"
use anyhow::Result;
use solana_client::nonblocking::rpc_client::RpcClient;
use solana_sdk::{commitment_config::CommitmentConfig, pubkey};

#[tokio::main]
async fn main() -> Result<()> {
    let client = RpcClient::new_with_commitment(
        String::from("https://api.devnet.solana.com"),
        CommitmentConfig::confirmed(),
    );

    // !hover pubkey
    let token_address = pubkey!("48gpnn8nsmkvkgso7462Z1nFhUrprGQ71u1YLBPzizbY");
    let token_acc_bal = client.get_token_account_balance(&token_address).await?;

    println!("{:#?}", token_acc_bal);

    Ok(())
}
```

### !params

#### !! pubkey

!type string !required

クエリするトークンアカウントのpubkey（base-58エンコード文字列として）

#### !! 1

!type object

以下のフィールドを含む設定オブジェクト：

##### !! commitment

!type string

コミットメントは、その時点でブロックがどの程度確定しているかを示します。
[ステートコミットメントの設定](/docs/rpc#configuring-state-commitment)を参照してください。

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  "result": {
    "context": { "slot": 1114 },
    // !hover(1:6) value
    "value": {
      // !hover amount
      "amount": "9864",
      // !hover decimals
      "decimals": 2,
      // !hover uiAmount
      "uiAmount": 98.64,
      // !hover uiAmountString
      "uiAmountString": "98.64"
    }
  },
  "id": 1
}
```

!type object

結果は以下を含むJSONオブジェクトになります：

##### !! amount

!type string

小数点なしの生の残高、u64の文字列表現

##### !! decimals

!type u8

小数点の右側の10進数の桁数

##### !! uiAmount

!type number | null

ミントで規定された小数点を使用した残高 **非推奨**

##### !! uiAmountString

!type string

ミントで規定された小数点を使用した文字列形式の残高

</APIMethod>

</Callout>
