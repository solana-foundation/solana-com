Solanaとのほとんどの機能をサポートする[Python SDK](https://github.com/michaelhly/solana-py)があります。Solanaプログラムに対してAnchorを使用している場合は、idlと共に[anchor-py](https://github.com/kevinheavey/anchorpy)を使用してPython互換のクライアントを生成することができます。

## インストール

```bash title="Terminal"
pip install solders
pip install solana
```

## 詳細なドキュメント

SolanaでこのPython
SDKを使用する方法の詳細については、[Python SDKのドキュメント](https://github.com/michaelhly/solana-py?tab=readme-ov-file#general-usage)をご覧ください。
