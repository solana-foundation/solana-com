## スケールドUI金額拡張機能とは何ですか？

スケールドUI金額拡張機能により、発行者はトークンのUI金額に更新可能な乗数を適用することができます。これは、トークンが株式や配当金などの実世界の資産を表す場合に役立ちます。

例えば、株式分割ではすべての保有者の株式数が2倍になります。この拡張機能がなければ、分割イベント時に効率的かつスケーラブルな方法ですべての保有者に新しいトークンを発行することは現実的ではありません。

しかし、Token-2022拡張モデルでは、トークンのUI金額の表示方法を変更する可能性があります。ScaledUiAmount拡張機能と`amount_to_ui_amount`命令を使用して、トークンにUI乗数を設定し、いつでもそのUI金額を取得できます。
[ソースコード](https://github.com/solana-program/token-2022/tree/main/program/src/extension/scaled_ui_amount)。

この機能は配当金や利回りの分配にも使用できます。

注意：新しいトークンが作成されることはありません。UI金額は生のトークン量に現在の乗数を掛けたものを返します。この機能は完全に見た目上のものです。

<Callout type="info">
  一部の拡張機能は互いに互換性がなく、同じトークンミントやtoken
  accountで同時に有効にすることはできません。`ScaledUiAmount`拡張機能と`InterestBearingMint`拡張機能を同じトークンミントやtoken
  accountで有効にすべきではありません。
</Callout>
