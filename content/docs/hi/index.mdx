## सोलाना आज़माएं: 2048 खेलें

सोलाना पर 2048 खेलें जहां हर चाल एक ट्रांजैक्शन भेजती है। शुरू करने के लिए
"प्ले" पर क्लिक करें और एक फंडेड डेवनेट वॉलेट के साथ शुरुआत करें और टाइल्स को
नियंत्रित करने के लिए एरो कीज़ का उपयोग करें। यदि मोबाइल पर हैं, तो टाइल्स को
नियंत्रित करने के लिए स्वाइप करें।

### क्लाइंट साइड डेवलपमेंट

यदि आप क्लाइंट साइड पर डेवलप करते हैं, तो निम्नलिखित कम्युनिटी-योगदान किए गए SDK
लोकप्रिय भाषाओं में नेटवर्क के साथ इंटरैक्ट करने में आपकी मदद करते हैं:

| भाषा       | SDK                                                                                                                                       |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| Rust       | [solana_sdk](https://docs.rs/solana-sdk/latest/solana_sdk/)                                                                               |
| Typescript | [@solana/web3.js](https://github.com/anza-xyz/solana-web3.js)                                                                             |
| Python     | [solders](https://github.com/kevinheavey/solders)                                                                                         |
| Java       | [sava](https://sava.software) या [solanaj](https://github.com/skynetcap/solanaj) या [solana4j](https://github.com/LMAX-Exchange/solana4j) |
| C++        | [solcpp](https://github.com/mschneider/solcpp) या [solana-c](https://github.com/VAR-META-Tech/solana-c-sdk)                               |
| Go         | [solana-go](https://github.com/gagliardetto/solana-go)                                                                                    |
| Kotlin     | [sol4k](https://github.com/sol4k/sol4k) या [solanaKT](https://github.com/metaplex-foundation/SolanaKT)                                    |
| Dart       | [solana](https://github.com/espresso-cash/espresso-cash-public/tree/master/packages/solana)                                               |
| C#         | [solnet](https://github.com/bmresearch/Solnet)                                                                                            |
| GdScript   | [godot](https://github.com/Virus-Axel/godot-solana-sdk/)                                                                                  |
