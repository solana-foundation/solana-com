h1: Solana के लिए Python SDK
---

एक [Python SDK](https://github.com/michaelhly/solana-py) है जो Solana के साथ
इंटरैक्ट करने के लिए अधिकांश फंक्शन्स का समर्थन करती है। यदि आप अपने Solana
प्रोग्राम्स के लिए Anchor का उपयोग कर रहे हैं, तो आप अपने idl के साथ
[anchor-py](https://github.com/kevinheavey/anchorpy) का उपयोग करके Python संगत
क्लाइंट बना सकेंगे।

## इंस्टॉलेशन

```bash title="Terminal"
pip install solders
pip install solana
```
