h1: Solana के लिए Go SDK

एक [Go SDK](https://github.com/gagliardetto/solana-go) है जो Solana के साथ
इंटरैक्ट करने के लिए अधिकांश फंक्शन्स का समर्थन करती है। यदि आप अपने Solana
प्रोग्राम्स के लिए Anchor का उपयोग कर रहे हैं, तो आप
[anchor-go](https://github.com/gagliardetto/anchor-go) का उपयोग अपने idl के साथ
करके Golang कंपैटिबल क्लाइंट बना सकेंगे।

## इंस्टॉलेशन

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```

## अधिक दस्तावेज़ीकरण

Solana के साथ इस Go SDK के उपयोग के बारे में अधिक जानकारी के लिए कृपया
[Go SDK दस्तावेज़ीकरण](https://github.com/gagliardetto/solana-go?tab=readme-ov-file#installation)
देखें।
