## Rust Crates

निम्नलिखित Solana विकास के लिए सबसे महत्वपूर्ण और आमतौर पर उपयोग किए जाने वाले
Rust crates हैं:

- [`solana-program`] &mdash; Solana पर चलने वाले प्रोग्रामों द्वारा आयात किया
  जाता है, SBF में संकलित किया जाता है। इस crate में कई मौलिक डेटा प्रकार हैं और
  यह [`solana-sdk`] से पुनः निर्यात किया जाता है, जिसे Solana प्रोग्राम से आयात
  नहीं किया जा सकता।

- [`solana-sdk`] &mdash; मूल ऑफचेन SDK, यह [`solana-program`] को पुनः निर्यात
  करता है और उसके ऊपर अधिक APIs जोड़ता है। अधिकांश Solana प्रोग्राम जो ऑन-चेन
  नहीं चलते हैं, इसे आयात करेंगे।

- [`solana-client`] &mdash; [JSON RPC API](/docs/rpc) के माध्यम से Solana नोड के
  साथ इंटरैक्ट करने के लिए।

- [`solana-cli-config`] &mdash; Solana CLI कॉन्फ़िगरेशन फ़ाइल को लोड करने और
  सहेजने के लिए।

- [`solana-clap-utils`] &mdash; [`clap`] का उपयोग करके CLI सेटअप के लिए रूटीन,
  जैसा कि मुख्य Solana CLI द्वारा उपयोग किया जाता है। इसमें CLI द्वारा समर्थित
  सभी प्रकार के हस्ताक्षरकर्ताओं को लोड करने के लिए फ़ंक्शन शामिल हैं।

## इंस्टॉलेशन

Solana पर लेनदेन भेजने और इंटरैक्ट करने के लिए:

```shell title="Terminal"
cargo add solana-sdk solana-client
```

Solana प्रोग्राम बनाने के लिए, इंस्टॉल करें:

```shell title="Terminal"
cargo add solana-program
```
