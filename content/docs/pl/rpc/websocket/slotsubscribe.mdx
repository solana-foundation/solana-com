h1: Metoda RPC slotSubscribe
---

Subskrybuj, aby otrzymywać powiadomienia za każdym razem, gdy slot zostanie
przetworzony przez validator

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "slotSubscribe"
}
```

### !params

**Brak**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

Identyfikator subskrypcji (potrzebny do anulowania subskrypcji)

</APIMethod>

#### Format powiadomienia:

Powiadomienie będzie obiektem z następującymi polami:

- `parent: <u64>` - Slot nadrzędny
- `root: <u64>` - Bieżący slot główny
- `slot: <u64>` - Nowo ustawiona wartość slotu

Przykład:

```json
{
  "jsonrpc": "2.0",
  "method": "slotNotification",
  "params": {
    "result": {
      "parent": 75,
      "root": 44,
      "slot": 76
    },
    "subscription": 0
  }
}
```
