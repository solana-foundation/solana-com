## Czym jest rozszerzenie Skalowana ilość UI?

Rozszerzenie skalowanej ilości UI pozwala emitentom zastosować aktualizowalny
mnożnik do ilości UI tokena. Jest to przydatne w scenariuszach, w których token
reprezentuje rzeczywisty zasób, taki jak akcje lub dywidendy.

Na przykład podział akcji podwaja liczbę udziałów dla wszystkich posiadaczy. Bez
to rozszerzenie, obecnie jest to niepraktyczne, aby efektywnie i skalowalnie
wyemitować nowe tokeny wszystkim posiadaczom podczas wydarzenia podziału.

Jednak dzięki modelowi rozszerzeń Token-2022 mamy możliwość zmiany sposobu
reprezentacji ilości UI tokenów. Korzystając z rozszerzenia Skalowana ilość UI i
instrukcji `amount_to_ui_amount`, możesz ustawić mnożnik UI dla swojego tokena i
pobrać jego ilość UI w dowolnym momencie.
[kod źródłowy](https://github.com/solana-program/token-2022/tree/main/program/src/extension/scaled_ui_amount).

Ta funkcja może być również używana do dywidend lub dystrybucji zysków.

Uwaga: Nigdy nie są tworzone nowe tokeny, ilość UI zwraca surową ilość tokenów
pomnożoną przez aktualny mnożnik. Funkcja ma charakter wyłącznie kosmetyczny.

<Callout type="info">
  Niektóre rozszerzenia są ze sobą niekompatybilne i nie można ich włączyć
  jednocześnie na tym samym tokenie mint lub token account. Nie należy włączać
  zarówno rozszerzenia `ScaledUiAmount`, jak i rozszerzenia
  `InterestBearingMint` na tym samym tokenie mint lub token account.
</Callout>
