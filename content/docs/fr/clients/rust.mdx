Les crates Rust de Solana sont
[publiés sur crates.io](https://crates.io/search?q=solana-) et peuvent être
trouvés [sur docs.rs](https://docs.rs/releases/search?query=solana-) avec le
préfixe `solana-`.

## Crates Rust

Voici les crates Rust les plus importants et les plus couramment utilisés pour
le développement Solana :

- [`solana-program`] &mdash; Importé par les programmes s'exécutant sur Solana,
  compilé en SBF. Ce crate contient de nombreux types de données fondamentaux et
  est ré-exporté depuis [`solana-sdk`], qui ne peut pas être importé depuis un
  programme Solana.

- [`solana-sdk`] &mdash; Le SDK offchain de base, il ré-exporte
  [`solana-program`] et ajoute plus d'APIs par-dessus. La plupart des programmes
  Solana qui ne s'exécutent pas on-chain importeront celui-ci.

- [`solana-client`] &mdash; Pour interagir avec un nœud Solana via
  l'[API JSON RPC](/docs/rpc).

- [`solana-cli-config`] &mdash; Chargement et sauvegarde du fichier de
  configuration de la CLI Solana.

- [`solana-clap-utils`] &mdash; Routines pour configurer une CLI, utilisant
  [`clap`], comme utilisé par la CLI principale de Solana. Inclut des fonctions
  pour charger tous les types de signataires pris en charge par la CLI.

[`solana-program`]: https://docs.rs/solana-program
[`solana-sdk`]: https://docs.rs/solana-sdk
[`solana-client`]: https://docs.rs/solana-client
[`solana-cli-config`]: https://docs.rs/solana-cli-config

## Installation

Pour envoyer des transactions et interagir avec Solana :

```shell title="Terminal"
cargo add solana-sdk solana-client
```

Pour construire des programmes Solana, installez :

```shell title="Terminal"
cargo add solana-program
```
