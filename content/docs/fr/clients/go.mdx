h1: SDK Go pour Solana
---

Il existe un [SDK Go](https://github.com/gagliardetto/solana-go) qui prend en
charge la plupart des fonctions pour interagir avec Solana. Si vous utilisez
Anchor pour vos programmes Solana, vous pourrez utiliser
[anchor-go](https://github.com/gagliardetto/anchor-go) avec votre idl pour
produire un client compatible avec Golang.

## Installation

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```

## Plus de documentation

Veuillez consulter la
