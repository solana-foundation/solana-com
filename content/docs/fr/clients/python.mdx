h1: SDK Python pour Solana
---

Il existe un [SDK Python](https://github.com/michaelhly/solana-py) qui prend en
charge la plupart des fonctions pour interagir avec Solana. Si vous utilisez
Anchor pour vos programmes Solana, vous pourrez utiliser
[anchor-py](https://github.com/kevinheavey/anchorpy) avec votre idl pour
produire un client compatible avec Python.

## Installation

```bash title="Terminal"
pip install solders
pip install solana
```
