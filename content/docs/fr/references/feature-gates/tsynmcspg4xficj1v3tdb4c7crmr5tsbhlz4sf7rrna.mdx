**Clé de fonctionnalité**:
[`tSynMCspg4xFiCj1v3TDb4c7crMR5tSBhLz4sF7rrNA`](https://explorer.solana.com/address/tSynMCspg4xFiCj1v3TDb4c7crMR5tSBhLz4sF7rrNA)

Cette fonctionnalité ajoute une nouvelle instruction de vote comme option par
défaut. Les instructions de vote existantes continueront à fonctionner, mais les
nouveaux votes utiliseront le nouveau format d'instruction à l'avenir.

## Impact

**Les fournisseurs de données** devront être mis à jour pour suivre la nouvelle
instruction de vote, car le nouveau format d'instruction pourrait ne pas être
capturé dans les pipelines de vote existants.

## Activation

Cette fonctionnalité est active sur le mainnet beta depuis
l'[epoch 749](https://explorer.solana.com/epoch/748) (environ 22h00 ET le
2025-02-27)
