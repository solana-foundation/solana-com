## Qu'est-ce que l'extension Montant UI à l'échelle ?

L'extension de montant UI à l'échelle permet aux émetteurs d'appliquer un
multiplicateur modifiable au montant UI d'un token. Cela est utile dans les
scénarios où le token est utilisé pour représenter un actif du monde réel, comme
une action ou un dividende.

Par exemple, une division d'actions double le nombre d'actions pour tous les
détenteurs. Sans cette extension, il est actuellement peu pratique d'émettre de
nouveaux tokens à tous les détenteurs lors d'un événement de division de manière
efficace et évolutive.

Avec le modèle d'extension Token-2022, nous avons la possibilité de changer la
façon dont le montant UI des tokens est représenté. En utilisant l'extension
ScaledUiAmount et l'instruction `amount_to_ui_amount`, vous pouvez définir un
multiplicateur UI sur votre token et récupérer son montant UI à tout moment.
[code source](https://github.com/solana-program/token-2022/tree/main/program/src/extension/scaled_ui_amount).

Cette fonctionnalité peut également être utilisée pour les dividendes ou la
distribution de rendements.

Remarque : Aucun nouveau token n'est jamais créé, le montant UI renvoie le
montant brut de tokens multiplié par le multiplicateur actuel. La fonctionnalité
est entièrement cosmétique.

<Callout type="info">
  Certaines extensions sont incompatibles entre elles et vous ne pouvez pas les
  activer simultanément sur le même mint de token ou compte de token. Vous ne
  devriez pas activer à la fois l'extension `ScaledUiAmount` et l'extension
  `InterestBearingMint` sur le même mint de token ou compte de token.
</Callout>
