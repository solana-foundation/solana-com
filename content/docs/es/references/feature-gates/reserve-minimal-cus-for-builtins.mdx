**Clave de característica**:
[`C9oAhLxDBm3ssWtJx1yBGzPY55r2rArHmN1pbQn6HogH`](https://explorer.solana.com/address/C9oAhLxDBm3ssWtJx1yBGzPY55r2rArHmN1pbQn6HogH)

Esta puerta de característica reduce la asignación predeterminada de CU para las
instrucciones de programas integrados de 200.000 a 3.000 CUs para mejorar el
rendimiento del validador.

## Impacto

Menos del 0,00001% de las transacciones se verán afectadas por este cambio.

Los **desarrolladores** siempre deben usar límites de presupuesto de cómputo
para asignar explícitamente CUs para sus transacciones. Este cambio no afectará
a las transacciones que asignan CUs explícitamente.
