h1: SDK de Go para Solana
---

Existe un [SDK de Go](https://github.com/gagliardetto/solana-go) que admite la
mayoría de funciones para interactuar con Solana. Si estás usando Anchor para
tus programas de Solana, podrás utilizar
[anchor-go](https://github.com/gagliardetto/anchor-go) con tu idl para generar
un cliente compatible con Golang.

## Instalación

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```

## Más documentación

Por favor, visita la
[documentación del SDK de Go](https://github.com/gagliardetto/solana-go?tab=readme-ov-file#installation)
