h1: SDK de Python para Solana
---

Existe un [SDK de Python](https://github.com/michaelhly/solana-py) que admite la
mayoría de las funciones para interactuar con Solana. Si estás usando Anchor
para tus programas de Solana, podrás utilizar
[anchor-py](https://github.com/kevinheavey/anchorpy) con tu idl para generar un
cliente compatible con Python.

## Instalación

```bash title="Terminal"
pip install solders
pip install solana
```

## Más documentación

Por favor, visita la
[documentación del SDK de Python](https://github.com/michaelhly/solana-py?tab=readme-ov-file#general-usage)
