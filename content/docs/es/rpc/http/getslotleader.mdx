h1: Método RPC getSlotLeader
---

Devuelve el líder del slot actual

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getSlotLeader",
  "params": [
    // !hover(1:3) 0
    {
      // !hover commitment
      "commitment": "finalized"
    }
  ]
}
```

```ts !!request title="Kit"
import { createSolanaRpc } from "@solana/kit";

const rpc_url = "https://api.devnet.solana.com";
const rpc = createSolanaRpc(rpc_url);

let slotLeader = await rpc.getSlotLeader().send();

console.log(slotLeader);
```

```ts !!request title="web3.js"
import { Connection, clusterApiUrl } from "@solana/web3.js";

const connection = new Connection(clusterApiUrl("devnet"), "confirmed");

let slotLeader = await connection.getSlotLeader();

console.log(slotLeader);
```

### !params

#### !! 0

!type object

Objeto de configuración que contiene los siguientes campos:

##### !! commitment

!type string

El commitment describe qué tan finalizado está un bloque en ese momento. Ver
[Configuración del compromiso de estado](/docs/rpc#configuring-state-commitment).

##### !! minContextSlot

!type number

El slot mínimo en el que se puede evaluar la solicitud
