## 스케일된 UI 금액 확장이란 무엇인가요?

스케일된 UI 금액 확장은 발행자가 토큰의 UI 금액에 업데이트 가능한 승수를 적용할
수 있게 해줍니다. 이는 토큰이 주식이나 배당금과 같은 실제 자산을 나타내는
시나리오에서 유용합니다.

예를 들어, 주식 분할은 모든 보유자의 주식 수를 두 배로 늘립니다. 이 확장 기능이
없다면, 분할 이벤트 중에 모든 보유자에게 효율적이고 확장 가능한 방식으로 새
토큰을 발행하는 것은 현실적으로 어렵습니다.

그러나 Token-2022 확장 모델을 사용하면 토큰의 UI 금액이 표시되는 방식을 변경할
수 있습니다. ScaledUiAmount 확장과 `amount_to_ui_amount` 명령을 사용하여 토큰에
UI 승수를 설정하고 언제든지 UI 금액을 가져올 수 있습니다.
[소스 코드](https://github.com/solana-program/token-2022/tree/main/program/src/extension/scaled_ui_amount).

이 기능은 배당금이나 수익 분배에도 사용될 수 있습니다.

참고: 새 토큰은 절대 생성되지 않으며, UI 금액은 원시 토큰 양에 현재 승수를 곱한
값을 반환합니다. 이 기능은 완전히 시각적인 효과만 있습니다.

<Callout type="info">
  일부 확장 기능은 서로 호환되지 않으며 동일한 토큰 민트나 token account에서
  동시에 활성화할 수 없습니다. `ScaledUiAmount` 확장과 `InterestBearingMint`
  확장을 동일한 토큰 민트나 token account에서 함께 활성화해서는 안 됩니다.
</Callout>
