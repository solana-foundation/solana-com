h1: Solana용 Python SDK

Solana와 상호작용하기 위한 대부분의 기능을 지원하는
[Python SDK](https://github.com/michaelhly/solana-py)가 있습니다. Solana
프로그램에 Anchor를 사용하고 있다면, idl과 함께
[anchor-py](https://github.com/kevinheavey/anchorpy)를 사용하여 Python 호환
클라이언트를 생성할 수 있습니다.

## 설치

```bash title="Terminal"
pip install solders
pip install solana
```
