솔라나는 낮은 수수료와 빠른 트랜잭션을 제공하는 고성능 블록체인으로, 이를
중심으로 훌륭한 게임 개발자 커뮤니티가 형성되었습니다.

여기서는 생태계 내에서 사용 가능한 솔라나 게임 개발 SDK에 대한 세부 정보를
확인할 수 있습니다. 모든 SDK는 오픈 소스이며 커뮤니티에 의해 구축되었습니다.
직접 SDK를 개발 중이라면 이 페이지에 PR을 열고 추가하는 것을 환영합니다.

## 유니티 SDK

유니티 게임 엔진은 초보자 친화적인 접근 방식과 WebGL, iOS, Android를 포함한
크로스 플랫폼 지원으로 유명합니다. 한 번 빌드하고 어디서나 내보낼 수 있습니다.

- [문서](https://docs.magicblock.gg/introduction)
- [소스](https://github.com/magicblock-labs/Solana.Unity-SDK)
- [예제 게임](https://github.com/solana-developers/solana-game-examples)
- [틱택토](https://blog.magicblock.gg/bolt-tic-tac-toe/)

솔라나 유니티 SDK는 [Magicblock](https://www.magicblock.gg/)에서 유지 관리하며
다음과 같은 기능을 제공합니다:

- NFT 지원
- 캔디 머신 통합
- 트랜잭션
- RPC 함수
- 팬텀 딥 링크
- WebGL 커넥터
- 안정적인 WebSocket 연결 지원
- 모바일 지갑 어댑터
- 트랜잭션 자동 승인을 위한 세션 키
- Anchor 클라이언트 코드 생성 등

## 고도 SDK

[고도 엔진](https://godotengine.org/)은 최근 몇 년간 많은 지원을 받은 오픈 소스
게임 엔진입니다. 고도용 솔라나 SDK는
[ZenRepublic](https://twitter.com/ZenRepublicNDM)과
[Virus-Axel](https://twitter.com/AxelBenjam)에서 유지 관리합니다. 이 SDK는 지갑
어댑터 통합, 트랜잭션, RPC 함수, Anchor 클라이언트 코드 생성 기능을 제공합니다.
심지어 완전히 기능적인 Metaplex
[캔디 머신 통합](https://zenwiki.gitbook.io/solana-godot-sdk-docs/guides/setup-candy-machine)도
포함되어 있습니다.

- [솔라나 고도 엔진 SDK](https://github.com/Virus-Axel/godot-solana-sdk)
- [문서](https://zenwiki.gitbook.io/solana-godot-sdk-docs)
- [데모](https://github.com/ZenRepublic/GodotSolanaSDKDemoPackage)
- [튜토리얼](https://www.youtube.com/watch?v=tszFPInYmXQ)
- [유지 관리자](https://twitter.com/ZenRepublicNDM)

## Turbo.Computer - 러스트 게임 엔진

터보는 솔라나에 초점을 맞춰 처음부터 작성된 러스트 게임 엔진으로, 경량
아키텍처와 빠른 반복 시간을 제공합니다. 초보자에게 친숙하며 완전한 솔라나 RPC
지원을 제공합니다. 심지어 AI 도구를 사용하여 완전한 게임을 생성할 수도 있습니다.

- [웹사이트](https://turbo.computer/)
- [문서](https://docs.turbo.computer/)
- [트위터 계정](https://twitter.com/makegamesfast)
- [관리자](https://twitter.com/jozanza)

# 허니콤 프로토콜

허니콤은 솔라나 가상 머신(SVM) 내에서 모든 중요한 게임 라이프사이클 기능과
상호운용성을 처리하는 온체인 프로그램과 상태 압축 도구 모음을 제공합니다.
스테이킹, 미션, 루트 박스, 플레이어 프로필, 상태 압축, 트랜잭션 자동 승인 등
다양한 NFT 기능을 지원합니다.

- [트위터 계정](https://twitter.com/honeycomb_prtcl)
- [문서](https://docs.honeycombprotocol.com/)

# 언리얼 SDK

언리얼 엔진은 뛰어난 비주얼과 노드 기반 스크립팅 프레임워크로 유명합니다. 여러
회사에서 관리하는 다양한 솔라나 SDK가 있습니다.

### 바메타 언리얼 SDK

VAR META는 언리얼 엔진 프로젝트에 솔라나 블록체인 기능을 통합하기 위한 오픈소스
언리얼 SDK를 제공합니다. 개발자가 언리얼 엔진 내에서 직접 솔라나의 스마트 계약과
상호작용하고, 지갑을 관리하며, 트랜잭션을 실행할 수 있게 해줍니다.

- [바메타의 언리얼 SDK](https://github.com/VAR-META-Tech/UnrealSolanaSDK)
- [관리자](https://www.var-meta.com/)

### 비트프로스트 언리얼 SDK

비트프로스트는 최근 솔라나 게임 잼에서 제작된 여러 게임에 이미 사용된 언리얼
SDK를 개발 중입니다. C++와 블루프린트에서 C# `solnet` 지원, 메타플렉스 NFT 민팅,
게임 내 지갑 결제 처리기 예제 등을 제공합니다.

- [비트프로스트의 솔라나 언리얼 SDK](https://github.com/Bifrost-Technologies/Solana-Unreal-SDK)
- [튜토리얼](https://www.youtube.com/watch?v=S8fm8mFeUkk)
- [관리자](https://twitter.com/BifrostTitan)

### 터그즈 언리얼 SDK

터그즈는 오픈소스 솔라나용 언리얼 SDK를 유지 관리하는 기술 스튜디오이자 NFT
프로젝트입니다. NFT 중심 기능이 많이 포함되어 있으며 이 플러그인은 iOS 및
Android와도 호환됩니다.

- [Thugz 블록체인 플러그인](https://www.fab.com/listings/a7e220ba-93e5-4a3b-99e7-17b13da76f05)
- [리포지토리](https://github.com/ThugzLabs/Thugz-BC-Plugin-Packaged-for-UE5.4)
- [문서](https://thugzblockchainplugin-docs.vercel.app/)
- [비디오 튜토리얼](https://www.youtube.com/watch?v=dS7sTZd_E9U&ab_channel=ThugzNFT)

### Star Atlas 파운데이션 킷

[Star Atlas](https://staratlas.com/) 팀은 *파운데이션 킷*이라고 불리는 SDK
스택의 일부를 오픈소스로 공개했습니다. 현재 적극적으로 유지보수되고 있지는
않지만, 프로젝트를 시작하는 데 좋은 출발점이 될 수 있습니다.

- [Star Atlas 언리얼 SDK](https://github.com/staratlasmeta/FoundationKit)
- [튜토리얼](https://www.youtube.com/watch?v=S8fm8mFeUkk)

## Next.js/React + Anchor

Solana에서 개발하는 가장 쉬운 방법 중 하나는 Web3js 자바스크립트 프레임워크와
Solana Anchor 프레임워크를 함께 사용하는 것입니다. 하지만 더 복잡한 게임의 경우,
[Unity](#unity-sdk)나 [언리얼](#unreal-sdks)과 같은 게임 엔진 사용을 권장합니다.

Next.js 기반 게임을 설정하는 가장 빠른 방법:

```shell
npx create-solana-game your-game-name
```

이렇게 하면 `wallet-adapter` 지원, Anchor 프로그램, React 앱, 그리고 함께
작동하도록 이미 구성된 Unity 클라이언트가 포함된 훌륭한 시작 애플리케이션이
생성됩니다. Next.js를 사용하는 이점은 백엔드와 프론트엔드에서 동일한 코드를
사용할 수 있어 개발 속도를 높일 수 있다는 것입니다.

- [Web3.js](https://solana.com/docs/clients/javascript)
- [비디오 튜토리얼](https://www.youtube.com/watch?v=fnhivg_pemI&t=1s&ab_channel=Solana)

> Solana SDK를 개발 중이고 프리셋에 추가하고 싶다면 다음 위치에 PR을 열 수
> 있습니다:
> [Solana 게임 프리셋](https://github.com/solana-developers/solana-game-preset)

## Phaser

Canvas와 WebGL을 기반으로 한 브라우저 게임을 위한 빠르고, 무료이며, 재미있는
오픈 소스 프레임워크입니다. [Phaser](https://phaser.io/)는 Solana에서 게임
개발을 시작하기에 좋은 방법입니다. Phaser Solana 플랫포머 템플릿은 게임을
시작하기에 좋은 출발점입니다.

- [소스 코드](https://github.com/Bread-Heads-NFT/phaser-solana-platformer-template)
- [유지보수 담당자](https://twitter.com/blockiosaurus)

## Flutter

Flutter는 Google이 유지보수하는 오픈 소스 프레임워크로, 단일 코드베이스에서
아름답고 네이티브로 컴파일된 멀티 플랫폼 애플리케이션을 구축할 수 있습니다.
Solana Flutter SDK는 [Espresso Cash](https://www.espressocash.com/) 팀이
유지보수하고 있습니다.

- [소스 코드](https://github.com/espresso-cash/espresso-cash-public)

## 파이썬

파이썬은 배우기 쉽고 널리 사용되는 프로그래밍 언어입니다(주로 머신러닝
애플리케이션에서 사용됨). Seahorse 프레임워크를 통해 개발자는 파이썬으로 Solana
프로그램을 작성할 수 있습니다. Seahorse는 Anchor 프레임워크 위에 구축되어 있으며
파이썬 코드를 Anchor 기반 러스트 코드로 변환합니다. Seahorse는 현재 베타
버전입니다.

- [문서](https://www.seahorse.dev/)
- [깃허브 저장소](https://github.com/solana-developers/seahorse)
- [Anchor 플레이그라운드 예제](https://beta.solpg.io/tutorials/hello-seahorse)

코드 생성과 같은 다양한 기능을 제공합니다.

- [소스 및 문서](https://github.com/bmresearch/Solnet/blob/master/docs/articles/getting_started.md)
