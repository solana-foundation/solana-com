h1: rootSubscribe RPC Metodu
---

Validator tarafından yeni bir root belirlendiğinde bildirim almak için abone
olun.

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**Yok**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

abonelik kimliği (abonelikten çıkmak için gereklidir)

</APIMethod>

#### Bildirim Formatı:

Sonuç, en son root slot numarasıdır.

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
