## Ölçeklendirilmiş UI Miktarı uzantısı nedir?

Ölçeklendirilmiş UI miktarı uzantısı, ihraççıların bir token'ın UI miktarına
güncellenebilir bir çarpan uygulamasına olanak tanır. Bu, token'ın bir hisse
senedi veya temettü gibi gerçek dünya varlıklarını temsil ettiği senaryolar için
kullanışlıdır.

Örneğin, bir hisse senedi bölünmesi, tüm sahipler için hisse sayısını ikiye
katlar. Bu uzantı olmadan, bir bölünme olayı sırasında tüm sahiplere verimli ve
ölçeklenebilir bir şekilde yeni token'lar basmak şu anda pratik değildir.

Ancak Token-2022 uzantı modeliyle, token'ların UI miktarlarının nasıl temsil
edildiğini değiştirme imkanına sahibiz. ScaledUiAmount uzantısını ve
`amount_to_ui_amount` talimatını kullanarak, token'ınıza bir UI çarpanı
ayarlayabilir ve UI miktarını istediğiniz zaman alabilirsiniz.
[kaynak kodu](https://github.com/solana-program/token-2022/tree/main/program/src/extension/scaled_ui_amount).

Bu özellik ayrıca temettüler veya getiri dağıtımı için de kullanılabilir.

Not: Hiçbir zaman yeni token oluşturulmaz, UI miktarı, ham token miktarını
mevcut çarpanla çarparak döndürür. Özellik tamamen kozmetiktir.

<Callout type="info">
  Bazı uzantılar birbiriyle uyumsuzdur ve aynı token mint veya token account
  üzerinde eş zamanlı olarak etkinleştirilemezler. `ScaledUiAmount` uzantısı ile
  `InterestBearingMint` uzantısını aynı token mint veya token account üzerinde
  etkinleştirmemelisiniz.
</Callout>
