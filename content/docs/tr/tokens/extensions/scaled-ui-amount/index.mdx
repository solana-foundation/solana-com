## Ölçeklendirilmiş UI Miktarı uzantısı nedir?

Ölçeklendirilmiş UI miktarı uzantısı, ihraççıların bir tokenin UI miktarına
güncellenebilir bir çarpan uygulamasına olanak tanır. Bu, tokenin hisse senedi
veya temettü gibi gerçek dünya varlıklarını temsil etmek için kullanıldığı
senaryolarda faydalıdır.

Örneğin, bir hisse senedi bölünmesi tüm sahipler için hisse sayısını ikiye
katlar. Bu uzantı olmadan, bir bölünme olayı sırasında tüm sahiplere verimli ve
ölçeklenebilir bir şekilde yeni tokenler basmak şu anda pratik değildir.

Ancak Token-2022 uzantı modeliyle, tokenlerin UI miktarlarının nasıl temsil
edildiğini değiştirme imkanına sahibiz. ScaledUiAmount uzantısını ve
`amount_to_ui_amount` talimatını kullanarak, tokeninize bir UI çarpanı
ayarlayabilir ve UI miktarını istediğiniz zaman alabilirsiniz.
[kaynak kodu](https://github.com/solana-program/token-2022/tree/main/program/src/extension/scaled_ui_amount).

Bu özellik ayrıca temettüler veya getiri dağıtımı için de kullanılabilir.

Not: Hiçbir zaman yeni tokenler oluşturulmaz, UI miktarı ham token miktarını
mevcut çarpanla çarpılmış olarak döndürür. Özellik tamamen kozmetiktir.

<Callout type="info">
  Bazı uzantılar birbiriyle uyumsuzdur ve aynı token mint veya token account
  üzerinde eş zamanlı olarak etkinleştirilemezler. Aynı token mint veya token
  account üzerinde hem `ScaledUiAmount` uzantısını hem de `InterestBearingMint`
  uzantısını etkinleştirmemelisiniz.
</Callout>
