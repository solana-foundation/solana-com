## Ölçeklendirilmiş UI Miktar uzantısı nasıl kullanılır

Ölçeklendirilmiş UI Miktar uzantısını kullanmak için, bunu bir token mint veya
token account üzerinde etkinleştirmeniz gerekir.

### Bir token mint üzerinde Ölçeklendirilmiş UI Miktar uzantısını etkinleştirme

Bir token mint üzerinde Ölçeklendirilmiş UI Miktar uzantısını etkinleştirmek
için, `Mint` hesabındaki `scaled_ui_amount_extension` alanını `true` olarak
ayarlamanız gerekir. İşte `spl-token` CLI kullanarak Ölçeklendirilmiş UI Miktar
uzantısı etkinleştirilmiş bir token oluşturma örneği:

```terminal title="Create Token"
$ spl-token --program-id TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb create-token --ui-amount-multiplier 1.5
Creating token 66EV4CaihdqyQ1fbsr51wBsoqKLgAG5KiYz7r5XNrxUM under program TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb

Address:  66EV4CaihdqyQ1fbsr51wBsoqKLgAG5KiYz7r5XNrxUM
Decimals:  9

Signature: 2sPziXu9M3duTCvsDvxQE9UKC9nBiLayi8muDvnjhA2qYvfXSZuaUieoq39MFjg4kf8xFrw6crmYSkPyV59dvudF
```

### UI miktar çarpanını güncelleme

UI miktar çarpanını güncellemek için, `update-ui-amount-multiplier` komutunu
kullanmanız gerekir. Zaman damgası isteğe bağlıdır ve yeni çarpan için özel bir
başlangıç zamanı belirlemek için kullanılabilir. Eğer zaman damgası
belirtilmezse, mevcut zaman damgası kullanılacaktır.

```terminal title="Update UI Amount Multiplier"
$ spl-token update-ui-amount-multiplier 66EV4CaihdqyQ1fbsr51wBsoqKLgAG5KiYz7r5XNrxUM 1.5 -- 1746471400
```

### Bakiyeyi Getirme

Bakiyeyi getirmek için, `balance` komutunu kullanmanız gerekir.

```terminal title="Fetch Balance"
$ spl-token balance 66EV4CaihdqyQ1fbsr51wBsoqKLgAG5KiYz7r5XNrxUM
```
