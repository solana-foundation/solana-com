bulunabilir.

## Rust paketleri

Aşağıdakiler, Solana geliştirme için en önemli ve yaygın olarak kullanılan Rust
paketleridir:

- [`solana-program`] &mdash; Solana üzerinde çalışan, SBF'ye derlenen programlar
  tarafından içe aktarılır. Bu paket birçok temel veri türü içerir ve Solana
  programından içe aktarılamayan [`solana-sdk`]'den yeniden dışa aktarılır.

- [`solana-sdk`] &mdash; Temel zincir dışı SDK, [`solana-program`]'yi yeniden
  dışa aktarır ve üzerine daha fazla API ekler. Zincir üzerinde çalışmayan çoğu
  Solana programı bunu içe aktaracaktır.

- [`solana-client`] &mdash; [JSON RPC API](/docs/rpc) aracılığıyla bir Solana
  düğümü ile etkileşim kurmak için.

- [`solana-cli-config`] &mdash; Solana CLI yapılandırma dosyasını yükleme ve
  kaydetme.

- [`solana-clap-utils`] &mdash; Ana Solana CLI tarafından kullanılan [`clap`]
  kullanarak bir CLI kurulumu için rutinler. CLI tarafından desteklenen tüm
  imzalayıcı türlerini yükleme fonksiyonlarını içerir.

## Kurulum

İşlem göndermek ve Solana ile etkileşim kurmak için:

```shell title="Terminal"
cargo add solana-sdk solana-client
```

Solana programlarını oluşturmak için şunu yükleyin:

```shell title="Terminal"
cargo add solana-program
```
