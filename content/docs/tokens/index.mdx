title: Token Programs
description: Learn about Solana's Token Programs.
---

On Solana, two main token programs exist (developed by
[Anza](https://www.anza.xyz/), formerly Solana Labs).

You can find the source code for both programs below.

<Cards>
  <Card icon={<Coins/>} title="Token Program (Original)" href="https://github.com/solana-program/token">
   - Basic token capability (mint, transfer, etc.)
   - Immutable and widely used
  </Card>

  <Card icon={<Coins/>} title="Token Extension Program (Token 2022)" href="https://github.com/solana-program/token-2022">
   - Includes all original Token Program features
   - Adds more features through "extensions"
   - Recommended for new tokens
  </Card>
</Cards>
