Esiste un [SDK Python](https://github.com/michaelhly/solana-py) che supporta la
maggior parte delle funzioni per interagire con Solana. Se stai utilizzando
Anchor per i tuoi programmi Solana, potrai utilizzare
[anchor-py](https://github.com/kevinheavey/anchorpy) con il tuo idl per creare
un client compatibile con Python.

## Installazione

```bash title="Terminal"
pip install solders
pip install solana
```
