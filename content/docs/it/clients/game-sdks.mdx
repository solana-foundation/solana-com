Essendo Solana una blockchain ad alte prestazioni con commissioni basse e
transazioni veloci, si è formata attorno ad essa una grande comunità di
sviluppatori di giochi.

Qui puoi trovare dettagli sugli SDK per lo sviluppo di giochi su Solana
disponibili nell'ecosistema. Tutti gli SDK sono open source e costruiti dalla
comunità. Se stai lavorando tu stesso su un SDK, sentiti libero di aprire una PR
a questa pagina e aggiungerlo.

## SDK per Unity

Il motore di gioco Unity è noto per il suo approccio amichevole per i
principianti e il supporto multi-piattaforma che include WebGL, iOS e Android.
Costruisci una volta, esporta ovunque.

- [Documentazione](https://docs.magicblock.gg/introduction)
- [Codice sorgente](https://github.com/magicblock-labs/Solana.Unity-SDK)
- [Giochi di esempio](https://github.com/solana-developers/solana-game-examples)
- [Tris](https://blog.magicblock.gg/bolt-tic-tac-toe/)

L'SDK Unity per Solana è mantenuto da [Magicblock](https://www.magicblock.gg/) e
include:

- Supporto per NFT
- Integrazione con candy machine
- Transazioni
- Funzioni RPC
- Deep link per Phantom
- Connettore WebGL
- Supporto affidabile per connessioni WebSocket
- Wallet-adapter per dispositivi mobili
- Chiavi di sessione per l'approvazione automatica delle transazioni
- Generazione di codice client per Anchor e altro ancora.

## SDK per Godot

Il [Godot Engine](https://godotengine.org/) è un motore di gioco open source che
ha guadagnato molto supporto negli ultimi anni. L'SDK Solana per Godot è
mantenuto da [ZenRepublic](https://twitter.com/ZenRepublicNDM) e
[Virus-Axel](https://twitter.com/AxelBenjam). Include un'integrazione con wallet
adapter, transazioni, funzioni RPC e generazione di codice client per Anchor.
Include persino un'integrazione completamente funzionale con Metaplex
[candy machine](https://zenwiki.gitbook.io/solana-godot-sdk-docs/guides/setup-candy-machine).

- [SDK Solana per Godot Engine](https://github.com/Virus-Axel/godot-solana-sdk)
- [Documentazione](https://zenwiki.gitbook.io/solana-godot-sdk-docs)
- [Demo](https://github.com/ZenRepublic/GodotSolanaSDKDemoPackage)
- [Tutorial](https://www.youtube.com/watch?v=tszFPInYmXQ)
- [Maintainer](https://twitter.com/ZenRepublicNDM)

## Turbo.Computer - Motore di gioco Rust

Turbo è un motore di gioco in rust scritto da zero per concentrarsi su Solana,
con un'architettura leggera e tempi di iterazione rapidi. È adatto ai
principianti e include il supporto completo per Solana RPC. Puoi persino
utilizzare i suoi strumenti di IA per generare giochi completi.

- [Sito web](https://turbo.computer/)
- [Documentazione](https://docs.turbo.computer/)
- [Account Twitter](https://twitter.com/makegamesfast)
- [Maintainer](https://twitter.com/jozanza)

# Honeycomb Protocol

Honeycomb fornisce una suite di programmi onchain e strumenti di compressione
dello stato che gestiscono tutte le funzioni cruciali del ciclo di vita del
gioco e la componibilità all'interno della Solana Virtual Machine (SVM).
Supporta molte funzionalità NFT come lo staking, le missioni, i loot box, i
profili dei giocatori, la compressione dello stato, l'approvazione automatica
delle transazioni e altro ancora.

- [Account Twitter](https://twitter.com/honeycomb_prtcl)
- [Documentazione](https://docs.honeycombprotocol.com/)

# SDK per Unreal

Unreal Engine è noto per la sua eccellente grafica e il framework di scripting
basato su nodi. Esistono diversi SDK Solana mantenuti da diverse aziende.

### SDK Unreal di Varmeta

VAR META fornisce un SDK Unreal open-source per integrare le funzionalità della
blockchain Solana nei progetti Unreal Engine. Permette agli sviluppatori di
interagire con gli smart contract di Solana, gestire wallet ed eseguire
transazioni direttamente all'interno di Unreal Engine.

- [SDK Unreal di Varmeta](https://github.com/VAR-META-Tech/UnrealSolanaSDK)
- [Maintainer](https://www.var-meta.com/)

### SDK Unreal di Bitfrost

Bitfrost sta lavorando a un SDK Unreal che è già stato utilizzato in diversi
giochi sviluppati durante recenti game jam di Solana. Include supporto C#
`solnet` in C++ e Blueprints, conio di NFT metaplex, esempi di processori di
pagamento nel wallet di gioco e altro ancora.

- [SDK Unreal Solana di Bitfrost](https://github.com/Bifrost-Technologies/Solana-Unreal-SDK)
- [Tutorial](https://www.youtube.com/watch?v=S8fm8mFeUkk)
- [Maintainer](https://twitter.com/BifrostTitan)

### SDK Unreal di Thugz

Thugz è uno studio tecnologico e un progetto NFT che mantiene anche in open
source un SDK Unreal per Solana. Include molte funzionalità incentrate sugli NFT
e il plugin è compatibile anche con iOS e Android.

- [Plugin Thugz Blockchain](https://www.fab.com/listings/a7e220ba-93e5-4a3b-99e7-17b13da76f05)
- [Repository](https://github.com/ThugzLabs/Thugz-BC-Plugin-Packaged-for-UE5.4)
- [Documentazione](https://thugzblockchainplugin-docs.vercel.app/)
- [Tutorial video](https://www.youtube.com/watch?v=dS7sTZd_E9U&ab_channel=ThugzNFT)

### Star Atlas Foundation Kit

Il team di [Star Atlas](https://staratlas.com/) ha reso open source parte del
loro stack SDK chiamato _Foundation Kit_. Non è mantenuto attivamente ma può
essere un buon punto di partenza per il tuo progetto.

- [Star Atlas Unreal SDK](https://github.com/staratlasmeta/FoundationKit)
- [Tutorial](https://www.youtube.com/watch?v=S8fm8mFeUkk)

## Next.js/React + Anchor

Uno dei modi più semplici per sviluppare su Solana è utilizzare il framework
JavaScript Web3js in combinazione con il framework Solana Anchor. Per giochi più
complessi, consigliamo di considerare l'utilizzo di un Game Engine come
[Unity](#unity-sdk) o [Unreal](#unreal-sdks).

Il modo più veloce per configurare un gioco basato su Next.js:

```shell
npx create-solana-game your-game-name
```

Questo genererà un'ottima applicazione di partenza con supporto
`wallet-adapter`, un programma Anchor, un'app React e un client Unity già
configurati per funzionare insieme. Un vantaggio dell'utilizzo di Next.js è che
puoi usare lo stesso codice sia nel backend che nel frontend, accelerando lo
sviluppo.

- [Web3.js](https://solana.com/docs/clients/javascript)
- [Tutorial video](https://www.youtube.com/watch?v=fnhivg_pemI&t=1s&ab_channel=Solana)

> Se stai lavorando su un SDK Solana e vuoi aggiungere il tuo preset al preset
> puoi aprire una PR qui:
> [Solana games preset](https://github.com/solana-developers/solana-game-preset)

## Phaser

Un framework open source veloce, gratuito e divertente per giochi basati su
browser con Canvas e WebGL. [Phaser](https://phaser.io/) è un ottimo modo per
iniziare con lo sviluppo di giochi su Solana. Il template Phaser Solana
Platformer è un eccellente punto di partenza per il tuo gioco.

- [Codice sorgente](https://github.com/Bread-Heads-NFT/phaser-solana-platformer-template)
- [Maintainer](https://twitter.com/blockiosaurus)

## Flutter

Flutter è un framework open source, mantenuto da Google, per creare applicazioni
multi-piattaforma belle e compilate nativamente da un'unica base di codice.
L'SDK Solana Flutter è mantenuto dal team di
[Espresso Cash](https://www.espressocash.com/).

- [Codice Sorgente](https://github.com/espresso-cash/espresso-cash-public)

## Python

Python è un linguaggio di programmazione facile da imparare e ampiamente
utilizzato (spesso usato nelle applicazioni di machine learning). Il framework
Seahorse permette agli sviluppatori di scrivere programmi Solana in Python.
Seahorse è costruito sulla base del framework Anchor e converte il codice Python
in codice rust basato su Anchor. Seahorse è attualmente in versione beta.

- [Documentazione](https://www.seahorse.dev/)
- [Repository Github](https://github.com/solana-developers/seahorse)
- [Esempio Anchor Playground](https://beta.solpg.io/tutorials/hello-seahorse)
