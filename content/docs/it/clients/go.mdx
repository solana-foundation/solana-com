Esiste un [SDK Go](https://github.com/gagliardetto/solana-go) che supporta la
maggior parte delle funzioni per interagire con Solana. Se stai utilizzando
Anchor per i tuoi programmi Solana, potrai utilizzare
[anchor-go](https://github.com/gagliardetto/anchor-go) con il tuo idl per creare
un client compatibile con Golang.

## Installazione

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```
