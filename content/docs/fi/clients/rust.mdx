Solanan Rust-paketit on
[julkaistu crates.io-palvelussa](https://crates.io/search?q=solana-) ja ne
löytyvät [docs.rs-sivustolta](https://docs.rs/releases/search?query=solana-)
etuliitteellä `solana-` .

## Rust-paketit

Seuraavat ovat tärkeimmät ja yleisimmin käytetyt Rust-paketit
Solana-kehityksessä:

- [`solana-program`] &mdash; Tuodaan Solanassa suoritettaviin ohjelmiin,
  käännetään SBF:ksi. Tämä paketti sisältää monia perustavanlaatuisia
  datatyyppejä ja se uudelleenviedään [`solana-sdk`]-paketista, jota ei voida
  tuoda Solana-ohjelmasta.

- [`solana-sdk`] &mdash; Perus offchain SDK, se uudelleenvienti
  [`solana-program`]-paketin ja lisää sen päälle enemmän API:ja. Useimmat
  Solana-ohjelmat, jotka eivät suorita lohkoketjussa, tuovat tämän.

- [`solana-client`] &mdash; Solana-solmun kanssa vuorovaikuttamiseen
  [JSON RPC API](/docs/rpc):n kautta.

- [`solana-cli-config`] &mdash; Solana CLI -konfiguraatiotiedoston lataaminen ja
  tallentaminen.

- [`solana-clap-utils`] &mdash; Rutiinit CLI:n määrittämiseen käyttäen
  [`clap`]-pakettia, kuten pääasiallisessa Solana CLI:ssä. Sisältää funktioita
  kaikkien CLI:n tukemien allekirjoittajatyyppien lataamiseen.

## Asennus

Transaktioiden lähettämiseen ja Solanan kanssa vuorovaikuttamiseen:

```shell title="Terminal"
cargo add solana-sdk solana-client
```

Solana-ohjelmien rakentamiseen asenna:

```shell title="Terminal"
cargo add solana-program
```
