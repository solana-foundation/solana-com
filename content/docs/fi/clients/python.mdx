On olemassa [Python SDK](https://github.com/michaelhly/solana-py), joka tukee
useimpia toimintoja Solanan kanssa vuorovaikuttamiseen. Jos käytät Anchoria
Solana- ohjelmissasi, voit käyttää
[anchor-py](https://github.com/kevinheavey/anchorpy) -kirjastoa idl:n kanssa
luodaksesi Python-yhteensopivan asiakassovelluksen.

## Asennus

```bash title="Terminal"
pip install solders
pip install solana
```
