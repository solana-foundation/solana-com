## Kuinka käyttää Skaalatun käyttöliittymäsumman laajennusta

Käyttääksesi Skaalatun käyttöliittymäsumman laajennusta, sinun täytyy ottaa se
käyttöön token mintissä tai token accountissa.

### Ota Skaalatun käyttöliittymäsumman laajennus käyttöön token mintissä

Ottaaksesi Skaalatun käyttöliittymäsumman laajennuksen käyttöön token mintissä,
sinun täytyy asettaa `scaled_ui_amount_extension` -kenttä arvoon `true` `Mint`
-tilissä. Tässä on esimerkki siitä, kuinka luoda token, jossa Skaalatun
käyttöliittymäsumman laajennus on käytössä käyttäen `spl-token` CLI:tä:

```terminal title="Create Token"
$ spl-token --program-id TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb create-token --ui-amount-multiplier 1.5
Creating token 66EV4CaihdqyQ1fbsr51wBsoqKLgAG5KiYz7r5XNrxUM under program TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb

Address:  66EV4CaihdqyQ1fbsr51wBsoqKLgAG5KiYz7r5XNrxUM
Decimals:  9

Signature: 2sPziXu9M3duTCvsDvxQE9UKC9nBiLayi8muDvnjhA2qYvfXSZuaUieoq39MFjg4kf8xFrw6crmYSkPyV59dvudF
```

### Päivitä käyttöliittymäsumman kerroin

Päivittääksesi käyttöliittymäsumman kertoimen, sinun täytyy käyttää
`update-ui-amount-multiplier` -komentoa. Aikaleima on valinnainen ja sitä
voidaan käyttää asettamaan mukautettu aloitusaika uudelle kertoimelle. Jos
aikaleimaa ei anneta, käytetään nykyistä aikaleimaa.

```terminal title="Update UI Amount Multiplier"
$ spl-token update-ui-amount-multiplier 66EV4CaihdqyQ1fbsr51wBsoqKLgAG5KiYz7r5XNrxUM 1.5 -- 1746471400
```

### Hae saldo

Hakeaksesi saldon, sinun täytyy käyttää `balance` -komentoa.

```terminal title="Fetch Balance"
$ spl-token balance 66EV4CaihdqyQ1fbsr51wBsoqKLgAG5KiYz7r5XNrxUM
```
