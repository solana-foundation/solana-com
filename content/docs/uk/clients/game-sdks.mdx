Оскільки Solana є високопродуктивним блокчейном з низькими комісіями та швидкими
транзакціями, навколо нього сформувалася чудова спільнота розробників ігор.

Тут ви можете знайти інформацію про SDK для розробки ігор на Solana, доступні в
екосистемі. Усі SDK є з відкритим кодом і створені спільнотою. Якщо ви працюєте
над власним SDK, не соромтеся відкрити PR до цієї сторінки та додати його.

## Unity SDK

Ігровий рушій Unity відомий своїм дружнім до початківців підходом та підтримкою
різних платформ, включаючи WebGL, iOS та Android. Створюйте один раз,
експортуйте всюди.

- [Документація](https://docs.magicblock.gg/introduction)
- [Вихідний код](https://github.com/magicblock-labs/Solana.Unity-SDK)
- [Приклади ігор](https://github.com/solana-developers/solana-game-examples)
- [Хрестики-нулики](https://blog.magicblock.gg/bolt-tic-tac-toe/)

SDK Solana для Unity підтримується компанією
[Magicblock](https://www.magicblock.gg/) і включає:

- Підтримку NFT
- Інтеграцію з Candy machine
- Транзакції
- Функції RPC
- Глибокі посилання Phantom
- Конектор WebGL
- Надійну підтримку WebSocket з'єднань
- Адаптер для мобільних гаманців
- Сесійні ключі для автоматичного підтвердження транзакцій
- Генерацію клієнтського коду Anchor та багато іншого.

## Godot SDK

[Godot Engine](https://godotengine.org/) — це ігровий рушій з відкритим кодом,
який отримав значну підтримку за останні кілька років. SDK Solana для Godot
підтримується [ZenRepublic](https://twitter.com/ZenRepublicNDM) та
[Virus-Axel](https://twitter.com/AxelBenjam). Він включає інтеграцію з адаптером
гаманця, транзакції, функції RPC та генерацію клієнтського коду Anchor. Навіть
повністю функціональну інтеграцію Metaplex
[candy machine](https://zenwiki.gitbook.io/solana-godot-sdk-docs/guides/setup-candy-machine).

- [SDK Solana для Godot Engine](https://github.com/Virus-Axel/godot-solana-sdk)
- [Документація](https://zenwiki.gitbook.io/solana-godot-sdk-docs)
- [Демо](https://github.com/ZenRepublic/GodotSolanaSDKDemoPackage)
- [Туторіал](https://www.youtube.com/watch?v=tszFPInYmXQ)
- [Розробник](https://twitter.com/ZenRepublicNDM)

## Turbo.Computer - ігровий рушій на Rust

Turbo — це ігровий рушій на Rust, створений з нуля з фокусом на Solana, з легкою
архітектурою та швидким циклом розробки. Він дружній для початківців і
постачається з повною підтримкою Solana RPC. Ви навіть можете використовувати
його інструменти штучного інтелекту для генерації повноцінних ігор.

- [Вебсайт](https://turbo.computer/)
- [Документація](https://docs.turbo.computer/)
- [Twitter акаунт](https://twitter.com/makegamesfast)
- [Розробник](https://twitter.com/jozanza)

# Honeycomb Protocol

Honeycomb надає набір онлайн-програм та інструментів стиснення стану, які
обробляють усі важливі функції життєвого циклу гри та забезпечують
композиційність у Solana Virtual Machine (SVM). Він підтримує багато функцій
NFT, таких як стейкінг, місії, лутбокси, профілі гравців, стиснення стану,
автоматичне підтвердження транзакцій тощо.

- [Twitter акаунт](https://twitter.com/honeycomb_prtcl)
- [Документація](https://docs.honeycombprotocol.com/)

# Unreal SDK

Unreal Engine відомий своєю чудовою графікою та системою візуального
програмування на основі вузлів. Існує кілька SDK для Solana, які підтримуються
різними компаніями.

### Varmeta Unreal SDK

VAR META надає SDK з відкритим кодом для Unreal, що дозволяє інтегрувати
функціональність блокчейну Solana в проєкти Unreal Engine. Він дозволяє
розробникам взаємодіяти з смарт-контрактами Solana, керувати гаманцями та
виконувати транзакції безпосередньо в Unreal Engine.

- [Unreal SDK від Varmeta](https://github.com/VAR-META-Tech/UnrealSolanaSDK)
- [Розробник](https://www.var-meta.com/)

### Bitfrost Unreal SDK

Bitfrost працює над SDK для Unreal, який вже використовувався в кількох іграх,
створених під час нещодавніх геймджемів Solana. Він постачається з підтримкою C#
`solnet` в C++ та Blueprints, карбуванням NFT на базі Metaplex, прикладами
процесорів платежів, ігровим гаманцем та іншим.

- [Solana Unreal SDK від Bitfrost](https://github.com/Bifrost-Technologies/Solana-Unreal-SDK)
- [Туторіал](https://www.youtube.com/watch?v=S8fm8mFeUkk)
- [Розробник](https://twitter.com/BifrostTitan)

### Thugz Unreal SDK

Thugz — це технологічна студія та NFT проєкт, який також підтримує SDK з
відкритим кодом для Unreal на Solana. Він містить багато функцій, орієнтованих
на NFT, і плагін сумісний також з iOS та Android.

- [Плагін Thugz Blockchain](https://www.fab.com/listings/a7e220ba-93e5-4a3b-99e7-17b13da76f05)
- [Репозиторій](https://github.com/ThugzLabs/Thugz-BC-Plugin-Packaged-for-UE5.4)
- [Документація](https://thugzblockchainplugin-docs.vercel.app/)
- [Відеоінструкція](https://www.youtube.com/watch?v=dS7sTZd_E9U&ab_channel=ThugzNFT)

### Star Atlas Foundation Kit

Команда [Star Atlas](https://staratlas.com/) відкрила доступ до частини свого
SDK стеку під назвою _Foundation Kit_. Він не підтримується активно, але може
стати відправною точкою для вашого проєкту.

- [Star Atlas Unreal SDK](https://github.com/staratlasmeta/FoundationKit)
- [Інструкція](https://www.youtube.com/watch?v=S8fm8mFeUkk)

## Next.js/React + Anchor

Один із найпростіших способів розробки на Solana — це використання JavaScript
фреймворку Web3js у поєднанні з фреймворком Solana Anchor. Однак для складніших
ігор ми рекомендуємо розглянути використання ігрового рушія, як-от
[Unity](#unity-sdk) або [Unreal](#unreal-sdks).

Найшвидший спосіб налаштувати гру на основі Next.js:

```shell
npx create-solana-game your-game-name
```

Це згенерує чудовий стартовий додаток з підтримкою `wallet-adapter`, програмою
Anchor, React-додатком та Unity-клієнтом, які вже налаштовані для спільної
роботи. Перевага використання Next.js полягає в тому, що ви можете
використовувати один і той самий код як на бекенді, так і на фронтенді,
прискорюючи розробку.

- [Web3.js](https://solana.com/docs/clients/javascript)
- [Відеоінструкція](https://www.youtube.com/watch?v=fnhivg_pemI&t=1s&ab_channel=Solana)

> Якщо ви працюєте над SDK для Solana і хочете додати свій пресет до наявних, ви
> можете відкрити PR тут:
> [Пресет для ігор Solana](https://github.com/solana-developers/solana-game-preset)

## Phaser

Швидкий, безкоштовний і захопливий фреймворк з відкритим кодом для створення
ігор на основі Canvas та WebGL для браузерів. [Phaser](https://phaser.io/) —
чудовий спосіб почати розробку ігор на Solana. Шаблон Phaser Solana Platformer є
відмінною відправною точкою для вашої гри.

- [Вихідний код](https://github.com/Bread-Heads-NFT/phaser-solana-platformer-template)
- [Розробник](https://twitter.com/blockiosaurus)

## Flutter

Flutter — це фреймворк з відкритим кодом, який підтримується Google, для
створення красивих, нативно скомпільованих, мультиплатформних додатків з єдиної
кодової бази. SDK Solana для Flutter підтримується командою
[Espresso Cash](https://www.espressocash.com/).

- [Вихідний код](https://github.com/espresso-cash/espresso-cash-public)

## Python

Python — це легка для вивчення та широко використовувана мова програмування
(часто застосовується в додатках машинного навчання). Фреймворк Seahorse
дозволяє розробникам писати програми Solana мовою Python. Seahorse побудований
на основі фреймворку Anchor і перетворює код Python у код rust на основі Anchor.
Seahorse наразі перебуває в бета-версії.

- [Документація](https://www.seahorse.dev/)
- [Репозиторій Github](https://github.com/solana-developers/seahorse)
- [Приклад Anchor Playground](https://beta.solpg.io/tutorials/hello-seahorse)

## Нативний C#

Оригінальний порт Web3js на C#. Він містить багато функціональних можливостей,
таких як транзакції, функції RPC та генерацію клієнтського коду Anchor.

- [Вихідний код і документація](https://github.com/bmresearch/Solnet/blob/master/docs/articles/getting_started.md)
