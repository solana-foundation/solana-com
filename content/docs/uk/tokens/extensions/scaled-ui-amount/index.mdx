## Що таке розширення Scaled UI Amount?

Розширення масштабованої суми інтерфейсу дозволяє емітентам застосовувати
оновлюваний множник до суми токена, що відображається в інтерфейсі. Це корисно
для сценаріїв, коли токен використовується для представлення реального активу,
наприклад, акції або дивіденду.

Наприклад, спліт акцій подвоює кількість акцій для всіх власників. Без цього
розширення наразі непрактично випускати нові токени всім власникам під час події
спліту ефективним і масштабованим способом.

Однак з моделлю розширення Token-2022 ми маємо можливість змінити спосіб
відображення сум токенів в інтерфейсі. Використовуючи розширення ScaledUiAmount
та інструкцію `amount_to_ui_amount`, ви можете встановити множник інтерфейсу для
вашого токена та отримувати його суму в інтерфейсі в будь-який час.
[вихідний код](https://github.com/solana-program/token-2022/tree/main/program/src/extension/scaled_ui_amount).

Ця функція також може використовуватися для дивідендів або розподілу прибутку.

Примітка: Нові токени ніколи не створюються, сума в інтерфейсі повертає сиру
кількість токенів, помножену на поточний множник. Ця функція є повністю
косметичною.

<Callout type="info">
  Деякі розширення несумісні між собою, і ви не можете активувати їх одночасно
  для одного й того ж випуску токена або token account. Не слід активувати
  одночасно розширення `ScaledUiAmount` та розширення `InterestBearingMint` для
  одного й того ж випуску токена або token account.
</Callout>
