h1: RPC-метод getTransactionCount
---

Повертає поточну кількість транзакцій з реєстру

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getTransactionCount",
  "params": [
    // !hover(1:3) 0
    {
      // !hover commitment
      "commitment": "finalized"
    }
  ]
}
```

```ts !!request title="Kit"
import { createSolanaRpc } from "@solana/kit";

const rpc_url = "https://api.devnet.solana.com";
const rpc = createSolanaRpc(rpc_url);

let txCount = await rpc.getTransactionCount().send();

console.log(txCount);
```

```ts !!request title="web3.js"
import { Connection, clusterApiUrl } from "@solana/web3.js";

const connection = new Connection(clusterApiUrl("devnet"), "confirmed");

let txCount = await connection.getTransactionCount();

console.log(txCount);
```

```rs !!request title="Rust"
use anyhow::Result;
use solana_client::nonblocking::rpc_client::RpcClient;
use solana_sdk::commitment_config::CommitmentConfig;

#[tokio::main]
async fn main() -> Result<()> {
    let client = RpcClient::new_with_commitment(
        String::from("https://api.devnet.solana.com"),
        CommitmentConfig::confirmed(),
    );

    let tx_count = client.get_transaction_count().await?;

    println!("{:#?}", tx_count);

    Ok(())
}
```

### !params

#### !! 0

!type object !optional

Об'єкт конфігурації, що містить такі поля:

##### !! commitment

!type string

Параметр commitment описує, наскільки фіналізованим є блок у певний момент часу.
Дивіться
[Налаштування підтвердження стану](/docs/rpc#configuring-state-commitment).

##### !! minContextSlot

!type number

Мінімальний slot, на якому може бути оцінений запит

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 268,
  "id": 1
}
```

!type u64

Поточна кількість транзакцій з реєстру

</APIMethod>
