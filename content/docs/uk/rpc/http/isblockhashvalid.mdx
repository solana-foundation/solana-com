h1: RPC метод isBlockhashValid
---

Повертає інформацію про те, чи є blockhash все ще дійсним чи ні

<Callout type="warn" title={"Обмеження версії"}>
  Цей метод доступний лише в `solana-core` v1.9 або новіших версіях. Будь ласка,
  використовуйте
  [getFeeCalculatorForBlockhash](/docs/rpc/http/getfeecalculatorforblockhash)
  для `solana-core` v1.8 та нижче.
</Callout>

```ts !!request title="Kit"
import { createSolanaRpc, type Blockhash } from "@solana/kit";

const rpc_url = "https://api.devnet.solana.com";
const rpc = createSolanaRpc(rpc_url);

// !hover blockhash
let blockhash = "J7rBdM6AecPDEZp8aPq5iPSNKVkU5Q76F3oAV4eW5wsW";

let isValid = await rpc.isBlockhashValid(blockhash as Blockhash).send();

console.log(isValid);
```

```rs !!request title="Rust"
use std::str::FromStr;

use anyhow::Result;
use solana_client::nonblocking::rpc_client::RpcClient;
use solana_sdk::{commitment_config::CommitmentConfig, hash::Hash};

#[tokio::main]
async fn main() -> Result<()> {
    let client = RpcClient::new_with_commitment(
        String::from("https://api.devnet.solana.com"),
        CommitmentConfig::confirmed(),
    );

    // !hover blockhash
    let blockhash = Hash::from_str("J7rBdM6AecPDEZp8aPq5iPSNKVkU5Q76F3oAV4eW5wsW")?;

    let is_valid = client
        .is_blockhash_valid(&blockhash, CommitmentConfig::finalized())
        .await?;

    println!("{:#?}", is_valid);

    Ok(())
}
```

### !params

#### !! blockhash

!type string
!required

Blockhash блоку для оцінки, як рядок у кодуванні base-58

#### !! config

!type object

Об'єкт конфігурації, що містить такі поля:

##### !! commitment

!type string

Параметр commitment описує, наскільки фіналізованим є блок у певний момент часу.
Дивіться
[Налаштування підтвердження стану](/docs/rpc#configuring-state-commitment).

##### !! minContextSlot

!type number

Мінімальний slot, на якому може бути оцінений запит
