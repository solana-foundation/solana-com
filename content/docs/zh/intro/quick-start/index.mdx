欢迎来到Solana快速入门指南。这个实践指南将向你介绍Solana构建的核心概念，无论你之前有什么经验。

## 你将学到什么

在本教程中，你将了解：

* **Solana账户**：了解Solana网络如何存储数据。
* **发送交易**：学习通过发送交易与Solana网络交互。
* **构建和部署程序**：创建你的第一个Solana程序并将其部署到网络。
* **程序派生地址（PDAs）**：学习如何使用PDAs为账户创建确定性地址。
* **跨程序调用（CPIs）**：学习如何从你的程序内调用其他程序，实现Solana上不同程序之间的复杂交互和组合性。

最棒的是什么？你不需要安装任何东西。本指南使用Solana
Playground，一个基于浏览器的开发环境。跟随指南，复制和粘贴代码，并立即看到结果，全部通过你的网页浏览器完成。基本的编程知识有帮助但不是必需的。

现在是时候深入并开始在Solana上构建了。

## Solana Playground

Solana
Playground（Solpg）提供了一个基于浏览器的开发环境，让你能够快速开发、部署和测试Solana程序。

在你的网页浏览器中打开一个新标签页并导航到[https://beta.solpg.io/](https://beta.solpg.io/)。

<Steps>
  <Step>
    ### 创建Playground钱包

    新的Solana Playground用户应该首先创建一个Playground钱包。这个钱包使你能够直接从浏览器与Solana网络交互。

    #### 步骤 1. 连接到 Playground

    点击屏幕左下角的"未连接"按钮。

    ![Not Connected](/assets/docs/intro/quickstart/pg-not-connected.png)

    #### 步骤 2. 创建您的钱包

    系统会提示您保存钱包的密钥对。准备好后，点击"继续"进行下一步。

    ![Create Playground Wallet](/assets/docs/intro/quickstart/pg-create-wallet.png)

    现在您应该可以在窗口底部看到您的钱包地址、SOL 余额和已连接的集群（默认为 devnet）。

    ![Connected](/assets/docs/intro/quickstart/pg-connected.png)

    <Callout>
      您的 Playground 钱包保存在浏览器的本地存储中。清除浏览器缓存将删除您保存的钱包。Playground 钱包仅用于测试和开发。请勿向此地址发送真实资产（来自主网）。
    </Callout>

    以下是一些可能对您有帮助的定义：

    * *钱包地址*：来自 Ed25519 密钥对的 32 字节公钥，通常显示为 base-58 编码的字符串（例如，`7MNj7pL1y7XpPnN7ZeuaE4ctwg3WeufbX5o85sA91J1`）。相应的私钥用于签署来自此地址的交易。在 Solana 上，地址作为用户钱包、程序（智能合约）或网络上任何其他账户的唯一标识符。
    * *已连接集群*：您当前交互的 Solana 网络。常见的集群包括：
      * `devnet`：供开发者实验的开发网络
      * `testnet`：为验证者测试保留的网络（应用开发者请勿使用）
      * `mainnet-beta`：用于实时交易的 Solana 主网络
  </Step>

  <Step>
    ### 获取 Devnet SOL

    在开始开发之前，您需要获取一些 devnet SOL。

    作为开发者，您需要 SOL 主要用于两种情况：

    * 创建新账户以在网络上存储数据或部署程序
    * 与 Solana 网络交互时支付交易费用

    为您的钱包充值 devnet SOL 的两种方法：

    #### 选项1：使用Playground终端

    为您的Playground钱包充值开发网SOL。在Playground终端中，运行：

    ```terminal
    $ solana airdrop 5
    ```

    #### 选项2：使用开发网水龙头

    如果空投命令不起作用（由于速率限制或错误），您可以使用
    [网页水龙头](https://faucet.solana.com/)。

    * 输入您的钱包地址（在Playground屏幕底部可找到）并
      选择金额
    * 点击"确认空投"以接收您的开发网SOL

    ![Faucet Airdrop](/assets/docs/intro/quickstart/faucet-airdrop.gif)
  </Step>
</Steps>
