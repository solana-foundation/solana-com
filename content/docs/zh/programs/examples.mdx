GitHub上的
[Solana程序示例](https://github.com/solana-developers/program-examples)
仓库提供了多个子文件夹，每个文件夹包含不同Solana编程范式和语言的代码示例，旨在帮助开发者学习和体验Solana区块链开发。

你可以在`solana-developers/program-examples`中找到这些示例，以及解释如何运行不同示例的README文件。大多数示例都是独立的，并且有原生Rust（即不使用框架）和[Anchor](https://www.anchor-lang.com/docs/installation)版本。它还包含了一系列我们希望看到的[贡献示例](https://github.com/solana-developers/program-examples?tab=readme-ov-file#examples-wed-love-to-see)。

在仓库中，你会找到以下子文件夹，每个文件夹中都有各种示例程序：

* [基础知识](#basics)
* [压缩](#compression)
* [预言机](#oracles)
* [代币](#tokens)
* [Token 2022（代币扩展）](#token-2022-token-extensions)
* [Break](#break)
  * [构建和运行](#build-and-run)

## Token 2022（代币扩展）

Token
2022是Solana上代币的新标准。它更加灵活，允许您为代币铸造添加16种不同的扩展，以增加更多功能。完整的扩展列表可以在[入门指南](/developers/guides/token-extensions/getting-started)

| 示例名称                                                                                                                     | 描述                                  | 语言     |
| ------------------------------------------------------------------------------------------------------------------------ | ----------------------------------- | ------ |
| [基础知识](https://github.com/solana-developers/program-examples/tree/main/tokens/token-2022/basics/anchor)                  | 如何创建代币、铸造和转移它。                      | Anchor |
| [默认账户状态](https://github.com/solana-developers/program-examples/tree/main/tokens/token-2022/default-account-state/native) | 此扩展允许您创建具有特定状态的代币账户，例如冻结状态。         | 原生     |
| [铸币关闭权限](https://github.com/solana-developers/program-examples/tree/main/tokens/token-2022/mint-close-authority)         | 在旧版代币程序中，无法关闭铸币。现在可以了。              | 原生     |
| [多重扩展](https://github.com/solana-developers/program-examples/tree/main/tokens/token-2022/multiple-extensions)            | 向您展示如何为单个铸币添加多个扩展                   | 原生     |
| [NFT 元数据指针](https://github.com/solana-developers/program-examples/tree/main/tokens/token-2022/nft-meta-data-pointer)     | 可以使用元数据扩展创建 NFT 并添加动态链上元数据。         | Anchor |
| [不可转让](https://github.com/solana-developers/program-examples/tree/main/tokens/token-2022/non-transferable/native)        | 对于成就、推荐计划或任何灵魂绑定代币等场景非常有用。          | 原生     |
| [转账费用](https://github.com/solana-developers/program-examples/tree/main/tokens/token-2022/transfer-fees)                  | 每次代币转账都会在代币账户中保留一部分代币，之后可以被收集。      | 原生     |
| [转账钩子](https://github.com/solana-developers/program-examples/tree/main/tokens/token-2022/transfer-hook)                  | 四个示例，通过从代币程序到您的程序的 CPI 为您的代币添加额外功能。 | Anchor |

## Break

[Break](https://break.solana.com/)
是一个 React 应用，让用户直观感受 Solana 网络的高速和高性能。您能否*break*
Solana区块链？在15秒的游戏过程中，每次点击按钮或按键都会向集群发送一个新的交易。尽可能快地敲击键盘，观察您的交易在网络稳定运行的同时实时完成确认！

Break可以在我们的Devnet、Testnet和Mainnet
Beta网络上玩。在Devnet和Testnet上免费玩，游戏会由网络水龙头提供资金。在Mainnet
Beta上，用户需要支付每局0.08
SOL。会话账户可以通过本地密钥库钱包充值，或者通过扫描Trust
Wallet的二维码转账代币。

[点击这里玩Break](https://break.solana.com/)
