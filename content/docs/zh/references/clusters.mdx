h1: 集群和公共 RPC 端点
---

Solana 区块链有几个不同的验证者组，称为集群。每个集群在整个生态系统中服务于不同的目的，并包含专用的 API 节点来满足[JSON-RPC](/docs/rpc/)请求为各自的集群提供服务。

集群内的各个节点由第三方拥有和运营，每个集群都有一个可用的公共端点。

## Solana 公共 RPC 端点

Solana
Labs 组织为每个集群运营一个公共 RPC 端点。这些公共端点都受到速率限制，但可供用户和开发者与 Solana 区块链进行交互。

> 公共端点的速率限制可能会发生变化。本文档中列出的具体速率限制不保证是最新的。

### 在不同集群上使用区块浏览器

许多流行的 Solana 区块链浏览器支持选择任何集群，通常也允许高级用户添加自定义/私有 RPC 端点。

Solana 区块链浏览器的一些例子包括：

* [http://explorer.solana.com/](https://explorer.solana.com/)。
* [http://solana.fm/](https://solana.fm/)。
* [http://solscan.io/](https://solscan.io/)。
* [http://solanabeach.io/](http://solanabeach.io/)。
* [http://validators.app/](http://validators.app/)。

## 概述

* 主网：已部署应用程序的实时生产环境。
* 开发网：为开发者提供公共可访问性的测试环境，用于试验他们的应用程序。
* 测试网：用于网络升级和验证者性能的压力测试。

**使用场景示例**：您可能想在开发网上调试新程序或在测试网上验证性能指标，然后再部署到主网。

| **集群** | **端点**                                | **用途**    | **备注**      |
| ------ | ------------------------------------- | --------- | ----------- |
| 主网     | `https://api.mainnet-beta.solana.com` | 实时生产环境    | 交易需要SOL     |
| 开发网    | `https://api.devnet.solana.com`       | 公共测试和开发   | 免费SOL空投用于测试 |
| 测试网    | `https://api.testnet.solana.com`      | 验证节点和压力测试 | 可能会有间歇性停机   |

## 开发网

开发网是任何想要试用Solana的人的游乐场，无论是用户、代币持有者、应用开发者还是验证节点。

* 应用开发者应该以开发网为目标。
* 潜在的验证节点应该首先以开发网为目标。
* 开发网和主网Beta之间的主要区别：
  * 开发网代币**不是真实的**
  * 开发网包含用于应用测试的代币水龙头空投
  * 开发网可能会进行账本重置
  * 开发网通常运行与主网Beta相同的软件发布分支版本，但可能运行比主网Beta更新的次要版本。
* 开发网的Gossip入口点：`entrypoint.devnet.solana.com:8001`

### Devnet 端点

* `https://api.devnet.solana.com` - 单个 Solana Labs 托管的 API 节点；有速率限制

#### 示例 `solana` 命令行配置

要使用 Solana CLI 连接到 `devnet` 集群：

```shell
solana config set --url https://api.devnet.solana.com
```

### Devnet 速率限制

* 每个 IP 每 10 秒最大请求数：100
* 每个 IP 每 10 秒对单个 RPC 的最大请求数：40
* 每个 IP 的最大并发连接数：40
* 每个 IP 每 10 秒的最大连接速率：40
* 每 30 秒的最大数据量：100 MB

## Testnet

Testnet 是 Solana 核心贡献者在实时集群上压力测试最新发布功能的地方，特别关注网络性能、稳定性和验证者行为。

* Testnet 代币 **不是真实的**
* Testnet 可能会进行账本重置。
* Testnet 包含用于应用程序测试的代币水龙头空投
* Testnet 通常运行比 Devnet 和 Mainnet Beta 更新的软件发布分支
* Testnet 的 Gossip 入口点：`entrypoint.testnet.solana.com:8001`

### Testnet 端点

* `https://api.testnet.solana.com` - 单个Solana Labs API节点；有速率限制

#### 示例 `solana` 命令行配置

使用Solana CLI连接到 `testnet` 集群：

```shell
solana config set --url https://api.testnet.solana.com
```

### 测试网速率限制

* 每10秒每IP最大请求数：100
* 每10秒每IP单个RPC最大请求数：40
* 每IP最大并发连接数：40
* 每10秒每IP最大连接速率：40
* 每30秒最大数据量：100 MB

## 主网测试版

为Solana用户、开发者、验证者和代币持有者提供的无需许可的永久集群。

* 在主网测试版上发行的代币是 **真实的** SOL
* 主网测试版的Gossip入口点： `entrypoint.mainnet-beta.solana.com:8001`

### 主网测试版端点

* `https://api.mainnet-beta.solana.com` - Solana
  Labs托管的API节点集群，由负载均衡器支持；有速率限制

#### 示例 `solana` 命令行配置

要使用Solana CLI连接到`mainnet-beta`集群：

```shell
solana config set --url https://api.mainnet-beta.solana.com
```

### 主网测试版速率限制

* 每个IP每10秒最大请求数：100
* 每个IP每10秒单个RPC最大请求数：40
* 每个IP最大并发连接数：40
* 每个IP每10秒最大连接速率：40
* 每30秒最大数据量：100 MB

> 公共RPC端点不适用于生产应用程序。请在启动应用程序、发布NFT等时使用专用/私有RPC服务器。公共服务可能被滥用，且速率限制可能在未事先通知的情况下更改。同样，高流量网站可能在未事先通知的情况下被阻止。

## 常见HTTP错误代码

* 403 -- 您的IP地址或网站已被阻止。是时候运行您自己的RPC服务器或寻找私有服务了。
* 429
  \-- 您的IP地址超出了速率限制。请放慢速度！使用[Retry-After](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Retry-After)HTTP响应头来确定在发出另一个请求前需要等待多长时间。
