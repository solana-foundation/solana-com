通过质押您的 SOL 代币，您可以帮助保护网络，同时[赚取奖励](https://docs.anza.xyz/implemented-proposals/staking-rewards)。

您可以通过将代币委托给处理交易并运行网络的验证者来进行质押。

委托质押是一种风险共担、收益共享的财务模式，可能为长期委托的代币持有者提供回报。这是通过将代币持有者（委托者）与其委托的验证者的财务激励相一致来实现的。

委托给验证者的质押越多，该验证者被选中将新交易写入账本的频率就越高。验证者写入的交易越多，验证者及其委托者获得的奖励就越多。配置系统以处理更多交易的验证者将获得更多奖励，因为他们确保网络尽可能快速和顺畅地运行。

验证者通过运行和维护其系统产生成本，这些成本以从奖励中收取一定百分比的费用形式转嫁给委托者。这种费用被称为*佣金*。由于委托的质押越多，验证者获得的奖励越多，他们可能会相互竞争以提供最低的佣金服务。

尽管目前 Solana 协议尚未实施这一机制，但未来通过一种称为*削减*的过程，委托者在质押时可能面临代币损失的风险。削减是指在验证者存在恶意行为（如创建无效交易或审查某些类型的交易或网络参与者）时，移除并销毁部分验证者的 SOL。

目前协议中没有实现惩罚机制。有关惩罚的更多信息，请参阅[惩罚路线图](https://docs.anza.xyz/proposals/optimistic-confirmation-and-slashing#slashing-roadmap)。

## 我如何质押我的 SOL 代币？

您可以通过将代币转移到支持质押的钱包中来质押 SOL。钱包会提供创建质押账户和进行委托的步骤。

### 支持的钱包

许多网页和移动钱包支持 Solana 的质押操作。请向您喜欢的钱包维护者确认支持状态。

### Solana 命令行工具

- Solana 命令行工具可以结合 CLI 生成的密钥对文件钱包、纸质钱包或连接的 Ledger
  Nano 执行所有质押操作。
  [使用 Solana 命令行工具的质押命令](https://docs.anza.xyz/cli/examples/delegate-stake)。

### 创建质押账户

按照钱包的说明创建质押账户。此账户类型与用于简单发送和接收代币的账户不同。

### 选择验证者

按照钱包的说明选择验证者。您可以通过以下链接获取有关潜在高性能验证者的信息。Solana 基金会不推荐任何特定的验证者。

网站 solanabeach.io 由我们的一个验证者 Staking
Facilities 构建和维护。它提供了关于整个网络的一些高级图形信息，以及每个验证者的列表和一些最近的性能统计数据。

- https://solanabeach.io

要查看区块生产统计数据，请使用 Solana 命令行工具：

- `solana validators`
- `solana block-production`

Solana 团队不对如何解释这些信息提供建议。请自行进行尽职调查。

### 委托您的权益

按照钱包的指示，将您的权益委托给您选择的validator。
