> [这里](https://github.com/solana-foundation/mucho/issues/new?title=%5Btoolkit%5D%20)。

```shell
npx create-solana-dapp
```

此命令生成一个新项目，将 Solana 程序连接到带有钱包连接器的前端。它支持多个流行的前端技术栈和 UI 库，包括：NextJS、React、Tailwind 等。

## 构建和测试

在进行任何修改之前，按照以下步骤测试此项目：

<Steps>
<Step>

### 构建智能合约

```shell
npm run anchor-build
```

</Step>
<Step>

### 启动本地验证器

```shell
npm run anchor-localnet
```

</Step>
<Step>

### 运行测试

```shell
npm run anchor-test
```

</Step>
<Step>

### 部署到本地 validator

```shell
npm run anchor deploy --provider.cluster localnet
```

</Step>
<Step>

### 构建 Web 应用

```shell
npm run build
```

</Step>
<Step>

### 运行 Web 应用

```shell
npm run dev
```

</Step>
</Steps>
