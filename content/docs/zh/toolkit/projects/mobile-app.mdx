> [这里](https://github.com/solana-foundation/mucho/issues/new?title=%5Btoolkit%5D%20)。

```shell
yarn create expo-app --template @solana-mobile/solana-mobile-expo-template
```

## 概述

这将使用 [Expo](https://expo.dev)
框架初始化一个新项目，该框架专为创建与 Solana 区块链交互的移动应用而设计。

按照此模板的指南 "[运行应用](https://docs.solanamobile.com/react-native/expo#running-the-app)" 以自定义开发构建的方式启动模板，并在您的 Android 模拟器上运行。一旦您构建了程序并使用 Expo 运行开发客户端，每次保存代码时，模拟器都会自动更新。

## 先决条件

要使用此模板，您还需要设置以下内容：

- [Android Studio 和模拟器](https://docs.solanamobile.com/getting-started/development-setup)
- [React Native](https://reactnative.dev/docs/environment-setup?platform=android)
- [EAS CLI 和账户](https://docs.expo.dev/build/setup/)
