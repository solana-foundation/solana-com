从以下脚手架中选择一个来生成新的项目工作区：

- [Anchor](#anchor) - 一个流行的基于 Rust 的框架，用于创建 Solana 程序。
- [`create-solana-program`](#create-solana-program) - 深入的工作区生成器，可用于 Anchor 程序开发或原生程序开发，包括 JavaScript 和 Rust 客户端。
- [Web 应用模板](#web-app-template) - 用于生成新项目的生成器，将 Solana 程序连接到各种前端技术栈，并包括钱包连接器设置。

## Anchor

```shell
anchor init
```

这将生成一个基本的工作区，用于编写 Anchor
Rust 程序、构建、测试和部署。更多信息请阅读
[`anchor init` 文档](/docs/toolkit/projects/anchor-init)。

## 创建 Solana 程序

```shell
npx create-solana-program
```

这将生成一个深入的工作区，用于 Anchor 程序开发或原生程序开发，支持 JavaScript 客户端、Rust 客户端或两者兼有。更多信息请阅读
[`create-solana-program` 文档](/docs/toolkit/projects/solana-program)。

## Web 应用模板

```shell
npx create-solana-dapp
```

这将初始化一个新项目，将 Solana 程序连接到带有钱包连接器的 TypeScript 前端。更多信息请阅读
[Web 应用模板文档](/docs/toolkit/projects/web-app)。

## 移动应用模板

```shell
yarn create expo-app --template @solana-mobile/solana-mobile-expo-template
```

这是使用 Expo 框架初始化一个新项目，该框架专为创建与 Solana 区块链交互的移动应用程序而设计。

## 更新现有项目

```shell
npx create-solana-program
```

您可以通过遵循此[指南](/docs/toolkit/projects/existing-project)将 Solana 程序模板添加到现有项目中。
