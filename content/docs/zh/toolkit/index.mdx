Solana 程序开发工具包以
[`mucho` npm 包](https://www.npmjs.com/package/mucho)
的形式发布。`mucho` 命令将用于运行大多数 Solana 程序开发工具 -
_mucho 工具，一个 CLI_。

## 安装

使用以下命令安装 Solana 工具包和 `mucho`
CLI：

```shell
npx -y mucho@latest install
```

通过以下命令验证 mucho 是否已安装：

```shell
mucho --version
```

有关更多信息，请访问 [入门指南](/docs/toolkit/getting-started)。

## 创建项目

```shell
pnpm create solana-program
```

有关更多信息，请访问 [项目概览](/docs/toolkit/projects/overview)。

[源代码](https://github.com/solana-program/create-solana-program)。
