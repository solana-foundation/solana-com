> 这是 [Solana 工具包](/docs/toolkit/)
> 的测试版，目前仍在开发中（WIP）。请将所有反馈作为 GitHub 问题发布在
> [这里](https://github.com/solana-foundation/mucho/issues/new?title=%5Btoolkit%5D%20)。

在 Solana 工具包中，有多个资源可用于测试 Solana 智能合约，包括：

- 一个模糊测试工具。
- 一个代码覆盖率工具。
- 一个用于在 NodeJS 中测试 Solana 程序的框架，该框架会启动一个轻量级的 `BanksServer`，类似于 RPC 节点，但速度更快，并创建一个 `BanksClient` 与服务器通信。
- 一个快速且轻量的库，用于在 Rust 中测试 Solana 程序，通过创建一个针对程序开发者优化的进程内 Solana
  VM 来工作。
- 一个工具，用于扫描您的代码库中常见的安全漏洞并提供修复建议。
