## 静态分析工具

[Radar](https://github.com/Auditware/radar?tab=readme-ov-file) 是一个用于 Anchor
Rust 程序的静态分析工具。它允许您编写、共享和使用模板，通过强大的基于 Python 的规则引擎，利用逻辑表达式自动检测 Rust 程序中的安全问题和漏洞代码模式。

[Xray](https://github.com/sec3-product/x-ray)
是一个开源的跨平台命令行界面（CLI）工具，专为 Solana 程序和 Rust 编写的程序的静态分析而设计。
