## 安装

通过运行以下命令安装 Solana 工具包和 mucho CLI：

```shell
npx -y mucho@latest install
```

## 构建

```shell
mucho build
```

## 启动 Localnet

```shell
mucho validator
```

## 运行测试

Anchor 程序：

```shell
anchor test
```

原生程序：

```shell
cargo test
```

## 部署

```shell
mucho deploy
```

有关本地验证器自定义和命令的更多信息，请阅读
[Solana 测试验证器指南](/developers/guides/getstarted/solana-test-validator)。
