Solana 程序开发工具包以
[`mucho` npm 包](https://www.npmjs.com/package/mucho)
的形式发布。`mucho` 命令将用于运行工具包中的大多数 Solana 程序开发工具——_mucho 工具，一个 CLI_。

## 安装

开始使用，请安装 Solana 工具包：

```shell
npx -y mucho@latest install
mucho --version
```

这将安装以下最新版本的工具：

- [Solana CLI / Agave Tool Suite](https://docs.anza.xyz/cli/)：一个命令行工具，允许开发者与 Solana 区块链交互，管理账户，发送交易，以及部署程序。
- [Mucho CLI](https://github.com/solana-developers/mucho)：Solana 生态系统中流行开发工具的超集，用于简化 Solana 区块链程序的开发和测试。
- [Rust](https://doc.rust-lang.org/book/)：Solana 智能合约所使用的编程语言。
- [Anchor](https://www.anchor-lang.com/)：一个用于编写 Solana 程序的框架，通过抽象许多复杂性来加速智能合约开发。
- [Fuzz Tester](https://ackee.xyz/trident/docs/latest/)：基于 Rust 的 Solana 程序模糊测试框架，帮助您发布安全代码。
- [Code Coverage Tool](https://github.com/LimeChain/zest?tab=readme-ov-file)：一个用于 Solana 程序的代码覆盖率 CLI 工具。

安装完成后，所有 `mucho` 命令可以如下运行：

```shell
mucho --help
```

## 生成一个密钥对

如果是全新安装
[Solana CLI](https://docs.anza.xyz/cli/)，您需要生成一个新的 keypair。

```shell
solana-keygen new
```

_这将会将您的新 keypair 存储在 Solana
CLI 的默认路径 (_`~/.config/solana/id.json`_)，并打印生成的 pubkey。_

## 获取您的 keypair 的公钥

```shell
solana address
```

## 为您的 keypair 提供资金

```shell
solana airdrop 10 --url localhost
```

## 设置您的网络配置

检查您当前的配置：

```shell
solana config get
```

要使用此工具包，请更新您的配置以连接到 localhost：

```shell
solana config set --url localhost
```

要在本地测试，您还需要启动一个本地节点以运行在您的 localhost 上：

```shell
mucho validator
```

有关更多信息，请阅读 [本地测试指南](/docs/toolkit/local-validator)。
