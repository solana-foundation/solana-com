h1: getBlocksWithLimit RPC 方法
---

返回从指定 slot 开始的已确认区块列表

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getBlocksWithLimit",
  "params": [
    // !hover start slot
    5,
    // !hover limit
    3
  ]
}
```

### !参数

#### !! 起始 slot

!类型 u64  
!必填

起始 slot

#### !! 限制

!类型 u64

限制（不能超过 `start_slot` 的 500,000 个区块）

#### !! 2

!类型 object

配置对象

##### !! 承诺级别

!类型 string

承诺级别描述了区块在某一时间点的最终状态。详见
[配置状态承诺](/docs/rpc#configuring-state-commitment)。

- 不支持 "processed"

### !!结果

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": [5, 6, 7],
  "id": 1
}
```

!类型 array

一个包含 u64 整数的数组，列出从 `start_slot` 开始的已确认区块，最多包含 `limit` 个区块（包括在内）。

</APIMethod>
