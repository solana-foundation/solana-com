h1: minimumLedgerSlot RPC 方法
---

返回节点在其账本中拥有信息的最低 slot。

<Callout type="info">
  如果节点配置为清除较旧的账本数据，该值可能会随着时间增加。
</Callout>

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "minimumLedgerSlot"
}
```

### !params

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 1234,
  "id": 1
}
```

!type u64

最低账本 slot 编号

</APIMethod>
