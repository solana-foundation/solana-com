h1: getVersion RPC 方法
---

返回节点上运行的当前 Solana 版本

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getVersion"
}
```

### !参数

### !!结果

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover(1:4) result
  "result": {
    // !hover solana-core
    "solana-core": "1.16.7",
    // !hover feature-set
    "feature-set": 2891131721
  },
  "id": 1
}
```

!类型 object

包含以下内容的 JSON 对象：

##### !! solana-core

!类型 string

solana-core 的软件版本

##### !! feature-set

!类型 u32

当前软件功能集的唯一标识符

</APIMethod>
