h1: getIdentity RPC 方法

返回当前节点的身份 pubkey

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getIdentity"
}
```

### !参数

### !!结果

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover(1:3) result
  "result": {
    // !hover identity
    "identity": "2r1F4iWqVcb8M1DbAjQuFpebkQHY9hcVU4WuW2DJBppN"
  },
  "id": 1
}
```

!类型 object

结果字段将是一个包含以下字段的 JSON 对象：

##### !! 身份

!类型 string

当前节点的身份 pubkey（以 base-58 编码的字符串形式）

</APIMethod>
