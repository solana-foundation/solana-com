h1: getSlotLeader RPC 方法
---

返回当前的 slot leader

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getSlotLeader"
}
```

### !参数

#### !! 0

!类型 object

包含以下字段的配置对象：

##### !! commitment

!类型 string

commitment 描述了区块在某一时间点的最终确定程度。详见
[配置状态承诺](/docs/rpc#configuring-state-commitment)。

##### !! minContextSlot

!类型 number

请求可以评估的最小 slot

### !!结果

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": "ENvAW7JScgYq6o4zKZwewtkzzJgDzuJAFxYasvmEQdpS",
  "id": 1
}
```

!类型 string

节点身份 Pubkey，使用 base-58 编码的字符串

</APIMethod>
