h1: getSupply RPC 方法
---

返回当前供应量的信息。

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getSupply"
}
```

### !params

#### !! 0

!type object

配置对象，包含以下字段：

##### !! commitment

!type string

commitment 描述了在某一时间点区块的最终确定程度。详见
[配置状态承诺](/docs/rpc#configuring-state-commitment)。

##### !! excludeNonCirculatingAccountsList

!type bool

从响应中排除非流通账户列表

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  "result": {
    "context": { "slot": 1114 },
    // !hover(1:11) result
    "value": {
      // !hover total
      "total": 1016000,
      // !hover circulating
      "circulating": 16000,
      // !hover nonCirculating
      "nonCirculating": 1000000,
      // !hover(1:6) nonCirculatingAccounts
      "nonCirculatingAccounts": [
        "FEy8pTbP5fEoqMV1GdTz83byuA8EKByqYat1PKDgVAq5",
        "9huDUZfxoJ7wGMTffUE7vh1xePqef7gyrLJu9NApncqA",
        "3mi1GmwEE3zo2jmfDuzvjSX9ovRXsDUKHvsntpkhuLJ9",
        "BYxEJTDerkaRWBem3XgnVcdhppktBXa2HbkHPKj2Ui4Z"
      ]
    }
  },
  "id": 1
}
```

!type object

结果将是一个包含以下内容的 JSON 对象：

##### !! total

!type u64

以 lamports 为单位的总供应量

##### !! circulating

!type u64

以 lamports 为单位的流通供应量

##### !! nonCirculating

!类型 u64

非流通供应量（以 lamports 为单位）

##### !! nonCirculatingAccounts

!类型 数组

一个包含非流通账户地址的字符串数组。如果 `excludeNonCirculatingAccountsList` 已启用，返回的数组将为空。

</APIMethod>
