返回有关集群中所有参与节点的信息

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getClusterNodes"
}
```

### !参数

### !!结果

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover(1:9) result
  "result": [
    {
      // !hover gossip
      "gossip": "10.239.6.48:8001",
      // !hover pubkey
      "pubkey": "9QzsJf7LPLj8GkXbYT3LFDKqsj2hHG7TA3xinJHu8epQ",
      // !hover rpc
      "rpc": "10.239.6.48:8899",
      // !hover tpu
      "tpu": "10.239.6.48:8856",
      // !hover version
      "version": "1.0.0 c375ce1f"
    }
  ],
  "id": 1
}
```

!类型 数组

结果字段将是一个包含 JSON 对象的数组，每个对象具有以下子字段：

##### !! pubkey

!类型 字符串

节点公钥，作为 base-58 编码的字符串

##### !! gossip

!类型 字符串 | null

节点的 Gossip 网络地址

##### !! tpu

!类型 字符串 | null

节点的 TPU 网络地址

##### !! rpc

!类型 字符串 | null

节点的 JSON
RPC 网络地址，或者 `null` 如果未启用 JSON
RPC 服务

##### !! version

!类型 字符串 | null

节点的软件版本，或者 `null` 如果版本信息不可用

##### !! featureSet

!type u32 | null

节点功能集的唯一标识符

##### !! shredVersion

!type u16 | null

节点配置使用的分片版本

</APIMethod>
