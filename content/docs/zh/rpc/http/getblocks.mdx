h1: getBlocks RPC 方法
---

返回两个 slot 之间已确认区块的列表

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getBlocks",
  "params": [
    // !hover start slot
    5,
    // !hover end slot
    10
  ]
}
```

### !参数

#### !! 起始 slot

!type u64  
!required

起始 slot

#### !! 结束 slot

!type u64

结束 slot（必须不超过 `start_slot`
500,000 个区块）

#### !! 2

!type object

配置对象

##### !! commitment

!type string

commitment 描述了区块在某一时间点的最终确定状态。详见
[配置状态承诺](/docs/rpc#configuring-state-commitment)。

- 不支持 "processed"

### !!结果

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": [5, 6, 7, 8, 9, 10],
  "id": 1
}
```

!type array

一个 u64 整数数组，列出 `start_slot` 和 `end_slot`（如果提供）之间，或最新确认 slot（包括在内）之间的已确认区块。最大范围为 500,000 个 slot。

</APIMethod>
