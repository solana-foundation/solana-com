返回节点的当前区块高度

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getBlockHeight",
  "params": []
}
```

### !参数

#### !! 0

!类型 object

配置对象

##### !! commitment

!类型 string

commitment 描述了在某一时刻区块的最终确定程度。详见
[配置状态承诺](/docs/rpc#configuring-state-commitment)。

##### !! minContextSlot

!类型 number

请求可以被评估的最小 slot。

### !!结果

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 1233,
  "id": 1
}
```

!类型 u64

当前区块高度。

</APIMethod>
