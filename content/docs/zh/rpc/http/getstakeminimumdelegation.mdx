h1: getStakeMinimumDelegation RPC 方法
---

返回最低质押委托金额，以 lamports 为单位。

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getStakeMinimumDelegation"
}
```

### !参数

#### !! 0

!类型 object

包含以下字段的配置对象：

##### !! commitment

!类型 string

commitment 描述了在某一时刻区块的最终确定程度。详见
[配置状态承诺](/docs/rpc#configuring-state-commitment)。

### !!结果

```jsonc !response
{
  "jsonrpc": "2.0",
  "result": {
    "context": { "slot": 501 },
    // !hover result
    "value": 1000000000
  },
  "id": 1
}
```

!类型 u64

最低质押委托金额，以 lamports 为单位

</APIMethod>
