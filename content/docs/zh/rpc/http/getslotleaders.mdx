h1: getSlotLeaders RPC 方法

返回指定 slot 范围内的 slot leaders

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getSlotLeaders",
  "params": [
    // !hover start slot
    100,
    // !hover limit
    10
  ]
}
```

### !参数

#### !! 起始 slot

!类型 u64  
!必填

起始 slot，类型为 u64 整数

#### !! 限制

!类型 u64  
!必填

限制，类型为 u64 整数（范围为 1 到 5,000）
