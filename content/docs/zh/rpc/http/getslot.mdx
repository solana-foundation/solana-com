h1: getSlot RPC 方法
---

返回已达到[指定或默认确认级别](/docs/rpc/#configuring-state-commitment)的 slot。

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getSlot"
}
```

### !参数

#### !! 0

!类型 object

包含以下字段的配置对象：

##### !! commitment

!类型 string

确认级别描述了区块在该时间点的最终状态。详见[配置状态确认](/docs/rpc#configuring-state-commitment)。

##### !! minContextSlot

!类型 number

请求可以评估的最小 slot。

### !!结果

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 1234,
  "id": 1
}
```

!类型 u64

当前 slot

</APIMethod>
