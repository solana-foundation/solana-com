h1: getHighestSnapshotSlot RPC 方法
---

返回节点拥有快照的最高 slot 信息。

这将找到最高的完整快照 slot，以及基于该完整快照 slot 的最高增量快照 slot（如果存在）。

<Callout type="warn" title={"版本限制"}>
  此方法仅适用于 `solana-core`
  v1.9 或更新版本。 请使用 [getSnapshotSlot](/docs/rpc/http/getsnapshotslot)
  适用于 `solana-core` v1.8
  及以下版本。
</Callout>

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getHighestSnapshotSlot"
}
```

### !参数

### !!结果 快照

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover(1:4) result
  "result": {
    // !hover full
    "full": 100,
    // !hover incremental
    "incremental": 110
  },
  "id": 1
}
```

!类型 object

当节点有快照时，将返回一个包含以下字段的 JSON 对象：

##### !! 完整快照

!类型 u64

最高的完整快照 slot

##### !! 增量快照

!类型 u64 | null

基于 `full` 的最高增量快照 slot

### !!结果 无快照

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover(1:4) result
  "error": {
    "code": -32008,
    "message": "No snapshot"
  },
  "id": 1
}
```

!类型 object

如果节点没有快照，将返回一个 JSON RPC 错误响应。

</APIMethod>
