h1: getBalance RPC 方法
---

返回提供的 Pubkey 所对应账户的 lamport 余额。

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getBalance",
  "params": [
    // !hover pubkey
    "83astBRguLMdt2h5U1Tpdq5tjFoJ6noeGwaY3mDLVcri"
  ]
}
```

### !参数

#### !! pubkey

!类型 string  
!必填

要查询账户的 Pubkey，使用 base-58 编码的字符串。

#### !! 1

!类型 object

配置对象。

##### !! commitment

!类型 string

承诺值描述了某个时间点区块的最终确定程度。详见
[配置状态承诺](/docs/rpc#configuring-state-commitment)。

##### !! minContextSlot

!类型 number

请求可以被评估的最小 slot。

### !!结果

```jsonc !response
{
  "jsonrpc": "2.0",
  "result": {
    "context": { "slot": 1 },
    // !hover result
    "value": 0
  },
  "id": 1
}
```

!类型 u64

包含值字段设置为余额的 RpcResponse JSON 对象。

</APIMethod>
