h1: getInflationGovernor RPC 方法
---

返回当前的通胀管理器

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getInflationGovernor"
}
```

### !params

#### !! 0

!type object

配置对象。

##### !! commitment

!type string

承诺描述了在某一时间点区块的最终确定程度。详见
[配置状态承诺](/docs/rpc#configuring-state-commitment)。

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover(1:7) result
  "result": {
    // !hover foundation
    "foundation": 0.05,
    // !hover foundationTerm
    "foundationTerm": 7,
    // !hover initial
    "initial": 0.15,
    // !hover taper
    "taper": 0.15,
    // !hover terminal
    "terminal": 0.015
  },
  "id": 1
}
```

!type object

结果字段将是一个包含以下字段的 JSON 对象：

##### !! foundation

!type f64

分配给基金会的总通胀百分比

##### !! foundationTerm

!type f64

基金会池通胀的持续时间（以年为单位）

##### !! initial

!type f64

从时间 0 开始的初始通胀百分比

##### !! 锥化

!类型 f64

每年降低通胀的速率。（速率的减少是基于创世配置中的目标 slot 时间计算得出）

##### !! 终端

!类型 f64

终端通胀百分比

</APIMethod>
