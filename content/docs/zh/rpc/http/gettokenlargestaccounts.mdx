h1: getTokenLargestAccounts RPC 方法
---

返回特定 SPL Token 类型的 20 个最大账户。

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getTokenLargestAccounts",
  "params": [
    // !hover pubkey
    "3wyAj7Rt1TWVPZVteFJPLa26JmLvdb1CAKEFZm3NY75E"
  ]
}
```

### !参数

#### !! pubkey

!类型 string  
!必填

要查询的代币 Mint 的公钥，使用 base-58 编码的字符串

#### !! 1

!类型 object

包含以下字段的配置对象：

##### !! commitment

!类型 string

承诺描述了在某一时间点区块的最终确定程度。详见
[配置状态承诺](/docs/rpc#configuring-state-commitment)。

### !!结果

```jsonc !response

  "jsonrpc": "2.0",
  "result": {
    "context": { "slot": 1114 },
    // !hover(1:16) result
    "value": [
      {
        // !hover address
        "address": "FYjHNoFtSQ5uijKrZFyYAxvEr87hsKXkXcxkcmkBAf4r",

        // !hover decimals

      {

        "uiAmountString": "2.29"
      }
    ]

```

!类型 array

包含以下内容的 JSON 对象数组：

##### !! address

!类型 string

代币账户的地址

##### !! amount

!类型 string

未经过小数处理的原始代币账户余额，u64 的字符串表示

##### !! 小数位数

!类型 u8

小数点右侧的以 10 为基数的位数

##### !! uiAmount

!类型 number | null

代币账户余额，使用铸币规定的小数位数 **已弃用**

##### !! uiAmountString

!类型 string

代币账户余额作为字符串，使用铸币规定的小数位数

</APIMethod>
