返回账本中的当前交易计数

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getTransactionCount"
}
```

### !参数

#### !! 0

!类型 object  
!可选

包含以下字段的配置对象：

##### !! commitment

!类型 string

承诺描述了在该时间点区块的最终确定程度。详见
[配置状态承诺](/docs/rpc#configuring-state-commitment)。

##### !! minContextSlot

!类型 number

请求可以评估的最小 slot

### !!结果

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 268,
  "id": 1
}
```

!类型 u64

账本中的当前交易计数

</APIMethod>
