h1: rootSubscribe RPC 方法
---

订阅以在验证器设置新的 root 时接收通知。

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !参数

**无**

### !!结果

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!类型 整数

订阅 ID（用于取消订阅）

</APIMethod>

#### 通知格式：

结果是最新的 root slot 编号。

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
