h1: programUnsubscribe RPC 方法
---

取消订阅由程序拥有的账户变更通知

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "programUnsubscribe",
  "params": [
    // !hover 0
    0
  ]
}
```

### !参数

#### !! 0

!类型 数字  
!必需

要取消的账户订阅 ID

### !!结果

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": true,
  "id": 1
}
```

!类型 布尔值

取消订阅成功消息

</APIMethod>
