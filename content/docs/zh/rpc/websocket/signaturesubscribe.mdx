h1: signatureSubscribe RPC 方法
---

订阅以在具有指定签名的交易达到指定的承诺级别时接收通知。

<Callout type="warn">
  这是对单个通知的订阅。一旦通知
  `signatureNotification` 被 RPC
  服务器发送后，订阅将自动取消。
</Callout>

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "signatureSubscribe",
  "params": [
    // !hover 0
    "2EBVM6cB8vAAD93Ktr6Vd8p67XPbQzCJX47MpReuiCXJAtcjaxpvWpcg9Ege1Nr5Tk3a2GFrByT7WPBjdsTycY9b",
    // !hover(1:4) 1
    {
      // !hover commitment
      "commitment": "finalized",
      // !hover enableReceivedNotification
      "enableReceivedNotification": false
    }
  ]
}
```

### !参数

#### !! 0

!类型 string  
!必需

交易签名，作为 base-58 编码的字符串

<Callout type="info">
  交易签名必须是交易的第一个签名 （有关详细信息，请参阅[交易
  ID](/docs/references/terminology#transaction-id)）。
</Callout>

#### !! 1

!类型 object

包含以下字段的配置对象：

##### !! commitment

!类型 string

承诺描述了在某一时刻区块的最终确定程度。详见
[配置状态承诺](/docs/rpc#configuring-state-commitment)。

##### !! enableReceivedNotification

!类型 boolean

是否订阅当签名被 RPC 接收到时的通知，除了处理完成时的通知。

### !!结果

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!类型 integer

订阅 ID（用于取消订阅）

</APIMethod>

#### 通知格式：

通知将是一个 RpcResponse JSON 对象，其中的 value 包含一个对象，包含：

- `slot: <u64>` - 对应的 slot。
- `value: <object|string>` - 一个通知值
  [`RpcSignatureResult`](https://github.com/solana-labs/solana/blob/6d28fd455b07e3557fc6c0c3ddf3ba03e3fe8482/rpc-client-api/src/response.rs#L265-L268)，结果可能是：
  - 当 `enableReceivedNotification` 为 `true` 且签名已接收时：字面字符串
    [`"receivedSignature"`](https://github.com/solana-labs/solana/blob/6d28fd455b07e3557fc6c0c3ddf3ba03e3fe8482/rpc-client-api/src/response.rs#L286-L288)，或
  - 当签名已处理时：`err: <object|null>`：
    - `null` 如果交易在指定的承诺级别成功处理，或
    - 一个
      [`TransactionError`](https://github.com/solana-labs/solana/blob/6d28fd455b07e3557fc6c0c3ddf3ba03e3fe8482/sdk/src/transaction/error.rs#L15-L164)，如果交易失败

#### 示例响应：

以下是一个成功**处理**交易的通知示例响应：

```json
{
  "jsonrpc": "2.0",
  "method": "signatureNotification",
  "params": {
    "result": {
      "context": {
        "slot": 5207624
      },
      "value": {
        "err": null
      }
    },
    "subscription": 24006
  }
}
```

以下是一个成功**接收**交易签名的通知示例响应：

```json
{
  "jsonrpc": "2.0",
  "method": "signatureNotification",
  "params": {
    "result": {
      "context": {
        "slot": 5207624
      },
      "value": "receivedSignature"
    },
    "subscription": 24006
  }
}
```
