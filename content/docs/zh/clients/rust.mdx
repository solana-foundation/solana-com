Solana 的 Rust crates 已
[发布到 crates.io](https://crates.io/search?q=solana-)，并可在
[docs.rs 上找到](https://docs.rs/releases/search?query=solana-)，它们的前缀为 `solana-`。

## Rust Crates

以下是 Solana 开发中最重要和最常用的 Rust crates：

- [`solana-program`]
  &mdash; 被运行在 Solana 上的程序导入，编译为 SBF。此 crate 包含许多基础数据类型，并从
  [`solana-sdk`]
  中重新导出，而后者无法从 Solana 程序中导入。

- [`solana-sdk`]
  &mdash; 基本的链下 SDK，它重新导出
  [`solana-program`]
  并在其基础上添加更多 API。大多数不在链上运行的 Solana 程序都会导入此 crate。

- [`solana-client`]
  &mdash; 用于通过 [JSON RPC API](/docs/rpc) 与 Solana 节点交互。

- [`solana-cli-config`]
  &mdash; 加载和保存 Solana CLI 配置文件。

- [`solana-clap-utils`] &mdash; 使用
  [`clap`]
  设置 CLI 的例程，正如主要的 Solana
  CLI 所使用的那样。包括加载 CLI 支持的所有类型签名者的功能。

## 安装

用于发送交易和与 Solana 交互：

```shell title="Terminal"
cargo add solana-sdk solana-client
```

构建 Solana 程序，安装：

```shell title="Terminal"
cargo add solana-program
```
