Solana的Rust程序包
[发布在crates.io上](https://crates.io/search?q=solana-)，可以在
[docs.rs](https://docs.rs/releases/search?query=solana-)上找到，带有`solana-`
前缀。

## Rust程序包

以下是Solana开发中最重要和最常用的Rust程序包：

* [`solana-program`]——由运行在Solana上的程序导入，编译为SBF。这个程序包包含许多基本数据类型，并从
  [`solana-sdk`]中重新导出，而solana-sdk不能从Solana程序中导入。

* [`solana-sdk`]——基本的链下SDK，它重新导出
  [`solana-program`]并在其基础上添加更多API。大多数不在链上运行的Solana程序都会导入这个。

* [`solana-client`]——通过 [JSON RPC API](/docs/rpc)与Solana节点交互。

* [`solana-cli-config`]——加载和保存Solana CLI配置文件。

* [`solana-clap-utils`]
  — 用于设置CLI的例程，使用[`clap`]，与Solana主CLI使用的相同。包含加载CLI支持的所有类型签名者的函数。

[`solana-program`]: https://docs.rs/solana-program

[`solana-sdk`]: https://docs.rs/solana-sdk

[`solana-client`]: https://docs.rs/solana-client

[`solana-cli-config`]: https://docs.rs/solana-cli-config

[`solana-clap-utils`]: https://docs.rs/solana-clap-utils

## 安装

用于发送交易和与Solana交互：

```shell title="Terminal"
cargo add solana-sdk solana-client
```

构建solana程序，安装：

```shell title="Terminal"
cargo add solana-program
```
