h1: Solana 的 Go SDK
---

有一个
[Go SDK](https://github.com/gagliardetto/solana-go)，支持与 Solana 交互的大多数功能。如果您正在为 Solana 程序使用 Anchor，您可以使用
[anchor-go](https://github.com/gagliardetto/anchor-go)
和您的 idl 一起生成兼容 Golang 的客户端。

## 安装

```bash  title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```
