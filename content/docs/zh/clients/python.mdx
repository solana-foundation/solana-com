h1: Solana的Python SDK
---

有一个[Python SDK](https://github.com/michaelhly/solana-py)支持大多数与Solana交互的功能。如果你正在使用Anchor开发Solana程序，你可以使用[anchor-py](https://github.com/kevinheavey/anchorpy)配合你的idl生成Python兼容的客户端。

## 安装

```bash title="Terminal"
pip install solders
pip install solana
```
