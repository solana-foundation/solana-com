---

Có một [Go SDK](https://github.com/gagliardetto/solana-go) hỗ trợ hầu hết các
chức năng để tương tác với Solana. Nếu bạn đang sử dụng Anchor cho các chương
trình Solana của mình, bạn sẽ có thể sử dụng
[anchor-go](https://github.com/gagliardetto/anchor-go) với idl của bạn để tạo
một client tương thích với Golang.

## Cài đặt

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```

## Tài liệu thêm

Vui lòng truy cập
[tài liệu Go SDK](https://github.com/gagliardetto/solana-go?tab=readme-ov-file#installation)
để biết thêm thông tin về việc sử dụng Go SDK này với Solana.
