Các crate Rust của Solana được
[xuất bản trên crates.io](https://crates.io/search?q=solana-) và có thể được tìm
thấy [trên docs.rs](https://docs.rs/releases/search?query=solana-) với tiền tố
`solana-`.

## Các crate Rust

Sau đây là những crate Rust quan trọng và thường được sử dụng nhất cho phát
triển trên Solana:

- [`solana-program`] &mdash; Được import bởi các chương trình chạy trên Solana,
  biên dịch sang SBF. Crate này chứa nhiều kiểu dữ liệu cơ bản và được tái xuất
  từ [`solana-sdk`], thứ không thể được import từ một chương trình Solana.

- [`solana-sdk`] &mdash; SDK offchain cơ bản, nó tái xuất [`solana-program`] và
  thêm nhiều API bên trên đó. Hầu hết các chương trình Solana không chạy
  on-chain sẽ import cái này.

- [`solana-client`] &mdash; Để tương tác với một node Solana thông qua
  [JSON RPC API](/docs/rpc).

- [`solana-cli-config`] &mdash; Tải và lưu tệp cấu hình CLI Solana.

- [`solana-clap-utils`] &mdash; Các quy trình để thiết lập CLI, sử dụng
  [`clap`], như được sử dụng bởi CLI Solana chính. Bao gồm các hàm để tải tất cả
  các loại người ký được hỗ trợ bởi CLI.

[`solana-sdk`]: https://docs.rs/solana-sdk

## Cài đặt

Để gửi giao dịch và tương tác với Solana:

```shell title="Terminal"
cargo add solana-sdk solana-client
```

Để xây dựng các chương trình Solana, hãy cài đặt:

```shell title="Terminal"
cargo add solana-program
```
