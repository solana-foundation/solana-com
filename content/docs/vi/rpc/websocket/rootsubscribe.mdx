h1: Phương thức RPC rootSubscribe
---

Đăng ký để nhận thông báo bất cứ khi nào một root mới được thiết lập bởi
validator.

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**Không có**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

id đăng ký (cần thiết để hủy đăng ký)

</APIMethod>

#### Định dạng thông báo:

Kết quả là số slot root mới nhất.

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
