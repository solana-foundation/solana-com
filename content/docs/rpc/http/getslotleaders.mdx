h1: getSlotLeaders RPC Method

Returns the slot leaders for a given slot range

<APIMethod>

```jsonc !!request with-client
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getSlotLeaders",
  "params": [
    // !hover 0
    100,
    // !hover 1
    10
  ]
}
```

### !params

#### !! 0

!type u64  
!required

Start slot, as u64 integer

#### !! 1

!type u64  
!required

Limit, as u64 integer (between 1 and 5,000)
