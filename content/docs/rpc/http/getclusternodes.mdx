h1: getClusterNodes RPC Method
---

Returns information about all the nodes participating in the cluster

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getClusterNodes"
}
```

### !params

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover(1:9) result
  "result": [
    {
      // !hover gossip
      "gossip": "10.239.6.48:8001",
      // !hover pubkey
      "pubkey": "9QzsJf7LPLj8GkXbYT3LFDKqsj2hHG7TA3xinJHu8epQ",
      // !hover rpc
      "rpc": "10.239.6.48:8899",
      // !hover tpu
      "tpu": "10.239.6.48:8856",
      // !hover version
      "version": "1.0.0 c375ce1f"
    }
  ],
  "id": 1
}
```

!type array

The result field will be an array of JSON objects, each with the following sub
fields:

##### !! pubkey

!type string

Node public key, as base-58 encoded string

##### !! gossip

!type string | null

Gossip network address for the node

##### !! tpu

!type string | null

TPU network address for the node

##### !! rpc

!type string | null

JSON RPC network address for the node, or `null` if the JSON RPC service is not
enabled

##### !! version

!type string | null

The software version of the node, or `null` if the version information is not
available

##### !! featureSet

!type u32 | null

The unique identifier of the node's feature set

##### !! shredVersion

!type u16 | null

The shred version the node has been configured to use

</APIMethod>
