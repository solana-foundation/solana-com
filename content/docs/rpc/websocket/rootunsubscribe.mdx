Unsubscribe from root notifications

<DocSideBySide>

<DocLeftSide>

### Parameters

<Parameter type={"number"} required={true}>
  subscription id to cancel
</Parameter>

### Result

`<bool>` - unsubscribe success message

</DocLeftSide>

<DocRightSide>

### Code sample

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootUnsubscribe",
  "params": [0]
}
```

### Response

```json
{ "jsonrpc": "2.0", "result": true, "id": 1 }
```

</DocRightSide>
</DocSideBySide>
