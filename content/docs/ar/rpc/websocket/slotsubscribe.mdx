h1: طريقة RPC لـ slotSubscribe
---

اشترك لتلقي إشعار في أي وقت تتم فيه معالجة slot بواسطة validator

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "slotSubscribe"
}
```

### !المعلمات

**لا شيء**

### !!النتيجة

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!النوع عدد صحيح

معرف الاشتراك (مطلوب لإلغاء الاشتراك)

</APIMethod>

#### تنسيق الإشعار:

سيكون الإشعار عبارة عن كائن يحتوي على الحقول التالية:

- `parent: <u64>` - الـ slot الأصل
- `root: <u64>` - الـ slot الجذر الحالي
- `slot: <u64>` - قيمة الـ slot المعينة حديثًا

مثال:

```json
{
  "jsonrpc": "2.0",
  "method": "slotNotification",
  "params": {
    "result": {
      "parent": 75,
      "root": 44,
      "slot": 76
    },
    "subscription": 0
  }
}
```
