h1: طريقة RPC لـ rootSubscribe
---

اشترك لتلقي إشعار في أي وقت يتم فيه تعيين جذر جديد بواسطة الـ validator.

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**لا شيء**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

معرف الاشتراك (مطلوب لإلغاء الاشتراك)

</APIMethod>

#### تنسيق الإشعار:

النتيجة هي رقم الـ slot الجذر الأحدث.

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
