هناك [مكتبة Python SDK](https://github.com/michaelhly/solana-py) تدعم معظم
الوظائف للتفاعل مع Solana. إذا كنت تستخدم Anchor لبرامج Solana الخاصة بك، ستتمكن
من استخدام [anchor-py](https://github.com/kevinheavey/anchorpy) مع ملف idl الخاص
بك لإنشاء عميل متوافق مع Python.

## التثبيت

```bash title="Terminal"
pip install solders
pip install solana
```
