هناك [مكتبة Go SDK](https://github.com/gagliardetto/solana-go) تدعم معظم الوظائف
للتفاعل مع سولانا. إذا كنت تستخدم Anchor لبرامج سولانا الخاصة بك، ستتمكن من
استخدام [anchor-go](https://github.com/gagliardetto/anchor-go) مع ملف idl الخاص
بك لإنتاج عميل متوافق مع لغة Golang.

## التثبيت

```bash title="Terminal"
go get github.com/gagliardetto/solana-go@v1.12.0
```

## المزيد من الوثائق

يرجى زيارة
[وثائق مكتبة Go SDK](https://github.com/gagliardetto/solana-go?tab=readme-ov-file#installation)
للمزيد من المعلومات حول استخدام مكتبة Go SDK هذه مع سولانا.
