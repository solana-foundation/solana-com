**Kunci Fitur**:
[`C9oAhLxDBm3ssWtJx1yBGzPY55r2rArHmN1pbQn6HogH`](https://explorer.solana.com/address/C9oAhLxDBm3ssWtJx1yBGzPY55r2rArHmN1pbQn6HogH)

Gerbang fitur ini mengurangi alokasi CU default untuk instruksi program builtin
dari 200.000 menjadi 3.000 CU untuk meningkatkan kinerja validator.

## Dampak

Kurang dari 0,00001% transaksi akan terpengaruh oleh perubahan ini.

**Pengembang** harus selalu menggunakan batas anggaran komputasi untuk secara
eksplisit mengalokasikan CU untuk transaksi mereka. Perubahan ini tidak akan
memengaruhi transaksi yang secara eksplisit mengalokasikan CU.
