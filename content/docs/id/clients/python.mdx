h1: Python SDK untuk Solana
---

Ada [Python SDK](https://github.com/michaelhly/solana-py) yang mendukung
sebagian besar fungsi untuk berinteraksi dengan Solana. Jika Anda menggunakan
Anchor untuk program Solana Anda, Anda akan dapat menggunakan
[anchor-py](https://github.com/kevinheavey/anchorpy) dengan idl Anda untuk
menghasilkan klien yang kompatibel dengan Python.

## Instalasi

```bash title="Terminal"
pip install solders
pip install solana
```
