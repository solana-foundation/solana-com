h1: Metode RPC rootSubscribe
---

Berlangganan untuk menerima notifikasi setiap kali root baru ditetapkan oleh
validator.

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

### !params

**Tidak ada**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

id langganan (diperlukan untuk berhenti berlangganan)

</APIMethod>

#### Format Notifikasi:

Hasilnya adalah nomor slot root terbaru.

```json
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```
