h1: Metode RPC slotSubscribe
---

Berlangganan untuk menerima notifikasi setiap kali slot diproses oleh validator

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "slotSubscribe"
}
```

### !params

**Tidak ada**

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 0,
  "id": 1
}
```

!type integer

Id langganan (diperlukan untuk berhenti berlangganan)

</APIMethod>

#### Format Notifikasi:

Notifikasi akan berupa objek dengan bidang-bidang berikut:

- `parent: <u64>` - Slot induk
- `root: <u64>` - Slot root saat ini
- `slot: <u64>` - Nilai slot yang baru ditetapkan

Contoh:

```json
{
  "jsonrpc": "2.0",
  "method": "slotNotification",
  "params": {
    "result": {
      "parent": 75,
      "root": 44,
      "slot": 76
    },
    "subscription": 0
  }
}
```
