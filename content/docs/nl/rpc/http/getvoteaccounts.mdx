h1: getVoteAccounts RPC-methode

Geeft de accountinformatie en bijbehorende stake voor alle stemaccounts in de
huidige bank.

<APIMethod>

```jsonc !!request curl
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getVoteAccounts",
  "params": [
    // !hover(1:3) 0
    {
      // !hover commitment
      "commitment": "finalized",
      // !hover votePubkey
      "votePubkey": "i7NyKBMJCA9bLM2nsGyAGCKHECuR2L5eh4GqFciuwNT"
    }
  ]
}
```

```ts !!request title="Kit"
import { address, createSolanaRpc } from "@solana/kit";

const rpc_url = "https://api.devnet.solana.com";
const rpc = createSolanaRpc(rpc_url);

// !hover votePubkey
let votePubkey = address("5ZWgXcyqrrNpQHCme5SdC5hCeYb2o3fEJhF7Gok3bTVN");

let voteAccounts = await rpc
  .getVoteAccounts({
    votePubkey
  })
  .send();

console.log(voteAccounts);
```

```ts !!request title="web3.js"
import { Connection, PublicKey, clusterApiUrl } from "@solana/web3.js";

const connection = new Connection(clusterApiUrl("devnet"), "confirmed");

let version = await connection.getVoteAccounts();

console.log(version);
```

```rs !!request title="Rust"
use anyhow::Result;
use solana_client::{nonblocking::rpc_client::RpcClient, rpc_config::RpcGetVoteAccountsConfig};
use solana_sdk::commitment_config::CommitmentConfig;

#[tokio::main]
async fn main() -> Result<()> {
    let client = RpcClient::new_with_commitment(
        String::from("https://api.devnet.solana.com"),
        CommitmentConfig::confirmed(),
    );

    // !hover votePubkey
    let vote_pubkey = String::from("5ZWgXcyqrrNpQHCme5SdC5hCeYb2o3fEJhF7Gok3bTVN");

    let config = RpcGetVoteAccountsConfig {
        vote_pubkey: Some(vote_pubkey),
        commitment: CommitmentConfig::finalized().into(),
        keep_unstaked_delinquents: None,
        delinquent_slot_distance: None,
    };

    let vote_accounts = client.get_vote_accounts_with_config(config).await?;

    println!("{:#?}", vote_accounts);

    Ok(())
}
```

### !params

#### !! 0

!type object !optional

Configuratieobject met de volgende velden:

##### !! commitment

!type string

De commitment beschrijft hoe definitief een blok is op dat moment. Zie
[Configureren van state commitment](/docs/rpc#configuring-state-commitment).

##### !! votePubkey

!type string

Geef alleen resultaten terug voor dit validator stemadres (base-58 gecodeerd)

##### !! keepUnstakedDelinquents

!type bool

Filter geen nalatige validators zonder stake uit

##### !! delinquentSlotDistance

!type u64

Specificeer het aantal slots achter de tip dat een validator moet vallen om als
nalatig te worden beschouwd. **OPMERKING:** Omwille van de consistentie tussen
ecosysteem producten, _wordt **niet** aanbevolen om dit argument te
specificeren._

!type object

Het resultatenveld zal een JSON-object zijn van `current` en `delinquent`
accounts, elk met een array van JSON-objecten met de volgende subvelden:

##### !! activatedStake

!type u64

De stake, in lamports, gedelegeerd aan dit stemaccount en actief in deze epoch

##### !! commission

!type number

Percentage (0-100) van beloningsuitbetaling verschuldigd aan het stemaccount

##### !! epochCredits

!type array

Meest recente geschiedenis van verdiende credits voor maximaal vijf epochs, als
een array van arrays die het volgende bevatten:
`[epoch, credits, previousCredits]`

##### !! epochVoteAccount

!type bool

Of het stemaccount is gestaked voor deze epoch

##### !! lastVote

!type u64

Meest recente slot waarop door dit stemaccount is gestemd

##### !! nodePubkey

!type string

Validator-identiteit, als base-58 gecodeerde string

##### !! rootSlot

!type u64

Huidige root slot voor dit stemaccount

##### !! votePubkey

!type string

Stemaccountadres, als base-58 gecodeerde string

</APIMethod>
