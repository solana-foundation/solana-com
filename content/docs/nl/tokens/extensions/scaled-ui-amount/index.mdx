## Wat is de Geschaalde UI-hoeveelheid extensie?

De geschaalde UI-hoeveelheid extensie stelt uitgevers in staat om een
bijwerkbare vermenigvuldiger toe te passen op de UI-hoeveelheid van een token.
Dit is nuttig voor scenario's waarin het token wordt gebruikt om een reëel
activum te vertegenwoordigen, zoals een aandeel of dividend.

Bijvoorbeeld, bij een aandelensplitsing verdubbelt het aantal aandelen voor alle
houders. Zonder deze extensie is het momenteel onpraktisch om op een efficiënte
en schaalbare manier nieuwe tokens uit te geven aan alle houders tijdens een
splitsingsgebeurtenis.

Met het Token-2022 extensiemodel hebben we echter de mogelijkheid om te
veranderen hoe de UI-hoeveelheid van tokens wordt weergegeven. Door de
ScaledUiAmount extensie en de `amount_to_ui_amount` instructie te gebruiken, kun
je een UI-vermenigvuldiger instellen op je token en de UI-hoeveelheid op elk
moment opvragen.
[broncode](https://github.com/solana-program/token-2022/tree/main/program/src/extension/scaled_ui_amount).

Deze functie kan ook worden gebruikt voor dividenden of het uitkeren van
rendement.

Opmerking: Er worden nooit nieuwe tokens aangemaakt, de UI-hoeveelheid geeft de
ruwe hoeveelheid tokens vermenigvuldigd met de huidige vermenigvuldiger weer. De
functie is volledig cosmetisch.

<Callout type="info">
  Sommige extensies zijn niet compatibel met elkaar en kunnen niet
  tegelijkertijd worden ingeschakeld op dezelfde token mint of token account. Je
  moet niet zowel de `ScaledUiAmount` extensie als de `InterestBearingMint`
  extensie inschakelen op dezelfde token mint of token account.
</Callout>
