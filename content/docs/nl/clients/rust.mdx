Solana's Rust crates worden
[gepubliceerd op crates.io](https://crates.io/search?q=solana-) en zijn te
vinden [op docs.rs](https://docs.rs/releases/search?query=solana-) met het
`solana-` voorvoegsel.

## Rust Crates

De volgende zijn de belangrijkste en meest gebruikte Rust crates voor Solana
ontwikkeling:

- [`solana-program`] &mdash; Geïmporteerd door programma's die op Solana
  draaien, gecompileerd naar SBF. Deze crate bevat veel fundamentele datatypen
  en wordt opnieuw geëxporteerd vanuit [`solana-sdk`], die niet kan worden
  geïmporteerd vanuit een Solana-programma.

- [`solana-sdk`] &mdash; De basis offchain SDK, het exporteert
  [`solana-program`] opnieuw en voegt daar meer API's aan toe. De meeste
  Solana-programma's die niet on-chain draaien zullen dit importeren.

- [`solana-client`] &mdash; Voor interactie met een Solana-node via de
  [JSON RPC API](/docs/rpc).

- [`solana-cli-config`] &mdash; Laden en opslaan van het Solana
  CLI-configuratiebestand.

- [`solana-clap-utils`] &mdash; Routines voor het opzetten van een CLI, met
  behulp van [`clap`], zoals gebruikt door de belangrijkste Solana CLI. Bevat
  functies voor het laden van alle soorten ondertekenaars die worden ondersteund
  door de CLI.

[`solana-program`]: https://docs.rs/solana-program
[`solana-sdk`]: https://docs.rs/solana-sdk
[`solana-client`]: https://docs.rs/solana-client
[`solana-cli-config`]: https://docs.rs/solana-cli-config
[`solana-clap-utils`]: https://docs.rs/solana-clap-utils

## Installatie

Voor het verzenden van transacties en interactie met Solana:

```shell title="Terminal"
cargo add solana-sdk solana-client
```

Voor het bouwen van Solana-programma's, installeer:

```shell title="Terminal"
cargo add solana-program
```
