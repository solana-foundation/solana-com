## Installation

Install the Solana Toolkit and mucho CLI by running the following command:

```shell
npx -y mucho@latest install
```

## Build

```shell
mucho build
```

## Start Localnet

```shell
mucho validator
```

## Run Tests

Anchor Programs:

```shell
anchor test
```

Native Programs:

```shell
cargo test
```

## Deploy

```shell
mucho deploy
```

For more information on local validator customization and commands, read the
[Solana Test Validator Guide](/developers/guides/getstarted/solana-test-validator).
