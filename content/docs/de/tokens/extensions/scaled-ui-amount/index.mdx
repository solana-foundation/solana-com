## Was ist die Scaled UI Amount-Erweiterung?

Die Scaled UI Amount-Erweiterung ermöglicht es Emittenten, einen
aktualisierbaren Multiplikator auf den UI-Betrag eines Tokens anzuwenden. Dies
ist nützlich für Szenarien, in denen der Token zur Darstellung eines realen
Vermögenswerts verwendet wird, wie beispielsweise einer Aktie oder Dividende.

Beispielsweise verdoppelt ein Aktiensplit die Anzahl der Aktien für alle
Inhaber. Ohne diese Erweiterung ist es derzeit unpraktisch, während eines
Split-Events auf effiziente und skalierbare Weise neue Tokens an alle Inhaber
auszugeben.

Mit dem Token-2022-Erweiterungsmodell haben wir jedoch die Möglichkeit, zu
ändern, wie der UI-Betrag von Tokens dargestellt wird. Mit der
ScaledUiAmount-Erweiterung und der `amount_to_ui_amount` Anweisung können Sie
einen UI-Multiplikator für Ihren Token festlegen und seinen UI-Betrag jederzeit
abrufen.
[Quellcode](https://github.com/solana-program/token-2022/tree/main/program/src/extension/scaled_ui_amount).

Diese Funktion kann auch für Dividenden oder die Verteilung von Renditen
verwendet werden.

Hinweis: Es werden niemals neue Tokens erstellt, der UI-Betrag gibt den
Rohbetrag der Tokens multipliziert mit dem aktuellen Multiplikator zurück. Die
Funktion ist rein kosmetisch.

<Callout type="info">
  Einige Erweiterungen sind nicht miteinander kompatibel und können nicht
  gleichzeitig für denselben Token-Mint oder dasselbe token account aktiviert
  werden. Sie sollten nicht sowohl die `ScaledUiAmount` Erweiterung als auch die
  `InterestBearingMint` Erweiterung auf demselben Token-Mint oder token account
  aktivieren.
</Callout>
