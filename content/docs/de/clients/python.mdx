h1: Python SDK für Solana
---

Es gibt ein [Python SDK](https://github.com/michaelhly/solana-py), das die
meisten Funktionen zur Interaktion mit Solana unterstützt. Wenn Sie Anchor für
Ihre Solana-Programme verwenden, können Sie
[anchor-py](https://github.com/kevinheavey/anchorpy) mit Ihrer IDL nutzen, um
einen Python-kompatiblen Client zu erstellen.

## Installation

```bash title="Terminal"
pip install solders
pip install solana
```
