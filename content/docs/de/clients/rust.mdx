Solanas Rust-Crates werden
[auf crates.io veröffentlicht](https://crates.io/search?q=solana-) und können
[auf docs.rs](https://docs.rs/releases/search?query=solana-) mit dem `solana-`
Präfix gefunden werden.

## Rust-Crates

Die folgenden sind die wichtigsten und am häufigsten verwendeten Rust-Crates für
die Solana-Entwicklung:

- [`solana-program`] &mdash; Wird von Programmen importiert, die auf Solana
  laufen und zu SBF kompiliert werden. Dieses Crate enthält viele grundlegende
  Datentypen und wird von [`solana-sdk`] re-exportiert, das nicht von einem
  Solana-Programm importiert werden kann.

- [`solana-sdk`] &mdash; Das grundlegende Offchain-SDK, es re-exportiert
  [`solana-program`] und fügt weitere APIs hinzu. Die meisten Solana-Programme,
  die nicht on-chain laufen, werden dieses importieren.

- [`solana-client`] &mdash; Für die Interaktion mit einem Solana-Node über die
  [JSON RPC API](/docs/rpc).

- [`solana-cli-config`] &mdash; Laden und Speichern der Solana
  CLI-Konfigurationsdatei.

- [`solana-clap-utils`] &mdash; Routinen für die Einrichtung einer CLI mit
  [`clap`], wie sie von der Haupt-Solana-CLI verwendet werden. Enthält
  Funktionen zum Laden aller Arten von Signern, die von der CLI unterstützt
  werden.

[`solana-program`]: https://docs.rs/solana-program
[`solana-sdk`]: https://docs.rs/solana-sdk
[`solana-client`]: https://docs.rs/solana-client

## Installation

Für das Senden von Transaktionen und die Interaktion mit Solana:

```shell title="Terminal"
cargo add solana-sdk solana-client
```

Für die Entwicklung von Solana-Programmen installieren Sie:

```shell title="Terminal"
cargo add solana-program
```
