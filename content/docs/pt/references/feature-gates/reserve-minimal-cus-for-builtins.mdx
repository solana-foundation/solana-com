**Chave do Recurso**:
[`C9oAhLxDBm3ssWtJx1yBGzPY55r2rArHmN1pbQn6HogH`](https://explorer.solana.com/address/C9oAhLxDBm3ssWtJx1yBGzPY55r2rArHmN1pbQn6HogH)

Este gate de recurso reduz a alocação padrão de CU para instruções de programas
builtin de 200.000 para 3.000 CUs para melhorar o desempenho do validator.

## Impacto

Menos de 0,00001% das transações serão afetadas por esta mudança.

**Desenvolvedores** devem sempre usar limites de orçamento computacional para
alocar explicitamente CUs para suas transações. Esta mudança não afetará
transações que alocam CUs explicitamente.
