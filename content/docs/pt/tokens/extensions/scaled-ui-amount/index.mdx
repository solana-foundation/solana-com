## O que é a extensão Valor de UI Escalado?

A extensão de valor de UI escalado permite que os emissores apliquem um
multiplicador atualizável ao valor de UI de um token. Isso é útil para cenários
onde o token é usado para representar um ativo do mundo real, como uma ação ou
dividendo.

Por exemplo, um desdobramento de ações dobra o número de ações para todos os
detentores. Sem esta extensão, atualmente é impraticável emitir novos tokens
para todos os detentores durante um evento de desdobramento de forma eficiente e
escalável.

Com o modelo de extensão Token-2022, no entanto, temos a possibilidade de mudar
como o valor de UI dos tokens é representado. Usando a extensão ScaledUiAmount e
a instrução `amount_to_ui_amount`, você pode definir um multiplicador de UI em
seu token e buscar seu valor de UI a qualquer momento.
[código fonte](https://github.com/solana-program/token-2022/tree/main/program/src/extension/scaled_ui_amount).

Este recurso também pode ser usado para dividendos ou distribuição de
rendimentos.

Nota: Nenhum novo token é criado, o valor de UI retorna a quantidade bruta de
tokens multiplicada pelo multiplicador atual. O recurso é inteiramente
cosmético.

<Callout type="info">
  Algumas extensões são incompatíveis entre si e você não pode habilitá-las
  simultaneamente no mesmo mint de token ou conta de token. Você não deve
  habilitar tanto a extensão `ScaledUiAmount` quanto a extensão
  `InterestBearingMint` no mesmo mint de token ou conta de token.
</Callout>
