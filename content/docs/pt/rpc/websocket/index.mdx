h1: Métodos Websocket RPC da Solana

Após conectar ao websocket RPC PubSub em `ws://<ADDRESS>/`:

- Envie solicitações de assinatura para o websocket usando os métodos abaixo
- Várias assinaturas podem estar ativas ao mesmo tempo
- Muitas assinaturas aceitam o parâmetro opcional
  [`commitment`](/docs/rpc/#configuring-state-commitment), definindo quão
  finalizada uma alteração deve estar para acionar uma notificação. Para
  assinaturas, se o commitment não for especificado, o valor padrão é
  `finalized`.

## Endpoint WebSocket RPC PubSub

Porta padrão: `8900`

- ws://localhost:8900
- http://192.168.1.88:8900
