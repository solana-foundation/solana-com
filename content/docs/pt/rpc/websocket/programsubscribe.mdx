h1: Método RPC programSubscribe
---

Inscreva-se em um programa para receber notificações quando os lamports ou dados
de uma conta pertencente ao programa especificado forem alterados

<APIMethod>

```jsonc !!request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "programSubscribe",
  "params": [
    // !hover 0
    "11111111111111111111111111111111",
    // !hover(1:4) 1
    {
      // !hover encoding
      "encoding": "base64",
      // !hover filters
      "filters": [{ "dataSize": 80 }]
    }
  ]
}
```

### !params

#### !! 0

!type string  
!required

Pubkey do `program_id`, como string codificada em base-58

#### !! 1

!type object

Objeto de configuração contendo os seguintes campos:

##### !! commitment

!type string

O commitment descreve quão finalizado está um bloco naquele momento. Veja
[Configurando State Commitment](/docs/rpc#configuring-state-commitment).

##### !! filters

!type array

Filtra resultados usando vários objetos de filtro. Veja
[Filtragem](/docs/rpc#filter-criteria).

<Callout type="info">
  A conta resultante deve atender a **TODOS** os critérios de filtro para ser
  incluída nos resultados retornados
</Callout>

##### !! encoding

!type string  
!values base58 base64 base64+zstd jsonParsed

Formato de codificação para dados da Conta

- `base58` é lento.
- A codificação `jsonParsed` tenta usar analisadores de estado específicos do
  programa para retornar dados de estado da conta mais legíveis e explícitos.
- Se `jsonParsed` for solicitado, mas um analisador não puder ser encontrado, o
  campo volta para a codificação `base64`, detectável quando o campo `data` for
  do tipo `string`.

### !!result

```jsonc !response
{
  "jsonrpc": "2.0",
  // !hover result
  "result": 24040,
  "id": 1
}
```

!type integer

ID de assinatura (necessário para cancelar a assinatura)

</APIMethod>

#### Formato de notificação

O formato de notificação é um objeto de program account <b>único</b> como visto
no método HTTP RPC [getProgramAccounts](/docs/rpc/http/getprogramaccounts).

Codificação Base58:

```json
{
  "jsonrpc": "2.0",
  "method": "programNotification",
  "params": {
    "result": {
      "context": {
        "slot": 5208469
      },
      "value": {
        "pubkey": "H4vnBqifaSACnKa7acsxstsY1iV1bvJNxsCY7enrd1hq",
        "account": {
          "data": [
            "11116bv5nS2h3y12kD1yUKeMZvGcKLSjQgX6BeV7u1FrjeJcKfsHPXHRDEHrBesJhZyqnnq9qJeUuF7WHxiuLuL5twc38w2TXNLxnDbjmuR",
            "base58"
          ],
          "executable": false,
          "lamports": 33594,
          "owner": "11111111111111111111111111111111",
          "rentEpoch": 636,
          "space": 80
        }
      }
    },
    "subscription": 24040
  }
}
```
